# Planning Center MCP Server

**Category:** Business & Commerce MCP Servers  
**Tags:** CRM, Events, Nonprofit

## Overview
The Planning Center MCP Server is an MCP-compatible integration for Planning Center, a suite of church management tools used to organize information, coordinate events, communicate with teams, and connect with congregations. It is provided via Pipedream Connect as a static MCP endpoint that can be added to any supported chat client or MCP-capable application.

## Features
- **MCP-compatible server**: Exposes Planning Center functionality through the Model Context Protocol, enabling use with MCP-aware agents and chat clients.
- **Static server endpoint**: Uses a single static URL for all clients:  
  `https://mcp.pipedream.net/v2`
- **Client-agnostic integration**: Can be added to different chat clients and MCP-enabled applications; authentication occurs when adding the server.
- **Backed by Planning Center platform**: Integrates with Planning Center’s church management tools designed to:
  - Organize church and congregation information
  - Coordinate events and activities
  - Communicate with staff and volunteer teams
  - Support engagement and connection with the congregation

## Usage
- Add the MCP server to your preferred MCP-compatible chat client or application using the static URL.
- Authenticate during setup within your client to connect to Planning Center via the MCP server.
- Refer to the provider’s MCP configuration documentation ("Configuration" page) for client-specific setup steps.
