# timeBuzzer MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Pipedream  
**Website:** https://mcp.pipedream.com/app/timebuzzer

## Description
The timeBuzzer MCP Server is a Model Context Protocol (MCP) integration for timeBuzzer that allows MCP-compatible clients (such as chat-based tools) to perform accurate, programmatic time tracking operations.

## Features
- MCP server endpoint for timeBuzzer: `https://mcp.pipedream.net/v2` (static URL for all compatible clients)
- Supports authentication when adding the server to an MCP client or application
- Designed for precise and efficient time tracking workflows
- Accessible from multiple MCP-enabled chat or AI clients
- Central configuration and setup guidance available via the Pipedream MCP Configuration page

- **Protocol:** Model Context Protocol (MCP)
