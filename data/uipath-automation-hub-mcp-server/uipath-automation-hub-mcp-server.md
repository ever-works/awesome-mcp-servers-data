## Overview
The UiPath Automation Hub MCP Server is an integration that connects UiPath Automation Hub to MCP-compatible AI tools. Through Pipedreamâ€™s MCP server, it enables discovery and centralized management of automation opportunities from within AI-powered developer and productivity environments.

Source: <https://mcp.pipedream.com/app/uipath_automation_hub>

## Features
- **MCP Server Integration**
  - Exposes UiPath Automation Hub via the Model Context Protocol (MCP) so MCP clients can interact with automation data.
- **UiPath Automation Hub Connectivity**
  - Connects directly to a UiPath Automation Hub account to access automation opportunities and related resources.
- **Discovery of Automation Opportunities**
  - Allows MCP clients to discover and reference automation opportunities managed in UiPath Automation Hub.
- **Centralized Management Access**
  - Provides a centralized interface (through MCP) to view and work with automation opportunities from multiple AI tools.
- **AI Tool Integration**
  - Works with AI tools that support MCP, including:
    - Cursor
    - Claude
    - Windsurf
- **RPA & Workflow Automation Context**
  - Designed for robotic process automation (RPA) and workflow-automation use cases where automation ideas and implementations are tracked in UiPath Automation Hub.
