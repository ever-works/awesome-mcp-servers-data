# Blocknative MCP Server

**Category:** Blockchain & Crypto MCP Servers  
**Slug:** `blocknative-mcp-server`

## Overview
The Blocknative MCP Server provides Model Context Protocol (MCP) access to Blocknative’s real-time core blockchain infrastructure and mempool data. It is designed to let AI agents and tools interact programmatically with blockchain and mempool information through a standardized MCP endpoint.

- **Provider / Brand:** Blocknative (via Pipedream)
- **Primary Use Case:** Real-time blockchain and mempool data access for AI and tools
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server**  
  - Exposes Blocknative capabilities over the Model Context Protocol.  
  - Can be added as a server in MCP-compatible chat or agent clients.

- **Static MCP URL**  
  - Single static endpoint: `https://mcp.pipedream.net/v2`.  
  - Same URL for all supported clients; authentication happens when adding the server to your app.

- **Real-time blockchain infrastructure access**  
  - Integrates with Blocknative’s core blockchain infrastructure.  
  - Intended for applications that need up-to-date chain data and mempool visibility.

- **Mempool data access**  
  - Designed to provide real-time mempool information suitable for AI and agent workflows (exact tool list is loaded dynamically and not detailed in the source content).

- **Client-agnostic integration**

  - Documentation hints at client-specific setup instructions via a configuration page.

- **Configuration via Pipedream**  
  - Server is hosted/managed through Pipedream’s MCP infrastructure.
