# Qwen_Max MCP Server

## Overview
Qwen_Max MCP Server is an open-source Model Context Protocol (MCP) server that exposes Qwen Max models as tools, allowing MCP-compatible clients to interact with Qwen LLMs.

Source: <https://github.com/66julienmartin/MCP-server-Qwen_Max>

## Features
- **MCP server implementation** for the Qwen Max model.
- **Exposes Qwen models as MCP tools**, making them accessible to any MCP-compatible client.
- **Supports interaction with Qwen LLMs** through the standardized MCP tooling interface.
- **Dockerfile included** for containerized deployment.
- **Node.js/TypeScript project setup**:
  - `package.json` and `package-lock.json` for dependency management.
  - `tsconfig.json` for TypeScript configuration.
- **Smithery configuration (`smithery.yaml`)** for integration/registration with MCP tooling directories that use this format.
