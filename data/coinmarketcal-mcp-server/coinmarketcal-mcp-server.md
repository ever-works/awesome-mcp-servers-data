# CoinMarketCal MCP Server

An MCP (Model Context Protocol) server that exposes CoinMarketCal’s cryptocurrency events database so MCP-compatible clients can query upcoming and historical crypto events.

- **Name:** CoinMarketCal MCP Server  
- **Category:** Blockchain & Crypto MCP Servers  
- **Brand:** CoinMarketCal  
- **Source URL:** https://mcp.pipedream.com/app/coinmarketcal  
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features

- Provides access to CoinMarketCal’s crypto events database via MCP.  
- Supports querying **upcoming** cryptocurrency events.  
- Supports querying **historical** cryptocurrency events.  
- Exposes data suitable for use in MCP-compatible chat or agent clients.  
- Uses a **single static MCP server URL** (`https://mcp.pipedream.net/v2`) that works across all clients.  
- Authentication is handled when adding/configuring the server within your application.  
- Built and hosted via **Pipedream Connect** infrastructure.

## Integration & Usage

- Add the static MCP server URL to any MCP-enabled chat client or application.  
- Configure authentication as required by your client when registering the server.  
- Refer to the platform’s configuration documentation (e.g., a “Configuration” page) for client-specific setup steps.
