## Overview

The Sanity MCP server connects your structured content directly to AI agents like Claude and Cursor, letting you manage content through natural language conversations. The server is a secure gateway between your agent and your Sanity projects.

## Key Features

- **Schema-Awareness**: AI models understand your content structure and perform operations through natural language instructions
- **40+ Tools**: Document operations, schema management, content releases, and AI-powered media generation
- **Real-Time Schema Understanding**: The MCP server reads your deployed schema manifest, giving the AI a perfect, real-time understanding of your content model
- **Governed Access**: Direct, governed access to structured content

## Hosted Infrastructure

The Sanity MCP server is hosted on Sanity's own infrastructure at https://mcp.sanity.io. It follows Anthropic's official MCP specification and works with any MCP-compatible client.

## Benefits for Structured Content

- **No Duplicate Data**: Eliminates duplicate data stores and custom integration work
- **Simple Conversations**: Transform content operations from complex queries to simple conversations
- **Team Superpowers**: Give your team superpowers without sacrificing structure
- **AI Content Operating System**: Powers intelligence with structure and agents with context
