# PDF Charts MCP Server

**Description**

PDF Charts MCP Server exposes PDF Chartsâ€™ drag-and-drop chart design and PDF generation capabilities over a standard MCP endpoint. It lets MCP-compatible agents create chart layouts, bind them to dynamic data via API calls, and download the resulting PDFs.

**Features**

- MCP-compatible server exposed at a static URL (`https://mcp.pipedream.net/v2`).
- Drag-and-drop chart design interface (via PDF Charts) for creating custom chart layouts.
- Chart customization capabilities (e.g., layout and styling) driven by the PDF Charts design tools.
- Ability to add dynamic data to charts through simple API calls from MCP agents.
- PDF generation from designed charts, producing downloadable chart PDFs.
- Works with any MCP client that can connect to the static server URL and handle authentication.

**Integration Details**

- Accessed via the MCP server URL: `https://mcp.pipedream.net/v2`.
- Authentication occurs when adding the server to your MCP-compatible application or chat client.
- Configuration guidance is provided through the associated configuration page (outside this summary).
