# Mailmodo MCP Server

Mailmodo MCP Server is an MCP (Model Context Protocol) server integration that connects Mailmodo with MCP-compatible applications, allowing you to create and send interactive email campaigns from within your MCP tools.

## Overview
- **Type:** MCP server integration
- **Category:** Business & Commerce – MCP Servers
- **Use cases:** Email marketing, sending campaigns, interactive email flows from MCP-enabled apps
- **Endpoint URL:** `https://mcp.pipedream.net/v2`

## Features
- **Mailmodo integration via MCP**  
  - Connects Mailmodo to MCP-compatible chat or agent applications.  
  - Uses a static MCP server URL (`https://mcp.pipedream.net/v2`) for all clients.

- **Account connection workflow**  
  - Option to configure Mailmodo by connecting your Mailmodo account to the MCP server.  
  - Account status checking (“Checking your account…”) to verify connection.

- **Client-agnostic setup**  
  - The same MCP server URL works across different chat clients and MCP applications.  
  - Authentication is performed when you add the server to each specific client/app.

- **Tooling within MCP**  
  - Exposes Mailmodo-related tools/actions inside MCP-compatible apps (e.g., to create and send email campaigns).  
  - Tools are loaded dynamically in the client interface (“Loading actions…”, “Loading available tools…”).

- **Configuration guidance**  
  - Step-by-step guidance to copy the MCP server URL and add it to supported clients.  
  - Link to a full configuration page for detailed setup instructions.
