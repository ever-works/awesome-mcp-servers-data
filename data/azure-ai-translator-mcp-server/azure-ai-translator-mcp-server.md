# Azure AI Translator MCP Server

**Category:** AI Integration – MCP Servers  
**Brand:** Microsoft Azure  
**Source:** https://mcp.pipedream.com/app/microsoft_azure_ai_translator

## Overview
Azure AI Translator MCP Server is an MCP-compatible integration for Microsoft Azure AI Translator. It enables MCP-based agents and applications to perform instant or batch text translation across more than 100 languages, leveraging Azure’s machine translation capabilities.

## Features
- **MCP-compatible server**: Exposes Azure AI Translator capabilities through the Model Context Protocol (MCP) for use in MCP-based agents and chat clients.
- **Instant text translation**: Translate text in real time across 100+ languages.
- **Batch translation**: Submit multiple texts or larger workloads for batch translation.
- **100+ language support**: Access a broad language set powered by Azure AI Translator.
- **Static server URL**: Use a single, static MCP endpoint that works for all supported clients:  
  - MCP server URL: `https://mcp.pipedream.net/v2`
- **Client-agnostic setup**: Works with any compatible MCP client; you authenticate when adding the server to your application.
- **Configuration docs**: A dedicated configuration page provides setup instructions for different chat clients (linked from the app page).

## Integration & Usage
- Add the MCP server URL `https://mcp.pipedream.net/v2` to your MCP-enabled app or chat client.
- Authenticate within your client when prompted during server setup.
- Use available MCP tools/endpoints (as exposed by your client) to perform single or batch translations.
