# Claid AI MCP Server

**Category:** Media Processing MCP Servers  
**Tags:** image-processing, e-commerce, media-management

## Overview
Claid AI MCP Server is an MCP-compatible service (via Pipedream) for transforming e-commerce visuals. It focuses on automated product image enhancement, including upscaling, background removal, and color/lighting correction to create professional-looking product photos.

**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features

- **Static MCP Server Endpoint**  
  - Single static URL (`https://mcp.pipedream.net/v2`) usable for all clients.  
  - Authentication is handled when adding the server to your MCP-compatible application.

- **E-commerce Visual Optimization**  
  - Tailored for e-commerce product imagery.  
  - Supports AI-based enhancements to improve visual quality and conversion potential.

- **Available Tools / Actions**  
  The MCP server exposes three main actions as tools:

  1. **Upscale Image**  
     - Enlarges images to a higher resolution.  
     - Improves clarity and sharpness of product visuals.

  2. **Remove Background**  
     - Automatically removes the background of an image.  
     - Isolates the main subject (e.g., product) for clean, professional presentation.  
     - Useful for creating marketplace-ready or catalog-ready product images.

  3. **Correct Color & Lighting**  
     - Automatically adjusts color balance and lighting.  
     - Uses HDR-style processing to enhance dynamic range.  
     - Helps recover detail in dark or overexposed areas.  
     - Produces more consistent, true-to-life product photos.

- **Client Integration**  
  - Designed to be added to chat or MCP-compatible clients as a server.  
  - Works with multiple clients via the same static URL.
