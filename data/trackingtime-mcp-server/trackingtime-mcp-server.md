# TrackingTime MCP Server

## Overview
TrackingTime MCP Server is an MCP-compatible integration that connects TrackingTime’s time-tracking capabilities to MCP-based applications and chat clients. It is designed for freelancers and teams who want to track time and manage productivity from within their existing MCP ecosystem.

- **Category:** Business & Commerce MCP Server
- **Use cases:** Time tracking for projects and clients, team productivity tracking, integrating TrackingTime into chat or MCP-enabled tools.

## Features
- **MCP-compatible server endpoint**  
  - Static server URL: `https://mcp.pipedream.net/v2`  
  - Works with any supported MCP client.

- **Account integration**  
  - Connect a TrackingTime account through the MCP server.  
  - Select clients / accounts after connection (as implied by "select your client below").

- **Authentication at application level**  
  - Authentication occurs when adding the server to the MCP-compatible application, rather than per-URL.

- **Chat client integration**  
  - Designed to be added to different chat clients that support MCP.  
  - Provides tools and actions inside those clients once configured (e.g., “Available tools,” “Loading actions…” placeholders).

- **Central configuration documentation**  
  - Can be configured using a general MCP configuration flow via a dedicated configuration page on Pipedream.

## Technical Details
- **Server URL:** `https://mcp.pipedream.net/v2`  
- **Hosting / provider:** Pipedream (MCP server runtime and configuration).
