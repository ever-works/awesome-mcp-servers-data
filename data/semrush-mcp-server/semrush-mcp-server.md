# Semrush MCP Server

**Category:** Business & Commerce MCP Servers  
**Tags:** SEO, marketing analytics, competitor research

## Overview
The Semrush MCP Server is an MCP integration that connects your applications to Semrush. It provides access to SEO, content marketing, competitor research, PPC, and social media marketing data through a single MCP endpoint.

## Features
- **Unified MCP Endpoint**  
  - Single static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same URL works across all compatible MCP clients

- **Semrush Data Access via MCP**  
  - SEO-related data access  
  - Content marketing–related data access  
  - Competitor research–related data access  
  - PPC (pay-per-click) marketing–related data access  
  - Social media marketing–related data access

- **Client-Agnostic Integration**  
  - Can be added to multiple chat or MCP-compatible clients  
  - Authentication handled when adding the server to each application

- **Configuration Resources**  
  - Dedicated configuration documentation available through the Configuration page (client-specific setup guidance)

## Setup
- Use the static MCP server URL: `https://mcp.pipedream.net/v2` when adding the server to your MCP-compatible client.  
- Authenticate within your client during server setup.
