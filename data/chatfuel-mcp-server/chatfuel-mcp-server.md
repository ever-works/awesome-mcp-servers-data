## Chatfuel MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Chatfuel  
**Source:** https://mcp.pipedream.com/app/chatfuel_dashboard_api_

### Description
The Chatfuel MCP Server exposes Chatfuel’s Dashboard API through the Model Context Protocol (MCP), allowing applications and development tools to interact with internal methods of the Chatfuel no-code chatbot platform for Facebook, Instagram, and Messenger.

The server is hosted via Pipedream Connect and is accessed through a static MCP endpoint.

### Features
- **MCP-compatible server** providing access to Chatfuel’s Dashboard API.
- **Integration with Chatfuel’s internal dashboard methods** used by the Chatfuel no-code platform.
- **Supports social channels**: designed for chatbots on Facebook, Instagram, and Messenger.
- **Static MCP endpoint** for all clients: `https://mcp.pipedream.net/v2`.
- **Client-agnostic configuration**: same server URL works across different MCP-capable clients; authentication occurs when you add the server to your application.
- **Configuration documentation available** via Pipedream’s Configuration page for setup details.

### Technical Details
- **MCP endpoint:** `https://mcp.pipedream.net/v2`  
- **Integration platform:** Pipedream Connect
