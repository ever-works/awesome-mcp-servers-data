## Overview
The ChartMogul MCP Server connects MCP-compatible tools and chat clients to ChartMogul, enabling retrieval and analysis of subscription analytics and revenue metrics through a standardized MCP endpoint.

## Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Works for all clients; authentication is handled when adding the server to each application.

- **ChartMogul integration**  
  - Connects to a ChartMogul account for access to subscription and revenue data.  
  - Designed for subscription analytics and revenue reporting use cases.

- **Client-agnostic configuration**  
  - Can be added to multiple MCP-capable chat clients or applications.  
  - Setup guidance available per client via “Select your chat client to learn how to get started.”

- **Central configuration documentation**  
  - Additional setup and configuration details accessible via a dedicated configuration page (linked as “Configuration page”).

## Usage
1. Connect your ChartMogul account in the Pipedream interface.  
2. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.  
3. Add the server URL to your MCP-compatible app or chat client and authenticate.  
4. Use the available tools exposed by the server to query subscription analytics and revenue metrics.
