## Description
SMTP2GO MCP Server is a Model Context Protocol (MCP) server that enables MCP-compatible clients to send, track, and manage transactional and marketing email programmatically using SMTP2GO’s email delivery infrastructure.

## Features
- MCP server endpoint for SMTP2GO: `https://mcp.pipedream.net/v2` (static URL for all clients).
- Supports authentication when adding the server to an MCP-compatible application.
- Programmatic sending of email via SMTP2GO.
- Email tracking capabilities (e.g., for delivery/engagement tracking) through SMTP2GO’s infrastructure.
- Designed for both transactional and marketing email use cases.
- Usable from multiple MCP-aware chat or agent clients (configured client-side).
