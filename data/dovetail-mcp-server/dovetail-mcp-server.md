# Dovetail MCP Server

An MCP server that connects MCP-compatible environments to Dovetail’s customer insights hub, enabling access to research and feedback data.

- **Category:** Business & Commerce – MCP Servers  
- **Brand:** Dovetail  
- **Source URL:** https://mcp.pipedream.com/app/dovetail  
- **Slug:** `dovetail-mcp-server`

## Features

- **MCP server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Single URL works for all Dovetail clients / environments.

- **Dovetail integration**  
  - Connects MCP-based chat or agent applications to Dovetail’s customer insights hub.  
  - Provides access to research and customer feedback data stored in Dovetail (details of individual tools/actions are not exposed in the provided content).

- **Account-based configuration**  
  - Connect a Dovetail account and select the appropriate client to start using the server.  
  - Authentication is performed when the server is added to the MCP-compatible application, not at URL copy time.

- **Client-agnostic setup**  
  - Instructions available per supported chat client to help add the MCP server to different applications.  
  - Reference to a full configuration page for more detailed setup steps.

- **Tooling integration (MCP tools)**  
  - Designed to expose “available tools” (actions) from Dovetail through MCP (exact tools are not listed in the provided content).  
  - Indicates dynamic loading of actions and tools from the Dovetail integration.

## Usage

1. Connect your Dovetail account in the Pipedream Dovetail MCP Server interface.  
2. Select your Dovetail client within the interface.  
3. Copy the static MCP server URL: `https://mcp.pipedream.net/v2`.  
4. Add this MCP server URL to your chosen MCP-compatible chat/agent client and authenticate when prompted.  
5. Use the client’s configuration or the linked configuration page for further setup details.
