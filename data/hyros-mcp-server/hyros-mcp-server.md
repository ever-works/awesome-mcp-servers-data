# Hyros MCP Server

Advertising Sales & AI Sales Attribution

## Overview
Hyros MCP Server is an MCP (Model Context Protocol) server integration that connects AI agents and chat clients to Hyros advertising and sales attribution data. It enables programmatic access to marketing performance insights so agents can analyze and optimize advertising and sales funnels.

- **Category:** Business & Commerce – MCP Servers
- **Provider / Brand:** Hyros (via Pipedream Connect)
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- **MCP Server Integration**  
  - Exposes Hyros capabilities through the MCP protocol for use in compatible chat and AI applications.
  - Uses a single static base URL for all clients; authentication is handled when adding the server to the application.

- **Access to Hyros Data**  
  - Provides AI agents access to Hyros advertising, sales, and attribution data.  
  - Supports use cases around marketing performance analysis and optimization.

- **Static Server URL**  
  - A single MCP server URL works across implementations:  
    - `https://mcp.pipedream.net/v2`

- **Client Integration Guidance**  
  - Instructions available (via the Configuration page on the source site) for adding the server to different chat clients.

- **Pipedream Connect Integration**  
  - Hosted and powered by Pipedream Connect, integrating Hyros with other tools and workflows that support MCP servers.

## Use Cases
- Enable AI copilots to inspect and interpret advertising performance and sales attribution data.
- Build conversational interfaces that answer questions about marketing performance using Hyros data.
- Automate reporting and analysis tasks related to ad campaigns and sales funnels.

## Access & Configuration
- **MCP Server URL:** `https://mcp.pipedream.net/v2` (static for all clients).  
- Authentication is performed when the server is added within the target application or chat client.  
- Additional setup details are available via the provider’s configuration documentation (not reproduced here).
