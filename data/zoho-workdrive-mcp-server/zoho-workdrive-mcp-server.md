# Zoho WorkDrive MCP Server

An MCP server integration for Zoho WorkDrive that enables MCP-based tools and chat clients to manage online team file storage and collaboration via Zoho WorkDrive APIs.

## Overview
- **Type:** MCP server / integration
- **Category:** File management MCP servers
- **Service:** Zoho WorkDrive (online team file management and collaboration)
- **Provider Platform:** Pipedream Connect
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **Zoho WorkDrive integration**
  - Connects MCP-based tools and chat clients to Zoho WorkDrive APIs.
  - Enables interaction with online team file storage and collaboration features provided by Zoho WorkDrive (exact API operations not detailed on the page).

- **Static MCP server endpoint**
  - Uses a single static server URL (`https://mcp.pipedream.net/v2`) that works for every supported MCP client.
  - Authentication is handled when adding the server to the client application.

- **Multi‑client compatibility**
  - Designed to be added to various chat clients that support MCP servers.
  - Configuration guidance available via a dedicated configuration page (linked as “Configuration page”).

- **Hosted by Pipedream**
  - Runs on Pipedream’s MCP infrastructure (Pipedream Connect).
  - Uses Pipedream’s standard terms and privacy controls.

## Usage
- Add the MCP server URL `https://mcp.pipedream.net/v2` to your MCP-compatible chat client or application.
- Authenticate with Zoho / Pipedream when prompted during setup.
- Optionally refer to the linked configuration page for client-specific setup steps.
