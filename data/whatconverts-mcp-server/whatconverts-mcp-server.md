# WhatConverts MCP Server

An MCP Server integration for WhatConverts that brings call tracking and lead tracking data into MCP-based workflows for marketing analytics.

- **Type:** MCP Server integration

- **Brand:** WhatConverts (via Pipedream Connect)
- **Primary use cases:**
  - Integrate call tracking data into MCP workflows
  - Integrate lead tracking data into MCP workflows
  - Support marketing analytics and reporting using WhatConverts data

## Features
- **Static MCP Server URL**  
  - Single shared endpoint for all clients:  
    `https://mcp.pipedream.net/v2`  
  - Authentication performed when adding the server to your MCP-compatible application.

- **Call Tracking Integration**  
  - Access call tracking data from WhatConverts through MCP workflows (e.g., for analysis, reporting, or routing in connected tools).

- **Lead Tracking Integration**  
  - Access lead tracking data from WhatConverts to power marketing analytics and workflow automation.

- **MCP Client Compatibility**  
  - Can be added to various MCP-compatible chat or agent clients (configuration done in the client, using the static URL and authentication).

- **Centralized Configuration Documentation**  
  - Full configuration and setup details are available via a dedicated configuration page (linked from the app page).
