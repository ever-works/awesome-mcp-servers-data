# Instant MCP Server

**Website:** https://www.instantmcp.com/

An MCP server for Instant (InstantDB) that allows MCP clients to interact with Instant’s application database services via OAuth-based authentication.

---

## Features

- MCP server integration for Instant (InstantDB)
- Enables interaction with Instant’s application database services from MCP-compatible clients
- OAuth authentication for secure access
- Listed within a marketplace (InstantMCP) for discovering MCP servers
- Supports customizable subscriptions for MCP server access and monetization
