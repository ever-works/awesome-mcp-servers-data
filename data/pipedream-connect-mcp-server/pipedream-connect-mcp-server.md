# Pipedream Connect MCP Server

## Overview
Pipedream Connect MCP Server exposes Pipedream Connect’s unified API integrations through the Model Context Protocol (MCP), allowing applications and AI agents to access thousands of APIs via a single SDK and a common MCP endpoint.

- **Name:** Pipedream Connect MCP Server  
- **Category:** Workflow Automation MCP Servers  
- **Provider / Brand:** Pipedream  
- **Website / Source:** https://mcp.pipedream.com/app/pipedream_connect  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **Unified access to APIs**  
  - Access thousands of API integrations from a single MCP server.  
  - Use one SDK to work with many third‑party services.

- **MCP compatibility**  
  - Exposes integrations via the Model Context Protocol for use in MCP‑compatible apps and AI agents.  
  - Static server URL (`https://mcp.pipedream.net/v2`) usable across different MCP clients.

- **Managed authentication**  
  - Centralized, managed auth for connected APIs.  
  - Uses approved client IDs, so apps don’t need to manage OAuth client registration per integration.

- **Durable components & infrastructure**  
  - Provides durable components for building and running workflows.  
  - Backed by infrastructure for running serverless functions as part of integrations.

- **Developer experience**  
  - Single SDK to integrate multiple APIs into applications or AI agents.  
  - Configuration guides available via a dedicated configuration page.

## Integration & Usage
- **MCP Server endpoint:**  
  - Use `https://mcp.pipedream.net/v2` as the MCP server URL in your MCP‑compatible client or application.  
  - Authentication occurs when adding/configuring the server in your app or chat client.
