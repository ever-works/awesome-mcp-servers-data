# FormTitan MCP Server

**Category:** Business & Commerce MCP Servers  
**Slug:** `formtitan-mcp-server`  
**Website:** https://mcp.pipedream.com/app/formtitan

## Overview
FormTitan MCP Server exposes FormTitan’s no‑code Salesforce/CRM form and portal platform through the Model Context Protocol. It enables MCP-compatible clients (such as chat-based tools and other MCP-enabled apps) to build and manage dynamic forms, portals, and related CRM resources without writing code.

Titan (the underlying platform) is an all‑in‑one, no‑code solution for Salesforce and other CRMs, supporting portals, progressive and dynamic forms, document generation, scheduling, dashboards, and more.

## MCP Endpoint
- **Base MCP server URL:** `https://mcp.pipedream.net/v2`  
  This is a static URL used by all clients; authentication is handled when adding the server in each client.

## Features
- **Model Context Protocol integration**  
  - Exposes FormTitan’s capabilities as an MCP server.  
  - Allows MCP clients (e.g., AI chat tools) to interact with FormTitan resources.

- **No‑code Salesforce/CRM integration**  
  - Connects to Salesforce and other CRMs.  
  - Enables building CRM-integrated experiences without writing code.

- **Dynamic and progressive forms**  
  - Create dynamic forms that can change based on user input or context.  
  - Support for progressive forms that adapt or unfold over multiple steps.

- **Portals**  
  - Build CRM-connected portals (e.g., customer or partner portals) powered by FormTitan.

- **Document generation**  
  - Generate documents from CRM data using FormTitan’s document generation capabilities.

- **Scheduler**  
  - Use scheduling capabilities (e.g., appointments, bookings) integrated with CRM data.

- **Dashboards**  
  - Create dashboards that surface CRM and form data visually.

- **All-in-one platform access via MCP**  
  - Access multiple FormTitan components (forms, portals, docs, scheduler, dashboards) from a single MCP endpoint.

## Usage
- Add `https://mcp.pipedream.net/v2` as a server in any MCP-compatible client.  
- Authenticate within the client when prompted.  
- Once configured, use the MCP tools to build and manage FormTitan forms, portals, and related CRM-integrated resources.
