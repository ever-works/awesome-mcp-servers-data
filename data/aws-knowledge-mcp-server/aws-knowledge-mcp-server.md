# AWS Knowledge MCP Server

**Website:** https://knowledge-mcp.global.api.aws  
**Category:** Documentation & Learning Resources  
**Brand:** AWS

## Overview
AWS Knowledge MCP Server is an open Model Context Protocol (MCP) server that allows MCP-aware tools and agents to query AWS knowledge resources and documentation programmatically via the endpoint `https://knowledge-mcp.global.api.aws`.

## Features
- **Open MCP server endpoint**: Public MCP server available at `https://knowledge-mcp.global.api.aws`.
- **Programmatic access to AWS knowledge**: Designed for tools to query AWS knowledge resources and documentation rather than manual browsing.
- **MCP-aware tool integration**: Intended for integration with MCP-compatible clients, agents, or development tools that can consume MCP servers.
- **AWS documentation focus**: Targets AWS knowledge resources and documentation as its primary data source.
