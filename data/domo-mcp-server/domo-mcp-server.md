# Domo MCP Server

**Category:** Data Visualization  
**Brand:** Domo  
**Slug:** `domo-mcp-server`

## Description
Domo MCP Server is an MCP (Model Context Protocol) server integration for Domo’s AI and data products platform. It allows MCP-compatible tools and chat clients to interact with Domo for analytics and reporting use cases.

## Features
- MCP server endpoint for connecting Domo with MCP-compatible clients and tools
- Single static server URL usable across different clients
- Authentication handled when adding the server to each application
- Designed to interface with Domo’s AI and data products platform
- Supports analytics and reporting workflows driven via MCP tools

## Technical Details
- **MCP server URL:** `https://mcp.pipedream.net/v2`
- **Integration host:** Powered by Pipedream Connect

## Setup
1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add the server to your MCP-capable chat client or application.
3. Authenticate within your client when prompted.
4. (Optional) Refer to the platform’s configuration page for client-specific setup instructions.
