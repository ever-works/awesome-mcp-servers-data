# Congress.gov MCP Server

**Brand:** Library of Congress  
**Category:** Data Access & Integration – MCP Servers  
**Tags:** government, open-data, research

![Congress.gov MCP Server](https://www.congress.gov/img/congress-gov-screenshot.png)

## Overview
The Congress.gov MCP Server exposes the public Congress.gov API through the Model Context Protocol (MCP), enabling applications and AI agents to retrieve U.S. legislative data in a standardized way.

## Features
- **Access to Congress.gov API**: Interfaces with the official Congress.gov API, which shares U.S. Congressional data with the public.
- **MCP-compatible endpoint**: Provides a static MCP server URL that can be used from any compatible client.
- **Client-agnostic setup**: The same server URL works across different MCP clients and chat applications.
- **Authentication at client level**: Authentication is handled when adding the server to an application, enabling secure access to the underlying API.
- **Integration via Pipedream Connect**: Delivered as an MCP server powered by Pipedream’s infrastructure.

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Protocol:** Model Context Protocol (MCP)
- **Data Source:** Congress.gov (U.S. legislative data)

## Usage
- Add `https://mcp.pipedream.net/v2` as an MCP server endpoint in your MCP-compatible client or chat application.
- Configure authentication within your application when prompted by the client.
