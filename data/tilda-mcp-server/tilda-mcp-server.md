# Tilda MCP Server

MCP Server integration for Tilda’s website builder, enabling MCP-based management of websites and content built on Tilda.

- **Category:** Content Management MCP Servers  
- **Brand:** Tilda  
- **Source:** https://mcp.pipedream.com/app/tilda

## Features

- MCP server for integrating Tilda’s website builder with MCP-compatible clients.  
- Static MCP endpoint URL usable across clients:  
  - `https://mcp.pipedream.net/v2`  
- Works with multiple chat / MCP clients (added as a server within your chosen client).  
- Authentication handled at the time you add the server in your application.  
- Configuration details available via the Pipedream-hosted configuration page.

## Integration & Usage

- Add the MCP server URL to your preferred chat or MCP-compatible client.  
- Authenticate within the client to connect to your Tilda content.  
- Uses Pipedream Connect infrastructure as the underlying platform.
