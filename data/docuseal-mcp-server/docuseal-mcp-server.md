# DocuSeal MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** DocuSeal  
**Source:** https://mcp.pipedream.com/app/docuseal

## Description
DocuSeal MCP Server is an integration layer that connects DocuSeal’s e-signing automation API with MCP-compatible AI tools. It allows agents and developer tools to create, send, and manage electronic signing workflows programmatically through Pipedream’s MCP server.

## Features
- Integrates DocuSeal’s e-signing automation API with MCP-compatible agents
- Connects DocuSeal to AI tools such as Cursor, Claude, and Windsurf via Pipedream’s MCP server
- Programmatic creation of electronic signing workflows
- Programmatic sending of documents for e-signature
- Programmatic management of e-signing workflows (e.g., tracking and handling signing processes)
