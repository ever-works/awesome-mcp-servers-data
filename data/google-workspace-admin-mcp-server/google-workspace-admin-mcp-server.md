## Description
An MCP server, delivered via Pipedream, that connects to Google Workspace Admin APIs so MCP-aware clients can automate directory, groups, and collaboration administration tasks.

The server is exposed via a static MCP URL and can be added to compatible chat or developer clients to perform Google Workspace Admin reporting and activity queries.

## Configuration
- **MCP server URL (static):** `https://mcp.pipedream.net/v2`
- **Authentication:** Performed when adding the server to your MCP-aware application.
- **Client setup:** Add the MCP server URL to your chat or other MCP client; detailed instructions are available on the Pipedream configuration page.

## Features
- **Google Workspace Admin integration**
  - Connects directly to Google Workspace Admin APIs.
  - Designed for automating directory, group, and collaboration administration tasks.

- **MCP-compatible server**
  - Exposed as a single, static MCP endpoint usable across different MCP clients.
  - Can be integrated into chat-based and other MCP-aware applications.

- **Available tools (actions)**
  1. **List All Activities**  
     - Retrieves a report of all administrative activities performed for a Google Workspace account.
  2. **List Activities By Event Name**  
     - Retrieves a report of all activities for a specific event name.
  3. **List Activities By Event Name and Admin**  
     - Retrieves a report of all activities for a specific event name, filtered by a particular administrator.
  4. **List Activities By Admin**  
     - Retrieves a report of all Admin console activities done by a specific administrator.

- **Reporting & audit support**
  - Provides access to Google Workspace Admin activity logs for auditing and monitoring.
  - Enables fine-grained filtering by event type and administrator.
