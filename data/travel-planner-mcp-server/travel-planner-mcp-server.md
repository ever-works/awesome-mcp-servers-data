# Travel Planner MCP Server

A Model Context Protocol (MCP) server that integrates Google Maps APIs and travel planning services, enabling LLMs to perform travel-related tasks such as itinerary creation, location search, place details lookup, and travel/route time calculations.

## Features
- Model Context Protocol (MCP) server implementation
- Integration with Google Maps and related travel planning services
- Itinerary creation support from within LLM workflows
- Location search (e.g., finding destinations or points of interest)
- Place details lookup (e.g., detailed information about specific locations)
- Route and travel time calculations
- Designed for use directly from LLMs
- Compatible with Smithery and Claude Desktop (installable as an MCP server via Smithery)
