# KrispCall MCP Server

**Category:** Messaging MCP Servers  
**Brand:** krispcall  
**Source:** https://mcp.pipedream.com/app/krispcall

## Overview
KrispCall MCP Server is an MCP integration that connects a KrispCall account to AI tools via Pipedream’s MCP server. It exposes KrispCall’s AI-powered phone application capabilities to MCP-compatible tools for telephony-related workflows.

## Features
- Connects a KrispCall account through Pipedream’s MCP server
- Exposes KrispCall’s call management and telephony capabilities to MCP tools
- Enables AI-assisted call management workflows
- Integrates with AI tools such as:
  - Cursor
  - Claude
  - Windsurf
- Supports business-focused communications and telephony use cases

## Use Cases
- Managing calls and telephony flows from within AI coding or productivity environments
- Automating business communication workflows using AI tools connected via MCP
