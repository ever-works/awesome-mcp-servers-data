# Sellsy MCP Server

## Overview
Sellsy MCP Server connects the Sellsy platform—covering marketing, sales, invoicing, and cash-flow management—to the Model Context Protocol (MCP), enabling compatible chat or AI clients to interact with Sellsy through a standardized server endpoint.

- **Name:** Sellsy MCP Server  
- **Category:** Business & Commerce MCP Servers  
- **Tags:** CRM, sales, invoicing  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **Unified Sellsy access via MCP** – Exposes Sellsy’s marketing, sales, invoicing, and cash-flow management capabilities through a single MCP server endpoint.
- **Static endpoint for all clients** – Uses one static URL (`https://mcp.pipedream.net/v2`) that works for every compatible MCP client; authentication is handled when you add the server to your application.
- **Client-agnostic integration** – Designed to be added to various MCP-compatible chat or AI applications; configuration details are provided via a separate configuration page.
- **Pipedream-backed infrastructure** – Runs on Pipedream Connect infrastructure for hosting and serving the MCP server.

## Setup
1. Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add the server in your MCP-compatible client (chat app or other tool) and complete authentication during setup.
3. Refer to the platform’s "Configuration" page for detailed client-specific instructions.
