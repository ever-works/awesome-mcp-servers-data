# CommPeak MCP Server

**Description**  
CommPeak MCP Server is an MCP-compatible integration endpoint that connects applications to CommPeak’s cloud contact center solution to help automate and manage sales and communication workflows.

**Category**  
Messaging MCP Servers

**Features**  
- Static MCP server URL usable by any compatible client: `https://mcp.pipedream.net/v2`  
- Connects to CommPeak’s cloud contact center platform  
- Supports automation of sales workflows (e.g., outbound/inbound contact center processes)  
- Supports automation and management of communication workflows (e.g., call and messaging flows)  
- Authentication handled when adding the server within the client application  
- Usable across multiple chat or MCP-enabled clients

**Integration & Usage**  
- Add the MCP server URL to any supported chat client or MCP-enabled app  
- Configure per-client authentication during setup  
- Additional configuration guidance available via the Pipedream configuration documentation
