## Features
- **MCP-compatible server for Zoho Tables**
  - Integrates Zoho Tables with MCP-based agents, tools, and workflows.
  - Provides access to collaborative work management capabilities.
  - Exposes database-style tables for structured data handling.
- **Shared static endpoint**
  - One universal MCP URL (`https://mcp.pipedream.net/v2`) for all supported clients.
  - Simplifies configuration across different chat or agent clients.
- **Client-agnostic integration**
  - Works with multiple MCP-enabled chat clients and applications.
  - Configuration guidance available via a dedicated configuration page (client-specific setup instructions).  
- **Authentication at connection time**
  - Authentication handled when the server is added to a client or application (exact auth method depends on client and Pipedream Connect flow).

## Usage

- Follow client-specific instructions on the configuration page for setup and authentication.
