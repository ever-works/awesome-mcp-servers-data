# Path of Exile MCP Server

Path of Exile MCP Server exposes Path of Exile game data and services through the Model Context Protocol (MCP), enabling programmatic access for MCP-compatible clients.

- **Category:** Game Server MCP Implementations  
- **Brand:** Grinding Gear Games  
- **Source:** https://mcp.pipedream.com/app/path_of_exile

## Features

- Exposes Path of Exile game data and services via the Model Context Protocol (MCP)
- Compatible with MCP-enabled / MCP-compatible clients and tools
- Single, static MCP server endpoint for all clients:
  - `https://mcp.pipedream.net/v2`
- Authentication handled when adding the server to your application
- Can be integrated into chat-based or other MCP-aware applications

## Integration & Configuration

- Add the MCP server by pointing your client to the static URL: `https://mcp.pipedream.net/v2`
- Follow your chat client or MCP client’s instructions to register the server
- Additional configuration details are available via the provider’s configuration documentation (Pipedream Configuration page)
