# WOXO MCP Server

**Category:** Media Processing MCP Servers  
**Brand:** WOXO  
**Slug:** `woxo-mcp-server`

## Overview
WOXO MCP Server is an MCP-compatible server that connects your WOXO account to AI tools (such as Cursor, Claude, and Windsurf), enabling programmatic creation and editing of videos at scale using spreadsheet data.

Source: <https://mcp.pipedream.com/app/woxo>

## Features
- MCP-compatible server for WOXO
- Connects WOXO to AI tools like Cursor, Claude, and Windsurf via Pipedream's MCP server
- Programmatic video creation from spreadsheet data
- Programmatic video editing from spreadsheet data
- Designed for large-scale / bulk video generation workflows
- Automation of media generation tasks using WOXO through MCP-enabled agents
