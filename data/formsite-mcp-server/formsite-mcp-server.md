# Formsite MCP Server

**Category:** business-commerce-mcp-servers  
**Brand:** Formsite  
**Slug:** formsite-mcp-server

## Overview
Formsite MCP Server is a Model Context Protocol (MCP) server integration that connects MCP-compatible clients and AI agents with Formsite. It enables the creation and management of professional online forms, including order forms and surveys, using Formsite’s drag‑and‑drop form builder and integrations.

- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP integration**
  - Exposes Formsite capabilities through the Model Context Protocol for use by AI agents and MCP-compatible clients.
  - Uses a single static MCP server URL that works across compatible clients.

- **Form creation and management**
  - Create online forms using Formsite’s drag‑and‑drop builder.
  - Supports professional forms such as registration forms, contact forms, and more.
  - Build and manage order forms for commerce use cases.
  - Build and manage surveys for data collection and feedback.

- **Workflow and integrations**
  - Integrates with other services via Formsite’s integration capabilities (as exposed through the MCP server and underlying Formsite account).

- **Client setup**
  - Static MCP server URL: `https://mcp.pipedream.net/v2` for all supported clients.
  - Authentication is performed when adding the server to the chosen application or chat client.
  - Configuration details are available via a central configuration page (in the hosting environment, e.g., Pipedream Connect).

- **Hosting environment**
  - Delivered via Pipedream Connect as the MCP host for the Formsite integration.
