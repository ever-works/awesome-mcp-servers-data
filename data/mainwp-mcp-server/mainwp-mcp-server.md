# MainWP MCP Server

**Category:** Server Management Tools  
**Website / Source:** https://mcp.pipedream.com/app/mainwp  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Description
MainWP MCP Server exposes the MainWP WordPress management plugin as an MCP-compatible server, allowing you to integrate MainWP’s free, self-hosted, open source WordPress management capabilities into MCP-based automation and management solutions.

## Features
- Integrates MainWP (WordPress management plugin) with MCP-based applications.
- Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) for all clients.
- Authentication performed when adding the server to your MCP-compatible application.
- Designed for managing WordPress websites through automated or chat-based tools.
- Built and hosted via Pipedream Connect infrastructure.

## Pricing
- Described as **free**, matching MainWP’s free, self-hosted, open source model.  
- No additional pricing details are provided in the available content.
