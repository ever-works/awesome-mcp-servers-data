# Liondesk MCP Server

**Category:** Business & Commerce · MCP Servers  
**Tags:** CRM, Real Estate, Lead Management

## Overview
The Liondesk MCP Server connects MCP-compatible clients to LionDesk, a real-estate-focused CRM platform. It lets you access LionDesk’s lead, contact, transaction, and communication management capabilities directly from MCP-based applications via a static MCP server endpoint.

## Features
- **MCP-compatible server**
  - Exposes LionDesk CRM functionality as tools/actions to MCP clients.
  - Uses a single static server URL (`https://mcp.pipedream.net/v2`) for all clients.
- **LionDesk CRM integration**
  - Integrates with LionDesk’s real-estate-focused CRM platform.
  - Access to lead management features.
  - Access to contact management features.
  - Access to transaction management features.
  - Access to communication management features (e.g., centralized communications in LionDesk).
- **Account-based access**
  - Connects using your LionDesk account.
  - Authentication occurs when adding the server to the client application.
- **Client-agnostic setup**
  - Can be added to different MCP-compatible chat or agent clients.
  - Central configuration via the generic MCP server URL.

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Integration Host:** Pipedream MCP platform
- **Usage Pattern:**
  1. Connect your LionDesk account in Pipedream MCP.
  2. Copy the static MCP server URL.
  3. Add the server to any supported MCP client.
