# Nozbe Teams MCP Server

Cloud-based task management MCP server that connects Nozbe Teams with MCP-compatible clients for automated task and project handling.

## Basic Info
- **Name:** Nozbe Teams MCP Server  
- **Category:** Project Management MCP Servers  
- **Brand:** Nozbe  
- **Source URL:** https://mcp.pipedream.com/app/nozbe_teams  
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- Exposes Nozbe Teams’ cloud-based team task management capabilities via the MCP protocol.  
- Supports integration with any MCP-compatible client through a single static MCP server URL.  
- Authentication handled when adding the server to your MCP-enabled application.  
- Enables automated handling of tasks and projects in Nozbe Teams (e.g., creating, organizing, and managing team tasks and projects via MCP tools).  
- Usable across multiple chat or MCP clients without changing the server URL.  
- Configuration and setup details available through Pipedream’s configuration documentation.

## Integration & Usage
- Add the MCP server to your chat client or MCP application using the static URL: `https://mcp.pipedream.net/v2`.  
- Authenticate within your client when prompted to link Nozbe Teams.  
- Once connected, use the exposed MCP tools to work with Nozbe Teams tasks and projects from your client.
