# Winston AI MCP Server

An MCP server integration that exposes Winston AI’s AI-generated content detection engine to MCP-based applications via a static API endpoint.

## Overview
- **Name:** Winston AI MCP Server  
- **Category:** Security / Attestation MCP Servers  
- **Provider / Brand:** Winston AI (via Pipedream Connect)  
- **Primary Function:** AI-generated content detection accessible through the MCP protocol  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **AI-generated content detection engine**  
  - Integrates Winston AI’s detection capabilities into MCP-compatible clients.  
  - Designed to identify AI-generated text for compliance and content verification use cases.

- **Static MCP server endpoint**  
  - Uses a single, static URL (`https://mcp.pipedream.net/v2`) for all clients.  
  - Simplifies configuration across different MCP-based applications.

- **Client-agnostic integration**  
  - Can be added to various MCP-compatible chat or agent clients.  
  - Works with any client that supports configuring an MCP server URL.

- **Authentication on client setup**  
  - Authentication occurs when adding the server to an application (details handled during client configuration).

- **Configuration documentation**  
  - A dedicated configuration page (linked from the product page) for setup instructions and usage details.

- **Pipedream Connect infrastructure**  
  - Hosted and delivered via Pipedream Connect.  
  - Covered under Pipedream’s Terms and Privacy Policy.

## Use Cases
- Compliance checks for AI-generated content.  
- Content authenticity and provenance validation.  
- Integrating AI detection into chat clients, internal tools, or automated review workflows.
