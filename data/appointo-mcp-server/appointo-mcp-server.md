# Appointo MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** appointo  
**Source:** https://mcp.pipedream.com/app/appointo

## Overview
Appointo MCP Server connects your Appointo appointment scheduling account to MCP-compatible AI tools. It is provided through Pipedream’s MCP server, enabling agents and tools to book and manage appointments via Appointo.

## Features
- Connects an Appointo account to MCP-compatible environments
- Integration via Pipedream’s MCP server
- Enables AI tools to interact with Appointo’s scheduling capabilities
- Supports appointment booking and management workflows through agents/tools
- Usable with AI tools such as:
  - Cursor
  - Claude
  - Windsurf
