# V7 Go MCP Server

An MCP (Model Context Protocol) server that exposes V7 Go’s auto-annotation and computer vision tooling for integration into MCP-based workflows.

---

## Overview
- **Type:** MCP server / AI integration
- **Purpose:** Connect V7 Go’s computer vision and auto-annotation capabilities to MCP-compatible chat clients and applications.
- **Provider:** Pipedream (powered by Pipedream Connect)
- **Static MCP Server URL:** `https://mcp.pipedream.net/v2`

---

## Features
- **MCP-compatible server**
  - Provides a static MCP endpoint (`https://mcp.pipedream.net/v2`) that can be used across different MCP clients.
  - Authenticates when adding the server to your client or application.

- **V7 Go integration**
  - Exposes V7 Go’s auto-annotation and computer vision tooling through MCP.
  - Designed for building and enhancing AI workflows that rely on image understanding and annotation.

- **Multi-client support**
  - Usable with any MCP-capable chat client or application.
  - Central configuration via a single URL; no per-client endpoint changes needed.

- **Configuration resources**
  - Dedicated configuration documentation available via the platform’s Configuration page.
