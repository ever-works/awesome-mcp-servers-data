# Feedbin MCP Server

Modern RSS feed reader MCP server for integrating Feedbin with MCP-compatible chat clients and workflows.

- **Name:** Feedbin MCP Server  
- **Category:** Web Search MCP Servers  
- **Tags:** rss, web, content-discovery  
- **Source URL:** https://mcp.pipedream.com/app/feedbin  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Description
The Feedbin MCP Server exposes Feedbinâ€™s RSS reading capabilities to MCP-compatible applications, allowing workflows and chat clients to access and manage RSS feeds, subscriptions, and items via the Feedbin service.

## Features
- **MCP-compatible server endpoint**  
  - Single static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Works across all supported MCP clients.

- **Feedbin integration**  
  - Connects MCP workflows to the Feedbin RSS service.  
  - Enables access to RSS feeds, subscriptions, and items managed in Feedbin (per item description).

- **Client-agnostic setup**  
  - Same server URL for every client; authentication occurs when adding the server inside the client.  
  - Documentation and configuration guidance provided via the Pipedream configuration page.

- **Pipedream Connect platform**  
  - Delivered and hosted via Pipedream Connect infrastructure.

## Usage
1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.  
2. Add it as an MCP server in your compatible chat client or application.  
3. Authenticate within the client when prompted to link your Feedbin account.
