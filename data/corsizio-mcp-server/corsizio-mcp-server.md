# Corsizio MCP Server

**Category:** Business & Commerce MCP Servers  
**Slug:** corsizio-mcp-server  
**Brand:** Corsizio

## Description
Corsizio MCP Server is an MCP integration that connects Corsizio with compatible MCP clients, enabling management of online registrations, payments, and feedback for live teaching events such as in-person or online classes, courses, workshops, camps, and lessons.

## Features
- MCP server integration for Corsizio
- Supports management of:
  - Online registrations for live teaching events
  - Payment processing for events
  - Collection and handling of participant feedback
- Works with any compatible MCP client via a single static server URL
- Suitable for various live teaching formats:
  - In-person classes and workshops
  - Online classes and courses
  - Camps, lessons, and similar scheduled events

## Technical Details
- **MCP server URL:** `https://mcp.pipedream.net/v2`
- Single static URL for all clients (authentication handled when adding the server to the application)

## Getting Started
- Add the MCP server URL to your MCP-compatible chat or automation client.
- Follow the client-specific instructions for configuring an MCP server (as documented by your client or on the referenced configuration page).
