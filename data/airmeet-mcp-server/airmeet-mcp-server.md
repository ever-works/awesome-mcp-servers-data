# Airmeet MCP Server

MCP Server for Airmeet, enabling programmatic access to virtual and hybrid event experiences via the Model Context Protocol.

## Overview
The **Airmeet MCP Server** exposes Airmeet’s virtual and hybrid event capabilities through the **Model Context Protocol (MCP)**, allowing compatible AI/chat clients and applications to programmatically interact with Airmeet events.

- **Category:** Business & Commerce – MCP Servers  
- **Use cases:** events, webinars, virtual events, hybrid events  
- **MCP endpoint:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server for Airmeet**  
  - Provides a standardized MCP endpoint (`https://mcp.pipedream.net/v2`) that can be used across supported clients.
- **Programmatic access to Airmeet**  
  - Intended to enable automation and integration of Airmeet virtual and hybrid event experiences into MCP-aware tools and chat clients.
- **Static server URL**  
  - Single static URL that works for all MCP clients; authentication is handled when the server is added to an application.
- **Client-agnostic integration**  
  - Designed to be added to different chat or MCP-compatible clients, with client-specific setup instructions available through the configuration flow.
- **Configuration guidance**  
  - A dedicated configuration page (outside the provided excerpt) explains how to add and configure the server in various applications.
