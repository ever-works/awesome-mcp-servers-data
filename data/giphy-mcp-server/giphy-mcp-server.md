# Giphy MCP Server

GIFs & animated stickers from Giphy, accessible via the Model Context Protocol (MCP) for programmatic search and retrieval.

- **Category:** Media Processing MCP Servers  
- **Brand:** Giphy  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features

- Connects MCP-compatible clients to the Giphy platform.
- Supports programmatic search of GIFs and animated stickers.
- Returns GIF and animated sticker content suitable for embedding or further processing.
- Uses a **static MCP server URL** (`https://mcp.pipedream.net/v2`) that works across all supported MCP clients.
- Authentication is performed when adding/configuring the server in your MCP-compatible application.
- Provided and hosted via Pipedream Connect (no self-hosting details required on this page).
