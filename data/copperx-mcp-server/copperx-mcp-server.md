# Copperx MCP Server

**Category:** Blockchain & Crypto MCP Servers  
**Slug:** `copperx-mcp-server`  
**Brand:** Copperx

## Overview
Copperx MCP Server exposes Copperx’s Web3 payment APIs to MCP-compatible agents via a single MCP endpoint. It enables applications and AI agents to interact with Copperx’s crypto and fiat payment capabilities through the MCP protocol.

## Features
- **MCP-compatible payment server**
  - Accessible via a static MCP server URL: `https://mcp.pipedream.net/v2`
  - Same URL works across all compatible MCP clients
  - Authentication performed when adding the server to your application

- **Web3 payment APIs integration**
  - Designed for Web3 businesses and applications
  - Surfaces Copperx’s payment APIs to agents and tools through MCP

- **Crypto and fiat payments**
  - Supports accepting cryptocurrency payments
  - Supports accepting traditional fiat currency payments
  - Unified API for both payment types

- **Billing and commerce workflows**
  - **Recurring billing**: automate scheduled or subscription-style charges
  - **Checkout**: create payment checkout flows for one-time purchases
  - **Invoicing**: generate and manage invoices via the payment API

- **Client-agnostic configuration**
  - Static server URL usable by any MCP-enabled chat client
  - Can be added to a variety of MCP-compatible applications

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Integration model:** Add as an MCP server in your chat client or agent framework, then configure authentication for Copperx APIs.
