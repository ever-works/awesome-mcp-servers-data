# Gitea MCP Server

**Brand:** pipedream  
**Category:** repository-code-analysis-mcp-servers  
**Tags:** git, repository-management, devops

## Description
Gitea MCP Server is an MCP (Model Context Protocol) server that lets MCP-compatible clients interact with a Gitea Git hosting instance through Pipedream-powered workflows. It acts as a bridge between chat or MCP clients and Gitea’s code hosting and repository management features.

## Features
- **MCP-compatible server endpoint**  
  - Exposes a static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Designed to work with multiple MCP-capable clients.

- **Integration with Gitea**  
  - Connects to a Gitea instance, a lightweight, community-managed code hosting platform written in Go.  
  - Enables interaction with Gitea’s Git hosting and repository management features via MCP.

- **Client configuration flow**  
  - "Connect your account" step to authorize and link Gitea/Pipedream.  
  - Client selection to get tailored setup instructions for different chat or MCP clients.  
  - Authentication handled when adding the MCP server to applications.

- **Pipedream workflow backing**  
  - The MCP server is powered by Pipedream workflows, enabling automation and integration with Gitea operations.

- **Configuration resources**  
  - Dedicated configuration guidance available via a full configuration page (generic MCP configuration docs on Pipedream).
