# World Bank Data API MCP Server

## Overview
A Model Context Protocol (MCP) server for interacting with the open World Bank data API. It enables AI assistants to fetch, list, and analyze global development indicators for countries available in the World Bank’s open data catalog.

## Features
- **MCP-compliant server** for the World Bank open data API.
- **Access to World Bank indicators**: list available development indicators exposed by the API.
- **Country-level analysis**: analyze selected indicators for countries supported by the World Bank dataset.
- **Designed for AI assistants**: intended to be used by LLM-based tools and assistants to programmatically explore World Bank data.
- **Integration with ecosystem tools**:
  - Published as an MCP server compatible with clients that support the Model Context Protocol.
  - Smithery configuration (`smithery.yaml`) for easy deployment/usage via Smithery.

## Usage
- **Claude Desktop**: can be added as an MCP server via `claude_desktop_config.json` (exact configuration snippet is provided in the project’s README on GitHub).
