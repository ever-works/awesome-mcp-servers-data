# Flodesk MCP Server

**Category:** Business & Commerce · MCP Servers  
**Website:** https://mcp.pipedream.com/app/flodesk

Flodesk MCP Server exposes Flodesk’s email marketing platform via the Model Context Protocol (MCP), enabling programmatic management of subscribers, emails, and campaigns from compatible MCP clients.

## Features

- **MCP Access to Flodesk**  
  - Connects Flodesk’s email marketing platform to MCP-compatible applications.  
  - Designed for managing subscribers, emails, and campaigns programmatically.

- **Static MCP Server Endpoint**  
  - Single static server URL for all MCP clients:  
    - `https://mcp.pipedream.net/v2`  
  - Authentication is handled when adding the server to your MCP-compatible application.

- **Client-Agnostic Integration**  
  - Same endpoint works across different MCP chat clients and tools.  
  - Can be added to various applications that support custom MCP servers.

- **Configuration Documentation**  
  - Detailed configuration and setup instructions available via a dedicated configuration page (linked from the app page).
