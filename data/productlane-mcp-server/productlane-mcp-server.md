# Productlane MCP Server

**Category:** Business & Commerce • MCP Servers  
**Brand:** Productlane  
**Integration focus:** Linear  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Overview
Productlane MCP Server is an MCP-compatible backend for building workflows around product discovery and feedback management that integrates with Linear. It runs as a static MCP endpoint you can connect to from any compatible chat client or MCP-enabled application.

## Features
- **Static MCP endpoint**
  - Single, static server URL: `https://mcp.pipedream.net/v2`
  - Works with any compatible MCP client.
- **Client-agnostic setup**
  - Can be added to different chat clients / MCP applications.
  - Authentication handled when you add the server to your application.
- **Product discovery workflows**
  - Designed for product discovery and feedback-centric use cases.
  - Integrates these workflows with Linear for product management.
- **Linear integration**
  - Built specifically to work with Linear as the product management system.
  - Intended for teams managing product feedback and discovery in Linear.
- **Pipedream Connect infrastructure**
  - Hosted and powered by Pipedream Connect.
  - Uses Pipedream’s configuration and management pages for setup.

## Getting Started
- Use the MCP server URL: `https://mcp.pipedream.net/v2` when configuring your MCP client.
- Add the server in your chosen chat client or MCP-enabled app and complete authentication during setup.
- Refer to the provider’s configuration page (via Pipedream Connect) for client-specific connection steps.
