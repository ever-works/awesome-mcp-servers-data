# Botbaba MCP Server

A flow-based chatbot platform, exposed as an MCP server, that lets you create and manage chatbots in minutes.

- **Category:** Messaging MCP Servers  
- **Provider / Brand:** Pipedream  
- **Source URL:** https://mcp.pipedream.com/app/botbaba  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features

- **Flow-based chatbot creation**  
  - Design and manage chatbots using a flow-based model.  
  - Optimized for building conversational bots quickly (first bot in under ~10 minutes).

- **MCP server integration**  
  - Exposed as a Model Context Protocol (MCP) server for use with compatible chat clients and tools.  
  - Single static MCP endpoint (`https://mcp.pipedream.net/v2`) used across clients.

- **Client-agnostic server URL**  
  - Same MCP server URL works for every supported client.  
  - Authentication happens when adding the server to your application.

- **Chat client integration**  
  - Can be added to different chat clients that support MCP.  
  - Setup instructions are available per chat client via the configuration docs.

- **Central configuration docs**  
  - Full configuration details available on a dedicated configuration page (e.g., how to add the server, auth steps, and runtime details).

## Usage

1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.  
2. Add it to your MCP-compatible chat client.  
3. Authenticate when prompted.  
4. Use the provided tools to create and manage flow-based chatbots.
