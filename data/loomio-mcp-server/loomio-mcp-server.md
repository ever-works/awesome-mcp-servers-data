# Loomio MCP Server

**Category:** Project Management MCP Servers  
**Brand:** Loomio  
**Slug:** `loomio-mcp-server`

## Description
Loomio MCP Server is an integration that connects your Loomio account to AI tools via Pipedream’s MCP server. It enables collaborative decision-making, discussions, and proposal workflows to be coordinated and automated by MCP-compatible agents and editors.

## Features
- Connects Loomio to AI tools through Pipedream’s MCP server
- Supports integration with AI environments such as:
  - Cursor
  - Claude
  - Windsurf
- Enables orchestration of Loomio collaborative decision-making workflows via MCP agents
- Facilitates AI-assisted discussion and proposal management using your Loomio account
