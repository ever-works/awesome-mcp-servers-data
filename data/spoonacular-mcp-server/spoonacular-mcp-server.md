## Spoonacular MCP Server

**Category:** Data Access & Integration – MCP Servers  
**Brand:** spoonacular  
**Source:** https://mcp.pipedream.com/app/spoonacular

### Description
Spoonacular MCP Server exposes Spoonacular’s recipe, meal planning, and nutrition APIs through the Model Context Protocol (MCP), enabling food and nutrition workflows directly inside compatible MCP clients. It connects to Spoonacular’s meal planner and recipe database so you can work with recipes, meal plans, and nutritional information programmatically.

### Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Usable across different MCP clients with the same endpoint.

- **Spoonacular API integration**  
  - Access to Spoonacular’s recipe, meal planning, and nutrition APIs via MCP.  
  - Designed for programmatic food and nutrition workflows inside chat or agent-style clients.

- **Account-based configuration**  
  - Connect a Spoonacular-related account through Pipedream’s interface.  
  - Authentication is handled when adding the server to your MCP-compatible application.

- **Client-agnostic usage**  
  - Works with multiple MCP-enabled chat clients.  
  - Per-client setup instructions available via the configuration flow.

- **Tool discovery in clients**  
  - Once connected, available tools (actions) are exposed to the client through MCP (e.g., recipe, meal, or nutrition-related actions when supported by the underlying Spoonacular APIs).

- **Web-based configuration page**  
  - Central configuration page on Pipedream to manage and review the Spoonacular MCP server setup.
