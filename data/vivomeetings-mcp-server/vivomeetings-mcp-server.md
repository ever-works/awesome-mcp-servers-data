# Vivomeetings MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Vivomeetings  
**Slug:** `vivomeetings-mcp-server`

## Overview
The Vivomeetings MCP Server allows MCP-compatible clients to schedule and manage online meetings and webinars through Vivomeetings, including access to recordings and real-time transcription features.

## MCP Server URL
- Base MCP Server endpoint (static for all clients):
  - `https://mcp.pipedream.net/v2`
- Authentication is handled when adding the server to your MCP-compatible application.

## Features
- Schedule and manage online meetings via Vivomeetings
- Manage and access meeting recordings
- Real-time transcription for meetings
- Webinar mode for hosting webinars
- Integration via a static MCP server URL usable across different MCP clients

## Integration & Configuration
- Works with any MCP client that supports adding custom MCP servers
- Server is added by configuring the static URL in your chat or MCP-compatible application
- Additional configuration details are available on the provider’s configuration page (via Pipedream Connect)
