# Myphoner MCP Server

**Category:** Business & Commerce – MCP Servers  
**Website:** https://mcp.pipedream.com/app/myphoner

## Description
Myphoner MCP Server connects the Myphoner cold outreach and calling platform with MCP-compatible chat clients, enabling agents to work with leads and outbound calling workflows directly from their applications.

## Features
- **MCP integration:** Exposes Myphoner’s cold outreach and calling capabilities as tools within MCP-compatible chat or AI clients.
- **Lead and call workflow support:** Designed to assist with lead management and outbound call workflows for sales teams.
- **Remote sales use case:** Tailored for remote sales teams performing cold outreach and cold calling.
- **Static server endpoint:** Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) that works for all clients.
- **Client-agnostic setup:** Can be added to different chat clients that support MCP, with configuration guidance available via a dedicated configuration page on Pipedream.
- **Account-based configuration:** Requires connecting a Myphoner account and selecting the relevant client to start using the tools.

## Integration & Configuration
- Connect your Myphoner account in the Pipedream Myphoner MCP Server app.
- Use the static MCP server URL: `https://mcp.pipedream.net/v2` when adding the server to your MCP-compatible app.
- Follow client-specific steps in your chat client or the Pipedream configuration page to finish setup.
