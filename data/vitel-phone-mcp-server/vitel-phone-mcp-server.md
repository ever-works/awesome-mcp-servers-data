# Vitel Phone MCP Server

**Category:** Messaging MCP Servers  
**Provider:** Vitel Global

## Overview
Vitel Phone MCP Server integrates Vitel Global’s cloud-based business phone system into MCP-compatible environments, allowing applications and chat clients to interact with Vitel’s business communication features through a standard MCP server endpoint.

## Features
- Integration of Vitel Global’s cloud-based business phone system into MCP-enabled apps
- Central MCP server endpoint usable across multiple clients
- Authentication-based access when adding the server to an application
- Compatibility with multiple chat clients (configured per client after connection)
- Hosted and managed by Pipedream as an MCP server instance

## Configuration
- **MCP server URL (static):** `https://mcp.pipedream.net/v2`  
  This URL is shared across clients; authentication is performed when the server is added to an application.
- Connect a Vitel Global account, then select the desired client within the Pipedream interface.
- Additional setup details are available via the Pipedream configuration flow (per supported chat client).
