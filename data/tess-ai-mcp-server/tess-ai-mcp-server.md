# Tess AI MCP Server

**Category:** AI Integration – MCP Servers  
**Brand:** Pareto

## Overview
Tess AI MCP Server is an MCP (Model Context Protocol) server that connects your Tess AI by Pareto account to compatible AI tools. It exposes Tess AI’s generative AI capabilities through the MCP interface so they can be used directly within supported development and AI environments.

## Features
- Connects Tess AI by Pareto to AI tools via Pipedream’s MCP server
- Exposes Tess AI’s generative AI capabilities through the MCP interface
- Enables integration with tools such as:
  - Cursor
  - Claude
  - Windsurf
- Designed for company and enterprise use cases

## Integrations
- Tess AI by Pareto
- Cursor
- Claude
- Windsurf
- Pipedream MCP
