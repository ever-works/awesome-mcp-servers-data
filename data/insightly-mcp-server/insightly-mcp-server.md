# Insightly MCP Server

## Overview
Insightly MCP Server connects Insightly’s CRM and marketing automation platform to MCP-compatible tools and agents via a static MCP endpoint, enabling access to CRM data and automation workflows from MCP-enabled applications.

- **Category:** Business & Commerce – MCP Servers
- **Use cases:** CRM data access, marketing automation, sales operations, workflow automation via MCP
- **MCP server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP Integration Endpoint**  
  - Single static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same URL usable across different clients; authentication handled per client/app.

- **Insightly CRM & Marketing Automation Access**  
  - Designed to expose Insightly CRM and marketing automation capabilities to MCP-compatible tools and agents.  
  - Enables use of Insightly data and workflows within chat clients and other MCP-integrated applications.

- **Client-Agnostic Setup**  
  - Works with multiple chat clients and MCP-capable applications.  
  - Setup instructions are available per client type via the configuration documentation (not reproduced here).

- **Pipedream Connect Backing**  
  - MCP server is powered by Pipedream Connect’s infrastructure for integration and workflow execution.
