## Description
Quaderno MCP Server is a Pipedream-hosted Model Context Protocol (MCP) server that exposes Quaderno’s tax compliance and sales tax automation capabilities to MCP-compatible tools and AI agents. It allows applications and agents to integrate Quaderno’s sales tax handling and compliance logic directly via a standardized MCP endpoint.

## Features
- **Pipedream-hosted MCP server**
  - Runs as a managed MCP endpoint provided by Pipedream.
  - Accessible through a single static URL: `https://mcp.pipedream.net/v2`.
- **Model Context Protocol integration**
  - Conforms to the MCP standard so it can be used with MCP-compatible chat clients, tools, and AI agents.
  - Designed to be added as a server in various chat or agent applications.
- **Quaderno tax compliance exposure**
  - Surfaces Quaderno’s tax compliance functionality to MCP clients.
  - Focused on sales tax automation and compliance workflows for global transactions.
- **Authentication at client setup**
  - The same URL is used for all clients; authentication occurs when adding the server to each application.
- **Client-agnostic configuration**
  - Works with multiple MCP-compatible chat clients.
  - Additional setup details are provided via a separate configuration page (not included in the provided content).

## Technical Details
- **MCP server URL:** `https://mcp.pipedream.net/v2`
- **Hosting:** Managed by Pipedream
- **Protocol:** Model Context Protocol (MCP)
