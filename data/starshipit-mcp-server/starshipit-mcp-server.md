# Starshipit MCP Server

## Overview
Starshipit MCP Server is an MCP-compatible integration for Starshipit, a web-based shipping fulfilment platform for eCommerce retailers. It allows MCP clients (such as AI chat tools) to interact with Starshipit to automate shipping fulfilment, order processing, and label creation across supported eCommerce channels and marketplaces.

- **Category:** Business & Commerce – MCP Servers
- **Use cases:** Shipping fulfilment automation, multi-channel order processing, shipping label generation
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server endpoint**
  - Single static MCP server URL (`https://mcp.pipedream.net/v2`) that works across compatible clients.
  - Supports authentication at the time you add the server to your MCP client / application.

- **Shipping fulfilment automation**
  - Connects to Starshipit to automate key shipping workflows.
  - Designed for eCommerce retailers handling orders from multiple channels.

- **Order processing and fulfilment**
  - Enables MCP clients to trigger and manage order processing steps through Starshipit.
  - Helps move orders from placed → processed → fulfilled using automated flows.

- **Shipping label creation**
  - Supports label creation via Starshipit through MCP-compatible clients.
  - Intended to streamline label generation for orders from connected marketplaces and shopping carts.

- **Multi-channel eCommerce support**
  - Built to work with orders originating from popular marketplaces and shopping carts (via Starshipit’s existing integrations).

- **Client-agnostic configuration**
  - The same server URL can be used across different MCP chat clients.
  - Configuration guidance available via a dedicated configuration page (client-specific steps for adding the server).

## Setup
- Use the MCP server URL: `https://mcp.pipedream.net/v2` when adding Starshipit MCP Server to your MCP-compatible client.
- Authenticate within your client as prompted during server setup.
- Refer to the provider’s configuration documentation for client-specific instructions.
