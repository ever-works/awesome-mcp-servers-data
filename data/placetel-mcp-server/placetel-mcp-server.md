# Placetel MCP Server

An MCP Server integration for Placetel’s cloud telephony platform, enabling programmatic management of modern business telephony through MCP-compatible clients.

- **Website / Source**: https://mcp.pipedream.com/app/placetel
- **Category**: Messaging MCP Servers
- **Brand**: Placetel
- **Tags**: telephony, VoIP, business-communications

## Features

- MCP Server integration for Placetel’s cloud telephony platform.
- Enables programmatic management of modern business telephony via MCP-compatible applications.
- Exposed via a single static MCP server URL usable by any client:
  - `https://mcp.pipedream.net/v2`
- Authentication is handled when adding the server to your MCP-enabled application or chat client.
- Hosted and delivered via Pipedream Connect.

## Usage

- Add the MCP server URL (`https://mcp.pipedream.net/v2`) to your MCP-compatible chat client or application.
- Authenticate within your client during setup.
- Refer to the provider’s Configuration page for client-specific setup instructions.
