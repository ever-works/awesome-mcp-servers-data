# Polly MCP Server

MCP Server for Polly that allows capturing instant feedback and running polls in Slack and Microsoft Teams via MCP-enabled tools and agents.

## Features

- Capture instant feedback directly in Slack.
- Run polls in Slack using Polly via MCP-enabled tools and agents.
- Capture instant feedback in Microsoft Teams.
- Run polls in Microsoft Teams using Polly via MCP-enabled tools and agents.
- Single static MCP server URL for all compatible clients:
  - Server URL: `https://mcp.pipedream.net/v2`
- Works with any MCP-enabled client or agent that supports adding custom MCP servers.
- Authentication handled when adding the server to your application.

## Integration & Configuration

- Add the Polly MCP Server to your preferred chat client (e.g., Slack, Microsoft Teams) as an MCP server endpoint.
- Configuration details and setup instructions are available on the Pipedream MCP **Configuration** page (external link from the source site).
