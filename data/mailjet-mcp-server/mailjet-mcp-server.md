## Overview
Mailjet MCP Server is an MCP (Model Context Protocol) server that connects MCP-compatible tools and clients to Mailjet’s transactional and marketing email platform. It enables applications to send and manage email campaigns through a unified MCP interface.

## Features
- **MCP Server for Mailjet**
  - Exposes Mailjet’s email capabilities via the MCP protocol.
  - Designed to be added as a server in MCP-compatible chat or developer tools.

- **Transactional & Marketing Email Integration**
  - Connects to Mailjet’s transactional email features (e.g., programmatic email sending).
  - Connects to Mailjet’s marketing email features (e.g., campaigns and newsletters).

- **Static MCP Endpoint**
  - Uses a static MCP server URL for all clients: `https://mcp.pipedream.net/v2`.
  - The same endpoint is used regardless of client; authentication is handled when adding the server to the application.

- **Client-Agnostic Setup**
  - Can be added to different MCP-compatible chat clients or applications.
  - Configuration instructions are available via the referenced configuration page (client-specific setup not detailed in the provided content).

- **Pipedream Connect Integration**
  - The server is powered and hosted through Pipedream Connect infrastructure.

## Usage
- Add the MCP server to an MCP-compatible client using the static URL: `https://mcp.pipedream.net/v2`.
- Authenticate within the client during setup (specific auth steps depend on the client and are not detailed in the provided content).
