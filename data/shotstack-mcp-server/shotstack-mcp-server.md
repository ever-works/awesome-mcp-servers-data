## Shotstack MCP Server

**Category:** Media Processing MCP Servers  
**Brand:** Shotstack  
**URL:** https://mcp.pipedream.com/app/shotstack

### Description
Shotstack MCP Server is an MCP integration that connects Shotstack’s video automation API with AI tools. It enables MCP tools to programmatically create, edit, and distribute large volumes of data-driven videos via Shotstack.

### Features
- Connects a Shotstack account to AI tools such as Cursor, Claude, and Windsurf via Pipedream’s MCP server
- Programmatic video creation using Shotstack’s video automation API
- Programmatic video editing for data-driven and templated content
- Automated video distribution workflows triggered through MCP tools
- Scales to handle large volumes of video generation and processing
- API-based integration suitable for embedding in broader automation pipelines

### Integrations
- Cursor (via MCP)
- Claude (via MCP)
- Windsurf (via MCP)
- Shotstack Video Automation API
- Pipedream MCP Server
