# Intellexer API MCP Server

**Category:** Content Extraction & Summarization MCP Servers  
**Brand:** Intellexer  
**Slug:** intellexer-api-mcp-server

## Overview
The Intellexer API MCP Server exposes Intellexer’s semantic analysis products to MCP-based applications. It allows developers to embed Intellexer’s AI-powered text and document analysis capabilities into consumer or enterprise applications and web services via XML or JSON.

MCP server URL:
```text
https://mcp.pipedream.net/v2
```

## Features
- **MCP Server Integration**
  - Static MCP server URL usable by any MCP-compatible client.
  - Authentication handled when adding the server to the client/application.
  - Works with multiple chat or MCP-based clients.

- **Text Summarization**
  - Summarize arbitrary text using Intellexer’s semantic analysis.
  - Accessible as an MCP tool/action: **Summarize Text**.

- **Document Summarization**
  - Summarize full documents via the Intellexer API.
  - Accessible as an MCP tool/action: **Summarize Document**.

- **Named Entity Extraction**
  - Extract named entities (e.g., people, organizations, locations) from documents.
  - Accessible as an MCP tool/action: **Extract Named Entities**.

- **Language Detection**
  - Detect the language and encoding of an input text stream.
  - Accessible as an MCP tool/action: **Detect Language**.

- **Data Formats & Access**
  - Supports integrating Intellexer’s semantic products using **XML** or **JSON** payloads.

## Available Tools (Actions)
1. Summarize Text  
2. Summarize Document  
3. Extract Named Entities  
4. Detect Language
