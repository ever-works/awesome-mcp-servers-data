# Acronis Cyber Protect Cloud MCP Server

## Description
Acronis Cyber Protect Cloud MCP Server connects MCP-based workflows to Acronis Cyber Protect Cloud, enabling unified management of backup, AI-based anti-malware, antivirus, and endpoint protection from within compatible MCP clients.

## Features
- **Backup integration**: Access Acronis Cyber Protect Cloud backup capabilities through MCP-based workflows.
- **AI-based anti-malware**: Leverage next-generation, AI-driven malware detection and protection.
- **Antivirus protection**: Integrate antivirus scanning and protection features into MCP-connected applications.
- **Endpoint protection management**: Manage endpoint protection policies and operations from MCP workflows.
- **Unified security & backup**: Combine backup, anti-malware, antivirus, and endpoint protection management in a single integrated solution via the MCP server.
- **Static MCP server URL**: Use a single, static endpoint for all clients:
  - `https://mcp.pipedream.net/v2`
- **Client-agnostic access**: Works with any supported MCP-compatible chat or application client; authentication occurs when adding the server to the client.

## Technical Details
- **MCP server URL**: `https://mcp.pipedream.net/v2`
- **Integration layer**: Provided via Pipedream Connect as the MCP hosting/integration platform.
