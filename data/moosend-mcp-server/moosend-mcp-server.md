# Moosend MCP Server

Moosend MCP Server connects Moosend’s email marketing and automation platform to the Model Context Protocol (MCP), enabling programmatic access to campaigns and contacts from MCP-compatible clients.

---

## Overview
- **Product type:** MCP Server integration
- **Platform:** Moosend (email marketing & automations)
- **Provider:** Pipedream Connect
- **Category:** Business & Commerce MCP Servers
- **Use cases:** Programmatic management of email campaigns, contacts, and automations from MCP-enabled chat or agent applications.

---

## Server URL
- **MCP server base URL:**
  - `https://mcp.pipedream.net/v2`
- **Notes:**
  - Static URL used by all clients
  - Authentication occurs when adding the server to your application

---

## Features
- Connects Moosend’s email marketing and automation platform to MCP-compatible clients.
- Enables programmatic campaign and contact management via MCP.
- Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) for all clients.
- Authentication handled at the time you add/configure the server in your app.
- Provided and operated through Pipedream Connect infrastructure.
- Configuration guidance available via a dedicated configuration page (client-specific setup instructions).
