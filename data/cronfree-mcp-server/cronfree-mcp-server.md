# Cronfree MCP Server

**Category:** Workflow Automation MCP Servers  
**Brand:** Cronfree

Cronfree MCP Server connects MCP-compatible clients to the Cronfree Time Scheduler, enabling programmatic creation and management of recurring tasks and no‑code automation schedules.

---

## Description
Cronfree Time Scheduler allows users to define and control multiple recurring tasks using a no‑code automation approach. Through the MCP server interface, MCP clients can integrate scheduling capabilities directly into their applications or workflows, automating time-based actions programmatically.

---

## Features
- **MCP-compatible scheduling server** – Exposes Cronfree Time Scheduler as an MCP Server for use with any supported MCP client.
- **Recurring task management** – Create and manage multiple recurring tasks programmatically.
- **No-code automation schedules** – Configure and run time-based automations without writing traditional code.
- **Static MCP server URL** – Single, reusable endpoint for all MCP clients:
  - Base URL: `https://mcp.pipedream.net/v2`
- **Client-agnostic integration** – Works across different MCP-enabled chat or automation clients; you add the same server URL to each.
- **Authentication at connection time** – Authentication is performed when adding the server to an application (credentials / method handled during setup in the client).

---

## Integration & Usage
- Add the MCP server to any supported MCP client using:
  - **Server URL:** `https://mcp.pipedream.net/v2`
- Follow your client’s configuration flow to complete authentication and begin defining recurring tasks and schedules.
