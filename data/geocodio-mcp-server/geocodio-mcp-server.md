## Description
The Geocodio MCP Server is an MCP-compatible service that exposes Geocodio’s geocoding and reverse geocoding API through a single, static MCP server endpoint. It is hosted via Pipedream and intended for use with MCP-capable chat or AI clients.

## Features
- **MCP-compatible server** – Provides an MCP Server interface for Geocodio’s geocoding and reverse geocoding capabilities.
- **Static MCP endpoint** – Uses a single static URL for all MCP clients:
  - `https://mcp.pipedream.net/v2`
- **Client-agnostic integration** – Designed to work with any MCP-enabled chat/AI client; configuration instructions are provided per client.
- **Account-based authentication** – You authenticate when adding the server to your application after connecting your Geocodio account.
- **Central configuration flow** – "Configure Geocodio" page to connect your account and select a client to complete setup.
- **Hosted by Pipedream** – Runs on Pipedream’s infrastructure under the “Infrastructure & Cloud” category.
