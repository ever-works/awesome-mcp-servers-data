# Free Dictionary MCP Server

**Website:** https://mcp.pipedream.com/app/dictionary_api  
**Category:** Documentation & Learning Resources  
**Tags:** dictionary, reference, language

## Overview
The Free Dictionary MCP Server is an MCP (Model Context Protocol) server that exposes the Free Dictionary API, allowing dictionary and definition lookups to be integrated into MCP-compatible chat clients and workflows.

## Features
- **MCP-compatible dictionary server**: Implements an MCP server interface for use with MCP-based clients and tools.
- **Free Dictionary API integration**: Provides access to the Free Dictionary API for word lookups and language reference tasks.
- **Definition lookups**: Supports retrieving word definitions within MCP workflows and chat environments.
- **Language and reference utility**: Designed for dictionary, reference, and language-related use cases.
- **Static server endpoint**: Uses a single static MCP server URL for all clients:
  - `https://mcp.pipedream.net/v2`
- **Client-agnostic configuration**: The same server URL works across different MCP-compatible chat clients; authentication is handled when adding the server to each application.
- **Configuration docs available**: A full configuration page is provided (via the platform) to guide adding and using the server in applications.

## Usage
- Add `https://mcp.pipedream.net/v2` as an MCP server in your compatible chat client or application.
- Authenticate within your client when prompted to start performing dictionary and definition lookups.
