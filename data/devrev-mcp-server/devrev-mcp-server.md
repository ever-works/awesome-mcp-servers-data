# DevRev MCP Server

## Overview
DevRev MCP Server is an MCP-compatible integration that connects DevRev’s developer–customer relationship management and product work management capabilities to MCP-based applications via Pipedream. It enables agents and chat clients to create and manage DevRev work items programmatically through standardized MCP tools.

**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible DevRev integration**  
  - Exposes DevRev functionality as tools to MCP-enabled clients and agents.  
  - Uses a static MCP server URL for all clients, with authentication handled when adding the server to your application.

- **Account connection & configuration**  
  - Connect a DevRev account through Pipedream.  
  - Select a client/chat application and follow configuration steps (via the configuration page) to add the server.

- **Available tools (actions)**  
  - **Create Ticket**  
    - Create new tickets in DevRev.  
    - Suitable for tracking customer or internal requests that require follow-up.  
  - **Create Task**  
    - Create new tasks associated with work items or initiatives in DevRev.  
  - **Create or Update Custom Schema Fragment**  
    - Create new or update existing custom schema fragments in DevRev.  
    - Useful for extending DevRev’s data model with custom fields or structures.  
  - **Create Opportunity**  
    - Create new opportunities for managing sales or revenue-related work in DevRev.  
  - **Create Issue**  
    - Create new issues, for example bug reports or product defects, in DevRev.  
  - **Create Comment**  
    - Add comments to DevRev work items (e.g., tickets, tasks, issues, opportunities) for collaboration and status updates.

- **Use cases**  
  - Automate creation of DevRev items from MCP-compatible chat clients or agents.  
  - Integrate customer interactions directly into DevRev as tickets, issues, or opportunities.  
  - Maintain structured product and work data via custom schema fragments.
