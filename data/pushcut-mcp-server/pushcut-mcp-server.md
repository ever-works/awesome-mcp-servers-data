# Pushcut MCP Server

**Category:** Home Automation MCP Servers  
**Website:** https://mcp.pipedream.com/app/pushcut

## Description
Pushcut MCP Server connects your Pushcut smart home and notification workflows to MCP-compatible AI tools, enabling orchestration and automation across your devices and services.

## Features
- Connects a Pushcut account to MCP-compatible AI tools
- Works with tools like Cursor, Claude, and Windsurf
- Enables orchestration of smart home workflows via MCP
- Supports automation of notifications through Pushcut

## Integrations
- Cursor (via MCP)
- Claude (via MCP)
- Windsurf (via MCP)
- Pushcut (smart home and notifications)
