# WhatsApp Business MCP Server

**Category:** Messaging MCP Servers  
**Brand:** WhatsApp Business  
**Slug:** whatsapp-business-mcp-server

## Overview
The WhatsApp Business MCP Server is an MCP server integration that exposes WhatsApp Business APIs within MCP-based workflows. It enables businesses to engage audiences, support sales activities, and deliver customer service over WhatsAppâ€”directly from compatible MCP clients.

## Features
- **MCP server integration for WhatsApp Business**
  - Provides a static MCP server URL: `https://mcp.pipedream.net/v2`
  - Usable by any compatible MCP client.
- **Authentication at client setup**
  - Authentication is handled when adding the server to your application (per-client auth configuration).
- **WhatsApp Business API exposure**
  - Access WhatsApp Business products and APIs via MCP workflows.
  - Supports use cases such as:
    - Audience engagement and messaging campaigns.
    - Sales and commerce-related conversations.
    - Customer service communication over WhatsApp.
- **Multi-client configuration**
  - Can be added to different chat clients that support MCP.
  - Configuration guidance available via a dedicated configuration page (for setup specifics such as connecting the server and managing credentials).
- **Pipedream Connect integration**
  - Delivered as part of the Pipedream Connect ecosystem, enabling integration into broader automation and workflow setups.

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Integration type:** MCP server for WhatsApp Business APIs
- **Authentication:** Per-client, performed during server addition to the app
