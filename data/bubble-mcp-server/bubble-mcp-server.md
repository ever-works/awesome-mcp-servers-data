# Bubble MCP Server

**Category:** Development Tools – MCP Servers  
**Brand:** Bubble

An MCP (Model Context Protocol) server that exposes Bubble’s no-code application platform to MCP-compatible agents, enabling them to create, modify, or interact with Bubble applications programmatically.

---

## Features

- **MCP-compatible server**
  - Acts as an MCP endpoint that can be added to any supported MCP client or chat application.
- **Static server URL**
  - Single shared endpoint for all clients:
    - `https://mcp.pipedream.net/v2`
  - Authentication is handled when you add the server to your application.
- **No-code app integration**
  - Connects to Bubble’s no-code application platform, allowing agents to:
    - Create Bubble app resources programmatically.
    - Modify existing Bubble applications.
    - Interact with Bubble apps via structured MCP tools.
- **Client-agnostic setup**
  - Designed to work with multiple chat clients that support MCP.
  - Setup guidance is available per client via configuration docs.
- **Hosted by Pipedream Connect**
  - Managed/hosted MCP server infrastructure, so you don’t need to deploy or host the server yourself.

---

## Setup & Configuration

- Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server to your MCP-compatible client or chat app.
- Authenticate during the server-adding process in your application.
- Additional configuration instructions are available on the provider’s configuration page.
