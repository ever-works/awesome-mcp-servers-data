# Uberduck MCP Server

## Description
The Uberduck MCP Server is an MCP-compatible service that connects applications to Uberduck for generating AI-driven vocals, including singing and rapping, suited for creative agencies, musicians, and developers.

## Features
- **MCP-compatible server**: Exposes Uberduck functionality via the Model Context Protocol (MCP) for integration with MCP-enabled applications.
- **AI-generated vocals**: Supports synthetic vocals for music and audio projects.
- **Singing and rapping**: Tailored for generating both singing and rap performances.
- **Static server URL**: Uses a single MCP server URL (`https://mcp.pipedream.net/v2`) that works across supported clients.
- **Account-based authentication**: Authentication handled when adding the server to your MCP-compatible application.
- **Client-agnostic setup**: Can be added to multiple chat or MCP clients by configuring the same server URL.

## Integration & Configuration
- Connect an Uberduck account via the Pipedream-based configuration flow.
- Copy and use the static MCP server URL in supported clients.
- Configuration guidance is available per client type and via a central configuration page.
