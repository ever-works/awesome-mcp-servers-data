# Oracle Cloud Infrastructure MCP Server

**Brand:** Oracle  
**Category:** Cloud & DevOps MCP Servers  
**Website:** https://mcp.pipedream.com/app/oracle_cloud_infrastructure

## Description
Oracle Cloud Infrastructure MCP Server is an MCP-compatible server that connects AI agents and MCP-enabled clients to Oracle Cloud Infrastructure (OCI). It enables programmatic management and interaction with OCI resources and cloud services through the Model Context Protocol.

## Features
- MCP-compliant server endpoint for Oracle Cloud Infrastructure
- Programmatic access to OCI resources and cloud services via MCP
- Single static MCP server URL usable across compatible clients
- Authentication handled when adding the server within each client or application
- Hosted and operated via Pipedream Connect infrastructure
