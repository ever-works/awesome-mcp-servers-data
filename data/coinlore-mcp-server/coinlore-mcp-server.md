# CoinLore MCP Server

## Overview
The CoinLore MCP Server is an integration that exposes CoinLore’s cryptocurrency market data through the Model Context Protocol (MCP). It lets applications and chat clients access real-time and historical crypto data—such as prices, volumes, markets, and market caps—via a unified MCP endpoint.

- **Category:** Finance / Market Data MCP Servers  
- **Provider / Brand:** CoinLore (via Pipedream Connect)  
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- **Cryptocurrency price data**  
  - Access current cryptocurrency/coin prices.  
  - Prices are calculated using CoinLore’s own algorithm.

- **Market metrics**  
  - Coin market capitalization data.  
  - Market and exchange-related information.

- **Volume and liquidity metrics**  
  - Trading volume data for supported coins.

- **Historical data**  
  - Historical prices and charts for cryptocurrencies.

- **Blockchain information**  
  - Access blockchain-related info for supported coins (as exposed by CoinLore’s API).

- **Widgets and related tools**  
  - Underlying access to CoinLore’s widgets and additional metrics made available via their API.

- **MCP integration**  
  - Single static MCP server URL that works across compatible clients.  
  - Authentication handled when adding the server to your application.  
  - Can be used with multiple chat clients that support MCP.

## Integration & Usage
- **MCP server URL:** `https://mcp.pipedream.net/v2` (static for all clients).  
- **Client setup:** Add this MCP server URL in your MCP-compatible chat client or application and complete authentication as prompted.  
- **Configuration reference:** Additional setup and client-specific instructions are available on the Pipedream Configuration page.
