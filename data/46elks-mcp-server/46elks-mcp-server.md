# 46elks MCP Server

## Overview
The 46elks MCP Server exposes the 46elks telephony API (SMS, MMS, and phone calls) through the Model Context Protocol (MCP), allowing MCP‑compatible tools and chat clients to integrate telephony features with minimal code.

- **Type:** MCP server / telephony integration service
- **Brand:** 46elks
- **Category:** Messaging MCP Servers
- **Tags:** telephony, SMS, API integration
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **Telephony API Integration**
  - Access the 46elks API for sending SMS
  - Send MMS messages
  - Initiate and manage phone calls
- **MCP Compatibility**
  - Exposes 46elks functionality via an MCP server endpoint
  - Usable from MCP‑compatible tools and chat clients
- **Simple Setup**
  - Static MCP server URL (`https://mcp.pipedream.net/v2`) for all clients
  - Authentication handled when adding the server to your application
  - Configuration guidance available via a dedicated configuration page (per client setup instructions)
- **Low-Code Integration**
  - Designed to integrate telephony features with just a few lines of code

## Pricing
- Transparent pricing via 46elks (no startup fees, no minimum monthly costs).
- Specific per‑message or per‑call rates are not listed in the provided content.

## Usage Notes
- Add the MCP server URL to your MCP‑compatible chat client or application.
- Authenticate with your 46elks / Pipedream credentials during setup.
- Refer to the configuration page of your client for exact integration steps.
