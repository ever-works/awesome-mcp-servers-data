# SMS Partner MCP Server

**Category:** Messaging MCP Servers  
**Brand:** SMS Partner  
**Slug:** `sms-partner-mcp-server`

MCP Server for SMS Partner, exposing professional SMS platform capabilities via the Model Context Protocol for use in compatible chat or AI clients.

---

## Features

- **MCP-based integration**
  - Exposes SMS Partner functionality through the Model Context Protocol (MCP).
  - Usable from any MCP-compatible chat or AI client.

- **Static MCP server endpoint**
  - Single shared URL for all clients: `https://mcp.pipedream.net/v2`.
  - Configuration is client-agnostic; the same endpoint is used across tools.

- **Application-level authentication**
  - Authentication occurs when adding/configuring the MCP server in your application.
  - Supports per-application credentials rather than per-URL customization.

- **Professional SMS platform integration**
  - Designed to connect to SMS Partner’s professional SMS platform features (e.g., telecom / messaging workloads via SMS Partner’s infrastructure).

- **Client setup guidance (via Pipedream Connect)**
  - Documentation available on selecting a chat client and adding the MCP server.
  - Additional configuration details provided on a dedicated configuration page (external to this summary).

---

## Technical Details

- **MCP server URL:** `https://mcp.pipedream.net/v2`
- **Protocol:** Model Context Protocol (MCP)
- **Intended use:** Integrate SMS Partner SMS capabilities into MCP-compatible chat / AI applications.
