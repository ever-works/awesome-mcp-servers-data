# FlightRadar24 MCP Server

## Overview
FlightRadar24 MCP Server is a Model Context Protocol (MCP) server for Claude Desktop that enables real-time flight tracking using Flightradar24 data.

## Features
- Claude Desktop-compatible MCP server
- Uses live data from Flightradar24
- Real-time flight tracking capabilities
- Designed for use directly within Claude Desktop conversations

## Integration
- Integrates with Claude Desktop via the `mcpServers` configuration
- Supported configuration locations:
  - macOS: `~/Library/Application Support/Claude/claude_desktop_config.json`
  - Windows: `%APPDATA%/Claude/claude_desktop_config.json`

## Setup
- Clone the GitHub repository to your local machine
- Install project dependencies
- Build the project
- Add the server entry to the `mcpServers` section of your Claude Desktop configuration file
