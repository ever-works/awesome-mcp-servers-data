# Zoom Admin MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Zoom

## Overview
The Zoom Admin MCP Server is an MCP server designed for Zoom administrators. It provides account-level access to Zoom, enabling management of video conferencing resources, meetings, and account configurations via MCP-compatible workflows and clients.

## Features
- **MCP server for Zoom** designed specifically for admin-level use cases
- **Account-level scopes** to access and manage Zoom accounts and configurations (admin-focused rather than just user-level access)
- **Video conferencing management** capabilities oriented around Zoom meetings and related resources
- **Static MCP endpoint** usable across clients: `https://mcp.pipedream.net/v2`
- **Client-agnostic setup**: same server URL works with any supported MCP chat client; authentication happens during server addition in the client
- **Configuration documentation available** via a central configuration page (for step-by-step integration and setup details)
- **Powered by Pipedream Connect** infrastructure for hosting and serving the MCP endpoint

## Integration & Usage
- Use the static URL `https://mcp.pipedream.net/v2` to add the Zoom Admin MCP Server to compatible MCP clients.
- Authenticate within your chat or MCP client when adding this server to gain admin-level Zoom capabilities.
