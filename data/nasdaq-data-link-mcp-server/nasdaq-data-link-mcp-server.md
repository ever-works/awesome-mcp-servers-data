## Features
- **MCP-compatible server** for integrating Nasdaq Data Link data into MCP-enabled chat clients and tools.
- **Access to 250+ datasets** covering financial and market data.
- **Support for time series data** (e.g., historical prices, indices, and other temporal financial metrics).
- **Support for table data** for structured, tabular financial datasets.
- **API-based access** for programmatic querying and integration into applications and workflows.
- **Single static endpoint** (`https://mcp.pipedream.net/v2`) usable across all supported MCP clients.
- **Authentication at client setup time**, allowing secure use of the shared static MCP URL.
- **Cloud-based platform** hosted and managed via Pipedream Connect.

## Usage
- Add the MCP server URL `https://mcp.pipedream.net/v2` to an MCP-compatible client.
- Authenticate within your client when prompted to enable access to Nasdaq Data Link datasets.
