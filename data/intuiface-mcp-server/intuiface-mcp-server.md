# Intuiface MCP Server

**Description**  
An MCP (Model Context Protocol) server for Intuiface, enabling MCP-compatible agents and chat clients to interact with Intuiface’s no-code platform for building rich, interactive digital experiences and digital signage.

**Category**  
Content Management MCP Servers

**Tags**  
- interactive  
- no-code  
- digital-signage

**Key Details**
- **MCP Endpoint URL:** `https://mcp.pipedream.net/v2` (static URL used by all clients)
- **Integration Model:** Added as an MCP server within compatible chat or agent clients
- **Authentication:** Performed when adding the server to your application (client-side auth flow)
- **Provider:** Hosted and delivered via Pipedream Connect

## Features

- **MCP-compatible server** for connecting Intuiface to agents and chat clients that support the Model Context Protocol.
- **Static MCP URL** (`https://mcp.pipedream.net/v2`) usable across different clients, simplifying setup.
- **Client-based authentication** performed at the time you add the server to your app.
- **No-code integration target**: bridges MCP agents with Intuiface’s no-code platform for interactive digital experiences and digital signage.
- **Documentation & configuration support** via a dedicated configuration page (general MCP setup guidance through Pipedream).
