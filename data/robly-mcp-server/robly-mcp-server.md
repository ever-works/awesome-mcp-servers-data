# robly MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** robly  
**Type:** MCP server integration for email marketing

## Description
robly MCP Server is an MCP (Model Context Protocol) server for Robly, an email marketing platform used to build and send email campaigns. It is provided via Pipedream Connect and exposes Robly’s email marketing capabilities to MCP‑compatible clients.

## Features
- MCP server endpoint for Robly: `https://mcp.pipedream.net/v2` (static URL for all clients)
- Works with multiple MCP‑compatible chat or AI clients
- Authentication performed when adding the server in the client/application
- Integrates with Robly’s email marketing platform for creating and managing email campaigns (as described by the product summary)
- Hosted and managed via Pipedream Connect infrastructure
- Central configuration documentation available via a dedicated configuration page

## Integration & Usage
- Add the robly MCP Server to your MCP‑compatible app by using the static URL endpoint
- Authenticate within your client when prompted to connect to Robly
- Refer to the configuration page (on Pipedream) for client‑specific setup steps
