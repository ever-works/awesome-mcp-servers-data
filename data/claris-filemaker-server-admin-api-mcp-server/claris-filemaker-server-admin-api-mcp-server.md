## Claris FileMaker Server - Admin API MCP Server

**Category:** Server Management Tools  
**Brand:** Claris FileMaker  
**Source:** https://mcp.pipedream.com/app/claris_filemaker_server_admin_api

### Description
An MCP Server integration for the Claris FileMaker Server Admin API, enabling scripted administrative tasks and configuration management on FileMaker Server via MCP-compatible tools instead of the standard Admin Console UI.

### Features
- Provides access to the FileMaker Server Admin API through an MCP-compatible server endpoint.
- Enables web services and scripting tools to perform administrative tasks on FileMaker Server.
- Supports configuration and management operations that would otherwise require the FileMaker Admin Console UI.
- Uses a static MCP server URL that works for all clients: `https://mcp.pipedream.net/v2`.
- Integrates with chat and other MCP-capable clients for interactive server administration workflows.

### Technical Details
- MCP server endpoint: `https://mcp.pipedream.net/v2`
- Authentication is performed when adding the server to an application/client.
