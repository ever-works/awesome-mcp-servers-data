# Yumpu MCP Server

## Overview
Yumpu MCP Server is an MCP (Model Context Protocol) server integration that allows MCP-enabled tools and chat clients to interact with Yumpu’s digital publishing and reading platform. It is provided via Pipedream Connect as a static MCP endpoint.

- **Name:** Yumpu MCP Server  
- **Category:** Content Management MCP Servers  
- **Use cases:** Working with digital publications and documents on Yumpu via MCP-compatible clients  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP integration for Yumpu**: Exposes Yumpu’s digital publishing and reading capabilities to MCP-compatible tools and chat applications.
- **Single static endpoint**: Uses a single, static MCP server URL (`https://mcp.pipedream.net/v2`) that works across different MCP clients.
- **Client-agnostic setup**: Designed to be added to any supported MCP chat client; configuration is handled at the client level.
- **Authentication at connection time**: Authentication is performed when you add/configure the server in your MCP client, rather than via different URLs per client.
- **Configuration documentation**: A separate configuration page (on Pipedream) provides client-specific setup steps and details.
