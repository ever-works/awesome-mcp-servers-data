# ShipStation MCP Server

Import, manage, and ship orders using ShipStation’s shipping and order management features via an MCP (Model Context Protocol) server.

---

## Overview
- **Type:** MCP Server (integration service)
- **Category:** Business & Commerce – MCP Servers
- **Provider:** Pipedream (powered by Pipedream Connect)
- **Purpose:** Connect your MCP-compatible client to ShipStation so you can work with orders and shipping workflows directly from your chat or MCP-enabled applications.

---

## Features
- **Order Import**
  - Integrates ShipStation into MCP, allowing you to import orders from ShipStation into MCP-enabled tools.

- **Order Management**
  - Access and manage ShipStation orders via MCP.
  - Centralize order-related operations inside supported chat clients or applications.

- **Shipping Operations**
  - Use ShipStation’s shipping and order management capabilities through the MCP server.
  - Designed to help you handle shipping workflows (e.g., preparing orders for shipment) from your MCP client.

- **Static MCP Server Endpoint**
  - Uses a single static URL for all clients: `https://mcp.pipedream.net/v2`.
  - Authentication is performed when you add the server to your chosen MCP-compatible application.

- **Multi‑client Compatibility**
  - Works with any chat client or application that supports MCP servers.
  - Configuration guidance available via the provider’s configuration documentation.

---

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Authentication:** Performed during server setup in the client (details provided in client-specific or configuration docs).
