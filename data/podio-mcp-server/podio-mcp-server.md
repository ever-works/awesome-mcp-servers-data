# Podio MCP Server

Platform: **MCP Server** (via Pipedream)  
Category: **Project Management / No-Code / Workflow Automation**

## Overview
Podio MCP Server exposes Podioâ€™s no-code app, project, and workflow automation capabilities as tools within the MCP ecosystem. After connecting a Podio account, users can perform common Podio actions (items, tasks, files, statuses) directly from compatible MCP clients.

## Configuration
- Connect a Podio account and select a client to start using the server.
- Static MCP server URL (for all clients):
  - `https://mcp.pipedream.net/v2`
- Add this server URL to your MCP-compatible application, then authenticate when prompted.
- Additional client-specific setup guidance is available on the Configuration page (not specific to a single client).

## Features
- **Podio account integration**
  - Connect and manage a Podio account from within the MCP ecosystem.
  - Authenticate once per client using the shared server URL.

- **Available tools (actions)**
  1. **Upload From URL**
     - Upload a file to Podio from a remote URL.
  2. **Upload File**
     - Upload a new file from a local source to Podio.
  3. **Update an Item**
     - Update an existing item in a Podio app.
  4. **Find Item**
     - Filter and locate items in a specific Podio app based on criteria.
  5. **Create Task**
     - Create a task in a specified Podio workspace.
  6. **Create Status**
     - Create a status update in a specified Podio workspace.
  7. **Create an Item**
     - Add a new item to a specified Podio app.
  8. **Attach File**
     - Attach an uploaded file to a Podio object (e.g., item, task).

- **No-code workflow support**
  - Designed to help build no-code workflows, apps, and project automation using Podio as the backend.
