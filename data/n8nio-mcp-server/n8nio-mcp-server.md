# n8n.io MCP Server

**Category:** Workflow Automation MCP Servers  
**Brand:** n8n  
**Source:** https://mcp.pipedream.com/app/n8n_io

## Overview
The n8n.io MCP Server connects AI agents and MCP-compatible clients to n8n.io’s workflow automation platform. It allows orchestration of complex automations, integrations, and data flows in n8n directly from tools that support the Model Context Protocol (MCP).

## MCP Server URL
- Static MCP server endpoint (works for every compatible client):  
  `https://mcp.pipedream.net/v2`
- Authentication is performed when adding the server to your application.

## Features
- Connects AI agents and MCP clients to n8n.io’s workflow automation platform.
- Enables orchestration of complex automations and data flows via n8n from within MCP-compatible tools.
- Supports integrations and no-code workflows available in n8n.io.
- Uses a single static MCP server URL for all supported clients.
- Can be added to various chat or MCP-enabled applications as a remote server.

## Getting Started
- Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server to your MCP-compatible or chat-based application.
- Refer to the provider’s Configuration page for step-by-step setup instructions.
