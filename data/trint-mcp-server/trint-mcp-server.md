# Trint MCP Server

## Overview
The Trint MCP Server exposes Trint’s speech-to-text and media management capabilities to Model Context Protocol (MCP) clients. It enables transcription and search over audio and video content from within compatible chat or automation environments.

- **Category:** Media Processing MCP Server  
- **Brand:** Trint  
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- **Speech-to-text transcription**  
  - Use Trint’s speech-to-text platform via MCP tools.  
  - Transcribe audio and video files into text.

- **Searchable media**  
  - Make audio and video content searchable through generated transcripts.  
  - Supports search across transcribed content for specific words or phrases.

- **Editable and shareable transcripts**  
  - Access text outputs that can be edited and shared after transcription.  
  - Integrates Trint’s media management capabilities for ongoing content handling.

- **MCP integration**  
  - Single static MCP server URL (`https://mcp.pipedream.net/v2`) used across different MCP clients.  
  - Authentication handled when adding the server to the chosen application.  
  - Compatible with multiple chat / MCP client applications via standard configuration.

- **Account-based configuration**  
  - Connect a Trint account through the Pipedream-based configuration flow.  
  - Central configuration page available for managing MCP server setup.

## Setup
1. **Copy MCP server URL**  
   - Use `https://mcp.pipedream.net/v2` as the MCP server endpoint.
2. **Add to client**  
   - Add this endpoint in your MCP-compatible chat or automation client.  
   - Authenticate with your Trint/Pipedream-connected account as prompted.
3. **Configure account**  
   - Use the configuration page (from the Pipedream app) to connect and manage your Trint account.
