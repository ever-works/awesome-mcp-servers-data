# Interseller MCP Server

## Overview
The **Interseller MCP Server** connects Interseller’s outbound automation and prospecting capabilities into MCP-compatible environments, enabling you to use Interseller features from within MCP-aware chat clients and applications.

- **Name:** Interseller MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Use cases:** Sales prospecting, outbound outreach automation, lead generation workflows within MCP clients

## MCP Server URL
Use this static URL for all MCP clients (authentication is handled when you add the server to your app):

```text
https://mcp.pipedream.net/v2
```

## Features
- **MCP-compatible integration:** Exposes Interseller’s outbound automation and prospecting capabilities through the MCP protocol so they can be accessed from compatible chat clients and tools.
- **Single static endpoint:** One URL (`https://mcp.pipedream.net/v2`) works across all supported MCP clients, simplifying configuration.
- **Client-agnostic setup:** Designed to be added to multiple MCP-aware chat clients; configuration is handled on the client side.
- **Authentication during setup:** Authentication to Interseller occurs when adding the MCP server to your application (no per-request manual auth needed in the URL itself).
- **Central configuration reference:** A dedicated configuration page (via Pipedream) provides full details for client-specific setup and options.
