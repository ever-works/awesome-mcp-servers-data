# Formbricks MCP Server

**Category:** Business & Commerce / MCP Servers  
**Brand:** Formbricks  
**Source:** https://mcp.pipedream.com/app/formbricks

## Overview
Formbricks MCP Server integrates Formbricks’ open‑source experience management and user research capabilities into MCP-compatible clients. It enables running surveys and feedback flows directly from MCP-based chat applications, helping teams understand what customers think and feel about their product with a privacy-first approach.

## MCP Server URL
- Base MCP server endpoint (static for all clients):  
  `https://mcp.pipedream.net/v2`
- Authentication is performed when adding the server to your MCP-compatible application.

## Features
- **MCP integration**
  - Exposes Formbricks functionality as an MCP server
  - Works with any compatible MCP client via a single static URL
- **Surveys & feedback flows**
  - Supports running surveys through MCP clients
  - Facilitates customer and user feedback collection from within chat-based tools
- **User research capabilities**
  - Brings Formbricks’ user research tools into MCP environments
  - Helps teams understand customer sentiment and product experience
- **Open-source experience management**
  - Built on Formbricks’ open-source stack for experience management
- **Privacy-first design**
  - Emphasizes privacy-conscious handling of user research and feedback data
- **Minimal developer overhead**
  - Aims to integrate user research workflows with minimal developer attention
- **Configuration resources**
  - Central configuration and setup documentation available via the Pipedream configuration page

## Integration & Setup
- Use the static MCP server URL in your MCP-compatible chat client.
- Authentication occurs when the server is added to your app.
- Additional configuration guidance is provided on the linked configuration page from Pipedream.
