# Amplenote MCP Server

An MCP (Model Context Protocol) server that connects to Amplenote, enabling AI agents and MCP-compatible clients to work with notes and tasks from within their chat or development environment.

---

## Overview
- **Name:** Amplenote MCP Server  
- **Category:** Content Management MCP Servers  
- **Use Cases:** Notes, task management, productivity  
- **Provider / Platform:** Runs via Pipedream Connect  
- **Static MCP Endpoint:** `https://mcp.pipedream.net/v2`

This server integrates Amplenote with MCP-supported applications so agents can create, organize, and act on Amplenote content directly from compatible chat clients or tools.

---

## Features
- **MCP-Compatible Endpoint**  
  - Single static URL (`https://mcp.pipedream.net/v2`) for use across supported MCP clients.  
  - Authentication handled when adding the server in the client/application.

- **Amplenote Integration**  
  - Connects to an Amplenote account to work with notes and tasks.  
  - Designed to support workflows around capturing ideas, organizing information, and executing to-dos within Amplenote.

- **Multi-Client Support**  
  - Can be added to different MCP-compatible chat clients or applications.  
  - Each client follows its own setup flow to register and authenticate the server.

- **Configuration Documentation**  
  - Central Configuration page (linked from the app) describing how to add and configure the server for various clients.

---

## Setup
1. Use the static MCP server URL:  
   `https://mcp.pipedream.net/v2`
2. Add this server URL to your MCP-supported chat client or application.  
3. Complete the authentication flow as prompted by the client.  
4. Optionally refer to the full configuration page for per-client setup details.
