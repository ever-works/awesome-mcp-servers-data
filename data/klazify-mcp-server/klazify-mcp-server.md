# Klazify MCP Server

An MCP server that exposes the Klazify Content Classification API via Pipedream for analyzing and categorizing domains, URLs, and companies.

- **Website / Source**: https://mcp.pipedream.com/app/klazify  
- **Server URL**: `https://mcp.pipedream.net/v2`  
- **Category**: Data Analytics, Data Analysis & Exploration MCP Servers  
- **Tags**: classification, content-analysis, api-integration

## Features

### MCP Server & Integration
- Static MCP server URL (`https://mcp.pipedream.net/v2`) usable for all Klazify clients.
- Authentication handled when adding the server to your application or chat client.
- Can be added to various chat / client applications (via the app’s configuration flow).
- Central configuration page available for detailed setup instructions.

### Available Tools / Actions (7 total)
1. **Real Time Categorization**  
   - Analyzes URL content or an email’s domain (if the input is an email).  
   - Classifies the resource into **385+ topic categories** using Klazify’s content classification API.

2. **Get Media Links Scraper**  
   - Retrieves media links associated with a given domain.

3. **Get Logo**  
   - Fetches the logo of a company based on its domain or associated data.

4. **Get Domain Tech Stack**  
   - Returns information on the technology stack used by a specified domain.

5. **Get Domain Expiration**  
   - Retrieves the expiration date of a given domain.

6. **Get Company Domain Info**  
   - Provides detailed information about a company’s domain.

7. **Categorize Website**  
   - Submits a website URL to the Klazify API for content categorization.

## Use Cases
- Classifying websites or email domains into detailed topic categories for analytics, personalization, or filtering.
- Enriching company/domain data in applications with logos, tech stack, expiration, and domain intelligence.
- Building data workflows that rely on automated content analysis and domain enrichment via MCP-compatible clients.
