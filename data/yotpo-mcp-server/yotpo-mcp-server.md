# Yotpo MCP Server

## Overview
Yotpo MCP Server is an MCP (Model Context Protocol) server that connects Yotpoâ€™s reviews and ratings platform to MCP-compatible agents or chat clients. It enables automated access to eCommerce reviews, ratings, and related marketing data directly within MCP-based applications.

## Features
- **MCP-compatible server endpoint**  
  - Static server URL for all clients: `https://mcp.pipedream.net/v2`  
  - Designed to be added as an MCP server in supported chat / agent clients.

- **Integration with Yotpo Reviews & Ratings**  
  - Access to customer product reviews data.  
  - Access to ratings associated with products and services.  
  - Exposure of Yotpo reviews/ratings data to MCP agents for automated workflows and reasoning.

- **eCommerce marketing data access**  
  - Integration point for broader Yotpo eCommerce marketing information (e.g., reviews and related marketing metadata) for use in AI agents and tools.

- **Authentication handled at client setup**  
  - A single shared server URL; authentication is performed when adding the server to each application/client.

## Setup
- Use the static MCP server URL: `https://mcp.pipedream.net/v2`  
- Add the server to an MCP-compatible chat client or agent and authenticate as prompted.
