# BoldSign MCP Server

**Category:** Business & Commerce – MCP Servers  
**Website:** https://mcp.pipedream.com/app/boldsign

## Overview
The BoldSign MCP Server exposes BoldSign’s electronic signature APIs and workflows to MCP-compatible automation tools and chat clients. It provides a unified MCP endpoint you can add to your application to work with BoldSign’s e-signature capabilities through the MCP protocol.

## MCP Server URL
- Static MCP endpoint for all clients:  
  `https://mcp.pipedream.net/v2`
- Authentication is handled when adding the server inside your client or application.

## Features
- **MCP-compatible e-signature access**: Integrates BoldSign’s electronic signature functionality with tools and chat clients that support the Model Context Protocol (MCP).
- **Workflow exposure via MCP**: Makes BoldSign’s signature workflows available as MCP actions/operations for automation.
- **Single static endpoint**: Uses a single, static MCP URL (`https://mcp.pipedream.net/v2`) for all clients, simplifying configuration and reuse.
- **Client-agnostic setup**: Designed to work with multiple MCP-compatible chat clients and applications; you authenticate within your chosen client.
- **Configuration documentation**: A dedicated configuration page is available (linked from the app page) to guide setup and usage of the MCP server.

## Integration & Usage
- Add the server to any MCP-compatible chat client or automation environment using the provided URL.
- Follow your client’s MCP server setup flow, where you’ll authenticate with BoldSign as needed.
