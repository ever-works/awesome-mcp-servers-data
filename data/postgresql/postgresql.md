**Category:** MCP Server Directories & Lists

### Overview
The PostgreSQL MCP server provides read-only access to PostgreSQL databases, enabling schema inspection and querying for analytics and data exploration use cases.

### Features
- Read-only access to PostgreSQL databases
- Schema inspection capabilities
- Query execution for analytics
- Query execution for data exploration
