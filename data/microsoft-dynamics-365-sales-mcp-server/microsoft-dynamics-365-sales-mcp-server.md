# Microsoft Dynamics 365 Sales MCP Server

Microsoft Dynamics 365 Sales MCP Server provides MCP-based access to Dynamics 365 Sales, enabling AI-powered sales and CRM workflows to be integrated into MCP-driven automations and agents.

## Features

### MCP Server & Connectivity
- Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`  
- Works with multiple MCP-compatible chat clients (e.g., ChatGPT / OpenAI) via configuration guides.  
- Uses your Microsoft Dynamics 365 Sales account for authentication when adding the server to your application.

### CRM Access & Automation
- Provides MCP-based access to Microsoft Dynamics 365 Sales CRM data.  
- Integrates AI-powered sales capabilities and deal management into MCP-driven workflows, automations, and agents.

### Available Tools (Actions)
1. **Find Contact**  
   - Search for a contact by:  
     - ID  
     - Name  
     - Custom filter criteria  
   - Returns matching contact data from Dynamics 365 Sales.

2. **Create Custom Entity**  
   - Create a custom entity record in Microsoft Dynamics 365 Sales.  
   - Supports specifying custom fields as defined in your Dynamics 365 Sales environment.

### Configuration & Usage
- Connect your Microsoft Dynamics 365 Sales account through the Pipedream MCP interface.  
- Select and configure the desired client (e.g., ChatGPT) to use the MCP server tools.  
- Documentation for each action is available via linked GitHub component files.
