# MetaMCP

**Category:** MCP Middleware & Orchestration  
**Website/Source:** https://github.com/metatool-ai/metamcp

## Description
MetaMCP is a unified middleware MCP server that can aggregate, orchestrate, and route requests across multiple MCP servers. It provides a central hub, deployable in a single Docker container, with a graphical interface for configuring and managing many MCP connections.

## Features
- **MCP Aggregation**: Connects to multiple MCP servers and exposes them through a single unified MCP endpoint.
- **Orchestration Layer**: Coordinates and routes requests across different MCP backends from one place.
- **Middleware / Gateway**: Acts as a gateway in front of multiple MCP services, handling request flow between clients and servers.
- **Single-Docker Deployment**: Packaged to run as an all-in-one MCP aggregator/orchestrator/middleware/gateway via Docker.
- **Central Configuration Hub**: Provides a central point to configure and manage many MCP connections.
- **GUI Management**: Includes a graphical user interface to manage connections, configuration, and routing behavior.
