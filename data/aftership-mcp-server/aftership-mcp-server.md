# AfterShip MCP Server

AfterShip MCP Server exposes AfterShip’s shipment tracking and post‑purchase experience capabilities through the Model Context Protocol (MCP), allowing tools and AI agents to access logistics and tracking data via an MCP server.

- **Tags**: shipping, logistics, tracking

## Features

- **MCP-based access to AfterShip**
  - Exposes AfterShip shipment tracking and post-purchase experience capabilities through the Model Context Protocol.
  - Designed for use by tools and AI agents that need programmatic access to logistics and tracking data.

- **Standard MCP server endpoint**
  - Static MCP server URL for all compatible clients:
    - `https://mcp.pipedream.net/v2`
  - Authentication is handled when adding the server to your MCP-compatible application or chat client.

- **Client-agnostic integration**
  - Works with any MCP-capable chat client or application.
  - Can be added to different apps by configuring the shared server URL and authentication.

- **Hosted by Pipedream Connect**
  - Server infrastructure and hosting provided via Pipedream Connect.
