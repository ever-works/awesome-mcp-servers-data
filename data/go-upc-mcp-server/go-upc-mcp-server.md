# Go-UPC MCP Server

**Category:** Data Access & Integration – MCP Servers  
**Brand:** go-upc  
**Source:** https://mcp.pipedream.com/app/go_upc

## Description
Go-UPC MCP Server exposes the Go-UPC barcode database through the Model Context Protocol (MCP), enabling AI tools to look up product information by UPC, EAN, and ISBN codes.

## Features
- **MCP-based integration**: Provides an MCP server interface for accessing the Go-UPC barcode database.
- **Barcode lookups**: Supports product lookup via:
  - UPC (Universal Product Code)
  - EAN (European Article Number)
  - ISBN (International Standard Book Number)
- **AI tool connectivity**: Designed to connect a Go-UPC account to AI tools via Pipedream’s MCP server, including (as examples):
  - Cursor
  - Claude
  - Windsurf
- **Product data access**: Retrieves structured product data associated with barcodes from the Go-UPC database.
- **API-style integration**: Acts as an integration layer so AI agents and tools can query product information programmatically.
