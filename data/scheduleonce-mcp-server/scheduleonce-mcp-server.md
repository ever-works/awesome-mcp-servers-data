# ScheduleOnce MCP Server

## Overview
The ScheduleOnce MCP Server is an MCP (Model Context Protocol) server that lets AI agents manage and interact with lead scheduling and booking workflows powered by ScheduleOnce.

- **Category:** Business & Commerce MCP Servers  
- **Use cases:** Lead scheduling, booking management, and lead engagement workflows
- **MCP server URL:** `https://mcp.pipedream.net/v2`

## Features
- **Lead scheduling workflows** – Enables AI agents to work with ScheduleOnce to schedule and manage meetings with leads.
- **Booking workflows** – Supports interaction with booking processes, allowing automated handling of appointment bookings.
- **Lead management support** – Designed to help capture, qualify, and engage inbound leads through scheduling and booking flows.
- **Static MCP server endpoint** – Uses a single static URL (`https://mcp.pipedream.net/v2`) that works across clients.
- **Client-agnostic integration** – The same MCP server URL can be added to any compatible chat or AI client.
- **Authentication at client level** – Authentication is performed when adding the server to the client/application rather than using per-user URLs.

## Configuration
- Add the MCP server to your chat or AI client using the static URL: `https://mcp.pipedream.net/v2`.
- Follow the client-specific steps in the MCP configuration documentation (referenced as a “Configuration page” in the source content).
