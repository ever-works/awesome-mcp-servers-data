# FedEx MCP Server

An MCP Server integration for FedEx that enables MCP tools to create, manage, track, and support delivery operations using FedEx shipping services.

## Features

- FedEx integration exposed as an MCP server endpoint usable by multiple MCP-compatible clients.
- Single static MCP server URL (`https://mcp.pipedream.net/v2`) for all clients; authentication handled in the client/app configuration.
- Designed to support:  
  - Creating and managing FedEx shipments (shipping operations).  
  - Tracking shipments and delivery status.  
  - Managing logistics flows related to FedEx delivery operations.
- Compatible with various chat or MCP-enabled applications that can add external MCP servers.
- Configuration details available via a dedicated configuration page (client-specific setup instructions).

## Integration & Usage

- Add the MCP server URL to your MCP-compatible or chat-based application.  
- Authenticate within your client when prompted (credentials / tokens managed at the client side).  
- Once configured, MCP tools can invoke FedEx-related operations via this server for shipping, logistics, and tracking workflows.
