# Lucid MCP Server

**Category:** Content Management MCP Servers  
**Brand:** Lucid  
**Slug:** lucid-mcp-server

## Description
Lucid MCP Server exposes Lucid’s Visual Collaboration Suite to MCP-compatible clients, allowing tools to create and manage visual diagrams and collaborative workspaces for hybrid and enterprise teams via a unified MCP endpoint.

## MCP Server URL
- Base MCP server URL (static for all clients):
  - `https://mcp.pipedream.net/v2`
- Authentication is performed when adding the server to your MCP-compatible application or chat client.

## Features
- **Visual Collaboration Integration**
  - Connects MCP tools to Lucid’s Visual Collaboration Suite.
  - Supports hybrid and enterprise teams of various sizes.
- **Static MCP Endpoint**
  - Single, static server URL usable across different MCP clients.
  - Simplifies configuration and reuse across tools.
- **Client-Agnostic Setup**
  - Designed to be added to multiple chat or MCP-enabled clients.
  - Works with any compatible MCP client that can consume the given URL.
- **Configuration Documentation**
  - Full configuration instructions available via a dedicated configuration page (external to this summary).
- **Pipedream Connect Integration**
  - MCP server is powered and hosted by Pipedream Connect.
