# Proofly MCP Server

**Description**  
An MCP (Model Context Protocol) server integration for the Proofly social proof and user‑behavior platform, enabling MCP‑compatible agents and AI workflows to trigger, query, and manage Proofly campaigns.

- **Category:** Business & Commerce → MCP Servers  
- **Slug:** `proofly-mcp-server`  
- **Brand:** Proofly  
- **Source URL:** https://mcp.pipedream.com/app/proofly

## Features

- **MCP-compatible server**  
  - Exposes Proofly functionality over the Model Context Protocol.  
  - Designed to be added as a server within MCP‑compatible chat and AI clients.

- **Campaign interaction**  
  - Lets AI agents trigger and interact with Proofly social‑proof campaigns.  
  - Enables querying and managing campaigns as part of automated workflows (per item description).

- **User behavior & social proof integration**  
  - Connects to the Proofly platform, which displays user behavior on a website and supports social‑proof use cases (e.g., showing activity, creating urgency, building trust, and driving conversions).

- **Static MCP endpoint**  
  - Uses a single, static MCP server URL for all clients:  
    - `https://mcp.pipedream.net/v2`  
  - Authentication is handled when the server is added to the client application.

- **Client-agnostic setup**  
  - Works with multiple MCP‑compatible chat clients.  
  - Setup instructions are available per client via the configuration documentation.

- **Configuration documentation**  
  - Full configuration and setup details are available through the provided Configuration page (hosted by Pipedream).

## Integration & Usage

- Add the server URL `https://mcp.pipedream.net/v2` to an MCP‑compatible app or chat client.
- Authenticate within the client when prompted.
- Use within AI workflows to programmatically work with Proofly campaigns and social‑proof behavior data.
