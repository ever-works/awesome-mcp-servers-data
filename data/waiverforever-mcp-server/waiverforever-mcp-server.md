## Features
- Integration with WaiverForever for working with digital waivers.  
- Support for kiosk-based signature workflows.  
- Paperless creation, signing, and storage of waivers.  
- Centralized, unified tool for managing customized electronic waivers for kiosk environments.  
- MCP-compatible server endpoint (`/v2`) that can be added to any MCP-capable chat client.  
- Authentication handled when adding the server to the client (static URL shared across clients).

## Integration & Usage
- Use the static MCP server URL `https://mcp.pipedream.net/v2` for all clients.  
- Add the server in any supported MCP-compatible chat client and authenticate during setup.  
- Configuration details and client-specific setup instructions are available on the associated configuration page of the provider (via Pipedream Connect).
