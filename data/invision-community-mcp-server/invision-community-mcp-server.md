# Invision Community MCP Server

MCP Server for Invision Community, enabling programmatic control and integration of community platform features via the MCP protocol.

- **Category:** Messaging MCP Servers  
- **Brand:** Invision Community  
- **Tags:** community, forum, collaboration  
- **Source URL:** https://mcp.pipedream.com/app/invision_community

## MCP Server Endpoint

- **Base MCP Server URL:** `https://mcp.pipedream.net/v2`  
  - Static URL shared across clients.  
  - Authentication is performed when adding the server in your client or application.

## Features

- Provides an MCP-compatible server interface for Invision Community.  
- Enables programmatic control of Invision Community features via the MCP protocol.  
- Supports integration of community / forum functionality into MCP-capable chat and AI clients.  
- Uses a single static MCP endpoint (`https://mcp.pipedream.net/v2`) for all clients.  
- Designed to be added as an MCP server in supported applications (chat clients, AI tools, etc.).

## Integration & Configuration
