# Tolstoy MCP Server

Deliver personal conversations with Tolstoy interactive videos directly through MCP-compatible clients.

## Overview
Tolstoy MCP Server is an MCP (Model Context Protocol) server integration for Tolstoy that allows MCP clients (such as AI chat applications) to work with Tolstoy’s interactive video experiences. It enables automated access to and management of Tolstoy conversations from within MCP-enabled tools.

- **Category:** Media Processing MCP Servers  
- **Slug:** `tolstoy-mcp-server`  
- **Brand:** Tolstoy  
- **Source URL:** https://mcp.pipedream.com/app/tolstoy

## Features
- **MCP server integration for Tolstoy** – Exposes Tolstoy interactive video capabilities via the Model Context Protocol so MCP-compatible clients can interact with Tolstoy.
- **Interactive video conversations** – Works with Tolstoy interactive video experiences and conversations for engagement and marketing use cases.
- **Static MCP server URL** – Single static endpoint usable across clients:  
  `https://mcp.pipedream.net/v2`
- **Client-agnostic setup** – Designed to work with multiple MCP clients; same URL can be added to different chat or AI applications.
- **Authentication at configuration time** – Authentication is performed when adding the MCP server to your application (per-client auth rather than per-URL).
- **Configuration documentation** – A dedicated configuration page (via Pipedream Connect) explains how to add the server to various chat clients.
- **Pipedream Connect integration** – Hosted and delivered via Pipedream’s Connect platform.
- **Use cases** – Suitable for interactive video marketing, video-based engagement, and personalized conversational experiences.
