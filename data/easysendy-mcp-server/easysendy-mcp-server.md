# EasySendy MCP Server

## Description
EasySendy MCP Server is a Model Context Protocol (MCP) server hosted on Pipedream that exposes EasySendy’s APIs to manage and automate drip email campaigns and broader email marketing workflows from MCP‑compatible tools and chat clients.

## Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Can be added to MCP-capable applications and chat clients.

- **EasySendy integration**  
  - Connects an EasySendy account to MCP-compatible tools.  
  - Enables control of EasySendy email marketing features via the MCP interface.

- **Drip email campaign automation**  
  - Designed to manage and automate drip email campaigns.  
  - Intended to run campaigns triggered and managed through your website or connected apps.

- **Email marketing workflows**  
  - Exposes EasySendy APIs for broader email marketing workflows beyond basic campaigns.  
  - Suitable for integrating campaign operations into automated workflows built on Pipedream or other MCP-aware tools.

- **Multi-client support**  
  - Static server URL works for every client; authentication occurs when adding the server to each application.  
  - Supports configuration with different chat clients via the MCP specification.

- **Configuration guidance**  
  - Guided flow to connect your EasySendy account and select a client.  
  - Documentation available via a configuration page (on Pipedream) for setup details.
