# Hullo MCP Server

An MCP Server integration for Hullo on Pipedream, providing business SMS capabilities inside automated workflows.

## Overview
Hullo MCP Server exposes a static MCP endpoint that can be added to compatible chat or MCP-enabled clients, enabling integration of next-generation business SMS features into your applications and workflows.

## MCP Server URL
- Static MCP server endpoint (works for every client):
  - `https://mcp.pipedream.net/v2`
- Authentication occurs when you add the server to your application.

## Features
- **Static MCP endpoint** usable across all supported MCP clients.
- **Business SMS integration** via Hullo within Pipedream-powered workflows.
- **Client-agnostic setup**: same URL for every client; configuration handled per client.
- **Configurable via chat clients**: can be added to supported chat or MCP-enabled applications.
- **Centralized configuration reference** available through a dedicated configuration page on Pipedream (for step-by-step setup in different clients).

## Setup
- Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server to your chat or MCP-enabled app.
- Follow client-specific instructions from the configuration page if needed.
