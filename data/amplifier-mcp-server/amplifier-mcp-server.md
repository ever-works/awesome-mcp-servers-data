# Amplifier MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Amplifier  
**Slug:** amplifier-mcp-server

## Overview
Amplifier MCP Server is an MCP integration that connects MCP-aware agents to Amplifier’s vertically integrated merchandise logistics platform. It enables agents and applications to interact with fulfillment and logistics workflows for community commerce and e‑commerce brands via a standard MCP endpoint.

## Features
- MCP integration for Amplifier’s merchandise logistics platform
- Supports interaction with fulfillment workflows
- Supports interaction with logistics operations for community commerce brands
- Single static MCP server URL usable by any compatible client
- Authentication handled when adding the server to an MCP-aware application
- Hosted and powered via Pipedream Connect infrastructure

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Integration type:** MCP server (for MCP-aware chat/agent clients)
- **Usage:** Add the static server URL to your MCP-compatible client, then authenticate within the client
