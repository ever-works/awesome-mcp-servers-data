# Griptape MCP Server

**Description**  
An MCP server for the Griptape AI agent framework and development platform, enabling MCP-based agents and clients to access Griptape’s tooling and workflows via Pipedream Connect.

**Category**  
AI Integration → MCP Servers

**Links**  
- Source / Setup: https://mcp.pipedream.com/app/griptape
- MCP Server URL: `https://mcp.pipedream.net/v2`
- Configuration Guide: /configuration (on Pipedream)

---

## Features

- **MCP-compatible server** for integrating Griptape with MCP-based chat clients and agents.
- **Static MCP endpoint** (`https://mcp.pipedream.net/v2`) that works for all supported clients.
- **Client-agnostic setup**: same server URL can be used across different MCP-enabled chat applications.
- **Authentication at client setup**: authentication handled when adding the server to your app, not via per-request URLs.
- **Integration via Pipedream Connect**: runs on Pipedream’s MCP infrastructure, benefiting from its connection and management layer.
- **Configuration guidance**: documentation available via a dedicated configuration page for adding the server to supported clients.
