# AWS IAM MCP Server

**Category:** Security Attestation MCP Servers  
**Brand:** AWS  
**Website:** https://modelcontextprotocol.io/mcp/servers/iam-mcp-server

## Overview
AWS IAM MCP Server is an MCP server designed for managing AWS Identity and Access Management (IAM) resources. It focuses on user, role, group, and policy management following AWS security best practices.

## Features
- IAM user management (create, update, manage users)
- IAM role management (create, update, manage roles)
- IAM group management (create, update, manage groups)
- IAM policy management (create, attach, detach, and manage policies)
- Alignment with AWS IAM security best practices for access control and permissions design
