## Overview
The Govee MCP Server is a Model Context Protocol (MCP) server that lets MCP-compatible agents and chat clients control and interact with Govee smart home devices. It focuses on ambient lighting, home appliances, and other IoT systems in the Govee ecosystem.

## Features
- **Static MCP Server Endpoint**  
  - Single static URL for all clients: `https://mcp.pipedream.net/v2`  
  - Works across different MCP-compatible chat applications  
  - Authentication handled when adding the server to your application

- **Account Integration**  
  - Connect and authenticate your Govee account via Pipedream  
  - Manage connected Govee accounts for use with MCP clients

- **Device Information Retrieval**  
  - Tool: **Retrieve Device Info**  
  - Fetch current status and metadata for a specific Govee device  
  - Access attributes such as:  
    - Power state (on/off)  
    - Color  
    - Mode  
    - Device health and related metadata

- **Device Control**  
  - Tool: **Control Device**  
  - Send commands to Govee devices, including:  
    - Turn power on/off  
    - Adjust brightness  
    - Change color  
    - Modify other supported device parameters and modes

- **Smart Home Automation Use Cases**  
  - Integrate Govee ambient lighting and appliances into AI workflows  
  - Allow MCP agents and chat clients to automate and orchestrate Govee devices as part of broader smart home or IoT flows

- **Client Configuration Support**  
  - Designed to be added to various chat clients that support MCP  
  - Additional setup guidance available via the Pipedream configuration page
