# PDF-API.io MCP Server

## Overview
PDF-API.io MCP Server is an integration that connects your PDF-API.io account to AI tools through Pipedream’s MCP server, enabling programmatic PDF generation and manipulation within MCP-powered workflows.

## Features
- Connects PDF-API.io to AI tools (e.g., Cursor, Claude, Windsurf) via Pipedream’s MCP server
- Provides programmatic access to PDF-API.io capabilities within MCP workflows
- Supports dynamic PDF generation
- Supports PDF manipulation operations (e.g., modifying or processing PDF content) through API-based calls

## Integrations
- Pipedream MCP server
- Cursor
- Claude
- Windsurf
