# Awesome MCP Servers (wong2)

**Type:** Open-source curated directory / awesome list  
**Category:** MCP server directories & lists  
**Repository:** https://github.com/wong2/awesome-mcp-servers  
**Website & submissions:** https://mcpservers.org (submit at https://mcpservers.org/submit)  
**License:** MIT

## Overview
Awesome MCP Servers is a curated list of Model Context Protocol (MCP) servers maintained by wong2. It catalogs notable MCP server implementations and integrations, and is mirrored on the public directory at mcpservers.org.

## Features
- **Curated MCP server directory**
  - Focused specifically on Model Context Protocol (MCP) server implementations and integrations.
  - Organized as an "awesome list" in a public GitHub repository.

- **Reference servers section**
  - Highlights servers intended to demonstrate MCP features and official SDK usage:
    - **Everything** – reference/test server with prompts, resources, and tools.
    - **Fetch** – server focused on fetching web content.
  - Designed to showcase MCP capabilities with the TypeScript and Python SDKs.

- **Web directory integration**
  - Connected to the public directory at **mcpservers.org**.
  - New MCP servers are submitted via a dedicated web form (not via GitHub pull requests).

- **Open-source repository**
  - Hosted on GitHub under the MIT license.
  - Publicly accessible for browsing, forking, and local use.

- **Ecosystem visibility**
  - Includes a sponsor section referencing MCP-related services and data providers (e.g., data access, monitoring, financial data MCP servers).

## Contribution / Submission
- Pull requests to the GitHub repository are **not** accepted.
- New MCP servers should be submitted through: **https://mcpservers.org/submit**.
