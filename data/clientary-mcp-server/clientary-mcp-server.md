# Clientary MCP Server

**Category:** Business & Commerce – MCP Servers  
**Tags:** invoicing, client-management, small-business

An MCP server that connects MCP-compatible clients to the Clientary platform, enabling management of clients, invoices, projects, proposals, estimates, hours, payments, contractors, and staff via a unified endpoint.

---

## Features

- **MCP-compatible server**
  - Exposes Clientary functionality through the Model Context Protocol (MCP)
  - Single static MCP server URL: `https://mcp.pipedream.net/v2`
  - Works with any compatible chat or MCP client; authentication is handled when adding the server to the application

- **Client management**
  - Access and manage client records from Clientary
  - Allows selecting clients after connecting your account

- **Invoicing and billing**
  - Integrates Clientary’s invoice management features via MCP
  - Supports workflows related to invoices and payments

- **Project tracking**
  - Access project-related data from Clientary
  - Includes support for hours / time-tracking information

- **Proposals & estimates**
  - Surface and manage proposals and estimates stored in Clientary

- **Payments**
  - Access payment information connected to Clientary invoices

- **Team & contractor management**
  - Access data related to contractors and staff within Clientary

- **Configuration through Pipedream**
  - Connect your Clientary account via Pipedream
  - Use Pipedream’s configuration flow to select clients and initialize the connection

## Setup

1. Configure Clientary via the Pipedream interface.
2. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
3. Add this MCP server to your MCP-compatible application or chat client.
4. Authenticate when prompted inside your application to link your Clientary account.
