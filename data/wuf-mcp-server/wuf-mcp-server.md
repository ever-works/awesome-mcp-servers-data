# Wuf MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Wuf  
**Source:** https://mcp.pipedream.com/app/wuf

## Description
Wuf MCP Server is an MCP server that integrates the Wuf mobile push notification service into MCP-compatible clients and workflows. It enables sending and managing mobile push notifications as part of automated workflows.

## Features
- MCP-compatible server endpoint for Wuf push notifications
- Integration of Wuf mobile push notification service into workflows
- Supports sending mobile push notifications via MCP
- Supports managing mobile push notifications via MCP (e.g., as part of workflow logic)
- Static MCP server URL usable across different clients: `https://mcp.pipedream.net/v2`
- Authentication handled when adding the server to an MCP-compatible application
- Works with multiple chat or MCP clients (configured per client)

## Configuration
- Use the static MCP server URL: `https://mcp.pipedream.net/v2`
- Add the server within your MCP/chat client and authenticate as prompted
- Additional configuration details are available on the linked configuration page from the source site
