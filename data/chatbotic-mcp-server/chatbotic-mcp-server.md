## Chatbotic MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Chatbotic  
**Slug:** `chatbotic-mcp-server`

Chatbotic MCP Server is an MCP (Model Context Protocol) server for the Chatbotic platform, enabling smart, contextual responses to visitor inquiries within compatible chat clients.

---

### Features

- **MCP-compatible server endpoint**  
  - Static base URL for all clients: `https://mcp.pipedream.net/v2`  
  - Works as a shared MCP server endpoint across different applications.

- **Contextual response capability**  
  - Designed to power Chatboticâ€™s ability to deliver contextual, intelligent responses to visitor questions.

- **Integration with chat clients**  
  - Can be added as a server in MCP-compatible chat clients.  
  - Works within the broader Pipedream / Pipedream Connect ecosystem.

- **Hosted infrastructure**  
  - Server is hosted and maintained by Pipedream (now part of Workday).  
  - No separate server deployment required by the user.

- **Configuration resources**  
  - Full configuration documentation available via the Pipedream Configuration page (for setup and usage patterns).

---

### Integration & Usage

- Use the static MCP server URL when configuring your MCP-enabled chat client.
- Authenticate within your application when adding the server (authentication is not embedded in the URL itself).
