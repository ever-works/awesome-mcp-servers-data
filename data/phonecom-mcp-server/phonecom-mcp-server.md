# Phone.com MCP Server

An MCP Server integration for Phone.com, exposing cloud business phone, voice, video, text, and call management capabilities via MCP.

- **Category:** Messaging MCP Servers  
- **Tags:** telephony, VoIP, business-communications  
- **Provider / Brand:** Phone.com  
- **Source URL:** https://mcp.pipedream.com/app/phone_com

## Features

- Connect a business phone number to any device (desktop, mobile, etc.).
- Access Phone.com’s cloud-based small business communication solution through an MCP server.
- Voice calling features comparable to an enterprise phone system.
- Video communication capabilities.
- Business text messaging (SMS / messaging) support.
- Call management features (e.g., routing, handling, and control via Phone.com’s system).
- Collaboration-oriented communication tools (e.g., team-oriented calling / conferencing features exposed via Phone.com).
- Static MCP server URL usable across compatible MCP clients:  
  - `https://mcp.pipedream.net/v2`
- Works with multiple MCP-compatible chat clients (add the server URL and authenticate within the client).

## Integration & Configuration

- Add the MCP server URL directly to your MCP-compatible application or chat client.
- Authentication is performed when the server is added to the client.
- Additional configuration guidance is available via the Pipedream Connect configuration page (external documentation).
