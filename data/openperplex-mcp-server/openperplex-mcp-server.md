## Description
OpenPerplex MCP Server is an MCP server that connects your OpenPerplex account to MCP-integrated AI tools, enabling advanced content retrieval and search capabilities. It is available via Pipedream and can be used with tools such as Cursor, Claude, and Windsurf.

Source: <https://mcp.pipedream.com/app/openperplex>
