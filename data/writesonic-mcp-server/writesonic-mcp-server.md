# Writesonic MCP Server

**Category:** AI Integration – MCP Servers  
**Brand:** Writesonic  
**Slug:** writesonic-mcp-server

## Description
Writesonic MCP Server connects the Writesonic AI writing platform to MCP-based clients, allowing users to programmatically generate marketing and written content such as blog posts, emails, and landing pages.

## Features
- Connects Writesonic’s AI writing capabilities to any MCP-compatible client.
- Supports programmatic generation of:
  - Blog posts
  - Marketing emails
  - Landing page copy
  - Other marketing content
- Uses a single static MCP server URL for all supported clients: `https://mcp.pipedream.net/v2`.
- Authentication handled when adding the server within each client or application.
- Documentation and configuration guidance available via the Pipedream Configuration page.

## Integration & Usage
- Add the MCP server URL to your MCP-enabled chat client or application.
- Authenticate through the client when prompted.
- Use the integrated Writesonic tools to generate or refine marketing and other written content directly from your MCP-based environment.
