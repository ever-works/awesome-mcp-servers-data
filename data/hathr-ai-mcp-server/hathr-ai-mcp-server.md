# Hathr AI MCP Server

**Category:** AI Integration – MCP Servers  
**Brand:** Hathr AI

## Overview
Hathr AI MCP Server is a Model Context Protocol (MCP) server that exposes Hathr AI’s HIPAA-compliant AI tools and HIPAA-compliant Claude models for use within MCP-compatible chat clients and applications. It is designed for healthcare and other environments that require HIPAA-aligned handling of data.

## Features
- **MCP-compatible server**
  - Provides an MCP server endpoint that can be added to any MCP-capable client or application.
- **Static server URL**
  - Single endpoint for all clients: `https://mcp.pipedream.net/v2`.
  - Same URL works across different supported chat clients.
- **Authentication at connection time**
  - Authentication is handled when adding the server to a client or application (details via client-specific setup or configuration docs).
- **HIPAA-focused AI access**
  - Access to Hathr AI’s HIPAA-compliant AI tools tailored for healthcare use cases.
  - Access to HIPAA-compliant Claude models through MCP.
- **Client integration guidance**
  - Setup instructions available per supported chat client (via configuration documentation).

## Compliance
- Designed for use in healthcare contexts requiring HIPAA-compliant AI tooling and model access.

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Integration model:** Add as an external MCP server in compatible chat clients / apps.
