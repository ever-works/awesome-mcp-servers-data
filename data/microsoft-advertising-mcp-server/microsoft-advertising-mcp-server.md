# Microsoft Advertising MCP Server

## Overview
Microsoft Advertising MCP Server connects Microsoft Advertising accounts to AI-powered tools, enabling management and automation of search advertising campaigns via the MCP (Model Context Protocol) ecosystem.

## Features
- **MCP integration**: Exposes Microsoft Advertising functions through an MCP server.
- **AI tool connectivity**: Connects Microsoft Advertising to AI tools such as Cursor, Claude, and Windsurf.
- **Campaign management support**: Designed to enable management and automation of search ad campaigns through AI workflows.
- **Pipedream-based implementation**: Runs as a Pipedream MCP server for easier setup and integration.

## Use Cases
- Automating routine campaign management tasks via AI agents.
- Allowing code editors or AI IDEs (e.g., Cursor) to interact with Microsoft Advertising accounts.
- Letting conversational AI (e.g., Claude) analyze and adjust search ad campaigns through MCP.
