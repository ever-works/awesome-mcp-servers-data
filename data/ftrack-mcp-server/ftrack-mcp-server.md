# FTrack MCP Server

**Category:** Project Management MCP Servers  
**Brand:** ftrack

An MCP server that connects ftrack’s team collaboration and media review platform to MCP, enabling AI agents to interact with projects, assets, and review workflows.

---

## MCP Server URL

Use this static MCP server URL (authentication occurs when you add it to your client):

```text
https://mcp.pipedream.net/v2
```

---

## Features

- Connects ftrack’s collaboration and media review platform to the MCP ecosystem.
- Allows AI agents to:
  - Access and interact with ftrack projects.
  - Work with assets managed in ftrack.
  - Integrate into review workflows for media and creative work.
- Single static MCP endpoint (`https://mcp.pipedream.net/v2`) usable across compatible MCP clients.
- Hosted and powered by Pipedream Connect, so no custom server hosting is required for basic use.

---

## Setup

- Add the MCP server URL to your MCP-compatible chat client or application.
- Authenticate within your client when prompted.
- Refer to your client’s configuration instructions or the general configuration documentation for MCP-compatible apps.
