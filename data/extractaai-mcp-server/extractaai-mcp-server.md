# Extracta.ai MCP Server

AI-driven data extraction MCP server for documents and images, provided via Pipedream Connect.

## Overview
- **Type:** MCP (Model Context Protocol) server
- **Function:** AI-based data extraction from documents and images
- **Protocol:** Model Context Protocol (MCP)
- **Provider / Host:** Pipedream Connect
- **Static MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- AI-driven extraction of structured data from unstructured sources
- Supports documents and image inputs
- Accessible via a single static MCP server URL that works across compatible clients
- Authentication performed when adding the MCP server to an application
- Designed to integrate with MCP-compatible chat or agent clients

## Integration & Setup
