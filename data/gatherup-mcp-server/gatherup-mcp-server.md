# GatherUp MCP Server

**Category:** Business & Commerce · MCP Servers  
**Website:** https://mcp.pipedream.com/app/gatherup

## Overview
GatherUp MCP Server is an MCP integration for GatherUp’s customer experience and review management platform. It allows AI agents and MCP-compatible applications to access customer feedback, reviews, and CX data from GatherUp within MCP-based workflows.

## Features
- MCP server interface for the GatherUp platform.
- Enables AI agents to retrieve and work with customer experience (CX) data.
- Access to customer feedback and online reviews managed in GatherUp.
- Static MCP server URL that can be used by any supported client.
- Authentication handled when adding the server to an MCP-compatible application.
- Works within broader MCP workflows powered by Pipedream Connect.

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Integration Host:** Pipedream (Pipedream Connect)
