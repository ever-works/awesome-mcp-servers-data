# Loops.so MCP Server

## Overview
Loops.so MCP Server integrates the Loops.so email platform into MCP-compatible applications, enabling product, marketing, and transactional email workflows directly from your chat or agent environment.

- **Category:** Messaging MCP Servers
- **Use cases:** Product emails, marketing campaigns, transactional notifications, contact and list management within MCP workflows

## Configuration
- Static MCP server URL used for all clients: `https://mcp.pipedream.net/v2`
- Authentication occurs when adding the server to your MCP-compatible application.

## Features

### Email & Event Operations
- **Send Transactional Email**  
  Trigger transactional emails (e.g., receipts, alerts, confirmations) via MCP workflows.

- **Send Event**  
  Send events tied to an email address to drive behavior-based messaging and automations in Loops.so.

### Contact Management
- **Create Contact**  
  Create new contacts in Loops.so from your MCP workflows.

- **Update Contact**  
  Update an existing contact by email; if the email does not exist, a new contact is created.

- **Find Contact**  
  Look up a contact by email address to retrieve its details or use it in downstream actions.

- **Delete Contact**  
  Remove an existing contact from Loops.so.

### Lists & Custom Fields
- **List Mailing Lists**  
  Retrieve all mailing lists in your Loops.so account for use in automations or UI selection.

- **List Custom Fields**  
  Retrieve custom contact properties (custom fields) defined in your Loops.so account for mapping and personalization.

## Tools Summary (MCP Actions)
