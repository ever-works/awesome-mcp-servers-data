# Ringover MCP Server

## Overview
Ringover MCP Server is an MCP (Model Context Protocol) server integration that connects Ringover’s business communications platform to MCP-compatible chat clients and applications. It exposes Ringover’s telephony and messaging capabilities as programmable workflows, enabling automation around calls, messages, and related communication events.

- **Category:** Messaging MCP Servers  
- **Use cases:** Telephony integration, messaging automation, task and workflow automation  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible endpoint**  
  - Single static URL (`https://mcp.pipedream.net/v2`) used to connect from any supported MCP client.  
  - Authentication handled when adding the server to the client/application.

- **Ringover integration**  
  - Interfaces with Ringover’s business communications (telephony and messaging) via MCP.  
  - Designed to turn Ringover communication events into programmable workflows.

- **Client-agnostic setup**  
  - Works with any MCP-capable chat client or tool.  
  - Configuration performed once per client using the shared URL.

- **Workflow and task automation**  
  - Intended to integrate calls, messages, and other Ringover interactions into automated business processes (e.g., logging events, triggering tasks, orchestrating communication flows).

- **Configuration resources**  
  - Dedicated configuration documentation available via the "Configuration" page (linked from the app page) for setup and usage details.
