---

## Features

- **MCP Integration Endpoint**  
  - Provides a static MCP server URL: `https://mcp.pipedream.net/v2`.  
  - Same URL can be used across different MCP-compatible clients.

- **Noor Team Chat Connectivity**  
  - Connects the Noor team chat platform to MCP so chat workflows and automations can be driven via MCP tools.

- **Client-Agnostic Setup**  
  - The static MCP URL is designed to work with every supported MCP client.  
  - Users authenticate when adding the server within their chosen application/client.

- **Configurable via Documentation**  
  - A configuration guide ("Configuration" page) is available to walk through adding the server to various chat clients.

- **Pipedream Connect Infrastructure**  
  - Runs on Pipedream Connect, providing the MCP server hosting and connectivity layer.

---

## Usage

1. Use the MCP server URL `https://mcp.pipedream.net/v2` in your MCP-compatible client.  
2. Authenticate when prompted by your application/client.  
3. Follow Noor MCP Server configuration instructions for your specific chat client to complete the setup.
