# Docmosis MCP Server

**Category:** Document Management MCP Servers  
**Brand:** Docmosis  
**Slug:** `docmosis-mcp-server`

## Description
Docmosis MCP Server is an integration that connects your Docmosis account to MCP-compatible AI tools (such as Cursor, Claude, and Windsurf). It enables template-based generation of custom PDF and Word documents within MCP-driven document workflows.

## Features
- **MCP server integration**: Runs as an MCP server that can be accessed by MCP-compatible clients and AI tools.
- **Docmosis account connectivity**: Links directly to your existing Docmosis account for document generation.
- **Template-based document generation**: Uses Docmosis templates to produce documents populated with dynamic data.
- **PDF output**: Generates documents in PDF format.
- **Word document output**: Generates documents in Word (e.g., DOC/DOCX) format.
- **Workflow integration**: Designed to fit into MCP-driven document workflows, allowing automated document creation as part of broader processes.
- **AI tool compatibility**: Can be used from AI tools like Cursor, Claude, and Windsurf via the MCP protocol.
- **Document customization**: Supports creation of custom documents based on provided templates and data (e.g., reports, contracts, forms).

## Use Cases
- Generating PDF or Word reports from AI-driven workflows.
- Creating templated contracts, letters, or forms via MCP-compatible IDEs or assistants.
- Automating document outputs as part of multi-step MCP workflows.
