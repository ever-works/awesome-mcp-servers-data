# Cincopa MCP Server

**Category:** Media Processing MCP Servers  
**Brand:** Cincopa  
**Slug:** cincopa-mcp-server  
**Tags:** media, video

## Description
The Cincopa MCP Server is an MCP-compatible integration that connects MCP tools to Cincopa’s photo and video hosting platform. It provides access to Cincopa’s photo and video hosting, galleries, slideshows, video player designs, and private video hosting capabilities through a unified MCP server endpoint.

## Features
- Connects MCP tools to Cincopa’s photo and video hosting services
- Supports various slideshow designs
- Supports multiple photo gallery layouts
- Integrates with Cincopa video players
- Enables access to private video hosting solutions
- Uses a single static MCP server URL usable across clients

## MCP Server Endpoint
- **Server URL:** `https://mcp.pipedream.net/v2`
- **Authentication:** Performed when adding the server to your MCP-compatible application

## Integration & Setup
- Add the static MCP server URL to any supported MCP client
- Configuration and client-specific setup details are available via the provider’s configuration documentation (referenced as the “Configuration” page in the source content)
