## Overview
KickoffLabs MCP Server is an MCP (Model Context Protocol) server that connects AI clients to KickoffLabs, enabling workflows for refer-a-friend style giveaways, sweepstakes, and product launches. It is hosted and powered by Pipedream Connect.

## Features
- **MCP-compatible endpoint**  
  - Single static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same URL works across supported MCP-enabled clients.

- **Integration with KickoffLabs campaigns**  
  - Designed to support refer-a-friend style giveaways.  
  - Supports sweepstakes-style promotions.  
  - Supports product launch campaigns that use referrals or similar mechanics.

- **Client-agnostic setup**  
  - Can be added to different MCP-compatible chat or AI clients.  
  - Authentication is performed when adding the server to the application (per-client auth flow).

- **Configuration resources**  
  - Dedicated configuration documentation available via the "Configuration" page on Pipedream.

- **Hosted by Pipedream Connect**  
  - Managed endpoint provided by Pipedream (no self-hosting details required from the user).

## Usage
- Copy the MCP server URL: `https://mcp.pipedream.net/v2`.  
- Add it as an MCP server in your preferred compatible chat client.  
- Authenticate when prompted in the client to connect to KickoffLabs-related workflows.
