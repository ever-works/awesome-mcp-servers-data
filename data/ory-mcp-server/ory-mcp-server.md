# Ory MCP Server

**Category:** Security & Attestation MCP Servers  
**Brand:** Ory  
**Website / Source:** https://mcp.pipedream.com/app/ory

## Overview
Ory MCP Server is an MCP (Model Context Protocol) server that connects AI agents to Ory’s identity and access management stack. It enables authentication, authorization, and user management operations through a standardized MCP interface, suitable for secure, global, GDPR-compliant applications.

## Features
- **MCP-compatible identity server**
  - Exposes Ory’s identity and access management capabilities via MCP.
  - Usable by various MCP-compatible chat or AI clients.

- **Authentication operations**
  - Allows AI agents / clients to perform authentication flows through Ory.

- **Authorization and access control**
  - Supports authorization checks and access control decisions using Ory’s stack.

- **User management**
  - Enables user-related operations (e.g., managing identities) via MCP.

- **Static MCP server URL**
  - Single endpoint for all clients:  
    `https://mcp.pipedream.net/v2`
  - Authentication occurs when adding the server to an application.

- **API-first design**
  - Built around Ory’s API-first identity and access management platform.

- **Compliance-oriented**
  - Intended for use in secure, global, GDPR-compliant applications.

- **Client-agnostic setup**
  - Can be added to different chat / AI clients that support MCP.
  - Additional setup details available via a configuration page (referenced by the app).

## Integration & Usage
- Add the MCP server URL `https://mcp.pipedream.net/v2` to an MCP-compatible client or application.
- Authenticate as part of the server-add / configuration process.
- Use within chat or agent environments to perform identity, authentication, authorization, and user management operations.
