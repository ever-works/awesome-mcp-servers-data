# Textlocal MCP Server

Bulk SMS marketing and business messaging MCP server integration.

## Overview
The Textlocal MCP Server is an MCP-compatible service endpoint that connects MCP clients to Textlocal’s bulk SMS marketing and business messaging platform. It enables programmatic sending and management of SMS campaigns from tools that support the Model Context Protocol.

- **Type:** MCP server integration
- **Category:** Messaging / SMS
- **Provider / Brand:** Textlocal (via Pipedream Connect)
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible endpoint** – Static MCP server URL that can be added to any MCP-capable client.
- **Textlocal integration** – Connects to Textlocal’s bulk SMS marketing and business messaging services.
- **Programmatic SMS access** – Designed to allow sending and managing SMS messages and campaigns through MCP clients.
- **Client-agnostic setup** – The same server URL works across different MCP-enabled chat or automation clients.
- **Authentication at client level** – Authentication is performed when adding/configuring the server in your application (credentials handled per client).

## Setup
- Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
- Add this server URL within your MCP-compatible client’s configuration.
- Authenticate to Textlocal when prompted by your client during server setup.
