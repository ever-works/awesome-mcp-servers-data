# MoDeck MCP Server

An MCP server that connects MoDeck’s online editable video templates and After Effects–based workflows to MCP clients for automated or assisted video content generation.

- **Website / Source**: https://mcp.pipedream.com/app/modeck
- **Category**: Media Processing MCP Servers
- **Brand**: MoDeck
- **Slug**: `modeck-mcp-server`
- **Tags**: video, content-generation

## Features

- Connects MoDeck’s online editable video templates to MCP-compatible clients.
- Integrates with After Effects–based workflows for templated video creation.
- Supports automated or assisted content generation workflows via MCP.
- Uses a single static MCP server URL for all clients:
  - `https://mcp.pipedream.net/v2`
- Authentication occurs when adding the server to an MCP-compatible application.
- Designed for workflows where creatives define the video template in After Effects, while non-technical users can edit content online within MoDeck.

## Usage

- Copy and use the static MCP server URL in your MCP-compatible chat or automation client.
- Configure authentication and other settings within your client or application configuration.
