# Gong MCP Server

An MCP server that exposes Gong’s revenue intelligence platform capabilities—such as sales insights and call data—into MCP-based tools and workflows.

- **Category:** Business & Commerce MCP Servers  
- **Tags:** analytics, sales, AI insights  
- **Provider:** Gong (via Pipedream Connect)

## MCP Server Endpoint

- **Base URL:** `https://mcp.pipedream.net/v2`  
- **Usage:**
  - Static URL used by all clients.
  - Authentication is performed when adding the server to your MCP-compatible application or chat client.

## Features

- Integrates Gong’s revenue intelligence data (e.g., sales insights and call data) into MCP-based tools and workflows.
- Exposes Gong functionality through a standardized MCP server interface.
- Works with multiple MCP-compatible chat clients and applications using a single shared endpoint.
- Configuration guidance available via the Pipedream MCP configuration documentation.

## Integration & Setup

- Add the server to any supported MCP chat client or application using the static URL.
- Follow client-specific instructions or the general configuration guide on the Pipedream configuration page.
