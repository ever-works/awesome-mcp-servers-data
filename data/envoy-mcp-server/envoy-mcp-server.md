# Envoy MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Envoy  
**Website:** https://mcp.pipedream.com/app/envoy

Envoy MCP Server is a Model Context Protocol (MCP) server integration for Envoy, used to support managing hybrid workplaces and workplace experiences via compatible MCP-aware clients.

---

## Features

- **MCP Server Integration**
  - Provides an Envoy-specific MCP server endpoint for use with MCP-compatible applications and chat clients.
  - Enables integration of Envoy workplace and visitor management workflows into tools that support the MCP standard.

- **Static Server URL**
  - Single static MCP server URL for all clients:  
    `https://mcp.pipedream.net/v2`
  - Same endpoint works across different supported chat clients and applications.

- **Client-Agnostic Setup**
  - Designed to be added to various chat clients and MCP-capable applications.
  - Authentication is handled when adding the server to the client, rather than via different URLs.

- **Hybrid Workplace Use Cases**
  - Targets organizations using Envoy to manage hybrid workplaces and workplace experiences.
  - Oriented toward workplace management, visitor management, and operations workflows.
