# Mailerlite MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** MailerLite  
**Slug:** mailerlite-mcp-server

## Description
MailerLite MCP Server is an integration that connects MailerLite’s email marketing platform with MCP-compatible clients using a static MCP endpoint hosted by Pipedream. It enables applications to interact with MailerLite for managing email marketing workflows such as subscribers and campaigns through MCP.

## Features
- **MCP-compatible endpoint**
  - Single static MCP server URL for all clients: `https://mcp.pipedream.net/v2`
  - Can be added to any MCP-capable chat or agent client.

- **MailerLite integration (conceptual / from item definition)**
  - Connects to MailerLite’s email marketing platform via MCP.
  - Designed to support automated management of:
    - Email subscribers
    - Marketing campaigns
  - Suitable for building workflows around email marketing and marketing automation within MCP-aware tools.

- **Hosted by Pipedream**
  - Server infrastructure and endpoint provided by Pipedream Connect.
  - Centralized configuration documentation available via Pipedream’s configuration page.

## Configuration
- Add the MCP server URL to your MCP client: `https://mcp.pipedream.net/v2`.
- Authentication is performed when adding / configuring the server in your application (details depend on the client and MailerLite account setup).
- Additional configuration guidance is available on Pipedream’s Configuration page (not specific to this single server).
