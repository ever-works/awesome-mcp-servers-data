## Overview
mcp-linear is an MCP (Model Context Protocol) server that integrates with the Linear project management platform. It enables AI assistants and other MCP clients to interact with Linear via natural language, including retrieving and managing issues, projects, and teams.

## Features
- **Linear integration via MCP**  
  - Connects the Linear project management system to any MCP-compatible client or AI assistant.

- **Issue management**  
  - Retrieve Linear issues.  
  - Create new issues.  
  - Update existing issues.

- **Project and workflow management**  
  - Retrieve and manage Linear projects.  
  - Interact with Linear workflows through MCP (e.g., status changes, organizational structure; specific operations depend on configured tools in the server).

- **Team-level operations**  
  - Retrieve and manage information about teams in Linear.

- **Natural-language interaction**  
  - Designed so AI assistants can perform Linear operations in response to natural-language user requests.

- **Configuration and development assets**  
  - Example environment configuration file (`.env.example`) for setting required environment variables.  
  - Development guidelines in `DEVELOPMENT.md`.  
  - VS Code and linting/formatting configuration (`.vscode`, `.eslintrc.json`, `.prettierrc`, `.editorconfig`).  
  - Dockerfile for containerized deployment.
