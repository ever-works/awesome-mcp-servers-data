# StealthGPT MCP Server

## Overview
StealthGPT MCP Server is an MCP (Model Context Protocol) server that connects your StealthGPT account to compatible AI tools. It enables undetectable AI text generation and AI detection capabilities to be accessed from MCP-enabled environments.

- **Category:** AI Integration – MCP Servers  
- **Brand:** StealthGPT  
- **Source:** https://mcp.pipedream.com/app/stealthgpt

## Features
- **MCP Protocol Integration**: Exposes StealthGPT’s functionality via the MCP protocol for use in tools that support MCP.
- **Undetectable AI Text Generation**: Provides access to StealthGPT’s text generation designed to appear human-written.
- **AI Detection Tools**: Offers AI detection capabilities to analyze text and determine whether it is likely AI-generated.
- **Pipedream MCP Server**: Runs as a Pipedream-hosted MCP server, simplifying setup and integration.
- **Multi-Tool Connectivity**: Allows connecting a StealthGPT account to various AI tools, including:
  - Cursor
  - Claude
  - Windsurf
- **Account-Based Access**: Uses your existing StealthGPT account as the underlying provider for generation and detection.

## Use Cases
- Integrate StealthGPT text generation directly into MCP-compatible editors and IDEs.
- Run AI detection checks on content from within tools like Cursor or Windsurf.
- Build workflows that combine StealthGPT generation and detection in a single MCP-enabled environment.
