# Google Dialogflow MCP Server

**Category:** AI Integration – MCP Servers  
**Brand:** Google Cloud  
**Slug:** google-dialogflow-mcp-server

## Description
The Google Dialogflow MCP Server is a Model Context Protocol (MCP) server that connects MCP-compatible clients to Google Dialogflow. It lets you create and manage conversational experiences and Dialogflow agents across devices and platforms via a unified MCP endpoint.

## Features
- **MCP-compatible server endpoint**  
  - Provides a static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Single URL works for all supported MCP clients.

- **Dialogflow integration**  
  - Connects MCP clients to Google Dialogflow agents.  
  - Supports creating and managing conversational experiences that can run across multiple devices and platforms.

- **Account-based configuration**  
  - Requires connecting a Google Dialogflow account within the Pipedream interface.  
  - Once connected, you can select the desired client and configure it against the same MCP server URL.

- **Client-agnostic setup**  
  - Designed to work with multiple MCP-compatible “chat clients.”  
  - Each client follows its own setup instructions while using the same MCP server URL.

- **Hosted by Pipedream**  
  - Server infrastructure and MCP hosting handled by Pipedream, avoiding the need to deploy and maintain your own MCP server.
