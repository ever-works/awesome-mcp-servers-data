## Overview
Enormail MCP Server exposes Enormail’s email marketing capabilities over the Model Context Protocol (MCP), enabling AI tools and MCP-compatible clients to create and automate email campaigns using your Enormail account. It is provided as a Pipedream MCP server, allowing integration with AI tools like Cursor, Claude, and Windsurf.

## Features
- **MCP-based access to Enormail:** Use Enormail’s email marketing functions programmatically through the MCP interface.
- **Email campaign creation:** Enable MCP clients and AI tools to create email campaigns in your Enormail account.
- **Campaign automation:** Automate email marketing workflows and sequences via MCP-connected tools.
- **AI tool integrations:** Connect Enormail to AI-powered environments such as Cursor, Claude, and Windsurf using Pipedream’s MCP server.
- **Business growth focus:** Designed to support marketing activities aimed at growing business and revenue through email campaigns.

## Integrations
