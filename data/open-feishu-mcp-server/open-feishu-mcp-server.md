A Model Context Protocol (MCP) server for Feishu that provides secure, OAuth-based access to Feishu resources and supports remote connections for integration with MCP-compatible clients.

---

## Features

- **Model Context Protocol (MCP) server for Feishu**
  - Implements MCP to expose Feishu capabilities to MCP-compatible clients.
  - Designed for interaction with Feishu messaging and collaboration resources.

- **Built-in OAuth authentication**
  - Uses OAuth to authenticate against Feishu.
  - Enables secure, token-based access to Feishu APIs and data.

- **Secure interaction with Feishu resources**
  - Accesses Feishu resources (e.g., messaging/collaboration features) through authenticated calls.
  - Focus on controlled, permission-based access to Feishu data.

- **Remote connection support**
  - Can be connected to remotely by MCP clients.
  - Suitable for distributed setups where the MCP server runs separately from the client.

- **Open-source implementation**
  - Source available on GitHub.
  - Licensed under the MIT license for flexible use and modification.
