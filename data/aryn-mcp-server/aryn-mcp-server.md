# Aryn MCP Server

An MCP (Model Context Protocol) server for Aryn that enables applications to access and work with unstructured data.

## Overview
- **Name:** Aryn MCP Server  
- **Category:** Content Extraction & Summarization MCP Servers  
- **Provider/Brand:** Aryn (hosted via Pipedream Connect)  
- **Purpose:** Connect applications to Aryn’s capabilities for working with unstructured data over the Model Context Protocol.

## MCP Server Endpoint
- **Base MCP Server URL:** `https://mcp.pipedream.net/v2`  
- This is a static URL intended to be used by any compatible MCP client.  
- Authentication is performed when adding/configuring the server in the client or application.

- **Model Context Protocol Support**
