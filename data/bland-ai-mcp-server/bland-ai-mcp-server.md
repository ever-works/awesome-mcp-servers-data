# Bland AI MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Pipedream  
**Source:** https://mcp.pipedream.com/app/bland_ai

## Overview
The Bland AI MCP Server integrates Bland AI’s programmable voice agents into any MCP‑compatible client, enabling applications to send and receive high volumes of human‑like phone calls through a single static MCP endpoint.

## Features
- **Programmable voice agents**: Use Bland AI’s voice agents that sound and feel human for automated phone interactions.
- **High‑volume calling**: Designed to send or receive very large volumes of phone calls (millions per day).
- **MCP integration**: Exposes functionality via the Model Context Protocol so any compatible client can use it as a server tool.
- **Static server URL**: Single, client‑agnostic endpoint for all integrations:  
  `https://mcp.pipedream.net/v2`
- **Per‑client authentication**: Authentication is performed when adding the server to each application or chat client.
- **Multi‑client support**: Can be added to various MCP‑enabled chat clients; setup instructions are provided per client.
- **Central configuration documentation**: A dedicated configuration page ("Configuration" on Pipedream) provides detailed setup guidance for different environments.
