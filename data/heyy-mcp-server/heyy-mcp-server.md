# Heyy MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Pipedream  
**Slug:** heyy-mcp-server

## Description
Heyy MCP Server is a Model Context Protocol (MCP) server hosted on Pipedream that connects to Heyy to automate customer communication across multiple channels, including WhatsApp, SMS, Facebook Messenger, and Instagram. It provides a single, static MCP endpoint that you can add to compatible chat or agent clients to enable omnichannel messaging workflows.

## Features
- **Omnichannel messaging support**  
  - Connect with customers on WhatsApp  
  - Send and receive SMS messages  
  - Integrate with Facebook Messenger  
  - Communicate via Instagram messaging

- **MCP server endpoint**  
  - Static MCP server URL for all clients:  
    - `https://mcp.pipedream.net/v2`  
  - Works with any compatible MCP client or chat application

- **Authentication at client setup**  
  - Authentication is handled when adding the server to your application (per-client auth rather than per-URL)

- **Client integration guidance**  
  - Instructions available for adding the server to different chat clients  
  - Additional details via a configuration page for more advanced setup

- **Pipedream Connect integration**  
  - Runs on Pipedream Connect infrastructure  
  - Can be used as part of broader automation workflows built on Pipedream

## Setup
- Use the static MCP server URL: `https://mcp.pipedream.net/v2`  
- Add this URL as an MCP server in your compatible chat or agent client.  
- Complete authentication when prompted by your client.  
- Refer to the configuration page on Pipedream for client-specific setup details.
