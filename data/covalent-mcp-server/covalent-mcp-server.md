# Covalent MCP Server

A unified API for accessing extensive blockchain data through MCP-compatible tools.

## Overview
- **Type:** MCP Server integration
- **Category:** Blockchain & Crypto MCP Servers
- **Provider:** Covalent (via Pipedream Connect)
- **Use Case:** Unified access to billions of blockchain data points (e.g., for DeFi and general blockchain data analysis) through MCP-enabled clients.

## MCP Server URL
- **Base URL (static for all clients):**
  ```
  https://mcp.pipedream.net/v2
  ```
- Authentication is performed when adding the server to your application.

## Features
- Unified API for blockchain data access.
- Visibility into billions of on-chain data points.
- Compatible with MCP clients (e.g., chat-based or tool-based MCP integrations).
- Single static MCP endpoint for all supported clients.
- Integration surfaced and managed via Pipedream Connect.

## Configuration & Setup
- Add the MCP server to your application using the static URL: `https://mcp.pipedream.net/v2`.
- Client-specific setup instructions available via the **Configuration** page on Pipedream.
