## Description
A curated, opinionated GitHub list of high‑quality **remote** Model Context Protocol (MCP) servers. It focuses on official, well‑maintained MCP servers that are accessible over the internet, helping developers find reliable, production‑ready MCP services (including OAuth 2.0–enabled servers) for integration with LLM clients.

## Features
- Curated directory of **remote MCP servers** (internet‑accessible servers rather than local ones)
- Emphasis on **official** and **well‑maintained** MCP server implementations
- Focus on **production‑ready** MCP services suitable for real AI applications
- Includes **categories** of MCP servers for easier discovery and organization
- Includes **authentication details**, with attention to **OAuth 2.0–enabled** remote MCP servers
- Provides **usage examples** for integrating listed servers with LLM clients
- Oriented toward use with major LLM providers’ **Connectors** / remote MCP integration features (e.g., OpenAI, Anthropic terminology)
- Open repository under the **MIT License**
- Community‑driven: supports contributions via standard GitHub workflows (CONTRIBUTING and CODE_OF_CONDUCT files present)

## Technical/Domain Context
- Based on the [Model Context Protocol](https://modelcontextprotocol.io/), which defines how AI models interact with external tools and resources
- Targets **remote MCP servers**, enabling secure network-based access from clients
