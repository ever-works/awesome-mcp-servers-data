# DocRaptor MCP Server

## Overview
DocRaptor MCP Server exposes DocRaptor’s HTML-to-PDF conversion API as an MCP (Model Context Protocol) server, enabling PDF generation from HTML content within MCP-compatible tools and chat clients. It is provided via Pipedream Connect.

## MCP Server URL
- Base MCP server URL (static for all clients):
  - `https://mcp.pipedream.net/v2`
- Authentication is performed when adding the server to your application.

## Features
- **HTML-to-PDF conversion via MCP**
  - Wraps DocRaptor’s HTML-to-PDF API as MCP actions.
  - Allows generating PDFs from HTML content in MCP-compatible environments.
- **Static MCP endpoint**
  - Single, static server URL that works across all supported clients.
- **Client-agnostic integration**
  - Designed to be added to various chat clients and MCP-capable applications.
- **Configuration guidance**
  - Central configuration documentation available through the Pipedream Configuration page.
- **Hosted by Pipedream Connect**
  - Runs on Pipedream’s infrastructure, with terms and privacy policy managed by Pipedream.

## Integration & Setup
- Add the MCP server URL `https://mcp.pipedream.net/v2` to an MCP-compatible client or application.
- Follow client-specific instructions (via Pipedream’s configuration documentation) to complete setup and authentication.
