# Oyster MCP Server

Automated global employment platform access via MCP.

## Overview
The Oyster MCP Server exposes Oyster’s automated global employment platform through the Model Context Protocol (MCP), allowing compatible chat or AI clients to connect to Oyster’s HR, global employment, and payroll capabilities via a standardized server endpoint.

## MCP Server Endpoint
- Base MCP server URL (static for all clients):
  - `https://mcp.pipedream.net/v2`
- Authentication is performed when adding the server to your application or client.

## Features
- **MCP-based integration**
  - Provides an MCP-compliant server endpoint for connecting Oyster to AI/chat applications.
  - Works with any MCP-compatible client using the same static URL.
- **Automated global employment platform access**
  - Connects to Oyster’s automated global employment capabilities (HR, global employment, payroll) through MCP.
- **Centralized configuration**
  - Can be added to supported chat clients via their MCP/server configuration workflows.
  - Additional setup details available through a configuration page (not client-specific).

## Integration & Usage
- Use the static server URL when adding a new MCP server in your chat or AI application.
- Complete authentication during the server connection process in your client.
- Refer to your client’s MCP/server configuration instructions for detailed setup.
