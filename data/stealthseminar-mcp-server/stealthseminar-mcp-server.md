# StealthSeminar MCP Server

## Overview
StealthSeminar MCP Server is a Model Context Protocol (MCP) server that lets compatible AI/chat clients interact with the StealthSeminar webinar platform. It focuses on automating the management of pre-recorded webinars that run as live events.

## MCP Server URL
- Base MCP server URL (static for all clients):
  - `https://mcp.pipedream.net/v2`
- Authentication is performed when adding the server to a compatible MCP-enabled application.

## Features
- **MCP integration**
  - Exposes StealthSeminar functionality via the Model Context Protocol.
  - Usable from MCP-compatible chat or AI clients.

- **Static server endpoint**
  - Single, static MCP server URL for all clients (`https://mcp.pipedream.net/v2`).
  - Simplifies configuration across different tools and environments.

- **Webinar automation focus**
  - Designed to interact with StealthSeminar’s automated webinar platform.
  - Supports workflows around pre-recorded webinars that are presented as live events (e.g., scheduling, managing, or automating such events via MCP tools).

- **Client-agnostic usage**
  - Can be added to various chat clients that support MCP.
  - Configuration guidance is provided via a generic “add the server to your app” flow and a dedicated configuration page (on Pipedream Connect).

- **Pipedream Connect integration**
  - MCP server is provided and hosted via Pipedream Connect infrastructure.

## Configuration & Integration
- Add the MCP server URL to any MCP-compatible chat client or application.
- Authentication is handled at the time you connect the server in your client.
- Additional configuration details are available on the Pipedream-hosted configuration page (not included in the provided content).
