# Ecologi MCP Server

## Overview
The Ecologi MCP Server is an MCP (Model Context Protocol) integration provided via Pipedream that connects Ecologi’s climate action platform with AI tools and MCP-compatible clients.

## Key Details
- **Name:** Ecologi MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Brand:** Ecologi  
- **Source URL:** https://mcp.pipedream.com/app/ecologi  
- **Slug:** ecologi-mcp-server  
- **Tags:** climate, sustainability, nonprofit

## Description
An MCP server that allows AI tools and MCP clients to interact with Ecologi’s climate action platform and related APIs through Pipedream. It is designed to connect Ecologi accounts with AI development environments and assistants.

## Features
- Connects an Ecologi account to AI tools using Pipedream’s MCP server.  
- Supports integration with popular AI tools such as:
  - Cursor  
  - Claude  
  - Windsurf  
- Exposes Ecologi’s climate action platform and related APIs to MCP-compatible clients (e.g., AI IDEs, assistants).

## Integrations
- **Pipedream MCP Server** as the hosting/integration layer.  
- **AI Tools:** Cursor, Claude, Windsurf (via MCP).
