# Schedule MCP Server

**Brand:** Pipedream  
**Category:** Workflow Automation MCP Servers  
**Tags:** scheduling, task-automation, cron

## Description
Schedule MCP Server is a scheduling-focused Model Context Protocol (MCP) server that triggers AI-driven workflows on a defined interval or cron schedule via a static MCP endpoint.

## Features
- **Interval-based triggers** – Run workflows on a recurring time interval (e.g., every N minutes/hours/days).
- **Cron-based scheduling** – Configure cron expressions to trigger workflows at specific times and patterns.
- **Static MCP server URL** – Single endpoint usable across compatible MCP clients:
  - `https://mcp.pipedream.net/v2`
- **Client-agnostic usage** – Designed to work with any MCP-compatible chat client or application.
- **Authentication on client add** – You authenticate when adding the server to your app, not per-URL instance.
- **Central configuration docs** – Additional setup and configuration details available through the Pipedream Configuration page.

## Integration
- Works within the Model Context Protocol ecosystem as an MCP server.
- Intended to be added to MCP-enabled chat clients and AI applications for scheduled, automated workflow execution.
