# Pushsafer MCP Server

An MCP server that connects to the Pushsafer notification service so MCP clients can send and receive push notifications.

- **Category:** Messaging MCP Servers  
- **Brand:** Pushsafer  
- **Source:** https://mcp.pipedream.com/app/pushsafer  
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features

- Connects MCP-compatible clients to the Pushsafer notification service.
- Supports sending push notifications via Pushsafer.
- Supports receiving push notifications from Pushsafer.
- Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) that works across clients.
- Authentication handled when adding the server to the client/application.
- Usable from multiple chat or MCP-enabled clients (e.g., “select your chat client to get started”).
- Configuration details available via a dedicated configuration page on Pipedream.

## Integration & Usage

1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add this server URL to your MCP-compatible client or chat application.
3. Authenticate within your client when prompted.
4. Configure notification behavior following the Pipedream configuration documentation.
