# Mural MCP Server

**Description**  
An MCP server that connects Mural’s visual collaboration and whiteboarding platform to MCP-compatible clients, enabling collaborative diagramming and planning within your chat or agent environment.

**Category**  
Project Management MCP Servers

**Provider / Brand**  
Mural (via Pipedream Connect)

**Server URL**  
`https://mcp.pipedream.net/v2`  
This is a static MCP server URL used by all compatible clients. Authentication is handled when you add the server to your application.

## Features
- Exposes Mural’s visual collaboration capabilities to MCP clients
- Supports whiteboarding for team brainstorming and visual thinking
- Enables collaborative diagramming and planning workflows
- Integrates Mural’s visual work platform into chat / agent applications via MCP
- Uses a single static MCP endpoint (`https://mcp.pipedream.net/v2`) for all supported clients
- Compatible with multiple chat clients that support MCP (configuration handled per client)

## Integration & Configuration
- Add the server URL to your MCP-compatible chat or agent client
- Authenticate to Mural when registering the server in your application
- Additional configuration details available on the platform’s Configuration page (per client instructions)
