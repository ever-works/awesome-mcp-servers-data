# ShortPixel MCP Server

**Brand:** ShortPixel  
**Category:** Media Processing MCP Servers  
**Slug:** `shortpixel-mcp-server`

ShortPixel MCP Server is a Pipedream-based MCP integration that optimizes images for better website performance by resizing, compressing, and serving them via CDN, producing smaller files that remain visually similar to the originals.

---

## Features

- **Image resizing**  
  - Adjust image dimensions to match site or application requirements.

- **Image compression**  
  - Compress images to reduce file size while maintaining visual similarity to the original.

- **CDN delivery**  
  - Serve optimized images from a Content Delivery Network to improve load times and performance.

- **Website performance optimization**  
  - Reduce page weight by using smaller, optimized images.

- **Static MCP server endpoint**  
  - Uses a single, static MCP server URL for all clients:  
    - `https://mcp.pipedream.net/v2`

- **Client-agnostic integration**  
  - The same MCP server URL works across different chat or MCP-compatible clients.

- **Authentication at client setup**  
  - Authentication is performed when adding the server to an application, rather than per-URL.

- **Configuration documentation**  
  - Additional configuration details are available via a dedicated configuration page (through Pipedream Connect).

---

## Integration & Usage

- Add the MCP server URL (`https://mcp.pipedream.net/v2`) to any compatible chat client or MCP-enabled application.
- Configure authentication and workflows through Pipedream / Pipedream Connect.
