# Ortto MCP Server

**Description:**  
Ortto MCP Server exposes Ortto’s unified customer journey and marketing automation capabilities through the Model Context Protocol (MCP), enabling AI-driven workflows to interact with customer journeys, campaigns, and related automation features.

**Category:**  
Business & Commerce · MCP Servers

**Tags:**  
- Marketing automation  
- Customer journey  
- Campaigns

---

## MCP Server URL

Use this static URL for all MCP-compatible clients:

```text
https://mcp.pipedream.net/v2
```

Authentication is performed when you add the server to your MCP-enabled application.

---

## Features

- **MCP-compatible endpoint** – Single static base URL (`https://mcp.pipedream.net/v2`) usable across different MCP clients.
- **Ortto integration** – Connects to Ortto, a unified customer journey platform for online businesses.
- **Customer journey exposure** – Surfaces customer journey data and actions into MCP so AI agents and tools can use them in workflows.
- **Marketing automation integration** – Provides access to Ortto’s marketing automation features via MCP for programmatic and AI-driven automation.
- **Campaign-related operations** – Designed to work with marketing campaigns (creation, orchestration, or analysis, depending on client implementation).
- **AI workflow enablement** – Lets AI assistants and MCP-aware tools trigger and manage Ortto workflows as part of broader automations.
- **Client-agnostic server** – Same server URL works for any MCP client; configuration is handled client-side.

> Note: The public page does not list individual tools/endpoints; actual available tools and methods depend on the MCP client’s configuration for this server.

---

## Integration & Setup

- Add the MCP server endpoint `https://mcp.pipedream.net/v2` in your MCP-compatible chat client or application.
- Authenticate to Ortto when prompted by the client during server configuration.
- For detailed client-specific steps, refer to the client’s MCP configuration instructions (e.g., the platform’s "Configuration" page).
