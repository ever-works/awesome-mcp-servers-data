# ChatBot MCP Server

**Category:** Messaging MCP Servers  
**Brand:** ChatBot  
**Source:** https://mcp.pipedream.com/app/chatbot

## Description
ChatBot MCP Server connects the ChatBot platform—used to create, deploy, and track chatbots across multiple channels—to MCP-compatible clients via a static server endpoint.

## Features
- **All-in-one chatbot integration**: Connects to ChatBot, a platform for creating, deploying, and tracking chatbots.
- **Omnichannel support**: Designed for use with chatbots that operate across multiple channels.
- **Static MCP server URL**: Uses a single, static server URL for all clients:
  - `https://mcp.pipedream.net/v2`
- **Client-agnostic**: The same MCP endpoint works with any compatible client; authentication occurs when adding the server to the application.
- **Configurable setup**: Supports configuration through a dedicated configuration page (linked from the app) for more detailed setup instructions.
- **Pipedream-backed infrastructure**: Hosted and operated via Pipedream Connect.

## Authentication
- Authentication is performed when you add the MCP server to your application, using the static URL `https://mcp.pipedream.net/v2`.

## Usage
1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add the server in your MCP-compatible client.
3. Authenticate when prompted during server addition.
4. Optionally, refer to the Configuration page linked from the app for client-specific setup.
