# Current RMS MCP Server

## Overview
The Current RMS MCP Server exposes Current RMS rental management capabilities through the Model Context Protocol (MCP), enabling applications and chat clients to interact with cloud-based rental, inventory, and booking operations used in AV, production, events, and broadcast businesses.

## Features
- **MCP-compatible endpoint**
  - Provides a static MCP server URL that works for all clients.
  - Can be added to any MCP-capable application or chat client.

- **Current RMS rental management integration**
  - Interfaces with Current RMS cloud rental management software.
  - Targets use cases in AV, production, events, and broadcast rental industries.

- **Access to operational data and actions**
  - Designed to expose rental operations.
  - Designed to expose inventory-related operations.
  - Designed to expose booking-related operations.

- **Authentication at client setup**
  - Authentication occurs when adding the server to your application or chat client.

- **Configuration resources**
  - Documentation and setup details available via a central configuration page (for supported chat clients and apps).
