# Agendor MCP Server

**Description**  
Agendor MCP Server connects Agendor’s sales improvement and CRM platform (focused on long-cycle B2B sales in Brazil) to MCP-compatible tools and workflows via Pipedream Connect.

**Category**  
Business & Commerce → MCP Servers

**Key Details**
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Provider / Host:** Pipedream Connect
- **Brand:** Agendor

## Features
- Integrates Agendor’s CRM and sales improvement platform with external tools through the MCP protocol.
- Supports use in multiple chat or MCP-enabled clients (one static server URL for all clients).
- Centralized server endpoint (`https://mcp.pipedream.net/v2`) for configuration and authentication.
- Designed for workflows involving Brazilian companies with long sales cycles.
- Web and mobile-compatible underlying Agendor platform.

## Integration & Configuration
- Use the static MCP server URL in your MCP-compatible chat or workflow client.
- Authentication is performed when adding the server in the client.
- Additional configuration details are available on Pipedream’s Configuration page (not client-specific here).
