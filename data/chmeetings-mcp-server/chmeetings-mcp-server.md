## Overview
ChMeetings MCP Server is an integration layer that connects a ChMeetings church management account to MCP-based AI tools. It enables assistants and developer tools such as Cursor, Claude, and Windsurf to interact with ChMeetings data and functionality.

## Features
- Connects a ChMeetings account to Pipedream's MCP server.
- Enables MCP-based AI tools (e.g., Cursor, Claude, Windsurf) to access ChMeetings-related functionality.
- Exposes church management and CRM capabilities from ChMeetings to compatible AI assistants and tools.
