# EODHD APIs MCP Server

**Category:** Finance – Market Data MCP Servers

**Brand:** eodhd  
**Source URL:** https://mcp.pipedream.com/app/eodhd_apis

## Description
EODHD APIs MCP Server provides access to stock market fundamentals and historical price data for global securities, including stocks, ETFs, mutual funds, and bonds. It exposes EODHD’s market data through an MCP (Model Context Protocol) server endpoint that can be integrated into compatible chat or AI clients.

## Features
- MCP server interface for EODHD market data
- Access to stock market fundamental data
- Access to historical price data
- Coverage of global markets
- Support for multiple instrument types:
  - Stocks
  - Exchange-traded funds (ETFs)
  - Mutual funds
  - Bonds
- Static MCP server URL usable across compatible clients
- Authentication handled when adding the server to an application

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Integration:** Add the MCP server to any supported MCP client / chat application using the static URL
