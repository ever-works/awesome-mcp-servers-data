## Overview
Tutor LMS MCP Server is a Model Context Protocol (MCP) server that integrates with Tutor LMS on WordPress. It enables AI-capable, MCP-compatible clients (such as chat-based applications) to interact with a Tutor LMS-powered eLearning site to help create, manage, and sell online courses.

## Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Usable by any MCP-compatible client.

- **Tutor LMS integration**  
  - Connects to Tutor LMS running on WordPress.  
  - Designed for AI-assisted course operations (creation, management, and selling of eLearning content).

- **AI-driven course workflows**  
  - Supports AI-driven creation of eLearning courses.  
  - Supports AI-assisted management of existing courses.  
  - Supports AI-assisted selling and related course commerce workflows.

- **Client-agnostic setup**  
  - Single static URL works for every supported MCP client.  
  - Authentication handled when adding the server within the client/application.

- **Configuration resources**  
  - Documentation and setup instructions available via the Pipedream Connect configuration pages.
