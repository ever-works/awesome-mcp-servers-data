# DocuMerge MCP Server

Automated document generation MCP server for use within MCP-compatible chat clients and workflows.

## Overview
- **Name:** DocuMerge MCP Server  
- **Category:** Document Management MCP Servers  
- **Provider / Brand:** DocuMerge (via Pipedream Connect)  
- **Description:** Provides automated, templated document generation and merging capabilities within MCP flows.

## Features
- Automated document generation within MCP-compatible applications.  
- Supports templated document creation (template-based document output).  
- Document merging capabilities as part of MCP workflows.  
- Single static MCP server URL usable across different clients: `https://mcp.pipedream.net/v2`.  
- Authentication handled when adding the server to a client/application.  
- Works with multiple chat clients that support MCP.  
- Central configuration and setup guidance available via a configuration page on Pipedream.

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Integration Model:** Added as an MCP server to chat clients / applications that support MCP.
