# Zonka Feedback MCP Server

## Overview
Zonka Feedback MCP Server connects AI tools and chat clients to Zonka Feedback, a multi-channel feedback and survey platform. It exposes Zonka’s capabilities through the MCP (Model Context Protocol) via a single static endpoint.

- **Type:** MCP server / integration
- **Category:** Business & Commerce MCP Servers
- **Provider / Brand:** Zonka Feedback (via Pipedream)
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- **MCP Integration Endpoint**  
  - Single static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Works with any compatible MCP client; authentication is performed when adding the server in the client.

- **Multi‑Channel Feedback & Surveys**  
  - Access Zonka Feedback’s multi-channel feedback collection and survey tools from AI clients.

- **Real-Time Reporting**  
  - Use Zonka’s real-time reporting capabilities through the MCP-connected tools (e.g., to surface up-to-date feedback data).

- **Collaborative Inbox**  
  - Integrates Zonka’s collaborative inbox features, enabling teams to manage and respond to feedback within connected workflows.

- **Client-Agnostic Configuration**  
  - Can be added to different chat/AI clients that support MCP.  
  - Configuration guidance available via a generic “Configuration” page (no client-specific lock-in).

## Setup
- Use the static MCP URL: `https://mcp.pipedream.net/v2` when adding a new MCP server in your client.  
- Authenticate within your chosen client as prompted during server setup.
