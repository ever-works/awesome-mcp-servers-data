# SeaTable MCP Server

## Overview
The SeaTable MCP Server integrates SeaTable with Model Context Protocol (MCP)–compatible clients, enabling AI agents and chat applications to work with SeaTable’s spreadsheet-like structured data, tasks, projects, and assets.

## Key Details
- **Name:** SeaTable MCP Server  
- **Category:** Data access & integration MCP server  
- **Vendor / Brand:** SeaTable (via Pipedream)  
- **Primary Function:** Connect AI / chat clients to SeaTable bases and data structures through a standard MCP server endpoint.  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server endpoint**  
  - Provides a static MCP server URL that can be used across different compatible clients.  
  - Authentication occurs when adding the server to your application.

- **SeaTable integration**  
  - Connects to a SeaTable account to work with structured, spreadsheet-like data.  
  - Designed for organizing and accessing tasks, assets, projects, and ideas stored in SeaTable.  
  - Supports use in the cloud or with self-hosted SeaTable instances (as supported by your SeaTable configuration).

- **Multi-client support**  
  - Can be added to various MCP-capable chat clients.  
  - Client-specific setup instructions are provided via the configuration pages on Pipedream.

- **Configuration flow**  
  - Connect your SeaTable account through the Pipedream-based configuration.  
  - Select a client and follow guided steps to add the MCP server URL.  
  - Tools and actions are dynamically loaded from the server once connected (exact tools are not listed in the available content).

## Usage
1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.  
2. Add this server URL to your MCP-compatible chat or AI client.  
3. Authenticate when prompted and connect to your SeaTable account.  
4. Use the available SeaTable-related tools and actions exposed by the MCP server inside your client.
