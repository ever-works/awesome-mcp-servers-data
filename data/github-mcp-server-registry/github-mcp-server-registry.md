**Category:** MCP server directories & lists  
**Tags:** directory, mcp, tool-discovery  
**Brand:** GitHub

## Overview
The GitHub MCP Server Registry is a centralized GitHub organization that aggregates Model Context Protocol (MCP) server repositories. It serves as a primary entry point to discover, browse, and integrate MCP servers that connect AI models to tools, data sources, and automation workflows.

## Features
- **Centralized MCP Directory** – Hosts a curated list of MCP server repositories under the `github.com/mcp` organization.
- **Server Discovery** – Allows users to find MCP servers for a variety of tasks, including tools, data access, automation, and integrations.
- **Ecosystem Coverage** – Includes servers built and maintained by different organizations and projects across the ecosystem.
- **Search & Browsing** – Supports searching and browsing through available MCP servers directly on GitHub.
- **Integration Entry Point** – Acts as a starting point for integrating MCP servers into AI assistants, agents, and other MCP-compatible clients.
- **Community-Contributed Servers** – Surfaces servers contributed by the community, not limited to a single vendor.
