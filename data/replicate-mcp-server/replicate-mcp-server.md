# Replicate MCP Server

**Brand:** replicate  
**Category:** MCP server directories/lists

## Description
Replicate MCP Server is an open-source Model Context Protocol (MCP) server for Replicate’s API. It enables LLMs to search, run, and manage machine learning models hosted on Replicate, including creating and monitoring predictions and working with generated images.

Source: [GitHub – deepfates/mcp-replicate](https://github.com/deepfates/mcp-replicate)

## Features
- Implements a Model Context Protocol (MCP) server for Replicate’s API.
- Allows LLMs to browse and search available models on Replicate.
- Supports creating and running predictions using Replicate-hosted machine learning models.
- Enables tracking and querying prediction statuses.
- Handles generated image outputs from Replicate models.
- Facilitates basic management of model runs via MCP tools.
- Distributed as an npm package (`mcp-replicate`), with TypeScript/Node.js source.
