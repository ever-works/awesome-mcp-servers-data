# CloudCart MCP Server

## Overview
CloudCart MCP Server is an MCP server integration that exposes CloudCart e‑commerce and marketing capabilities through the MCP protocol, enabling compatible chat or agent clients to interact with CloudCart stores.

## Features
- MCP server endpoint for CloudCart: static URL `https://mcp.pipedream.net/v2`
- Works with multiple MCP-compatible chat/agent applications
- Authentication handled when adding the server to your MCP-compatible app
- Provides access (via MCP tools/actions) to CloudCart e‑commerce store and marketing functionality
- Central configuration via a CloudCart–Pipedream configuration page

> Note: The page mentions “Available tools” but does not list them explicitly; only that CloudCart store and marketing functionality are exposed through the MCP protocol.

## Integration & Setup
- Connect your CloudCart account through the configuration interface
- Use the static MCP server URL in your MCP-compatible client
- Authenticate within your application when registering the server
