# Discourse MCP Server

**Description**

The Discourse MCP Server exposes Discourse’s community discussion and forum platform APIs to MCP-compatible clients, enabling community management and automation directly from your chat or MCP client.

- **Category:** Messaging MCP Servers
- **Tags:** community, forum, collaboration
- **Provider:** Pipedream
- **Static MCP Server URL:** `https://mcp.pipedream.net/v2`

---

## Features

- **Discourse API exposure via MCP**
  - Provides access to Discourse’s community discussion and forum platform APIs through the MCP protocol.
  - Designed for use with MCP-compatible chat or automation clients.

- **Static server endpoint**
  - Single static MCP server URL (`https://mcp.pipedream.net/v2`) used across all supported clients.
  - Authentication is handled when adding the server to your application.

- **Client-agnostic configuration**
  - Works with multiple MCP-enabled chat clients.
  - Setup flow: connect your Discourse account in Pipedream, then select your client to complete configuration.

- **Tooling integration**
  - Exposes a set of “tools” (actions) that can be loaded and invoked from MCP clients (exact tools not listed in the provided content).

- **Centralized configuration page**
  - Additional configuration details are available via a dedicated configuration page on Pipedream.

---

## Usage

1. Connect your Discourse account in Pipedream.
2. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
3. Add the server to your MCP-compatible app or chat client.
4. Authenticate when prompted and load available tools/actions.
