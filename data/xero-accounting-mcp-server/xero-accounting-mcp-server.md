# Xero Accounting MCP Server

**Category:** Business & Commerce MCP Servers  
**Tags:** accounting, invoicing, finance  
**Provider:** Pipedream  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Description
The Xero Accounting MCP Server is an MCP-based integration that connects applications to Xero’s online accounting platform. It provides tools to work with Xero accounting data, including invoices, contacts, and other financial information, via the Model Context Protocol.

## Features
- MCP-based server for integrating Xero Accounting into compatible clients
- Access to Xero’s online accounting capabilities
  - Work with invoices
  - Manage contacts
  - Access financial data (e.g., accounting records and related entities exposed by Xero)
- Single static MCP server URL (`https://mcp.pipedream.net/v2`) usable by any client
- Authentication handled when adding the MCP server within your application

## Setup
- Use the static MCP server URL in your MCP-compatible chat client or application
- Authenticate to Xero when prompted by your client during server configuration
- Additional configuration details are available on the provider’s configuration page (via Pipedream Connect)
