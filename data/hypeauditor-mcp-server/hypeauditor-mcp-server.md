# HypeAuditor MCP Server

All-in-one MCP Server integration that exposes HypeAuditor’s influencer marketing analytics and management capabilities to MCP-aware tools and workflows.

## Overview
- **Type:** MCP Server integration
- **Category:** Business & Commerce · MCP Servers
- **Brand:** HypeAuditor
- **Slug:** `hypeauditor-mcp-server`
- **Source URL:** https://mcp.pipedream.com/app/hypeauditor
- **Provider / Infrastructure:** Powered by Pipedream Connect

## Features
- **MCP Server endpoint**
  - Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`
  - Works with any MCP-aware client that supports adding custom MCP servers.

- **Authentication at client configuration time**
  - Authentication is handled when you add the server to your application (client-side configuration).

- **Client-agnostic integration**
  - Designed to be used with multiple chat or MCP-capable clients.
  - Configuration instructions are available per client type via the “Select your chat client” flow.

- **Central configuration documentation**
  - Detailed setup instructions are provided on a dedicated configuration page (linked as “Configuration page”).

- **Influencer marketing focus (via HypeAuditor)**
  - Integrates HypeAuditor’s influencer marketing analytics and management capabilities into MCP workflows (e.g., for influencer discovery, analytics, and campaign management) — exact tools depend on HypeAuditor’s underlying platform.

## Integration & Setup
- Use the static MCP URL: `https://mcp.pipedream.net/v2` when adding the server to your MCP-aware app or chat client.
- Follow client-specific steps via the “Select your chat client” options or the full Configuration page.
