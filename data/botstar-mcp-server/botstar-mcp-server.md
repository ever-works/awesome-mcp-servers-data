# Botstar MCP Server

Bot application

## Overview
The Botstar MCP Server is an MCP server integration for Botstar, a bot application platform for building and managing chatbots. It is powered by Pipedream Connect and provides a static MCP endpoint that can be used from any compatible client.

## MCP Server URL
- Static MCP server URL (works for all clients):
  - `https://mcp.pipedream.net/v2`

## Features
- MCP server integration specifically for Botstar-based chatbot applications
- Single static endpoint usable by multiple MCP-compatible clients
- Authentication is handled when adding the server within the client/application
- Works with various chat clients that support MCP server connections
- Configuration details available via a dedicated configuration page on Pipedream

## Usage
1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add the server to your MCP-compatible chat client or application.
3. Authenticate as prompted by your client when adding the server.
4. Refer to the configuration documentation on Pipedream for client-specific setup.
