# Dux Soup MCP Server

## Overview
Dux Soup MCP Server is an MCP-compatible integration that connects to Dux-Soup, enabling LinkedIn prospecting and marketing automation features to be used from MCP-compatible tools and chat clients.

- **Category:** Business & Commerce – MCP Servers  
- **Use cases:** LinkedIn lead generation, outreach automation, and prospect engagement via MCP-compatible applications.

## Features
- **MCP server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same URL works for all clients; authentication is handled when adding the server to your application.

- **Dux-Soup integration**  
  - Connects your Dux-Soup account to MCP-compatible tools.  
  - Designed to expose LinkedIn prospecting and marketing automation capabilities through MCP.

- **Client configuration**  
  - “Configure Dux Soup” flow to connect your account.  
  - Select your chat client to view client-specific connection instructions.  
  - Links to a full configuration page for detailed setup steps.

- **Tool discovery**  
  - Supports loading and listing “available tools” / “actions” once the server is connected (exact actions are not listed in the provided content).
