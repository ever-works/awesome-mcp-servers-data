# Files.com MCP Server

## Overview
The Files.com MCP Server exposes Files.com’s secure SaaS FTP and file transfer capabilities—transfer, sharing, and storage of files—to AI agents and MCP-based automations via a standard MCP endpoint.

- **Name:** Files.com MCP Server  
- **Category:** File Management MCP Servers  
- **Brand:** Files.com  
- **Website / Source:** https://mcp.pipedream.com/app/files_com

## MCP Server Endpoint
- **MCP Server URL (static for all clients):** `https://mcp.pipedream.net/v2`  
  You authenticate when adding the server to your MCP-compatible application or chat client.

## Features
- **SaaS FTP Platform Integration**  
  - Connects to Files.com, a SaaS FTP file transfer platform.  
  - Uses Files.com’s secure infrastructure for file operations.

- **Secure File Transfer**  
  - Transfer files through Files.com using the MCP server.  
  - Designed to integrate file transfer actions into AI agents and automations.

- **File Sharing**  
  - Exposes sharing capabilities of Files.com (e.g., sharing files with others) via MCP.

- **File Storage**  
  - Store and manage files on Files.com using MCP-based tools.

- **AI & Automation Integration**  
  - Accessible from MCP-compatible chat clients and applications.  
  - Intended for use by AI agents and MCP-based workflows to perform file operations programmatically.

- **Unified Endpoint**  
  - Single, static MCP URL for all clients; configuration handled through the client’s MCP settings.

## Usage Notes
- Add `https://mcp.pipedream.net/v2` as an MCP server in your MCP-enabled chat client or application.  
- Authentication is performed when you configure the server in your app.
