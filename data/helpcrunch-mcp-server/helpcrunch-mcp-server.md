# HelpCrunch MCP Server

**Category:** Business & Commerce · MCP Servers  
**Brand:** HelpCrunch  
**Slug:** helpcrunch-mcp-server

## Overview
HelpCrunch MCP Server is an MCP server integration that connects HelpCrunch’s all-in-one customer communication, live chat, and CRM capabilities to MCP-compatible tools via Pipedream. It exposes HelpCrunch functionality through a standardized MCP endpoint so you can use it from supported chat clients and applications.

## Features
- **MCP server integration for HelpCrunch**: Provides an MCP-accessible interface to HelpCrunch’s platform.
- **Access to customer communication tools**: Integrates HelpCrunch’s messaging and communication features through MCP.
- **Live chat integration**: Enables MCP tools and chat clients to interact with HelpCrunch live chat capabilities.
- **CRM connectivity**: Allows MCP-compatible applications to leverage HelpCrunch CRM data and workflows.
- **Single static MCP endpoint**: Uses a shared MCP server URL for all clients:
  - Endpoint: `https://mcp.pipedream.net/v2`
- **Client-agnostic configuration**: The same MCP URL works across different chat clients; authentication is handled when adding the server to each application.
- **Configuration via Pipedream**: Setup and management are handled through Pipedream Connect and its configuration interface.

## Technical Details
- **MCP endpoint**: `https://mcp.pipedream.net/v2`
- **Authentication model**: Authentication occurs when you add the MCP server to your specific chat client or application.
- **Provider**: Delivered and hosted via Pipedream Connect.
