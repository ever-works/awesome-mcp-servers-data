# Synthflow MCP Server

Automate phone calls with AI voice agents via an MCP-compatible server.

## Overview
- **Type:** MCP server (Messaging / AI integration)
- **Purpose:** Enable MCP agents and clients to automate phone calls and interact programmatically with AI voice agents.
- **Provider:** Synthflow (hosted via Pipedream Connect)

## Features
- **MCP-compatible endpoint**
  - Provides a static MCP server URL that can be used by any compatible client.
  - Central URL for all clients: `https://mcp.pipedream.net/v2`.

- **AI voice agent integration**
  - Designed for programmatic interaction with AI voice agents.
  - Supports automating phone calls through MCP agents.

- **Client-agnostic usage**
  - The same server URL works across different MCP-capable chat clients.
  - Authentication occurs when adding the server in each client/application.

- **Configuration documentation**
  - Dedicated configuration guidance available via a “Configuration” page (client setup instructions, authentication steps, etc.).

## Authentication
- Authentication is handled when you add the MCP server to your application or chat client.
- The URL itself is static; credentials and auth configuration are applied client-side.

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Category:** Messaging MCP servers
- **Tags:** telecom, voice, AI integration
