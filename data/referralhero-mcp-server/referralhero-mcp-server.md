# ReferralHero MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** ReferralHero  
**Source:** https://mcp.pipedream.com/app/referralhero

## Overview
ReferralHero MCP Server is an integration built on Pipedream that connects your ReferralHero account to AI tools via the Model Context Protocol (MCP). It exposes ReferralHero’s referral program features—such as creation, management, and tracking—to AI-powered development environments and assistants.

## Features
- **MCP-based integration**: Exposes ReferralHero functionality through the Model Context Protocol for standardized access from AI tools.
- **Referral program creation**: Programmatic creation of new referral and loyalty campaigns (as supported by ReferralHero’s API).
- **Program management**: Manage existing referral programs and campaigns, including configuration and updates via MCP.
- **Referral tracking**: Access and track referral performance data and campaign metrics from ReferralHero.
- **Loyalty and campaigns support**: Work with loyalty programs and campaigns defined in your ReferralHero account.
- **AI tool connectivity via Pipedream**: Connects ReferralHero to AI tools such as Cursor, Claude, and Windsurf using Pipedream’s MCP server.
- **Model-aware context**: Enables AI agents to query and use referral program data as part of their context when assisting with business or marketing tasks.

## Integrations
- **Pipedream MCP Server** (host environment)
- **AI Tools** (via MCP):
  - Cursor
  - Claude
  - Windsurf
