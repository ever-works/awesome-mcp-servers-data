# PAGE X MCP Server

**Category:** Business & Commerce MCP Servers  
**Tags:** CRM, sales, lead-management

PAGE X MCP Server exposes PAGE X — a CRM focused on sales tracking, lead management, task automation, and service efficiency — via a standardized MCP endpoint that applications and chat clients can connect to.

---

## MCP Server URL

Use this static MCP server URL for all clients (authentication occurs when adding the server to your app):

```text
https://mcp.pipedream.net/v2
```

---

## Features

- **CRM integration**
  - Connects to PAGE X, a CRM system oriented around managing sales processes and customer relationships.
- **Sales tracking**
  - Designed to work with PAGE X capabilities for tracking deals, opportunities, and sales performance.
- **Lead management**
  - Supports workflows that involve capturing, organizing, and following up on leads managed in PAGE X.
- **Task automation**
  - Interfaces with PAGE X automation features to streamline repetitive CRM-related tasks.
- **Service efficiency**
  - Aims to improve service operations by exposing PAGE X tools for organizing customer interactions and follow-ups.
- **Standard MCP endpoint**
  - Single static URL (`https://mcp.pipedream.net/v2`) for all clients.
  - Authentication handled when adding the server in your chat client or application.
- **Multi-client compatibility**
  - Can be added to different MCP-compatible chat clients and applications.
- **Centralized configuration docs**
  - Detailed connection and setup instructions available via the general Configuration page on Pipedream.

---

## Setup & Integration

1. Copy the MCP server URL:
   - `https://mcp.pipedream.net/v2`
2. Add the server in your MCP-compatible app or chat client.
3. Authenticate when prompted by your client.
4. (Optional) Refer to the Pipedream Configuration page for detailed, client-specific setup steps.
