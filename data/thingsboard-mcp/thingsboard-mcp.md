## Overview

thingsboard-mcp is the official ThingsBoard MCP (Model Context Protocol) server. It provides a natural language interface that allows LLMs and AI agents to access and interact with ThingsBoard IoT data and services programmatically.

- **Repository:** https://github.com/thingsboard/thingsboard-mcp  
- **License:** Apache-2.0

## Features

- **MCP server for ThingsBoard**  
  - Implements the Model Context Protocol to expose ThingsBoard capabilities to LLMs and AI agents.
