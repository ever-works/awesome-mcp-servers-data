## Overview

SegmentStream MCP turns AI workspaces into agentic marketing analysts by exposing nine core measurement technologies developed over seven years managing $100M+ ad spend.

## Features

- Identity Graph: cross-device user profiles
- Cross-Channel Attribution: click-time revenue optimization
- Self-Reported Reattribution: LLM classification for dark funnel channels
- Marketing Mix Optimization: automated budget reallocation
- Incrementality Testing: geo holdout experiments
- Signal Quality: synthetic conversions via CAPI
- Visit Scoring: ML-based session quality
- Predictive Lead Scoring: lead-to-conversion probability
- Customer LTV Prediction: lifetime value modeling

## Capabilities

- Continuous Optimization Loop (Measure → Predict → Validate → Optimize → Learn)
- 100+ pre-built skills for campaign performance, anomaly detection, budget modeling
- Data stored in BigQuery for full SQL access
