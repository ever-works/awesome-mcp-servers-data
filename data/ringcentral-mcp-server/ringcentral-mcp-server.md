## RingCentral MCP Server

**Category:** Messaging MCP Servers  
**Website:** https://mcp.pipedream.com/app/ringcentral

### Description
RingCentral MCP Server is an MCP-compatible integration that connects MCP-enabled applications to RingCentral’s cloud communications platform, covering phone, meetings, and contact-center capabilities.

### MCP Server URL
- Static MCP server endpoint (for all clients):  
  `https://mcp.pipedream.net/v2`
- Authentication is performed when adding the server to your MCP-enabled application.

### Features
- **MCP Integration**
  - Provides a single static MCP server URL usable across compatible chat and MCP clients.
  - Designed to be added as an external tool/server in MCP-enabled applications.

- **RingCentral Platform Access**
  - Integrates with RingCentral’s cloud communications platform.
  - Supports intelligent phone-related workflows (e.g., calling features via RingCentral APIs, where configured).
  - Supports meetings-related workflows (e.g., scheduling or managing RingCentral meetings, where supported by the integration).
  - Supports contact center-related workflows (e.g., call/contact handling via RingCentral contact center, where supported).

- **Client Configuration Support**
  - Can be added from multiple chat clients that support MCP.
  - Centralized configuration guidance available via the Pipedream Connect configuration page.
