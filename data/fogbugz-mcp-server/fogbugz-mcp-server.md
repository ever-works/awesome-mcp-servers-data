# FogBugz MCP Server

**Brand:** pipedream  
**Category:** Project Management MCP Servers  
**Tags:** issue-tracking, bug-tracking, project-management

## Description
FogBugz MCP Server is an MCP integration that connects your FogBugz project management and issue tracking account with MCP-compatible AI tools via Pipedream. It enables workflows such as issue tracking and project coordination through AI-driven clients.

## Features
- Connects a FogBugz account to MCP-compatible clients via Pipedream
- Supports integration with AI tools such as Cursor, Claude, and Windsurf
- Enables AI-assisted issue tracking workflows
- Facilitates bug-tracking operations through MCP clients
- Supports project management and project coordination tasks via AI interfaces

## Integrations
- FogBugz
- Cursor
- Claude
- Windsurf
- Pipedream MCP framework
