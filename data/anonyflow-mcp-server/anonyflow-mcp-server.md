## Overview
AnonyFlow MCP Server is an MCP-compatible service that provides encryption-based data anonymization and privacy tooling to help maintain GDPR, CCPA, and HIPAA data protection compliance within MCP workflows. It runs as a static MCP server endpoint that can be integrated with various chat or MCP-compatible clients.

## Features
- **Encryption-based data anonymization**: Uses encryption methods to anonymize sensitive data while preserving utility for downstream processing.
- **Privacy tools for compliance workflows**: Designed to support privacy controls and transformations within MCP-powered workflows.
- **Regulatory support**:
  - GDPR-aligned anonymization practices
  - CCPA-focused data privacy handling
  - HIPAA-oriented protection of sensitive health-related data
- **MCP server endpoint**:
  - Static server URL usable across supported MCP clients: `https://mcp.pipedream.net/v2`
  - Authentication handled when adding the server to your chosen application or client
- **Multi-client compatibility**: Can be added to different chat/MCP clients (instructions provided per client via the configuration page).

## Integration & Usage
- Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server to a supported chat or MCP client following client-specific setup instructions.
- Use within workflows that require anonymization and privacy-preserving handling of sensitive data.
