# Flutterwave MCP Server

## Overview
Flutterwave MCP Server provides MCP-based access to Flutterwave’s payment infrastructure, enabling you to build and manage secure payment flows. It connects your Flutterwave account to AI tools such as Cursor, Claude, and Windsurf via Pipedream’s MCP server.

## Features
- MCP-based integration with Flutterwave’s payment infrastructure
- Build and manage secure payment flows
- Connect a Flutterwave account to AI tools
  - Cursor
  - Claude
  - Windsurf
- Operates via Pipedream’s MCP server for orchestration and connectivity
