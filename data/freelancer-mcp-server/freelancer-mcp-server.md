# Freelancer MCP Server

An MCP (Model Context Protocol) server that connects to Freelancer, an online marketplace for hiring freelancers and finding freelance jobs.

## Basic Info
- **Name:** Freelancer MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Brand:** Freelancer  
- **Source URL:** https://mcp.pipedream.com/app/freelancer  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Description
The Freelancer MCP Server provides an MCP-compatible endpoint that lets applications integrate with the Freelancer platform for hiring freelancers and finding freelance jobs. It is a static MCP server URL that can be added to any compatible client, with authentication handled when the server is added to the application.

## Features
- **Static MCP Endpoint**  
  - Uses a single, static URL (`https://mcp.pipedream.net/v2`) for all compatible MCP clients.  
- **Freelancer Platform Integration**  
  - Connects to Freelancer, enabling workflows around hiring freelancers and discovering freelance jobs (exact tools and methods are not detailed on the page).  
- **Client-Agnostic Configuration**  
  - Designed to work with multiple chat or MCP-enabled clients; users "select your chat client" to learn setup steps.  
- **Application-Level Authentication**  
  - Authentication is performed when you add the server to your application, rather than per-URL instance.  
- **Configuration Guidance**  
  - Additional setup and usage details available via a dedicated configuration page (not expanded in the provided content).  
- **Hosted by Pipedream Connect**  
  - The server is operated and provided via Pipedream’s Connect infrastructure.
