# Klaxoon MCP Server

An MCP (Model Context Protocol) server that connects Klaxoon’s collaborative meeting and teamwork tools to any compatible MCP client.

## Overview
- **Name:** Klaxoon MCP Server  
- **Category:** Project Management MCP Servers  
- **Tags:** collaboration, meetings, work-management  
- **Provider / Platform:** Pipedream Connect  
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- Integrates Klaxoon’s collaborative environment (meetings, brainstorming, planning, and co-creation) with MCP-compatible chat and AI clients.
- Single static MCP server URL that works across different MCP clients.
- Authentication handled when adding the server to each client (no per-client URL required).
- Usable from multiple chat clients; documentation and setup guidance are available via a configuration page.

## Integration & Usage
- Use the MCP server URL `https://mcp.pipedream.net/v2` when configuring your MCP-compatible application.
- Follow your specific chat client’s instructions to add and authenticate the Klaxoon MCP Server.
- Additional technical and configuration details are available on the Pipedream configuration page.
