# Raven Tools MCP Server

## Overview
Raven Tools MCP Server is an MCP (Model Context Protocol) integration for the Raven Tools Link Manager. It allows you to connect Raven Tools to compatible chat or AI applications so you can manage white-hat backlink campaigns, including storing, organizing, and tracking link opportunities, as well as monitoring links gained or lost.

- **Category:** Business & Commerce MCP Servers  
- **Use cases:** SEO, marketing, link management

## Features
- **Raven Tools Link Manager integration**
  - Connects an MCP-compatible app to the Raven Tools Link Manager.  
  - Supports management of large-scale white-hat backlink campaigns.

- **Backlink opportunity management**
  - Store and organize thousands of link opportunities.  
  - Search and find link opportunities efficiently from within a connected client.

- **Link monitoring**
  - Monitor links you have gained.  
  - Track links you have lost to maintain backlink profile quality.

- **Static MCP server URL**
  - Uses a single static MCP server endpoint for all clients:  
    - `https://mcp.pipedream.net/v2`  
  - Authentication occurs when adding the server to your MCP-compatible application.

- **Multi-client compatibility**
  - Can be added to various MCP-aware chat clients or applications.  
  - Central configuration through the app’s configuration or “Add server” flow.

- **Account-based configuration**
  - Connect your Raven Tools account.  
  - Select the appropriate client (within Raven Tools) after connecting to start using tools.

## Setup & Configuration
- Copy the MCP server URL: `https://mcp.pipedream.net/v2`.  
- Add it as a server in your MCP-compatible chat client or application.  
- Authenticate with your Raven Tools account when prompted.  
- Select the Raven Tools client (site/account) you wish to manage.
