# 10,000ft MCP Server (Resource Management by Smartsheet)

## Overview
The 10,000ft MCP Server (Resource Management by Smartsheet MCP Server) is an MCP server integration that enables simple resource allocation and business management capabilities within MCP-compatible applications.

## Features
- **MCP Server Integration**: Exposes Resource Management by Smartsheet (formerly 10,000ft) through the MCP ecosystem.
- **Simple Resource Allocation**: Designed to support straightforward resource allocation workflows from within chat or MCP-enabled clients.
- **Static MCP Endpoint**: Uses a single static MCP server URL for all clients:
  - `https://mcp.pipedream.net/v2`
- **Account-Based Authentication**: Authentication occurs when adding the server to your MCP-compatible application after connecting your Resource Management by Smartsheet account.
- **Client-Agnostic Setup**: Can be added to multiple chat clients or MCP-compatible apps; setup instructions are provided per client via the configuration flow.
- **Central Configuration Page**: Additional configuration details and setup steps are available via a dedicated configuration page (linked from the interface).

## Usage
1. Connect your Resource Management by Smartsheet account in the Pipedream interface.
2. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
3. Add this server URL to your chosen MCP-compatible app or chat client.
4. Authenticate when prompted and select the appropriate client/workspace to begin using the available resource management tools.
