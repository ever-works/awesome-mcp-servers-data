# GoDaddy MCP Server

An MCP server that integrates GoDaddy services (domains, hosting, DNS) into MCP-compatible applications, hosted and orchestrated via Pipedream.

- **Category:** Cloud & DevOps MCP Servers
- **Provider / Brand:** Pipedream
- **Slug:** `godaddy-mcp-server`
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features

- **GoDaddy integration for MCP**
  - Connects GoDaddy services (such as domains, hosting, and DNS) into MCP-based workflows.
  - Designed to be added as an MCP server in compatible chat or agent clients.

- **Static MCP server endpoint**
  - Single static URL: `https://mcp.pipedream.net/v2` used for all clients.
  - Authentication handled when you add the server to your MCP-compatible application.

- **Pipedream-hosted orchestration**
  - Uses Pipedream as the hosting and orchestration layer for the MCP server.
  - Centralized configuration via Pipedream’s configuration page.

- **Client-agnostic setup**
  - Works with multiple chat / agent clients that support MCP.
  - Per-client instructions are available once a client is selected in the Pipedream UI.

- **Configuration workflow**
  - “Configure GoDaddy” flow to connect your GoDaddy account.
  - Account verification step (“Checking your account…”) before tools are made available.
  - Tools / actions are dynamically loaded after configuration (exact tools not listed in the provided content).
