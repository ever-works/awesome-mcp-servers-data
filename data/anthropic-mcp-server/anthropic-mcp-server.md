# Anthropic MCP Server

## Overview
Anthropic (Claude) MCP Server is a Model Context Protocol (MCP) server integration that connects MCP-compatible clients to Anthropic’s Claude models via Pipedream. It provides a static server endpoint that can be added to supported chat clients and authenticated through Pipedream.

- **Category:** AI Integration – MCP Servers  
- **Brand:** Anthropic  
- **Focus:** Access to Claude-based AI assistants through MCP-compatible tools

## Features
- **MCP server integration for Claude**  
  Provides an MCP-compliant server to connect Anthropic’s Claude models with MCP clients.

- **Static MCP server URL**  
  - Single, reusable endpoint for all clients:  
    `https://mcp.pipedream.net/v2`  
  - Authentication handled when adding the server to your application via Pipedream.

- **Client integration support**  
  - Designed to be added to various MCP-compatible chat clients.  
  - Example guidance available for ChatGPT (OpenAI) as a client.  
  - Additional configuration instructions available on a dedicated configuration page.

- **Account connection via Pipedream**  
  - Connect an Anthropic (Claude) account through Pipedream’s interface.  
  - Central place to manage Anthropic connections for MCP usage.

- **MCP tools discovery (via Pipedream)**  
  - Interface to list available MCP tools/actions exposed by the Anthropic (Claude) integration.  
  - Tools are loaded and displayed within the Pipedream MCP UI (e.g., “Available tools” section).

- **General AI assistant capabilities (via Claude)**  
  - Access to Claude models as AI assistants suitable for a wide range of tasks and scales.  
  - Emphasis on AI research and products that prioritize safety and scalable capability (as provided by Anthropic’s Claude).

## Configuration & Usage
- Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server to a supported MCP client (e.g., ChatGPT with MCP support) following the specific client guide.
- Authenticate and manage your Anthropic (Claude) connection via Pipedream after signing in.
- Optionally refer to the full configuration page on Pipedream for detailed setup steps.
