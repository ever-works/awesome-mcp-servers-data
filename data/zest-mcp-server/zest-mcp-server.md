# Zest MCP Server

Zest MCP Server is an MCP (Model Context Protocol) server integration that exposes Zest’s business gifting and rewards capabilities to compatible chat or AI clients via a single static endpoint.

## Overview
- **Type:** MCP server integration
- **Category:** Business & Commerce MCP Servers
- **Purpose:** Provide MCP-based access to Zest’s business gifting and rewards functionality from supported applications.
- **Server URL:** `https://mcp.pipedream.net/v2` (static URL used by all clients; authentication is handled when adding the server to your app)

## Features
- **MCP-based integration**
  - Connects Zest’s gifting and rewards platform to MCP-compatible chat or AI clients.
  - Uses a single static MCP server URL for all clients.
- **Business gifting and rewards access**
  - Interfaces with Zest’s business gifting and rewards functionality for use in workflows and conversational tools.
- **Client-agnostic endpoint**
  - Same MCP server URL works across different supported chat clients and applications.
- **Configurable via client settings**
  - Added to your app by configuring the MCP server URL and authenticating when prompted.
- **Hosted by Pipedream**
  - Runs on Pipedream Connect infrastructure, with centralized configuration and management.
