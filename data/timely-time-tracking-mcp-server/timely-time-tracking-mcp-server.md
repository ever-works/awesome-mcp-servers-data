# Timely Time Tracking MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Timely

Timely Time Tracking MCP Server connects the Timely AI-powered time tracking platform with MCP-based workflows. It lets you integrate Timely’s automated time tracking and timesheet creation capabilities into compatible chat or MCP-enabled applications.

---

## Features

- **MCP-compatible server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same URL works across clients; authentication handled when adding the server to your application.

- **Timely integration**  
  - Connects to a Timely account to enable AI-powered time tracking within MCP workflows.  
  - Supports automated timesheet creation driven by Timely’s core platform.

- **Client-agnostic usage**  
  - Designed to be added to different chat clients or MCP-compatible applications.  
  - Configuration guided per client (via “Select your chat client to learn how to get started”).

- **Configuration support**  
  - Central configuration via a dedicated configuration page (linked as “Configuration page”).  
  - Authentication flow handled at the time you add the server to your app.

- **Tooling discovery (actions)**  
  - Exposes “available tools” / “actions” through the MCP interface (listed dynamically in the client as “Loading actions…”, “Loading available tools…”).
