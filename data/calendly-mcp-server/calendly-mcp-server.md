# Calendly MCP Server

An MCP server that connects to Calendly v2, enabling MCP-compatible applications to schedule and manage meetings and appointments via Calendly.

## Overview
- **Type:** MCP server / workflow-automation integration
- **Category:** Workflow Automation MCP Servers
- **Brand:** Calendly
- **Integration:** Calendly v2 API
- **Primary use:** Programmatic scheduling and management of meetings and appointments through MCP-compatible clients.

## Features
- **Calendly v2 integration**
  - Connects directly to Calendly’s v2 API.
  - Uses your existing Calendly account and calendars.

- **MCP-compatible server endpoint**
  - Single static MCP server URL: `https://mcp.pipedream.net/v2`.
  - Works with any MCP-compatible client; authentication is performed when you add the server to your application.

- **Programmatic scheduling & appointments**
  - Designed to automate appointment scheduling using Calendly.
  - Helps avoid double booking by leveraging Calendly’s scheduling logic and calendar awareness.

- **Client-agnostic configuration**
  - Can be added to different MCP-enabled chat or automation clients.
  - Configuration guidance available per client type via a configuration page.

## Configuration
- Copy and use the MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server URL to your MCP-compatible application and authenticate using your Calendly account.
- Additional client-specific setup instructions are available on the configuration page linked from the product.
