# Sellix MCP Server

- **Website**: https://mcp.pipedream.com/app/sellix
- **Category**: Business & Commerce – MCP Servers
- **Vendor / Brand**: Pipedream
- **Product Type**: Model Context Protocol (MCP) server integration for Sellix

## Description
Sellix MCP Server is a Model Context Protocol server hosted on Pipedream that connects MCP-aware agents and chat clients to Sellix, a digital ecommerce platform for online stores. It exposes Sellix’s store management and automation capabilities so that agents can interact with and manage a Sellix-powered digital store programmatically.

## Features
- **MCP server endpoint**
  - Static MCP server URL: `https://mcp.pipedream.net/v2`
  - Works as a shared endpoint for all supported MCP clients.

- **Sellix account integration**
  - Connects to a Sellix account from within Pipedream.
  - Uses account authentication when adding the MCP server to an application.

- **MCP-aware client compatibility**
  - Designed to be added as a server to various MCP-compatible chat clients and agents.
  - Client-specific setup guidance available from the app (select a chat client and follow instructions).

- **Access to Sellix ecommerce capabilities**
  - Exposes Sellix’s digital ecommerce store management and automation features to MCP agents (e.g., for creating, growing, and managing an online digital-product store).
