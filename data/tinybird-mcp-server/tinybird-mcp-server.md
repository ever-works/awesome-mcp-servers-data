# Tinybird MCP Server

An MCP server that integrates with Tinybird to help agents build and query real-time data products and APIs on streaming data.

- **Category:** data-analysis-exploration-mcp-servers  
- **Brand:** tinybird  
- **Source:** https://mcp.pipedream.com/app/tinybird  
- **MCP Endpoint URL:** `https://mcp.pipedream.net/v2`

## Features

- **Tinybird integration** – Connects MCP agents to Tinybird so they can work with streaming and real-time data.  
- **Real-time data products** – Enables building and querying real-time data products and APIs based on streaming data.  
- **Unified static MCP URL** – Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) that works across supported clients.  
- **Client-agnostic configuration** – Designed to be added to various chat/MCP clients; configuration details are provided per client via documentation.  
- **Authentication at client setup** – Authentication is handled when adding the server to your application, not via different URLs.  
- **Pipedream Connect platform** – Runs on Pipedream Connect infrastructure for hosting and managing the MCP server.

## Setup & Integration

- Use the static URL `https://mcp.pipedream.net/v2` when adding the Tinybird MCP Server to your MCP-compatible chat client or application.  
- Follow client-specific instructions on the configuration page (linked from the source) to complete authentication and setup.
