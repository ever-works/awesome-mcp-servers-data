# Dispatch MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Dispatch  
**Slug:** `dispatch-mcp-server`

An MCP server for Dispatch that connects your Dispatch account to MCP‑compatible AI tools, enabling coordination and automation of on‑demand delivery and logistics workflows.

## Description
Dispatch MCP Server is an MCP server integration, provided via Pipedream, that links your Dispatch account with AI tools such as Cursor, Claude, and Windsurf. It is designed to help MCP clients coordinate and automate on‑demand delivery operations and logistics tasks through standardized MCP tools.

## Features
- **MCP Server for Dispatch**: Exposes your Dispatch account through the Model Context Protocol so MCP‑compatible clients can interact with Dispatch data and actions.
- **AI Tool Integration**: Connects Dispatch to AI tools like:
  - Cursor
  - Claude
  - Windsurf
- **Logistics Coordination**: Supports coordination of on‑demand delivery operations and logistics workflows via MCP tools.
- **Automation of Delivery Tasks**: Enables AI agents/clients to automate logistics and delivery‑related tasks (e.g., managing operations, handling on‑demand requests) through Dispatch.
- **Pipedream‑Hosted Integration**: Provided as a Pipedream MCP server, simplifying connection between Dispatch and MCP clients.

## Use Cases
- Connecting Dispatch to AI coding or IDE assistants (e.g., Cursor, Windsurf) to build and run logistics workflows.
- Allowing AI agents (e.g., via Claude) to interact with Dispatch for on‑demand delivery coordination.
