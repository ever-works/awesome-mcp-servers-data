# Cisco Meraki MCP Server

## Overview
The Cisco Meraki MCP Server is a cloud-based integration endpoint that connects applications to Cisco Meraki’s cloud networking platform via MCP. It provides access to Meraki services such as wireless, switching, security and SD-WAN, smart cameras, sensors, and related open APIs.

MCP server base URL:
```text
https://mcp.pipedream.net/v2
```

## Features
- **Cloud-based MCP server endpoint**
  - Static MCP server URL usable by any compatible client: `https://mcp.pipedream.net/v2`.
  - Authentication handled when adding the server to your application.

- **Cisco Meraki platform integration**
  - Connects to Cisco Meraki’s cloud networking platform.
  - Enables access to:
    - Wireless networking services.
    - Switching infrastructure.
    - Security and SD-WAN capabilities.
    - Smart cameras.
    - Sensors.

- **API access**
  - Provides access to Cisco Meraki open APIs through MCP.
  - Supports integration of networking, security, and device telemetry into custom applications and workflows.

- **Cloud-first operations**
  - Designed around Meraki’s cloud-first operations model.
  - Suitable for managing and automating network-related tasks via an MCP-compatible client.

## Getting Started
- Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
- Add this server URL in your chat client or application that supports MCP.
- Configure authentication during server setup within your client.
- Additional configuration details are available via the provider’s configuration documentation.
