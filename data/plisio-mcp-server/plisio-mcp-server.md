# Plisio MCP Server

An MCP Server integration for Plisio that lets you accept and manage cryptocurrency payments via MCP-based workflows.

## Overview
Plisio MCP Server is an MCP-compatible endpoint, provided through Pipedream Connect, that you can add to MCP-enabled chat clients or applications to work with Plisio crypto payments.

- **Static MCP server URL:** `https://mcp.pipedream.net/v2`
- **Authentication:** Performed when adding/configuring the server in your MCP-compatible client or app.

## Features
- MCP server endpoint for integrating Plisio crypto payment capabilities into MCP-based tools and workflows.
- Single static server URL that works across all compatible MCP clients.
- Authentication handled during client/app configuration instead of per-URL customization.
- Works within Pipedream Connectâ€™s ecosystem for easier connection and management of integrations.
- Configurable through a dedicated configuration flow in supported MCP clients (via the referenced Configuration page).

## Integration & Usage
- Add the static MCP server URL to your MCP-enabled chat client or application.
- Complete the authentication step when prompted by your client/app.
- Refer to the associated configuration documentation ("Configuration" page) for client-specific setup details.
