# Gravity Forms MCP Server

**Category:** Content Management MCP Servers  
**Tags:** forms, WordPress, content-management

## Overview
Gravity Forms MCP Server is an MCP integration for the Gravity Forms WordPress plugin. It allows MCP-compatible clients (such as chat-based tools or other MCP apps) to build, manage, and query WordPress-based web forms powered by Gravity Forms.

MCP access is provided via a static server URL that you add to your MCP-compatible application.

## Features
- **MCP integration for Gravity Forms**  
  - Connects MCP clients to the Gravity Forms plugin on a WordPress site.  
  - Exposes form-related capabilities through the MCP protocol.

- **Build and manage WordPress web forms**  
  - Create and configure Gravity Forms forms via an MCP client.  
  - Manage existing forms (e.g., structure, fields) on a WordPress site.

- **Query forms and form data**  
  - Enable MCP clients to query available forms.  
  - Retrieve information related to WordPress-based Gravity Forms forms.

- **Static MCP server endpoint**  
  - Single shared MCP server URL for all clients:  
    - `https://mcp.pipedream.net/v2`  
  - Authentication occurs when adding the server to an application.

- **Works with multiple MCP clients**  
  - Designed for any compatible MCP chat client or application.  
  - Can be added to various tools that support MCP servers.

## Technical Details
- **Server URL:** `https://mcp.pipedream.net/v2`  
- **Platform:** WordPress (via Gravity Forms plugin)  
- **Integration Type:** MCP server (remote, static URL)
