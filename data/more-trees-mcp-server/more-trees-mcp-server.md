# More Trees MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** More Trees  
**Slug:** more-trees-mcp-server

## Description
More Trees MCP Server exposes the More Trees API via the Model Context Protocol (MCP), enabling automated CO₂ offsetting workflows. It allows users and businesses to programmatically plant trees, supporting sustainability and environmental initiatives within MCP-compatible applications.

## Features
- **MCP-compatible API access**: Exposes the More Trees API through a standard MCP server endpoint.
- **Static MCP server URL**: Single endpoint for all clients: `https://mcp.pipedream.net/v2`.
- **Programmatic tree planting**: Supports automated workflows to plant trees on behalf of individuals or businesses.
- **Automated CO₂ offsetting**: Designed for integrating CO₂ sequestration and offsetting into applications and workflows.
- **Integration with chat / MCP clients**: Can be added as a server to supported chat clients and MCP-enabled tools.
- **Centralized configuration**: Setup and usage details are available via a configuration page (for supported clients and environments).
