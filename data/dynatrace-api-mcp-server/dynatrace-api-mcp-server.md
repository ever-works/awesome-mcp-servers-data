# Dynatrace API MCP Server

An MCP server that integrates with the Dynatrace API to give MCP-compatible agents access to Dynatrace observability, security, and AIOps data.

## Overview
- **Type:** MCP server / integration
- **Category:** Monitoring
- **Brand:** Dynatrace
- **Slug:** `dynatrace-api-mcp-server`
- **Source URL:** https://mcp.pipedream.com/app/dynatrace_api
- **Static MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- Connects MCP-based agents to Dynatrace via the Dynatrace API.
- Access to full-stack observability data from Dynatrace.
- Access to runtime application security insights.
- Access to AIOps data and insights from Dynatrace.
- Uses a single static MCP server URL that works across supported clients.
- Authentication handled when adding the server to your application.

## Integration & Usage
