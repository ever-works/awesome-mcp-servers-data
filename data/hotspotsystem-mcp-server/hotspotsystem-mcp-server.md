### Features
- MCP server endpoint for HotspotSystem: `https://mcp.pipedream.net/v2`
- Works as a shared static MCP URL for all clients
- Authentication handled when adding the server inside your MCP-compatible application or chat client
- Designed to support automation around:
  - WiFi hotspot management
  - Reseller operations for HotspotSystem
- Built on Pipedream Connect infrastructure
