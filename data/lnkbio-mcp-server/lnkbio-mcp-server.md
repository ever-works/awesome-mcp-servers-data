# Lnk.Bio MCP Server

**Category:** Content Management MCP Servers  
**Brand:** lnkbio  
**Slug:** lnkbio-mcp-server

## Description
Lnk.Bio MCP Server is an integration that connects your Lnk.Bio account to AI tools via Pipedream’s MCP server, enabling management of multiple links from a single bio URL through MCP-based automation and tools.

Source: <https://mcp.pipedream.com/app/lnk_bio>

## Features
- Connects a Lnk.Bio account to AI tools through Pipedream’s MCP server
- Supports integration with AI tools such as Cursor, Claude, and Windsurf
- Enables management of multiple links from a single Lnk.Bio bio URL
- Allows automation and tooling around link management using the MCP protocol
