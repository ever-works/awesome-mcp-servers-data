# TidyCal MCP Server

**Category:** Project Management MCP Servers  
**Brand:** TidyCal  
**Tags:** scheduling, calendar, appointments

## Overview
TidyCal MCP Server is an MCP (Model Context Protocol) server that connects TidyCal’s scheduling capabilities to MCP-based agents and applications. It enables programmatic scheduling and calendar interactions (such as booking and managing appointments) through a standardized MCP endpoint.

## Features
- Connects TidyCal’s scheduling system to MCP-compatible clients and agents.
- Supports programmatic scheduling workflows (e.g., creating and managing appointments) via MCP.
- Provides calendar interaction capabilities for automation and integration scenarios.
- Uses a single static MCP server URL that works across all compatible clients.
- Authentication handled when adding the server to your application (per-client auth, shared endpoint).
- Can be integrated into various chat or agent clients that support MCP.

## MCP Server URL
- Static MCP endpoint (for all clients):
  - `https://mcp.pipedream.net/v2`

## Integration
- Add the server to any MCP-capable app or chat client using the static URL.
- Configuration details are provided in the associated configuration documentation (per client setup instructions).
