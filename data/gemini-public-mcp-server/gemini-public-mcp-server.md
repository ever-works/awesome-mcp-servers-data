# Gemini Public MCP Server

**Category:** Blockchain & Crypto MCP Servers  
**Brand:** Gemini  
**Source:** https://mcp.pipedream.com/app/gemini_public  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Description
Gemini Public MCP Server exposes Gemini cryptocurrency exchange APIs over the Model Context Protocol (MCP), allowing applications and chat clients to access Gemini’s trading and market data capabilities through a unified MCP endpoint.

## Features
- **MCP-compatible endpoint**: Single static MCP server URL (`https://mcp.pipedream.net/v2`) that can be used with any supported MCP client.
- **Gemini exchange integration**: Connects to Gemini’s cryptocurrency exchange APIs for Bitcoin, Ether, and other supported assets.
- **Trading access**: Designed to enable interaction with Gemini’s trading APIs (e.g., order placement and related trading operations), subject to authentication and permissions.
- **Market data access**: Provides access to Gemini market data (such as prices and related exchange data) through MCP.
- **Client-agnostic usage**: Same server URL works across different chat clients and MCP-compatible applications.
- **Authentication at client setup**: Authentication is performed when adding the server to your application or client, rather than requiring per-URL customization.
- **Configuration documentation**: A separate configuration page explains how to add and use the server with various clients.
- **Pipedream Connect integration**: Operated and hosted via Pipedream Connect infrastructure.

## Integration & Usage
- Use the static MCP URL `https://mcp.pipedream.net/v2` when configuring the server in your MCP-compatible client or application.
- Authenticate within your app or chat client as prompted during setup.
- Refer to the provider’s configuration documentation for client-specific setup steps.
