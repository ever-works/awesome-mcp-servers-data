# Retailed MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Pipedream  
**Website:** https://mcp.pipedream.com/app/retailed  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Description
Retailed MCP Server is a Model Context Protocol server on Pipedream that connects the Retailed Suite inventory and reseller tools to MCP-compatible clients and assistants. It allows inventory and reseller workflows from Retailed Suite to be integrated into MCP-driven applications through a single static server URL.

## Features
- **MCP-compatible server**: Exposes Retailed Suite inventory and reseller capabilities via the Model Context Protocol.
- **Static server endpoint**: Uses a single static URL (`https://mcp.pipedream.net/v2`) that works across different MCP clients.
- **Per-client authentication**: Authentication is handled when adding the server to each MCP-compatible application or chat client.
- **Integration with chat clients**: Can be added to supported chat or assistant clients to enable inventory and reseller workflows inside conversations.
- **Configuration guidance**: Documentation available via a configuration page to help set up and integrate the server into applications.
