# Connecteam MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Connecteam  
**Source:** https://mcp.pipedream.com/app/connecteam

Connecteam MCP Server is an MCP (Model Context Protocol) integration that exposes Connecteam’s employee app and workforce management capabilities through a unified MCP endpoint provided by Pipedream.

---

## Features

- **Static MCP Server Endpoint**  
  - Single static URL for all clients: `https://mcp.pipedream.net/v2`  
  - Authentication handled when adding the server to your MCP-compatible application.

- **Connecteam Integration via MCP**  
  - Provides access to Connecteam’s employee app and workforce management functionality through MCP.  
  - Designed to integrate with chat clients and other MCP-aware tools.

- **Client-Agnostic Setup**  
  - Same server URL for all users and organizations.  
  - Configuration is done on the client side (e.g., your chat or MCP client) using the provided URL.

- **Documentation & Configuration Guidance**  
  - Step-by-step instructions for adding the server to supported chat clients.  
  - Central configuration documentation available via the linked Configuration page (on Pipedream Connect).

---

## Setup

1. Copy the MCP server URL:  
   `https://mcp.pipedream.net/v2`
2. Add this URL as an MCP server in your MCP-compatible app or chat client.  
3. Authenticate when prompted by your application during setup.
