# Givebutter MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Givebutter  
**Slug:** givebutter-mcp-server

## Overview
The Givebutter MCP Server exposes Givebutter’s fundraising and donation‑management capabilities through the Model Context Protocol (MCP), enabling AI/chat clients to integrate with and orchestrate nonprofit fundraising workflows and campaigns.

Server base URL (static for all clients):
```text
https://mcp.pipedream.net/v2
```

## Features
- **MCP-compatible server**
  - Implements a standard MCP server endpoint usable from any compatible chat or AI client.
  - Uses a single static URL (`https://mcp.pipedream.net/v2`) for all clients.
  - Authentication is performed when adding/configuring the server in the client.

- **Access to Givebutter fundraising capabilities**
  - Surfaces Givebutter’s fundraising and donation‑management features via MCP for use in workflows that support nonprofits and fundraising campaigns.
  - Designed to power MCP‑driven workflows (e.g., campaign management, donation handling, nonprofit operations) inside chat/AI tools.

- **Pipedream Connect integration**
  - Hosted and powered by Pipedream Connect.
  - Uses Pipedream’s infrastructure for MCP server delivery and configuration.

- **Client-agnostic configuration**
  - Works with multiple chat clients; users select their client and follow setup instructions there.
  - Central configuration is documented on the Pipedream MCP Configuration page.
