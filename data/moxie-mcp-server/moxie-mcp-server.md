# Moxie MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Moxie  
**Source:** https://mcp.pipedream.com/app/moxie

## Overview
The Moxie MCP Server is an MCP-compatible server that connects MCP-based tools and chat clients to Moxie’s business management capabilities, including client management, invoicing, project management, time tracking, and accounting.

## Features
- **MCP-compatible server endpoint**  
  - Provides a static MCP server URL usable by any compatible client: `https://mcp.pipedream.net/v2`  
  - Authentication is performed when adding the server to an MCP-enabled application.

- **Client management integration**  
  - Access and work with client records managed in Moxie.

- **Invoicing integration**  
  - Connects to Moxie’s invoicing features for creating and managing invoices.

- **Project management integration**  
  - Interfaces with Moxie’s project management tools for tracking and managing projects.

- **Time tracking integration**  
  - Links MCP-based tools to Moxie’s time tracking capabilities.

- **Accounting integration**  
  - Connects to Moxie’s accounting features for financial tracking and bookkeeping tasks.

- **Collaboration support**  
  - Integrates with Moxie’s collaborator features to support working with team members.

- **Proposals integration**  
  - Accesses Moxie’s proposal-related tools (e.g., creating or managing proposals) through MCP.

- **Multi-client compatibility**  
  - Designed to be added to various chat or MCP-enabled clients via configuration.

## Setup
- Use the static MCP URL: `https://mcp.pipedream.net/v2` when configuring your MCP-compatible client.  
- Authenticate within your application when adding the Moxie MCP Server.
