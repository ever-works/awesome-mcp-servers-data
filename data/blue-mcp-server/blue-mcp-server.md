# Blue MCP Server

**Category:** Project Management MCP Servers  
**Brand:** Blue  
**Description:** An MCP server that provides MCP-based access to the Blue teamwork and collaboration platform.

---

## Features

- **MCP-based access to Blue**  
  - Exposes the Blue teamwork and collaboration platform through the Model Context Protocol (MCP), enabling compatible clients to interact with Blue.

- **Static MCP server URL**  
  - Single, static endpoint for all supported MCP clients:  
    - `https://mcp.pipedream.net/v2`  
  - Same URL works across different applications; authentication is handled when adding the server in each client.

- **Client-agnostic integration**  
  - Designed to be added to multiple chat or MCP-capable clients.  
  - Configuration guidance is available per client via the platform’s configuration resources.

- **Hosted by Pipedream Connect**  
  - Server is hosted and delivered via Pipedream’s "Connect" infrastructure.

---

## Setup & Integration

1. Use the static MCP server URL: `https://mcp.pipedream.net/v2`.  
2. Add the server in your MCP-compatible or chat client.  
3. Authenticate within your client when prompted.  
4. Optionally refer to the provider’s full configuration documentation for client-specific setup steps.
