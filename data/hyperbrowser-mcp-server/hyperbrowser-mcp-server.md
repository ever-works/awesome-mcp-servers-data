# Hyperbrowser MCP Server

An MCP (Model Context Protocol) server implementation for Hyperbrowser, providing AI agents with tools for scalable browser automation and web interaction.

- **Website/Source**: https://github.com/hyperbrowserai/mcp
- **Category**: Workflow Automation MCP Servers
- **Tags**: browser-automation, web-automation, scraping
- **License**: MIT

## Features

- Implements a Model Context Protocol (MCP) server specifically for Hyperbrowser.
- Enables AI agents to perform browser-based automation tasks.
- Supports scalable web interaction and automation workflows (e.g., browsing, scraping, scripted interactions) via Hyperbrowser.
- Distributed as an open-source project with Dockerfile and scripts for containerized or scripted deployment.
- Node.js/TypeScript-based codebase (package.json, tsconfig.json) for extensibility and customization.

## Technical Details

- Repository structure includes:
  - `src/` – main server implementation source code.
  - `docker/` – Docker-related configuration and assets.
  - `scripts/` – helper scripts for running or managing the server.
- Includes `run_server.sh` for simplified local/server startup.
- Configuration via `smithery.yaml` for MCP-related integration/metadata.
