# Simple Analytics MCP Server

## Overview
Simple Analytics MCP Server is an MCP-compatible backend that connects to Simple Analytics, allowing privacy-friendly website analytics (no cookies, no personal data) to be accessed and queried from MCP-based tools and chat clients.

- **Category:** Business & Commerce – MCP Servers  
- **Brand:** Simple Analytics  
- **Slug:** `simple-analytics-mcp-server`  
- **Source URL:** https://mcp.pipedream.com/app/simple_analytics

## Features
- **MCP server for Simple Analytics**  
  - Exposes Simple Analytics data to MCP-based tools and chat clients.
- **Privacy-friendly analytics**  
  - No cookies.  
  - No personal data collection.
- **Static MCP endpoint**  
  - Server URL: `https://mcp.pipedream.net/v2` (shared endpoint used for this integration).  
  - Authentication handled when adding the server to your application.
- **Account connection workflow**  
  - Connect your Simple Analytics account via the Pipedream interface.  
  - Select your Simple Analytics client/site after authentication.
- **Client integration guidance**  
  - Instructions for adding the MCP server to different chat clients.  
  - Link to a full configuration page for detailed setup.
- **Tooling integration**  
  - Exposes “available tools” / actions (loaded dynamically) that operate on Simple Analytics data.

## Usage
1. Connect your Simple Analytics account in the Pipedream Simple Analytics MCP app.  
2. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.  
3. Add this MCP server URL to your compatible chat client or application.  
4. Authenticate when prompted and select the Simple Analytics client/site to query.
