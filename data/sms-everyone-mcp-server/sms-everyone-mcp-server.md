# SMS Everyone MCP Server

## Overview
SMS Everyone MCP Server is an MCP (Model Context Protocol) integration that enables sending bulk SMS marketing and business SMS messages via SMS Everyone from MCP-compatible chat clients and tools.

- **Category:** Messaging MCP Servers
- **Brand:** SMS Everyone
- **Slug:** `sms-everyone-mcp-server`

## Features
- **Bulk SMS marketing:** Send large-volume SMS campaigns through SMS Everyone using MCP-compatible applications.
- **Business SMS messaging:** Support for transactional and business-oriented SMS use cases (e.g., notifications, alerts, reminders).
- **MCP server endpoint:** Uses a static MCP server URL: `https://mcp.pipedream.net/v2`.
- **Central configuration:** Configure SMS Everyone once and connect it to your MCP client to start sending messages.
- **Account-based authentication:** Authentication occurs when adding the MCP server to your application, tied to your SMS Everyone account.
- **Multi-client compatibility:** Can be added to different MCP-enabled chat clients / applications.

## Setup
1. Connect your SMS Everyone account in the configuration interface.
2. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
3. Add the MCP server to your MCP-compatible app or chat client.
4. Authenticate when prompted and select the relevant client/account.
