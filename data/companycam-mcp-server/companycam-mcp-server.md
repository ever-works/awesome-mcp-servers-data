# CompanyCam MCP Server

CompanyCam MCP Server integrates the CompanyCam photo and job site documentation platform into MCP-compatible applications, enabling automated access to project photos and related data.

## Overview
- **Type:** MCP server / integration
- **Category:** Business & Commerce – MCP Servers
- **Use Cases:** Field service, job site photo capture, project documentation, visual records management
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP Integration Endpoint**  
  - Provides a single static MCP server URL that works across compatible chat or agent clients.  
  - Authentication is handled when adding the server to a client or application.

- **CompanyCam Platform Connectivity**  
  - Connects MCP-based tools to CompanyCam’s photo and documentation platform.  
  - Enables automated access to project photos and job site documentation data (e.g., images and related project context).

- **Client-Agnostic Setup**  
  - Designed to be added to multiple chat or agent clients that support MCP.  
  - Central configuration via the static URL rather than per-client endpoints.

- **Configuration Resources**  
  - Guided configuration steps available via a dedicated configuration page (outside this summary).  
  - Integration is powered by Pipedream Connect, leveraging their infrastructure and policies.
