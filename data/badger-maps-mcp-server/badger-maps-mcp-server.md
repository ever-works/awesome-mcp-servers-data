# Badger Maps MCP Server

An MCP server integration that brings Badger Maps’ field sales route planning capabilities into MCP-driven automation and chat-based workflows.

- **Website / Source**: https://mcp.pipedream.com/app/badger_maps
- **Category**: Business & Commerce – MCP Servers
- **Tags**: sales, route-planning, maps
- **MCP Server URL**: `https://mcp.pipedream.net/v2`

## Features

- **MCP integration for Badger Maps**
  - Exposes Badger Maps route-planning capabilities through the Model Context Protocol (MCP).
  - Can be used from any compatible MCP client or chat application.

- **Static server endpoint**
  - Uses a single static MCP server URL: `https://mcp.pipedream.net/v2`.
  - Same URL works across all supported clients; authentication is handled when adding the server to your app.

- **Field sales routing use case**
  - Designed specifically to support field sales route planning scenarios within MCP-driven automations and assistants.

- **Client-agnostic setup**
  - Can be added to different chat clients; configuration instructions are provided per client via the linked configuration page.

- **Pipedream-hosted**
  - Hosted and operated by Pipedream (Pipedream Connect), with terms and privacy controls managed via Pipedream’s platform.
