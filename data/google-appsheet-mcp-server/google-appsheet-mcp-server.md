# Google AppSheet MCP Server

**Category:** Workflow Automation MCP Servers  
**Brand:** Google  
**Source:** https://mcp.pipedream.com/app/google_appsheet

## Description
The Google AppSheet MCP Server is an integration that lets MCP-compatible tools build and control no-code Google AppSheet solutions programmatically. It exposes AppSheet capabilities through the Model Context Protocol (MCP), enabling automation and app management from supported chat clients and developer tools.

## Features
- Programmatic control of Google AppSheet solutions via MCP
- No-code app development support through integration with AppSheet
- Single static MCP server endpoint for all clients: `https://mcp.pipedream.net/v2`
- Works with any MCP-compatible chat client or tool
- Authentication handled when adding the server in the client/application
- Central configuration and setup guidance via a dedicated configuration page

## Usage
1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add this server URL to your MCP-compatible chat client or application.
3. Authenticate when prompted within your client.
