# X (Twitter) MCP Server by EnesCinr

An open-source Model Context Protocol (MCP) server that connects LLM environments to the X (Twitter) API, allowing models to post tweets and search Twitter directly.

## Features
- **MCP server for X (Twitter)**
  - Implements a Model Context Protocol server for interacting with Twitter from MCP-compatible clients.
- **Post tweets**
  - Allows LLM clients to create and post tweets via the Twitter API.
- **Search tweets by query**
  - Enables searching Twitter based on a text query from within the LLM environment.
- **LLM environment integration**
  - Designed to be used from MCP-enabled LLM clients (e.g., tools that support MCP servers) so models can perform live Twitter operations.
- **Docker support**
  - Repository includes a `Dockerfile` for containerized deployment and running the server in a consistent environment.
- **TypeScript/Node.js implementation**
  - Uses a Node.js + TypeScript stack (as indicated by `package.json` and `tsconfig.json`) for the server implementation.
- **Configuration via smithery.ai**
