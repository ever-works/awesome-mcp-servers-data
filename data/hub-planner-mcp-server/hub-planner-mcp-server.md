# HUB Planner MCP Server

**Category:** Project Management MCP Servers  
**Brand:** HUB Planner  
**Slug:** `hub-planner-mcp-server`

MCP Server integration for HUB Planner that exposes resource management, planning, scheduling, and timesheet data to AI tools via Pipedream’s MCP server.

---

## Description
The HUB Planner MCP Server connects your HUB Planner account to AI-powered development and productivity tools (such as Cursor, Claude, and Windsurf) using Pipedream’s MCP server. This allows AI tools to access and work with your HUB Planner data for planning, scheduling, and time tracking workflows.

---

## Features
- **MCP-based integration**: Provides an MCP Server interface for HUB Planner, enabling standardized access to HUB Planner data.
- **Resource management access**: Exposes resource management data (e.g., team members, allocations) to AI tools.
- **Planning & scheduling data**: Allows AI tools to read and potentially manipulate planning and scheduling information from HUB Planner.
- **Timesheet data access**: Gives AI workflows access to time-tracking / timesheet information stored in HUB Planner.
- **Pipedream integration**: Runs as a Pipedream MCP server, making it easier to deploy and connect within existing Pipedream-based automations.
- **AI tool connectivity**: Designed to be used by AI tools like Cursor, Claude, and Windsurf to build workflows, assistants, or automations around HUB Planner data.

---

## Integrations
- **HUB Planner** – primary data source for resource management, planning, scheduling, and timesheets.
- **Pipedream MCP Server** – infrastructure layer for the MCP integration.
- **AI / Dev Tools** (examples mentioned):
  - Cursor
  - Claude
  - Windsurf
