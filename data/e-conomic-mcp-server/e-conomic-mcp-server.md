# E-conomic MCP Server

**Brand:** Visma E-conomic  
**Category:** Business & Commerce MCP Servers  
**Tags:** accounting, SME, finance

## Overview
E-conomic MCP Server is an MCP integration for Visma E-conomic, an online accounting application for small and medium-sized organizations. It exposes key accounting operations as MCP tools so chat-based clients and agents can work with accounting and financial data through the MCP protocol.

## Features
- **MCP server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Single URL works for all clients; authentication occurs when adding the server to your application.

- **Visma E-conomic integration**  
  - Connects an existing Visma E-conomic account.  
  - Select and work within a specific client/company context after connecting.  
  - Designed for small and medium-sized organizations using E-conomic for online accounting.

- **Available MCP tools (actions)**  
 1. **Create Invoice**  
    - Creates a new draft invoice in Visma E-conomic.

 2. **Update Customer**  
    - Updates data for an existing customer record.

 3. **List Invoices**  
    - Retrieves a list of invoices from the connected E-conomic account.

 4. **Create Voucher**  
    - Creates a new voucher entry.

 5. **Create Customer**  
    - Creates a new customer record.

 6. **Book Invoice**  
    - Books (posts) an invoice in the accounting system.

- **Client integration**  
  - Can be added to MCP-compatible chat clients.  
  - Works with configuration instructions provided on the linked configuration page.

## Setup
- Connect a Visma E-conomic account and choose the client/company to manage.  
- Copy and use the MCP server URL in your MCP-compatible application.  
- Authenticate within your app when adding the server.
