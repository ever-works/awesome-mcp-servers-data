# Instabot MCP Server

**Category:** AI Integration – MCP Servers  
**Brand:** Instabot  
**Source:** https://mcp.pipedream.com/app/instabot_chatbot_platform

Instabot MCP Server exposes the Instabot chatbot platform’s conversation, integration, and deployment capabilities to MCP-compatible agents and workflows, allowing you to connect Instabot bots with tools that speak the Model Context Protocol.

---

## Features

- **MCP-compatible server endpoint**  
  - Single static MCP server URL for all clients: `https://mcp.pipedream.net/v2`  
  - Can be added to any MCP-capable application or chat client.

- **Instabot chatbot platform integration**  
  - Access Instabot’s conversation features through MCP-based tools and workflows.  
  - Connect Instabot bots to where your users already are (e.g., website, mobile app, email) via MCP-compatible agents.

- **Campaign and deployment support**  
  - Use Instabot within broader AI workflows to support marketing or product campaigns.  
  - Designed to be quickly integrated and launched in existing user touchpoints.

- **Configuration documentation**  
  - A dedicated configuration page (linked from the app page) explains how to add the server to specific chat clients and applications.

- **Pipedream Connect integration**  
  - Hosted and provided via Pipedream Connect infrastructure, allowing use alongside other Pipedream-based integrations.

---

## Getting Started

1. Use the MCP server URL: `https://mcp.pipedream.net/v2`.  
2. Add the server to your MCP-compatible app or chat client.  
3. Authenticate within your client when prompted.  
4. Refer to the linked configuration page for client-specific setup details.
