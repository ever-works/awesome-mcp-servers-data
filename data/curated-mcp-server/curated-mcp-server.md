# Curated MCP Server

## Overview
Curated MCP Server is a Model Context Protocol (MCP) server that connects MCP-compatible clients to Curated, a platform for creating and publishing digest newsletters via email and the web. It allows programmatic access to Curated’s newsletter publishing and link curation capabilities from chat or other MCP-aware applications.

## Features
- **MCP integration with Curated**: Exposes Curated’s newsletter and link curation functionality as MCP tools that can be used by compatible clients.
- **Programmatic newsletter creation workflow**:
  - Collect and manage links intended for a digest newsletter.
  - Select and curate the best links.
  - Add commentary to curated links.
  - Prepare content for publication as a digest newsletter.
- **Multichannel publishing support**:
  - Publish newsletters to email subscribers.
  - Publish the same digest content on the web.
- **Static MCP server endpoint**:
  - Single, reusable server URL for all MCP clients: `https://mcp.pipedream.net/v2`.
  - Authentication handled when adding the server to each client.
- **Client-agnostic usage**:
  - Designed to work with multiple MCP-compatible chat or agent clients.
  - Configuration guidance available per client type.

## Technical Details
- **Server URL**: `https://mcp.pipedream.net/v2`
- **Protocol**: Model Context Protocol (MCP)
- **Host platform**: Pipedream MCP infrastructure
- **Category**: Business & Commerce MCP Servers
- **Intended use cases**:
  - Automated or semi-automated newsletter drafting from within chat/agent environments.
  - Streamlined link collection and curation pipelines.
  - Programmatic publishing of curated content to email and web.
