# Marketo MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Adobe Marketo Engage  
**Slug:** marketo-mcp-server

## Description
Marketo MCP Server is an MCP (Model Context Protocol) server for Adobe Marketo Engage. It exposes Marketo’s marketing automation and customer engagement capabilities to MCP-compatible applications via a static MCP endpoint, allowing you to integrate Marketo functions into chat-based or other MCP-enabled clients.

## Features
- **MCP-compatible server**
  - Provides a Model Context Protocol endpoint for Adobe Marketo Engage.
  - Static MCP server URL usable across different clients.
- **Adobe Marketo Engage integration**
  - Access to Marketo’s marketing automation features through MCP.
  - Supports workflows related to attracting, segmenting, and nurturing customers.
  - Can be used to incorporate campaign and engagement data into MCP-aware tools.
- **Single static endpoint**
  - Server URL: `https://mcp.pipedream.net/v2` (same for all clients).
  - Authentication is handled when adding the server within each client.
- **Client-agnostic usage**
  - Designed to be added to various chat or MCP-compatible applications.
  - Central configuration model via the MCP server URL.

## Usage
- Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server in your MCP-compatible or chat client.
- Authenticate when prompted by the client during setup.
- Optionally refer to the full configuration documentation on the provider’s configuration page.
