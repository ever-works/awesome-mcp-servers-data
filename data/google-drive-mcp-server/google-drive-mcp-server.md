# Google Drive MCP Server

**Category:** File Management MCP Servers  
**Brand:** Google  
**Source:** https://mcp.pipedream.com/app/google_drive  
**MCP Endpoint:** `https://mcp.pipedream.net/v2`

Google Drive MCP Server is an MCP-compatible service that lets AI agents and MCP-enabled applications interact with Google Drive: storing, accessing, and managing files and shared drives through Pipedream MCP.

---

## Features

### MCP Integration
- Static MCP server URL (`https://mcp.pipedream.net/v2`) usable from any compatible client.
- Authentication handled when adding the server to your MCP-enabled application.
- Works with multiple chat or agent clients (configured via the app’s MCP configuration page).

### Google Drive File & Folder Management
- **Upload File**: Upload files to Google Drive.
- **Update File**: Update a file’s metadata and/or content.
- **Move File**: Move a file from one folder to another.
- **Move File to Trash**: Move a file or folder to the trash.
- **Create New File From Template**: Create a new Google Docs file from a template, with optional placeholder replacement.

### Comments & Replies
- **Update Comment**: Modify the content of an existing comment.
- **Reply to Comment**: Add a reply to an existing comment.
- **Update Reply**: Update an existing reply on a specific comment.
- **List Replies**: List replies associated with a specific comment.
- **Resolve Comment**: Mark a comment as resolved.

### Access & Sharing
- **Resolve Access Proposals**: Accept or deny access requests to files or folders in Google Drive.

### Shared Drives
- **Update Shared Drive**: Update the configuration or metadata of an existing shared drive.
- **Search for Shared Drives**: Search for shared drives using query options.

> Note: The server exposes **37 actions** in total as tools. Only those explicitly listed in the provided content are detailed above; additional actions exist but are not enumerated in the source text.
