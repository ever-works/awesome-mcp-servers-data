# Zoho Books MCP Server

**Category:** Business & Commerce MCP Servers  
**Tags:** accounting, invoicing, small-business

## Overview
Zoho Books MCP Server is an MCP-compatible integration that lets you connect a Zoho Books account and perform online accounting operations—such as managing invoices, expenses, sales orders, items, contacts, customers, employees, and payments—directly from MCP-driven workflows.

The server is accessed via a static MCP URL:

```text
https://mcp.pipedream.net/v2
```

## Features

### MCP Integration & Configuration
- Static MCP server URL that works for all Zoho Books clients.
- Authentication handled when adding the server to your MCP-compatible application.
- Designed to be used from various chat or MCP clients via configuration.

### Accounting & Commerce Actions (Tools)
15 actions are available as MCP tools, including:

1. **Update Sales Order**  
   - Update an existing sales order in Zoho Books.

2. **Make API Call**  
   - Perform an arbitrary call to the Zoho Books API.
   - Useful for accessing endpoints or operations not covered by predefined actions.

3. **List Invoices**  
   - Retrieve a list of all invoices.

4. **List Expenses**  
   - Retrieve a list of all expenses.

5. **List Contacts**  
   - List all contacts for a given `organization_id`.

6. **Get Item**  
   - Fetch details of an existing item.

7. **Get Invoice**  
   - Fetch details of a specific invoice.

8. **Delete Contact**  
   - Delete an existing contact.

9. **Create Sales Order**  
   - Create a new sales order.

10. **Create Item**  
    - Create a new item in your Zoho Books account.

11. **Create Invoice**  
    - Create an invoice for a customer.

12. **Create Estimate**  
    - Create a new estimate (quote).

13. **Create Employee**  
    - Create an employee record for expense-related workflows.

14. **Create Customer**  
    - Create a new customer record.

15. **Create Customer Payment**  
    - Create a new payment associated with a customer.
