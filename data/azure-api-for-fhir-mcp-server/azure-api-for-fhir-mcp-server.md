Managed MCP server that connects MCP-compatible clients to the Azure API for FHIR, a standards-based healthcare data platform.

## Features

- Connects MCP clients to Azure API for FHIR
- Uses a **static MCP server URL** that works across compatible clients: `https://mcp.pipedream.net/v2`
- Authentication is handled when adding the server within your client/application
- Designed for healthcare data scenarios using the FHIR standard
- Hosted as part of the Pipedream Connect MCP server ecosystem

## Integration & Usage

- Add the MCP server URL to any supported MCP-compatible chat or tooling client
- Configuration details and client-specific setup instructions are available on the linked configuration page (external to this summary)
