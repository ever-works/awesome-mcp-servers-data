# Papyrs MCP Server

**Category:** Content Management MCP Servers  
**Tags:** intranet, wiki, collaboration

Papyrs MCP Server connects a Papyrs intranet / internal wiki to AI tools via the Model Context Protocol (MCP), enabling AI-powered interactions with your internal pages, forms, and files.

---

## Overview

Papyrs MCP Server is an integration that exposes Papyrs intranet and internal wiki capabilities—such as pages, forms, files, and collaborative content—through MCP tools. It is designed to work with AI-enabled development and assistant environments.

It uses Pipedream’s MCP server to link a Papyrs account with AI tools including Cursor, Claude, and Windsurf.

---

## Features

- **Papyrs Integration via MCP**  
  - Connects your Papyrs account to the Model Context Protocol ecosystem.  
  - Makes intranet and internal wiki resources available as MCP tools.

- **Intranet & Wiki Access**  
  - Exposes Papyrs pages for use within AI tools.  
  - Supports internal wiki-style documentation access through MCP.

- **Forms & Files Exposure**  
  - Provides access to Papyrs forms via MCP tools (e.g., for reading or interacting through AI).  
  - Makes Papyrs files available to connected AI tools.

- **Collaboration Content Integration**  
  - Surfaces collaborative content stored in Papyrs (pages, documents, shared resources) to AI assistants.

- **Pipedream MCP Server Usage**  
  - Runs on Pipedream’s MCP server infrastructure.  
  - Uses Pipedream as the bridge between Papyrs and AI environments.

- **AI Tool Compatibility**  
  - Can be connected to AI tools that support MCP, including:  
    - Cursor  
    - Claude  
    - Windsurf
