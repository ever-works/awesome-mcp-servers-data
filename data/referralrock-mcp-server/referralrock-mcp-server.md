# ReferralRock MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** ReferralRock  
**Slug:** `referralrock-mcp-server`

An MCP Server integration for ReferralRock that allows MCP‑compatible tools to orchestrate and measure word‑of‑mouth and referral programs.

---

## Features

- MCP-compatible server for integrating ReferralRock into MCP‑enabled clients and tools.
- Supports orchestration of referral and word‑of‑mouth marketing programs via MCP.
- Enables measurement and tracking of referral program performance from MCP-compatible interfaces.
- Uses a single static MCP server URL for all clients:  
  - `https://mcp.pipedream.net/v2`
- Authentication handled when adding the server in your MCP client or application.
- Documentation and setup details available via a configuration page (through the hosting platform).

---

## Technical Details

- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Hosting / Provider:** Powered by Pipedream Connect.
