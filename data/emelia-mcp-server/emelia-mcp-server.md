# Emelia MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Emelia  
**Slug:** emelia-mcp-server

## Overview
Emelia MCP Server is an integration endpoint for MCP-compatible clients that connects to Emelia’s B2B growth marketing and outreach platform. It enables automation and management of B2B growth marketing and outreach workflows through Emelia via an MCP server hosted on Pipedream.

## MCP Server URL
- Static MCP server endpoint (works for all MCP clients):  
  `https://mcp.pipedream.net/v2`
- Authentication is performed within the client when adding the server.

## Features
- **MCP-compatible server**: Exposes Emelia’s functionality through the Model Context Protocol so MCP clients (e.g., AI/chat tools) can interact with Emelia.
- **Growth marketing workflow integration**: Designed for growth marketing teams and B2B automation use cases, enabling orchestration of outreach and marketing workflows via MCP.
- **B2B outreach automation**: Supports automation and management of B2B outreach processes through Emelia’s platform (e.g., email-based campaigns and related outreach tasks, as implied by the platform focus).
- **Static, reusable endpoint**: Single static URL for all clients, simplifying configuration across different MCP-compatible tools.
- **Client-agnostic setup**: Can be added to any MCP-capable chat or automation client that supports custom MCP servers.

## Getting Started
- Use the static server URL `https://mcp.pipedream.net/v2` when configuring the MCP server in your client.
- Authenticate within your MCP client as prompted during server setup.
- Optional: Refer to the platform’s full configuration documentation (via the "Configuration" page on the hosting site) for detailed client-specific setup steps.
