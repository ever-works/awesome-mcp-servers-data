# Flash MCP Server (by Velora AI)

**Category:** Business & Commerce – MCP Servers  
**Source:** https://mcp.pipedream.com/app/flash_by_velora_ai

Flash MCP Server provides access to Flash, an AI copilot designed for product and growth leaders, via the Model Context Protocol (MCP).

---

## Overview

Flash (by Velora AI) is an AI copilot focused on supporting product and growth leaders. Through this MCP server, it can be integrated into compatible chat or MCP-enabled clients using a static endpoint hosted by Pipedream.

---

## Features

- **AI copilot for product and growth leaders**  
  - Designed to assist roles focused on product management and growth/analytics.  
  - Accessible via the Model Context Protocol for use in compatible tools and chat clients.

- **Static MCP server endpoint**  
  - Single URL works across MCP-compatible clients:  
    - `https://mcp.pipedream.net/v2`
  - Authentication is performed when adding the server to the chosen application.

- **Client-agnostic integration**  
  - Can be added to different chat clients that support MCP.  
  - Additional configuration details are available through a dedicated configuration page (referenced on the source site).

- **Hosted by Pipedream**  
  - MCP server is powered by Pipedream Connect infrastructure.  
  - Uses Pipedream’s standard terms and privacy policy.

---

## Configuration & Integration

- **MCP Server URL**: `https://mcp.pipedream.net/v2`  
- **Setup**: Add this URL as an MCP server in your MCP-compatible client and authenticate within that client.  
- **Further configuration**: Additional setup steps are provided on the Pipedream configuration page linked from the source site.
