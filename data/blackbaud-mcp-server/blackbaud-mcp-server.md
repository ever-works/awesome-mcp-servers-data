# Blackbaud MCP Server

**Category:** Business & Commerce – MCP Servers  
**Tags:** nonprofit, fundraising, CRM

## Overview
Blackbaud MCP Server is an MCP-compatible integration that connects Blackbaud’s social good software solutions and data to MCP-based agents and tools via a static MCP server endpoint.

## MCP Server Endpoint
- **Server URL:** `https://mcp.pipedream.net/v2`  
- This static URL is used for all clients; authentication occurs when adding the server to your application.

## Usage
- Add the MCP server URL in your MCP-compatible chat client or application.
- Follow your client’s MCP configuration flow to authenticate and connect.
- Additional setup details are available on the platform’s configuration page (not specific to Blackbaud).

## Features
- MCP-compatible server endpoint for integrating Blackbaud with agents and tools.
- Single static URL for all clients, simplifying configuration.
- Authentication handled during server addition in the client/app.
- Built and hosted via Pipedream Connect infrastructure.
