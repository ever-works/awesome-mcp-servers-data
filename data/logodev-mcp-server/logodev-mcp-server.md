# Logo.dev MCP Server

## Description
Logo.dev MCP Server connects your Logo.dev account to Model Context Protocol (MCP)–compatible AI tools, enabling programmatic access to Logo.dev’s logo generation and logo API features.

## Features
- Programmatic access to Logo.dev’s logo generation capabilities
- Access to Logo.dev’s logo API from MCP-compatible clients
- Integration via Pipedream’s MCP server
- Works with AI tools that support MCP, including:
  - Cursor
  - Claude
  - Windsurf
- Suitable for branding and image-generation workflows
