## Overview
Official Tatum MCP for unified blockchain access.

## Features
- RPC for EVM, Bitcoin, Solana, etc.
- Wallet creation and management
- Transaction signing/broadcast
- NFT mint/deploy
- KYT/AML compliance checks

## Use Cases
- Cross-chain DeFi bots
- Multi-chain NFT minting/portfolio mgmt
- Payment gateways

## Comparisons
Vs Solana MCP: Tatum supports 100+ chains vs Solana-only; broader but less optimized for high TPS.
