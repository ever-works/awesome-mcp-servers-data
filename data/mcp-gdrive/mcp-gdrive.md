# mcp-gdrive

**Category:** Document Management MCP Servers  
**Tags:** google-drive, google-sheets, cloud-storage

## Overview
mcp-gdrive is a Model Context Protocol (MCP) server that allows language models and MCP-compatible clients to access Google Drive files and edit Google Sheets. It enables reading and manipulating cloud documents stored in Google Drive through a standardized MCP interface.

- **License:** MIT  
- **Source:** https://github.com/isaacphi/mcp-gdrive

## Features
- **Google Drive integration**  
  - Read files stored in Google Drive via MCP.  
  - Access cloud documents from MCP-compatible tools.

- **Google Sheets editing**  
  - Edit Google Sheets documents through the MCP server.  
  - Programmatic manipulation of spreadsheet content by LLMs.

- **Model Context Protocol (MCP) support**  
  - Exposes Google Drive and Sheets operations as MCP tools/endpoints.  
  - Designed for use with MCP-enabled LLM clients.

- **Cloud document access**  
  - Enables LLMs to interact with cloud-stored files rather than only local data.  
  - Centralized access point to Drive/Sheets within an MCP ecosystem.
