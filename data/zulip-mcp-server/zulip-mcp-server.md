# Zulip MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Zulip

An MCP server integration for Zulip that exposes Zulip’s threaded team chat features via the Model Context Protocol (MCP), enabling message and stream management from compatible MCP clients.

---

## Features

- **MCP Integration Endpoint**  
  - Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`  
  - Authentication handled when adding the server in your MCP‑compatible application.

- **Zulip Chat Capabilities (via MCP)**  
  - Access to Zulip’s threaded team chat model.  
  - Supports real-time messaging with an email-style threading structure.  
  - Designed to help users focus on relevant conversations and ignore irrelevant ones.

- **Client-Agnostic Setup**  
  - Same MCP server URL works across different MCP clients.  
  - Configuration guided by selecting your chat client or using the general configuration documentation.

- **Hosted by Pipedream Connect**  
  - MCP server is operated and delivered through Pipedream’s Connect infrastructure.

---

## Setup & Configuration

- Use the static MCP server URL: `https://mcp.pipedream.net/v2`.  
- Add this server URL in your MCP-compatible chat or AI client.  
- Authenticate within your client when prompted.  
- Optional: Refer to the platform’s full configuration page for client-specific steps.
