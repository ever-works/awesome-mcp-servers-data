# Blazemeter MCP Server

**Category:** Testing & Debugging Tools  
**Tags:** performance-testing, load-testing, quality-assurance

## Overview
Blazemeter MCP Server is an MCP server integration for Blazemeter that allows MCP-compatible tools and clients to run open source–based performance tests against websites, applications, and APIs at large scale using the Blazemeter platform.

## Features
- MCP server integration for Blazemeter
- Supports open source–based performance testing workflows
- Enables large-scale load and performance tests on:
  - Websites
  - Web or mobile applications
  - APIs
- Provides a single static MCP server URL usable from multiple clients
- Compatible with various MCP-enabled chat clients and tools (added as an external MCP server)

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Integration host:** Powered by Pipedream Connect

## Usage
- Copy the MCP server URL
- Add the server URL to any compatible MCP client or chat application
- Authenticate within the client when prompted
