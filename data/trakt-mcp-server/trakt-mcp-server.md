# Trakt MCP Server

## Overview
The Trakt MCP Server is an MCP-compatible service that connects to Trakt, enabling MCP agents and chat-based clients to access and work with TV and movie watch data tracked via Trakt.

## Features
- Integrates with Trakt to access TV and movie watch data.
- Uses a static MCP server URL (`https://mcp.pipedream.net/v2`) that works across supported MCP clients.
- Authentication occurs when adding the server to a compatible application or chat client.
- Can be added to various chat clients for conversational access to Trakt data (via MCP protocol).
- Centralized configuration via a dedicated configuration page (for client-specific setup instructions).

## Configuration
- Connect your Trakt account, then select your MCP-compatible client to start.
- Use the static MCP server URL:
  - `https://mcp.pipedream.net/v2`
- Follow client-specific instructions (from the configuration page) to add the Trakt MCP server to your application.
