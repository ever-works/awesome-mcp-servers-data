# Zenler MCP Server

Zenler MCP Server is an MCP integration for the Zenler all‑in‑one course creation, delivery, and marketing platform. It allows AI clients and MCP-compatible applications to access and manage Zenler course and marketing workflows via a standardized MCP endpoint.

---

## Overview
- **Type:** MCP server / integration
- **Platform:** Zenler (online course creation and marketing)
- **Category:** Learning & Education; Business & Commerce MCP Servers
- **Provider:** Pipedream
- **MCP Base URL:** `https://mcp.pipedream.net/v2`

---

## Features

- **Standard MCP Endpoint**
  - Uses a static MCP server URL: `https://mcp.pipedream.net/v2`.
  - Same URL works for all Zenler clients; authentication is handled when adding the server to a client app.

- **Zenler Platform Integration**
  - Connects to Zenler’s course creation, delivery, and marketing environment.
  - Designed to let AI tools and MCP-compatible clients interact with Zenler course content and marketing workflows (e.g., accessing, managing, or automating operations exposed as MCP tools/actions).

- **Account Connection Flow**
  - “Configure Zenler” step to connect a Zenler account.
  - After connection, users can select the appropriate client (chat or other MCP-capable application) to begin using the tools.

- **Client-Agnostic Setup**
  - Documentation and configuration guidance provided per chat client / MCP client.
  - Users can follow client-specific instructions to add the MCP server to their app.

- **Tooling via MCP**
  - Exposes Zenler-related “tools” (actions) that can be invoked programmatically from MCP clients.
  - Tools are loaded dynamically from the Zenler integration (shown as “Loading actions… / Loading available tools…” in the UI), enabling course and marketing operations through MCP.

- **Centralized Configuration Resources**
  - Linked configuration page (Pipedream Configuration) for detailed setup and management of the MCP server.
