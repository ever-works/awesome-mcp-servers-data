# Feathery MCP Server

**Category:** Business & Commerce – MCP Servers  
**Tags:** forms, workflow, automation

## Overview
Feathery MCP Server is an MCP (Model Context Protocol) server integration that lets tools and chat clients programmatically interact with Feathery’s form and survey experiences. It is delivered via Pipedream and exposes Feathery form capabilities to MCP-compatible applications through a single static server URL.

- **Provider / Brand:** Feathery (via Pipedream)
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP server integration for Feathery**  
  Exposes Feathery’s forms and surveys to MCP-compatible clients so they can be used programmatically.

- **Static server endpoint**  
  Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) that works for all Feathery clients, simplifying configuration.

- **Authentication at application level**  
  Authentication is performed when adding the server to your application, allowing multiple clients to share the same base MCP URL while using their own credentials.

- **Chat client integration**  
  Designed to be added to chat-based or MCP-enabled applications, enabling interaction with Feathery forms directly from those clients.

- **Pipedream-based hosting**  
  Hosted and delivered via Pipedream’s integration platform, benefiting from its infrastructure and configuration flow.

- **Configurable client connection**  
  Requires connecting a Feathery account and selecting a client within the Pipedream app before using the tools.

- **Tool-based interaction**  
  Provides a set of MCP tools / actions (loaded dynamically) that allow applications to perform operations related to Feathery forms and surveys. (Specific tools are not listed in the provided content.)
