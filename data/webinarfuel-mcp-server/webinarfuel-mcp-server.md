# WebinarFuel MCP Server

## Overview
WebinarFuel MCP Server is an MCP-compatible integration endpoint for WebinarFuel’s all‑in‑one automated webinar software. It enables you to connect WebinarFuel to MCP-based chat clients and workflows via a static server URL, handling authentication at the application level.

- **Name:** WebinarFuel MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Brand:** WebinarFuel  
- **Integration Platform:** Pipedream Connect  
- **Primary Use Cases:** Webinar automation, marketing workflows, integrations via MCP

## Features
- **MCP-Compatible Server Endpoint**  
  - Provides a static MCP server URL for all clients: `https://mcp.pipedream.net/v2`.
  - Designed to be added as a server within MCP-capable chat or agent clients.

- **Authentication at Client Setup**  
  - Uses a single shared server URL; individual users authenticate when adding the server to their own application/client.

- **Chat Client Integration**  
  - Can be added to supported chat or agent clients that understand MCP servers.  
  - Guided setup is available per chat client type (via the “Select your chat client to learn how to get started” flow on the source site).

- **Configuration Documentation**  
  - Full configuration details are available on a dedicated Configuration page (linked from the source content).

- **Pipedream Connect Backing**  
  - Runs on top of Pipedream Connect infrastructure, allowing it to participate in broader Pipedream-powered workflows and automations.
