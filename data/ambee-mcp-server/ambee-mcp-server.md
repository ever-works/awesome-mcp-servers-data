# Ambee MCP Server

An MCP (Model Context Protocol) server that exposes Ambee climate and environmental intelligence data—such as air quality and weather—to AI agents via a static MCP endpoint.

---

## Overview

- **Type:** MCP Server (data-access integration)
- **Provider / Brand:** Ambee (hosted via Pipedream)
- **Category:** Data Access & Integration – MCP Servers
- **Use Cases:**
  - Enrich AI agents with real-time climate and environmental context
  - Retrieve air quality and weather data programmatically inside MCP-compatible chat clients or applications

---

## Features

- **Static MCP Server URL**
  - Single static endpoint for all clients: `https://mcp.pipedream.net/v2`
  - Same URL can be reused across different MCP-compatible chat clients.

- **MCP Integration**
  - Designed to be added as an MCP server to AI/chat applications.
  - Supports authentication at the time of adding the server to the application.

- **Climate & Environmental Intelligence Data Access**
  - Access to Ambee data (as described):
    - Climate intelligence
    - Environmental intelligence
    - Air quality data
    - Weather data
  - Intended for consumption by AI agents for context-aware responses.

- **Client-Agnostic Setup**
  - Works with multiple “chat clients” that support MCP.
  - Documentation/steps provided per client type (via “Select your chat client to learn how to get started”).

- **Configuration Guidance**
  - Central configuration page (linked as “Configuration page”) for detailed setup.
  - Flow includes connecting an account and selecting a client.

- **Hosted by Pipedream**
  - Served and managed as an integration on Pipedream’s platform.
