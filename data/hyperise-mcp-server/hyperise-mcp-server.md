**Category:** Media Processing MCP Servers

## Overview
Hyperise MCP Server provides programmatic access, via the Model Context Protocol (MCP), to Hyperiseâ€™s personalized image generation and marketing technology (MarTech) integrations. It enables dynamic, personalized images for outreach campaigns and website landing pages from within MCP-compatible clients.

## Features
- **Personalized image generation**
  - Create personalized images suitable for outreach sequences and landing pages.
  - Use dynamic fields to tailor images to individual recipients or audiences.

- **MCP-based programmatic access**
  - Exposes Hyperise capabilities through a standard MCP server endpoint.
  - Accessible from any MCP-compatible chat or AI client.

- **Static MCP server endpoint**
  - Single static MCP URL for all clients: `https://mcp.pipedream.net/v2`.
  - Authentication is handled when adding the server to the client/application.

- **MarTech integrations**
  - Integrates with thousands of marketing and sales tools (MarTech ecosystem).
  - Designed to plug into common outreach platforms and website builders.
  - Supports use in email outreach, sales sequences, and landing page experiences.

- **Usage in outreach and landing pages**
  - Embed personalized images in outbound campaigns to increase relevance.
  - Use personalized visuals on website landing pages for tailored experiences.

- **Client-agnostic setup**
  - Works with multiple MCP-compatible chat clients.
  - Central configuration documentation available via a shared configuration page.

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Integration pattern:** Add as an MCP server to your chosen client, then authenticate within that client.
