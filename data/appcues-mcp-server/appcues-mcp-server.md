# Appcues MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Appcues  
**Slug:** `appcues-mcp-server`  
**Source:** https://mcp.pipedream.com/app/appcues

## Description
Appcues MCP Server exposes Appcues’ product adoption and user onboarding capabilities through the Model Context Protocol (MCP). It enables chat-based and automated workflows to interact with Appcues for building and managing in-product guidance, onboarding flows, and customer engagement experiences without requiring direct developer implementation.

## Features
- **MCP-based integration**: Provides an MCP-compliant server endpoint for connecting Appcues to MCP-compatible clients and agents.
- **Static MCP server URL**: Uses a single static URL usable by any MCP client:
  - `https://mcp.pipedream.net/v2`
- **Per-client authentication**: Authentication is performed when adding the server in each client or application, keeping the base URL the same across environments.
- **Product onboarding workflows**: Exposes Appcues functionality focused on creating and managing user onboarding experiences.
- **Product adoption tools**: Surfaces capabilities to measure and improve product adoption via Appcues within MCP-driven automations.
- **Customer engagement**: Enables use of Appcues guidance and engagement features from within MCP-aware chat clients and tools.
- **In-context guidance integrations**: Supports building in-context help and guidance flows controlled via MCP workflows.
- **Works with multiple clients**: Can be added to various MCP-compatible chat clients; configuration details are provided per client.
- **Central configuration documentation**: A full configuration page is available (via Pipedream) to set up and manage the MCP server.

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Protocol:** Model Context Protocol (MCP)  
- **Host platform:** Pipedream Connect
