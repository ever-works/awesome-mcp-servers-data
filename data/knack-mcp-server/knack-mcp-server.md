# Knack MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Pipedream  
**Website:** https://mcp.pipedream.com/app/knack

## Description
Knack MCP Server connects Knack’s business app platform APIs to MCP-compatible tools via Pipedream. It provides a unified MCP endpoint you can add to your chat or MCP client to access and automate data and workflows built on Knack.

## MCP Server Endpoint
- **Server URL:** `https://mcp.pipedream.net/v2`  
  - Static URL used by all clients.
  - Authentication is performed when adding the server to your application.

## Features
- **MCP-compatible server** that exposes Knack-related functionality through the MCP protocol.
- **Static, shared endpoint** (`https://mcp.pipedream.net/v2`) for all clients, simplifying configuration.
- **Authentication at client setup time**, not per-URL, allowing secure access to Knack via your own credentials.
- **Integration via Pipedream Connect**, leveraging Pipedream’s infrastructure to connect Knack’s business app platform APIs to tools that support MCP.
- **Usable from multiple chat / MCP clients** by adding the same server URL to each client.
- **Central configuration documentation** via the Pipedream MCP Configuration page (covers how to add and use the server across supported clients).

## Usage
1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add it as a new MCP server in your chat or MCP client.
3. Authenticate when prompted to link your Knack / Pipedream account.
4. Start using Knack data and automation capabilities from within your MCP-enabled tools.
