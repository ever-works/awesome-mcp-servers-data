## Description
An MCP server integration for PostGrid Verify that enables applications and chat clients to perform address autocomplete, verification, validation, and standardization according to local postal standards. It is delivered via Pipedream Connect and is configured as a static MCP endpoint that you authenticate against from compatible clients.

## Features
- **Address Autocomplete**  
  - Suggests address completions as users type.  
  - Helps reduce input errors and speeds up address entry.

- **Address Verification**  
  - Confirms that an address exists and can be used for mail delivery.  
  - Targets improved deliverability and reduced undeliverable mail.

- **Address Validation**  
  - Checks addresses for correctness and completeness.  
  - Identifies invalid or incomplete address components.

- **Address Standardization**  
  - Normalizes addresses to local postal standards and formats.  
  - Produces consistent address records suitable for mailing and databases.

- **Postal Deliverability Optimization**  
  - Designed to improve mailing deliverability and reduce return mail.  
  - Helps maintain up-to-date and accurate address records.

- **Static MCP Endpoint**  
  - Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) compatible with all supported MCP clients.  
  - Authentication occurs when adding the server to your application or chat client.

- **Integration via Chat Clients / Apps**  
  - Can be added to compatible chat clients or applications as an MCP server.  
  - Configuration instructions available via the linked configuration page (per-client setup details).

- **Pipedream Connect Integration**  
  - Delivered and managed through Pipedream Connect infrastructure.

## Configuration
- **MCP Server URL:** `https://mcp.pipedream.net/v2` (static for all clients).  
- **Setup:** Add this URL as an MCP server in your MCP-compatible application or chat client, then authenticate as prompted.
