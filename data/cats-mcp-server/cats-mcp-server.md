# CATS MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** CATS  
**Source:** https://mcp.pipedream.com/app/cats

## Description
CATS MCP Server exposes the CATS Applicant Tracking System (ATS) and recruiting platform through the Model Context Protocol (MCP), enabling integration with compatible chat or AI clients.

## Features
- Provides MCP-based access to CATS applicant tracking and recruiting functionality (via Pipedream Connect)
- Uses a single static MCP server endpoint for all clients
- Supports authentication at the time you add/configure the server in your MCP-compatible application
- Can be added to different chat or AI clients that support MCP
- Central configuration and documentation available via a dedicated configuration page

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2` (static for all clients)
- **Provider:** Pipedream Connect
