### Overview
The Undetectable AI MCP Server is an MCP-compatible service (hosted via Pipedream Connect) that exposes Undetectable AIâ€™s capabilities for AI text detection and content optimization. It is designed to help applications analyze text for AI-generated characteristics and adjust content so it is less likely to be flagged as AI-written.

### Features
- **MCP-compatible server**
  - Provides a static MCP server endpoint: `https://mcp.pipedream.net/v2`
  - Usable from any compatible MCP client or chat application.

- **AI text detection**
  - Analyzes text to determine how likely it is to be flagged as AI-written.

- **Content optimization for AI detection**
  - Supports optimizing or adjusting text to reduce the likelihood of being identified as AI-generated.

- **Client-agnostic configuration**
  - Single static URL works for every MCP client.
  - Authentication is handled when adding the server to a client or application.

- **Pipedream integration**
  - Hosted and delivered via Pipedream Connect infrastructure.
  - Configuration and usage details available on the Pipedream Configuration page.

### Integration & Usage
- **Server URL:** `https://mcp.pipedream.net/v2`
- **Setup:**
  - Add the MCP server URL to a supported chat client or MCP-compatible tool.
  - Authenticate within the client during setup.
  - Refer to the Pipedream Configuration page for client-specific instructions.
