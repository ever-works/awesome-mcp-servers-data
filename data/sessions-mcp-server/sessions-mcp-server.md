# Sessions MCP Server

**Description**

Sessions MCP Server is an MCP-compatible server that allows MCP clients to schedule and run online meetings and webinars using the Sessions platform via a unified MCP endpoint.

**Server URL**

- MCP server base URL (static, for all clients):
  - `https://mcp.pipedream.net/v2`

**Integration & Usage**

- Designed to be added as an MCP server within compatible chat or MCP client applications.
- Authentication is performed when adding/configuring the server in the client.
- Works with multiple MCP clients through the same static endpoint.

**Features**

- MCP-compliant server for the Sessions platform.
- Supports online meeting management through MCP clients.
- Supports webinar management through MCP clients.
- Single, static MCP URL usable across different clients.
- Authentication handled at the time of adding the server to the client.
