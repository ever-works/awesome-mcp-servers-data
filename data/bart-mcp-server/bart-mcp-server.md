# BART MCP Server

**Description**  
An MCP server that exposes Bay Area Rapid Transit (BART) train service information for use within MCP-compatible chat or agent clients, enabling queries about BART service across the San Francisco Bay Area.

**Source**  
<https://mcp.pipedream.com/app/bart>

**Category**  
Data Access & Integration – MCP Servers

**Tags**  
- transportation  
- open-data  
- api

---

## Features

- Provides access to BART (Bay Area Rapid Transit) train service information.  
- Covers routes and services throughout the San Francisco Bay Area.  
- Exposed as an MCP (Model Context Protocol) server compatible with MCP-enabled clients.  
- Uses a single static base URL for all clients: `https://mcp.pipedream.net/v2`.  
- Authentication is handled when adding/configuring the server within your application.  
- Documentation and setup guidance available via the Pipedream Configuration page.  
- Hosted and operated via Pipedream Connect.

---

## Configuration

- **MCP Server URL**: `https://mcp.pipedream.net/v2`  
- Add this URL as an MCP server in your compatible chat or agent client, then follow that client’s flow to authenticate.  
- Additional configuration instructions are available on Pipedream’s configuration page (linked from the app page).
