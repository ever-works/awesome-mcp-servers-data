# Core MCP Server

MCP server that provides a starting point for using AWS MCP servers through a dynamic proxy server strategy based on role-based environment variables.

## Features

### Planning and Orchestration
- Acts as a core orchestration layer for AWS-focused MCP servers.
- Uses a dynamic proxy server strategy to route and coordinate requests.
- Supports role-based environment variables to control how different MCP servers are accessed and composed.
- Designed as a starting point for building workflows that span multiple AWS MCP servers.
