# Weglot MCP Server

**Category:** Content Management MCP Servers  
**Brand:** Weglot  
**Slug:** `weglot-mcp-server`

An MCP (Model Context Protocol) server integration that connects your Weglot account to AI tools, enabling website translation and localization workflows directly from compatible AI environments.

![Weglot interface](https://weglot.com/wp-content/uploads/2023/10/Weglot_Hero@2x.png)

---

## Features

- **MCP Server Integration**  
  - Implements the Model Context Protocol to expose Weglot capabilities to AI tools.

- **Weglot Account Connectivity**  
  - Connects directly to your existing Weglot account through Pipedream’s MCP server.

- **AI Tool Integration**  
  - Designed to be used with AI development and IDE assistants such as:
    - Cursor
    - Claude
    - Windsurf
  - Enables these tools to interact with your Weglot project context.

- **Website Translation & Localization Context**  
  - Provides access to Weglot’s translation and localization features via MCP (e.g., for website content translation workflows).

---

## Use Cases

- Integrate website translation tasks into AI-assisted coding or content workflows.
- Allow AI tools to reference or manage translation-related context from a Weglot project.
