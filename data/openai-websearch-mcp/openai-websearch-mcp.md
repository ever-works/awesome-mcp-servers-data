## Overview
openai-websearch-mcp is a Python-based Model Context Protocol (MCP) server that wraps and exposes OpenAI’s built-in `web_search` tool to MCP-enabled clients. It enables applications and MCP-compatible agents to perform OpenAI-powered web searches through a standardized protocol interface.

## Features
- **MCP server implementation**
  - Implements an MCP-compatible server exposing web search as a tool.
  - Designed to integrate with MCP-enabled clients and frameworks.

- **OpenAI web_search integration**
  - Wraps OpenAI’s built-in `web_search` tool.
  - Provides a consistent interface to perform web searches via OpenAI models.

- **Python-based package**
  - Distributed as a Python package on PyPI: `openai-websearch-mcp`.
  - Includes project configuration via `pyproject.toml`.
  - Contains a dedicated package module under `src/openai_websearch_mcp`.

- **Runtime and compatibility**
  - Supports Python 3.10+.
  - Explicitly marked as “MCP Compatible” for use with Model Context Protocol tooling.

- **Documentation assets**
  - Includes a `docs` directory for project documentation (setup, usage, etc.).

- **Open-source licensing**
  - Released under the MIT License.
