# Mocean API MCP Server

Mocean API MCP Server connects the Mocean communications platform with MCP-compatible clients, enabling programmatic messaging and communication workflows.

## Overview
- **Type:** MCP server integration
- **Category:** Messaging MCP Servers
- **Provider / Brand:** Mocean (via Pipedream Connect)
- **Primary Use Case:** Programmatically send and receive SMS and perform other communication functions via Mocean’s APIs from any MCP-enabled client.

## Features
- **Global SMS Messaging**
  - Send SMS programmatically using Mocean’s communications APIs.
  - Receive SMS programmatically where supported by Mocean.
- **Communication APIs & SDKs**
  - Access Mocean’s API and SDK designed for software developers.
  - Integrate SMS and related communication features into applications.
- **MCP Integration**
  - Exposes Mocean communication capabilities as MCP tools for compatible clients.
  - Single, static MCP server URL usable across different MCP clients.
- **Worldwide Reach**
  - Designed for sending and receiving SMS worldwide (subject to Mocean’s network coverage).

## MCP Server Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Usage Pattern:**
  - Add the static MCP server URL to any supported MCP client.
  - Authenticate during server configuration in your application.
