# Procore MCP Server

An MCP server that surfaces Procore construction project management capabilities—such as projects, documents, and workflows—inside MCP-compatible applications.

## Overview
- **Type:** MCP Server (integration via Pipedream Connect)
- **Category:** Project Management MCP Servers
- **Domain:** Construction project management
- **Base URL:** `https://mcp.pipedream.net/v2` (static for all clients; authentication occurs when adding the server to an application)

## Features
- Exposes Procore construction project management functionality to MCP-compatible clients
- Supports working with:
  - Projects
  - Documents
  - Workflows
- Uses a single, static MCP server URL for all clients
- Can be added to various chat or MCP-enabled applications as a server endpoint

## Usage
- Add the MCP server to an MCP-compatible app using:
  - **Server URL:** `https://mcp.pipedream.net/v2`
  - Authenticate during application setup (per client instructions)
- Configuration details and client-specific setup are available on the provider’s configuration page.
