## DeepSeek MCP Server

### Overview
DeepSeek MCP Server is an open-source Model Context Protocol (MCP) server that connects MCP-compatible clients to DeepSeek’s language models. It also exposes additional API endpoints tailored for MCP clients.

### Features
- **Model Context Protocol server** implementation for use with MCP-compatible clients.
- **Integration with DeepSeek language models**, enabling access to DeepSeek’s LLMs through MCP.
- **Additional API endpoints** exposed for MCP clients beyond basic model access (as defined in the server implementation).
- **Containerized deployment support** via the provided `Dockerfile`.
- **Environment-based configuration** using an example `.env.example` file for setting keys and runtime options.
- **Open-source repository structure** with source code under `src/`, assets in `assets/`, and GitHub workflows in `.github/`.
