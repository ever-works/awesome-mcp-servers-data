## Overview

Appsmith MCP server provides AI-powered access to the open-source low-code platform for building internal tools.

## Features

- Open Source: Self-hosted low-code platform
- 45+ Data Sources: Connect to databases, APIs, and SaaS tools
- Drag-and-Drop UI: Visual application builder
- Custom JavaScript: Add business logic with code
- Version Control: Git-based workflow

## Use Cases

- IDE-integrated app prototyping
- Database admin panels
- Workflow automation tools

## Pricing

Free and open-source (cloud plans available).
