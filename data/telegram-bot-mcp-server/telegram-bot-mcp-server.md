# Telegram Bot MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Telegram  
**Source:** https://mcp.pipedream.com/app/telegram_bot_api

## Description
Telegram Bot MCP Server is an MCP server for the Telegram Bot API that exposes Telegram’s cloud-based, cross-platform, encrypted instant messaging and bot capabilities to MCP-compatible clients.

## Features
- MCP-based access to the Telegram Bot API
- Integrates Telegram’s cloud messaging capabilities into MCP clients
- Supports cross-platform, encrypted instant messaging via Telegram
- Static MCP server URL usable from any compatible client
- Authentication handled when adding the server to your application
- Works with multiple chat / MCP-enabled clients

## MCP Server URL
- Base MCP server URL (static for all clients):
  - `https://mcp.pipedream.net/v2`

## Usage Notes
- Add the static server URL to your MCP-compatible app or chat client.
- Authentication occurs during the server-connection/setup flow in your application.
