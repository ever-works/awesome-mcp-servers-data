# Best of MCP Servers

**Description**

Best of MCP Servers is a GitHub-hosted, automatically ranked and weekly updated curated list of roughly 400 Model Context Protocol (MCP) servers. Projects are grouped into 34 categories and scored using project-quality metrics collected from GitHub and various package managers. The list serves as a centralized, data-driven directory for discovering top MCP servers and their ecosystems.

## Features

- **Curated MCP server list**: Manually and programmatically curated collection of MCP servers and related tooling.
- **Automatic ranking**: Servers are ranked using project-quality metrics derived from GitHub and package managers (e.g., stars, activity, downloads, etc., as available in the source data).
- **Weekly updates**: Rankings and entries are updated on a weekly basis for freshness.
- **Category organization**: ~400 servers are organized into 34 thematic categories for easier discovery and browsing.
- **GitHub & package manager metrics**: Uses data pulled from GitHub and various package registries to inform scoring.
- **History tracking**: Includes a `history` directory and `latest-changes.md` to track how the list and rankings evolve over time.
- **Configuration-driven**: Uses configuration files (e.g., `config` directory, `projects.yaml`) to define and maintain the set of projects and scoring.
- **Open contribution model**: Provides `CONTRIBUTING.md` for users to propose new MCP servers or improvements to existing entries.
- **Open content license**: Distributed under CC-BY-SA-4.0, enabling reuse and remixing with attribution and share-alike.
- **Part of best-of ecosystem**: Generated from the `best-of-lists/best-of` tooling, aligning with other “best-of” curated lists.
