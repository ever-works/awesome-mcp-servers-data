# Brave Search

## Description
Brave Search is an MCP (Model Context Protocol) server that enables privacy-focused web and local search through the Brave Search API, allowing LLMs to perform internet queries without direct browser access.

## Features
- Provides web search capabilities via the Brave Search API
- Supports local search functionality (e.g., more constrained or region-specific queries) through the same API
- Designed to be used as an MCP server for tools and LLM-based applications
