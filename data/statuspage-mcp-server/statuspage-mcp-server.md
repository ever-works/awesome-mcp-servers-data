# Statuspage MCP Server

**Category:** Monitoring  
**Website:** https://mcp.pipedream.com/app/statuspage

An MCP (Model Context Protocol) server that connects Statuspage to MCP-compatible tools and chat clients so they can query and manage real-time incident and status information.

---

## Features

- **MCP-compatible status integration**
  - Exposes Statuspage data through the MCP protocol for use in MCP-enabled tools and chat clients.
  - Designed to let tools query and manage real-time incident and status information from Statuspage.

- **Static MCP server endpoint**
  - Single server URL for all clients: `https://mcp.pipedream.net/v2`.
  - Same endpoint reused across different applications; authentication is handled when adding the server to each app.

- **Multi-client support**
  - Can be added to various MCP-compatible chat clients and applications.
  - Configuration guidance available via the platformâ€™s configuration page (generic MCP setup).

- **Incident and status data access**
  - Built specifically for Statuspage, enabling:
    - Retrieval of current status.
    - Access to incident information.
    - Use of this information inside chat / AI workflows for communication with users.

- **Pipedream Connect integration**
  - Runs on top of Pipedream Connect infrastructure, simplifying connection and configuration.
