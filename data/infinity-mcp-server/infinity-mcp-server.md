# Infinity MCP Server

An MCP (Model Context Protocol) server that connects to Infinity, enabling access to its work and project organization capabilities directly from compatible chat or AI clients.

## Overview
- **Type:** MCP server / integration
- **Category:** Project management MCP servers
- **Use cases:** Project organization, task management, team collaboration, centralized work management via MCP-compatible clients
- **Source / Setup URL:** https://mcp.pipedream.com/app/infinity
- **MCP Endpoint URL:** `https://mcp.pipedream.net/v2`

## Features
- **Infinity integration via MCP**
  - Connects Infinity’s work and project organization features to MCP-compatible applications.
  - Provides access to Infinity data and structure from within chat or AI tools that support MCP.

- **Centralized work organization**
  - Designed to help organize projects, tasks, and related work items in one place.
  - Supports using Infinity as a single workspace for storing and organizing work.

- **Collaboration support**
  - Intended for teams to work together using Infinity’s collaborative features through MCP-enabled clients.

- **Static MCP server URL**
  - Uses a single, static URL (`https://mcp.pipedream.net/v2`) that works across supported clients.
  - Simplifies configuration: the same endpoint can be reused in multiple applications.

- **Client-agnostic connection**
  - Can be added to different MCP-capable chat clients or AI assistants.
  - Configuration details and examples are available via the associated configuration page.

- **Authentication at connection time**
  - Authentication occurs when adding the MCP server to your application (exact method depends on the client and Pipedream/Infinity setup).

## Integration & Configuration
- **How to connect**
  - Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
  - Add it to your MCP-compatible chat client or application.
  - Follow client-specific steps described on the configuration page (linked from the source URL).
