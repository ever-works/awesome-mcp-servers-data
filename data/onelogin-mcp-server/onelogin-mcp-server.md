# OneLogin MCP Server

**Category:** Security & Attestation MCP Servers  
**Brand:** OneLogin  
**Source:** https://mcp.pipedream.com/app/onelogin

## Overview
OneLogin MCP Server is an integration that exposes OneLogin identity and access management capabilities—such as user, role, and application management—to AI agents via the Model Context Protocol (MCP). It is provided as a static MCP endpoint powered by Pipedream Connect.

## Features
- **MCP-based integration**
  - Exposes OneLogin identity and access management (IAM) capabilities to AI agents over MCP.
  - Designed for use with MCP-compatible chat or agent clients.

- **Identity & access management scope**
  - User management (e.g., working with user entities in OneLogin).
  - Role management.
  - Application management.

- **Static MCP server URL**
  - Single static endpoint for all clients: 
    - `https://mcp.pipedream.net/v2`
  - Authentication is performed when adding the server to your application.

- **Client integration**
  - Can be added to various MCP-enabled chat/agent clients.
  - Configuration is documented via a dedicated configuration page (referenced in the source content).

- **Underlying platform**
  - Powered by Pipedream Connect.

## Usage
- Add the MCP server to a compatible chat or agent client using the static URL.
- Authenticate during setup within your chosen client.
- Use the exposed IAM tools (users, roles, applications) within AI workflows that support MCP.
