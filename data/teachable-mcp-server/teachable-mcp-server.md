# Teachable MCP Server

**Category:** Business & Commerce · MCP Servers  
**Brand:** Teachable  
**Source:** https://mcp.pipedream.com/app/teachable

Teachable MCP Server is an MCP server that exposes Teachable’s course and digital product platform to AI tools, enabling course and product management and automation from MCP-compatible clients.

---

## Features

- MCP server interface for Teachable’s platform
- Supports management and automation of:
  - Online courses
  - Digital products
- Accessible via a static MCP server URL usable by any compatible client
- Authentication handled when adding the server to an application
- Compatible with multiple chat / AI clients that support MCP
- Hosted and operated via Pipedream Connect infrastructure

---

## Technical Details

- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
  (Static URL shared across clients; authentication occurs during client setup.)
