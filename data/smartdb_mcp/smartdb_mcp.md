## SmartDB_MCP

**Category:** Database & Messaging MCP Servers  
**Tags:** multi-database, query, SQL

### Description
SmartDB_MCP is a universal database MCP server that can connect to multiple databases simultaneously and expose tools for multi-database operations through the Model Context Protocol (MCP).

### Features
- **Multi-database connectivity**
  - Supports connecting to multiple databases at the same time.
  - Current supported databases (from visible content):
    - MySQL
    - PostgreSQL
    - SQL Server
    - MariaDB
    - DM8
    - Oracle
- **MCP integration**
  - Implements an MCP server that exposes database operations as MCP tools.
  - Enables multi-DB operations through a unified MCP interface.
- **Authentication & security**
  - Supports OAuth 2.0 authentication for database access.
- **Health & monitoring**
  - Database connection health checks.
- **Performance & optimization**
  - SQL optimization capabilities.
  - Index health detection to analyze and monitor index status.
