# Drata MCP Server

An MCP server integration for Drata that surfaces compliance automation capabilities through the MCP interface.

- **Category:** Security & Attestation MCP Servers  
- **Brand:** Drata  
- **Source URL:** https://mcp.pipedream.com/app/drata  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Overview
The Drata MCP Server exposes Drata’s compliance automation features—such as evidence collection, control monitoring, and audit readiness—via a standardized MCP endpoint. The same static MCP server URL is used for all clients, with authentication handled when adding the server to your application.

## Features
- **Standard MCP endpoint**  
  - Single static MCP server URL (`https://mcp.pipedream.net/v2`) for all clients.  
  - Authentication occurs during server setup in your MCP-compatible application.

- **Compliance automation integration**  
  - Surfaces Drata’s compliance automation capabilities through MCP, including:  
    - Evidence collection workflows.  
    - Control monitoring and tracking.  
    - Tools to support audit-readiness processes.

- **Client-agnostic setup**  
  - Works with multiple MCP-compatible chat or AI clients.  
  - Configuration instructions available per client via the “Add the server to your app” flow and the full configuration page.

- **Pipedream Connect-based**  
  - Implemented and hosted via Pipedream Connect infrastructure.

## Setup
- Use the static MCP server URL: `https://mcp.pipedream.net/v2`.  
- Add this server in your MCP-compatible client and complete the authentication flow.  
- Optional: Refer to the provider’s configuration page for detailed, client-specific setup steps.
