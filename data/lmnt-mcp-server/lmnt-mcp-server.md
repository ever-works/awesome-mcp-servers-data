# LMNT MCP Server

MCP Server for LMNT, enabling creation of expressive, human-like speech and custom voices from within MCP-compatible tools.

- **Website / Source**: https://mcp.pipedream.com/app/lmnt
- **Category**: Media Processing MCP Servers
- **Brand**: lmnt
- **Slug**: `lmnt-mcp-server`
- **Tags**: `speech-to-text`, `text-to-speech`, `audio-processing`

## Features

- Provides an MCP (Model Context Protocol) server interface for LMNT.
- Enables creation of expressive, emotive, human-like speech.
- Supports generation of custom voices.
- Designed to be used from any MCP-compatible client or tool.
- Uses a single static MCP server URL for all clients: `https://mcp.pipedream.net/v2`.
- Authentication handled when adding the server to an application.
- Configuration details available via a centralized configuration page (via Pipedream Connect).

## Integration & Usage

- Add the MCP server URL to MCP-compatible chat clients or applications.
- Follow client-specific setup instructions (select chat client to learn how to get started).
- Additional configuration guidance available on the Pipedream Configuration page.
