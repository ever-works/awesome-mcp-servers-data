# Jotform MCP Server

Jotform MCP Server is an MCP server that integrates with Jotform’s online form and fillable PDF platform, enabling automated form creation and response collection from within MCP-capable clients.

- **Website / Source**: https://mcp.pipedream.com/app/jotform
- **Category**: Business & Commerce – MCP Servers
- **Brand**: Jotform
- **Slug**: `jotform-mcp-server`
- **Tags**: forms, surveys, workflow-automation

## Features

- **MCP server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Usable with any compatible MCP client.  
  - Authentication is performed when adding the server to an application.

- **Jotform integration**  
  - Connects MCP clients to Jotform’s online form service.  
  - Supports creating online forms.  
  - Enables collection of form responses (e.g., delivered to email via Jotform).  
  - Integrates with Jotform’s fillable PDF forms.

- **Client compatibility**  
  - Designed to be added as an MCP server in various chat or MCP-enabled clients.  
  - Central configuration documentation available via the Pipedream MCP configuration page.

- **Infrastructure**  
  - Powered by Pipedream Connect as the underlying integration platform.

## Use Cases

- Build and manage online forms from MCP-capable chat or AI clients.  
- Collect and review form or survey responses initiated from conversations.  
- Generate and work with fillable PDF forms through MCP integrations.
