## Overview

Powerful MCP server for integrating Jotform forms with AI assistants like Cursor, VS Code, and Goose.

## Features

- **form_list**: Retrieve all forms.
- **create_form**: Generate new forms.
- **edit_form**: Modify existing forms.
- **create_submission**: Submit new data.
- **get_submissions**: Fetch form responses.
- OAuth 2.0 for secure, one-time client approvals.
- Simple setup by pointing MCP clients to the server URL.
- Actively maintained with expanding tools.
