# Zenscrape MCP Server

**Category:** Content Extraction & Summarization MCP Servers  
**Brand:** Zenscrape  
**Slug:** `zenscrape-mcp-server`

## Overview
Zenscrape MCP Server is an MCP server integration that connects Zenscrape’s web scraping API to MCP-compatible clients. It enables large-scale web scraping while automatically handling proxy management and reducing the likelihood of request blocking.

## Features
- **MCP server integration** for Zenscrape’s web scraping API.
- **Large-scale web scraping** support for data collection across many pages/sites.
- **Automatic proxy management** handled by Zenscrape’s API.
- **Blocking minimization** to reduce failed or blocked HTTP requests when scraping.
- **Static MCP server URL** for all clients: `https://mcp.pipedream.net/v2`.
- **Client-agnostic setup**: same server URL works across different MCP-compatible chat or AI clients.
- **Account-based authentication**: authenticate your Zenscrape account when adding the MCP server to your application.

## Usage
- Connect your Zenscrape account via the provided MCP configuration flow.
- Add the static MCP server URL (`https://mcp.pipedream.net/v2`) to your MCP-compatible client.
- Use the exposed MCP tools to perform web scraping and data extraction through Zenscrape’s API.
