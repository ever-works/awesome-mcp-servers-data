# Miyn MCP Server

Miyn MCP Server integrates Miyn’s appointment scheduling system with MCP-compatible clients, enabling automated booking and scheduling workflows.

## Overview
- **Type:** MCP server / integration
- **Category:** Business & Commerce – MCP Servers
- **Purpose:** Connect Miyn’s appointment scheduling software to MCP-compatible chat or automation clients via a standard MCP server endpoint.
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP Integration**
  - Exposes Miyn’s appointment scheduling capabilities through an MCP-compatible server.
  - Uses a static MCP server URL that can be added to any supported client.

- **Appointment Scheduling**
  - Integrates with Miyn’s appointment scheduling software, suitable for small businesses.
  - Supports automated booking and scheduling workflows when used with MCP clients.

- **Client-Agnostic Server URL**
  - Single static endpoint (`https://mcp.pipedream.net/v2`) works across all MCP-compatible clients.
  - Authentication is handled when the server is added/configured in the client application.

- **Configuration Guidance**
  - Can be added to various chat or MCP-compatible apps.
  - Additional configuration details are available via the platform’s configuration documentation (referenced as a “Configuration page”).
