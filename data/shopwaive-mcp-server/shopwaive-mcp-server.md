# Shopwaive MCP Server

## Overview
Shopwaive MCP Server is an MCP-compatible service that connects to Shopwaive for managing checkout, store credit, and gift cards across eCommerce platforms. It is designed to integrate into MCP workflows and supports headless commerce architectures as well as platforms like Shopify Plus and Wix.

- **Category:** Business & Commerce MCP Servers  
- **Use cases:** Checkout operations, customer store credit handling, gift card management, multi-platform eCommerce workflows

## Features
- **Checkout management** across supported eCommerce platforms via MCP workflows.
- **Store credit management**, enabling workflows that read or update customer credits in Shopwaive.
- **Gift card management**, including using and managing gift cards as part of the checkout process.
- **Multi-platform support**, including:
  - Headless eCommerce implementations
  - Shopify Plus
  - Wix
- **MCP integration**: Exposes Shopwaive capabilities as an MCP server endpoint that can be added to compatible chat or workflow clients.
- **Static MCP server endpoint** using a single URL for all clients: `https://mcp.pipedream.net/v2` (authentication performed when adding the server to an app).

## Integration & Configuration
- Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server within your MCP-compatible client or application, then authenticate as prompted.
