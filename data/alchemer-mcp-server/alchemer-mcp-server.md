# Alchemer MCP Server

**Category:** Business & Commerce · MCP Servers  
**Tags:** surveys, feedback, analytics

## Overview
The Alchemer MCP Server is an MCP (Model Context Protocol) integration that lets MCP-compatible clients interact with Alchemer’s enterprise online survey and feedback collection platform. It is provided via Pipedream Connect as a static MCP endpoint that you can add to compatible chat or AI applications.

- **Name:** Alchemer MCP Server  
- **Platform:** Pipedream Connect  
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible endpoint** – Exposes Alchemer functionality through a standardized MCP server interface for use with MCP clients (e.g., AI/chat applications that support MCP).
- **Static server URL** – Uses a single static URL (`https://mcp.pipedream.net/v2`) for all clients, simplifying configuration and reuse across tools.
- **Client-side authentication** – Authentication is handled when adding the server to your application, allowing secure access per client without changing the server URL.
- **Integration with Alchemer platform** – Designed to work with Alchemer’s enterprise survey and feedback collection capabilities (e.g., online surveys, feedback workflows, analytics) via MCP-aware tools.
- **Works with multiple chat clients** – Can be added to different MCP-capable chat clients; configuration guidance is available per client type.
- **Central configuration reference** – A dedicated configuration page (via Pipedream) provides full setup details and options for connecting the MCP server to your environment.

## Usage
- Use the static MCP URL when adding a new MCP server in your client.
- Complete authentication in your client during server setup to authorize access to Alchemer resources.
- Refer to the configuration documentation (on Pipedream) for client-specific setup steps.
