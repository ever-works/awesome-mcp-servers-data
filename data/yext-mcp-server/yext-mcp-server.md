# Yext MCP Server

An MCP Server integration that lets MCP-compatible tools create and deliver content across channels using Yext via Pipedream.

## Overview
- **Type:** MCP server integration
- **Category:** Business & Commerce â€“ MCP Servers
- **Provider:** Yext (via Pipedream)
- **Use case:** Enable chat clients and other MCP-compatible applications to interact with Yext APIs for content creation and delivery.

## Features
- **MCP-compatible server endpoint**
  - Static server URL usable across clients: `https://mcp.pipedream.net/v2`.
  - Designed to be added as an MCP server in supported chat clients and tools.

- **Yext account connection**
  - Connect a Yext account within Pipedream.
  - Select a specific client / configuration after connecting.

- **Content creation and delivery**
  - Intended to create and deliver content to multiple channels through Yext APIs (omnichannel publishing).

- **Client-agnostic configuration**
  - Same server URL works for all supported clients.
  - Authentication happens when adding the server to each application.

- **Guided setup for chat clients**
  - Option to select your chat client and view tailored setup instructions.
  - Link to a full configuration page for more detailed setup steps.

- **Tool-based interactions**
  - Exposes Yext-related tools / actions to MCP-compatible applications (tools are loaded dynamically within the interface).

## Configuration
- Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server to your MCP-compatible app or chat client.
- Authenticate with your Yext account when prompted.
- (Optional) Follow the full configuration guide on the linked configuration page.
