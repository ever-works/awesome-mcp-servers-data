# Corrently MCP Server

**Category:** Finance & Market Data MCP Servers  
**Brand:** Corrently  
**Slug:** `corrently-mcp-server`

## Description
Corrently MCP Server exposes Corrently’s electricity tariff and loyalty program via the Model Context Protocol (MCP). It enables applications and chat clients to access dynamic electricity tariff data and integrate loyalty and micro‑investment workflows into renewable energy generation.

MCP endpoint: `https://mcp.pipedream.net/v2`

## Features
- **Electricity Tariff Access**  
  - Access Corrently’s electricity tariff information through an MCP-compatible endpoint.
  - Designed for use with any MCP-enabled client via a static server URL.

- **Loyalty Program Integration**  
  - Integrate Corrently’s loyalty program features into apps or chat clients.  
  - Link electricity usage or tariff data with loyalty benefits.

- **Micro‑Investment Workflows**  
  - Enable micro‑investments into renewable energy generation facilities.  
  - Support workflows where electricity customers can invest small amounts in green energy projects.

- **MCP Server Characteristics**  
  - Single static URL (`https://mcp.pipedream.net/v2`) usable across different MCP clients.  
  - Authentication handled when adding the server to the application.  
  - Compatible with configuration flows documented on the Pipedream MCP configuration page.

## Integration & Usage
- Add the Corrently MCP Server via your MCP-compatible chat client or application using the static URL.  
- Configure authentication during server addition as guided by your client or the Pipedream MCP configuration docs.
