## Description
Logistia Route Planner MCP Server integrates the Logistia delivery management and route planning system with MCP-compatible agents, enabling route optimization and delivery orchestration through a standardized MCP endpoint.

## Features
- MCP-compatible server for integrating Logistia with agentic / chat-based clients
- Static MCP server URL usable across different clients
- Supports delivery management workflows
- Enables route planning and route optimization capabilities via MCP
- Facilitates orchestration of delivery operations through connected applications

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Hosting / Infrastructure:** Powered by Pipedream Connect
