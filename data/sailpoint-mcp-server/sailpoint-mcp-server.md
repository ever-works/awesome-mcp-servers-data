# SailPoint MCP Server

An MCP (Model Context Protocol) server for SailPoint that exposes SailPoint identity security and governance capabilities to MCP-compatible workflows and applications.

- **Website:** https://mcp.pipedream.com/app/sailpoint
- **Category:** Security & Attestation MCP Servers
- **Brand:** SailPoint
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Description
The SailPoint MCP Server, provided via Pipedream Connect, enables applications and chat clients that support MCP to connect to SailPoint’s identity security and governance features through a unified MCP endpoint.

## Features
- **Standard MCP Endpoint**  
  - Single static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same URL works across all compatible MCP clients.

- **Client-Agnostic Integration**  
  - Designed to be added to various chat clients and MCP-enabled applications.  
  - Authentication handled when adding the server within each client.

- **Pipedream Connect Integration**  
  - Hosted and powered by Pipedream Connect infrastructure.  
  - Uses Pipedream’s common configuration and connection patterns for MCP servers.

- **Identity Security & Governance Access**  
  - Provides access to SailPoint’s identity security and governance capabilities through MCP-compatible tools (exact operations depend on client configuration and SailPoint setup).

## Setup
- Use the static MCP URL `https://mcp.pipedream.net/v2` when configuring the server in your MCP-compatible client.  
- Follow your client’s instructions for adding an MCP server and completing authentication.

For detailed configuration instructions, see the provider’s configuration page linked from the main app listing.
