# NetHunt CRM MCP Server

## Overview
NetHunt CRM MCP Server provides access to NetHunt, a Gmail-based CRM, through the MCP protocol. It lets you interact with CRM data and sales workflows from compatible MCP-enabled applications while keeping NetHunt embedded in your inbox.

## Features
- **Gmail-based CRM integration**
  - Works directly with Gmail, bringing CRM functionality into your inbox.
- **Sales automation**
  - Supports automated sales workflows powered by NetHunt CRM.
- **Drip campaigns**
  - Enables configuration and use of drip email campaigns within your existing email environment.
- **MCP server access**
  - Provides a static MCP server URL (`https://mcp.pipedream.net/v2`) usable across MCP-compatible clients.
  - Authentication handled when adding the server to your MCP-enabled application.
- **Client-agnostic setup**
  - Can be connected from multiple chat or MCP clients using the same endpoint.

## Integration & Configuration
- Connect a NetHunt CRM account through the MCP server.
- Add the static server URL to your MCP-compatible app and authenticate when prompted.
- Additional setup and usage details are available on the Pipedream configuration page (referenced from the source page).
