## Features

- Access Figma design files
- Generate code from designs using Cursor agent

## Use Cases

- Convert Figma designs to code in IDE

## Setup

- Ensure Node.js installed
- Visit https://mcp.composio.dev/figma, generate npx command
- Run in terminal
- Refresh Cursor, find in MCP settings
- Initiate connection, provide Figma file URL
