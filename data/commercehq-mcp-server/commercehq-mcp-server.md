# CommerceHQ MCP Server

**Category:** Business & Commerce · MCP Servers  
**Slug:** commercehq-mcp-server  
**Source:** https://mcp.pipedream.com/app/commercehq

## Overview
CommerceHQ MCP Server is an MCP integration that exposes CommerceHQ’s e‑commerce and conversion-focused store capabilities through the Model Context Protocol (MCP). It is delivered via Pipedream Connect and can be added to compatible chat or MCP-enabled clients using a single static endpoint.

## Features
- **MCP-compatible server endpoint**  
  - Single static URL usable by any MCP client:  
    `https://mcp.pipedream.net/v2`
  - Authentication occurs when adding the server to the client.

- **CommerceHQ platform integration**  
  - Surfaces CommerceHQ’s e-commerce store functionality via MCP.  
  - Provides access to CommerceHQ’s conversion-focused platform features (high-converting apps and themes) through MCP-aware tools and workflows.

- **Multi-client support**  
  - Designed to work with multiple chat/MCP clients using the same URL.  
  - Configuration guidance available via the app’s configuration page on Pipedream.

- **Pipedream Connect-based delivery**  
  - Hosted and provided through Pipedream Connect.  
  - Operates within Pipedream’s infrastructure, terms, and privacy framework.
