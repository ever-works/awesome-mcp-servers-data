# Google reCAPTCHA MCP Server

**Category:** Security & Attestation MCP Servers  
**Provider / Brand:** Pipedream  
**Source:** https://mcp.pipedream.com/app/google_recaptcha

Security service MCP server that exposes Google reCAPTCHA’s bot-detection and anti‑fraud capabilities to MCP‑compatible tools via Pipedream.

---

## Features

- **Google reCAPTCHA integration via MCP**  
  - Provides access to Google reCAPTCHA’s security and bot‑detection services from MCP‑compatible clients and tools.

- **Static MCP server endpoint**  
  - Single, static server URL for all supported clients:  
    - `https://mcp.pipedream.net/v2`
  - Authentication handled when adding the server to your application.

- **Client‑agnostic setup**  
  - Designed to work with multiple chat or MCP clients.  
  - You select your chat client and follow its specific setup flow (as documented externally).

- **Configuration documentation**  
  - Centralized configuration guidance available via a dedicated configuration page (linked from the app page).

- **Hosted by Pipedream**  
  - Runs on Pipedream Connect infrastructure.  
  - Governed by Pipedream’s Terms and Privacy Policy.
