# Zoho Creator MCP Server

**Category:** Workflow Automation MCP Servers  
**Brand:** Zoho  
**Description:** An MCP server for Zoho Creator that lets agents interact with Zoho’s low‑code application development platform and automate app workflows.

---

## Features

- **MCP server for Zoho Creator**
  - Exposes Zoho Creator capabilities to MCP-compatible chat clients and agents.
  - Designed for integrating with Zoho’s low-code application development environment.
  - Intended to support workflow automation within Zoho Creator apps.

- **Static MCP endpoint**
  - Single server URL for all clients: `https://mcp.pipedream.net/v2`.
  - Authentication is performed when adding the server in each client, not by changing the URL.

- **Client-agnostic setup**
  - Same MCP server URL works across different chat clients.
  - Can be added to compatible applications following their MCP/server configuration flows.

- **Configuration resources**
  - Additional setup and configuration guidance available via the platform’s Configuration page (general MCP server configuration, not Zoho-specific details in the provided content).

---

## Setup

1. Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add the server in your MCP-compatible chat client or application.
3. Complete authentication during the add/connect flow in the client.
4. Refer to the platform’s Configuration page for client-specific instructions.
