## Overview
BILL MCP Server is a Model Context Protocol (MCP) server that connects AI agents to the BILL platform. It enables programmatic work with bill creation, payments, invoicing, and receivables from within MCP-compatible chat or agent clients.

## Features
- **Bill management**
  - Work with bill creation workflows in the BILL platform
  - Enable AI-driven handling of bills via MCP-compatible tools

- **Payments and receivables**
  - Interact with payments functionality in BILL
  - Support workflows related to receivables through AI agents

- **Invoicing**
  - Allow AI agents to work with invoicing operations in BILL

- **MCP integration**
  - Exposes BILL functionality through the Model Context Protocol
  - Designed to be added as an MCP server to compatible chat / agent applications

- **Static MCP server endpoint**
  - MCP server URL (static for all clients):
    ```
    https://mcp.pipedream.net/v2
    ```
  - Authentication is performed when adding the server to an application

- **Client-agnostic setup**
  - The same server URL can be used across different MCP-compatible chat clients
  - Configuration details are available via the Pipedream configuration page

## Technical Details
- **Protocol:** Model Context Protocol (MCP)
- **Server type:** Remote MCP server
- **Provider:** Pipedream Connect (integrated with BILL)
