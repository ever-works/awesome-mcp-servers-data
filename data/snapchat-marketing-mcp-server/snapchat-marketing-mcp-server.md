# Snapchat Marketing MCP Server

## Overview
The Snapchat Marketing MCP Server integrates Snapchat Ads and Public Profiles with the Model Context Protocol (MCP), enabling programmatic access to Snapchat marketing capabilities (such as ad campaigns and marketing assets) from compatible MCP clients.

- **Name:** Snapchat Marketing MCP Server
- **Category:** Business & Commerce MCP Servers
- **Brand:** Snapchat
- **Slug:** `snapchat-marketing-mcp-server`
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- Integration of Snapchat Marketing (Ads and Public Profiles) with the MCP ecosystem.
- Programmatic management of Snapchat ad campaigns and related marketing assets via MCP-compatible tools.
- Single static MCP server URL (`https://mcp.pipedream.net/v2`) usable across different clients.
- Authentication handled when adding the server to an MCP-compatible application.
- Works with multiple chat or MCP clients (configured per client in their respective settings).
