## Overview

GameBrain API clients via MCP allow querying vast game databases for metadata, platforms, genres, and more.

## Features

- Search millions of video games
- Filter by platform, genre, release date
- Retrieve detailed game info, ratings, covers
- Multi-platform coverage (PC, console, mobile)

## Use Cases

- Game recommendation engines
- AI game research and analysis
- Content generation for gaming apps

## Pricing

API usage-based; open-source client.
