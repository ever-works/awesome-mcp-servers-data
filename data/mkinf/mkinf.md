## mkinf

**Website:** https://mkinf.io  
**Category:** MCP Server Directories & Lists  
**Tags:** directory, MCP, curated list

### Overview
mkinf is an open‑source registry of hosted Model Context Protocol (MCP) servers. It focuses on helping AI agents and developers quickly discover and use available MCP server endpoints to accelerate agent workflows.

### Features
- **Open‑source registry**
  - Public, open‑source project for listing MCP servers.
  - Community‑oriented approach to maintaining and extending the registry.

- **Hosted MCP server listings**
  - Central directory of MCP servers that are already hosted and available via endpoints.
  - Designed to make it easier to plug MCP servers into AI agents and tooling.

- **Discovery for AI agents**
  - Structured information to help AI agents and developers find relevant MCP servers.
  - Oriented around accelerating AI agent workflows by simplifying endpoint discovery and selection.

- **MCP‑focused catalog**
  - Specialization in Model Context Protocol servers, rather than generic APIs.
  - Curated list organized around MCP usage and integration.
