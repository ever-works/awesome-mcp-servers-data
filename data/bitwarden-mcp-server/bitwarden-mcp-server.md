# Bitwarden MCP Server

Open-source password manager MCP server that integrates Bitwarden credential management into MCP-enabled applications.

## Overview
- **Type:** MCP server (Model Context Protocol)
- **Provider / Host:** Pipedream
- **Category:** Security / Credential management
- **Static MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Use Case:** Securely access and manage Bitwarden-stored credentials within MCP workflows and chat clients.

## Features
- **Static MCP Endpoint**  
  - Single, static MCP server URL (`https://mcp.pipedream.net/v2`) usable across all compatible clients.  
  - No per-client URL variation required.

- **Bitwarden Integration**

- **Client-Agnostic Setup**  
  - Designed to be added to various chat or MCP-compatible clients.  
  - Authentication occurs when the server is added to the application.

- **Configuration Documentation**  
  - Full configuration details available via a dedicated configuration page (linked from the app page).
