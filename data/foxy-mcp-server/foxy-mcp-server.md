# Foxy MCP Server

Foxy MCP Server connects MCP clients to Foxy’s ecommerce platform so you can manage carts, products, and transactions from compatible chat or MCP-enabled applications.

- **Website / Source**: https://mcp.pipedream.com/app/foxy
- **Category**: Business & Commerce – MCP Servers
- **Tags**: e-commerce, payments, transactions
- **MCP Server URL**: `https://mcp.pipedream.net/v2`

## Features

- **MCP Integration**
  - Exposes Foxy’s ecommerce capabilities via the Model Context Protocol (MCP).
  - Works with any compatible MCP client using a single static server URL.
  - Authentication is handled when adding the server to your specific application/client.

- **Ecommerce Platform Access**
  - Connects to Foxy’s flexible ecommerce platform.
  - Designed for managing:
    - Shopping carts
    - Products
    - Transactions / payments (via Foxy’s platform capabilities)

- **Client-Agnostic Server URL**
  - Uses one static MCP server endpoint: `https://mcp.pipedream.net/v2`.
  - Same URL for all clients; configuration is done per client within its settings.

- **Configuration Resources**
  - Step-by-step guidance available for different chat / MCP clients.
  - Central configuration documentation via the main Configuration page on Pipedream.
