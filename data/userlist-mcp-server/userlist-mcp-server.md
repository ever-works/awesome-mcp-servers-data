# Userlist MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Userlist  
**Source:** https://mcp.pipedream.com/app/userlist

An MCP Server that integrates with Userlist, enabling MCP-aware tools and chat clients to orchestrate SaaS email automation and user messaging workflows.

---

## Features

- **MCP integration endpoint**  
  - Static server URL for all MCP-compatible clients:  
    - `https://mcp.pipedream.net/v2`  
  - Authentication is performed when adding the server to your application.

- **Userlist integration**  
  - Connects MCP-aware tools to Userlistâ€™s email and user messaging capabilities.  
  - Designed for SaaS products that need full-stack email automation and user messaging workflows.

- **Client-agnostic configuration**  
  - Single MCP server URL works across different chat clients.  
  - Configuration guidance available via the linked configuration documentation (per client setup not detailed in the provided content).

- **Pipedream Connect platform**  
  - MCP server is hosted and powered by Pipedream Connect infrastructure.

---

## Usage

1. Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add the server to your MCP-aware chat client or application.
3. Authenticate during server addition as required by your client.  
4. Use the integration to orchestrate SaaS email automation and user messaging via Userlist.
