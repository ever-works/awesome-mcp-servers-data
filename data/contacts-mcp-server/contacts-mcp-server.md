# Contacts+ MCP Server

Cross-platform contacts management via MCP.

## Overview
The Contacts+ MCP Server is an MCP (Model Context Protocol) server that exposes Contacts+ functionality to compatible chat and AI clients. It provides standardized, MCP-based access to cross-platform contacts data and related operations.

- **Type:** MCP server (integration)
- **Category:** Business & Commerce MCP Servers
- **Brand:** Contacts+
- **MCP Endpoint URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-based access to Contacts+**
  - Connects Contacts+ contact data and operations to any MCP-compatible client.
  - Uses a single, static MCP server URL for all supported clients.

- **Cross-platform contacts integration**
  - Designed around a "Cross-Platform Contacts App" model, enabling access to contacts across platforms supported by Contacts+.

- **Standard MCP server behavior**
  - Can be added to various chat or AI applications that support MCP.
  - Authentication occurs when the server is added/configured in the client.

- **Centralized configuration**
  - One static URL (`https://mcp.pipedream.net/v2`) works for all clients.
  - Additional setup and configuration guidance available via a dedicated configuration page (client-specific instructions).

- **Pipedream Connect integration**
  - Implemented and operated via Pipedream Connect infrastructure.

## Usage
- Use the static URL `https://mcp.pipedream.net/v2` as the MCP server endpoint in your MCP-compatible chat or AI application.
- Authenticate within your client when prompted during server setup.
