# Z-API MCP Server

An MCP server that integrates with Z-API to enable WhatsApp messaging and automation within MCP-compatible clients.

## Overview
- **Category:** Messaging MCP Server
- **Integration:** WhatsApp via Z-API
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`
- **Provider:** Pipedream / Pipedream Connect

## Features
- WhatsApp integration through Z-API
- Supports sending messages over WhatsApp via MCP workflows
- Designed to plug into multiple MCP-compatible chat clients
- Uses a single static MCP server URL for all clients
- Authentication handled when adding the server to your application
- Configuration details available via the Pipedream Configuration page (client-specific setup instructions)

## Usage
- Add the MCP server to your MCP-compatible app using:
  - **MCP Server URL:** `https://mcp.pipedream.net/v2`
- Authenticate within your client when prompted.
