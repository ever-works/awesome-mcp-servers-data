# OANDA MCP Server

**Category:** Finance / Market Data MCP Servers  
**Website:** https://mcp.pipedream.com/app/oanda

## Description
OANDA MCP Server exposes OANDA’s multi-asset trading, FX data, and payments functionality through MCP-compatible AI workflows. It allows AI applications and chat clients that support MCP to interact with OANDA services via a standardized server endpoint.

## Features
- Access to OANDA’s multi-asset trading capabilities via MCP-compatible clients.
- Retrieval of foreign exchange (FX) and other market data suitable for AI workflows.
- Integration with OANDA-related corporate payments and FX services via MCP.
- Single static MCP server URL for all clients: `https://mcp.pipedream.net/v2`.
- Authentication handled when adding / configuring the server in the client application.
- Usable from multiple MCP-compatible chat clients and AI tools.

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Integration platform:** Powered by Pipedream Connect.
- **Usage pattern:** Add the MCP server URL to a compatible chat client or AI application and configure authentication as required.
