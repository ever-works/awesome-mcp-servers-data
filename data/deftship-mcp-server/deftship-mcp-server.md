# Deftship MCP Server

## Overview
The Deftship MCP Server provides a Machine Connection Protocol (MCP) interface to DeftShip, a free multi-carrier shipping software platform. It enables programmatic access to DeftShip’s shipping and logistics capabilities via compatible MCP-enabled chat or automation clients.

- **Name:** Deftship MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Use Cases:** Shipping, logistics, automation  
- **Provider / Host:** Pipedream Connect  
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server endpoint**  
  - Single static URL (`https://mcp.pipedream.net/v2`) used across clients.
- **Client-agnostic configuration**  
  - Works with any MCP-capable chat or automation client.  
  - Authentication handled when the server is added to the client/application.
- **Integration with DeftShip**  
  - Connects to DeftShip’s free multi-carrier shipping software to support shipping and logistics automation workflows.
- **Central configuration documentation**  
  - Setup and configuration details available via a shared configuration page (through Pipedream / Pipedream Connect).

> Note: The underlying DeftShip platform is described as multi-carrier shipping software; the page does not list specific API methods or carrier integrations, so those cannot be detailed here.
