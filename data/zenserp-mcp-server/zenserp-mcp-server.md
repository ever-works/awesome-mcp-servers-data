# Zenserp MCP Server

**Description**  
Zenserp MCP Server is an MCP-compatible endpoint that lets agents programmatically retrieve and scrape search engine results pages (SERPs) via Zenserp’s API. It is designed for fast, scalable access to search results that can be integrated into MCP-enabled chat clients and applications.

## Features

- **MCP-compatible server**  
  - Exposes Zenserp search capabilities through the Model Context Protocol (MCP), allowing MCP agents and chat clients to query SERPs programmatically.

- **Static server endpoint**  
  - Uses a single static MCP server URL for all clients:  
    - `https://mcp.pipedream.net/v2`
  - Authentication is handled when adding the server to each application/client.

- **Search engine results scraping**  
  - Accesses search engine results pages (SERPs) through Zenserp’s API.  
  - Built for fast, scalable scraping of search results.

- **Client-agnostic integration**  
  - Works with multiple MCP-enabled chat clients and tools.  
  - Can be added to different apps by configuring the same server URL and providing appropriate credentials.

- **Configuration guidance**  
  - Documentation available via a configuration page (linked from the app page) that explains how to add and use the server in various clients.
