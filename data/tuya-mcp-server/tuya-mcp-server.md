# Tuya MCP Server

**Category:** Home Automation MCP Servers  
**Brand:** Tuya  
**Website:** https://mcp.pipedream.com/app/tuya

## Description
Tuya MCP Server is an MCP (Model Context Protocol) server that connects Tuya’s global AI+IoT cloud platform with AI agents. It enables management and control of Tuya-connected smart devices and services through MCP-compatible clients.

## Features
- Integrates Tuya’s AI+IoT cloud platform with AI agents via MCP
- Supports management and control of Tuya-connected smart home and IoT devices
- Uses a static MCP server URL compatible with all supported clients:
  - `https://mcp.pipedream.net/v2`
- Authentication handled when adding the server within the client/application
- Can be added to various MCP-compatible chat or agent clients
- Centralized configuration guidance available via a dedicated configuration page (Pipedream Configuration)

## Technical Details
- MCP endpoint: `https://mcp.pipedream.net/v2`
- Protocol: Model Context Protocol (MCP)
- Deployment/hosting: Provided via Pipedream Connect infrastructure
