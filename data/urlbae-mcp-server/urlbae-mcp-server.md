# UrlBae MCP Server

**Category:** Business & Commerce · MCP Servers  
**Brand:** UrlBae

## Overview
UrlBae MCP Server is a Model Context Protocol (MCP) server that allows AI agents and MCP-compatible clients to work with the UrlBae platform. It enables programmatic creation and management of dynamic short links, QR codes, and bio pages, as well as retrieval of associated analytics.

MCP endpoint:
- **Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **Dynamic Link Management**  
  - Create short links through the UrlBae platform.  
  - Manage existing short links (e.g., update targets or settings, where supported by UrlBae and the MCP tools).

- **QR Code Generation & Management**  
  - Create QR codes associated with UrlBae links or destinations.  
  - Manage QR assets via MCP-compatible tools.

- **Bio Page Creation & Management**  
  - Create bio / profile pages (link-in-bio style) using UrlBae.  
  - Manage content and structure of bio pages through MCP integrations (where available).

- **Analytics Retrieval**  
  - Retrieve analytics for UrlBae assets (short links, QR codes, bio pages), including campaign performance metrics exposed by UrlBae.  
  - Use analytics data within AI workflows for reporting, optimization, or decision-making.

- **MCP Integration**  
  - Uses a static MCP server URL (`https://mcp.pipedream.net/v2`) that works across supported clients.  
  - Authentication is handled when adding the server to the MCP-compatible application.  
  - Designed to be used in chat or agent clients that support MCP for tool calling.

- **Pipedream Connect Integration**  
  - Hosted and powered via Pipedream Connect’s infrastructure.  
  - Can be configured following Pipedream’s MCP configuration guidance.

## Use Cases
- Marketing and campaign tracking with dynamic links and QR codes.  
- Managing link-in-bio style pages programmatically.  
- Pulling UrlBae analytics into AI-driven reports or dashboards.  
- Allowing AI agents to generate and manage tracking links during conversations.
