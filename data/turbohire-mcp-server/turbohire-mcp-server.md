# TurboHire MCP Server

## Overview
TurboHire MCP Server integrates TurboHire’s hiring and recruitment automation capabilities into MCP-compatible tools and workflows via a Pipedream-hosted MCP endpoint.

- **Type:** MCP server (Model Context Protocol)
- **Category:** Business & Commerce – MCP Servers
- **Use cases:** Recruiting, HR workflows, hiring automation within MCP-enabled chat or developer tools.

## MCP Endpoint
- **Server URL (static for all clients):** `https://mcp.pipedream.net/v2`
- The same URL is used across supported MCP clients; authentication is handled when you add the server to your application.

## Features
- **MCP Integration**
  - Exposes TurboHire recruitment and hiring automation features through the MCP protocol.
  - Usable from any MCP-compatible client (e.g., supported chat environments and tools).
- **Pipedream Hosting**
  - Server is provided and hosted via Pipedream Connect.
  - Uses a single, static base URL for all clients.
- **Configurable in Multiple Clients**
  - Can be added to different MCP-enabled chat clients or tools using the same server URL.
  - Additional configuration guidance is available through the referenced configuration page on Pipedream.
