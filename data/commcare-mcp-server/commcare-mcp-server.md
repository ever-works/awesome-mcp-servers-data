# CommCare MCP Server

**Category:** Workflow Automation MCP Servers  
**Brand:** CommCare  
**Source:** https://mcp.pipedream.com/app/commcare

## Overview
CommCare MCP Server is a Model Context Protocol (MCP) server that lets compatible AI/chat clients design and interact with CommCare mobile and web application workflows. It exposes CommCare-related tools so you can configure and work with CommCare from within your MCP-enabled applications.

## Features
- **MCP-compliant server**
  - Provides a static MCP server endpoint: `https://mcp.pipedream.net/v2`.
  - Designed to work with multiple MCP-compatible chat clients.

- **CommCare workflow integration**
  - Enables model-context tools to design CommCare mobile and web app workflows.
  - Supports interaction with CommCare app workflows from within your client.

- **Account-based configuration**
  - Connect a CommCare account to configure access and tools.
  - Configuration and client-specific setup documented via a dedicated configuration page.

- **Multi-client support**
  - Can be added to different chat clients; instructions are provided per client.

- **Dynamic tools (actions) exposure**
  - Exposes available tools/actions for CommCare once the server is connected and configured.

## Integration & Setup
- Copy the static MCP server URL (`https://mcp.pipedream.net/v2`).
- Add the server to your MCP-compatible app or chat client.
- Authenticate when adding the server to grant access to your CommCare account.
- Refer to the full configuration page linked from the app for client-specific steps.
