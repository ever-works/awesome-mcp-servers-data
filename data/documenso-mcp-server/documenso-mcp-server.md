# Documenso MCP Server

**Category:** Document Management MCP Servers  
**Tags:** document-management, e-signature, workflow-automation

## Overview
The Documenso MCP Server integrates Documenso’s open-source document signing and trust infrastructure with MCP-based tools and chat clients. It provides a static MCP endpoint that can be added to compatible applications to enable document management and e-signature workflows.

- **Name:** Documenso MCP Server  
- **Provider / Brand:** Documenso (via Pipedream)  
- **Purpose:** Connect Documenso’s document signing capabilities into MCP-enabled applications and workflows.

## Features
- **MCP Integration Endpoint**  
  - Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`  
  - Usable across multiple MCP-compatible chat clients and applications.

- **Documenso Account Integration**  
  - Connect a Documenso account via Pipedream.  
  - Configure Documenso as a file storage / document handling backend.

- **Client Configuration Flow**  
  - Select your chat client and follow client-specific steps to add the MCP server.  
  - Authentication occurs when adding the server to the chosen application.

- **Tooling / Actions Support**  
  - Exposes Documenso-related tools / actions through the MCP interface (listed dynamically as “Available tools” within the Pipedream UI).  
  - Designed for use in document management, e-signature, and workflow automation contexts.

- **Centralized Configuration Reference**  
  - Additional setup and usage details available via the Pipedream configuration page.
