# ClickHelp MCP Server

**Category:** Documentation & Learning Resources  
**Brand:** ClickHelp  
**Slug:** `clickhelp-mcp-server`

## Description
The ClickHelp MCP Server is a Model Context Protocol (MCP) server that connects ClickHelp’s documentation and knowledge management platform to AI tools through the Pipedream MCP environment. It allows MCP-compatible clients to interface with ClickHelp’s AI-driven documentation and knowledge features.

## Features
- Integrates ClickHelp with Pipedream’s MCP environment
- Enables MCP clients to access ClickHelp documentation and knowledge resources
- Connects a ClickHelp account to AI tools such as:
  - Cursor
  - Claude
  - Windsurf
- Supports AI-powered documentation and knowledge management workflows via MCP
