# Nutshell MCP Server

**Category:** Business & Commerce MCP Servers  
**Slug:** nutshell-mcp-server  
**Brand:** Nutshell  
**Source URL:** https://mcp.pipedream.com/app/nutshell

All-in-one CRM and email marketing MCP server integration for Nutshell, designed for B2B organizations to manage contacts, deals, and email campaigns directly through MCP-compatible clients.

## MCP Server URL

Use this static URL for all clients (authentication is handled when you add the server to your application):

```text
https://mcp.pipedream.net/v2
```

## Features

- **CRM Integration**
  - Connects to Nutshell, an all-in-one CRM platform for B2B organizations.
  - Centralizes customer and deal data for use within MCP-compatible chat or agent clients.

- **Email Marketing Integration**
  - Interfaces with Nutshellâ€™s email marketing capabilities via the MCP server layer.
  - Enables workflows that combine CRM records with email marketing actions (where supported by the client/implementation).

- **B2B Sales & Collaboration Focus**
  - Tailored for B2B organizations collaborating on sales pipelines.
  - Supports workflows oriented around winning and managing deals.

- **Standard MCP Server Endpoint**
  - Single, static MCP server URL (`https://mcp.pipedream.net/v2`) usable across different MCP clients.
  - Authentication occurs when adding the server to the chosen application.

- **Pipedream Connect Integration**
  - Operates via Pipedream Connect infrastructure.
  - Can be combined with other Pipedream-powered integrations in broader automation setups (depending on client).

## Configuration

- Add the server to your MCP-compatible chat or agent client using:
  - **Server URL:** `https://mcp.pipedream.net/v2`
  - Complete authentication and configuration within your client following its MCP setup instructions.
- Additional generic setup information is available on the Pipedream MCP **Configuration** page (not Nutshell-specific).
