# Leiga MCP Server

MCP Server for Leiga, bringing AI-powered teamwork and collaboration features into MCP-compatible tools.

---

## Overview

Leiga MCP Server integrates Leiga’s AI-powered teamwork and collaboration capabilities into any MCP-compatible client via a single static endpoint. Once added to your chat or MCP client, you authenticate within the application and can then use Leiga’s project and task collaboration features through the MCP interface.

- **Category:** Project Management MCP Server  
- **Use Cases:** Collaboration, project management, task management within MCP-compatible tools

---

## MCP Server Endpoint

- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Usage:**
  - Works as a **static URL** for all supported MCP clients
  - Authentication occurs when you add / configure the server inside your application
