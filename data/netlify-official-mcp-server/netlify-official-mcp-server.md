## Overview

Netlify MCP Server follows the Model Context Protocol to enable code agents to use the Netlify API and CLI to create new projects and to build and manage your deployments. This integration provides serverless, scalable backend deployment with zero infrastructure headaches through natural language interactions.

## Features

- **Project Creation**: Create new Netlify projects through AI assistants
- **Deployment Management**: Build, deploy, and manage site deployments
- **API Integration**: Full access to Netlify API functionality
- **CLI Operations**: Execute Netlify CLI commands conversationally
- **Git Integration**: Deploy from Git repositories automatically
- **Serverless Functions**: Manage Netlify Functions deployment
- **Domain Management**: Configure custom domains and DNS
- **Build Configuration**: Manage build settings and environment variables

## Deployment Benefits

Hosting MCP endpoints on Netlify provides:
- Serverless, scalable backend with zero infrastructure management
- Automatic scaling based on traffic
- Deploy, update, and scale via Git or CLI
- Global CDN distribution
- Instant rollbacks and preview deployments

## Use Cases

- Creating and deploying Jamstack sites
- Managing serverless function deployments
- Automated deployment workflows
- Site configuration management
- Build troubleshooting and optimization
- Multi-environment deployment coordination

## Integration Methods

Deploy MCP servers to Netlify using:
- Git-based continuous deployment
- Netlify CLI for manual deployments
- API-driven deployment automation

## Technical Implementation

Utilizes Netlify's serverless functions to deploy MCP servers, streamlining development and reducing overhead. Simplifies management through automated scaling and deployment workflows.

## Supported Clients

Claude Code, Cursor, Windsurf, and any MCP-compatible AI assistant.

## Authentication

Requires Netlify personal access token for API and deployment access.
