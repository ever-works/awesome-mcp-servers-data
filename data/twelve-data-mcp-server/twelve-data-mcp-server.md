# Twelve Data MCP Server

**Category:** Finance Market Data MCP Servers  
**Brand:** Twelve Data

Twelve Data MCP Server integrates Twelve Data’s stock, forex, and crypto market data APIs into MCP-based applications via a static MCP endpoint.

---

## Features

- **MCP-compatible market data server**  
  Integrates Twelve Data’s APIs with MCP-based applications.

- **Multi-asset coverage**  
  - Stocks  
  - Forex  
  - Crypto

- **Static MCP server URL**  
  - Single endpoint for all MCP clients:  
    `https://mcp.pipedream.net/v2`  
  - Authentication handled when adding the server to your application.

- **Client-agnostic integration**  
  - Works with multiple chat / MCP clients.  
  - Setup instructions available per client via the configuration flow.

- **Configuration workflow**  
  - Connect a Twelve Data account.  
  - Select a client and follow guided setup.  
  - Centralized documentation via a configuration page.
