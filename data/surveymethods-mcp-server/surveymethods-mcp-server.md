# SurveyMethods MCP Server

## Overview
SurveyMethods MCP Server is an integration that exposes SurveyMethods’ online survey platform through the Model Context Protocol (MCP). It enables applications and chat-based clients to create and manage surveys, perform advanced analysis, send email newsletters, and interact with the SurveyMethods API via a unified MCP endpoint.

- **Name:** SurveyMethods MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Brand:** SurveyMethods  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **Online survey creation and management**  
  - Access SurveyMethods’ online survey software via MCP.  
  - Create, configure, and manage surveys programmatically.

- **Advanced data analysis**  
  - Run advanced analysis on survey responses through the integration.  
  - Use SurveyMethods’ analytics capabilities from within MCP-enabled clients.

- **Email newsletters**  
  - Send and manage email newsletters related to surveys (e.g., invitations, reminders, follow-ups) via the MCP integration.

- **Survey API access**  
  - Utilize the comprehensive SurveyMethods API through a single MCP server endpoint.  
  - Integrate survey functionality into MCP-compatible chat clients and applications.

- **Static MCP endpoint**  
  - Uses a single static server URL (`https://mcp.pipedream.net/v2`) for all clients.  
  - Authentication is handled when adding/configuring the server in your application.

- **Client-agnostic integration**  
  - Designed to work with multiple MCP-compatible chat clients and tools.  
  - Configuration guidance available via the associated configuration page (client-specific instructions).
