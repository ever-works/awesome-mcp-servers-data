## Overview
The Wealthbox MCP Server connects MCP-based tools and chat clients to the Wealthbox CRM platform used by financial advisors, enterprise firms, and broker-dealers. It enables applications to interact with Wealthbox data and workflows for managing client relationships and streamlining operations.

## Features
- **MCP Server Endpoint**  
  - Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`  
  - Authentication handled when adding the server to a compatible application or chat client.

- **CRM Integration Focus**  
  - Designed for Wealthbox, a modern CRM tailored for financial advisors.  
  - Supports use cases for:  
    - Individual financial advisory practices  
    - Enterprise financial firms  
    - Broker-dealers managing advisor networks

- **Client & Workflow Enablement**  
  - Allows MCP-based tools and chat clients to connect to Wealthbox CRM.  
  - Intended to enable management of CRM data and advisor workflows (e.g., client relationship and operations management) through MCP-compatible interfaces.

- **Configuration Resources**  
  - Central configuration entry point via the Pipedream "Configuration" page for setup details.  
  - Server can be added from multiple chat clients, following client-specific instructions.

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Integration Host:** Pipedream Connect  
- **Underlying CRM:** Wealthbox (financial advisor CRM)
