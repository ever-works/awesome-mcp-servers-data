# GitLab MCP Server

**Category:** Development Tools · MCP Servers  
**Vendor:** Pipedream  
**Source:** https://mcp.pipedream.com/app/gitlab

## Overview
GitLab MCP Server is an MCP-compatible integration that connects GitLab’s DevSecOps platform to MCP-based applications via Pipedream. It enables automated workflows around repositories, branches, commits, issues, and epics using a static MCP server URL.

- **MCP server URL:** `https://mcp.pipedream.net/v2`

## Features

### MCP Integration
- Static MCP server URL usable across supported MCP clients.
- Authentication handled when adding the server to your client/application.
- Works with multiple chat or MCP-enabled clients (configuration docs provided via Pipedream).

### GitLab Issue Management
- **Create Issue**: Create new GitLab project issues programmatically.
- **Get Issue**: Retrieve a single issue from a repository.
- **Update Issue**: Update fields of an existing project issue.
- **Search Issues**: Search for issues in a repository using a query.

### GitLab Epic Management
- **Create Epic**: Create new epics in GitLab.
- **Update Epic**: Update existing epics.

### Repository & Branch Management
- **List Repo Branches**: Retrieve a list of branches for a given project repository.
- **Get Repo Branch**: Fetch details of a single repository branch.
- **Create Branch**: Create a new branch in a repository.
- **List Commits**: List commits in a specific repository branch.
