# ProWorkflow MCP Server

**Category:** Project Management MCP Servers  
**Brand:** ProWorkflow  
**Source:** https://mcp.pipedream.com/app/proworkflow

## Overview
ProWorkflow MCP Server exposes ProWorkflow’s project, task, contact, and client management capabilities as tools within an MCP-compatible environment. It allows you to orchestrate core ProWorkflow operations (projects, tasks, and related entities) programmatically from your chat or MCP client.

## Features
- **MCP integration**
  - Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`
  - Authentication handled when adding the server to your MCP-compatible application
  - Works with multiple chat / MCP clients via a standard configuration flow

- **Project management tools**
  - **Add Project**
    - Create new projects in ProWorkflow
    - Configure project details programmatically via the MCP environment
  - **Delete Project**
    - Remove existing projects from ProWorkflow
    - Useful for automated cleanup or lifecycle workflows

- **Task management tools**
  - **Add Project Task**
    - Create new tasks within a specific project
    - Use within automations to break work into structured tasks
  - **Update Project Task**
    - Modify existing project tasks
    - Supports updating task details from automated workflows or agent actions

- **Contact & client tools**
  - **Add Contact**
    - Create new contacts in ProWorkflow
    - Integrate with other systems to keep contact records in sync
  - **Add Client to Project**
    - Associate a client with a project
    - Useful for automated client–project assignment based on external triggers

- **Use cases within MCP**
  - Orchestrate project and task creation from conversational interfaces
  - Automate time and task management workflows that rely on ProWorkflow data
  - Keep client and contact information aligned with project structures via tools

## Configuration
- Use the static server URL: `https://mcp.pipedream.net/v2`
- Add the server to your MCP client and authenticate during setup
- Further configuration details are available on the linked configuration page from Pipedream.
