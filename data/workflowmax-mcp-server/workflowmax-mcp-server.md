# WorkflowMax MCP Server

All-in-one project and job management MCP server integration for WorkflowMax.

## Overview
The WorkflowMax MCP Server exposes WorkflowMax’s project and job management capabilities through the Model Context Protocol (MCP), allowing compatible chat or agent clients to interact with WorkflowMax via a unified MCP endpoint.

## MCP Server URL
- Base MCP server URL (static for all clients):
  - `https://mcp.pipedream.net/v2`
- Authentication is performed when adding the server to your MCP-compatible application.

## Features
- **MCP-based access** to WorkflowMax
  - Connects WorkflowMax’s all-in-one project and job management features to MCP-enabled chat/agent clients.
  - Uses a single, shared MCP endpoint for all clients.
- **Static server endpoint**
  - One universal URL (`https://mcp.pipedream.net/v2`) works for all clients.
- **Client-agnostic integration**
  - Can be added to any supported MCP-compatible chat client or application.
- **Configuration documentation**
  - A dedicated configuration page (linked from the app) explains how to add and configure the server in different clients.

## Getting Started
1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add this server URL to your MCP-compatible chat client or application.
3. Authenticate within your client when prompted.
4. (Optionally) Refer to the linked configuration page in the app for client-specific setup steps.
