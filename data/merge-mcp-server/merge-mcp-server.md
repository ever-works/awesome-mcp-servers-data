# Merge MCP Server

**Category:** Data Access & Integration – MCP Servers  
**Brand:** Merge  
**Slug:** `merge-mcp-server`

## Description
Merge MCP Server exposes Merge’s unified API for HR, payroll, accounting, ticketing, CRM, and ATS systems as an MCP-compatible service. It lets you integrate and work with multiple business systems through a single MCP endpoint instead of maintaining separate connections to each provider.

Source: <https://mcp.pipedream.com/app/merge>

## Features
- **Unified API across domains**
  - Single integration layer for:
    - HR systems
    - Payroll platforms
    - Accounting tools
    - Ticketing systems
    - CRM platforms
    - ATS (Applicant Tracking Systems)
- **MCP-compatible server**
  - Exposes Merge’s unified API as a Model Context Protocol (MCP) server
  - Designed to be added as a server to MCP-compatible chat / agent clients
- **Single static endpoint**
  - MCP server URL: `https://mcp.pipedream.net/v2`
  - Same URL for all clients; authentication handled when adding the server to your application
- **Multi-system integrations via one endpoint**
  - Access multiple third-party business tools through one MCP endpoint
  - Reduces need for separate point-to-point integrations
- **Configuration guidance**
  - Integration instructions available through a dedicated configuration page (linked from the app)

## Integration & Usage
- Add `https://mcp.pipedream.net/v2` as an MCP server in your compatible client.
- Authenticate during server setup within your application.
- Use MCP tools to query or interact with connected HR, payroll, accounting, ticketing, CRM, and ATS systems via Merge’s unified API.
