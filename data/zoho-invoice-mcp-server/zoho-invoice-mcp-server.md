# Zoho Invoice MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Zoho  
**Slug:** zoho-invoice-mcp-server

## Description
Zoho Invoice MCP Server provides a Model Context Protocol (MCP) interface to Zoho Invoice, an online invoicing platform for freelancers and small businesses. It enables AI agents and MCP-compatible clients to create and manage invoices, handle payment collection workflows, and automate billing processes via Zoho Invoice.

## Features
- MCP interface to Zoho Invoice for use with compatible AI/chat clients
- Supports creating and managing invoices
- Supports handling payment collection workflows
- Enables automation of billing processes for freelancers and small businesses
- Uses a static MCP server URL for all clients: `https://mcp.pipedream.net/v2`

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2` (static for all clients; authentication occurs when adding the server in the client)
- **Integration platform:** Powered by Pipedream Connect
