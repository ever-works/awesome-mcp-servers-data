# Payhere MCP Server

Easy payment link creation and management for MCP-based automations.

## Overview
The Payhere MCP Server is an MCP-compatible service (hosted via Pipedream Connect) that lets you generate and manage payment links from within MCP-enabled applications and chat clients using a single static server URL.

- **Category:** Business & Commerce – MCP Servers  
- **Tags:** payments, billing, subscriptions  
- **Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **Static MCP Server Endpoint**  
  - Uses a single URL (`https://mcp.pipedream.net/v2`) for all clients.  
  - Authentication is performed when adding the server to your MCP-compatible application.

- **MCP Integration**  
  - Designed to plug into MCP-based automations and chat clients.  
  - Can be added to various MCP-compatible chat interfaces via their MCP server configuration.

- **Payment Link Workflows**  
  - Supports creating Payhere payment links suitable for payments, billing, and subscriptions (as described in the item metadata).  
  - Intended for use in automated workflows where payment links need to be generated or managed programmatically.

- **Centralized Configuration**  
  - Configuration and setup details are available through a dedicated configuration page on the Pipedream site.  
  - Managed and hosted by Pipedream Connect.

## Integration & Usage
- Add the MCP server URL to your MCP-compatible application or chat client.  
- Authenticate during setup within the client.  
- Use the server’s tools (via MCP) to create and manage Payhere payment links as part of your automations.
