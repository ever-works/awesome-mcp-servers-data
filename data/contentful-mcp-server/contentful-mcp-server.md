# Contentful MCP Server

API-first content platform MCP server using OAuth authentication, exposing Contentful’s content APIs to MCP-based tools.

- **Name:** Contentful MCP Server (OAuth)
- **Category:** Content Management MCP Servers
- **Brand:** Contentful
- **Tags:** cms, content-management, api-first

## Features

- **MCP Server Endpoint**  
  - Static MCP server URL usable across compatible MCP clients:  
    - `https://mcp.pipedream.net/v2`
  - Designed to integrate with any MCP-capable chat or agent client.

- **Authentication**  
  - Uses OAuth authentication to connect to Contentful.  
  - Authentication occurs when adding the server within your MCP-compatible application.

- **Contentful Integration**  
  - Exposes Contentful’s API-first content platform to MCP tools.  
  - Enables programmatic access to Contentful content via MCP-based workflows and clients.

- **Client Integration Guidance**  
  - Can be added to various chat/agent clients that support MCP.  
  - Additional setup and configuration details available via the referenced configuration documentation (outside this summary).
