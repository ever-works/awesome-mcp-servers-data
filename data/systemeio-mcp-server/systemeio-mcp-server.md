# Systeme.io MCP Server

## Overview
Systeme.io MCP Server exposes Systeme.io’s all‑in‑one marketing platform (funnels, email, automation, CRM) to Model Context Protocol (MCP) clients via a static MCP endpoint operated by Pipedream.

- **Name:** Systeme.io MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Tags:** marketing-automation, funnels, CRM  
- **Provider / Brand:** Systeme.io (served via Pipedream Connect)  
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server** – Accessible from any MCP-enabled client using a single static URL.
- **Systeme.io integration surface** – Designed to expose Systeme.io’s core marketing capabilities to MCP clients, including:
  - Sales and marketing funnels
  - Email marketing tools
  - Marketing automation workflows
  - CRM-style contact and lead management
- **Client-agnostic endpoint** – Same URL works across different MCP chat or agent clients; authentication is handled when adding the server in each client.
- **Configuration documentation** – A dedicated configuration page (via Pipedream) describes how to add and use the server in supported apps.
- **Operated by Pipedream Connect** – Server hosting and connection layer provided by Pipedream’s integration platform.

## Usage
- **MCP server URL:**
  ```text
  https://mcp.pipedream.net/v2
  ```
- Add this URL as an MCP server in your chat/agent client, then follow that client’s steps to authenticate with Systeme.io.
