# Locate MCP Server

Cloud-based inventory management MCP server that exposes the Locate inventory management system via a standard MCP endpoint, enabling programmatic access from compatible chat or agent clients.

- **Name:** Locate MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Brand:** Locate  
- **MCP Endpoint URL:** `https://mcp.pipedream.net/v2`  
- **Source URL:** https://mcp.pipedream.com/app/locate  
- **Intended Users:** Small to mid-size businesses using Locate for inventory management

## Features

- **Locate MCP endpoint** – Exposes the Locate cloud-based inventory management system as an MCP server so agents and tools can interact with it programmatically.
- **Programmatic inventory access** – Designed to allow querying and updating of inventory data via MCP-compatible clients (e.g., chat-based agents and tools).
- **Static server URL** – Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) that works for all clients.
- **Per-client authentication** – Authentication is performed when adding the server in each application / chat client (exact auth details depend on the client’s configuration flow).
- **Multi-client compatibility** – Can be added to different chat or agent clients that support MCP, using their respective configuration flows.
- **Configuration documentation** – A dedicated configuration page (via Pipedream) provides setup instructions for supported clients.
- **Cloud-based infrastructure** – Runs as a cloud service, suitable for small to mid-sized businesses already using Locate inventory management.

> Note: The underlying capabilities of Locate (inventory, warehouse, and order management) are made available through the MCP interface, but specific operation lists, schemas, or field-level details are not described in the provided content.
