# GoDial MCP Server

MCP Server for GoDial, exposing auto‑dialing, contact list management, and sales call tracking features to MCP tools.

---

## Overview
The GoDial MCP Server connects GoDial’s mobile calling and sales app to MCP-compatible clients. It lets you work with contact lists, automate outbound calling, and track call outcomes from within your MCP tools via a single static endpoint.

**MCP Server URL**  
`https://mcp.pipedream.net/v2`

This static URL is used for all clients; authentication is handled when you add the server to your application.

---

## Features
- **Auto-dialing**
  - Initiate automatic outbound calls to contacts from within MCP-enabled tools.
- **Contact list management**
  - Import contact lists into GoDial.
  - Access and manage contacts exposed through the MCP server.
- **Sales call tracking**
  - Save call status (e.g., completed, unanswered, etc.).
  - Track sales-related call outcomes via MCP.
- **Callback scheduling**
  - Schedule follow-up calls (callbacks) for contacts.
- **Call notes**
  - Add and store notes about calls to support sales workflows.
- **MCP integration**
  - Single, static MCP server endpoint for all clients.
  - Usable from multiple chat or MCP-compatible applications once configured.
