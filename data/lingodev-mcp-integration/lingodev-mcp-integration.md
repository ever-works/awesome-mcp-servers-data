# Lingo.dev MCP Integration

**Category:** MCP Server / Tool Directory Entry  
**Brand:** lingodev  
**Source:** <https://github.com/lingodotdev/lingo.dev/blob/main/mcp.md>

## Overview
The Lingo.dev MCP Integration exposes Lingo.dev’s localization engine through the Model Context Protocol (MCP), allowing agents and tools that speak MCP to perform multilingual and localization-aware interactions.

## Features
- **MCP-compatible integration** – Provides an MCP interface so agents can call Lingo.dev as an external tool.
- **Access to Lingo.dev localization engine** – Uses Lingo.dev’s core localization capabilities from within MCP-based workflows.
- **Multilingual interaction support** – Enables agents to handle interactions across many languages via Lingo.dev.
- **Localization for agent inputs/outputs** – Designed to help agents localize or translate content as part of their conversational or task flows.
