# Chatrace MCP Server

**Category:** Messaging MCP Servers  
**Tags:** chatbots, marketing-automation, customer-support

## Overview
Chatrace MCP Server exposes Chatrace’s chatbot-driven sales, marketing, and customer support automation capabilities to MCP-compatible tools and chat clients. It enables you to automate interactions and workflows across multiple channels through a unified MCP endpoint.

## Features
- **MCP-compatible endpoint**: Provides a static MCP server URL that can be used from any compatible client.
- **Chatbot-driven automation**: Connects to Chatrace’s chatbot engine to automate:
  - Sales interactions
  - Marketing campaigns and flows
  - Customer support conversations
- **Omnichannel support**: Designed to operate "across all channels" supported by Chatrace, allowing centralized automation of user conversations.
- **Reusable server URL**: One static URL (`https://mcp.pipedream.net/v2`) works for every client; authentication is handled when adding the server to an app.
- **Client-agnostic integration**: Can be added to different chat clients that support MCP, enabling consistent access to Chatrace features.
- **Configuration documentation**: A dedicated configuration page (via Pipedream) provides setup details for various clients and environments.

## MCP Server URL
- **Base URL:** `https://mcp.pipedream.net/v2`
- **Usage:** Same URL for all clients; authentication is performed during server addition/configuration.

## Integration & Setup
- Add the MCP server to any supported chat client using the static URL.
- Follow client-specific instructions from the configuration page hosted by Pipedream.
