# Remote MCP Directory Server

**Website:** https://mcp.remote-mcp.com  
**Category:** MCP Server Directories & Lists  
**Brand:** remote-mcp

## Overview
Remote MCP Directory Server is an open directory service for Model Context Protocol (MCP) servers. It indexes and exposes available MCP servers so that agents and tools can discover, browse, and integrate them more easily.

## Features
- **Open MCP directory server** – Publicly accessible at `https://mcp.remote-mcp.com`.
- **MCP server indexing** – Collects and maintains a list of available MCP servers.
- **Discovery endpoint** – Exposes directory data so agents can discover compatible MCP servers.
- **Integration-oriented** – Designed to be consumed by agents/tools that speak MCP for easier integration workflows.
