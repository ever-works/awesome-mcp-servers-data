## Overview
Dingo is an open‑source tool for evaluating the quality of AI data, models, and applications. This MCP server exposes Dingo’s rule‑based data quality checks through the Model Context Protocol, allowing AI agents and tools to run and interact with Dingo evaluations programmatically.

## Features
- **Comprehensive quality evaluation** for AI data, models, and applications.
- **Rule-based data quality checks** that can be defined and executed against datasets.
- **MCP server integration**, enabling access via the Model Context Protocol.
- **Agent compatibility**, allowing AI agents to trigger and consume evaluation results.
- **Open-source implementation** under the Apache-2.0 license.
- **Example projects and scripts** in the repository to demonstrate usage.
- **Multiple app front-ends** (e.g., standard app and Gradio-based app) present in the codebase, indicating both UI and programmatic interaction options.

## Technical Details
- **License:** Apache-2.0
- **Repository structure includes:** core `dingo` package, `app` and `app_gradio` front-ends, `docs`, `examples`, `scripts`, and test suite.

## Use Cases
- Running automated data quality checks before training or deploying AI models.
- Integrating data and model quality evaluation into AI agent workflows via MCP.
- Evaluating the reliability of AI applications using rule-based criteria.
