# Enrow MCP Server

**Category:** Business & Commerce – MCP Servers  
**Website:** https://mcp.pipedream.com/app/enrow

## Overview
Enrow MCP Server is a Model Context Protocol (MCP) server hosted via Pipedream that connects applications to Enrow. It focuses on using deeply verified email addresses to support lead generation and sales outreach workflows.

## Features
- **MCP server endpoint**: Static MCP server URL (`https://mcp.pipedream.net/v2`) used to connect from compatible chat or MCP-enabled clients.
- **Enrow integration**: Connects to an Enrow account to access Enrow’s verified email data and outreach capabilities.
- **Lead generation support**: Designed to help identify and reach leads using verified contact information.
- **Sales workflow support**: Intended for use within sales and outreach workflows to help secure more deals with accurate contact data.
- **Multi-client compatibility**: Can be added to various chat or MCP-compatible clients (setup is guided per client type).
- **Central configuration**: Configuration handled through the Pipedream Enrow MCP configuration page, where you connect your Enrow account and select a client.

## Setup
- Use the static MCP endpoint: `https://mcp.pipedream.net/v2`.
- Connect your Enrow account through the Pipedream configuration interface.
- Add the server to your chat/MCP client following client-specific instructions.
