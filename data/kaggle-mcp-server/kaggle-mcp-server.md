# Kaggle MCP Server

**Website:** https://mcp.pipedream.com/app/kaggle  
**Category:** Documentation & Learning Resources  
**Tags:** datasets, machine-learning, community

## Description
Kaggle MCP Server is an MCP (Model Context Protocol) server that connects Kaggle’s machine learning and data science resources and community datasets to AI tools. It runs on Pipedream’s MCP server to let compatible AI environments access and work with Kaggle data.

## Features
- Connects a Kaggle account to AI tools via MCP
- Works with AI environments such as Cursor, Claude, and Windsurf
- Provides access to Kaggle’s machine learning and data science resources
- Enables use of Kaggle community datasets within AI tools

## Integrations
- Cursor (via MCP)
- Claude (via MCP)
- Windsurf (via MCP)
- Pipedream MCP server
