# Honeybadger MCP Server

**Category:** Monitoring  
**Brand:** Honeybadger  
**Slug:** honeybadger-mcp-server  
**Source:** https://mcp.pipedream.com/app/honeybadger

## Description
The Honeybadger MCP Server exposes Honeybadger error, outage, and service degradation data through the Model Context Protocol (MCP), allowing AI agents and MCP-compatible clients to incorporate production monitoring and incident information directly into their workflows.

## Features
- **MCP-compatible server endpoint**
  - Static MCP server URL: `https://mcp.pipedream.net/v2`
  - Works across any supported MCP client.
- **Honeybadger integration**
  - Connects to a Honeybadger account to access monitoring data.
  - Designed for zero-instrumentation access to existing Honeybadger error and outage data.
- **Error and incident visibility**
  - Surfaces information about errors, outages, and service degradation for use by AI agents.
  - Supports monitoring and incident management workflows through MCP tools.
- **Client-agnostic configuration**
  - Can be added to different chat or MCP-capable applications.
  - Configuration guidance available via a centralized configuration page (per client instructions).
- **Tool-based interactions**
  - Exposes Honeybadger data as “tools” within MCP clients (actions load dynamically within the client environment).
