# Triggre MCP Server

MCP Server integration for Triggre’s no-code platform, allowing MCP-compatible agents to create and manipulate custom business applications and workflows built on Triggre.

## Overview
- **Name:** Triggre MCP Server  
- **Category:** Workflow automation MCP servers  
- **Brand:** Triggre  
- **Source URL:** https://mcp.pipedream.com/app/triggre  
- **MCP Server Base URL:** `https://mcp.pipedream.net/v2`

## Features
- **No-code application integration**  
  - Connects MCP-compatible agents to Triggre’s no-code platform.  
  - Enables creation and manipulation of custom business applications built on Triggre.

- **Workflow automation**  
  - Allows agents to work with Triggre-based workflows, supporting automation of internal processes.

- **Static MCP server URL**  
  - Uses a single, static URL (`https://mcp.pipedream.net/v2`) for all clients.  
  - Authentication occurs when adding the server to the client / application.

- **Client-agnostic setup**  
  - Can be added to different MCP-capable chat clients or applications using the same server URL.  
  - Setup instructions are available per chat client via the configuration flow on the source site.

- **Pipedream Connect integration**  
  - Hosted and powered by Pipedream Connect as the MCP hosting layer.

## Usage
- Add the server URL `https://mcp.pipedream.net/v2` to your MCP-compatible chat client or application.  
- Authenticate during the add-server flow as prompted by the client.
