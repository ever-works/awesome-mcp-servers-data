# UniOne MCP Server

An MCP server that exposes UniOne’s email delivery service to MCP-compatible tools, enabling programmatic email sending and infrastructure management.

## Overview
- **Type:** MCP server / integration
- **Category:** Messaging MCP Servers
- **Provider / Brand:** UniOne (via Pipedream MCP)
- **Purpose:** Provide reliable email delivery and management capabilities to applications and chat clients through the Model Context Protocol (MCP).

## Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Works across all supported MCP clients.

- **UniOne email delivery integration**  
  - Bridges UniOne’s email delivery infrastructure into MCP tools.  
  - Designed for reliable transactional and messaging use cases (based on UniOne’s core service focus).

- **Account-based configuration**  
  - Connect and configure a UniOne account before use.  
  - Account check / verification step when setting up the server.

- **Client-agnostic usage**  
  - Intended for use from multiple MCP-compatible chat clients and applications.  
  - Per-client setup instructions available via the UI (not detailed in the provided content).

- **Tool exposure via MCP**  
  - Exposes UniOne’s capabilities as MCP tools / actions (specific tools are not listed in the provided content and appear as “Loading actions...”).

- **Configuration documentation**  
  - Central configuration reference available via a dedicated configuration page (exact steps not detailed in the provided content).
