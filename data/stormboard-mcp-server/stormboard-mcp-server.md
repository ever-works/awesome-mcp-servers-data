# Stormboard MCP Server

## Overview
Stormboard MCP Server connects MCP-compatible agents to Stormboard’s online sticky-note whiteboard, enabling AI agents and tools to collaborate with and manipulate Stormboard boards for meetings, brainstorming, and creative projects.

- **Category:** Project Management MCP Server
- **Provider / Brand:** Pipedream
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-Compatible Server**
  - Exposes Stormboard functionality through the Model Context Protocol (MCP) so MCP-enabled chat clients and agents can integrate with Stormboard.
- **Stormboard Integration**
  - Connects to Stormboard’s online sticky-note whiteboard environment.
  - Supports collaborative use cases such as meetings, brainstorming sessions, and creative project planning.
- **Static Server URL**
  - Uses a single static MCP server URL: `https://mcp.pipedream.net/v2`.
  - Same URL works across all compatible MCP clients.
- **Client-Agnostic Setup**
  - Designed to be added to various MCP-capable chat clients or applications.
  - Authentication occurs when adding/configuring the server in the client.
- **Collaboration & Brainstorming Support**
  - Allows agents to work with sticky notes and boards to facilitate collaboration, ideation, and structured discussions.

## Usage
- Add the MCP server URL `https://mcp.pipedream.net/v2` to your MCP-compatible chat client or application.
- Authenticate during setup as required by the client.
- Use the client’s MCP integration features to interact with Stormboard boards (e.g., for whiteboarding, collaboration, and brainstorming flows).
