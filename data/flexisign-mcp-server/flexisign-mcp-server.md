# FlexiSign MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** FlexiSign  
**Slug:** `flexisign-mcp-server`

## Overview
FlexiSign MCP Server is an MCP integration that allows tools and applications to create and manage electronic signatures using FlexiSign’s e-signing capabilities. It exposes a static MCP server endpoint that can be added to compatible chat or agent clients.

## Features
- **MCP server integration for FlexiSign** – Connects MCP-compatible tools and agents to FlexiSign’s e-signature functionality.
- **Electronic signature creation** – Supports creating new e-signatures through FlexiSign.
- **Signature management** – Enables MCP tools to manage e-signatures using FlexiSign’s electronic signing features.
- **Static MCP server URL** – Uses a single static endpoint for all clients:  
  `https://mcp.pipedream.net/v2`
- **Client-agnostic endpoint** – Same URL works across different MCP-compatible chat or agent clients.
- **Authentication at client setup** – Authentication is handled when adding the MCP server to your application or chat client.
- **Configuration via client or docs** – Can be set up directly in supported clients or via the general MCP configuration documentation.

## Technical Details
- **MCP server URL:** `https://mcp.pipedream.net/v2`
- **Integration type:** Hosted MCP server powered by Pipedream Connect
