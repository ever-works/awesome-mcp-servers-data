# sendSMS MCP Server

**Category:** Messaging MCP Servers  
**Brand:** sendSMS  
**Slug:** sendsms-mcp-server

## Description
sendSMS MCP Server integrates the sendSMS platform into MCP-based applications to support chat, SMS marketing, and mobile advertising workflows via Pipedream Connect.

## Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL usable across compatible clients:  
    - `https://mcp.pipedream.net/v2`
  - Designed to be added as an external MCP server to chat clients and other MCP-enabled tools.

- **Authentication at client setup**  
  - Authentication is performed when adding the server to the application rather than via per-client URLs.

- **Chat integration**  
  - Can be used within chat clients that support MCP servers to trigger or manage SMS and mobile messaging workflows (implied by "Select your chat client to learn how to get started").

- **SMS marketing workflows**  
  - Supports use of sendSMS for SMS-based marketing campaigns.

- **Mobile advertising workflows**  
  - Enables mobile advertising use cases built on top of sendSMS via MCP.

- **Configuration guidance**  
  - Additional setup and configuration details are available on a dedicated configuration page (not included in the provided content but referenced as the source of full instructions).
