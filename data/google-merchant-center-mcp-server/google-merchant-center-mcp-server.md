# Google Merchant Center MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Google

## Overview
The Google Merchant Center MCP Server integrates Google Merchant Center with MCP-compatible clients, allowing you to manage products and feeds for Google Shopping and other Google surfaces directly from your chat or developer tools.

Server base URL:
```text
https://mcp.pipedream.net/v2
```

## Features
- **MCP-compatible server**  
  - Exposes Google Merchant Center operations as MCP tools/actions.  
  - Uses a single static server URL (`https://mcp.pipedream.net/v2`) for all supported clients.

- **Google Merchant Center integration**  
  - Connect a Google Merchant Center account for product management.  
  - Manage product data used across Google Shopping and other Google surfaces.

- **Available tools / actions**  
  - **Create Product**  
    - Creates a new product in your Google Merchant Center account.  
    - Action implementation and parameters documented in the Pipedream GitHub repository.  
  - **Update Product**  
    - Updates an existing product in your Google Merchant Center account.  
    - Action implementation and parameters documented in the Pipedream GitHub repository.

- **Client setup**  
  - Works with multiple MCP-compatible chat clients (e.g., ChatGPT).  
  - Authentication occurs when adding the server to your application.  
  - Additional setup and configuration details available on the Pipedream MCP configuration page.
