# DopplerAI MCP Server

## Overview
DopplerAI MCP Server is an MCP (Model Context Protocol) server that exposes the DopplerAI platform through a single API, providing unified access to:
- AI memory
- Data processing capabilities
- Scalable vector database operations

It is delivered via Pipedream Connect and can be used as a static MCP endpoint in compatible AI/chat clients.

## Features
- **Single unified API** for DopplerAI functionality
- **Memory access** for AI agents via MCP
- **Data processing integration** accessible through the MCP server
- **Scalable vector database operations** exposed through MCP
- **Static MCP endpoint** that works across supported clients
- **Client-agnostic configuration**: same base URL for all MCP-compatible chat clients

## MCP Server Endpoint
- Base MCP Server URL: `https://mcp.pipedream.net/v2`
- Static URL shared across clients; authentication is handled when adding the server to your application

## Integration & Configuration
- Can be added to various MCP-compatible chat clients using the static URL
- Detailed setup and configuration instructions are available on the associated Configuration page (via Pipedream)
