# Autodesk MCP Server

Autodesk MCP Server is an MCP integration that exposes Autodesk’s 3D design, engineering, and construction software capabilities to MCP-compatible AI agents and tools.

- **Website / Source**: https://mcp.pipedream.com/app/autodesk
- **Category**: Media Processing MCP Servers
- **Brand**: Autodesk
- **MCP Server URL**: `https://mcp.pipedream.net/v2`

## Features

- **MCP-compatible integration**: Connects Autodesk capabilities to any MCP-compatible AI agent or tool.
- **3D design support**: Interfaces with Autodesk tools used for 3D design workflows.
- **Engineering workflows**: Exposes engineering-related functionality from Autodesk software.
- **Construction software capabilities**: Provides access to Autodesk construction-focused features through MCP.
- **Static, reusable server URL**: Uses a single static MCP endpoint (`https://mcp.pipedream.net/v2`) for all clients.
- **Client-agnostic usage**: Designed to work with any MCP-capable chat client or application.
- **Authentication on add/connect**: Authentication occurs when adding the server to an application or client.

## Usage

- Add the MCP server to an MCP-compatible chat client or application using:
  - **Server URL**: `https://mcp.pipedream.net/v2`
- Follow your client’s MCP configuration flow to authenticate and start accessing Autodesk capabilities.
