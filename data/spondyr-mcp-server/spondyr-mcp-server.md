# Spondyr MCP Server

**Category:** Document Management MCP Servers  
**Brand:** Spondyr  
**Slug:** `spondyr-mcp-server`  
**Source URL:** https://mcp.pipedream.com/app/spondyr

Spondyr MCP Server exposes the Spondyr correspondence template management and distribution API to MCP-compatible environments, enabling automated document and communication workflows from within MCP clients.

---

## Features

- **MCP-compatible server**  
  - Accessible via a static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Works with any supported MCP client or chat environment.

- **Correspondence template management**  
  - Integrates with Spondyr’s API for managing correspondence templates used in documents and communications.

- **Template-based document and communication workflows**  
  - Enables applications to automate generation and distribution of templated documents and messages.

- **Distribution API exposure**  
  - Surfaces Spondyr’s distribution capabilities through MCP so clients can trigger sending or distributing correspondence from within their workflows.

- **Static configuration model**  
  - Single static URL for all clients; authentication is handled when adding the server to an application.

- **Pipedream Connect integration**  
  - Provided through Pipedream Connect infrastructure for easy integration into existing automation setups.

- **Documentation & configuration guidance**  
  - Full configuration instructions available via the linked configuration page on Pipedream.

---

## Integration

- Add the MCP server by configuring your chat client or MCP-compatible app with the static URL: `https://mcp.pipedream.net/v2`.
- Authentication is performed during server addition to the client or application.
