## Overview

Oracle Autonomous AI Database MCP Server is a multi-tenant, built-in feature of Autonomous AI Database Serverless supporting database versions 19c and 26ai. It exposes Model Context Protocol (MCP) endpoints, enabling AI agents and clients such as Claude Desktop and OCI AI Agent to invoke tools defined with Select AI Agent.

## Features

- **Native Integration**: Run MCP Server natively inside Oracle Autonomous AI Database instance (19c/26ai)
- **No Standalone Infrastructure**: No need for standalone servers or cluster management
- **AI Client Compatibility**: Works with Claude Desktop, Visual Studio Code with Cline, and OCI AI Agent
- **Enterprise Security**: Deep integration with database security features including RBAC, encryption, auditing, VPD policies
- **Select AI Agent**: Define, run, and govern AI agents inside the database using in-database tools, external REST tools, or MCP Servers
- **OCI Console Management**: Directly manage MCP server within the OCI console

## Security & Governance

Unlike third-party database MCP servers, the Autonomous AI Database MCP Server offers:

- Existing roles, lockdown profiles, and access control lists (ACLs)
- Built-in auditing and encryption
- Role-based access control (RBAC)
- Virtual private database (VPD) policies
- Compliance with enterprise security standards
