# Zoho Bigin MCP Server

**Category:** Business & Commerce → MCP Servers  
**Brand:** Zoho  
**Slug:** `zoho-bigin-mcp-server`

## Overview
Zoho Bigin MCP Server is an MCP-based integration that connects to Zoho Bigin, a lightweight CRM, to expose tools for working with customer and pipeline data through compatible MCP chat or agent clients.

It uses a static MCP server URL (`https://mcp.pipedream.net/v2`) and authenticates when you add the server to your chosen application.

## Features
- **MCP-based Zoho Bigin integration**
  - Provides an MCP Server endpoint for interacting with Zoho Bigin from MCP-compatible apps.
  - Uses a single static MCP URL for all clients (`https://mcp.pipedream.net/v2`).

- **Customer & pipeline data access**
  - Designed to work with Zoho Bigin’s customer records and pipeline (sales) data.
  - Enables CRM-style interactions through tools exposed by the server (actions list is dynamically loaded in the UI).

- **Account configuration**
  - Connect a Zoho Bigin account via the configuration flow.
  - After connection, you can select your client (chat or other MCP client) to start using the tools.

- **Multi-client support**
  - Can be added to different MCP-enabled chat clients or applications.
  - Each client authenticates against the same static server URL.

- **Central configuration reference**
  - A dedicated configuration page (linked as "Configuration page") provides additional setup details and steps for various clients.

## Setup Basics
- Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server to your MCP-compatible app or chat client.
- Authenticate with your Zoho Bigin account during or after server addition.
