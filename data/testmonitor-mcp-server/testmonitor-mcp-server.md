# TestMonitor MCP Server

An MCP server that connects TestMonitor with AI tools to enable AI-driven interaction with functional, user, and acceptance test management workflows.

## Overview
TestMonitor MCP Server integrates your TestMonitor account with MCP-compatible AI tools such as Cursor, Claude, and Windsurf, allowing AI systems to interact with and help manage your testing workflows.

## Features
- Connects a TestMonitor account to Pipedream's MCP server
- Enables AI-driven interaction with TestMonitor test management workflows
- Supports functional, user, and acceptance testing workflows
- Compatible with AI tools that support MCP, including:
  - Cursor
  - Claude
  - Windsurf

## Integrations
- TestMonitor
- Pipedream MCP
- Cursor
- Claude
- Windsurf
