# Demandbase MCP Server

**Category:** Business & Commerce MCP Servers  
**Vendor:** Demandbase / Pipedream  
**Slug:** `demandbase-mcp-server`

An MCP server that exposes Demandbase’s account-based marketing (ABM) capabilities to MCP-compatible clients, enabling B2B sales and marketing workflows directly from chat or other MCP applications.

---

## Features

- **MCP-compatible ABM server**  
  - Provides a static MCP server endpoint: `https://mcp.pipedream.net/v2`  
  - Can be added to any MCP-capable chat or client application.

- **Demandbase account connection**  
  - Connects to a Demandbase account for use within MCP clients.  
  - Authentication occurs when adding the server to an application.

- **Client-agnostic configuration**  
  - Same server URL works across different clients.  
  - Configuration guidance available via a dedicated configuration page.

- **B2B sales & marketing focus**  
  - Designed around account-based marketing (ABM) for B2B sales and marketing use cases.  
  - Intended to bring Demandbase ABM data and functionality into conversational or automation workflows.
