# Greenhouse Job Board MCP Server

## Overview
The Greenhouse Job Board MCP Server exposes the Greenhouse Job Board API via the Model Context Protocol (MCP), enabling applications and chat clients to build custom careers and department-level job pages and related job board functionality.

- **Type:** MCP server / developer integration
- **Category:** Business & Commerce MCP Servers
- **Brand:** Greenhouse (via Pipedream)
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- **Greenhouse Job Board API integration**  
  Access Greenhouse Job Board data through an MCP-compatible server.

- **Custom careers pages**  
  Use API data to build careers pages with a unique look and feel tailored to your site or application.

- **Department-level pages**  
  Construct pages focused on specific departments (e.g., engineering, marketing) using Greenhouse job board data.

- **Job board functionality for chat clients**  
  Integrate Greenhouse job listings and related operations into MCP-enabled chat clients and other MCP-aware tools.

- **Static MCP server URL**  
  Single static endpoint (`https://mcp.pipedream.net/v2`) usable by any compatible client.

- **Per-client authentication**  
  Authentication occurs when you add the MCP server to your application; the same URL works across clients.

- **Cross-client configuration support**  
  Designed to be added to different chat clients; configuration details are provided via the linked configuration documentation.

## Integration & Setup
