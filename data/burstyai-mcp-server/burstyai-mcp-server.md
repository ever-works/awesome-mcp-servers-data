# BurstyAI MCP Server

An MCP server integration for BurstyAI that exposes its AI autopilot workers for small business growth to MCP clients.

## Overview
- **Type:** MCP server integration
- **Category:** Business & Commerce MCP Servers
- **Use case:** Connect BurstyAI’s AI autopilot workers to MCP-compatible chat or agent clients to help automate and grow small businesses.
- **Provider:** BurstyAI (via Pipedream Connect)

## Features
- **MCP-compatible server**: Provides a Machine Control Protocol (MCP) server endpoint that can be added to any supported MCP client.
- **Static server URL**: Uses a single static URL for all clients:
  - `https://mcp.pipedream.net/v2`
- **Client-agnostic integration**: Designed to work with multiple MCP-capable chat clients and applications.
- **Authentication on add**: Authentication is handled when adding the server to your client/application (per-client auth flow, not encoded in the URL).
- **Pipedream Connect integration**: Hosted and powered by Pipedream Connect infrastructure.
