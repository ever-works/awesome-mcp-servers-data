# Mailcoach MCP Server

**Website:** https://mcp.pipedream.com/app/mailcoach  
**Category:** Business & Commerce – MCP Servers  
**Tags:** newsletter, email-marketing, campaigns

## Overview
Mailcoach MCP Server is an MCP (Model Context Protocol) server integration that connects MCP-compatible clients (such as AI/chat tools) with Mailcoach’s newsletter and email marketing platform. It allows workflows and automations built in MCP clients to interact with Mailcoach’s email delivery and list management capabilities.

## Features
- **MCP Server Integration**
  - Exposes Mailcoach functionality through a standard MCP server endpoint.
  - Designed to be added as a server in compatible chat or AI clients.

- **Static MCP Endpoint**
  - Single static URL for all clients:
    - `https://mcp.pipedream.net/v2`
  - The same URL works across different MCP-compatible applications.

- **Client-Agnostic Setup**
  - Can be added to multiple chat / MCP client applications.
  - Authentication occurs when adding the server to your chosen application.

- **Newsletter & Email Marketing Connectivity**
  - Bridges MCP clients to Mailcoach’s newsletter workflows.
  - Uses Mailcoach’s underlying email delivery and list management infrastructure (details not specified in source but implied by description and product scope).

- **Configuration Documentation**
  - Full configuration instructions available via the linked configuration page (on the Pipedream site).

## Integration & Usage
- Add `https://mcp.pipedream.net/v2` as an MCP server in your compatible client.
- Authenticate within the client during server setup.
- Use the Mailcoach MCP Server within workflows to connect newsletter-related actions to Mailcoach.
