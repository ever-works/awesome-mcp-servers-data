# Dappier MCP Server

**Category:** AI Integration – MCP Servers  
**Brand:** Dappier  
**Source:** https://github.com/DappierAI/dappier-mcp

## Description
Dappier MCP Server is a RAG-as-a-service Model Context Protocol (MCP) server that lets AI tools perform retrieval‑augmented generation over APIs and data sources. It is designed to enable fast, real-time web search and access to structured data from news, financial markets, sports, entertainment, weather, and other sources via Dappier.

## Features
- **RAG-as-a-service**
  - Provides retrieval-augmented generation capabilities as a service for AI tools and agents.
  - Integrates external APIs and data sources into LLM workflows via MCP.

- **Model Context Protocol (MCP) server**
  - Implements an MCP-compatible server for use with MCP clients and AI agent frameworks.
  - Exposes tools/endpoints that LLMs can call for retrieval and query execution.

- **API and data source integration**
  - Connects to Dappier-powered APIs for:
    - Real-time web search
    - News content
    - Financial markets data
    - Sports information
    - Entertainment content
    - Weather data
    - Other premium or trusted media data sources (via Dappier platform).

- **Real-time data access**
  - Designed for fast, up-to-date information retrieval suitable for AI agents that need live data.

- **Open-source implementation**
  - Available on GitHub under the MIT license.
  - Includes Dockerfile and Python project configuration (`pyproject.toml`) for deployment and development.

- **Tooling & ecosystem metadata**
  - `glama.json` and `smithery.yaml` provided for integration with MCP-related tooling and registries.

## Technical Details
- **Language:** Python (packaged as a Python project with `pyproject.toml`).
- **Containerization:** Dockerfile available for container-based deployment.
