# EZ Texting MCP Server

## Overview
The **EZ Texting MCP Server** exposes EZ Texting’s SMS marketing and messaging capabilities through the Model Context Protocol (MCP). It allows AI agents and MCP-compatible applications to send and manage text message campaigns via EZ Texting.

- **Type:** MCP server / messaging integration
- **Platform:** Pipedream MCP
- **Category:** Messaging MCP servers
- **Tags:** sms, marketing, campaigns
- **MCP server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-accessible SMS tools**
  - Provides an MCP server interface to EZ Texting’s SMS marketing and messaging platform.
  - Designed for use by AI agents and MCP-compatible chat clients.

- **Static MCP server endpoint**
  - Uses a single static URL (`https://mcp.pipedream.net/v2`) that works across clients.
  - Authentication occurs when adding the server to your MCP-compatible application.

- **Client integration workflow**
  - Connect an EZ Texting account via Pipedream.
  - Select a client/chat application to see specific integration steps.
  - Optionally reference a separate configuration page for full MCP configuration details.

- **Tool-based actions (MCP tools)**
  - The server exposes EZ Texting actions as “available tools” within MCP.
  - Tools are auto-discovered/loaded by compatible MCP clients (exact tools not listed in the source content but are presented as loadable actions).
