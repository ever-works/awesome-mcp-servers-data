# Talend MCP Server

**Category:** Data Access & Integration MCP Servers  
**Brand:** Talend  
**Source:** https://mcp.pipedream.com/app/talend

## Description
Talend MCP Server exposes Talend’s data integration and data management capabilities through an MCP-compatible server interface, enabling AI-driven data workflows to access and work with data via a standard MCP endpoint.

The server is hosted via Pipedream Connect and can be used by any MCP-compatible chat client or application.

## MCP Server Endpoint
- **Base URL:** `https://mcp.pipedream.net/v2`
- **Usage:** This static URL is used for all clients; authentication is performed when adding the server to the client/application.

## Features
- MCP-compatible interface to Talend capabilities
- Supports AI-driven data workflows over MCP
- Central static MCP endpoint usable by multiple clients
- Authentication handled at the time of adding the server to your application
- Integrates with MCP-compatible chat clients and applications

## Setup & Integration
- Add the server URL `https://mcp.pipedream.net/v2` to your MCP-compatible chat client or app
- Follow client-specific instructions (via the client’s "Add server" or configuration flow)
- Additional configuration details are available on the Pipedream MCP Configuration page
