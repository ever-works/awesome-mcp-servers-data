# BigMailer MCP Server

**Description**  
The BigMailer MCP Server is an integration that connects BigMailer’s email marketing platform with MCP-compatible AI tools, giving agencies and makers programmatic access to BigMailer features.

**Category**  
- Business & Commerce → MCP Servers

**Use Cases**  
- Automate email marketing workflows from AI coding tools
- Enable AI agents to create and manage email campaigns
- Integrate BigMailer actions into developer environments

## Features
- **MCP Integration**: Exposes BigMailer functionality through an MCP server endpoint.
- **AI Tool Connectivity**: Designed to work with MCP-capable AI tools such as:
  - Cursor
  - Claude
  - Windsurf
- **Email Marketing Access**: Provides access to BigMailer’s email marketing features suitable for agencies and individual makers (e.g., programmatic campaign operations, subscriber actions — specifics depend on BigMailer’s API capabilities).
- **Pipedream-Based Implementation**: Runs as part of Pipedream’s MCP server environment, enabling workflow-style integration and automation.
