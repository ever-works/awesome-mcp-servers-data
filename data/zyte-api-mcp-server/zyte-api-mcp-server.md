# Zyte API MCP Server

## Overview
Zyte API MCP Server is an MCP-compatible server that provides unified access to Zyte’s web data extraction services. It acts as a central entry point for high-quality web scraping and structured data collection, designed to be used from MCP-enabled clients.

- **Category:** Data Access & Integration – MCP Servers  
- **Provider / Brand:** Zyte  
- **Source URL:** https://mcp.pipedream.com/app/zyte_api  
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- **Unified access to Zyte API**  
  - Connects MCP-compatible applications to Zyte’s web data extraction platform through a single MCP server.
- **Web scraping and data extraction**  
  - Supports high-quality web scraping workflows and structured data collection via the Zyte API.
- **MCP-compatible integration**  
  - Exposes Zyte’s capabilities in a format suitable for MCP clients (e.g., chat-based tools and other MCP-aware applications).
- **Static MCP server URL**  
  - Uses a single static URL (`https://mcp.pipedream.net/v2`) that works across clients, simplifying configuration.
- **Client-agnostic setup**  
  - Can be added to any supported MCP client; authentication is handled when adding the server to the application.
- **Configuration documentation**  
  - Detailed configuration instructions available via the referenced configuration page on Pipedream.

## Usage
- Add the MCP server URL `https://mcp.pipedream.net/v2` to your MCP-compatible application.
- Authenticate through your client when prompted.
- Use the client’s MCP interface to access Zyte’s web data extraction and scraping capabilities.
