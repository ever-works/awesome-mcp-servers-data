# Alerty MCP Server

Send native notifications from your applications and servers via an MCP-based automation server.

**Website:** https://mcp.pipedream.com/app/alerty  
**Provider / Brand:** Pipedream  
**Category:** Messaging MCP Servers  
**Tags:** notifications, alerting, automation

## Features

- **MCP-based notification server**: Exposes an MCP (Model Context Protocol) server for integrating Alerty notifications into compatible clients and tools.
- **Native notifications**: Enables sending native (system-level or client-native) notifications from apps and servers.
- **Single static server URL**: Uses a universal MCP server endpoint that works across clients:
  - `https://mcp.pipedream.net/v2`
- **Client-agnostic setup**: Same MCP server URL can be added to different chat or automation clients; authentication is handled when adding the server to each application.
- **Configuration documentation**: Full setup and configuration details are provided via a dedicated configuration page (linked from the app page).

## Integration & Usage

- Add the MCP server URL (`https://mcp.pipedream.net/v2`) to your client or application as an MCP server.
- Authenticate within the client during setup to start sending notifications.
