## Features

- Authentication performed when adding the server to an MCP client or application
- Designed for small businesses and sole traders to accept card payments via SumUp
- Integrates via Pipedream Connect infrastructure

## Integration & Usage

- Configure authentication during client setup
- Additional setup and configuration details available on the Pipedream configuration page (linked from the app page)
