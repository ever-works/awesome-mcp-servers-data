# Woovi MCP Server

## Overview
The Woovi MCP Server is an MCP-compatible endpoint that allows tools and chat-based applications to interact with Woovi’s payments and customer loyalty platform. It is delivered via Pipedream Connect and exposes Woovi capabilities such as Pix payments, cashback, and installment payments (Pix Parcelado) to MCP-aware clients.

**MCP Server URL:**
```text
https://mcp.pipedream.net/v2
```

## Features
- **MCP-compatible server**
  - Static MCP endpoint usable across different MCP clients.
  - Authentication is handled when you add the server in your application or chat client.

- **Woovi platform integration**
  - Access to Woovi’s payment and commerce features through MCP tools.
  - Designed for use in workflows that involve business, commerce, and customer interactions.

- **Pix payments**
  - Integrates with Woovi’s Pix payment functionality (instant payments in Brazil).

- **Customer loyalty & rewards**
  - Supports Woovi’s loyalty-related features, such as cashback.

- **Pix Parcelado (installment payments)**
  - Access to Woovi’s Pix Parcelado capabilities for installment-style Pix payments.

- **Multi-client compatibility**
  - Can be added to different MCP-enabled chat clients.
  - Central configuration is documented via the Pipedream Configuration page.

## Setup
- Use the static MCP server URL `https://mcp.pipedream.net/v2` in your MCP-compatible client.
- Configure authentication within your app or chat client when adding the server.
- Refer to the Pipedream Configuration documentation (not included here) for client-specific setup steps.
