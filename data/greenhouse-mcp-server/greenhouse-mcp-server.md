# Greenhouse MCP Server

**Description**  
An MCP (Model Context Protocol) server integration for Greenhouse, a talent acquisition suite focused on structured hiring workflows.

**Key Details**
- **Integration type:** MCP server
- **Service:** Greenhouse (ATS / recruiting / HR)
- **Provider platform:** Pipedream Connect
- **MCP server URL:** `https://mcp.pipedream.net/v2`

**Features**
- Provides an MCP-compatible endpoint to integrate Greenhouse with MCP-enabled chat or AI clients.
- Uses a single static server URL (`https://mcp.pipedream.net/v2`) for all clients; authentication is handled when adding the server to the application.
- Designed to work with Greenhouse, a structured hiring / talent acquisition platform (applicant tracking, recruiting, HR use cases).
- Can be added to various chat clients that support MCP by following per-client setup instructions.
- Central configuration guidance available via a dedicated configuration page on Pipedream.
