# MeetingPulse MCP Server

**Category:** Business & Commerce – MCP Servers  
**Website:** https://mcp.pipedream.com/app/meetingpulse

## Overview
MeetingPulse MCP Server exposes MeetingPulse’s live audience engagement capabilities—such as polls, Q&A, and feedback—through the Model Context Protocol (MCP). This allows AI agents and chat-based applications to incorporate interactive audience tools for events, HR, online meetings, sales, and marketing workflows.

## Features
- **MCP-based server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same URL works for all MCP-compatible clients; authentication is handled when adding the server to the app.

- **Audience engagement tooling (via MeetingPulse)**  
  - Polling functionality for live audiences.  
  - Q&A collection and management.  
  - Feedback capture for sessions, events, and meetings.

- **Use cases**  
  - HR engagement (all-hands, town halls, internal meetings).  
  - Events and conferences (live polls, Q&A, feedback).  
  - Online meetings and webinars.  
  - Sales and marketing sessions requiring real-time audience interaction.

- **Integration with chat clients**  
  - Can be added to MCP-compatible chat clients to let AI workflows trigger and manage polls, questions, and feedback.  
  - Documentation available via a configuration page for setup and usage details (not included here).

- **Hosted and maintained by Pipedream**  
  - Delivered as part of Pipedream Connect’s MCP ecosystem.  
  - Centralized terms and privacy via Pipedream’s policies.
