# Apiary MCP Server

**Category:** Development Tools – MCP Servers  
**Brand:** Apiary

Platform MCP server that connects AI agents and tools to Apiary’s API design, development, and documentation platform via the Model Context Protocol.

---

## Features

- **MCP server endpoint**  
  - Static MCP server URL for all compatible clients:  
    - `https://mcp.pipedream.net/v2`  
  - Works as a shared endpoint; authentication is handled when adding the server in each client/app.

- **Integration with chat / AI clients**  
  - Designed to be added as an MCP server to supported chat or AI applications.  
  - Client-specific setup instructions are available via a configuration page (not specific to a single client).

- **Apiary platform connectivity**  
  - Connects tools and AI agents to Apiary’s API design, development, and documentation platform via MCP (as described in the item metadata).  
  - Intended to enable management of API blueprints, mocks, and documentation programmatically through MCP.

- **Pipedream Connect infrastructure**  
  - Hosted and powered by Pipedream Connect, which provides the MCP server runtime and connectivity.

---

## Usage

- Use the static URL `https://mcp.pipedream.net/v2` when configuring the MCP server in your preferred chat or AI client.
- Authenticate within your client when prompted after adding the server.
- Optionally consult the platform’s configuration documentation ("Configuration" page) for client-specific setup steps.
