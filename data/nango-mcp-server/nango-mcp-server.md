# Nango MCP Server

**Description**  
An MCP server for Nango that lets MCP clients use Nango’s open-source unified API to connect with third-party SaaS integrations.

**Website**  
https://mcp.pipedream.com/app/nango

**Category**  
Data Access & Integration · MCP Servers

**Tags**  
- unified-api  
- integration  
- open-source

**MCP Server URL**  
`https://mcp.pipedream.net/v2`

## Features

- **Open-source unified API** for connecting to multiple third-party SaaS integrations through Nango.
- **Static MCP server URL** (`https://mcp.pipedream.net/v2`) usable across compatible MCP clients.
- **Client-agnostic configuration**: add the same server URL to different chat / MCP clients, with authentication handled during setup.
- **Account connection & management** via Nango: sign in to connect Nango and manage integrations and connections.
- **Configuration documentation**: dedicated configuration page for step-by-step setup instructions (via Pipedream).

### Available Tools (Actions)

7 actions are exposed as MCP tools:

1. **Proxy Post Request**  
   - Make a POST request via Nango’s proxy.

2. **Proxy Get Request**  
   - Make a GET request via Nango’s proxy.

3. **List Integrations**  
   - Retrieve a list of existing integrations.

4. **List Connections**  
   - Retrieve a list of existing connections.

5. **Get Connection**  
   - Fetch details for a specific connection.

6. **Delete Connection**  
   - Delete a specific connection.

7. **Create Integration**  
   - Create a new integration.
