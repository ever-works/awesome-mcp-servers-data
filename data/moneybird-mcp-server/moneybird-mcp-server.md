# Moneybird MCP Server

Accounting software integration for the MCP/Pipedream environment.

- **Name:** Moneybird MCP Server  
- **Category:** Finance / Market Data MCP Servers  
- **Tags:** accounting, invoicing, small-business  
- **Source URL:** https://mcp.pipedream.com/app/moneybird  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Overview
Moneybird MCP Server is an integration that exposes Moneybird accounting software features through the MCP/Pipedream environment. It provides a static MCP server endpoint that can be added to compatible chat or MCP-enabled clients, where users then authenticate with their own credentials.

## Features
- **MCP-compatible server endpoint**  
  - Static server URL: `https://mcp.pipedream.net/v2`  
  - Same URL works for all clients; authentication is done per client when adding the server.

- **Integration with MCP / chat clients**  
  - Designed to be added to MCP-enabled chat applications.  
  - Configuration guidance available via a separate configuration page (not included in provided content).

- **Moneybird accounting integration**  
  - Connects to Moneybird accounting software via the MCP/Pipedream environment.  
  - Intended for accounting, invoicing, and small-business use cases.

- **Pipedream Connect platform**  
  - Integration is powered and hosted by Pipedream Connect.  
  - Operates under Pipedream’s platform terms and privacy policies.

## Setup
- Use the static MCP server URL: `https://mcp.pipedream.net/v2` when adding the server to your MCP-enabled app or chat client.
- Authentication occurs during the “add server” flow in your client (specific steps depend on the client; not detailed in the provided content).
