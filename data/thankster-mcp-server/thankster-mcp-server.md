# Thankster MCP Server

An MCP (Model Context Protocol) server integration for Thankster that allows agents and applications to automatically send handwritten-style cards for CRM and marketing automation workflows.

## Features
- **MCP-compatible server:** Exposes Thankster functionality through the Model Context Protocol so MCP-enabled agents and clients can trigger card mailings programmatically.
- **Automatic handwritten-style card mailing:** Enables automated sending of physical, handwritten-style cards for CRM, marketing, and related workflows.
- **Static MCP server endpoint:**
  - Server URL: `https://mcp.pipedream.net/v2`
  - Same endpoint used across compatible MCP clients.
- **Client-agnostic setup:** Designed to work with any MCP-capable chat client or application; you authenticate when you add the server to your chosen client.
- **Configuration documentation:** A dedicated configuration page (via Pipedream) guides you through adding and using the server in different clients.
- **Hosted by Pipedream:** Infrastructure, MCP serving, and integration are provided via Pipedream Connect.

## Setup
1. Use the MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add the server to your MCP-compatible chat client or application.
3. Authenticate within your client during setup.
4. Configure workflows to trigger Thankster handwritten-style card mailings from CRM or marketing automation flows.
