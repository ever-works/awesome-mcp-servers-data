# HigherGov MCP Server

## Overview
HigherGov MCP Server is a Model Context Protocol (MCP) server that connects MCP-compatible clients to HigherGov. It provides access to federal market intelligence, opportunity discovery, and business development data to help users find opportunities, build relationships, and pursue government contracts and grants.

## Key Use Cases
- Discover federal contract and grant opportunities
- Access market intelligence on the U.S. federal sector
- Support business development workflows focused on government customers
- Integrate HigherGov data into MCP-enabled chat and AI applications

## Features
- **MCP Server Endpoint**  
  - Static server URL for all clients: `https://mcp.pipedream.net/v2`  
  - Authentication handled when adding the server to the client application

- **HigherGov Integration**  
  - Connects to a HigherGov account via configuration in Pipedream  
  - Supports selection of a specific client/account context within HigherGov

- **MCP Client Compatibility**  
  - Designed to be added to MCP-compatible chat and AI clients  
  - Uses a single static URL usable across different MCP clients

- **Government Market Focus**  
  - Tools oriented around U.S. federal government contracts and grants  
  - Emphasis on opportunity discovery and relationship-building for business development

- **Configuration Guidance**  
  - Step-by-step flow: connect HigherGov account → select client → copy MCP server URL → add to MCP app  
  - Link to a broader configuration guide on Pipedream (general MCP configuration)
