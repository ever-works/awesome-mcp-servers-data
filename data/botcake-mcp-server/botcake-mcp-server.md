## Features
- **Static MCP Server URL**
  - Single, static endpoint for all MCP clients: `https://mcp.pipedream.net/v2`
  - Works across different MCP-compatible applications.

- **Client-Agnostic Integration**
  - Designed to be added to a variety of chat clients.
  - Configuration guidance available for different client types.

- **Authentication at Connection Time**
  - Authentication performed when adding the MCP server to your application (client config time), not baked into the URL.

- **Configuration Documentation**
  - Full configuration instructions available via a dedicated Configuration page (hosted by Pipedream).

- **Chatbot Marketing Use Case**
  - Intended to power chatbot marketing workflows when integrated via MCP.
