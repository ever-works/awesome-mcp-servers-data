## Canny MCP Server

**Category:** Project Management MCP Servers  
**Brand:** Canny  
**Source:** https://mcp.pipedream.com/app/canny

### Overview
The Canny MCP Server is a Model Context Protocol (MCP) server that connects Canny’s feature request tracking and customer feedback data to MCP-compatible AI tools via Pipedream. It enables agents and AI-powered environments to access and organize product feedback and roadmap insights.

### Features
- Connects a Canny account to AI tools through Pipedream’s MCP server
- Provides access to Canny feature request tracking data
- Exposes customer feedback data from Canny to MCP-compatible agents
- Allows AI tools to read roadmap insights stored in Canny
- Supports organizing and structuring roadmap and feedback insights programmatically
- Compatible with AI development environments and agents such as:
  - Cursor
  - Claude
  - Windsurf

### Integrations
- **Pipedream** (MCP server host)
- **Canny** (feature requests and customer feedback)
- **AI Tools / IDEs:** Cursor, Claude, Windsurf (via MCP)
