**Category:** Web Search MCP Servers

## Overview
NewsCatcher MCP Server is an MCP-compatible server for the NewsCatcher News API. It enables MCP tools and chat-based clients to search and retrieve online news articles from worldwide sources in multiple languages.

## Features
- MCP-compatible server implementation for the NewsCatcher News API
- Multi-language news search across global online news sources
- Retrieval of news articles published online
- Single static MCP server URL usable across compatible clients:
  - `https://mcp.pipedream.net/v2`
- Authentication performed when adding/configuring the server in your client or application
- Usable with multiple chat clients that support MCP (configured per client)

## Configuration
- Add the MCP server to your application using the static URL: `https://mcp.pipedream.net/v2`
- Configure authentication and client-specific settings within your MCP-compatible chat client
