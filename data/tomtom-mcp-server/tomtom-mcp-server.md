# TomTom MCP Server

Mapping and location technology MCP server that provides access to TomTom’s mapping, routing, and location capabilities via the Model Context Protocol (MCP).

- **Website / Source**: https://mcp.pipedream.com/app/tomtom
- **Category**: Data Access & Integration – MCP Servers
- **Brand**: TomTom
- **Slug**: `tomtom-mcp-server`
- **Tags**: `maps`, `navigation`, `geocoding`

## Features

- **MCP-compatible server**
  - Exposes TomTom mapping and location technology through the MCP protocol.
  - Designed to be added as an MCP server to compatible chat or AI clients.

- **Static server endpoint**
  - Single static MCP server URL for all clients:
    - `https://mcp.pipedream.net/v2`
  - Authentication is handled when adding/configuring the server in your application.

- **Integration with chat/AI clients**
  - Can be added to supported chat clients as an MCP server.
  - Configuration guidance available via the platform’s Configuration page (client-specific setup instructions).

- **Routing and location use cases**
  - Intended to help applications find routes and locations.
  - Suitable for map-based, navigation, and geocoding workflows (based on TomTom capabilities).

- **Hosted by Pipedream Connect**
  - Server is provided and hosted via Pipedream’s MCP infrastructure.
