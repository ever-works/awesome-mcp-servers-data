# Agenty MCP Server

Robotic process automation (RPA) MCP server for Agenty, exposing web scraping, data extraction, and automation capabilities via the Model Context Protocol.

- **Type:** MCP server / integration endpoint
- **Category:** Workflow automation MCP servers
- **Provider / Brand:** Agenty (hosted via Pipedream Connect)
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Website / Source:** https://mcp.pipedream.com/app/agenty

## Features

- **Model Context Protocol (MCP) server**
  - Exposes Agenty’s RPA, web scraping, and data extraction capabilities via a standard MCP endpoint.
  - Uses a single static MCP URL (`https://mcp.pipedream.net/v2`) that works across supported MCP clients.

- **Robotic Process Automation (RPA)**
  - Designed to orchestrate and automate web-based tasks using Agenty’s automation platform (details not fully listed on this page).

- **Web scraping & data extraction**
  - Integrates Agenty’s web scraping and structured data extraction features into MCP-compatible applications (high-level description only; no granular feature list provided on the page).

- **Client-agnostic configuration**
  - The same server URL is used for all clients; authentication occurs when adding the server within each client.
  - Documentation available via a separate Configuration page (linked from the listing).

- **Hosted by Pipedream Connect**
  - MCP server is provided through Pipedream’s infrastructure, with terms and privacy policy governed by Pipedream.

## How to Use

1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add this URL as a new MCP server in your compatible chat client or application.
3. Authenticate when prompted by the client.
4. Refer to the Pipedream MCP Configuration page for client-specific setup steps.
