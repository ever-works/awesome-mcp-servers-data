**Category:** Cloud & DevOps MCP Servers

## Description
The Supabase Management API MCP Server provides Model Context Protocol (MCP) access to the Supabase Management API, enabling programmatic management of Supabase organizations and projects from compatible chat or MCP-enabled clients.

## Features
- MCP server interface for the Supabase Management API.
- Programmatic management of Supabase organizations.
- Programmatic management of Supabase projects.
- Static MCP server URL usable across compatible clients: `https://mcp.pipedream.net/v2`.
- Authentication handled when adding the server to your application.

## Integration & Setup

- Add the server by configuring the static MCP URL in your client.
