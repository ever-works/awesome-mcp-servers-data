# shadcn-ui-mcp-server

## Overview
shadcn-ui-mcp-server is an open-source Model Context Protocol (MCP) server that provides AI assistants with structured context about shadcn/ui v4 components. It exposes component structure, usage patterns, installation details, and related metadata to enable UI-aware reasoning and code generation.

- **Type:** Open-source MCP server / development tool
- **Ecosystem:** shadcn/ui v4
- **Use cases:** UI-aware code generation, component-aware reasoning, automated UI scaffolding
- **License:** MIT

## Features
- **MCP server for shadcn/ui**
  - Implements the Model Context Protocol to serve shadcn/ui-specific information to compatible AI assistants and tooling.

- **Component structure awareness**
  - Provides detailed information about shadcn/ui v4 component structure so models can generate code that correctly composes and configures components.

- **Usage and installation details**
  - Exposes usage patterns and installation guidance for shadcn/ui components, assisting in generating ready-to-use UI code and setup instructions.

- **UI-aware reasoning and code generation**
  - Enables AI assistants to reason about UI layouts, component composition, and integration, improving the quality of generated React and related framework code.

- **Multi-framework compatibility**
  - Designed to be compatible with:
    - React
    - Svelte 5
    - Vue
    - React Native

- **shadcn/ui v4 support**
  - Targets the v4 version of shadcn/ui components, blocks, demos, and metadata.

- **Metadata, blocks, and demos access**
  - Grants access to shadcn/ui blocks, examples/demos, and component metadata to guide more accurate and idiomatic code generation.
