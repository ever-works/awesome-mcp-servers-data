# iScraper MCP Server

Harvest LinkedIn public data for profiles, companies, and employee lists via the iScraper API, exposed as an MCP (Model Context Protocol) server.

## Overview
- **Type:** MCP Server (data-access / integration)
- **Category:** data-access-integration-mcp-servers
- **Provider / Brand:** iScraper (via Pipedream Connect)
- **Slug:** `iscraper-mcp-server`
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- Access LinkedIn **public data** through the iScraper API
- Retrieve **individual LinkedIn public profiles**
- Retrieve **company profiles**
- Fetch **employees lists** for companies
- Designed to be **reliable** and **affordable** (per product description)
- Exposed as a **static MCP server endpoint** usable across compatible clients
- **Authentication handled on client setup** (you authenticate when adding the server to your application)
- Built and hosted via **Pipedream Connect** infrastructure

## Integration / Usage
- Use the static MCP endpoint: `https://mcp.pipedream.net/v2`
- Add the server within any supported MCP-compatible chat client or application
