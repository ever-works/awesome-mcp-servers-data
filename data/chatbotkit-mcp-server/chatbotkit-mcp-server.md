# ChatBotKit MCP Server

**Category:** AI Integration – MCP Servers  
**Website:** https://mcp.pipedream.com/app/chatbotkit  
**Provider / Brand:** ChatBotKit (via Pipedream)

## Description
ChatBotKit MCP Server is a Model Context Protocol (MCP) server that connects ChatBotKit with applications built on Pipedream. It enables you to build conversational AI chatbots that use your own data and deploy them across multiple channels, including websites, Slack, Discord, and WhatsApp.

## Features

- **MCP Server Endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Single URL works across compatible MCP clients.  
  - Authentication is handled when adding the server to your application.

- **Multichannel Chatbot Support**  
  - Use ChatBotKit to power chatbots on:  
    - Websites  
    - Slack  
    - Discord  
    - WhatsApp

- **Own-Data Conversational AI**  
  - Configure chatbots that communicate using your own data (e.g., site or app content, internal resources).  
  - Designed for natural language interactions with users.

- **Client Integration Guidance**  
  - Workflow to add the MCP server to your app via a “select your chat client” flow.  
  - Additional configuration details available on a dedicated configuration page (linked from the app).

- **Pipedream Connect Integration**  
  - MCP server is powered and hosted through Pipedream Connect.  
  - Can be used as part of broader Pipedream-based automation and integration workflows.

## Setup

- Use `https://mcp.pipedream.net/v2` as the MCP server URL in your MCP-compatible client or tooling.
- Authenticate when prompted by your client during server addition.
- Follow client-specific instructions (e.g., for Slack, Discord, website, WhatsApp) using the provider’s configuration guides.
