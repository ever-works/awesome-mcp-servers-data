# StackShare MCP Server

**Category:** Search & Discovery MCP Servers  
**Brand:** StackShare  
**Source URL:** https://mcp.pipedream.com/app/stackshare_api

## Overview
StackShare MCP Server is an MCP-compatible service that lets you query information about the technology stacks used by over 1.5 million companies directly from MCP-capable clients (e.g., AI chat or dev tools that support the Model Context Protocol).

## Features
- **Company tech stack lookup**: Access data about the tools and technologies used by more than 1.5M companies.
- **MCP-compatible endpoint**: Single static MCP server URL for all clients: `https://mcp.pipedream.net/v2`.
- **Client-agnostic configuration**: Works with any MCP-capable client; you authenticate when adding the server to your application.
- **Centralized configuration**: Can be configured via a dedicated configuration page in the Pipedream environment.
- **Tool discovery**: Use StackShareâ€™s dataset to discover new tools, services, and technologies used across companies.
- **API-based integration**: Designed to be integrated programmatically as a server within MCP workflows.

## Usage
- Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server to your MCP-capable application and authenticate during setup.
- Use available tools (actions) exposed by the server to query StackShare data about company tech stacks.
