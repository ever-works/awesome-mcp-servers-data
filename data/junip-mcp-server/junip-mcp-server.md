# Junip MCP Server

Junip MCP Server provides MCP-based access to Junip’s product review collection and display tools, allowing MCP clients to manage and use e‑commerce reviews programmatically.

- **Website / Source**: https://mcp.pipedream.com/app/junip
- **Category**: Business & Commerce – MCP Servers
- **Brand**: Junip
- **Slug**: `junip-mcp-server`
- **Tags**: reviews, e-commerce, customer-feedback

## Features

- **MCP-based access to Junip**
  - Connects to Junip’s product review collection and display tooling through the Model Context Protocol (MCP).
  - Designed for use from MCP-compatible chat or agent clients.

- **Programmatic review management**
  - Enables clients to manage e‑commerce product reviews programmatically (e.g., collecting, retrieving, and leveraging reviews in workflows or applications).

- **Static MCP server endpoint**
  - Single MCP server URL usable across clients:
    - `https://mcp.pipedream.net/v2`
  - Authentication is performed when adding the server within the client/application.

- **Client-agnostic setup**
  - Works with multiple MCP-compatible chat clients.
  - Setup guided via client-specific instructions and a general configuration page.

- **Pipedream integration**
  - Hosted and provided via Pipedream Connect infrastructure.
  - Uses Pipedream’s platform for connection and configuration.

## Configuration

- **Server URL**: `https://mcp.pipedream.net/v2`
- **Authentication**: Performed when adding the MCP server to the client/app.
- **Documentation**: Additional configuration details are available on the Pipedream MCP configuration page (client-specific setup instructions).
