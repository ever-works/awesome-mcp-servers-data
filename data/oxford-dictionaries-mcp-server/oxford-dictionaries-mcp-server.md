# Oxford Dictionaries MCP Server

**Category:** Documentation & Learning Resources  
**Brand:** Oxford University Press  
**Slug:** `oxford-dictionaries-mcp-server`

Oxford Dictionaries MCP Server provides access to authoritative dictionary and lexical resources from Oxford University Press via the MCP (Model Context Protocol), suitable for integration into chat clients and applications.

---

## Features

- **Oxford Dictionaries data**  
  - Access to world-renowned Oxford Dictionaries content and lexical resources.  
  - Authoritative language reference sourced from Oxford University Press.

- **MCP (Model Context Protocol) server**  
  - Exposes Oxford Dictionaries data as an MCP server for use with compatible clients.  
  - Designed to be added as a contextual tool to chat-based or other MCP-enabled applications.

- **Static server endpoint**  
  - Single static URL for all MCP clients:  
    - `https://mcp.pipedream.net/v2`  
  - Same endpoint across different applications simplifies configuration.

- **Authentication at client setup**  
  - Authentication is performed when adding the server to each client / application.  
  - Keeps the public endpoint static while securing access per user or app.

- **Multi-client integration**  
  - Can be added to various chat clients that support MCP.  
  - Configuration guidance is available per chat client and through a centralized configuration page.

- **Hosted by Pipedream Connect**  
  - MCP server infrastructure is provided and operated via Pipedream Connect.  
  - Centralized configuration and management via the Pipedream platform.

---

## Usage

- Use the static MCP server URL `https://mcp.pipedream.net/v2` when configuring an MCP-compatible client.
- Follow client-specific instructions in your chat/application environment, or refer to the associated configuration documentation page.
