# Alpaca MCP Server

Commission-free, API-first stock brokerage MCP server for integrating Alpaca trading and market data into MCP-compatible clients.

**Website:** https://mcp.pipedream.com/app/alpaca  
**MCP Server URL:** `https://mcp.pipedream.net/v2`  
**Category:** Finance / Market Data MCP Servers  
**Tags:** trading, stock-market, api-integration

## Features

- **MCP-compatible server endpoint**  
  - Single static URL (`https://mcp.pipedream.net/v2`) usable by any MCP client.  
  - Authentication handled when adding the server in the client/application.

- **Alpaca brokerage integration**

- **Client-agnostic setup**

  - Configuration instructions available via a centralized configuration page.

- **Pipedream Connect platform**  
  - Server is hosted and powered by Pipedream Connect infrastructure.  
  - Central terms, privacy, and cookie policy managed by Pipedream.
