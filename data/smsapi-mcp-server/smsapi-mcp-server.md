# SMSAPI MCP Server

**Category:** Messaging MCP Servers  
**Brand:** SMSAPI  
**Slug:** smsapi-mcp-server

## Overview
SMSAPI MCP Server is an MCP-compatible integration that lets AI agents interact with the SMSAPI messaging platform. It enables sending and managing SMS-based mobile marketing communications through a standardized MCP server endpoint.

## Features
- MCP-compatible server for integrating SMSAPI with AI agents and MCP-aware clients
- Supports sending SMS messages via the SMSAPI platform
- Designed for managing SMS-based mobile marketing communications
- Uses a single static MCP server URL for all compatible clients
- Authentication handled when adding the server to an MCP-enabled application

## MCP Server URL
- Static MCP server URL (for all clients):
  - `https://mcp.pipedream.net/v2`
