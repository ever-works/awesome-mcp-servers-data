# Spoke Phone MCP Server

MCP Server integration for Spoke Phone, enabling access to its customer conversation and telephony capabilities from MCP-compatible tools and chat clients.

---

## Overview
- **Name:** Spoke Phone MCP Server  
- **Category:** Messaging MCP Servers  
- **Tags:** telephony, voice, customer-conversations  
- **Provider/Brand:** Spoke Phone (via Pipedream Connect)  
- **Slug:** `spoke-phone-mcp-server`
- **Source URL:** https://mcp.pipedream.com/app/spoke_phone

---

## Features
- **MCP Server Integration**  
  - Exposes Spoke Phone’s customer conversation and telephony capabilities to MCP tools and compatible chat clients.
- **Static MCP Endpoint**  
  - Single static MCP server URL used across clients:  
    `https://mcp.pipedream.net/v2`  
  - Authentication is performed when adding the server to your application.
- **Multi‑Client Compatibility**  
  - Designed to be added to various MCP-enabled chat clients.  
  - Client-specific setup is available via the configuration instructions on the Pipedream MCP configuration page.
- **Pipedream Connect Integration**  
  - Runs on Pipedream Connect infrastructure, allowing use alongside other MCP servers offered through Pipedream.

---

## Configuration
- **Server URL:** `https://mcp.pipedream.net/v2`  
- **Setup:**  
  - Add the above URL as an MCP server in your chosen chat client.  
  - Authenticate during the add/connect flow as prompted by the client.  
  - More detailed configuration instructions are available on the Pipedream MCP **Configuration** page linked from the app listing.
