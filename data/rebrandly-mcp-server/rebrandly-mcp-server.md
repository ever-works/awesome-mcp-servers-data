# Rebrandly MCP Server

Custom URL Shortener MCP server that exposes Rebrandly’s URL shortening and link management capabilities to MCP-compatible AI tools and workflows.

- **Name:** Rebrandly MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Brand:** Rebrandly  
- **Source URL:** https://mcp.pipedream.com/app/rebrandly  
- **MCP Server Endpoint:** `https://mcp.pipedream.net/v2`

## Features

- **MCP-compatible server** for integrating Rebrandly with AI tools and chat-based clients that support the Model Context Protocol (MCP).
- **Custom URL shortening** capabilities exposed via MCP for use inside AI workflows.
- **Link management integration**: designed to surface Rebrandly’s link management features (e.g., managing shortened links) through MCP-compatible applications.
- **Static server URL** (`https://mcp.pipedream.net/v2`) used across all compatible clients, simplifying configuration.
- **Client-agnostic setup**: same MCP server URL works for every supported chat client; authentication is handled when adding the server within each app.
- **Configuration documentation** available via a dedicated configuration page linked from the app.

## Integration & Usage

- Add the MCP server to your MCP-compatible chat or AI client using the static URL: `https://mcp.pipedream.net/v2`.
- Authenticate within your chosen client when prompted during server setup.
- Use the client’s MCP integration flow or configuration guide to complete setup (via the referenced Configuration page).
