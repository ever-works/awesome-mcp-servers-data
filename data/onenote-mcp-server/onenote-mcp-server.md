# OneNote MCP Server

**Category:** Document Management MCP Servers  
**Brand:** Microsoft OneNote

A Model Context Protocol (MCP) server by Rajesh Vijay that connects to Microsoft OneNote using the Microsoft Graph API. It enables models/tools to read and create OneNote content—such as notebooks, sections, and pages—directly within their context.

---

## Features

- **Microsoft Graph integration**  
  - Connects to Microsoft OneNote via the Microsoft Graph API.

- **Notebook management**  
  - Read existing OneNote notebooks.  
  - Create new notebooks from within a model context.

- **Section management**  
  - Access and read sections within notebooks.  
  - Create new sections.

- **Page management**  
  - Read existing OneNote pages.  
  - Create new pages.

- **Model Context Protocol (MCP) server**  
  - Exposes OneNote operations as MCP tools/resources that can be used by MCP-compatible models and clients.
