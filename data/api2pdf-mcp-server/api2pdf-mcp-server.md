# Api2pdf MCP Server

**Category:** Document Management MCP Servers  
**Brand:** api2pdf  
**Source:** https://mcp.pipedream.com/app/api2pdf

## Overview
Api2pdf MCP Server is an MCP integration that connects Api2pdf with AI tools via Pipedream, enabling agents to generate PDFs from various sources through the Api2pdf API and streamline document creation workflows.

## Features
- Integrates Api2pdf with the MCP (Model Context Protocol) ecosystem via Pipedream’s MCP server.
- Allows AI agents and tools (e.g., Cursor, Claude, Windsurf) to programmatically generate PDFs.
- Supports PDF creation from various input sources (as exposed by the Api2pdf API), such as:
  - Web pages/URLs
  - HTML content
  - Other supported document formats (depending on Api2pdf API capabilities).
- Designed to automate and streamline document creation workflows within AI-driven environments.
- Centralized configuration via a Pipedream MCP server endpoint.

## Integrations
- **Pipedream MCP Server** – acts as the MCP host for the Api2pdf integration.
- **AI Tools** (via MCP):
  - Cursor
  - Claude
  - Windsurf
