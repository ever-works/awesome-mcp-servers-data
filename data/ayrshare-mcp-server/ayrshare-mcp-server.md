# Ayrshare MCP Server

**Description**  
An MCP Server integration for Ayrshare Social Media APIs, enabling MCP tools and agents to publish and manage social media posts via Ayrshare directly from within the MCP environment.

**Category**  
- Business & Commerce → MCP Servers

**Tags**  
- Social media  
- Marketing automation  
- Publishing

**Integration Details**  
- MCP server base URL (static for all clients): `https://mcp.pipedream.net/v2`  
- Hosted and provided via Pipedream  
- Authentication occurs when adding the server to your MCP-enabled application or chat client

## Features
- **Social media posting via APIs**: Uses Ayrshare’s Social Media APIs to send social media posts from within MCP-compatible tools.
- **Central MCP integration**: Exposes Ayrshare capabilities as MCP tools/actions accessible to agents and clients that support the MCP protocol.
- **Static MCP server URL**: Single, static endpoint (`https://mcp.pipedream.net/v2`) for all clients, simplifying configuration.
- **Account-based authentication**: Authentication handled when adding the server to your app or chat client, rather than per-URL.
- **Client-agnostic setup**: Can be added to different chat clients or MCP-enabled applications, with instructions available per client.
- **Configuration guidance**: Linked configuration documentation (full configuration page) for setup details within the MCP environment.

## Getting Started
1. Configure your Ayrshare connection within the Pipedream-hosted MCP server page.  
2. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.  
3. Add this server URL to your MCP-compatible app or chat client.  
4. Authenticate when prompted to enable tools for publishing and managing social media posts.
