# Benchmark Email MCP Server

**Category:** Business & Commerce · MCP Servers  
**Slug:** `benchmark-email-mcp-server`

An MCP server that connects MCP-based tools and workflows to Benchmark Email, enabling email marketing and campaign operations programmatically.

---

## Overview

Benchmark Email MCP Server exposes Benchmark Email’s email marketing and campaign features as MCP-compatible tools. It is delivered via a static MCP endpoint hosted by Pipedream Connect, allowing you to integrate Benchmark Email into compliant chat clients and automation workflows.

- **MCP endpoint:** `https://mcp.pipedream.net/v2`  
- **Authentication:** Performed when adding the server in your MCP client or app  
- **Integration host:** Pipedream Connect

- Static MCP server URL usable across multiple MCP-compatible clients.

### Email Marketing & Campaign Access
- Programmatic access to Benchmark Email email marketing capabilities from MCP-based tools and workflows.
- Designed to expose common marketing operations such as:
  - Managing email campaigns (e.g., create, update, schedule, send) via MCP tools.  
  - Interacting with subscriber lists and contact data (e.g., adding and updating subscribers).
