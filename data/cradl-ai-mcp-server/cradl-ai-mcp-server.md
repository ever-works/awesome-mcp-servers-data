# Cradl AI MCP Server

AI-powered MCP server for automating internal document processes using Cradl AI via Pipedream Connect.

- **Website / Source**: https://mcp.pipedream.com/app/cradl_ai
- **Category**: Content Extraction & Summarization MCP Servers
- **Tags**: document-processing, ai-integration
- **MCP Server URL**: `https://mcp.pipedream.net/v2`

## Overview
Cradl AI MCP Server exposes Cradl AI’s document understanding capabilities through the Model Context Protocol (MCP), enabling applications and chat clients to automate internal document workflows using AI-based document processing.

## Features
- **MCP-Compatible Server**
  - Provides a static MCP server endpoint usable from any compatible client.
  - Designed to be added as an external tool/server to chat-based or other MCP-enabled applications.

- **AI-Based Document Process Automation**
  - Integrates with Cradl AI to support AI-driven internal document workflows.
  - Focused on automating processing and understanding of documents (e.g., extraction, interpretation) within internal business processes.

- **Static, Reusable Endpoint**
  - Single static URL (`https://mcp.pipedream.net/v2`) works across different MCP clients.
  - Authentication handled when you add/configure the server in your application.

- **Pipedream Integration Layer**
  - Delivered via Pipedream Connect as the integration and hosting layer.
  - Central configuration entry point through the Pipedream "Configuration" page (for setup and client instructions).

- **Client-Agnostic Setup**
  - Documentation and flows for adding the server to different chat clients.
  - Intended to be plugged into various environments without changing the server URL.

## Configuration & Usage
- **Server URL**: Use `https://mcp.pipedream.net/v2` as the MCP server endpoint in your client or application.
- **Authentication**: Performed when adding/configuring the server within your chosen client or app (details handled in that environment’s setup flow).
- **Further Setup Docs**: Full configuration guidance is available on the Pipedream Configuration page linked from the app.
