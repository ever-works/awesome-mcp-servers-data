# Mode MCP Server

**Category:** Data Analysis & Exploration MCP Servers  
**Brand:** Mode  
**Slug:** `mode-mcp-server`

Mode MCP Server is an MCP-compatible server for Mode’s analytics platform, enabling AI-driven tools to query and work with your Mode data and analyses via Mode APIs.

---

## Description
Mode MCP Server connects Mode’s analytics environment (SQL, Python, R, and visualizations) to MCP-compatible AI clients. Through Pipedream’s MCP server, you can link your Mode account to AI tools such as Cursor, Claude, and Windsurf, allowing them to access and work with data, notebooks, and reports exposed via Mode APIs.

---

## Features
- **MCP-compatible server for Mode**  
  - Implements an MCP server interface tailored to Mode’s analytics platform.  
  - Allows MCP clients to connect to Mode via standardized MCP protocols.

- **Integration with AI tools**  
  - Connects Mode to AI tools like **Cursor**, **Claude**, and **Windsurf** through Pipedream’s MCP server.  
  - Lets AI assistants interact with Mode resources programmatically.

- **Mode API access**  
  - Uses **Mode APIs** as the underlying data and operations layer.  
  - Enables AI tools to query Mode-hosted data and analytics content.

- **Multi-language analytics support (via Mode)**  
  - Works with Mode’s support for **SQL**, **Python**, and **R** within the analytics environment.  
  - Allows AI tools to interact with queries, notebooks, and analyses involving these languages (as exposed via Mode APIs).

- **Visualization-aware access**  
  - Supports interaction with Mode **visualizations** through the API, enabling AI tools to reference or work with charts and dashboards (where exposed by Mode APIs).

- **Data querying and exploration**  
  - Lets MCP clients issue requests that result in **data queries** against Mode-connected data sources.  
  - Supports analytics and reporting workflows from within AI coding and productivity tools.

- **Analytics and reporting workflows**  
  - Designed for **analytics**, **SQL**, and **reporting** use cases.  
  - Facilitates AI-assisted report generation, query refinement, and exploration of Mode analyses.
