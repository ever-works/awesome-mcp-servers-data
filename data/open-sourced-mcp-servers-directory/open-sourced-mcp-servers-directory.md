### Open-Sourced MCP Servers Directory

**Website:** https://github.com/chatmcp/mcp-directory  
**Category:** MCP server directories & lists

#### Description
Open-Sourced MCP Servers Directory is a curated GitHub-based list of freely available, open-source MCP (Model Context Protocol) server implementations, maintained by the **chatmcp** project. It focuses on making community MCP servers easier to discover and explore.

#### Features
- Curated directory of MCP servers sourced from GitHub
- Focus exclusively on open-source MCP server implementations
- Emphasis on discoverability of freely available MCP servers
