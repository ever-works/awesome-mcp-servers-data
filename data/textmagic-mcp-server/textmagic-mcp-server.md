# TextMagic MCP Server

**Category:** Messaging MCP Servers  
**Brand:** TextMagic  
**Source:** https://mcp.pipedream.com/app/textmagic

## Overview
TextMagic MCP Server is an MCP-compatible integration that connects MCP-based tools and chat clients with TextMagicâ€™s international business SMS and customer messaging APIs. It allows applications using the Model Context Protocol (MCP) to send and manage SMS messages through TextMagic.

## Features
- MCP server endpoint available at a static URL: `https://mcp.pipedream.net/v2`
- Works with any MCP-compatible client that supports adding custom MCP servers
- Integrates with TextMagic APIs to support international business SMS and customer messaging
- Supports authentication at the time you add the server to your MCP-enabled application
- Usable across multiple chat clients; configuration guidance is provided per client via the configuration page

## Usage
- Add the MCP server URL (`https://mcp.pipedream.net/v2`) to your MCP-capable chat client or application.
- Authenticate with TextMagic when prompted by the client.
- Use MCP-based tools to send and manage SMS and customer messaging via TextMagic.
