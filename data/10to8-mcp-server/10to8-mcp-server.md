# 10to8 MCP Server

## Overview
The 10to8 MCP Server is a Model Context Protocol (MCP) server integration for the 10to8 appointment scheduling platform. It enables agents and tools to manage 10to8 bookings as part of automated workflows, aiming to reduce appointment administration time and support higher booking capacity.

- **Category:** Business & Commerce – MCP Servers  
- **Brand:** 10to8  
- **Integration Host:** Pipedream Connect  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server:** Exposes 10to8 appointment scheduling capabilities through the Model Context Protocol, allowing use in MCP-enabled chat clients and agent frameworks.
- **Central server endpoint:** Uses a single static URL (`https://mcp.pipedream.net/v2`) for all clients; authentication is handled when adding the server to your application.
- **Appointment scheduling integration:** Connects to 10to8 to support creating and managing bookings within automated workflows and tools.
- **Workflow automation support:** Designed to reduce manual appointment administration time (from minutes to seconds) by allowing agents / tools to handle booking tasks programmatically.
- **Multi-client compatibility:** Can be added to various MCP-aware chat clients; configuration details are available via the general configuration documentation.

## Installation & Configuration
- **Server URL to add:** `https://mcp.pipedream.net/v2`  
- **Setup:** Add the MCP server URL in your MCP-compatible client, then authenticate with your 10to8 / Pipedream-connected account as prompted.  
- **Documentation:** Additional setup and configuration options are available on the Pipedream MCP Configuration page (referenced as the “full Configuration page”).
