# Moaform MCP Server

**Category:** Business & Commerce · MCP Servers  
**Tags:** surveys, forms, data-collection

## Overview
Moaform MCP Server is an MCP server that connects to Moaform’s online survey platform. It enables MCP clients to create surveys and collect an unlimited number of responses directly through the MCP integration.

- **Provider / Brand:** Moaform (via Pipedream Connect)
- **MCP Endpoint URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP Integration**
  - Static MCP server URL (`https://mcp.pipedream.net/v2`) that works across compatible MCP clients.
  - Authentication handled when adding the server to your MCP-enabled application.

- **Survey Creation**
  - Connects to Moaform’s online survey platform for building surveys and forms.

- **Response Collection**
  - Supports unlimited survey responses via the Moaform platform.

- **Client-Agnostic Setup**
  - Designed to work with multiple chat / MCP clients.
  - Configuration guidance available via the generic configuration page (client-specific setup instructions referenced but not detailed in the content).

## Technical Details
- **Server Type:** MCP (Model Context Protocol) server
- **Base URL:** `https://mcp.pipedream.net/v2`
- **Hosting / Infra:** Powered by Pipedream Connect
