# Gistly MCP Server

**Category:** Content Extraction & Summarization MCP Servers  
**Brand:** Gistly  
**Slug:** gistly-mcp-server

## Description
Gistly MCP Server is an MCP (Model Context Protocol) integration that enables tools and chat clients to access YouTube AI summaries and related functionality powered by ChatGPT. It is delivered via Pipedream Connect as a static MCP endpoint that can be added to compatible clients.

## Features
- **YouTube AI summaries**: Generate AI-powered summaries of YouTube content using ChatGPT.
- **MCP server integration**: Exposes Gistly capabilities as an MCP server for use in MCP-compatible tools and chat clients.
- **Static MCP endpoint**: Uses a single static URL for all clients:
  - `https://mcp.pipedream.net/v2`
- **Client-agnostic configuration**: The same server URL works across different chat clients; authentication occurs when adding the server to each application.
- **Configuration documentation**: Full configuration instructions available via a dedicated configuration page (referenced as “Configuration page” on the site).

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Integration platform:** Powered by Pipedream Connect.
