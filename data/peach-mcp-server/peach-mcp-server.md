# Peach MCP Server

## Overview
Peach MCP Server is an MCP (Model Context Protocol) server integration that connects Peach’s interactive WhatsApp marketing capabilities to compatible chat clients and automated workflows. It enables programmatic access to WhatsApp-based marketing and messaging through a standard MCP endpoint.

- **Category:** Messaging MCP Servers
- **Use cases:** WhatsApp marketing campaigns, interactive customer messaging, automated messaging workflows via MCP-compatible clients
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server endpoint**  
  - Provides a static MCP server URL (`https://mcp.pipedream.net/v2`) that works across clients.
  - Designed to be added to MCP-capable chat or agent applications.

- **WhatsApp marketing integration**  
  - Connects to Peach’s WhatsApp marketing capabilities for interactive, conversation-driven campaigns.
  - Supports use in automated messaging workflows.

- **Client-agnostic setup**  
  - Same server URL can be used by multiple MCP clients; authentication is handled per client during setup.

- **Configuration via Pipedream Connect**  
  - Managed and delivered through Pipedream Connect’s infrastructure.
  - Additional configuration details available via Pipedream’s configuration documentation (not specific to a single client).
