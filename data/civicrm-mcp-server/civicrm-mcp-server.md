# CiviCRM MCP Server

## Overview
CiviCRM MCP Server exposes CiviCRM’s nonprofit-focused CRM capabilities—such as contact, contribution, and membership management—to MCP-compatible AI agents and tools. It is designed for non-profit and civic-sector organizations that use or want to integrate with the CiviCRM platform.

- **Type:** MCP server / integration
- **Category:** Business & Commerce – MCP Servers
- **Intended users:** Nonprofits and civic-sector organizations using CiviCRM
- **Base platform:** CiviCRM (open-source, web-based CRM)

## Features
- **CiviCRM integration via MCP**
  - Connects AI agents and MCP-compatible tools to a CiviCRM instance.
  - Exposes nonprofit-focused CRM data and operations.

- **Contact management**
  - Access and manage constituent (contact) records stored in CiviCRM.

- **Contribution management**
  - Work with contribution/donation data tracked in CiviCRM.

- **Membership management**
  - Interact with membership records and related data from CiviCRM.

- **Nonprofit and civic-sector focus**
  - Oriented around use cases common to non-profit and other civic-sector organizations.

- **Standard MCP endpoint**
  - Uses a static MCP server URL that works across clients: `https://mcp.pipedream.net/v2`.
  - Authentication is handled when adding the server to a compatible application.

- **Client-agnostic setup**
  - Same server URL for all supported MCP clients.
  - Can be added to different chat or agent applications that support the MCP protocol.

## Technical Details
- **MCP server URL:** `https://mcp.pipedream.net/v2`
- **Protocol:** MCP (Model Context Protocol)
- **Source / configuration:**
  - App page: https://mcp.pipedream.com/app/civicrm
  - Configuration guidance available via Pipedream’s configuration resources.
