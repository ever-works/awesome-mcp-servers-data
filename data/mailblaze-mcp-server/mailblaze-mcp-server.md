# MailBlaze MCP Server

## Overview
MailBlaze MCP Server is an MCP-compatible integration that exposes MailBlaze’s email marketing capabilities to AI tools and MCP-enabled chat clients. It enables programmatic management of email lists, campaigns, and automations through a standardized MCP server endpoint.

- **Category:** Business & Commerce – MCP Servers  
- **Provider / Brand:** MailBlaze (via Pipedream Connect)  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server**  
  - Provides a static MCP server URL usable across clients.  
  - Authenticates per client when the server is added to an MCP-compatible app.

- **Email marketing functionality exposure**  
  - Designed to surface MailBlaze’s core email marketing features to AI tools.  
  - Intended for managing:  
    - Email lists and subscribers.  
    - Campaign-related workflows.  
    - Marketing automations.

- **Integration with AI / chat clients**  
  - Can be added to supported MCP chat clients.  
  - Central configuration via a single endpoint, with client-specific setup handled at the application level.

- **Pipedream Connect integration**  
  - Hosted and powered by Pipedream Connect infrastructure.  
  - Configuration details and setup guidance available via Pipedream’s configuration documentation.

## Setup
- Use the static MCP server URL:  
  - `https://mcp.pipedream.net/v2`  
- Add this server in your MCP-compatible chat client and complete the authentication flow as required by that client.
