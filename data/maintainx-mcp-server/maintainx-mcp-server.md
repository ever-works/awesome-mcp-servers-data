## Overview

## MCP Server URL
- Static MCP server URL (same for all clients):
  - `https://mcp.pipedream.net/v2`
- Authentication is performed when adding the server to your MCP-compatible application or chat client.

## Features
- Connects MCP clients (e.g., chat-based tools) to the MaintainX platform via Pipedream.
- Supports access to MaintainX work order and procedure digitization capabilities.
- Enables mobile-first workflows for frontline and factory teams.
- Uses a single, static server URL for all clients.
- Configuration guided via a central configuration page on Pipedream.

## Integration & Configuration
- Add the server to supported chat or MCP clients using the static URL.
- Client-specific setup instructions are provided via the Pipedream configuration resources.
