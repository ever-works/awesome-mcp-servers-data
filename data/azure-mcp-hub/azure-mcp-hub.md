# Azure MCP Hub

Comprehensive curated directory of MCP servers and resources specifically for Azure developers by Arun Sekhar. Features categorized listings of Azure-compatible MCP servers, installation guides, deployment templates, and integration examples with Azure services. Use cases include accelerating MCP adoption for cloud-native AI agents on Azure Functions, App Service, Kubernetes, and Cosmos DB, enabling developers to discover, deploy, and manage MCP tools in Microsoft cloud environments.
