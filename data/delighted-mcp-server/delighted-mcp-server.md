# Delighted MCP Server

## Overview
Delighted MCP Server is an MCP (Model Context Protocol) server integration for Delighted, provided via Pipedream. It allows MCP‑aware tools and chat clients to access Delighted’s Net Promoter System® (NPS) and customer feedback data through a unified MCP endpoint.

- **Category:** Business & Commerce – MCP Servers
- **Provider / Brand:** Delighted (via Pipedream)
- **Slug:** delighted-mcp-server
- **Source URL:** https://mcp.pipedream.com/app/delighted

## Features
- **MCP-compatible endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Single URL works for all clients; authentication occurs when adding the server to the client/app.

- **Delighted integration**  
  - Integrates with Delighted’s NPS® and customer feedback platform.  
  - Intended to expose Delighted data (NPS surveys, responses, feedback metrics) to MCP‑aware tools.

- **Client-agnostic setup**  
  - Can be added to multiple chat clients or MCP‑aware applications.  
  - Guided setup per client (via “Select your chat client to learn how to get started”).

- **Central configuration via Pipedream**  
  - Account connection and configuration managed through the Pipedream app page.  
  - Uses a common configuration flow referenced from the Pipedream Configuration page.

## Setup / Configuration
1. Connect your Delighted account in the Pipedream Delighted MCP app.  
2. Copy the static MCP server URL: `https://mcp.pipedream.net/v2`.  
3. Add the server URL to your MCP‑aware client or application.  
4. Authenticate within your client when prompted.  
5. (Optional) Refer to the full Pipedream Configuration page for general MCP configuration steps.
