# Paved MCP Server

**Category:** Business & Commerce MCP Servers  
**Slug:** `paved-mcp-server`  
**Brand:** Paved

## Overview
Paved MCP Server is an MCP (Model Context Protocol) server that connects Paved’s newsletter sponsorship platform to MCP-compatible clients. It enables automated management of newsletter ads and sponsorships directly from supported chat or AI applications.

### MCP Server URL
- Base MCP server URL (static for all clients):
  ```
  https://mcp.pipedream.net/v2
  ```
- Authentication occurs when adding the server to your chosen MCP-compatible application.

## Features
- **MCP integration**: Exposes Paved’s newsletter sponsorship platform via the Model Context Protocol for use in compatible clients.
- **Newsletter sponsorship management**: Designed to support automation around newsletter ad and sponsorship workflows (e.g., managing sponsorships and ads programmatically from clients).
- **Static server endpoint**: Single, static URL (`https://mcp.pipedream.net/v2`) usable across all supported MCP clients.
- **Client-agnostic setup**: Works with any MCP-compatible chat or AI client; you select your client and configure the same server URL.
- **Configurable via documentation**: Additional configuration details are available through a dedicated configuration page (referenced as “Configuration page” on the source site).
- **Hosted by Pipedream Connect**: Runs on Pipedream’s infrastructure, simplifying deployment and connectivity.

## Setup / Usage
- Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server to your MCP-compatible app or chat client.
- Authenticate during the add/connect flow as prompted by your client.
- Optional: Refer to the platform’s configuration documentation for client-specific setup steps.
