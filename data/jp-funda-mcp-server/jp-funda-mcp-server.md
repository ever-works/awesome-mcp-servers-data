# JP Funda MCP Server

**Category:** Finance / Market Data MCP Servers  
**Brand:** JP Funda  
**Source:** https://mcp.pipedream.com/app/jp_funda

## Description
JP Funda MCP Server provides access to securities report data for Japanese companies via an MCP-compatible interface. It is designed to be added as a Model Context Protocol (MCP) server to compatible chat or agent clients, enabling programmatic retrieval of Japan-focused financial disclosure data.

## Features
- **Japanese securities report data access**: Acquire securities report data for Japanese companies.
- **MCP-compatible server**: Exposes functionality through the Model Context Protocol for integration with MCP-capable clients.
- **Static server endpoint**: Uses a shared static MCP server URL: `https://mcp.pipedream.net/v2`.
- **Client-agnostic URL**: The same endpoint works across different MCP clients; configuration is handled on the client side.
- **Authenticated access**: Authentication occurs when adding the server within your application or client configuration.
