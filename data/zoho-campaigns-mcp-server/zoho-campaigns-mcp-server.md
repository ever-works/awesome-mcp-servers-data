# Zoho Campaigns MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Zoho  
**Source:** <https://mcp.pipedream.com/app/zoho_campaigns>

## Overview
Zoho Campaigns MCP Server is an MCP (Model Context Protocol) integration for Zoho Campaigns. It lets MCP-compatible tools and chat clients interact with Zoho Campaigns to manage email marketing operations, such as handling email contact databases and sending marketing campaigns.

## Features
- **MCP integration endpoint**  
  - Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`  
  - Can be added to compatible chat clients or MCP-enabled applications.

- **Email marketing management** (via Zoho Campaigns)
  - Use Zoho Campaigns as the underlying email marketing platform.  
  - Manage an email database of leads and contacts.  
  - Send marketing emails to managed contact lists.

- **Client-agnostic configuration**
  - Same MCP server URL works across different chat clients.  
  - Authentication occurs when adding the server to an application.

## Usage
- Add `https://mcp.pipedream.net/v2` as an MCP server in a supported client.  
- Authenticate the connection to enable access to Zoho Campaigns resources (contacts, campaigns, and sends) from within your MCP-enabled tool or chat interface.
