# SamCart MCP Server

**Category:** Business & Commerce · MCP Servers  
**Vendor / Brand:** Pipedream  
**Website:** https://mcp.pipedream.com/app/samcart

SamCart MCP Server is a Pipedream-hosted Model Context Protocol (MCP) integration that lets MCP‑compatible applications interact with SamCart’s direct‑to‑consumer ecommerce platform.

---

## Features

- **MCP-compatible integration**  
  - Exposes SamCart functionality as an MCP server endpoint for any MCP‑capable client.

- **Static MCP server URL**  
  - Single server URL for all clients: `https://mcp.pipedream.net/v2`  
  - Authentication handled when adding the server to an MCP‑compatible application.

- **Pipedream-hosted infrastructure**  
  - MCP server is hosted and managed by Pipedream.

- **Multi-client support**  
  - Works with multiple chat / MCP clients (you select your client when configuring).  
  - Configuration guidance available per client via the app UI.

- **Configuration workflow**  
  - Connect a SamCart account via Pipedream.  
  - Select the desired client to complete MCP server setup.  
  - Uses a unified configuration page for more detailed setup.

- **Tooling exposure (actions)**  
  - Designed to expose “available tools” (actions) from SamCart into MCP clients.  
  - Tools are loaded dynamically (e.g., “Loading actions…”, “Loading available tools…”), indicating support for a set of SamCart-related operations once configured.  
  - Exact actions are not listed in the provided content.

- **Direct-to-consumer ecommerce focus**  
  - Integrates specifically with SamCart, a direct‑to‑consumer ecommerce and checkout platform.

---

## Technical Details

- **Protocol:** Model Context Protocol (MCP) server.  
- **Endpoint:** `https://mcp.pipedream.net/v2` (shared endpoint; auth per client/app).  
- **Hosting:** Managed by Pipedream.
