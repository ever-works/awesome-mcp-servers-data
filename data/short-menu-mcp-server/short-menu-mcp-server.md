# Short Menu MCP Server

## Overview
Short Menu MCP Server is an MCP-compatible integration for the Short Menu URL shortener service. It allows MCP clients to create and manage short links and branded URLs, and access related analytics.

- **Category:** Business & Commerce MCP Servers  
- **Service Type:** MCP server for URL shortening and link management  
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server** for use with any MCP-capable chat or agent client.  
- **URL shortening** using the Short Menu service.  
- **Branded links support** for creating and managing custom-domain short URLs.  
- **Link management** from MCP clients (e.g., creating and managing short links and branded URLs).  
- **Analytics access** for shortened and branded links (e.g., tracking link performance via Short Menu’s analytics).  
- **Static server URL** (`https://mcp.pipedream.net/v2`) that works across compatible clients, with authentication handled when adding the server.  
- **Configurable in multiple chat clients** via each client’s MCP / tools configuration.

## Integration & Configuration
- Add the MCP server to your MCP-compatible application using the static URL:  
  - `https://mcp.pipedream.net/v2`  
- Authentication is performed when you register the server in your client.  
- Additional configuration details are available on the provider’s configuration page (not included in the provided content).
