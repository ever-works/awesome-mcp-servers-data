# GetEmails MCP Server

## Overview
GetEmails MCP Server exposes GetEmail.io’s email discovery capabilities to MCP-compatible AI tools and clients. It allows applications to query GetEmail.io through a standardized MCP server endpoint as part of automated or AI-driven workflows.

- **Name:** GetEmails MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Tags:** lead-generation, email, prospecting  
- **Provider / Brand:** GetEmail.io (via Pipedream Connect)  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server endpoint**  
  - Single static URL (`https://mcp.pipedream.net/v2`) usable across MCP clients.  
  - Authentication is handled when adding the server to your MCP-enabled application.

- **Integration with GetEmail.io**  
  - Connects MCP clients to GetEmail.io’s email discovery service.  
  - Designed for lead generation, email prospecting, and related workflows.

- **Client-agnostic setup**  
  - Same server URL for all supported MCP chat clients.  
  - Configuration instructions are provided per client via the associated configuration page.

- **Workflow integration via Pipedream Connect**  
  - Operates as part of Pipedream Connect’s ecosystem, enabling use within broader automation and AI workflows.

## Usage
- Add the MCP server to your MCP-enabled chat client or application using:  
  `https://mcp.pipedream.net/v2`
- Authenticate during the server setup process in your chosen client.
- Consult the linked configuration page in the source site for client-specific setup steps.
