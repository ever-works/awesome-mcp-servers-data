# Planly MCP Server

All-in-one social media management MCP server that connects Planly’s social media scheduling capabilities to Model Context Protocol (MCP)-compatible chat clients.

- **Name:** Planly MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Tags:** social-media, scheduling, marketing  
- **Source URL:** https://mcp.pipedream.com/app/planly  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Description
The Planly MCP Server exposes Planly’s all-in-one social media management and scheduling functionality through the Model Context Protocol. It is designed to be added as a server in MCP-enabled chat or agent applications so agents can schedule and manage social media content programmatically.

## Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Works across any MCP client that supports adding external servers.

- **Planly integration for social media management**  
  - Integrates Planly’s all-in-one social media management capabilities with MCP.  
  - Enables scheduling and managing social media content via agents and chat-based tools.  
  - Authentication occurs when adding the server in the client (handled at configuration time).

- **Cross-client usage**  
  - Single static URL usable by multiple chat clients.  
  - Can be configured following each client’s MCP server setup instructions.  
  - Central configuration reference available via the linked configuration page.

- **Pipedream Connect platform**  
  - Hosted and provided through Pipedream Connect infrastructure.  
  - Governed by Pipedream’s terms and privacy policies.

## Setup & Configuration
- Use the static MCP server URL `https://mcp.pipedream.net/v2` when adding a new MCP server in your chat or agent application.
- Complete authentication during the server-adding flow in your chosen client.
- For detailed, client-specific setup instructions, refer to the provider’s configuration documentation.
