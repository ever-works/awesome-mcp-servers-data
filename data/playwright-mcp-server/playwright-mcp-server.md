# Playwright MCP Server

## Overview
Playwright MCP Server is an open-source Model Context Protocol (MCP) server that uses Microsoft Playwright to automate web browsers and APIs. It is designed to integrate with MCP-compatible clients such as Claude Desktop, Cline, Cursor IDE, and others, enabling automated web interactions and data extraction directly from those environments.

## Features

- **Playwright-based automation**  
  - Uses Playwright to control modern browsers (e.g., Chromium, Firefox, WebKit).  
  - Supports scripted navigation, interactions, and assertions.

- **Model Context Protocol (MCP) server**  
  - Exposes automation capabilities through the MCP standard.  
  - Works as a tool/server that can be called from MCP-aware applications.

- **Web browser automation**  
  - Automate page loading, clicking, typing, and form submission.  
  - Capture page content for analysis or data extraction.  
  - Suitable for end-to-end and functional test-style flows.

- **API automation**  
  - Automate HTTP API calls via MCP tooling, alongside browser steps.  
  - Combine browser and API interactions in a single workflow.

- **Integration with popular MCP clients**  
  - Designed to be used from:  
    - Claude Desktop  
    - Cline  
    - Cursor IDE  
    - Other MCP-compatible tools and IDEs.

- **Examples and documentation**  
  - `examples` directory in the repository for practical usage patterns.  
  - `docs` directory and project site: <https://executeautomation.github.io/mcp-playwright/>

- **Containerization & scripts**  
  - Docker-related configuration (`.dockerignore`, `image` folder) for containerized deployment.  
  - `scripts` folder for setup, build, or automation scripts (e.g., running the server, tooling).

- **Open-source development**  
  - Public GitHub repository with issue tracking, pull requests, and version tags.  
  - MIT license allows commercial and private use, modification, and redistribution.
