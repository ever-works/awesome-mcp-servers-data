# Builder.io MCP Server

**Category:** Content Management MCP Servers  
**Slug:** `builderio-mcp-server`  
**Source:** https://mcp.pipedream.com/app/builder_io

## Description
The Builder.io MCP Server is an integration that connects Builder.io’s visual page and section building capabilities to Model Context Protocol (MCP) tools. It allows AI-powered environments to manage content and layouts built with Builder.io on modern tech stacks.

## Features
- Connects a Builder.io account to MCP-compatible AI tools.  
- Supports integration with tools such as Cursor, Claude, and Windsurf via Pipedream’s MCP server.  
- Exposes Builder.io’s visual page-building capabilities to AI agents.  
- Exposes Builder.io’s visual section-building capabilities to AI agents.  
- Enables AI tools to manage content created in Builder.io.  
- Enables AI tools to manage layout structures built in Builder.io.  
- Designed to work across modern tech stacks that integrate Builder.io.
