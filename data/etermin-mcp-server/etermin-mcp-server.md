# eTermin MCP Server

Online appointment scheduling software integrated via an MCP-compatible server.

## Overview
The eTermin MCP Server connects MCP-compatible tools and chat clients with eTermin’s online appointment scheduling and calendar management capabilities using a single static MCP endpoint.

## Features
- MCP-compatible server for eTermin
- Online appointment scheduling integration
- Calendar-related operations (via eTermin’s scheduling system)
- Single static MCP endpoint usable across different MCP clients
- Authentication handled when adding the server to each client/app

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Integration Platform:** Powered by Pipedream Connect
- **Usage:** Add the MCP server URL to your MCP-compatible app or chat client, then authenticate within that client.
