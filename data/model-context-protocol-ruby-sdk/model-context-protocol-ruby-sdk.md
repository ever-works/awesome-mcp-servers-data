## Model Context Protocol Ruby SDK

- **Name:** Model Context Protocol Ruby SDK
- **Brand / Ecosystem:** model-context-protocol
- **Category:** mcp-server-directories-lists
- **Source:** <https://github.com/modelcontextprotocol/ruby-sdk>
- **Package:** <https://rubygems.org/gems/mcp>
- **License:** MIT

### Description
Official Ruby SDK for the Model Context Protocol, providing Ruby interfaces and utilities to build MCP-compatible servers and tools. The project is maintained in collaboration with Shopify.

### Features
- Official Ruby SDK implementation for the Model Context Protocol
- Ruby interfaces for working with MCP concepts
- Utilities to build MCP-compatible servers
- Utilities to build MCP-compatible tools
- Distributed as the `mcp` gem via RubyGems
- Open-source repository with examples, documentation, and tests (via `examples/`, `docs/`, and `test/` directories)
