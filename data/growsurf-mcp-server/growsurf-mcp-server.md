# Growsurf MCP Server

## Overview
The Growsurf MCP Server is an integration endpoint that connects Growsurf referral software with MCP-compatible applications. It is designed for referral-driven growth use cases across B2C, FinTech, and SaaS businesses.

- **Category:** Business & Commerce – MCP Servers  
- **Use Cases:** Managing and tracking referral programs, integrating referral data into chat or other MCP-enabled clients

## Features
- **MCP-compatible server endpoint**  
  - Provides a static MCP server URL for connecting from any supported MCP client:  
    `https://mcp.pipedream.net/v2`
- **Centralized configuration**  
  - "Configure Growsurf" flow to connect your Growsurf account.  
  - Account connection and client selection within the Pipedream interface.
- **Client-agnostic integration**  
  - Same MCP server URL works for every client; authentication occurs when adding the server to your specific application or chat client.
- **Guided setup for chat clients**  
  - Instructions available for adding the server to different chat clients.  
  - Option to reference a full configuration page for more detailed setup steps.
- **Referral program support**  
  - Built to integrate with Growsurf referral software used by B2C, FinTech, and SaaS companies to manage and track referral-driven growth.

## Configuration & Access
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Authentication:** Performed when adding the server to your application or chat client.  
- **Documentation:** Additional setup details are available on the referenced Configuration page within Pipedream.
