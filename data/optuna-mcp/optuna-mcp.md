## optuna-mcp

**Category:** AI Integration – MCP Servers

### Overview
optuna-mcp is the official Optuna Model Context Protocol (MCP) server. It allows AI assistants and MCP-compatible clients to interact with Optuna, orchestrating hyperparameter optimization and other search tasks programmatically.

  - Exposes Optuna’s optimization capabilities through the MCP interface.
