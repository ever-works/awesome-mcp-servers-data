# Salesforce MCP Connector

Third-party MCP server exposing Salesforce data and APIs, supporting create, update, and delete operations on CRM records using user access rights, including custom objects. Compatible with Salesforce editions providing API access like Enterprise, Unlimited, and Developer.

[https://developer.salesforce.com/docs/atlas.en-us.api_mcp.meta/api_mcp/intro_mcp.htm](https://developer.salesforce.com/docs/atlas.en-us.api_mcp.meta/api_mcp/intro_mcp.htm)
