# SimpleBackups MCP Server

All-in-one automation for backing up websites, databases, storage, and applications from within MCP-powered workflows.

## Overview
- **Type:** MCP server (for MCP-powered tools / clients)
- **Provider:** SimpleBackups (via Pipedream Connect)
- **Category:** Cloud / DevOps MCP servers
- **Primary use:** Centralized, automated backups for multiple resource types (websites, databases, storage, apps) through a single MCP endpoint.

## Features
- **Unified backup automation** for:
  - Websites
  - Databases
  - Storage (cloud or other supported storage targets)
  - Applications
- **MCP workflow integration**
  - Designed to be used directly from MCP-powered chat clients and tools.
  - Enables backups to be triggered / managed as part of automated workflows.
- **Single static server URL**
  - Uses a fixed MCP server endpoint: `https://mcp.pipedream.net/v2`.
  - Same URL works across all supported MCP clients.
- **Client-agnostic authentication**
  - Authentication is performed when adding the server to each application or chat client.
- **Configuration documentation**
  - Full configuration details available via a dedicated configuration page (linked from the app page).
