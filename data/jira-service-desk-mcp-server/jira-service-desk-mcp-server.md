# Jira Service Desk MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Atlassian  
**Source:** https://mcp.pipedream.com/app/jira_service_desk

## Overview
The Jira Service Desk MCP Server exposes Jira Service Desk request and service management entities through the Model Context Protocol (MCP), enabling AI-assisted IT service management (ITSM) and support workflows within compatible chat or AI clients.

## Features
- MCP-compliant server for Jira Service Desk
- Access to Jira Service Desk–specific entities (e.g., requests)
- Designed for AI-assisted ITSM and support flows
- Static MCP endpoint usable across different clients

## MCP Server URL
- Base MCP server URL: `https://mcp.pipedream.net/v2`
- Single static URL used by all clients; authentication occurs when adding the server to your application

## Integration & Setup
- Add the MCP server to any supported chat or AI client that can connect to MCP servers
- Configuration details available via the provider’s configuration documentation
