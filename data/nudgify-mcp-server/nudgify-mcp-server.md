# Nudgify MCP Server

**Category:** Business & Commerce – MCP Servers  
**Website:** https://mcp.pipedream.com/app/nudgify  
**Provider / Brand:** Nudgify (via Pipedream Connect)  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

Nudgify MCP Server lets MCP-compatible agents and chat clients integrate Nudgify’s social proof, FOMO, and UX nudges into websites using Nudgify’s platform APIs.

## Features

- **MCP-compatible server endpoint**  
  - Static, shared MCP server URL: `https://mcp.pipedream.net/v2`  
  - Can be added to any supported MCP client / chat application.

- **Authentication at client level**  
  - Authentication is performed when you add the server to your application (per-client authentication model).

- **Nudgify platform integration**  
  - Designed to expose Nudgify’s APIs to MCP agents.  
  - Enables agents to orchestrate:  
    - Social proof nudges (e.g., recent activity, signups, purchases)  
    - FOMO nudges (scarcity, urgency elements)  
    - UX nudges (on-site guidance, contextual prompts).

- **Real-time website context**  
  - Supports real-time context from Nudgify so agents can add and manage nudges that react to live website activity.

- **Multi-client support**  
  - Single static MCP URL works for “every client,” enabling integration across multiple chat clients and tools.

- **Configuration guidance**  
  - Documentation available via a central configuration page for setting up the server in different MCP clients.
