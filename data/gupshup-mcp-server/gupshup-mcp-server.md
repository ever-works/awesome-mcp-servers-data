# Gupshup MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Pipedream  
**Source:** https://mcp.pipedream.com/app/gupshup

## Description
Gupshup MCP Server is a Pipedream-hosted MCP (Model Context Protocol) server that connects Gupshup’s omnichannel messaging APIs, bot-building tools, and mobile marketing capabilities to AI tools like Cursor, Claude, and Windsurf for workflow automation.

## Features
- Connects a Gupshup account to MCP-compatible AI tools (e.g., Cursor, Claude, Windsurf)
- Integrates with Gupshup’s omnichannel messaging APIs
- Supports use of Gupshup bot-building tools within automated workflows
- Exposes Gupshup’s mobile marketing capabilities to AI-driven automations
- Enables building and running messaging workflows via Pipedream as the MCP server layer
