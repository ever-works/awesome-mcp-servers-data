# Beehiiv MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** beehiiv  
**Slug:** `beehiiv-mcp-server`

An MCP (Model Context Protocol) server that connects MCP‑compatible clients to Beehiiv, enabling interaction with Beehiiv’s newsletter creation, growth, and monetization platform via a unified server endpoint.

---

## Features

- **MCP-compatible server**  
  - Exposes Beehiiv functionality to any client that supports the Model Context Protocol (MCP).

- **Newsletter platform integration**  
  - Connects to Beehiiv, a platform focused on newsletter creation, growth, and monetization.
  - Intended to let clients manage and interact with Beehiiv newsletters directly from MCP-enabled tools.

- **Single static server URL**  
  - MCP server base URL: `https://mcp.pipedream.net/v2`  
  - Same URL is used across all supported MCP clients.

- **Client-agnostic setup**  
  - Designed to work with multiple chat or MCP clients that support adding external MCP servers.
  - Configuration is handled on the client side when adding this server.

- **Authentication handled at connection time**  
  - Authentication is performed when adding the MCP server to an application/client (exact method depends on the client and Beehiiv/Pipedream setup).

- **Hosted by Pipedream Connect**  
  - Runs on Pipedream’s infrastructure as part of the Pipedream Connect ecosystem.
