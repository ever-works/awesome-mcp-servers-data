# Sonix MCP Server

MCP server integration for Sonix that provides automated transcription, translation, and subtitling capabilities in over 40 languages via MCP-based tools.

## Overview
- **Type:** MCP server / integration
- **Category:** Media processing MCP servers
- **Provider:** Sonix (via Pipedream Connect)
- **Use cases:** Audio and video transcription, translation, and subtitle generation

## Features
- Automated transcription of audio and video content
- Automated translation of transcriptions into multiple languages
- Automated subtitle generation (subtitling) for media content
- Supports over 40 languages
- Designed to be fast and accurate
- Accessible through MCP-based tools and compatible chat clients
- Uses a single static MCP server URL for all clients (`https://mcp.pipedream.net/v2`)
- Authentication handled when adding the server to your application
