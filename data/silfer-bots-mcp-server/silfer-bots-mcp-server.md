**Category:** Messaging MCP Servers

Facebook Messenger bot platform for sales automations, integrated as an MCP Server for use in MCP-based workflows.

## Description
SilFer Bots MCP Server connects the SilFer Bots Facebook Messenger sales automation platform to MCP-compatible applications. By adding the serverâ€™s static URL to an MCP client, users can incorporate Facebook Messenger chatbot sales flows into broader automated workflows.

---

## Features
- **MCP integration for SilFer Bots**  
  Integrates SilFer Bots (Facebook Messenger sales automation bots) as an MCP Server, allowing use from MCP-aware tools and clients.

- **Static MCP server endpoint**  
  Uses a single static URL (`https://mcp.pipedream.net/v2`) that works for all clients; authentication occurs when adding the server to an application.

- **Account-based configuration**  
  Connects to a SilFer Bots account, then lets you select the appropriate client / bot configuration to start using in workflows.

- **Client-agnostic setup**  
  Can be added to different MCP-compatible chat clients, with setup instructions available per client type.

- **Configuration guidance**  
  Linked configuration documentation (full configuration page) to help with connecting the MCP server and managing settings.

- **Tool discovery**  
  Supports listing and loading of available tools / actions exposed by the SilFer Bots MCP Server within the MCP client.

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Primary channel:** Facebook Messenger  
- **Use case focus:** Sales and sales automations via chatbots
