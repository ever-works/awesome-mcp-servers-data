# Getswift MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Getswift

An MCP server that connects your Getswift account and delivery management workflows with AI tooling through Pipedream. It enables integration between Getswift’s delivery and business systems platform and AI assistants such as Cursor, Claude, and Windsurf.

## Features
- Connects a Getswift account to AI tools via Pipedream’s MCP server
- Integrates with AI assistants including:
  - Cursor
  - Claude
  - Windsurf
- Bridges MCP-based workflows with Getswift’s delivery and business systems management platform
