# Withings MCP Server

**Category:** Business & Commerce · MCP Servers  
**Brand:** Withings  
**Website:** https://mcp.pipedream.com/app/withings

## Overview
Withings MCP Server is a Model Context Protocol (MCP) server that connects AI tools and chat clients to Withings smart scales, watches, and other health monitoring devices. It provides access to user health metrics and device data through a unified MCP-compatible endpoint.

## Features
- **MCP-compatible server** for integrating Withings data into AI agents and chat-based tools.
- **Single static endpoint** for all clients:
  - MCP server URL: `https://mcp.pipedream.net/v2`
- **Authentication on client setup** so each application authenticates users when the server is added.
- **Access to Withings devices and data**, including (as provided by Withings ecosystem):
  - Smart scales
  - Smart watches
  - Other health monitoring devices
- **Health metrics integration** enabling retrieval and use of health and device metrics within AI workflows (e.g., analytics, health tracking, personalized insights).
- **Client-agnostic setup**: works with multiple chat clients that support MCP; configuration handled per client.
- **Configuration documentation** available via a central configuration page for more detailed setup instructions.
- **Powered by Pipedream Connect**, leveraging Pipedream’s infrastructure for hosting and serving the MCP endpoint.

## Technical Details
- **Protocol:** Model Context Protocol (MCP)
- **Base URL:** `https://mcp.pipedream.net/v2`
- **Integration type:** API / MCP server used by AI clients and tools
