## Description
InsertChat MCP Server is an MCP-compatible service for creating AI chatbots that are powered by your own data and aligned with your brand, for use inside MCP-based workflows.

## Features
- MCP server endpoint for InsertChat: `https://mcp.pipedream.net/v2`
- Static MCP server URL that works across compatible MCP clients
- Authentication handled when adding the server to an MCP-enabled application
- Designed for building AI chatbots that use your own data
- Supports brand-aligned chatbot experiences within MCP workflows
- Integrates via Pipedream Connect infrastructure

## Integration & Usage
- Add the MCP server URL to your MCP-compatible chat client or application
- Configure and manage usage via the InsertChat MCP configuration page (linked from the app page)
