# Goody MCP Server

An MCP server that connects Goody’s corporate gifting platform to MCP-compatible AI agents, enabling automated creation and management of gifting workflows.

## Overview
- **Type:** MCP server / integration
- **Category:** Business & Commerce – MCP Servers
- **Use Case:** Corporate gifting, loyalty, and e-commerce workflows automated via AI agents
- **Provider:** Goody (delivered via Pipedream Connect)
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP integration endpoint**
  - Single static MCP server URL: `https://mcp.pipedream.net/v2`
  - Works with any compatible MCP client; authentication is handled when you add the server to your application.

- **Goody corporate gifting connection**
  - Integrates with Goody’s corporate gifting platform to power gifting-related workflows (e.g., sending and managing gifts) through MCP.

- **AI agent automation**
  - Designed for use by AI agents / chat-based tools to automate creation and management of gifting workflows.

- **Client-agnostic setup**
  - Static URL usable across different MCP-capable chat clients.
  - Configuration guidance available via a separate configuration page (not included here).

## Setup
- Add the MCP server to your MCP-compatible app or chat client using:
  - **Server URL:** `https://mcp.pipedream.net/v2`
- Authenticate within your client when prompted during server addition.
