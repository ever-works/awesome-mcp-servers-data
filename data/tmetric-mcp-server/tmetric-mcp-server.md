# TMetric MCP Server

TMetric MCP Server provides an MCP server integration for TMetric, a time tracking application designed to improve productivity and business efficiency.

## Overview
- **Name:** TMetric MCP Server  
- **Category:** Project Management MCP Servers  
- **Use Case:** Integrate TMetric time tracking capabilities into MCP-compatible clients and tooling.  
- **Base MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server endpoint**  
  - Exposes a static MCP server URL (`https://mcp.pipedream.net/v2`) that can be used by any compatible client.  
  - Central endpoint for interacting with the TMetric integration via MCP.

- **Client-agnostic configuration**  
  - Same server URL works for every supported MCP client.  
  - Configuration is handled in the client by adding this server endpoint.

- **Authentication at connection time**  
  - Users authenticate when adding the server to their application / chat client (rather than needing per-client URLs).

- **Time tracking integration**  
  - Connects to TMetric, a time tracking app focused on productivity and business efficiency.  
  - Intended for tracking time on tasks, projects, and activities through MCP-aware tools.

- **Documentation & configuration guidance**  
  - Step-by-step instructions available per chat client via the “Add the server to your app” flow.  
  - Central configuration reference available via the Pipedream MCP Configuration page.
