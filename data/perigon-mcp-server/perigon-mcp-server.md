# Perigon MCP Server

**Category:** Web Search MCP Servers  
**Website:** https://mcp.pipedream.com/app/perigon

## Overview
Perigon MCP Server is an MCP-compatible server that connects Perigon’s news intelligence platform to MCP clients. It enables applications and chat clients to compare news from multiple perspectives and convert large volumes of media content into structured, actionable insights.

## Features
- **News comparison from multiple perspectives**  
  - Compare coverage of the same topic or event from different news sources.
  - Analyze how narratives differ across outlets and regions.

- **Fact vs. fiction analysis support**  
  - Designed to help distinguish factual reporting from misleading or biased content using Perigon’s news intelligence data.

- **Media-to-insights transformation**  
  - Turn unstructured media content (e.g., news articles) into structured insights for downstream analysis.

- **MCP server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`.  
  - Single URL works for all clients; authentication occurs when adding the server to an application.

- **MCP client integration**  
  - Add the server to compatible MCP-based chat clients and applications.  
  - Configuration guided via Pipedream’s configuration pages.

- **Account-based configuration**  
  - Connect a Perigon account and select the appropriate client/profile before use.

- **Tool-based operations**  
  - Exposes “tools” (actions) over MCP for working with Perigon’s news intelligence data (exact tools are loaded dynamically within the interface).
