# Warp MCP Servers

**Vendor:** Warp  
**Category:** MCP server directories & lists

## Description
Warp MCP Servers is a built-in interface in the Warp terminal for running, configuring, and managing Model Context Protocol (MCP) servers. It provides a graphical configuration layer on top of MCP servers, allowing users to control servers and share configurations directly from the terminal.

## Features
- **Server lifecycle management**
  - Start and stop MCP servers from the MCP Servers page
  - Rename existing server configurations
  - Delete server configurations

- **Server inspection and tooling**
  - View available tools exposed by each MCP server
  - View resources provided by each MCP server

- **Configuration sharing and collaboration**
  - Share MCP server configurations with teammates
  - Automatically scrub sensitive data from configurations before sharing
  - Access shared, preconfigured MCP servers from a "Shared" section

- **Authentication options**
  - Environment variable–based authentication
  - OAuth-based one-click installation flows
  - Custom header–based authentication

- **Built-in / out-of-the-box servers**
  - Includes out-of-the-box MCP servers that can be installed directly from the Shared section

- **Integration in Warp terminal**
  - GUI and configuration layer integrated directly into the Warp terminal
  - Central place to manage, inspect, and collaborate on MCP servers
