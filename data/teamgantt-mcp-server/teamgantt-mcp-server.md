# TeamGantt MCP Server

**Category:** Project Management MCP Servers  
**Brand:** TeamGantt  
**Slug:** `teamgantt-mcp-server`

## Overview
TeamGantt MCP Server connects the TeamGantt project management and Gantt chart platform with MCP (Model Context Protocol), enabling AI agents and MCP-compatible clients to manage projects, tasks, and timelines through standardized MCP calls.

## MCP Server URL
- Static MCP server URL (for all clients):
  - `https://mcp.pipedream.net/v2`
- Authentication occurs when adding the server to your application.

## Features
- **MCP integration for TeamGantt**: Bridges TeamGantt’s project management and Gantt chart capabilities with MCP-compatible chat or agent clients.
- **Standardized MCP calls**: Allows AI agents to interact with TeamGantt data (e.g., projects and timelines) via a consistent protocol.
- **Static, reusable endpoint**: Single URL works across different MCP clients; no per-client URL variations required.
- **Client-agnostic setup**: Can be added to any supported MCP chat client or application.
- **Centralized configuration docs**: A separate configuration page (via Pipedream) provides setup details for various clients (referenced as the “Configuration page”).

## Use Cases
- Manage project timelines and Gantt charts directly from MCP-enabled chat or agent interfaces.
- Allow AI agents to read/update project schedules and tasks in TeamGantt via MCP.
