# Mailbox Power MCP Server

## Overview
Mailbox Power MCP Server is an MCP (Model Context Protocol) server integration that connects MCP-compatible clients to Mailbox Power, enabling them to trigger and manage personalized direct mail and customer engagement campaigns directly from MCP-powered applications.

- **Category:** Business & Commerce MCP Servers  
- **Use cases:** Personalized direct mail, customer engagement, and campaign management through MCP-enabled chat or automation clients.

## Features
- **MCP server endpoint**  
  - Provides a static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same URL works for every client; authentication occurs when adding the server to an application.

- **Mailbox Power integration**  
  - Connects an existing Mailbox Power account within the MCP server configuration.  
  - Lets users leverage Mailbox Powerâ€™s personalization capabilities for direct mail and engagement campaigns through MCP clients.

- **Client-agnostic setup**  
  - Designed to be added to various MCP-compatible chat or automation clients.  
  - Documentation available via a configuration page for detailed setup steps.

- **Campaign and engagement tooling (via MCP tools)**  
  - Exposes Mailbox Power tools as MCP actions (listed in the client once loaded).  
  - Intended to allow triggering and managing personalized direct mail and customer engagement workflows from within MCP environments.

## Configuration
- Connect a Mailbox Power account in the configuration interface.  
- Copy and use the static MCP server URL when adding the server to your MCP-compatible app.  
- Follow client-specific instructions (via the configuration page) to complete setup and authentication.
