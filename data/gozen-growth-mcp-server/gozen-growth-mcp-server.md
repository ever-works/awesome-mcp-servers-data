# GoZen Growth MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** GoZen  
**Slug:** `gozen-growth-mcp-server`

GoZen Growth MCP Server provides Model Context Protocol (MCP) access to GoZen’s email marketing and automation platform, allowing compatible chat or AI clients to interact with GoZen Growth features through a standardized server endpoint.

---

## MCP Server Endpoint

- **Base URL (static for all clients):**
  ```
  https://mcp.pipedream.net/v2
  ```
- Authentication is performed when adding the server to your MCP-compatible application.

---

## Features

- **MCP-based integration**
  - Exposes GoZen Growth (email marketing & automation) capabilities via the Model Context Protocol.
  - Designed to be added as an MCP server in compatible chat or AI applications.

- **Email marketing & automation access**
  - Connects your MCP client to GoZen’s email marketing and automation solution (“GoZen Growth”).
  - Intended to support operations around email campaigns and marketing workflows (exact tool list not detailed in the source content).

- **Static server URL**
  - Uses a single, static MCP server URL (`https://mcp.pipedream.net/v2`) for all clients.
  - Simplifies configuration across different MCP-enabled applications.

- **Client-agnostic setup**
  - Works with multiple MCP-compatible chat clients.
  - Setup instructions are provided per client type via the Pipedream configuration documentation.

- **Configuration documentation**
  - Full configuration guidance is available on a dedicated **Configuration** page (linked from the Pipedream app page).

---

## Integration & Setup

1. **Copy MCP server URL**
   - Use `https://mcp.pipedream.net/v2` as the MCP server endpoint.

2. **Add to your MCP-compatible client**
   - In your chat or AI application that supports MCP, add a new server using the URL above.
   - Complete authentication when prompted by your client.

3. **Optional: Refer to configuration docs**
   - For client-specific steps, follow the instructions on the Pipedream **Configuration** page.
