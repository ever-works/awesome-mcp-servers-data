# CUFinder MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** CUFinder  
**Slug:** cufinder-mcp-server

## Description
CUFinder MCP Server is an MCP Server integration for CUFinder’s B2B lead generation and enrichment platform. It allows MCP-based tools and chat clients to access CUFinder for B2B lead generation, data enrichment, and sales workflow automation.

## Features
- MCP Server integration for CUFinder’s B2B lead generation platform
- Supports lead generation and data enrichment for sales and growth workflows
- Accessible via a static MCP server URL usable across clients
- Designed to be added to compatible chat clients and MCP-based applications
- Authentication performed when adding/configuring the server in the client

### Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
