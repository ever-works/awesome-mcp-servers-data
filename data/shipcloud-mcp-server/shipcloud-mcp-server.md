# shipcloud MCP Server

**Category:** Business & Commerce MCP Servers  
**Provider / Brand:** Pipedream  
**Integration Type:** Model Context Protocol (MCP) server  
**Website:** https://mcp.pipedream.com/app/shipcloud

## Overview

shipcloud MCP Server is an MCP-compatible server hosted on Pipedream that connects MCP-based applications to shipcloud’s cloud-based shipping and carrier management services. It is designed to let applications work with multiple shipping providers through a unified integration.

## Features

- **MCP-compatible server**  
  - Exposes shipcloud functionality via the Model Context Protocol for use in MCP-enabled apps and chat clients.

- **Static server endpoint**  
  - Single MCP server URL for all clients:  
    `https://mcp.pipedream.net/v2`  
  - Authentication is handled when you add/configure the server in your client or application.

- **Shipcloud shipping integration**  
  - Connects to shipcloud’s cloud-based shipping and carrier management services.  
  - Intended for use by small and medium-sized online vendors that ship packages with multiple carriers.

- **Multi-carrier shipping support**  
  - Provides access to major shipping providers via the shipcloud service (through a single integration point).

- **Client-agnostic setup**  
  - The same MCP server URL works across different MCP-compatible chat clients and applications.  
  - Configuration instructions are available per client via Pipedream’s documentation (e.g., “Add the server to your app”).

- **Hosted by Pipedream**  
  - Runs on Pipedream’s MCP infrastructure (Pipedream Connect), removing the need to self-host the MCP server.  
  - Terms and privacy managed by Pipedream.

## Configuration & Usage

- **MCP server URL:** `https://mcp.pipedream.net/v2`  
- Add the server in your MCP-capable client or application and complete authentication as prompted.  
- Detailed setup steps and client-specific guides are available on Pipedream’s configuration pages.
