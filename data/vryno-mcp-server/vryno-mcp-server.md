## Overview
Vryno MCP Server exposes Vryno CRM capabilities through the Model Context Protocol (MCP), allowing compatible MCP clients to interact with the Vryno CRM platform. It is designed for modern businesses that need CRM functionality accessible directly from MCP-enabled applications.

Server URL (for MCP clients):
```text
https://mcp.pipedream.net/v2
```

## Features
- MCP-compatible server endpoint for Vryno CRM
- Provides access to Vryno CRM features from MCP clients (e.g., chat-based tools, MCP-enabled apps)
- Single static server URL usable across different MCP clients
- Authentication handled when adding/configuring the server in the client application
- Hosted and provided via Pipedream Connect infrastructure

## Integration & Usage
- Add `https://mcp.pipedream.net/v2` as an MCP server endpoint in your compatible client
- Authenticate through your client when prompted to link it with your Vryno CRM account
- Use within MCP-aware tools to interact with your CRM data and workflows
