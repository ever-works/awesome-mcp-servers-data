# ManyChat MCP Server

ManyChat MCP Server provides an MCP endpoint for interacting with ManyChat’s chat marketing platform, enabling automated chats and marketing flows across Messenger, SMS, email, and Instagram via MCP clients.

- **Category:** business-commerce-mcp-servers  
- **Brand:** manychat  
- **Source:** https://mcp.pipedream.com/app/manychat  
- **Slug:** manychat-mcp-server  
- **Tags:** chatbots, marketing-automation, omnichannel

## Features

- MCP server endpoint for ManyChat: static URL `https://mcp.pipedream.net/v2` usable across compatible MCP clients.
- Connects MCP clients to ManyChat’s chat marketing platform.
- Supports customer engagement across multiple channels:
  - Facebook Messenger
  - SMS
  - Email
  - Instagram (IG)
- Enables building and running automated chat and marketing flows via ManyChat from MCP-enabled applications.
- Works with multiple chat clients (MCP clients) by adding the shared server URL and authenticating per application.

## Integration & Setup

- Use the static MCP server URL `https://mcp.pipedream.net/v2` for all clients.
- Authentication occurs when adding the server to each application / client.
- Additional configuration details available via the platform’s configuration documentation (referenced as “Configuration page” in the source).
