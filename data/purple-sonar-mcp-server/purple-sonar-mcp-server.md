# Purple Sonar MCP Server

## Overview
Purple Sonar MCP Server is a Model Context Protocol (MCP) server integration that connects Purple Sonar with MCP-compatible chat or AI applications. It enables monitoring of target accounts and identifying new sales or opportunity signals directly from your client application via a static MCP server endpoint.

- **Type:** MCP server integration
- **Category:** Business & Commerce MCP Servers
- **Use cases:** Sales, account-based marketing (ABM), lead and opportunity intelligence
- **Source / Setup URL:** https://mcp.pipedream.com/app/sonar

## Features
- **Target account monitoring**  
  - Allows your application to monitor specified target accounts through Purple Sonar.  
  - Designed for sales and account-based marketing workflows.

- **Opportunity / signal detection**  
  - Surfaces opportunity signals and lead intelligence from Purple Sonar to your MCP-compatible client.  
  - Supports use in sales and lead-intelligence scenarios.

- **MCP server endpoint**  
  - Uses a static MCP server URL for all clients:  
    - `https://mcp.pipedream.net/v2`  
  - Authentication is performed when you add/configure the server in your application.

- **Client-agnostic integration**  
  - Can be added to different MCP-compatible chat or AI clients.  
  - Setup flow includes selecting your chat client and following client-specific instructions.

- **Account connection and configuration**  
  - Connect a Purple Sonar account and select the appropriate client to complete configuration.  
  - Supports centralized configuration via the Pipedream-based configuration page.

- **Tool / action exposure**  
  - Exposes Purple Sonar capabilities as “available tools” (MCP actions) that can be invoked from your client.  
  - Tools are dynamically loaded from the MCP server (details not enumerated in the provided content).
