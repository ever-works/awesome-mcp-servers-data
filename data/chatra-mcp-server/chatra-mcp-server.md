# Chatra MCP Server

**Description**

Chatra MCP Server provides an MCP (Model Context Protocol) interface for Chatra, a live chat messenger app that brings mobile messenger–style capabilities (e.g., conversational chat UI) to traditional website/live chat setups.

**Category**

- Messaging MCP Servers

**Tags**

- Live chat  
- Customer support  
- Messaging

**Integration Details**

- **MCP endpoint URL**: `https://mcp.pipedream.net/v2`  
  - Static URL used by all compatible MCP clients.  
  - Authentication is performed when adding the server to your application.

**Features**

- Exposes Chatra as an MCP-compatible server for use with MCP-aware clients.  
- Uses a single, static MCP server URL that can be reused across different clients.  
- Supports authentication at connection time (when adding the server to an app).  
- Designed to connect Chatra’s live chat and mobile-messenger-like experience into tools that support the MCP protocol.  
- Configuration guidance is available via a dedicated configuration page (client-specific setup instructions).

**Getting Started**

1. Use the MCP server URL: `https://mcp.pipedream.net/v2`.  
2. Add this server URL in your MCP-compatible client or application.  
3. Complete authentication when prompted by your client.  
4. (Optional) Refer to the configuration documentation for client-specific setup.
