# Cliengo MCP Server

An MCP server integration for Cliengo that exposes chatbot-driven lead capture and sales opportunity data to MCP-enabled assistants and workflows.

- **Website / Source**: https://mcp.pipedream.com/app/cliengo
- **Category**: Business & Commerce – MCP Servers
- **Tags**: CRM, lead management, sales
- **MCP Server URL**: `https://mcp.pipedream.net/v2`

## Overview
Cliengo MCP Server connects Cliengo’s chatbot platform—which engages website visitors and captures leads—to the Model Context Protocol (MCP) ecosystem. This allows assistants and automation workflows to access and act on Cliengo lead and sales opportunity data via a standardized MCP server.

## Features
- **MCP server endpoint**
  - Static MCP server URL (`https://mcp.pipedream.net/v2`) usable for all clients.
  - Supports authentication at the time you add the server to your MCP-enabled application.

- **Chatbot-driven lead capture integration**
  - Surfaces data generated by Cliengo’s chatbot that proactively chats with website visitors.
  - Exposes lead and sales opportunity information collected through Cliengo conversations to MCP clients (e.g., AI assistants, workflows).

- **MCP client compatibility**
  - Designed to be added to various MCP-compatible chat clients and tools.
  - Configuration guidance available via a centralized configuration page (on the source site).

- **Pipedream Connect integration**
  - Provided and operated via Pipedream Connect’s infrastructure.
  - Can be used as part of broader automation or workflow setups that use Pipedream.

## Setup & Integration
- Use the static MCP endpoint: `https://mcp.pipedream.net/v2`.
- Add the server to your MCP-compatible chat client or application.
- Authenticate during server addition as required by your client / environment.
- Additional configuration instructions are available on the provider’s configuration page (linked from the source site).
