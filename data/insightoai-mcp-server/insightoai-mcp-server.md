## Overview
Insighto.ai MCP Server is an AI agent builder exposed via the Model Context Protocol (MCP). It allows users to build, customize, and deploy AI agents, and connect them as tools to compatible chat or agent clients using a static MCP server URL.

## Features
- **AI agent builder**: Create, customize, and deploy AI agents using Insighto.ai through the MCP interface.
- **MCP protocol support**: Exposes Insighto.ai capabilities as an MCP server for use with MCP-compatible clients.
- **Static MCP server URL**: Single URL (`https://mcp.pipedream.net/v2`) used across clients; authentication is handled when adding the server to each application.
- **Multi-client integration**: Can be added to different chat / agent clients; configuration details are provided per client.
- **Configuration guidance**: Linked configuration page with instructions on how to connect accounts and select clients.
- **Account-based access**: Requires connecting an Insighto.ai/Pipedream account and selecting a client before using available tools.
- **Tool discovery**: Lists and loads available actions / tools from Insighto.ai through the MCP server for use within clients.

## Technical Details
- **MCP endpoint:** `https://mcp.pipedream.net/v2`
- **Hosting / provider:** Implemented and hosted via Pipedream.
