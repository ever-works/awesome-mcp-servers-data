# Sendy MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Sendy  
**Source:** https://mcp.pipedream.com/app/sendy

An MCP server that connects to Sendy, a self-hosted newsletter application using Amazon SES, enabling MCP tools and chat clients to manage and send email campaigns programmatically.

---

## Features

- **Sendy integration**
  - Connects directly to a self-hosted Sendy instance.
  - Uses Amazon SES as the underlying email delivery service.
  - Designed for programmatic control of newsletter and campaign operations via MCP-compatible clients.

- **MCP server endpoint**
  - Static MCP server URL: `https://mcp.pipedream.net/v2`.
  - Same URL works across all compatible MCP clients.
  - Authentication is handled when adding/configuring the server within the client.

- **Client compatibility**
  - Can be added to multiple chat or MCP-enabled clients.
  - Central configuration via a single endpoint for different tools.

- **Configuration and setup**
  - Instructions available per chat client through the Pipedream Connect interface.
  - Additional details on usage and configuration provided on the Configuration page (linked from the app page).
