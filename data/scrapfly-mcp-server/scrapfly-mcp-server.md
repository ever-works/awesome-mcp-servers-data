# Scrapfly MCP Server

## Overview
Scrapfly MCP Server is an MCP-compatible integration that connects to the Scrapfly Web Scraping API, providing developer-focused web scraping capabilities within MCP-enabled chat and development environments. It is delivered via Pipedream Connect using a static MCP server URL.

- **Category:** Content Extraction & Summarization MCP Servers
- **Brand:** Scrapfly
- **Slug:** `scrapfly-mcp-server`
- **Primary Use Cases:** Web scraping, data extraction from websites via API within MCP clients.

## Features
- **MCP Server for Scrapfly API**  
  Connects MCP-compatible clients and tools to the Scrapfly Web Scraping API.

- **Static MCP Endpoint**  
  Uses a single static MCP server URL for all compatible clients:  
  `https://mcp.pipedream.net/v2`

- **Client-Agnostic Integration**  
  The same MCP endpoint can be added to multiple chat or development clients that support MCP.

- **Authentication at Client Setup**  
  Authentication is handled when adding the server to your application/client, allowing secure access through a shared endpoint.

- **Pipedream Connect Integration**  
  Hosted and powered by Pipedream Connect, simplifying setup and management of the MCP server.

- **Configuration Documentation**  
  A dedicated configuration page is available (via Pipedream) to guide setup and usage in different clients.

## Getting Started
- Use the MCP server URL: `https://mcp.pipedream.net/v2`
- Add this URL in your MCP-compatible chat or development client.
- Authenticate when prompted during server setup in your application.
- Refer to the Pipedream-hosted configuration page for client-specific setup instructions.
