# Brex MCP Server

## Overview
Brex MCP Server is an MCP integration that connects your Brex financial account to AI tools, enabling automated interactions with payments, deposits, and card data.

- **Category:** Finance / Market Data MCP Servers  
- **Brand:** Brex  
- **Source:** https://mcp.pipedream.com/app/brex

## Features
- Integrates Brex’s financial platform with MCP-compatible AI tools.  
- Allows AI clients to interact with Brex payments, deposits, and card data.  
- Supports automated workflows involving Brex financial data.  
- Works with AI tools such as Cursor, Claude, and Windsurf via Pipedream’s MCP server.

## Integrations
- Cursor  
- Claude  
- Windsurf  
- Other MCP-compatible AI clients via Pipedream’s MCP server
