# UiPath MCP Server

Automation Platform – MCP server that connects MCP clients with UiPath’s RPA and automation platform.

- **Category:** Workflow Automation MCP Servers  
- **Slug:** `uipath-mcp-server`  
- **Source URL:** https://mcp.pipedream.com/app/uipath  
- **Provider / Brand:** UiPath (via Pipedream Connect)

## Features

- **Static MCP server endpoint**  
  - Single URL for all compatible MCP clients:  
    - `https://mcp.pipedream.net/v2`  
  - Same endpoint used across different chat or MCP-enabled applications.

- **MCP client integration**  
  - Designed to be added as an MCP server in various chat clients / MCP-capable apps.  
  - Authentication occurs when adding the server to the client application.

- **UiPath automation access**  
  - Bridges MCP clients with UiPath’s RPA and automation capabilities.  
  - Suitable for task and workflow automation scenarios (RPA, automation, task automation).

- **Configuration guidance**  
  - Additional setup and usage details available via a central configuration page (linked from the app page).

- **Pipedream Connect infrastructure**  
  - Server is hosted and delivered via Pipedream Connect.
