# Coupontools MCP Server

## Overview
Coupontools MCP Server is an MCP-compatible integration that connects your MCP-enabled applications to Coupontools’ promotional and coupon-based marketing features via a static server URL.

## Features
- MCP-compatible server endpoint for Coupontools
- Central access point to Coupontools’ promotional and coupon-based marketing capabilities
- Static MCP server URL usable across clients (`https://mcp.pipedream.net/v2`)
- Authentication handled at the time you add the server to your MCP-enabled application
- Works with multiple chat or MCP clients (add the same server URL to different apps)
- Configuration documentation available through a dedicated configuration page

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Integration Platform:** Powered by Pipedream Connect
- **Usage Model:** Add as an MCP server within your chat client or MCP-compatible application, then authenticate per client
