# SendinBlue MCP Server

An MCP (Model Context Protocol) server that connects applications to Sendinblue (Brevo) for advanced email marketing and automation capabilities.

## Overview
- **Type:** MCP server / integration
- **Provider:** Pipedream
- **Platform:** Sendinblue (Brevo)
- **Category:** Business & Commerce – MCP Servers
- **Primary Use Cases:** Email marketing, marketing automation, campaign operations via MCP-compatible clients

## Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL usable from any compatible client:  
    `https://mcp.pipedream.net/v2`
- **Sendinblue integration**  
  - Designed to expose Sendinblue’s email marketing and automation capabilities to MCP clients.
- **Client-agnostic setup**  
  - Same server URL for all clients; authentication handled when adding the server to your specific application.
- **Configuration documentation**  
  - Dedicated configuration guidance available via the Pipedream Connect configuration page.
