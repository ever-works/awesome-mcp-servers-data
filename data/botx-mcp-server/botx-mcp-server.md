# BotX MCP Server

An MCP server for BotX that lets MCP-compatible clients build and orchestrate no-code AI workflows with ChatGPT and other models.

- **Category:** Workflow Automation MCP Servers  
- **Tags:** no-code, AI workflows, automation  
- **Source:** https://mcp.pipedream.com/app/botx

---

## Features

- **MCP-compatible server**  
  - Exposes a Model Context Protocol (MCP) server endpoint for use with supported chat / AI clients.

- **Static server URL**  
  - Single MCP endpoint for all clients:  
    `https://mcp.pipedream.net/v2`  
  - Authentication handled when adding the server inside each client application.

- **No-code AI workflow orchestration**  
  - Designed for building and running no-code AI workflows.  
  - Works with ChatGPT and other AI models (via BotX and Pipedream Connect).

- **Client-agnostic configuration**  
  - Same MCP server URL for every compatible client.  
  - Per-client setup instructions available through a configuration guide.

- **Powered by Pipedream Connect**  
  - Integrates with Pipedreamâ€™s automation platform for connecting services and orchestrating workflows.

## Integration & Setup

- Add the MCP server to your preferred chat / AI client using the static URL.  
- Authenticate through the client when prompted.  
- Optional: refer to the full configuration documentation (linked from the source page) for client-specific setup.
