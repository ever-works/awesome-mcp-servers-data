# Freshlearn MCP Server

## Overview
Freshlearn MCP Server is an MCP (Model Context Protocol) server that connects AI/chat clients to FreshLearn, an all‑in‑one creator suite for monetizing knowledge products such as courses and digital content.

- **Name:** Freshlearn MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Brand:** Freshlearn  
- **Purpose:** Enable AI/chat applications to interact with FreshLearn to work with courses, digital products, and other monetized knowledge assets.

## Key Details
- **MCP server URL (static for all clients):**  
  `https://mcp.pipedream.net/v2`
- **Authentication:** Performed when adding the server to your application / client.
- **Integration platform:** Provided via Pipedream Connect.

## Features
Based on the available information, the server provides:

- **Standard MCP endpoint:**
  - Single static base URL (`https://mcp.pipedream.net/v2`) usable by any compatible MCP client.
  - Authentication handled during client/server configuration.

- **FreshLearn integration context:**
  - Designed to interact with FreshLearn, an all‑in‑one creator suite centered on:
    - Knowledge monetization
    - Online courses
    - Digital products / digital content

- **Client-agnostic setup:**
  - Can be added to different chat or AI clients that support MCP.
  - Configuration guidance available via a generic configuration page (not client‑specific in the core URL).
