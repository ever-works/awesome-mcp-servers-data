# Product Hunt MCP Server

**Category:** Search & Discovery MCP Servers  
**Brand:** Product Hunt

An MCP (Model Context Protocol) server that exposes Product Hunt data and functionality so agents and AI applications can discover and work with the best new tech products programmatically.

---

## Features

- Exposes Product Hunt data and functionality over the Model Context Protocol (MCP)
- Enables programmatic discovery of new tech products from Product Hunt
- Designed for integration with MCP-compatible chat and AI clients
- Uses a single static MCP server URL for all clients
- Authentication performed when adding the server to your application

---

## MCP Server URL

Use this static URL in any compatible client:

```text
https://mcp.pipedream.net/v2
```
