# Dribbble MCP Server

**Category:** Content Management MCP Servers  
**Brand:** Dribbble  
**Slug:** dribbble-mcp-server

An MCP server that integrates with Dribbble, enabling tools and applications that support the Model Context Protocol (MCP) to interact programmatically with Dribbble’s design sharing and discovery platform.

## Features

- Integration with Dribbble’s design sharing and discovery platform for digital designers and creatives.
- Provides a static MCP server endpoint usable across compatible clients:  
  - **MCP server URL:** `https://mcp.pipedream.net/v2`
- Designed to be added to MCP-compatible chat or AI clients for programmatic interaction with Dribbble content (e.g., design discovery and related workflows).
- Authentication handled when adding the server to an MCP-compatible application.
- Configuration details available via a dedicated configuration page (for supported clients and setup steps).

## Technical Details

- **Server type:** MCP (Model Context Protocol) server
- **Base endpoint:** `https://mcp.pipedream.net/v2`
- **Provider:** Pipedream Connect (infrastructure for hosting and connecting MCP servers)

## Use Cases

- Discovering and referencing design work from Dribbble within MCP-enabled tools.
- Integrating Dribbble design content into AI/chat workflows for designers and creative teams.
