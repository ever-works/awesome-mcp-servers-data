# Streak MCP Server

**Category:** Business & Commerce · MCP Servers  
**Tags:** CRM, Gmail, Sales

## Overview
Streak MCP Server provides Model Context Protocol (MCP) access to Streak, a CRM and process management system that runs inside Gmail. It enables AI tools and MCP-compatible clients to interact with Streak pipelines, contacts, and email-related data directly via a unified MCP endpoint.

## Features
- MCP-based access to Streak CRM
- Works with Streak pipelines and process management data
- Integration with contacts and email-related data inside Gmail
- Single static MCP endpoint usable from any compatible client:
  - Server URL: `https://mcp.pipedream.net/v2`
- Authentication handled when adding the server in the client/application
- Compatible with multiple chat / MCP clients (configured per client)

## Setup & Integration
- Use the static MCP server URL: `https://mcp.pipedream.net/v2`
- Add the server in your MCP-compatible app or chat client and complete authentication there
- Additional configuration details are available on the provider’s configuration page (per client instructions)
