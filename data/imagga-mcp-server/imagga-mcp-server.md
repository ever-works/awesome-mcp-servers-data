# Imagga MCP Server

An MCP Server integration for Imagga's Image Recognition API, enabling image tagging, categorization, visual search, and content moderation within MCP-compatible environments.

## Category
- Media Processing MCP Servers

## Tags
- Image recognition  
- Computer vision  
- Content moderation

## Features
- **MCP-compatible server URL**  
  - Static MCP server endpoint: `https://mcp.pipedream.net/v2`  
  - Works across supported MCP clients (e.g., ChatGPT / OpenAI clients)  
  - Authentication handled when adding the server to your application

- **Imagga Image Recognition integration**  
  - Uses Imagga’s Image Recognition API  
  - Supports image tagging and categorization  
  - Enables visual search use cases  
  - Provides tools for content moderation workflows

- **Available tools (actions)**  
  - **Process Batch of Images**  
    - Analyze a batch of images in a single operation  
    - Supports categorization  
    - Supports image tagging  
    - Supports color extraction from images
  
  - **Analyze Image**  
    - Analyze a single image  
    - Assigns a category based on the image’s visual content

- **Client configuration guidance**  
  - Instructions for adding the MCP server to supported chat clients (e.g., ChatGPT/OpenAI)  
  - Link to a general configuration page for additional setup details

## Integration
- Hosted and provided via Pipedream’s MCP platform
- Requires signing in with a Pipedream account to connect and manage Imagga credentials
