# ProcessPlan MCP Server

Visually track, manage, and automate business processes via MCP.

- **Website / Source**: https://mcp.pipedream.com/app/processplan
- **Category**: Workflow automation MCP servers
- **Brand**: ProcessPlan
- **Tags**: bpm, workflow-automation, process-management

## Overview
ProcessPlan MCP Server is an MCP-compatible server, hosted via Pipedream, that connects ProcessPlan’s business process management capabilities to MCP-enabled chat or AI clients. It focuses on visual tracking, management, and automation of business processes through a standardized MCP interface.

## Features
- **MCP-compatible server**
  - Exposes ProcessPlan functionality over the Model Context Protocol (MCP) for integration with MCP-capable applications.
- **Static server endpoint**
  - Single shared MCP endpoint for all clients: `https://mcp.pipedream.net/v2`.
  - Authentication is handled when adding the server within each client’s configuration.
- **Visual process tracking**
  - Designed to support visual tracking of business processes managed in ProcessPlan.
- **Process management**
  - Interfaces with ProcessPlan to manage and organize business processes through MCP.
- **Workflow automation**
  - Enables automation of workflows and processes defined in ProcessPlan from within MCP-aware tools.
- **Client-agnostic usage**
  - Works with any compatible chat or AI client that supports adding custom MCP servers.
- **Configuration guidance**
  - Documentation and setup instructions available via a central configuration page on Pipedream.

## Technical Details
- **MCP Server URL**: `https://mcp.pipedream.net/v2`
- **Hosting / Infrastructure**: Powered by Pipedream Connect.
