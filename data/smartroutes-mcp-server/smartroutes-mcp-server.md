# SmartRoutes MCP Server

## Overview
The SmartRoutes MCP Server connects MCP-compatible agents to the SmartRoutes logistics platform, enabling route planning, proof-of-delivery capture, automated notifications, and fleet tracking from within MCP-enabled applications.

## Features
- **MCP integration**
  - Static MCP server URL usable across clients
  - Authentication handled when adding the server to your MCP-enabled app or chat client
- **Route planning**
  - Supports planning and managing delivery routes via MCP tools (as exposed by SmartRoutes through this server)
- **Proof-of-delivery (POD) capture**
  - Allows MCP agents to initiate or manage proof-of-delivery capture workflows
- **Automated notifications**
  - Enables automation of customer or internal notifications related to routes and deliveries
- **Fleet tracking**
  - Provides access to fleet tracking capabilities through MCP tools
- **Client-based configuration**
  - Connect a SmartRoutes account and select a specific client/profile within that account for operations
- **Multi-client / chat client support**
  - Can be added to different MCP-compatible chat clients or applications using the same server URL

## Configuration
- Connect your SmartRoutes account in the Pipedream interface.
- Select the desired client within your SmartRoutes account to use with the MCP server.
- Copy and use the MCP server URL: `https://mcp.pipedream.net/v2` when configuring your MCP-compatible app or chat client.
