# Zylvie MCP Server

**Category:** Business & Commerce – MCP Servers  
**Website:** https://mcp.pipedream.com/app/zylvie

Zylvie MCP Server connects Zylvie’s high-converting cart and digital product subscription platform with MCP-compatible clients, enabling programmatic control over e-commerce workflows such as selling digital products and managing subscriptions.

---

## Features

- **MCP-compatible server endpoint**  
  - Static MCP server URL usable across compatible clients:  
    - `https://mcp.pipedream.net/v2`
  - Central endpoint for connecting MCP clients to Zylvie via Pipedream.

- **Integration with Zylvie platform**  
  - Connects to Zylvie’s cart and checkout system for digital products.  
  - Enables subscription workflows for recurring digital product billing.

- **Programmatic e-commerce workflow management**  
  - Designed for managing e-commerce flows (e.g., cart, checkout, subscriptions) via MCP-aware applications.

- **Client-agnostic setup**  
  - Single static URL works for all supported MCP clients; authentication is handled when adding the server to each application.

- **Configuration guidance**  
  - Documentation available via a configuration page on Pipedream for adding the server to different chat or MCP clients.

---

## Technical Details

- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Provider:** Zylvie, delivered through Pipedream Connect
