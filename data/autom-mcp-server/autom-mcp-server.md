# Autom MCP Server

**Brand:** autom  
**Category:** Web Search MCP Servers  
**Source:** https://mcp.pipedream.com/app/autom

## Overview
Autom MCP Server is a Model Context Protocol (MCP) server that provides programmatic access to Google search and SERP (Search Engine Results Page) data from within MCP-compatible clients.

## Features
- MCP-compatible server for integration with MCP clients and chat applications.
- Programmatic Google web search from within the Model Context Protocol.
- SERP (search results) API access for retrieving structured search result data.
- Optimized for fast retrieval of Google search results.
- Single static base URL for all clients: `https://mcp.pipedream.net/v2`.
- Authentication handled when adding/configuring the server in your application.
- Configuration guidance available via a dedicated configuration page (client-specific setup instructions).

## MCP Server URL
- **Base URL:** `https://mcp.pipedream.net/v2`
- **Usage:** Same static URL for every MCP client; authentication is applied when you add/configure the server in your app.
