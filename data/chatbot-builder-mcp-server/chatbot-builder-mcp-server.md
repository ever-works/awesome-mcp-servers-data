# Chatbot Builder MCP Server

## Overview
Chatbot Builder MCP Server is an MCP (Model Context Protocol) server that connects to Chatbot Builder so you can build and run custom chatbots and GPT-based assistants across multiple channels, including websites, social media, text messaging, and phone calls.

- **Use cases:** Custom conversational agents, multi-channel customer support, marketing bots, automated phone/text workflows.

## Features
- **MCP server for Chatbot Builder**
  - Provides a static MCP server endpoint: `https://mcp.pipedream.net/v2`.
  - Can be added as a server to compatible MCP-enabled applications.

- **Multi-channel chatbot deployment**
  - Build custom chatbots and GPTs for:
    - Websites
    - Social media platforms
    - Text messaging (SMS and similar channels)
    - Phone calls
    - Other supported messaging channels

- **Account-based configuration**
  - Connect your Chatbot Builder / Pipedream account.
  - Select your client (chat client) within the configuration flow.

- **Client-specific setup guidance**
  - Choose your chat client to view tailored setup instructions.
  - Option to reference a full configuration page for more detailed setup steps.

- **Authentication at application level**
  - The MCP server URL is shared across clients; authentication occurs when you add the server to your specific application.

## Integration & Setup
- Copy the static MCP server URL: `https://mcp.pipedream.net/v2`.
- Add it as a server in your MCP-compatible app.
- Authenticate and select the appropriate client to start using Chatbot Builder tools.
