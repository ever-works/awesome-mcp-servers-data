## Description
GetResponse MCP Server is an MCP (Model Context Protocol) server integration that connects applications to GetResponse, an inbound marketing and email automation platform. It enables MCP-compatible clients to access and work with GetResponse marketing and campaign capabilities through a standardized server endpoint.

## Features
- MCP server integration for GetResponse
- Access to GetResponse marketing and campaign-related capabilities via MCP
- Single static MCP server URL usable across clients: `https://mcp.pipedream.net/v2`
- Authentication handled when adding/configuring the server in the client application
- Compatible with multiple chat or MCP-enabled clients (added as an external server)

## Technical Details
- MCP Server URL: `https://mcp.pipedream.net/v2`
- Integration hosted and provided via Pipedream Connect
- Configuration details and client-specific setup available through the linked configuration page (external to this summary)
