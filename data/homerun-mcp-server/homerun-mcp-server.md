# Homerun MCP Server

**Category:** Business & Commerce – MCP Servers  
**Tags:** recruiting, HR  
**Provider / Brand:** Homerun (via Pipedream Connect)

An MCP (Model Context Protocol) server that connects MCP-compatible tools to Homerun’s ATS and hiring software, enabling interaction with recruiting and hiring features from within chat or agent-based clients.

---

## Features

- **Standard MCP Endpoint**  
  - Uses a single static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same URL works across all compatible clients.

- **Client-Agnostic Integration**  
  - Designed to be added to any MCP-compatible chat or agent client.  
  - Configuration handled at the client level; the server URL stays the same.

- **Authentication at Client Setup**  
  - Authentication occurs when adding the Homerun MCP server to your application/client.  
  - Keeps the server URL static while securing access per user/client.

- **Homerun ATS Connectivity**  
  - Bridges MCP tools with Homerun’s applicant tracking and hiring software for SMBs.  
  - Intended to surface and work with recruiting and hiring-related data and actions exposed by Homerun (e.g., jobs, candidates, hiring workflows), as supported by the MCP integration.

- **Pipedream Connect Integration**  
  - Operates as part of Pipedream’s "Connect" ecosystem, benefiting from their infrastructure for MCP servers.

- **Configuration Documentation**  
  - A dedicated configuration page (linked from the app) provides step-by-step instructions for adding the server to different MCP clients.

---

## Usage

1. Use the MCP server URL:  
   `https://mcp.pipedream.net/v2`
2. Add it in your MCP-compatible client’s configuration for servers.
3. Authenticate when prompted by your client to enable access to Homerun.
