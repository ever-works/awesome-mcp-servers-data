# Lifx MCP Server

An MCP server for controlling Wi‑Fi–enabled LIFX LED smart lights via MCP-compatible tools.

## Overview
- **Type:** MCP server (Machine Connection Protocol)
- **Purpose:** Enable chat clients and other MCP-compatible applications to control LIFX smart lighting devices over Wi‑Fi.
- **Provider:** Pipedream
- **Static MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **Static MCP Endpoint**
  - Single, static URL (`https://mcp.pipedream.net/v2`) used across all supported clients.
  - Authentication performed when adding the server in each client, not by changing the URL.

- **MCP Compatibility**
  - Designed to work with MCP-compatible chat clients and tooling.
  - Can be added as an MCP server in various applications following their MCP configuration process.

- **LIFX Smart Light Integration**
  - Targets Wi‑Fi–enabled LIFX LED smart lights.
  - Provides a unified MCP-based interface for device control (exact operations not detailed in the source content).

- **Configuration Resources**

  - Additional configuration details available via a central configuration page (not reproduced here).
