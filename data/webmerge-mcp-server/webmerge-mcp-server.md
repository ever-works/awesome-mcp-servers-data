# WebMerge MCP Server

**Category:** Document Management MCP Servers  
**Brand:** Pipedream  
**Source:** https://mcp.pipedream.com/app/webmerge

WebMerge MCP Server exposes WebMerge (now Formstack Documents) via the MCP protocol, enabling programmatic document generation and automated paperwork workflows from compatible MCP clients.

---

## Features

- **Automated document generation**  
  - Integrates with WebMerge / Formstack Documents to create documents programmatically.  
  - Suitable for generating PDFs and other document formats.

- **Paperwork workflow automation**  
  - Designed to automate end-to-end paperwork processes (e.g., form outputs, contracts, reports).  
  - Enables consistent, repeatable document workflows through MCP-compatible tools.

- **MCP protocol integration**  
  - Exposes WebMerge capabilities as an MCP server, allowing use from MCP-enabled chat clients and applications.

- **Static MCP server endpoint**  
  - Single shared server URL for all clients:  
    - `https://mcp.pipedream.net/v2`  
  - Authentication is handled when adding the server within your client/application.

- **Client-agnostic setup**  
  - Works with multiple MCP-compatible chat clients.  
  - Can be configured following the general MCP configuration instructions on the linked configuration page.

---

## Configuration

- **Server URL**: `https://mcp.pipedream.net/v2`  
- **Usage**: Add this URL as an MCP server in your MCP-compatible app or chat client, then authenticate as prompted.  
- **Additional details**: Full configuration guidance is available on the Pipedream MCP Configuration page (linked from the app page).
