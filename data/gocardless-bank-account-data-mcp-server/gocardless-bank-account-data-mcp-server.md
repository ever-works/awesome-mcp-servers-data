# GoCardless Bank Account Data MCP Server

**Description**  
GoCardless Bank Account Data MCP Server (formerly Nordigen) provides Model Context Protocol (MCP) access to GoCardlessâ€™s bank account data aggregation services, enabling applications to interact with open banking and financial account data through a standardized MCP interface.

**MCP Server URL**  
This static URL is used to connect any MCP-compatible client:

```text
https://mcp.pipedream.net/v2
```

You authenticate when adding the server to your application.

## Features
- MCP-based access to GoCardless Bank Account Data (formerly Nordigen).
- Bank account data aggregation via open banking connections.
- Single static MCP endpoint usable by all clients (`https://mcp.pipedream.net/v2`).
- Authentication handled at the time you add/configure the server in your client.
- Compatible with multiple MCP-enabled chat or agent clients.
