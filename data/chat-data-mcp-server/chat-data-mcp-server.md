## Overview

Chat Data MCP Server is an MCP (Model Context Protocol) server integration for the Chat Data platform. It lets MCP-compatible clients connect to and interact with Chat Data’s AI + live chat chatbot and related data services via a unified MCP endpoint hosted on Pipedream.

The server is exposed via a static MCP URL and is intended to be added to compatible chat clients that support MCP, enabling them to use Chat Data’s conversational and data capabilities.

## Features

- **MCP-compatible server**
  - Provides a standardized MCP endpoint for integrating Chat Data into MCP-capable chat clients and applications.

- **Static MCP server URL**
  - Single, static MCP URL for all clients:  
    `https://mcp.pipedream.net/v2`
  - Same URL works across different MCP client implementations.

- **Client-side authentication**
  - Authentication occurs when adding/configuring the server inside your application or client.

- **Integration with Chat Data platform**
  - Connects MCP clients to Chat Data’s AI + live chat chatbot.
  - Provides access to associated data services from the Chat Data platform (as exposed through the MCP server).

- **Pipedream Connect hosting**
  - MCP server is powered and hosted through Pipedream Connect infrastructure.

- **Multi-client support**
  - Intended to be added to a variety of MCP-compatible chat clients.
  - Documentation available per client type via “Select your chat client to learn how to get started”.

- **Centralized configuration resources**
  - Additional setup and configuration instructions are available on a dedicated configuration page ("Configuration"), covering how to integrate the server into different apps.

## Usage

- Use the static MCP server URL `https://mcp.pipedream.net/v2` when configuring an MCP client.
- Authenticate within your chosen client during setup.
- Optionally consult the full configuration page for client-specific instructions.
