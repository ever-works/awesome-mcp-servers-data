## Features
- Integrates Yr meteorological and weather data into MCP-based applications.
- Operates as an MCP server via Pipedream.
- Enables connection of a Yr account to AI tools and editors such as:
  - Cursor
  - Claude
  - Windsurf
- Provides programmatic access to open weather data from the Norwegian Meteorological Institute for use in AI workflows and automations.

## Integrations
- Works within the Pipedream MCP server environment.
- Connects Yr data to compatible MCP clients (e.g., Cursor, Claude, Windsurf).
