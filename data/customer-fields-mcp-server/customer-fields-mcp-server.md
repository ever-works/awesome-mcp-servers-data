# Customer Fields MCP Server

## Overview
Customer Fields MCP Server is an MCP-compatible service (via Pipedream Connect) that lets you create custom customer sign-up forms and store associated data on customer accounts.

- **Category:** Business & Commerce – MCP Servers  
- **Use case:** Custom customer onboarding, profile enrichment, and customer data management within MCP-enabled tools.

## Features
- **Custom sign-up forms**  
  - Create customer sign-up forms tailored to your business requirements.  
  - Support for capturing custom fields beyond standard customer details.

- **Customer data management**  
  - Save custom form responses directly to customer accounts.  
  - Store and manage structured customer attributes for downstream use (e.g., segmentation, personalization).

- **MCP compatibility**  
  - Exposes Customer Fields functionality as an MCP server, allowing integration with MCP-compatible clients and tools.  
  - Designed to work with a single static server endpoint.

- **Static MCP server endpoint**  
  - Uses a single, static server URL for all clients:  
    - `https://mcp.pipedream.net/v2`  
  - Authentication is performed when adding the server to your client/application.

- **Integration via Pipedream Connect**  
  - Provisioned and managed through Pipedream’s Connect platform.  
  - Can be added to supported chat or MCP clients following Pipedream’s configuration guides.

## Getting Started
- Add the following MCP server URL to your compatible client:  
  - `https://mcp.pipedream.net/v2`  
- Authenticate within your client when prompted.  
- Configure how your client will use Customer Fields for form creation and customer data access, following Pipedream’s configuration documentation.
