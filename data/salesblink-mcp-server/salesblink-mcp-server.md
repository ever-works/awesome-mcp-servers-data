# SalesBlink MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** SalesBlink  
**Slug:** `salesblink-mcp-server`

## Overview
SalesBlink MCP Server provides an MCP-compatible interface to SalesBlink, an all-in-one full-stack sales outreach platform designed to support the entire sales cycle from prospecting through outreach to closing.

The server is hosted and powered by Pipedream Connect and can be used by any MCP-compatible client via a static URL.

## MCP Server Endpoint
- **Base URL:** `https://mcp.pipedream.net/v2`
- **Usage:**
  - Static endpoint shared across clients
  - Authentication is performed when adding the server to your MCP-compatible application

## Features
- **MCP-Compatible Access to SalesBlink**  
  Provides a machine-consumable interface to SalesBlink’s platform so tools and agents can programmatically interact with its sales workflows.

- **Full-Stack Sales Workflow Coverage**  
  Designed to connect to SalesBlink’s capabilities across:
  - Prospecting
  - Outreach
  - Closing

- **Static MCP Server URL**  
  A single, static URL (`https://mcp.pipedream.net/v2`) that can be used with any supported MCP client.

- **Client-Agnostic Integration**  
  Works with multiple chat or MCP clients; configuration is handled at the client level using the same server URL.

- **Hosted by Pipedream Connect**  
  Infrastructure, hosting, and delivery managed by Pipedream, so the MCP server is available without self-hosting.

## Configuration
- Add `https://mcp.pipedream.net/v2` as an MCP server in your preferred MCP-compatible chat client or application.
- Authentication and any further configuration are performed when you register the server in that client.
- Additional configuration details are available on the Pipedream MCP configuration page (not included here).
