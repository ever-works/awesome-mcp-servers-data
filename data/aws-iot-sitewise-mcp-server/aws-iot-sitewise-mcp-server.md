# AWS IoT SiteWise MCP Server

**Category:** Cloud & DevOps MCP Servers  
**Brand:** AWS

## Overview
AWS IoT SiteWise MCP Server is a Model Context Protocol (MCP) server that exposes AWS IoT SiteWise capabilities to MCP-compatible clients. It is intended for industrial IoT scenarios involving asset management, data ingestion, monitoring, and analytics.

## Features
- **MCP-compatible server**  
  - Implements the Model Context Protocol (MCP) to integrate AWS IoT SiteWise with MCP-aware tools and agents.

- **Industrial IoT asset management**  
  - Surfaces AWS IoT SiteWise asset models and assets via MCP.  
  - Enables structured access to industrial equipment and process hierarchies.

- **Data ingestion integration**  
  - Exposes SiteWise data ingestion capabilities so MCP clients can interact with time-series data coming from industrial sensors and equipment.

- **Monitoring capabilities**  
  - Provides access to SiteWise metrics and time-series data for operational monitoring within MCP-compatible clients.

- **Analytics enablement**  
  - Makes SiteWise data and asset structures available for analytics workflows orchestrated through MCP (e.g., analysis, reporting, or optimization routines run by an MCP client).
