# Ringba MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Ringba  
**Source:** https://mcp.pipedream.com/app/ringba

## Overview
Ringba MCP Server is a Model Context Protocol (MCP) server that connects your Ringba account (inbound call tracking and pay‑per‑call analytics) to AI tools. It is provided via Pipedream’s MCP server integration.

## Features
- **Ringba account integration**  
  - Connects directly to your Ringba account through Pipedream’s MCP server.
- **MCP protocol support**  
  - Exposes Ringba capabilities via the Model Context Protocol so AI tools can query and use your call data.
- **AI tool connectivity**  
  - Designed to work with AI tools such as:
    - Cursor
    - Claude
    - Windsurf
- **Call tracking and analytics exposure**  
  - Surfaces Ringba’s inbound call tracking data and pay‑per‑call analytics to MCP‑compatible AI clients (e.g., for analysis, reporting, or workflow automation within those tools).

## Use Cases
- Enabling AI assistants (e.g., Claude) to access and analyze Ringba call tracking and pay‑per‑call data.  
- Integrating Ringba analytics into developer tools like Cursor and Windsurf for data‑driven prompts, insights, or automation.
