# Flexie MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Flexie  
**Source:** https://mcp.pipedream.com/app/flexie

## Overview
Flexie MCP Server is an MCP-compatible server hosted by Pipedream that connects Flexie’s flexible CRM and marketing automation capabilities to MCP-enabled agents and chat clients. It exposes Flexie CRM and workflow functionality as tools that agents can call programmatically.

## Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Single URL works for every Flexie client; authentication is handled when adding the server to an application.

- **Flexie CRM integration**  
  - Connects to a Flexie CRM account via Pipedream’s configuration UI.  
  - Allows MCP agents to access flexible CRM software features (contact and lead management, sales workflows, and related CRM data) through MCP tools.

- **Marketing and workflow automation access**  
  - Exposes Flexie marketing and automation workflows to MCP agents for programmatic use (e.g., campaigns, follow-ups, and other automated flows), as supported by the underlying Flexie account.

- **Multi-client support**  
  - Can be added to multiple chat clients or MCP-compatible applications.  
  - Each client authenticates separately while using the same base MCP server URL.

- **Configuration via web UI**  
  - “Configure Flexie” flow to connect an account and select a client before use.  
  - Links to a full configuration page for more detailed setup steps.  
  - Displays available tools/actions once configuration is complete.

- **Hosted by Pipedream**  
  - Managed hosting of the MCP server, including the tools layer that bridges Flexie CRM and MCP agents.
