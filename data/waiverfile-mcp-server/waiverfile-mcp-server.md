# WaiverFile MCP Server

**Category:** Document Management MCP Servers  
**Tags:** e-signature, forms, compliance

## Overview
The WaiverFile MCP Server integrates WaiverFileâ€™s digital waiver and release form capabilities into MCP-compatible applications. It enables users to create, manage, and process waiver forms and collect electronic signatures from computers, tablets, and mobile devices.

## Features
- **Digital waiver signing**: Customers can sign release waivers electronically on desktop, tablet, or mobile devices.
- **Multi-channel access**: Waivers can be completed from your website or at a physical kiosk.
- **Surveys & forms**: Supports configuration and use of forms and surveys in addition to waivers.
- **MCP integration**: Exposes WaiverFile functionality via an MCP Server so MCP agents / clients can interact with waiver data.
- **Static MCP endpoint**: Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) for all clients.
- **Account-based authentication**: Authentication is performed when adding the MCP server to your application, allowing secure access to your WaiverFile account.
- **Configurable connection**: Connect your WaiverFile account and select a client within the configuration flow to start using the tools.

## Technical Details
- **MCP server URL:** `https://mcp.pipedream.net/v2`  
- **Integration surface:** Add as an MCP server in your compatible chat or agent client, then authenticate with your WaiverFile account.
