# Utradea MCP Server

**Category:** Finance & Market Data MCP Servers  
**Website:** https://mcp.pipedream.com/app/utradea

## Overview
Utradea MCP Server is an MCP integration that connects applications to Utradea’s stock and crypto research environment. It exposes investment ideas, social sentiment analytics, and automated due diligence tools through a unified MCP endpoint hosted by Pipedream.

## Features
- **Unified MCP Endpoint**  
  - Static MCP server URL for all compatible clients: `https://mcp.pipedream.net/v2`  
  - Works across multiple MCP-compatible chat clients and tools.

- **Stock & Crypto Investment Ideas**  
  - Access to curated stock investment ideas.  
  - Access to crypto investment ideas.

- **Social Sentiment Analytics**  
  - Analytics on social sentiment related to stocks and cryptocurrencies (e.g., community and social media signals).

- **Automated Due Diligence Tools**  
  - Tools to automate parts of the investment research and due diligence process.  
  - Can be used to streamline evaluation of assets before making investment decisions.

- **Trade Alerts & Investment Community Integration**  
  - Access to trade alerts generated within the Utradea ecosystem.  
  - Integration with an investment-focused community for idea discovery and discussion (as exposed via MCP).

- **Pipedream-Hosted Integration**  
  - Powered by Pipedream Connect infrastructure.  
  - Central configuration and setup guidance available through Pipedream’s configuration page.

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Authentication:** Performed when adding the server to your MCP-compatible application.  
- **Client Compatibility:** Designed to be added to various MCP-enabled chat clients and tools.
