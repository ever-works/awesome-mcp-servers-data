# Dropmark MCP Server

**Category:** Content Management MCP Servers  
**Brand:** Dropmark  
**Slug:** `dropmark-mcp-server`

An MCP server for Dropmark that lets AI tools and MCP-compatible clients organize links, files, and notes into visual collections.

---

## Features

- **MCP-compatible server**  
  - Exposes Dropmark functionality over the Model Context Protocol (MCP).  
  - Designed to be used from compatible chat and AI clients.

- **Central server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`.  
  - Same URL works across all supported clients; authentication is handled when adding the server to the client.

- **Dropmark account integration**  
  - Connects to a Dropmark account via Pipedream’s configuration flow.  
  - Uses your Dropmark account to access and manage collections and items.

- **Content organization**  
  - Organize links, files, and notes.  
  - Arrange content into visual collections for easier browsing and management.

- **Client setup guidance**  
  - Per-client instructions for adding the MCP server to different chat / AI applications.  
  - Additional configuration details available on a dedicated configuration page.

- **File storage & management**  
  - Supports handling stored files through Dropmark as part of visual collections (as indicated by the “File Storage” section).
