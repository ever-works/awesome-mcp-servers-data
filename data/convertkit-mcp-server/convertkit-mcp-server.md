# ConvertKit MCP Server

Email marketing integration for online creators via the Model Context Protocol (MCP), powered by Pipedream Connect.

## Overview
The ConvertKit MCP Server allows MCP-compatible clients (such as AI/chat applications or other MCP-aware tools) to connect to ConvertKit and manage email marketing workflows and audience engagement for creators.

- **Type:** MCP server integration
- **Category:** Business & Commerce – MCP Servers
- **Provider/Brand:** ConvertKit (via Pipedream Connect)
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible endpoint**  
  - Uses a single static MCP endpoint URL: `https://mcp.pipedream.net/v2`.  
  - Works with any compatible MCP client; authentication is handled when adding the server in the client.

- **ConvertKit integration**  
  - Connects MCP clients to ConvertKit for email marketing and audience-engagement use cases for online creators.  
  - Intended to enable management of email marketing activities and subscriber/audience data directly from MCP-aware applications (details of specific tools/actions are not listed in the provided content).

- **Client-agnostic setup**  
  - The same server URL is used across different MCP-enabled chat or automation clients.  
  - Configuration is guided per client via the “Add the server to your app” flow and the full configuration page.

- **Pipedream Connect infrastructure**  
  - Hosted and operated via Pipedream Connect.  
  - Governed by Pipedream’s Terms and Privacy Policy.

## Configuration
- Use the static server URL in your MCP-capable client: `https://mcp.pipedream.net/v2`.
- Authenticate within your client when prompted during server addition.
- Client-specific setup instructions are available via the “Add the server to your app” and the main configuration page (not fully included in the provided content).
