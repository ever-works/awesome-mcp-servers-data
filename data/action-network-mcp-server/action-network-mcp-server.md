# Action Network MCP Server

An MCP server integration that exposes Action Network’s online toolset for managing and empowering progressive campaigns programmatically.

## Overview
- **Type:** MCP server integration
- **Provider:** Action Network (via Pipedream Connect)
- **Category:** Business & Commerce – MCP Servers
- **Use cases:** Programmatic access to tools for political or advocacy campaigns, organizing, and supporter engagement.

## Connection Details
- **MCP server URL:** `https://mcp.pipedream.net/v2`
  - This is a static URL used by all clients.
  - Authentication is handled when adding the server within your client/application.

## Features
- Provides programmatic access to Action Network’s online campaign tools (as exposed via Pipedream’s MCP server).
- Supports integration with compatible MCP-enabled chat or agent clients.
- Centralized, static endpoint for configuration across different clients.
- Configuration guidance available via the Pipedream Configuration page (client-specific setup instructions).
