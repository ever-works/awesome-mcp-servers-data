# Coda MCP Server

**Category:** Project Management MCP Servers  
**Brand:** Coda  
**Website:** https://mcp.pipedream.com/app/coda

## Description
Coda MCP Server exposes Coda’s docs-as-apps platform through the Model Context Protocol (MCP), allowing tools and agents to programmatically create, modify, and query Coda docs, tables, and workflows. It is delivered via Pipedream’s MCP server to connect your Coda account with AI tools such as Cursor, Claude, and Windsurf.

## Features
- **MCP integration**: Access Coda via the Model Context Protocol for standardized tool/agent interoperability.
- **Programmatic doc management**: Create and modify Coda documents through MCP-compatible tools.
- **Table operations**: Query and update tables in Coda docs programmatically.
- **Workflow interaction**: Integrate and trigger Coda-based workflows from external tools.
- **AI tool connectivity**: Connect Coda to AI tools like Cursor, Claude, and Windsurf using Pipedream’s MCP server.
- **Docs-as-apps platform access**: Leverage Coda’s docs-as-apps capabilities (docs, tables, and workflows) from within AI and agent environments.

## Integrations
- **Pipedream MCP Server** – hosting and MCP endpoint provider.
- **AI & Developer Tools** (via MCP):
