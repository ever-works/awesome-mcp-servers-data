# Refersion MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Pipedream  
**Website:** https://mcp.pipedream.com/app/refersion

## Overview
Refersion MCP Server is a Model Context Protocol (MCP) server hosted on Pipedream that gives AI agents programmatic access to Refersion’s affiliate, ambassador, and influencer marketing management capabilities. It is designed to help manage and track performance-based marketing programs via a standardized MCP interface.

## Features
- **MCP-compatible server** for integrating Refersion capabilities directly into MCP-aware chat clients and AI agents.
- **Affiliate program management** support via Refersion’s platform (accessible programmatically through the server).
- **Ambassador program management** to handle brand advocates and ambassadors.
- **Influencer marketing management** for tracking and coordinating influencer campaigns.
- **Tracking and analytics focus** for ambassador, influencer, and affiliate activity and performance (through Refersion’s underlying features).
- **Static MCP endpoint** using a single URL for all clients:  
  - Server URL: `https://mcp.pipedream.net/v2`
- **Client-agnostic usage**: the same MCP server URL works across different compatible chat clients.
- **Authentication at client setup**: authentication is handled when adding the server to your specific application/client.
- **Hosted and operated by Pipedream**, leveraging the Pipedream Connect infrastructure.

## Integration & Setup
- Use the static MCP server URL: `https://mcp.pipedream.net/v2` when configuring your MCP-compatible client.
- Authentication is required and completed during the “add server” or configuration step in your chosen chat client.
- Additional configuration details are available on Pipedream’s configuration page (linked from the app page).
