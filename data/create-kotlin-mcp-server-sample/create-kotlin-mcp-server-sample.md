# Create Kotlin MCP Server Sample

## Overview
Create Kotlin MCP Server Sample is a Kotlin-based reference implementation of a Model Context Protocol (MCP) server. It is distributed as part of the official Kotlin SDK samples and is intended as a starting point for developers who want to build custom MCP servers in Kotlin.

## Features
- **Kotlin-based MCP server template** – Provides a working example of an MCP server implemented in Kotlin.
- **Included in the Kotlin SDK samples** – Ships as part of the Model Context Protocol Kotlin SDK sample projects.
- **Best-practices reference** – Demonstrates recommended patterns and structure for implementing MCP servers in Kotlin.
- **Starter template for custom servers** – Designed to be cloned or adapted as the foundation of your own Kotlin MCP server.
- **GitHub-hosted project** – Available as source code on GitHub for inspection, modification, and integration.
