## Overview
KoliBri MCP Server is a streaming Model Context Protocol (MCP) server that exposes the public-ui KoliBri design system to MCP‑compatible AI assistants and developer tools. It provides access to more than 200 accessible web component samples, their specifications, and related resources so tools can generate and validate UI code aligned with accessibility standards.

## Features
- **Streaming MCP server**
  - Implements a streaming MCP interface for efficient, incremental responses.
  - Designed to integrate with MCP‑compatible AI assistants and IDE/devtools.

- **Accessible component catalog**
  - Provides access to 200+ KoliBri web components and patterns.
  - Focus on guaranteed accessibility and standards‑compliant HTML.
  - Components centered on UI building blocks (inputs, navigation, layout, etc.).

- **Samples and examples**
  - Delivers ready‑to‑use component samples for rapid prototyping.
  - Example snippets suitable for direct insertion into projects or AI‑generated code.

- **Specifications and API details**
  - Exposes component specifications (attributes, events, states).
  - Helps tools reason about valid configurations and usage constraints.

- **Related resources**
  - Provides links or metadata for documentation around the KoliBri design system.
  - Can be used by assistants to guide best practices for accessible UI.

- **Ecosystem & packaging**
  - Distributed via NPM as `@public-ui/mcp`.
  - Built on top of the KoliBri/public-ui repository and design system.
  - Licensed under EUPL-1.2 (for the overall repo).

## Use Cases
- Powering AI assistants to generate accessible HTML and web components.
- Enforcing accessibility‑first UI patterns in code generation tools.
- Providing structured component knowledge to IDE plugins via MCP.
