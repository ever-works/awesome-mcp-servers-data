# Zendesk Sell MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Zendesk  
**Source:** https://mcp.pipedream.com/app/zendesk_sell

## Overview
The Zendesk Sell MCP Server exposes Zendesk Sell’s modern sales CRM capabilities—such as pipeline visibility and productivity tooling—to MCP‑aware applications. It allows chat clients and other MCP-compatible tools to interact with Zendesk Sell via a unified MCP server endpoint.

## MCP Server URL
- Base MCP server endpoint (static for all clients):
  - `https://mcp.pipedream.net/v2`
- Authentication is performed when adding the server in the client/application.

## Features
- **MCP integration for Zendesk Sell**
  - Integrates Zendesk Sell’s CRM features into any MCP-aware application.
  - Uses a single, static MCP server URL for all supported clients.
- **Sales CRM capabilities (via Zendesk Sell)**
  - Modern sales CRM focused on:
    - Maximizing sales productivity.
    - Maintaining visibility into sales pipelines.
    - Supporting revenue growth workflows.
- **Client-agnostic configuration**
  - Same MCP URL across different chat clients and tools.
  - Configuration guided per-client through documentation and examples (via the configuration page).
- **Pipedream Connect integration**
  - MCP server is powered by Pipedream Connect infrastructure.

## Configuration & Setup
- Add the MCP server to your application using:
  - Server URL: `https://mcp.pipedream.net/v2`
- Follow client-specific setup instructions via the linked **Configuration** page from the source site.
