# Productboard MCP Server

## Overview
The Productboard MCP Server exposes Productboard’s customer-centric product management capabilities (feature ideas, feedback, roadmaps) through the Model Context Protocol (MCP), allowing compatible chat or MCP-enabled clients to interact with Productboard data and workflows.

- **Name:** Productboard MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Brand:** productboard  
- **Source URL:** https://mcp.pipedream.com/app/productboard  
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server**  
  - Exposes Productboard functionality via a standard MCP endpoint.  
  - Uses a single, static base URL: `https://mcp.pipedream.net/v2`.

- **Client-agnostic endpoint**  
  - Same server URL works for all supported MCP clients.  
  - Authentication is handled when adding the server within the client / application.

- **Productboard integration scope** *(from item metadata)*  
  - Work with **feature ideas** within Productboard.  
  - Interact with **customer feedback** stored in Productboard.  
  - Access or update **roadmaps** to support product planning workflows.

- **Configuration resources**  
  - Documentation available via a central **Configuration page** (client-specific setup instructions).  
  - Hosted and operated via **Pipedream Connect** infrastructure.

## Usage
- Add the MCP server to an MCP-compatible chat client or application.  
- Use the static URL `https://mcp.pipedream.net/v2` as the server endpoint.  
- Complete authentication during the add / configuration flow within the client.
