# Bird MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Pipedream  
**Slug:** `bird-mcp-server`

## Description
The Bird MCP Server connects a MessageBird (Bird) account to MCP-compatible AI tools, enabling business communication and automation capabilities to be used within MCP workflows.

## Features
- Connects a Bird (MessageBird) account to MCP-based workflows
- Integrates with AI tools that support MCP, such as Cursor, Claude, and Windsurf
- Exposes Bird’s business communication capabilities into MCP (e.g., for messaging and omnichannel scenarios)
- Supports using Bird’s automation capabilities inside AI-driven workflows

## Integrations
- Cursor (via MCP)
- Claude (via MCP)
- Windsurf (via MCP)
