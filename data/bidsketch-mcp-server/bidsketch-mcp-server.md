# Bidsketch MCP Server

**Category:** Business & Commerce – MCP Servers  
**Tags:** proposals, sales, document-automation

## Overview
The Bidsketch MCP Server connects Bidsketch proposal software with MCP-compatible tools, enabling automated proposal creation and management within sales workflows. It is provided via Pipedream Connect and uses a single static server URL for all clients.

## Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`  
  - Authentication handled when adding the server to an MCP-enabled application.

- **Proposal workflow automation**  
  - Supports automating proposal-related actions in sales pipelines using MCP tools.  
  - Enables integrating Bidsketch proposal creation, management, and approvals into chat clients or other MCP-aware applications.

- **Sales pipeline integration**  
  - Designed to help streamline proposal management as part of the overall sales process (e.g., shortening proposal turnaround, supporting up-sell workflows, and deal-closure steps) when orchestrated via MCP tools.

- **Client-agnostic setup**  
  - Same server URL for every supported client; configuration is handled per client within the MCP app.  
  - Can be added to various chat clients or other MCP-compatible interfaces.

## Configuration
- Add the MCP server to an MCP-compatible application or chat client using:  
  `https://mcp.pipedream.net/v2`  
- Authentication is performed during server addition within the client.  
- Further technical configuration details are available on the provider’s configuration documentation (referenced in the source as the “Configuration page”).
