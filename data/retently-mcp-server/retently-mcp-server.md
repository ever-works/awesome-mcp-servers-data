# Retently MCP Server

An MCP server integration for Retently to send and manage Net Promoter Score® (NPS) surveys and customer feedback analytics via MCP-compatible clients.

## Overview
- **Type:** MCP Server (Retently integration)
- **Category:** Business & Commerce – MCP Servers
- **Provider / Platform:** Pipedream Connect
- **Use Case:** Automate sending NPS® surveys to clients and collect structured customer feedback for analysis and follow‑up actions within MCP-enabled chat or agent clients.
- **MCP Endpoint URL:** `https://mcp.pipedream.net/v2`

## Features
- **NPS® Survey Delivery**
  - Send Net Promoter Score® surveys to customers from MCP clients.
  - Use a static MCP server URL that works across different clients.

- **Customer Feedback Collection**
  - Gather structured responses and qualitative feedback from survey recipients (via Retently's NPS flows).

- **Feedback Analytics (via Retently)**
  - Integrate Retently’s analytics capabilities into MCP-powered workflows for monitoring NPS and customer sentiment.

- **MCP Client Integration**
  - Add the Retently MCP server to compatible chat or agent applications.
  - Central server URL simplifies configuration across multiple clients.

- **Configuration Documentation**
  - Detailed configuration steps available via the referenced configuration page (for connecting the MCP server to supported clients).

## Authentication
- Authentication is performed when adding the MCP server to your application (exact method is defined in the app’s configuration flow).

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Server Type:** Static endpoint shared across clients; behavior is determined by configuration and authentication.
