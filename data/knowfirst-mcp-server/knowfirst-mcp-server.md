# KnowFirst MCP Server

Boost your business data with instant enrichment using KnowFirst’s Australian business intelligence API in MCP-compatible environments.

---

## Overview
- **Name:** KnowFirst MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Provider / Brand:** KnowFirst (via Pipedream Connect)  
- **Primary Function:** Enriches business data with Australian business intelligence through KnowFirst’s SaaS API, accessible as an MCP Server.

---

## Features
- **MCP-compatible server integration**  
  - Exposes KnowFirst’s capabilities as a Model Context Protocol (MCP) server for use in MCP-enabled chat clients and tools.

- **Australian business intelligence data**  
  - Focused on enriching business data with intelligence specific to Australian businesses (e.g., business details, metadata, and related insights via KnowFirst’s API).

- **Instant data enrichment**  
  - Designed for real-time or on-demand enrichment of existing business records from within MCP-compatible environments.

- **Static MCP server endpoint**  
  - Single static URL used by all clients:  
    - `https://mcp.pipedream.net/v2`  
  - Simplifies configuration and reuse across multiple applications.

- **SaaS API integration**  
  - Built on top of KnowFirst’s SaaS API for programmatic access to business intelligence data.

- **Client-agnostic setup**  
  - Works with any MCP-capable chat/client application; server URL is the same, and authentication is handled when adding it to each app.

- **Configuration documentation**  
  - Additional configuration guidance available via a dedicated configuration page (for adding the server to different apps / clients).

---

## Integration Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Usage:**  
  1. Copy the MCP server URL.  
  2. Add it to your MCP-enabled chat client or application.  
  3. Authenticate within your client when prompted.
