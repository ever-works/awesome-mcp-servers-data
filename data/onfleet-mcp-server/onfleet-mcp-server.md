# Onfleet MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Onfleet  
**Source:** https://mcp.pipedream.com/app/onfleet

## Description
Onfleet MCP Server is an MCP (Model Context Protocol) server integration that connects MCP-compatible agents and chat clients to Onfleet’s last‑mile delivery management platform. It is provided via Pipedream and exposes Onfleet functionality as tools that agents can call through a standard MCP endpoint.

## Features
- **MCP-compatible server endpoint**  
  - Provides a static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Works across all supported chat clients and MCP-compatible applications.

- **Onfleet account connectivity**  
  - Connects to an existing Onfleet account through the Pipedream interface.  
  - Uses your authenticated Onfleet client/account to execute actions via MCP.

- **Tooling exposure for agents**  
  - Exposes Onfleet tools / actions as MCP tools (listed as “Available tools” in the interface).  
  - Designed so agents can call Onfleet operations (e.g., logistics and delivery management tasks) from within a chat or other MCP-aware environments. *(Specific tools are not enumerated in the provided content.)*

- **Multi-client support**  
  - Single static server URL usable across multiple chat clients.  
  - Per-client authentication handled when adding the MCP server to each app.

- **Configuration guidance**  
  - Step-by-step instructions to:  
    - Copy and use the MCP server URL.  
    - Add the server to various supported chat clients.  
    - Access a central configuration page for more details.

## Technical Details
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`  
- **Integration Host:** Pipedream  
- **Protocol:** Model Context Protocol (MCP) for tool-based interaction with Onfleet.
