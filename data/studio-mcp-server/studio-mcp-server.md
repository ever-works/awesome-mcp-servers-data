# Studio MCP Server

**Brand:** AI21 Labs  
**Category:** AI Integration – MCP Servers  
**Slug:** `studio-mcp-server`

## Description
Studio MCP Server is an MCP-compatible server that exposes AI21 Labs’ Studio APIs, providing access to Jurassic-1 large language models for text generation and text comprehension within any MCP-enabled client or environment.

## Features
- **MCP-compatible server**: Works with MCP-capable chat clients and tools.
- **AI21 Studio API integration**: Connects directly to AI21 Labs’ Studio APIs.
- **Jurassic-1 LLM access**: Uses Jurassic-1 large language models for:
  - Text generation
  - Text comprehension
- **Static server endpoint**:
  - MCP server URL: `https://mcp.pipedream.net/v2`
  - Same URL can be used across different MCP clients.
- **Authentication at client setup**: Authentication is performed when adding the server to your application or chat client.
- **Configurable per client**: Can be added to various chat clients and configured via the associated configuration pages.

## Technical Details
- **Server URL:** `https://mcp.pipedream.net/v2`
- **Protocol:** MCP (Model Context Protocol)
- **Integration host:** Pipedream Connect
