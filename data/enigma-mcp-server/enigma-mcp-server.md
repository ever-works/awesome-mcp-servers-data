# Enigma MCP Server

Enigma MCP Server is an MCP (Model Context Protocol) server that allows MCP-compatible tools and applications to query fresh, accurate data about the identity and financial health of small businesses.

## Key Details
- **Name:** Enigma MCP Server  
- **Category:** Finance / Market Data MCP Servers  
- **Provider / Brand:** Enigma (via Pipedream Connect)  
- **URL (MCP server endpoint):** `https://mcp.pipedream.net/v2`  
- **Use Case Focus:** Small business identity and financial health data; risk and financial analysis scenarios

## Features
- **Fresh small business data**: Access up-to-date information on small businesses.  
- **Identity data**: Query data related to the identity of small businesses (e.g., entity details, identifiers, and related attributes as supported by Enigma).  
- **Financial health data**: Retrieve information reflecting the financial health of small businesses, useful for risk analysis, underwriting, and related evaluations.  
- **MCP-compatible server**: Designed as an MCP server so MCP tools and chat clients can use the data as part of their workflows and contextual reasoning.  
- **Static server URL**: Single, static endpoint (`https://mcp.pipedream.net/v2`) used across clients, simplifying configuration.  
- **Client-agnostic integration**: Can be added to multiple MCP-capable chat clients and applications using the same configuration URL.  
- **Authentication at configuration time**: Authentication is handled when adding the server to your application/client (no per-client URL changes required).  
- **Configuration docs available**: A dedicated configuration page (via Pipedream) explains how to add and use the server in supported chat clients.

## Integration & Setup
