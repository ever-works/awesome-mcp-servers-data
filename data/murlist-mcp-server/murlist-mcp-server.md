# MurList MCP Server

MCP Server that integrates with Murlist for AI-driven lead generation workflows.

## Overview
MurList MCP Server connects your MCP-compatible chat or AI client to Murlist, enabling automated lead generation, scoring, verification, and conversion via MCP tools.

- **Category:** Business & Commerce → MCP Servers  
- **Provider / Brand:** murlist  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **Lead generation automation**  
  - Integrates with Murlist to discover and collect potential leads.  
  - Designed to run on autopilot once configured.

- **Lead scoring**  
  - Uses Murlist’s logic to score leads for quality or priority.  
  - Accessible as MCP tools from within supported chat/AI clients.

- **Lead verification**  
  - Verifies collected lead data (e.g., contact details) through Murlist.  
  - Helps filter out invalid or low-quality leads.

- **Lead conversion support**  
  - Provides tools to help move verified leads further down the funnel.  
  - Can be orchestrated directly from your MCP-enabled application.

- **Static MCP server endpoint**  
  - Uses a single static URL for all clients: `https://mcp.pipedream.net/v2`.  
  - Authentication is handled when adding the server to your application.

- **Works with multiple chat clients**  
  - Can be added to various MCP-compatible chat / AI clients.  
  - Configuration instructions are available via the referenced configuration page in Pipedream.

## Integration & Setup
- Add `https://mcp.pipedream.net/v2` as an MCP server in your supported client.  
- Authenticate during setup within your client to connect to Murlist’s capabilities.  
- Further configuration details are available via Pipedream’s configuration resources.
