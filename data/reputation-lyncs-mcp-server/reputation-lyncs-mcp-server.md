# Reputation Lyncs MCP Server

## Overview
Reputation Lyncs MCP Server connects your Reputation Lyncs account to AI-powered tools through Pipedream’s MCP server, enabling automated workflows for customer review collection and reputation management.

## Features
- Connects a Reputation Lyncs account to compatible AI tools
- Operates via Pipedream’s MCP server infrastructure
- Supports integration with tools such as Cursor, Claude, and Windsurf
- Enables automation of customer review collection processes
- Facilitates reputation management workflows through MCP

## Integrations
- Cursor
- Claude
- Windsurf
- Pipedream MCP server
