# Integration App MCP Server

## Overview
Integration App MCP Server is an open‑source Model Context Protocol (MCP) server that connects Integration App with external SaaS applications. It is designed so agents can perform actions in third‑party SaaS tools on behalf of customers.

- **Project type:** MCP server implementation
- **Source:** [GitHub repository](https://github.com/integration-app/mcp-server)

## Features
- **MCP server for Integration App** – Implements an MCP server that can be used with Integration App.
- **SaaS integration layer** – Enables Integration App agents to interact with other SaaS applications.
- **Act on behalf of customers** – Supports workflows where agents perform operations in customers’ connected SaaS accounts.
