# Async Interview MCP Server

## Overview
Async Interview MCP Server integrates the Async Interview video interviewing platform with MCP-compatible chat clients. It enables interaction with pre-recorded, customizable video interviews for recruitment workflows through a standardized MCP endpoint.

- **Category:** Business & Commerce MCP Servers
- **Use cases:** Recruiting, video interviews, candidate assessment
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server endpoint**
  - Single static MCP URL (`https://mcp.pipedream.net/v2`) usable across all clients.
  - Authentication handled when adding the server within the client/application.

- **Async Interview platform integration**
  - Connects to Async Interview, a video interviewing platform.
  - Supports pre-recorded video interviews.
  - Designed for recruitment scenarios where interviews can be customized to specific roles or processes.

- **Client-agnostic setup**
  - Works with multiple MCP-enabled chat clients.
  - Configuration guided by client-specific setup instructions.
  - Central configuration reference via a dedicated configuration page (for detailed setup steps and options).

- **Pipedream Connect infrastructure**
  - MCP server is hosted and powered by Pipedream Connect.
