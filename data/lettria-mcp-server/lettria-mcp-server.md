# Lettria MCP Server

**Category:** AI Integration – MCP Servers  
**Brand:** lettria  
**Slug:** `lettria-mcp-server`

## Description
Lettria MCP Server is an MCP server integration that connects your Lettria account to AI tools, exposing Lettria’s end-to-end NLP capabilities for turning text data into structured insights.

## Features
- MCP server integration for Lettria
- Connects a Lettria account to AI tools via Pipedream’s MCP server
- Enables use with tools such as:
  - Cursor
  - Claude
  - Windsurf
- Provides end-to-end NLP capabilities
- Supports transforming unstructured text data into structured insights
- Designed for text analysis and NLP workflows
