# Upwave MCP Server

## Overview
Upwave MCP Server exposes Upwave, a visual collaboration platform for projects, portfolios, risk management, and daily tasks, as a Model Context Protocol (MCP) server that can be used from compatible chat or AI clients.

- **Item type:** MCP server / integration
- **Platform:** Upwave (visual project & portfolio collaboration)
- **Category:** Project management MCP servers
- **Tags:** project-management, collaboration, visualization
- **Source URL:** https://mcp.pipedream.com/app/upwave

## Features
- **MCP server endpoint**
  - Static MCP server URL for all supported clients: `https://mcp.pipedream.net/v2`
  - Same URL usable across different MCP-compatible chat or AI applications.

- **Upwave integration**
  - Connects to Upwave, a visual platform for:
    - Project collaboration
    - Portfolio management
    - Risk management
    - Daily task management

- **Client-agnostic setup**
  - Designed to be added to various MCP-compatible chat clients.
  - Authentication occurs when adding the server in the client.

- **Configuration resources**
  - Additional configuration details available via the linked configuration page (for step-by-step setup in different clients).

## Authentication
- Authentication is performed when you add the MCP server (`https://mcp.pipedream.net/v2`) to your chosen client.
