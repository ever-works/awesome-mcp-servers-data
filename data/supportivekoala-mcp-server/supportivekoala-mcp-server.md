## Description
Supportivekoala MCP Server is an MCP (Model Context Protocol) server that connects to the Supportivekoala service, enabling automated generation of marketing images based on input data. It is accessed through a static MCP endpoint and can be integrated with compatible chat or MCP-enabled clients.

## Features
- **Automated marketing image generation**
  - Generates images programmatically based on input data.
  - Designed for marketing and promotional use cases.
- **Model Context Protocol (MCP) integration**
  - Exposes Supportivekoala functionality as an MCP server.
  - Works with MCP-compatible chat clients and tools.
- **Static MCP server URL**
  - Single endpoint for all clients: `https://mcp.pipedream.net/v2`.
  - Authentication is handled when adding the server to the client/application.
- **Client-agnostic setup**
  - Same server URL works across different MCP-enabled chat clients.
  - Configuration guided via per-client instructions and a central configuration page.
- **Template- and data-driven workflows** (inferred from description and tags)
  - Uses input data to populate templates for consistent image output.
  - Suitable for repeated or large-scale image generation workflows.
