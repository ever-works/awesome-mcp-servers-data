# Sendbird AI chabot MCP Server

**Description**

Sendbird AI chabot MCP Server is an MCP (Model Context Protocol) server that connects to Sendbird’s AI chatbot solution, allowing MCP-compatible clients to create and manage custom AI chatbots for websites. It is provided as a static MCP endpoint that can be added to various chat clients.

**Link**

- Source / More info: https://mcp.pipedream.com/app/sendbird_ai_chabot

## Features

- **MCP-compatible server**: Implements a Model Context Protocol server that can be consumed by MCP-compatible clients and tools.
- **Sendbird AI chatbot integration**: Connects directly to Sendbird’s AI chatbot solution to power website chatbots.
- **Static MCP endpoint**: Uses a single static URL that works across all supported clients:  
  - MCP Server URL: `https://mcp.pipedream.net/v2`.
- **Client-agnostic integration**: Designed to work with multiple MCP clients; you add the same server URL and authenticate within each client.
- **Website chatbot creation**: Enables the creation of custom AI chatbots for websites with minimal setup.
- **Centralized configuration**: References a central configuration page (via Pipedream) for setup and additional options.

---

## Configuration / Usage

- **MCP server URL**:  
  - Add `https://mcp.pipedream.net/v2` as an MCP server in your chosen chat client.  
  - Authentication is handled when adding the server to your application.
- **Client setup**:  
  - Select your chat client (from the Pipedream UI) to view specific integration instructions.  
  - Additional configuration details are available on the Pipedream Configuration page.
