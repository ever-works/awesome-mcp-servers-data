# Close CRM MCP Server

**Category:** Business & Commerce · MCP Servers  
**Brand:** Close  
**Website:** https://www.close.com/integrations/klavis

## Overview
The Close CRM MCP Server is an integration built by Klavis AI that exposes Close CRM data and workflows through the Model Context Protocol (MCP). It provides a hosted MCP server that connects AI applications to Close via OAuth 2.1, removing the need to manage authentication or client-side integration code.

## Features
- **Hosted MCP Server for Close CRM**  
  - Provides a managed MCP endpoint for interacting with Close CRM data and workflows.

- **OAuth 2.1 Authentication**  
  - Uses OAuth 2.1 to securely authorize access to Close accounts.

- **MCP-Based Access to CRM Data & Workflows**  
  - Designed to let AI models and agents read and act on Close CRM data (e.g., leads, opportunities, pipelines, activities) via MCP tools.

- **No Client-Side Code Required**  
  - Klavis AI hosts the MCP server, eliminating the need to run your own server or embed complex auth/client logic.

- **Open-Source MCP Integrations**  
  - Built as part of Klavis AI’s open source MCP integrations for AI applications, enabling transparency and extensibility.

- **API for Easy Integration**  
  - A user-friendly API to configure and connect the hosted Close CRM MCP Server into AI apps, agents, and workflows.

- **Compatibility with AI Models & Agents**  
  - Intended for use with any MCP-compatible AI model, assistant, or agent framework.
