# Big Cartel MCP Server

MCP Server for Big Cartel, enabling AI and tools to interact with Big Cartel online store data for artists and makers through the Model Context Protocol.

---

## Overview
- **Name:** Big Cartel MCP Server  
- **Category:** Business & Commerce MCP Servers  
- **Brand:** Big Cartel  
- **URL:** https://mcp.pipedream.com/app/big_cartel  
- **MCP Server Endpoint:** `https://mcp.pipedream.net/v2`  
- **Description:** Allows AI agents and MCP-compatible tools to access and work with data from Big Cartel online stores used by artists and makers.

---

## Features
- **MCP-compatible server** for integrating Big Cartel data into Model Context Protocol clients.  
- **Static MCP endpoint** (`https://mcp.pipedream.net/v2`) usable across supported clients.  
- **Big Cartel store integration** intended for artists and makers operating online stores (e-commerce / digital products).  
- **Authentication at client setup time**, performed when adding the server to an application.  
- **Works with multiple chat / MCP clients**, added as an external server in the client configuration.  
- **Configuration documentation available** via a dedicated configuration page on Pipedream.

## Use Cases
- Connect an AI assistant to a Big Cartel store to query or work with store data.  
- Build workflows and tools around Big Cartel for artists and makers using MCP-compatible clients.

## Integration & Configuration
