# Softr MCP Server

**Category:** Development Tools – MCP Servers  
**Brand:** Softr  
**Source:** https://mcp.pipedream.com/app/softr

Softr MCP Server is a Model Context Protocol (MCP) server that lets agents build and manage custom apps and portals on the Softr no‑code platform using Airtable as a backend data source.

## Features

- MCP server integration for Softr, enabling AI agents and chat clients to interact with Softr apps and portals.
- Designed for building and managing custom web apps and portals backed by Airtable data.
- Supports no‑code workflows via the Softr platform (no traditional programming required).
- Static MCP endpoint URL usable from any compatible client:
  - `https://mcp.pipedream.net/v2`
- Authentication handled when adding the server to your client or application.
- Compatible with multiple chat clients (details via configuration docs).

## Configuration

- Add the Softr MCP Server to your preferred chat client by using the static MCP server URL.
- Additional setup and client-specific instructions are available on the Pipedream MCP Configuration page.
