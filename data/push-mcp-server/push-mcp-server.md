## Features

### MCP Server Capabilities
- Exposes Push by Techulus as an MCP server for use in MCP-compatible clients (e.g., ChatGPT / OpenAI clients).
- Uses a static MCP server URL (`https://mcp.pipedream.net/v2`) that works for all clients, with authentication handled when adding the server to an application.
- Integrates with Pipedream Connect for configuration and account management.

### Notification & Messaging
- Cross-platform push notification delivery.
- Supports both code-based and no-code workflows for sending notifications.
- Designed for real-time, transactional messaging (e.g., alerts, updates, and other event-driven notifications).

### Available Tools / Actions
- **Send Push Notification**  
  Sends a new Push notification to the user.
  - Exposed as an MCP tool/action within supported clients.
  - Documentation available via Pipedream GitHub component:  
    https://github.com/PipedreamHQ/pipedream/blob/master/components/push_by_techulus/actions/send-push-notification/send-push-notification.mjs

### Configuration & Integration
- Connect a Push by Techulus account through Pipedream to manage clients and credentials.
- "Configure Push by Techulus" flow to link accounts and select clients.
- Guides available for adding the server to specific chat clients (e.g., ChatGPT/OpenAI) via the Pipedream configuration pages.
