## Tools
- `search_datasets`: Searches for NASA datasets in CMR using keywords, date ranges, and filters.

## External APIs and technologies
- earthaccess (≥0.14.0): Python library for NASA Earth science data.
- NASA CMR: High-performance metadata system.
- FastMCP: Python SDK for MCP servers.
- Python 3.10+

## Configuration requirements
No explicit API keys; earthaccess may use NASA Earthdata Login.
