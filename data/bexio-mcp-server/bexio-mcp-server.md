# bexio MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** bexio  
**Source:** https://mcp.pipedream.com/app/bexio

## Overview
bexio MCP Server is an MCP server integration for bexio’s business software suite, designed for small businesses, startups, and self‑employed individuals. It allows MCP-compatible clients (such as chat-based or agent applications) to connect to bexio in order to access and automate business management workflows.

## Features
- **MCP server integration for bexio** – Exposes bexio’s business software capabilities to MCP clients via a standardized server interface.
- **Business management workflows** – Enables automation and interaction with bexio’s tools commonly used by small businesses, startups, and self‑employed users (e.g., core business management and accounting workflows, as supported by bexio).
- **Static MCP endpoint** – Uses a shared MCP server URL for connections: `https://mcp.pipedream.net/v2`.
- **Per-client authentication** – Authentication occurs when adding the server to an MCP-compatible application, allowing secure access for each client.
- **Multi-client compatibility** – Can be added to different MCP chat or agent clients; each client can follow its own setup flow.
- **Configuration guidance** – Includes a configuration page (via Pipedream) that walks users through connecting accounts and selecting the correct bexio client/tenant.
- **Pipedream-hosted infrastructure** – Runs on Pipedream’s MCP hosting, leveraging their platform for connectivity, terms of use, and privacy controls.

## Technical Details
- **MCP server URL:** `https://mcp.pipedream.net/v2`
- **Hosting provider:** Pipedream
- **Usage model:** Connect a bexio account, then add the MCP server URL to your MCP-compatible application and authenticate.
