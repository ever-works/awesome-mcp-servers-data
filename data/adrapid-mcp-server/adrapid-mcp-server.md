# Adrapid MCP Server

An MCP server for generating ad creatives from large data sets using MCP-compatible workflows on Pipedream.

## Overview
- **Type:** MCP server (Model Context Protocol)
- **Category:** Business & commerce – MCP servers
- **Provider / Platform:** Pipedream Connect
- **Brand:** Adrapid
- **Primary use case:** Automated ad creative generation from large or complex data sources within MCP-enabled chat or workflow clients.

## Features
- **MCP-compatible server endpoint**
  - Static server URL for all clients: `https://mcp.pipedream.net/v2`.
  - Can be added to any MCP-capable chat or agent client.

- **Ad creative generation from large data sets**
  - Designed to generate advertising creatives from “T-Rex sizes” (very large) data sets.
  - Intended for use in automated workflows and chat-based tools rather than manual UI.

- **Workflow integration via Pipedream**
  - Built to run on Pipedream’s infrastructure.
  - Integrates with Pipedream Connect workflows and automations.
  - Suitable for embedding into broader marketing / advertising automation pipelines.

- **Authentication at client level**
  - Shared static URL for all clients, with authentication handled when adding the server to each application.

## Technical Details
- **Protocol:** Model Context Protocol (MCP)
- **Server URL:** `https://mcp.pipedream.net/v2`
- **Configuration:** Added as an MCP server in compatible chat / agent applications (see platform configuration pages for client-specific steps).
