# Join MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Join  
**Source:** https://mcp.pipedream.com/app/join

## Overview
Join MCP Server exposes Join’s remote-control capabilities for Android and PC devices through an MCP-compatible agent. It enables actions such as SMS handling, notifications, clipboard operations, and other remote interactions via supported chat clients or tools.

## Features
- **Remote Android and PC control via MCP**  
  - Integrates Join’s remote operations with any MCP-compatible client.
- **SMS operations**  
  - Perform SMS-related actions on connected Android devices (e.g., sending or managing SMS via the MCP server).
- **Notifications**  
  - Interact with device notifications exposed by Join.
- **Clipboard actions**  
  - Access or manipulate clipboard content on connected devices.
- **Multi-device support**  
  - Works with Android and PC devices linked to Join.
- **Static MCP server URL**  
  - Single endpoint for all clients: `https://mcp.pipedream.net/v2`.
- **Client-agnostic setup**  
  - Same server URL works across different chat or MCP clients; authentication is handled when adding the server to a specific application.

## Setup
1. Use the MCP server URL: `https://mcp.pipedream.net/v2`.  
2. Add the server in your MCP-compatible app or chat client.  
3. Authenticate when prompted by your client.  
4. Optionally refer to the Pipedream Connect Configuration page for detailed integration steps.
