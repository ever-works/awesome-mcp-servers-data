## Overview

Dynatrace MCP Server integrates with the Dynatrace observability platform to provide real-time monitoring data access through AI assistants. This server enables developers to monitor application problems, security concerns, and performance metrics using natural language queries.

## Features

- **Real-Time Observability**: Access live performance and monitoring data
- **Problem Monitoring**: Query and analyze application problems and incidents
- **Security Insights**: Monitor security concerns and vulnerabilities
- **Metrics Access**: Retrieve and analyze performance metrics
- **Distributed Tracing**: Access distributed trace data for debugging
- **Log Analysis**: Query and analyze application logs
- **AI-Powered Insights**: Davis AI integration for intelligent problem detection

## Key Capabilities

- Query application performance metrics
- Investigate production problems and incidents
- Analyze security vulnerabilities
- Monitor infrastructure health
- Analyze user experience metrics
- Access real user monitoring data
- Query synthetic monitoring results

## Supported Environments

Works with Dynatrace OneAgent monitoring for applications, infrastructure, cloud platforms, and containerized environments.

## Authentication

Requires Dynatrace API token and environment URL for secure access to monitoring data.

## Compatibility

Compatible with Claude, Cursor, Windsurf, and any MCP-enabled AI assistant.
