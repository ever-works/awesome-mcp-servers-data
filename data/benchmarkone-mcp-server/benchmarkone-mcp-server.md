# BenchmarkONE MCP Server

All-in-one marketing automation MCP server integration for BenchmarkONE.

**Website:** https://mcp.pipedream.com/app/benchmarkone  
**Category:** Business & Commerce – MCP Servers  
**Tags:** marketing-automation, campaigns, CRM

---

## Overview

BenchmarkONE MCP Server connects BenchmarkONE’s all‑in‑one marketing automation and CRM platform to MCP-compatible chat or agent applications, enabling workflows around campaigns and contact management.

---

## Features

- **Static MCP server endpoint**  
  - Single shared MCP server URL for all clients:  
    `https://mcp.pipedream.net/v2`  
  - Authentication is handled at the time you add the server to your application.

- **MCP integration for BenchmarkONE**  
  - Integrates BenchmarkONE’s marketing automation and CRM capabilities into MCP workflows.  
  - Designed for campaign and contact management scenarios within MCP-compatible tools.

- **Multi-client compatibility**  
  - Same server URL works across different chat or agent clients that support MCP.

- **Configuration guidance**  
  - Setup instructions available via a "Configuration" page for adding the server to various chat clients.

---

## Setup

1. Use the MCP server URL: `https://mcp.pipedream.net/v2`.  
2. Add this server in your MCP-compatible chat or agent client.  
3. Authenticate within your client when prompted.  
4. (Optional) Refer to the linked Configuration page on the source site for client-specific setup steps.
