# Freshsales MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Freshworks  
**Website:** https://mcp.pipedream.com/app/freshsales

## Description
Freshsales MCP Server connects a Freshsales CRM account to Model Context Protocol (MCP)–enabled AI tools, enabling AI-driven interaction with customer and sales data.

## Features
- Connects a Freshsales CRM account via Pipedream’s MCP server
- Enables AI tools to interact with Freshsales for customer and sales cycle management
- Works with MCP-compatible tools such as:
  - Cursor
  - Claude
  - Windsurf
- Provides a standardized MCP interface for accessing Freshsales data and actions
