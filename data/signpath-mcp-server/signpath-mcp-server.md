# SignPath MCP Server

A Pipedream-hosted MCP server that exposes SignPath code-signing capabilities to MCP-compatible environments.

- **Category:** Security & Attestation MCP Servers  
- **Brand:** SignPath  
- **Source URL:** https://mcp.pipedream.com/app/signpath

## Description
SignPath MCP Server provides simple and secure code-signing capabilities through the Model Context Protocol (MCP). It is delivered as a Pipedream MCP server endpoint that can be added to any compatible chat client or MCP-enabled application.

## Features
- **MCP-compatible code signing** – Exposes SignPath code-signing operations through a standard MCP server for use in compatible tools and environments.
- **Static MCP server URL** – Single endpoint for all clients:
  - `https://mcp.pipedream.net/v2`
- **Client-agnostic integration** – Designed to work with any MCP-compatible chat client or application.
- **Authentication at configuration time** – Users authenticate when adding the server to their application, rather than using client-specific URLs.
- **Hosted by Pipedream** – Operated via Pipedream Connect infrastructure, removing the need to self-host the MCP server.
- **Configuration guidance** – Documentation available via a central configuration page for adding the server to different clients.

## Usage
- Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
- Add it to your MCP-compatible chat client or application.
- Authenticate when prompted during server setup.
- Refer to the configuration page (linked from the source URL) for client-specific setup steps.
