# Proxy Spider MCP Server

## Overview
Proxy Spider MCP Server is an MCP-compatible proxy service that provides rotating proxies with both random-per-request and sticky-port options, designed to integrate with MCP-capable applications via a single static server URL.

- **Category:** Cloud & DevOps – MCP Servers
- **Brand:** proxy-spider
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **Rotating proxies** for outbound requests
- **Random proxy assignment per request** for higher IP diversity
- **Sticky ports** to maintain the same exit IP across multiple requests when needed
- **Unlimited bandwidth** usage for proxy traffic
- **MCP-compatible endpoint** usable by any MCP client
- **Single static URL for all clients** (`https://mcp.pipedream.net/v2`), simplifying configuration
- **Authentication at add-time**: clients authenticate when adding the server to their application

## Integration & Usage
- Add the MCP server to any compatible chat or MCP client using the static URL
- Configuration and client-specific setup are documented on the provider’s configuration page (external to this summary)
