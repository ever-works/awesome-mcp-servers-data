# Urlbox.io MCP Server

MCP Server for Urlbox.io, providing screenshot capture and website rendering capabilities through the MCP interface.

## Overview
- **Type:** MCP server (integration via Pipedream Connect)
- **Purpose:** Reliable website screenshot capture and rendering
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- Website screenshot capture via Urlbox.io through the MCP interface
- Website rendering capabilities exposed as MCP tools
- Static MCP server URL usable from any compatible MCP client
- Authentication handled when adding the server to your application
- Designed as a dependable screenshot API replacement (focus on reliability and consistency)

## Configuration
- Add the MCP server using the static URL: `https://mcp.pipedream.net/v2`
- Configure authentication within your MCP-compatible client or application
- Additional configuration details are available on the Pipedream MCP configuration page (external to this summary)
