# Auth0 Management API MCP Server

**Category:** Security & Attestation MCP Servers  
**Brand:** Auth0  
**Source:** https://mcp.pipedream.com/app/auth0_management_api

## Overview
The Auth0 Management API MCP Server exposes Auth0’s Management API over the Model Context Protocol (MCP), enabling identity and access management tasks (such as working with users, roles, and clients) from compatible MCP-enabled applications.

## Features
- MCP server interface to Auth0’s Management API
- Supports identity and access management operations (e.g., users, roles, clients)
- Static MCP server URL usable across all compatible clients:
  - `https://mcp.pipedream.net/v2`
- Authenticates when you add the server to your MCP-enabled application
- Can be integrated into multiple chat or MCP clients
- Configuration guidance available via the Pipedream Configuration page

## Integration & Usage
- Add the MCP server to any supported MCP client using the static URL
- Authenticate during setup within your client or application
- Use MCP tooling to invoke Auth0 Management API operations programmatically from chat or other interfaces
