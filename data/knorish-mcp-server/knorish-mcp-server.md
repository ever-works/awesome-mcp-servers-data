# Knorish MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Knorish  
**Source URL:** https://mcp.pipedream.com/app/knorish  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Description
The Knorish MCP Server connects applications to Knorish, an all‑in‑one cloud platform for creating, selling, and marketing online courses, memberships, and digital downloads.

## Features
- **Standard MCP endpoint**: Uses a static MCP server URL (`https://mcp.pipedream.net/v2`) that works across compatible clients.
- **Client‑side authentication**: Authentication is handled when adding the server to your chat client or application.
- **Pipedream Connect integration**: Server is provided and operated via Pipedream’s Connect infrastructure.
- **Multi‑client support**: Designed to be added to various chat clients that support MCP servers.
- **Central configuration docs**: Full setup and configuration instructions available through a dedicated configuration page on Pipedream.

## Supported Use Cases
- Integrating Knorish capabilities into MCP‑compatible chat or agent clients.  
- Accessing Knorish resources related to:
  - Online courses
  - Memberships
  - Digital products / downloads
