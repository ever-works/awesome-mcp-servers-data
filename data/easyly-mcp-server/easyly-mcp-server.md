# Easyly MCP Server

Easyly MCP Server provides machine-controllable access to Easyly’s cloud-based CRM for sales, marketing, and workflow automation.

## Overview
- **Type:** MCP Server (Machine-Controlled Provider)
- **Category:** Business & Commerce MCP Servers
- **Platform:** Easyly cloud-based CRM via Pipedream Connect
- **Endpoint URL:** `https://mcp.pipedream.net/v2`

## Features
- Access to Easyly’s cloud-based CRM environment
- Supports sales-related CRM operations
- Supports marketing-related CRM operations
- Enables automation of everyday business tasks
- Customizable workflows for sales, marketing, and general task management
- Standard MCP static server URL usable across compatible clients
- Authentication handled when adding the server in the client/application

## Integration & Usage
- Use the static MCP server URL: `https://mcp.pipedream.net/v2`
- Add the server inside any MCP-compatible chat client or application
- Configuration details are available on the associated configuration page (via Pipedream Connect)
