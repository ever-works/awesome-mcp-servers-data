# Favro MCP Server

An MCP Server integration for Favro, enabling MCP-compatible tools (such as AI chat clients) to interact with Favro’s agile planning and collaboration features.

---

## Basic Information
- **Name:** Favro MCP Server  
- **Category:** Project Management MCP Servers  
- **Brand:** Favro  
- **Source URL:** https://mcp.pipedream.com/app/favro

---

## Features
- **MCP-compatible endpoint**  
  - Provides a single static MCP server URL that works across supported clients:  
    - `https://mcp.pipedream.net/v2`
  - Designed to be added as a remote MCP server in compatible chat or agent applications.

- **Favro integration**  
  - Connects MCP-based tools to Favro’s agile planning and collaboration environment (e.g., boards, tasks, and team workflows).  
  - Intended for use by distributed teams and leaders who manage work in Favro.

- **Client-agnostic server setup**  
  - Same MCP server URL for every client; authentication is handled when adding the server in the client / application.  
  - Documentation for adding the server is organized by chat client.

- **Configuration documentation**  
  - A dedicated configuration page (linked from the app page) explains how to set up and use the MCP server with different clients.

---

## Authentication
- Authentication occurs when you add the MCP server to your application; the URL itself is static and does not embed credentials.

---

## Usage
1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.  
2. Add it to your MCP-compatible chat client or application.  
3. Follow the client-specific setup instructions or the general configuration documentation.
