# smartlead-mcp-server-local

**Category:** Business & Commerce MCP Servers  
**Tags:** sales, marketing-automation, workflow-automation  
**Source:** https://github.com/jean-technologies/smartlead-mcp-server-local

## Description
A local Model Context Protocol (MCP) server for Smartlead that allows LLMs and automation platforms (including n8n) to interact with the Smartlead platform for communication and workflow automation use cases.

## Features
- Local MCP server implementation for Smartlead
- Designed for quick download and deployment
- Compatible with MCP-enabled LLM clients
- Integration support for automation platforms such as n8n
- Enables programmatic interaction with Smartlead for sales and marketing workflows
- Supports communication and workflow automation scenarios (e.g., sales outreach, marketing sequences)
- Example environment configuration via `.env.example`
- Example MCP client configuration via `mcp_settings_example.json`
- Dockerfile included for containerized deployment
- TypeScript-based source code (with `tsconfig.json`)
- Jest configuration present for testing
- Developer onboarding and setup documentation (`DEVELOPER_ONBOARDING.md`)
- LLM installation and integration notes (`llms-install.md`)
