# Customer.guru MCP Server

## Overview
Customer.guru MCP Server exposes Customer.guru’s Net Promoter Score (NPS) and customer satisfaction capabilities to MCP‑aware agents and applications. It allows tools and chat clients that support the Model Context Protocol (MCP) to connect to Customer.guru and work with customer feedback data.

The server is hosted via Pipedream Connect and is accessed through a static MCP endpoint URL.

## Features
- **MCP-compatible server**: Implements an MCP server interface so MCP-aware agents and chat clients can connect to Customer.guru.
- **Static server endpoint**: Single MCP server URL usable across clients:
  - `https://mcp.pipedream.net/v2`
- **Centralized configuration**: One endpoint for all clients, with authentication handled when adding the server to each application.
- **NPS-focused integration**: Designed to work with Customer.guru’s Net Promoter Score and customer satisfaction tracking features.
- **Customer feedback access**: Enables agents to track and work with NPS and customer satisfaction data sourced from Customer.guru.
- **Chat client integration**: Can be added to supported chat clients so that agents can query and interact with Customer.guru data from within conversations.
- **Configuration documentation**: Full configuration instructions available via a dedicated configuration page (linked from the product page).

## Technical Details

- **Hosting platform:** Pipedream Connect
