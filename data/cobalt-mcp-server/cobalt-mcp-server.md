# Cobalt MCP Server

An MCP server for Cobalt that exposes fund data and related investment information to MCP‑compatible tools and chat clients.

## Overview
- **Name:** Cobalt MCP Server  
- **Category:** Finance / Market Data MCP Servers  
- **Use Case:** Access and work with fund and investment data through MCP-enabled applications.
- **Provider / Platform:** Runs on Pipedream (Pipedream Connect)
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- Provides MCP-compatible access to Cobalt fund data.
- Designed for use with multiple chat clients and MCP tools via a single static URL.
- Authentication handled when adding the server into the client or application.
- Centralized configuration and setup details available via a dedicated configuration page.

## Integration & Usage
- Use the static MCP server URL: `https://mcp.pipedream.net/v2` in any supported MCP client.
- Configure authentication within your chosen chat client or MCP-compatible tool.
- Additional setup instructions are available on the platform’s configuration page.
