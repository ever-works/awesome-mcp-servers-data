## Description
GoCanvas MCP Server is a Model Context Protocol server that connects AI tools with the GoCanvas mobile forms platform. It enables AI-assisted creation and management of mobile data collection forms using a simple, drag-and-drop interface.

## Features
- **Model Context Protocol server** for integrating GoCanvas with MCP-compatible AI/chat clients.
- **Mobile forms integration** with the GoCanvas platform for data collection on mobile devices.
- **Drag-and-drop form builder** to design and modify mobile forms without coding.
- **Static MCP server URL** (`https://mcp.pipedream.net/v2`) that works across supported clients.
- **Per-client authentication** performed when adding the MCP server to an application.
- **Client-agnostic setup**: can be added to multiple chat/AI clients that support MCP.
- **Configuration guidance available** via a dedicated configuration page on Pipedream.
