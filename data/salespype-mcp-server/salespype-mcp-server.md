# Salespype MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Salespype  
**Slug:** `salespype-mcp-server`

Salespype MCP Server is an MCP-compatible backend (powered by Pipedream Connect) for automating Salespype lead qualification and conversion workflows via a standardized MCP endpoint.

---

## Features

- **MCP-based automation**
  - Exposes Salespype functionality through a Model Context Protocol (MCP) server.
  - Designed to automate lead qualification and conversion workflows.

- **Static MCP server endpoint**
  - Single, static server URL for all clients:  
    `https://mcp.pipedream.net/v2`
  - URL is reusable across different chat or MCP-compatible clients.

- **Per-client authentication**
  - Authentication occurs when adding the server to each client or application.
  - Allows shared infrastructure while keeping user- or app-specific credentials isolated.

- **Client-agnostic integration**
  - Can be added to multiple chat clients or MCP-enabled apps.
  - Configuration instructions are provided per client via the associated configuration page.

- **Pipedream Connect integration**
  - Operates on top of Pipedream Connect infrastructure.
  - Benefits from Pipedreamâ€™s integration and workflow capabilities behind the MCP server.

---

## Usage

- Use the MCP server URL `https://mcp.pipedream.net/v2` when configuring your MCP-capable client or application.
- Authenticate within your chosen client during setup.
- Refer to the configuration page (on the source site) for client-specific setup steps.
