# Lessonspace MCP Server

**Category:** Documentation & Learning Resources  
**Tags:** education, collaboration, online-learning

An MCP (Model Context Protocol) server integration for Lessonspace that exposes Lessonspace’s online teaching and collaborative classroom capabilities as tools accessible from compatible chat or AI clients, via Pipedream.

---

## Features

- **MCP Server Endpoint**  
  - Provides a static MCP server URL usable across different clients:  
    - `https://mcp.pipedream.net/v2`
  - Central entry point for accessing Lessonspace-related tools and actions via MCP.

- **Client-Agnostic Integration**  
  - Designed to work with multiple MCP-compatible chat or AI applications.  
  - Same server URL works for every supported client; authentication occurs when adding the server to the application.

- **Lessonspace Account Connection**  
  - Flow for connecting a Lessonspace account through Pipedream.  
  - Once connected, users can select the appropriate client and begin using Lessonspace tools.

- **Configurable via Pipedream**  
  - Configuration and management handled through Pipedream’s "Configure Lessonspace" interface.  
  - Additional setup guidance available on a dedicated configuration page (referenced as the “full Configuration page”).

- **Tooling Exposure (MCP Tools)**  
  - The server is designed to load and expose "available tools" and "actions" related to Lessonspace (exact tools are dynamically loaded and not listed explicitly in the provided content).  
  - Intended to give MCP clients access to Lessonspace’s online teaching and collaborative classroom features via structured actions.
