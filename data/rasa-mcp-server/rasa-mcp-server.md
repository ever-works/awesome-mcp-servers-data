# Rasa MCP Server

An MCP server that connects to Rasa’s email list and marketing capabilities, enabling MCP-based tools to orchestrate and analyze email campaigns via Rasa.

- **Category:** Business & Commerce – MCP Servers  
- **Brand:** Rasa  
- **Source URL:** https://mcp.pipedream.com/app/rasa  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features

- **MCP-based email orchestration**  
  - Exposes Rasa’s email list and marketing capabilities through the MCP protocol.  
  - Allows MCP-compatible clients and tools to coordinate and run email-related workflows.

- **Email campaign analysis**  
  - Designed to let tools analyze email campaigns that use Rasa’s marketing stack (e.g., performance, audience behavior, or campaign structure) via MCP.

- **Static MCP endpoint**  
  - Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) for all supported clients.  
  - Authentication is handled when adding the server to the client or application, not by changing the URL.

- **Client-agnostic integration**  
  - Can be added to different MCP-compatible chat or agent clients.  
  - Users select their client and follow client-specific instructions to connect to the server.

- **Configuration documentation**  
  - Full configuration and setup guidance is available via the linked configuration page (through the Pipedream Connect platform).
