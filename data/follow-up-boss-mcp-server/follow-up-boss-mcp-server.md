# Follow Up Boss MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Follow Up Boss  
**Slug:** `follow-up-boss-mcp-server`

MCP server that connects Follow Up Boss CRM with AI tools, enabling programmatic access to real estate lead management and follow‑up workflows.

## Features

- Connects a Follow Up Boss CRM account to AI tools via Pipedream’s MCP server
- Supports integration with tools such as:
  - Cursor
  - Claude
  - Windsurf
- Enables programmatic interaction with Follow Up Boss data and workflows (e.g., real estate lead management and follow‑up processes)

## Use Cases

- Automating real estate lead management from AI coding/editing environments
- Triggering and managing follow‑up workflows in Follow Up Boss from AI assistants
- Building custom AI-powered workflows around Follow Up Boss CRM data
