# Dromo MCP Server

**Category:** Data Access & Integration – MCP Servers  
**Brand:** Dromo  
**Source:** https://mcp.pipedream.com/app/dromo

## Overview
Dromo MCP Server provides self-service data file import capabilities (CSV, XLS, XLSX, and other tabular formats) through MCP-enabled agents and applications. It can be embedded directly into applications or triggered via API to handle file-based data ingestion.

## Features
- **Self-service data file importing** for end users.  
- **Supported file formats:** CSV, XLS, XLSX, and other common data file types.  
- **Embeddable in applications:** Integrate the importer directly into your app’s interface.  
- **API-triggered workflows:** Invoke imports programmatically via API.  
- **MCP-compatible server:** Exposes functionality to MCP-enabled agents and apps.  
- **Single static MCP server URL:**
  - Endpoint: `https://mcp.pipedream.net/v2`  
  - Same URL works for all clients; authentication is handled when adding the server to an application.  

## Integration & Configuration
- Add the MCP server to any supported chat or MCP client by using the static server URL.  
- Additional setup and configuration details are available via the platform’s configuration page (e.g., authentication and client-specific instructions).
