# Azure Speech Service MCP Server

**Category:** AI Integration – MCP Servers  
**Brand:** Microsoft Azure

## Description
Azure Speech Service MCP Server is a managed Model Context Protocol (MCP) server that exposes Azure Speech Service capabilities—speech-to-text, text-to-speech, speech translation, and speaker recognition—to MCP-compatible applications via a static server endpoint.

## Features
- **Managed MCP server**
  - Hosted MCP endpoint provided as a static URL: `https://mcp.pipedream.net/v2`
  - Same URL usable across different MCP-compatible clients
  - Authentication handled when adding the server in the client/application configuration

- **Speech-to-text**
  - Convert spoken audio into text using Azure Speech Service’s recognition capabilities.

- **Text-to-speech**
  - Convert text into spoken audio using Azure Speech Service.

- **Speech translation**
  - Translate spoken language into other languages via Azure’s speech translation features.

- **Speaker recognition**
  - Identify or verify speakers from audio using Azure’s speaker recognition capabilities.

- **MCP integration**
  - Designed specifically for use with the Model Context Protocol (MCP) so chat clients and tools can call Azure Speech functionality as part of their workflows.

## Configuration
- Static MCP server URL: `https://mcp.pipedream.net/v2`
- Requires authentication when adding the server to your MCP-compatible chat client or application.
- Additional setup and client-specific instructions are available on the provider’s configuration page (not included in this summary).
