# Workstack MCP Server

**Category:** Project Management MCP Servers  
**Brand:** Pipedream  
**Source:** https://mcp.pipedream.com/app/workstack

Workstack MCP Server connects MCP‑compatible agents to Workstack’s online project management system, enabling them to manage projects and team schedules through Workstack’s team calendar.

---

## Features

- MCP-compatible server for Workstack
- Uses a single static MCP server URL for all clients: `https://mcp.pipedream.net/v2`
- Supports authentication when adding the server in your MCP-compatible application
- Designed for online project management workflows
- Enables team scheduling and calendar-based task management via Workstack
- Allows agents to manage projects and stay on top of task scheduling
- Can be added to supported chat or MCP clients via standard MCP configuration

---

## Technical Details

- **MCP Endpoint:** `https://mcp.pipedream.net/v2`
- **Integration type:** MCP server powered by Pipedream Connect
- **Client support:** Works with any MCP-compatible agent or chat client that supports adding external MCP servers
