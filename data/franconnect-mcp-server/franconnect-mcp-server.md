# FranConnect MCP Server

FranConnect MCP Server is an MCP (Model Context Protocol) integration that exposes FranConnect’s enterprise franchise and multi-location business management capabilities to MCP-compatible tools and chat clients.

## Overview
- **Type:** MCP server integration
- **Platform:** Powered by Pipedream Connect
- **Purpose:** Enable MCP-based applications (e.g., chat clients) to connect to FranConnect’s franchise and multi-location business management system.

## Features
- **MCP Server Endpoint**
  - Static MCP server URL: `https://mcp.pipedream.net/v2`
  - Single, shared endpoint for all clients; authentication is handled when adding the server to an application.

- **MCP Integration for FranConnect**
  - Provides an MCP server interface to FranConnect, an enterprise software platform for:
    - Franchise businesses
    - Multi-location businesses
  - Intended to make FranConnect’s business management capabilities accessible inside MCP-enabled tools (such as chat-based interfaces).

- **Client Setup & Configuration**
  - Can be added to various chat clients that support MCP.
  - Additional configuration instructions available via a dedicated configuration page (linked from the integration page).

- **Pipedream Connect Infrastructure**
  - Integration is hosted and powered by Pipedream Connect.
  - Operates under Pipedream’s platform terms and privacy policies.

## Authentication
- Authentication is performed when adding the FranConnect MCP server to your application.
- Uses the static MCP base URL; credentials or auth details are configured at the client/application level (specific auth method not detailed in the provided content).
