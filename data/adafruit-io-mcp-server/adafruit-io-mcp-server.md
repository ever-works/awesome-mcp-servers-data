## Overview
The Adafruit IO MCP Server is an MCP server that connects to the Adafruit IO platform, allowing MCP-based tools and agents to display, respond to, and interact with IoT project data hosted on Adafruit IO.

## Features
- Connects MCP-compatible clients, tools, and agents to Adafruit IO.
- Enables display of Internet-of-Things (IoT) data stored on Adafruit IO.
- Supports responding to and interacting with live IoT project data.
- Uses a single static MCP server URL for all compatible clients: `https://mcp.pipedream.net/v2`.
- Authentication handled when adding the server to your application.
- Accessible setup documentation via a configuration page (per-client setup guidance).

## Integration & Usage
- **MCP Server URL:** `https://mcp.pipedream.net/v2` (shared across clients).
- Add this MCP server URL within your chat client or MCP-compatible application and complete authentication to connect to Adafruit IO.
