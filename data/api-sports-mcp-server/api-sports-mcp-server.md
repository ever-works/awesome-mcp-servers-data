## Overview
API Sports MCP Server is an MCP-compatible service that connects to the API Sports platform, providing structured access to multi-sport statistics and live sports data through a unified MCP interface.

Supported sports (per API Sports): football, basketball, baseball, hockey, rugby, Formula 1, handball, and volleyball.

## MCP Server Endpoint
- **MCP server URL (static):** `https://mcp.pipedream.net/v2`  
  - This URL is the same for all compatible clients.
  - Authentication is performed when adding the server to your application.

## Features
- **Multi-sport data access** via API Sports
  - Football
  - Basketball
  - Baseball
  - Hockey
  - Rugby
  - Formula 1
  - Handball
  - Volleyball
- **RESTful sports data APIs** exposed through MCP
  - Access structured statistics and live sports data.
- **MCP integration**
  - Compatible with MCP-enabled chat and agent clients.
  - Single static base URL for all clients.
- **Configuration support**
  - Setup and usage instructions available via the Pipedream MCP Configuration page.

## Usage
1. Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add the server in your MCP-compatible client.
3. Configure authentication when prompted by your client.
