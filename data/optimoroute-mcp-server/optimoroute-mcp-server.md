# OptimoRoute MCP Server

**Description**

OptimoRoute MCP Server is an MCP-compatible integration that exposes OptimoRoute’s route planning and optimization capabilities for delivery and field service businesses. It allows MCP clients (such as chat-based tools) to connect to an OptimoRoute account and perform route, schedule, and order management operations programmatically.

**Category:** Business & Commerce – MCP Servers  
**Brand:** OptimoRoute  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

---

## Features

- **MCP-compatible server**  
  - Static MCP server URL (`https://mcp.pipedream.net/v2`) usable across clients.  
  - Authentication handled when adding the server to compatible applications.

- **Account connection & configuration**  
  - Connect an OptimoRoute account through Pipedream.  
  - Configure the integration and select the desired client to start using tools.

- **Route planning & optimization for delivery and field service**  
  - Access OptimoRoute’s core capabilities for planning and optimizing routes and schedules for delivery and field service operations.

- **Available tools (actions)**  
  1. **Update Order Status**  
     - Update the status of an existing order in OptimoRoute.  
     - Useful for reflecting progress (e.g., in-progress, completed, canceled) from within MCP-compatible clients.

  2. **Start Planning**  
     - Initiate route and schedule planning for given orders.  
     - Triggers OptimoRoute’s planning engine from within your MCP client.

  3. **Search Orders**  
     - Search for orders stored in OptimoRoute.  
     - Enables querying orders by specified criteria from chat or other MCP-enabled interfaces.

  4. **Get Routes**  
     - Retrieve routes generated by OptimoRoute.  
     - Access route information for drivers, deliveries, or service visits.

  5. **Get Mobile Events**  
     - Fetch mobile events recorded by OptimoRoute’s mobile app (e.g., driver actions or on-the-road events).  
     - Useful for monitoring and reporting on field activity.

  6. **Create Order**  
     - Create new orders in OptimoRoute directly from MCP-compatible clients.  
     - Supports adding jobs/deliveries to be included in future route planning.

- **Client-agnostic setup**  
  - Works with any MCP-compatible chat or automation client that supports adding custom MCP servers.
