## Overview
Raisely MCP Server is a Model Context Protocol (MCP) server that connects AI/chat clients to Raisely, enabling interaction with online fundraising campaigns and donation management via MCP.

- **Product type:** MCP server integration for Raisely
- **Purpose:** Access and manage Raisely fundraising campaigns and donations from compatible MCP-enabled chat applications.
- **Base URL:** `https://mcp.pipedream.net/v2` (static URL used for all clients; authentication is handled when adding the server to the application).

## Features
- MCP server endpoint for Raisely: `https://mcp.pipedream.net/v2`
- Works with multiple MCP-compatible chat clients (added as a server in the client configuration)
- Supports authentication at the time you add the server to your application
- Hosted and powered by Pipedream Connect
- Central configuration documentation available via the Pipedream **Configuration** page

> Note: The source content does not list specific Raisely operations or tools exposed via MCP (e.g., create campaign, list donations). It only states that the server enables access to online fundraising campaigns and donation management via MCP.

## Configuration
- Add `https://mcp.pipedream.net/v2` as an MCP server in your chat client.
- Follow client-specific instructions on the Pipedream Configuration page for:
  - Supplying authentication/credentials
  - Enabling the server for your chosen chat client
