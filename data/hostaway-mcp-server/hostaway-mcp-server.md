# Hostaway MCP Server

An MCP Server for Hostaway that lets MCP agents interact with vacation rental management data, bookings, and channels via a unified server interface.

- **Category:** Business & Commerce – MCP Servers  
- **Brand:** Hostaway  
- **Tags:** Hospitality, Reservations, Property Management

## Features

- Unified MCP server interface to Hostaway’s vacation rental management platform.
- Supports interaction with:
  - Vacation rental management data.
  - Bookings and reservations.
  - Distribution channels (e.g., OTAs and connected booking channels) exposed via Hostaway.
- Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`.
- Authentication handled when adding the server to your MCP-compatible application.
- Designed to integrate with MCP-enabled chat or agent clients via configuration.

## Usage

- Add `https://mcp.pipedream.net/v2` as an MCP server in your chat or agent client.
- Authenticate within your application when prompted to connect to Hostaway data and operations.
