# PingOne MCP Server

**Category:** Security & Attestation MCP Servers  
**Brand:** Ping Identity

An MCP server integration that exposes Ping Identity’s PingOne single sign-on (SSO) capabilities to MCP-compatible clients and workflows via a static MCP endpoint.

---

## Features

- **PingOne SSO integration**
  - Provides access to Ping Identity’s PingOne single sign-on (SSO) services from MCP-compatible clients and workflows.
- **Static MCP server URL**
  - Uses a single, static endpoint for all clients:  
    `https://mcp.pipedream.net/v2`
  - Authentication occurs when adding the server to your MCP-compatible application.
- **Client-agnostic usage**
  - Designed to work with any MCP-compatible chat client or application.
- **Configuration guidance**
  - Supports setup through client-specific instructions and a general configuration page (via the platform hosting the MCP server).

---

## Usage

- Add the MCP server URL `https://mcp.pipedream.net/v2` to your MCP-compatible client or application.
- Complete authentication as prompted by the client when registering the server.
