# AddEvent MCP Server

**Description**  
MCP Server for AddEvent that connects your AddEvent account to AI tools and workflows, enabling “add to calendar” event scheduling and calendar services within MCP-compatible environments.

**Brand**  
- Name: AddEvent

**Category**  
- Business & Commerce MCP Servers

**Tags**  
- Calendar  
- Events  
- Scheduling

## Features
- Integrates AddEvent’s “add to calendar” functionality into MCP workflows.
- Connects your AddEvent account via Pipedream’s MCP server.
- Enables AI tools (such as Cursor, Claude, and Windsurf) to interact with your AddEvent data and services.
- Supports event scheduling use cases through MCP-compatible tools.
