## Features

- **MCP Server Integration**
  - Provides a static MCP server endpoint (`https://mcp.pipedream.net/v2`) usable by any compatible MCP client.
  - Authentication handled when adding the server within the client/application.

- **Mailsoftly Connectivity**
  - Connects MCP tools to Mailsoftly, an email marketing and automation platform.
  - Allows MCP-based workflows to trigger and manage Mailsoftly email marketing campaigns and automations.

- **Client-Agnostic Setup**
  - Single, static MCP URL works across different chat or MCP-enabled clients.
  - Can be added to multiple applications that support MCP server configuration.

## Usage

1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add the server to your MCP-compatible app or chat client.
3. Authenticate within your client when prompted.
