# ToneDen MCP Server

A Model Context Protocol (MCP) server that connects to ToneDen’s social marketing platform, enabling programmatic access to manage and automate social marketing campaigns from compatible clients.

## Overview
- **Type:** MCP server (backend API bridge)
- **Platform:** ToneDen social marketing platform
- **Use Case:** Social media and marketing campaign automation via MCP-compatible chat or AI clients
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- **MCP Integration**
  - Provides a static MCP server URL (`https://mcp.pipedream.net/v2`) usable across different MCP-compatible clients.
  - Authentication handled when adding the server to an application/client.

- **ToneDen Platform Access**
  - Connects to ToneDen’s social marketing platform for campaign-related operations (as described: manage and automate social marketing campaigns).
  - Designed to support social marketing workflows such as campaign management and automation from within MCP-aware tools.

- **Client-Agnostic Setup**
  - Works with multiple chat or MCP client applications.
  - Central configuration and setup instructions available via the linked configuration page (client-specific setup, not detailed in the provided content).

- **Pipedream Connect Integration**
  - Hosted and powered by Pipedream Connect infrastructure.
  - Uses Pipedream’s standardized MCP deployment model for consistent integration behavior.

## Configuration
- **Server URL:** `https://mcp.pipedream.net/v2`
- **Setup Flow:**
  - Add the above MCP server URL in a supported MCP client.
  - Authenticate during the add/connect step within the client.
  - Optional detailed instructions are available on the external configuration page (not included in the provided content).
