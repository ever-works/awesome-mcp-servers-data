## Overview

**Airtable MCP Server (felores)** is an open-source Model Context Protocol (MCP) server that exposes Airtable’s API to MCP-compatible clients (such as Claude Desktop). It enables large language models to manage Airtable bases, tables, fields, and records as part of database-driven workflows.

- **Type:** MCP server / developer tool
- **Ecosystem:** Airtable, Model Context Protocol (MCP), Claude Desktop and other MCP clients
- **Source:** [GitHub – felores/airtable-mcp](https://github.com/felores/airtable-mcp)

## Features

- **Airtable API integration**  
  - Connects directly to Airtable’s API via an MCP server implementation.  
  - Exposes Airtable data and operations to LLMs through MCP tools.

- **Resource management for Airtable**  
  - Work with **bases**, **tables**, **fields**, and **records**.  
  - Supports **searching** Airtable data.  
  - Supports **creating** new resources (e.g., records).  
  - Supports **updating** existing resources.

- **MCP / LLM workflow support**  
  - Designed for use with **Claude Desktop** and other MCP-compatible clients.  
  - Enables LLM-driven, programmatic workflows over Airtable data (e.g., database-backed assistants, automations, and integrations).
