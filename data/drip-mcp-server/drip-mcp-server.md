# Drip MCP Server

**Category:** Business & Commerce – MCP Servers  
**Use case:** Connect MCP-based applications to Drip’s marketing automation and e‑commerce CRM capabilities.

---

## Overview
Drip MCP Server is an MCP-compatible integration endpoint (powered by Pipedream Connect) that allows MCP-based clients and applications to interact with Drip. It enables you to incorporate Drip’s marketing automation and e‑commerce CRM features into chat clients or other MCP-enabled tools.

---

## MCP Server URL
- Base MCP server URL (static for all clients):
  - `https://mcp.pipedream.net/v2`
- Authentication is performed when adding the server to your application.

---

## Features
- **MCP-compatible server endpoint** that can be used by any MCP-capable client.
- **Static, shared URL** (`https://mcp.pipedream.net/v2`) for simplified configuration across clients.
- **Authentication at client setup time**, allowing secure access per application or user.
- **Integration with Drip** to leverage:
  - Marketing automation capabilities.
  - E‑commerce CRM functions.
- **Client-agnostic setup guidance** via instructions for different chat clients and a full configuration page.
- **Backed by Pipedream Connect**, providing the underlying infrastructure for the MCP server.

---

## Configuration & Setup
- Add the MCP server URL to your MCP-compatible chat client or application.
- Authenticate within your client when prompted.
- Optional: refer to the full configuration documentation on the provider’s Configuration page for client-specific setup steps.
