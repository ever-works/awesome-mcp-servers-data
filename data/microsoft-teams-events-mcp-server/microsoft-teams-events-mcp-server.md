# Microsoft Teams Events MCP Server

Microsoft Teams Events MCP Server integrates Microsoft Teams’ events and webinars features into MCP, enabling automation of event creation, management, and engagement workflows.

- **Category:** Workflow Automation MCP Servers
- **Brand:** Microsoft
- **Source URL:** https://mcp.pipedream.com/app/microsoft_teams_events
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features

- Integration of Microsoft Teams events and webinars into MCP-based workflows.
- Automation of event-related processes such as:
  - Event creation.
  - Event management.
  - Engagement workflows around events and webinars.
- Central MCP server endpoint (`https://mcp.pipedream.net/v2`) usable across compatible MCP clients.
- Client-agnostic setup: same server URL for every client, with authentication handled when adding the server to an application.

## Setup

- Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server to your MCP-compatible chat client or application.
- Follow client-specific instructions or refer to the platform’s configuration documentation.
