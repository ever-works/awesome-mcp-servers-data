## Description

yutu is an open‑source, fully featured Model Context Protocol (MCP) server and command‑line interface for YouTube. It automates YouTube‑related operations such as searching, fetching, and managing video data for use in LLM workflows.

## Features

- **MCP server for YouTube**
  - Exposes YouTube operations via the Model Context Protocol for integration with LLM tools and agents.
- **Command-line interface (CLI)**
  - Provides a CLI to perform YouTube operations from the terminal.
- **YouTube search automation**
  - Automates searching YouTube for videos based on user‑defined queries (e.g., keywords, topics) for use in LLM workflows.
- **Video data fetching**
  - Retrieves video metadata and related information from YouTube for consumption by downstream tools or agents.
- **Video data management**
  - Supports managing retrieved video data as part of automated or agent‑based workflows (e.g., using, organizing, or piping results into other systems).
- **LLM workflow integration**
  - Designed to plug into LLM/agent workflows, enabling programmatic access to YouTube content as context or data sources.
- **Open-source tooling**
  - Distributed under the Apache-2.0 license.
