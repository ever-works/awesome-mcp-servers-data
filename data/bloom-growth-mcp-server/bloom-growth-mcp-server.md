# Bloom Growth MCP Server

**Category:** Project Management MCP Servers  
**Tags:** performance-management, team-management, goals

## Overview
Bloom Growth MCP Server integrates the Bloom Growth team growth and performance ecosystem into MCP-driven workflows. It is provided via Pipedream Connect as a static MCP endpoint that can be added to any compatible chat client or MCP-enabled application.

- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Provider:** Pipedream / Bloom Growth
- **Use Case:** Connect Bloom Growth’s team and performance tooling into chat-based or MCP-based workflows.

## Features
- **Static MCP endpoint** usable across all compatible clients.
- **Centralized configuration:** same URL for every client, authentication handled when adding the server to each application.
- **Chat client integration:** designed to plug into supported chat clients for MCP usage.
- **MCP-driven workflows:** enables incorporating Bloom Growth’s team growth, performance, and goals ecosystem into automated or interactive workflows.

## Integration & Configuration
- Add the MCP server URL (`https://mcp.pipedream.net/v2`) to your chosen MCP-compatible client.
- Authenticate within the client when prompted.
- Additional configuration details are available via the provider’s configuration resources (e.g., a configuration page linked from the source site).
