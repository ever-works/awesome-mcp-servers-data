# AerisWeather MCP Server

## Overview
AerisWeather MCP Server is an MCP server integration that exposes the AerisWeather weather API and mapping platform, enabling AI agents and applications to access weather and environmental data through Pipedream.

- **Category:** Data-access & integration MCP servers
- **Use case:** Connect an AerisWeather account to AI tools (e.g., Cursor, Claude, Windsurf) for programmatic weather and environmental data retrieval.

## Features
- Exposes the **AerisWeather weather API** via an MCP server.
- Provides access to **weather and environmental data** for MCP agents and applications.
- Integrates with the **AerisWeather mapping platform** for geospatial/weather mapping data.
- Designed to run on **Pipedream's MCP server** infrastructure.
- Allows connecting an **AerisWeather account** to AI tools such as **Cursor, Claude, and Windsurf**.

## Integrations
- **Pipedream MCP**
- **AerisWeather API & mapping platform**
- AI / developer tools:
  - Cursor
  - Claude
  - Windsurf
