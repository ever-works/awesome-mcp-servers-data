# Translate.com MCP Server

**Category:** AI Integration – MCP Servers  
**Website:** https://mcp.pipedream.com/app/translate_com

## Overview
Translate.com MCP Server is an MCP-compatible server that exposes Translate.com’s machine and human translation services to MCP-enabled agents and chat clients. It allows applications to integrate translation workflows through a unified MCP endpoint.

## Features
- Wraps Translate.com’s **machine translation** service for automated language translation.
- Wraps Translate.com’s **human translation** service for higher-quality, human-reviewed translations.
- Provides a **single static MCP server URL** usable across compatible clients.
- Designed for **MCP-compatible agents and chat clients**, enabling translation tools directly within those environments.
- Operates via **Pipedream Connect** infrastructure.

## MCP Server URL
- Base MCP server endpoint (static for all clients):
  - `https://mcp.pipedream.net/v2`
- Authentication is handled when adding the server to your chosen client or application.

## Integration & Usage
- Add the server to any supported MCP client using the static URL.
- Further configuration instructions are available through the client-specific guides and the general Configuration page on the Pipedream site.
