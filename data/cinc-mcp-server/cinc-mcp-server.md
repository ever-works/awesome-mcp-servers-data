# CINC MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** CINC  
**Slug:** `cinc-mcp-server`

## Description
The CINC MCP Server exposes CINC’s real-estate CRM and lead management capabilities via the Model Context Protocol (MCP), allowing compatible chat or AI clients to interact with CINC data and workflows.

## MCP Server URL
- Base MCP server URL (static for all clients):
  - `https://mcp.pipedream.net/v2`

## Features
- MCP endpoint for integrating CINC’s real-estate CRM with MCP-compatible clients.
- Access to CINC lead management functionality via MCP (as supported by the underlying CINC API and Pipedream implementation).
- Static server URL used by all clients; authentication is handled when adding the server in the client.
- Hosted and provided through Pipedream’s MCP infrastructure.

## Integration & Setup
- Add the MCP server to any supported chat or AI client using the static URL.
- Client-specific setup instructions are available via the configuration guidance linked from the hosting page.
