# WisePops MCP Server

**Description**  
WisePops MCP Server is an MCP (Model Context Protocol) integration that connects WisePops—a web application for designing smart website popups—to MCP-compatible agents and applications. It enables agents to work with WisePops campaigns such as popups, surveys, and forms through MCP APIs.

## Features

- **MCP Server Endpoint**  
  - Provides a static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same endpoint usable across different clients; authentication handled when adding the server to the client/app.

- **WisePops Integration**  
  - Connects to a WisePops account for managing website popups.  
  - Designed for creating and managing “smart” popups to increase signups and conversions.

- **Campaign Types**  
  - Supports working with:  
    - Popups  
    - Surveys  
    - Forms

- **Account & Client Configuration**  
  - Flow to connect a WisePops account and select a specific client to work with.  
  - Account status check (“Checking your account…”) during configuration.

- **Client Integration Guidance**  
  - Instructions available for adding the MCP server to different chat clients.  
  - Links to a full configuration page for detailed setup steps.

- **Tooling via MCP**  
  - Exposes “available tools” and “actions” via MCP (loaded dynamically by the client) to interact with WisePops data and campaigns.

---

## Setup & Configuration

- Use the static MCP URL `https://mcp.pipedream.net/v2` as the server endpoint.  
- Add the server to an MCP-compatible chat client or application.  
- Authenticate when registering the server with the client.  
- Connect a WisePops account and select a client within WisePops to begin using tools.
