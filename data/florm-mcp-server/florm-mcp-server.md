# Florm MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Florm  
**Source:** https://mcp.pipedream.com/app/florm

## Overview
Florm MCP Server exposes Florm’s branded, logic‑driven forms, surveys, and data collection workflows to MCP-compatible clients and automations. It is powered by Pipedream Connect and can integrate with over 6,500 applications.

## MCP Server Endpoint
- **Server URL (static for all clients):**
  ```
  https://mcp.pipedream.net/v2
  ```
- Authentication is performed when adding the server to your MCP-compatible application or chat client.

## Features
- **Branded forms**
  - Create forms with custom branding and styling.
- **Logic-driven experiences**
  - Build forms and surveys that use conditional or logic-based flows.
- **Surveys**
  - Design and deliver structured surveys for data collection.
- **Data collection workflows**
  - Orchestrate end-to-end workflows around form and survey submissions.
- **MCP integration**
  - Exposes Florm capabilities as an MCP server for use within MCP clients and automations.
- **App integrations via Pipedream Connect**
  - Connect forms and workflows with 6,500+ third-party applications available on Pipedream (e.g., CRMs, messaging tools, productivity apps, etc.).
- **Static MCP endpoint**
  - Single, static server URL that works across different clients.

## Integration & Setup
- Add the MCP server URL to your chat client or MCP-compatible application.
- Authenticate during the server-connection process within your chosen client.
- Additional configuration details are available on the linked configuration page (not reproduced here).
