# Navigatr MCP Server

An MCP Server for Navigatr, a platform for discovering nearby learning activities, courses, and educational inspiration.

**Website:** https://mcp.pipedream.com/app/navigatr  
**Category:** Documentation & Learning Resources  
**Tags:** education, courses, discovery

## Overview
Navigatr MCP Server integrates Navigatrâ€™s learning discovery capabilities into MCP-compatible clients, allowing users to find popular learning activities, courses, and related educational opportunities directly from their chat or MCP-enabled applications.

---

## Features
- **Learning activity discovery**: Find popular learning activities near you.
- **Course search**: Discover courses you can join through Navigatr.
- **Inspiration browsing**: Explore educational ideas and opportunities to get inspired.
- **MCP compatibility**: Exposes Navigatr functionality via the Model Context Protocol for use in supported clients.
- **Static server endpoint**: Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) that works across clients.
- **Client-agnostic setup**: Can be added to various chat clients and MCP-enabled apps using the same endpoint.
- **Authentication at configuration time**: Authentication is performed when adding the server to your application.
- **Configuration documentation**: Additional configuration details available via a central configuration page (for supported clients).

---

## Integration & Usage
- **MCP server URL:** `https://mcp.pipedream.net/v2`
- **How to start:**
  - Add the server URL in your MCP-compatible chat client or app.
  - Authenticate when prompted during setup.
  - Use the integrated tools to search for nearby learning activities, courses, and educational inspiration.
