# Chatwork MCP Server

**Category:** Messaging MCP Servers  
**Website:** https://mcp.pipedream.com/app/chatwork

## Description
Chatwork MCP Server provides an MCP-compatible interface to Chatwork so AI tools can interact with Chatwork’s group chat, messaging, and collaboration features.

It is built on Pipedream’s MCP server to connect a Chatwork account with AI-powered developer tools.

## Features
- MCP-compatible server for Chatwork
- Access to Chatwork group chats and direct messages (via MCP)
- Integration with Chatwork collaboration features (e.g., shared spaces, team communication)
- Works through Pipedream’s MCP server infrastructure
- Usable from multiple AI / IDE tools, including:
  - Cursor
  - Claude
  - Windsurf
