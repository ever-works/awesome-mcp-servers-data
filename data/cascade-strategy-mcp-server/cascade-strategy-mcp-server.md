# Cascade Strategy MCP Server

**Category:** Project Management MCP Servers  
**Tags:** planning, strategy, work-management

## Overview
The Cascade Strategy MCP Server connects Cascade Strategy’s business planning and execution platform with MCP-compatible AI clients using Pipedream Connect. It exposes a static MCP server endpoint that can be added to supported chat or AI applications to integrate business planning and work-management workflows.

## MCP Server URL
- Base MCP server URL (static for all clients):
  - `https://mcp.pipedream.net/v2`
- Authentication occurs when you add and configure the server in your application.

## Integration & Setup
- Works with MCP-compatible chat and AI clients.
- Add the server by supplying the static MCP URL in your client’s MCP or tools configuration.
- Additional setup and configuration details are available via the platform’s configuration documentation (not included in the provided content).

## Features
- **MCP-compatible endpoint**: Provides a standardized MCP server interface at a single static URL.
- **Authentication at connect time**: Users authenticate when adding the server to their app, enabling secure access without per-client URLs.
- **Pipedream Connect integration**: Runs on Pipedream’s infrastructure, benefiting from its connectivity and reliability.
- **Designed for business planning & execution workflows**: Intended to interface with Cascade Strategy’s tools for planning, strategy, and work management (exact tool list not detailed in the provided content).
- **Multi-client support**: Same URL can be reused across different MCP-capable chat or AI clients.
