# Magnetic MCP Server

**Category:** Project Management MCP Servers  
**Brand:** Magnetic Software  
**Source:** https://mcp.pipedream.com/app/magnetic  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Overview
The Magnetic MCP Server exposes Magnetic Software’s integrated cloud platform—combining project management, sales CRM, and accounts—through the Model Context Protocol (MCP), making it accessible to MCP-compatible tools and agents.

## Features
- **MCP-compatible server**
  - Provides an MCP endpoint that can be added to any compatible chat client or agent framework.
  - Uses a static base URL (`https://mcp.pipedream.net/v2`) for all clients.

- **Magnetic Software integration**
  - Connects to Magnetic’s integrated cloud platform.
  - Surfaces capabilities related to:
    - Project management
    - Sales CRM
    - Accounts / financial data

- **Client-agnostic configuration**
  - Same server URL works across supported chat clients.
  - Authentication handled during server addition/configuration in the client.

- **Pipedream Connect infrastructure**
  - Hosted and powered by Pipedream Connect.
  - Configuration details available via Pipedream’s configuration documentation.

## Authentication
- The MCP server uses a static URL for all users.
- Actual authentication occurs when adding/configuring the server in your chosen client or application.

## Usage
1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add it to your MCP-compatible chat client or tool.
3. Complete the authentication flow as prompted by your client.
