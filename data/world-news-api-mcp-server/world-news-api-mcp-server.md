## Description
The World News API MCP Server is an MCP-compatible service that connects MCP clients to the World News API via Pipedream. It provides programmatic access to thousands of global news sources across more than 150 countries and in over 50 languages, enabling retrieval, monitoring, and analysis of news content.

## Features
- **MCP-compatible server**: Exposes the World News API as an MCP server that can be added to any MCP-capable client.
- **Unified server URL**: Uses a single static MCP server endpoint for all clients:
  - `https://mcp.pipedream.net/v2`
- **Global news coverage**:
  - Access to thousands of news sources.
  - Coverage across 150+ countries.
  - Articles available in 50+ languages.
- **Use cases**:
  - News retrieval from a large set of global sources.
  - Continuous monitoring of news topics or regions.
  - Analysis tasks such as trend tracking or content exploration via MCP clients.
- **Client integration**:
  - Designed to be added to various MCP-enabled chat or agent clients.
  - Authentication is performed when the server is added to the client/application.
- **Configuration resources**:
  - Full configuration guidance available via the linked configuration page on Pipedream.

## Integration Details
- **MCP Server URL**: `https://mcp.pipedream.net/v2`
- **Authentication**: Performed at the time you add the server to your application or chat client (details provided in the configuration documentation).
