# StoryScale MCP Server

Create and manage shareable product tours and demos through an MCP-compatible endpoint.

## Overview
StoryScale MCP Server is an MCP (Model Context Protocol) server integration that lets MCP-aware agents or workflows create and manage shareable product tours and demos. It is provided via a static MCP server URL that can be added to compatible chat clients or applications.

- **Name:** StoryScale MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Use Cases:** Product tours, interactive demos, product marketing flows, user onboarding

## Features
- **MCP-compatible server endpoint**  
  - Exposes StoryScale functionality over the MCP protocol for use with MCP-aware agents and workflows.

- **Static server URL**  
  - Single, static MCP server URL usable across clients:  
    `https://mcp.pipedream.net/v2`
  - Same URL for all supported clients; authentication happens when adding the server to an application.

- **Client-agnostic integration**  
  - Designed to be added to different MCP-capable chat clients or apps.  
  - Works within MCP-aware environments without per-client URL changes.

- **Shareable product tours and demos**  
  - Supports creation and management of product tours and demos that can be shared with users (exact UI/workflow not detailed in source, but enabled through StoryScale integration).

- **Workflow / agent integration**  
  - Can be called from MCP-aware agents or automated workflows to programmatically trigger or manage tours and demos.

- **Configuration documentation**  
  - Full configuration guidance is available via a dedicated configuration page (linked from the app page) to help set up the server in various clients.

## Integration & Setup
- Use the MCP server URL: `https://mcp.pipedream.net/v2`.
- Add this server URL to your chosen MCP-capable chat client or application.
- Authenticate during the add/connect flow as prompted by the client.
- Refer to the provider’s configuration page for client-specific setup steps.
