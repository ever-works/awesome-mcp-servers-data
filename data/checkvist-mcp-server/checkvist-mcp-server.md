# Checkvist MCP Server

**Category:** Project Management MCP Servers  
**Brand:** Checkvist  
**Source:** https://mcp.pipedream.com/app/checkvist

## Overview
Checkvist MCP Server is an MCP Server integration for Checkvist, providing MCP-compatible agents with access to Checkvist’s task management and outlining capabilities, including tags, due dates, search / filtering, and collaboration.

## Features
- **MCP server integration**
  - Exposes Checkvist functionality via the Model Context Protocol (MCP) for compatible agents and clients.
  - Uses a single static MCP server endpoint: `https://mcp.pipedream.net/v2`.
- **Task management**
  - Create and manage tasks and lists.
  - Organize work as an online outliner.
- **Outlining capabilities**
  - Hierarchical lists for structuring tasks and notes.
- **Tags**
  - Apply tags to tasks for organization and filtering.
- **Due dates**
  - Assign due dates to tasks for scheduling and time management.
- **Search and filtering**
  - Search tasks and lists.
  - Filter by criteria such as tags and other task attributes.
- **Collaboration**
  - Share and collaborate on tasks / outlines with others.
- **Keyboard-centric workflow**
  - “Outstanding” keyboard support for fast, power‑user style task and outline management.
- **Client-agnostic configuration**
  - One static MCP server URL works for every supported client.
  - Authentication is handled when adding the server to each application.

## MCP Server URL
- **Endpoint:** `https://mcp.pipedream.net/v2`
- **Usage:** Add this URL as an MCP server in your compatible chat client or application; authenticate during setup.
