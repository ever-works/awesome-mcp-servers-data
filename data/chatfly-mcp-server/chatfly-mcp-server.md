# ChatFly MCP Server

**Category:** Messaging MCP Servers  
**Brand:** ChatFly  
**Source:** https://mcp.pipedream.com/app/chatfly

## Description
ChatFly MCP Server is an MCP-compatible backend that connects ChatFly’s chatbot platform to compatible chat clients. It helps businesses create, test, launch, and manage AI chatbots for a variety of use cases and channels at scale.

## Features
- **MCP-compatible server endpoint**
  - Static MCP server URL: `https://mcp.pipedream.net/v2`
  - Works with any MCP-capable client; authentication is handled when adding the server to the application.

- **AI chatbot lifecycle support**
  - Supports creating AI chatbots for different business use cases.
  - Supports testing chatbots before deployment.
  - Supports launching chatbots into production.
  - Supports ongoing management of chatbots at scale.

- **Omnichannel and multi-use-case support**
  - Designed for AI chatbots that operate across multiple channels.
  - Suitable for varied use cases (e.g., support, sales, internal tools) within a single MCP integration.

- **Integration guidance**
  - Can be added to different chat clients; documentation is organized by client type.
  - Additional configuration details available via the platform’s configuration page (not included in provided content).

## Technical Details
- **Server URL:** `https://mcp.pipedream.net/v2`
- **Protocol:** MCP (Model Context Protocol) server compatible with MCP clients.
