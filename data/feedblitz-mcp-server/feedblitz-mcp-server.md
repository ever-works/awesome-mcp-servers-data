## Description
FeedBlitz MCP Server exposes FeedBlitzâ€™s full-service email marketing capabilities via the MCP protocol so MCP-compatible clients can orchestrate email campaigns, manage subscriber lists, and configure automation flows.

## Features
- **MCP protocol access**: Connects FeedBlitz email marketing functionality to any MCP-capable client.
- **Campaign orchestration**: Allows clients to create, manage, and coordinate email marketing campaigns through MCP.
- **Subscriber list management**: Supports operations on subscriber lists (e.g., adding, organizing, and managing subscribers) using MCP.
- **Automation flows**: Enables configuration and control of automated email sequences and workflows from MCP clients.
- **Static MCP server endpoint**: Uses a single static URL for all clients: `https://mcp.pipedream.net/v2`.
- **Client-agnostic integration**: Designed to work with multiple MCP-compatible chat or app clients; you authenticate when adding the server to your application.

## Technical Details
- **MCP server URL:** `https://mcp.pipedream.net/v2`
- **Integration model:** Add as an MCP server in a compatible client, then authenticate within that client.
