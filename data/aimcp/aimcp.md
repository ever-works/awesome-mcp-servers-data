# AiMCP

A directory of Model Context Protocol (MCP) clients and servers, helping developers discover and compare MCP-based tools and integrations.

**Website:** <https://www.aimcp.info>

## Features
- **MCP-focused directory**: Catalog of MCP servers, clients, and related tools in the Model Context Protocol ecosystem.
- **MCP Hub Gallery**: Central gallery view for browsing available MCP integrations.
- **Featured MCPs**: Highlighted section for notable or popular MCP servers and clients (e.g., Baidu Maps MCP Server, MiniMax-MCP, Magic UI MCP, Zed, ChatMCP, Continue, Notion MCP, Docker MCP, and others).
- **Integration coverage**:
  - Location services (e.g., Baidu Maps MCP Server)
  - AI media generation (e.g., MiniMax-MCP)
  - IDE and editor integrations (e.g., Zed, Cursor/Coding assistants)
  - Productivity and SaaS tools (e.g., Notion, Google Calendar, WooCommerce, Power Platform, Docker)
  - Developer tooling and infrastructure (e.g., OpenTelemetry Collector config, OpenAPI-to-MCP, filesystem access, Cloudflare Workers-based servers)
  - Domain-specific servers (e.g., genomic processing via SAMtools MCP, fitness data via Whoop MCP, Medusa JS integration)
- **Per-item metadata**: Each MCP entry includes a short description and author information (with avatar) to clarify ownership and purpose.
- **Filtering**: Basic filters by properties and tags to refine the list of MCPs.
- **Ecosystem overview**: Introduces Model Context Protocol as an open standard for secure, two-way connections between AI systems and data sources.
