# DarkSky API MCP Server

**Category:** Web Search MCP Servers  
**Brand:** darksky  
**Slug:** `darksky-api-mcp-server`

## Description
The DarkSky API MCP Server is a Model Context Protocol (MCP) server that exposes the DarkSky weather API to MCP-compatible clients. It allows applications and chat-based tools to programmatically query current conditions, weather forecasts, and historical weather data.

## Features
- **Weather forecasts**: Retrieve forecast data from the DarkSky API for specified locations.
- **Historical weather data**: Query past weather conditions using DarkSkyâ€™s historical data endpoints.
- **Smart home and automation use cases**: Intended for integration into smart home workflows (e.g., automations based on current or predicted weather).
- **MCP-compatible endpoint**: Provides a static MCP server URL (`https://mcp.pipedream.net/v2`) that can be used with any supported MCP client.
- **Account-based configuration**: Connect a DarkSky-capable account through Pipedream, then select an MCP client to begin using the tools.
- **Tool-based interaction**: Exposes DarkSky-related actions as tools that MCP clients can call programmatically (tool list is dynamically loaded by the platform).
- **Central configuration page**: Additional configuration options documented via a separate configuration page within Pipedream.

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Protocol:** Model Context Protocol (MCP), suitable for integration with MCP-aware chat clients and applications.
