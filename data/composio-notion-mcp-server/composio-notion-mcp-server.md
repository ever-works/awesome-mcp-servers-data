## Features

- Automated note-taking from conversations in Claude app
- Fetch any Notion document for context in discussions
- Integration with Cursor and Claude Desktop
- Secure OAuth authentication handled by Composio

## Use Cases

- Store conversation details in Notion
- Add product requirement documents to Cursor for feature creation

## Setup

- Install Node.js
- Visit https://mcp.composio.dev/notion, generate npx command
- Run `npx @composio/mcp@latest setup "URL" --client claude`
- Refresh app, initiate connection via hammer icon
