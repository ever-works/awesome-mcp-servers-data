# Exist MCP Server

**Category:** Data Analysis & Exploration MCP Servers  
**Website / Source:** https://mcp.pipedream.com/app/exist

Exist MCP Server is an MCP (Model Context Protocol) integration for **Exist**, a personal life analytics platform that aggregates and analyzes data about your daily activities and habits.

## Features

- **MCP integration for Exist**
  - Connects Exist’s personal life analytics data to MCP-compatible chat or AI clients.
  - Intended for accessing and working with aggregated data about daily activities and habits.

- **Static MCP server endpoint**
  - Single, client-agnostic URL for configuration:  
    `https://mcp.pipedream.net/v2`
  - Same URL can be used across different MCP-capable clients; authentication is handled when adding the server within each client.

- **Client-agnostic setup**
  - Designed to be added to various chat clients that support MCP.
  - Configuration instructions are provided per client via the Pipedream / configuration documentation.

- **Powered by Pipedream Connect**
  - Operates on Pipedream’s integration infrastructure, enabling standardized configuration and management of the MCP server.

## Configuration

- **MCP Server URL**
  - Use `https://mcp.pipedream.net/v2` as the server URL in your MCP-compatible application.
  - You’ll authenticate as part of the add/connect flow in your chosen client.

- **Further configuration details**
  - Additional setup and client-specific steps are available on the Pipedream **Configuration** page linked from the app listing.
