# Kindo MCP Server

**Category:** Security & Attestation MCP Servers  
**Website:** https://mcp.pipedream.com/app/kindo

## Description
Kindo MCP Server is an MCP server integration that brings enterprise-oriented security and policy controls to MCP-based AI workflows. It is delivered via a static MCP server endpoint that can be added to compatible chat or MCP clients, with authentication handled when the server is connected.

## Features
- MCP server endpoint for Kindo
- Designed for enterprise-grade / agentic security controls
- Policy-oriented controls for MCP-based AI workflows
- Static MCP server URL usable across supported clients
- Authentication performed when adding the server to an application
- Configuration guidance available via a dedicated configuration page

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Integration model:** Add as an MCP server in your chat or MCP-compatible client
