# Air MCP Server

**Category:** Project Management MCP Servers  
**Brand:** Air  
**Source URL:** https://mcp.pipedream.com/app/air

## Overview
Air MCP Server exposes Air’s creative operations platform through the Model Context Protocol (MCP), enabling MCP-compatible tools and chat clients to access and manage creative work and assets stored in Air.

## Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Usable across different MCP clients; authentication occurs when adding the server to each application.

- **Air account integration**  
  - Connect an existing Air account.  
  - Select a specific client/workspace within Air after connection.

- **Creative operations & asset access**  
  - Provides access to Air’s creative operations platform capabilities for:  
    - Managing creative work.  
    - Managing creative assets (file storage and asset organization).  
  - Designed for creative teams collaborating around shared assets.

- **Chat client integration**  
  - Can be added to supported chat or MCP clients.  
  - Client-specific setup instructions available via the configuration flow.

- **Configuration resources**  
  - Central configuration page for setup and management of the MCP server connection.

## Technical Details
- **Protocol:** Model Context Protocol (MCP).  
- **Server URL:** `https://mcp.pipedream.net/v2` (static across clients).  
- **Authentication:** Performed when the server is added to a given MCP-compatible application.
