# Smarty MCP Server

**Description**  
Smarty MCP Server is an MCP server integration that provides access to Smarty’s location data intelligence APIs, enabling address verification, validation, enrichment, standardization, geocoding, and auto-complete from compatible MCP-aware applications.

**Category**  
- Data Access & Integration – MCP Servers

**Brand**

**Source**  
- https://mcp.pipedream.com/app/smarty

---

## Features

- **MCP server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2` (used to add the server to MCP-compatible clients).

- **Address verification & validation**  
  - Access to Smarty APIs that verify and validate postal addresses.

- **Address standardization & enrichment**  
  - Standardizes address formats and enriches address records using Smarty’s APIs.

- **Geocoding**  
  - Converts addresses into geographic coordinates via Smarty’s geocoding services.

- **Address auto-complete**  
  - Supports type-ahead / auto-complete address lookups through Smarty’s APIs.

- **Integration with MCP clients**  
  - Can be added to various chat or MCP clients by configuring the static server URL and authenticating with the user’s Smarty account.

- **Account-based configuration**  
  - Uses a connected Smarty account for authentication and access control when the server is added to an application.
