# Clarify MCP Server

**Category:** Business & Commerce – MCP Servers  
**Tags:** CRM, Sales, Revenue

Clarify MCP Server exposes Clarify CRM via the Model Context Protocol (MCP), letting MCP-compatible tools and chat clients access and work with modern revenue-team CRM data and operations in a structured way.

---

## Features

- **MCP-based access to Clarify CRM**  
  - Integrates Clarify’s next-gen CRM with any MCP-compatible client.  
  - Provides structured access to CRM data and revenue operations for modern revenue teams.

- **Standard MCP endpoint**  
  - Single static MCP server URL for all clients:  
    - `https://mcp.pipedream.net/v2`
  - URL can be reused across different applications and chat clients.

- **Client-agnostic integration**  
  - Works with multiple chat / MCP clients; you select your chat client and follow its specific setup instructions.  
  - Centralized configuration guidance available via a dedicated configuration page.

- **Authentication handled on connect**  
  - Authentication occurs when adding the server to your application, not via custom per-client URLs.

- **Pipedream Connect infrastructure**  
  - Hosted and powered by Pipedream Connect, providing managed MCP server hosting and connectivity.

---

## Setup

- Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server to your MCP-compatible app or chat client and authenticate during setup.
- Optional: Refer to the linked configuration page (from the source site) for client-specific instructions.
