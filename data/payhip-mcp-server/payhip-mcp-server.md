# Payhip MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Payhip

Payhip MCP Server is an MCP server that lets MCP-compatible agents access Payhip’s e‑commerce capabilities for selling digital products, courses, and memberships.

---

## Features

- **MCP server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same URL works across compatible MCP clients; authentication is handled when adding the server to your application.

- **Payhip e‑commerce integration**  
  - Connects MCP agents to Payhip’s platform for:
    - Selling digital products
    - Selling online courses
    - Selling memberships
  - Designed to let agents operate on behalf of creators who sell directly to their fans and followers.

- **Client compatibility**  
  - Can be added to multiple MCP‑compatible chat clients.  
  - Setup instructions are provided per client via the configuration documentation.

- **Configuration documentation**  
  - Central configuration guide available via the platform’s Configuration page for details on adding and managing the server in different applications.
