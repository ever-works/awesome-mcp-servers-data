# MarketTime MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** MarketTime  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Description
MarketTime MCP Server connects the MarketTime integrated sales platform to the Model Context Protocol (MCP). It enables AI-driven workflows around B2B order management and sales operations by exposing MarketTime functionality to compatible chat clients and applications.

## Features
- **MCP-compatible endpoint**: Static server URL (`https://mcp.pipedream.net/v2`) usable across clients that support the Model Context Protocol.
- **MarketTime integration**: Connects to the MarketTime integrated sales platform for B2B sales, order management, and related operations.
- **AI workflow enablement**: Designed to support AI-driven workflows for:
  - Order management
  - Sales operations
  - B2B sales processes
- **Client-agnostic setup**: Same MCP server URL works for any supported client; authentication is handled when adding the server to the application.
- **Configuration documentation**: Additional setup and configuration details available via a dedicated configuration page (through Pipedream / Pipedream Connect).

## Usage
- Add `https://mcp.pipedream.net/v2` as an MCP server in a compatible chat client or application.
