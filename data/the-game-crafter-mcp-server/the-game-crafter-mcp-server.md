## Overview

Integrates The Game Crafter's print-on-demand service for tabletop games into MCP workflows, allowing AI agents to handle game design and production tasks.

## Features

- Browse product catalogs and components
- Create and manage print projects
- Calculate pricing and shipping
- Upload artwork and proofs
- Order prototypes and full runs

## Use Cases

- AI-assisted tabletop game prototyping
- Automated game component sourcing

## Pricing

Free and open-source. Platform fees apply for production.
