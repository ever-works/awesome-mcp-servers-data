# Anymail Finder MCP Server

## Overview
The Anymail Finder MCP Server is an MCP (Model Context Protocol) server provided by Pipedream that integrates the Anymail Finder SaaS service. It enables MCP-compatible agents or chat clients to programmatically discover and validate business email addresses associated with company employees.

- **Type:** MCP server / developer integration
- **Provider / Brand:** Pipedream
- **Category:** Business & Commerce – MCP Servers
- **Primary use case:** Lead generation and email validation within MCP-enabled applications

## Features
- **Email discovery:**
  - Find email addresses for employees at a given company.
  - Designed for business / professional email discovery and lead generation workflows.

- **Email validation:**
  - Validate discovered email addresses to ensure they are deliverable and usable.

- **MCP server endpoint:**
  - Static MCP server URL: `https://mcp.pipedream.net/v2`.
  - Single URL works for all clients; authentication is handled when adding the server to the application.

- **MCP client integration:**
  - Can be added to compatible chat clients or MCP-enabled applications.
  - Configuration guidance available via a dedicated configuration page (per-client setup instructions).

- **Account-based configuration:**
  - Requires connecting an Anymail Finder account via Pipedream.
  - Account is checked/validated during setup.

- **Tooling exposure to agents:**
  - Exposes Anymail Finder capabilities as “tools” to MCP agents (e.g., actions for finding and validating emails) once loading is complete.
