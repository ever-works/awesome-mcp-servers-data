# 1sec Mail MCP Server

**Brand:** Pipedream  
**Category:** Messaging MCP Servers  
**Tags:** email, testing, automation

## Overview
1sec Mail MCP Server is an MCP server integration for 1sec Mail that lets you programmatically create and receive disposable email accounts in seconds, useful for automated workflows and testing scenarios.

## Features
- Programmatic creation of disposable email accounts
- Programmatic receipt of emails for those disposable accounts
- Designed for fast setup and use (create and receive email in ~1 second)
- Suitable for automated workflows and testing environments (e.g., sign-up flows, verification emails)
- Exposed via a single static MCP server URL usable by any compatible client
- Authentication performed when adding the server to your application

## Technical details
- **MCP server base URL:** `https://mcp.pipedream.net/v2`
- **Integration type:** MCP server powered by Pipedream Connect
