# iHomefinder MCP Server

## Overview
The iHomefinder MCP Server provides MCP-based access to iHomefinder’s online property search technology and MLS® System data. It is designed to let developers and teams integrate real estate listing and search capabilities into MCP-compatible applications and workflows.

- **Category:** Business & Commerce – MCP Servers  
- **Use cases:** Real estate listing search, MLS® data access, property search integration in apps and workflows

## Features
- **MCP-based access to iHomefinder services**  
  Access iHomefinder’s property search technology and MLS® System data through the MCP protocol, enabling standardized integration with MCP clients.

- **Integration with real estate workflows**  
  Embed listing search and MLS® data retrieval into MCP-enabled tools (e.g., chat-based or workflow-based applications) to support real estate use cases.

- **Static MCP server endpoint**  
  Uses a single, static MCP server URL for all clients:  
  `https://mcp.pipedream.net/v2`

- **Authentication at application level**  
  Authentication is handled when adding the server to an MCP-compatible application, allowing secure access tied to your iHomefinder account.

- **Client-agnostic setup**  
  The same MCP server URL can be used across different MCP-compatible clients; configuration is done per client within its own interface.

- **Configuration via Pipedream**  
  The server is provided and hosted via Pipedream’s MCP infrastructure, with configuration handled through the Pipedream interface and configuration documentation.
