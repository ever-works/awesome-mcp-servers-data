# iLovePDF MCP Server

## Overview
The **iLovePDF MCP Server** connects AI agents and MCP-compatible clients to the iLovePDF service, enabling automated PDF and Office document processing via a standard MCP endpoint.

- **Category:** Document Management MCP Servers
- **Brand:** iLovePDF
- **Slug:** `ilovepdf-mcp-server`
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **PDF Merge** – Combine multiple PDF files into a single document.
- **PDF Split** – Separate a PDF into multiple files (e.g., by page ranges).
- **PDF Compression** – Reduce the file size of PDFs.
- **Office to PDF Conversion** – Convert Office documents (such as Word, Excel, PowerPoint) to PDF.
- **General PDF & Document Processing** – Access additional iLovePDF tools exposed via MCP for working with PDF and Office files.
- **MCP-Compatible Endpoint** – Single static URL that can be used across different MCP clients; authentication is handled when adding the server to the application.

## Technical Details
- **Server type:** MCP server powered by Pipedream Connect
- **Authentication:** Performed when adding the server to the client (details handled by the client / configuration flow)
- **Usage context:** Designed for integration into chat clients or agents that support MCP servers.
