# Forms On Fire MCP Server

## Overview
Forms On Fire MCP Server is an MCP-compatible server (powered by Pipedream Connect) that allows AI tools and chat clients to interact with Forms On Fire, a no-code platform for digitizing mobile and cloud-based forms and workflows.

- **Type:** MCP server / workflow-automation integration
- **Category:** workflow-automation-mcp-servers
- **Use cases:** Digitizing forms, submitting and managing mobile and cloud-based forms, automating workflows, supporting field service operations.

## MCP Server URL
- **Server URL:** `https://mcp.pipedream.net/v2`
- This is a static URL used by all MCP clients; authentication is performed when adding the server to your application.

## Features
- **MCP compatibility**
  - Exposes Forms On Fire capabilities to MCP-compatible AI tools and chat clients.
  - Can be added as an MCP server in supported applications using a single static URL.

- **Forms digitization and management**
  - Enables digitization of forms that would otherwise be handled on paper or in separate systems (via Forms On Fire).
  - Supports working with mobile and desktop form workflows.

- **Workflow automation**
  - Integrates with the Forms On Fire platform, which is designed to manage and automate form-driven workflows.
  - Suitable for field service and other operational use cases that rely on structured data capture and routing.

- **Cloud-based operation**
  - Connects to Forms On Fire’s cloud software through Pipedream’s infrastructure.

- **No-code platform integration**
  - Leverages Forms On Fire’s no-code environment, enabling non-developers to design and manage forms and workflows that MCP clients can then access.

- **Multi-client support**
  - Single, static MCP endpoint (`https://mcp.pipedream.net/v2`) that can be reused across different MCP-compatible chat clients.

## Getting Started
- Add the server URL `https://mcp.pipedream.net/v2` to your MCP-compatible application.
- Authenticate within your client when prompted.
- Optional: Refer to the platform’s configuration documentation ("Configuration" page on Pipedream) for detailed setup steps.
