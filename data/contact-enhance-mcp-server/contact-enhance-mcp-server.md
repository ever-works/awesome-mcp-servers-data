# Contact Enhance MCP Server

MCP server that integrates Contact Enhance to enrich business contact data via an AI-powered bot.

**Website:** https://mcp.pipedream.com/app/contact_enhance  
**Category:** Business & Commerce MCP Servers  
**Tags:** data-enrichment, contacts, lead-enrichment

---

## Overview
Contact Enhance MCP Server is an MCP-compatible bot that connects to the Contact Enhance service to enrich business contact and lead data within MCP-enabled chat clients and applications.

---

## Features
- **Business contact enrichment**: Enhances existing business contact and lead records using Contact Enhance data.
- **AI-powered interaction**: Operates as a bot within MCP-compatible chat clients.
- **Standard MCP server endpoint**: Uses a static MCP server URL (`https://mcp.pipedream.net/v2`) that works across supported clients.
- **Client-agnostic integration**: Can be added to various MCP-enabled chat applications; configuration details are provided per client.
- **Central configuration**: Full setup and configuration instructions are available via a dedicated configuration page.

---

## Integration & Usage
- **MCP server URL:** `https://mcp.pipedream.net/v2`
- **Authentication:** Performed when adding the server to the client/application.
- **Setup:** Select your chat client on the provider site or use the full configuration documentation.
