# Gan.AI MCP Server

An MCP server that connects to Gan.AI, enabling agents and tools to generate and manage personalized videos at scale.

**Source:** <https://mcp.pipedream.com/app/gan_ai>

---

## Overview

Gan.AI MCP Server exposes Gan.AI’s personalized video capabilities via the Model Context Protocol (MCP), so MCP-aware clients and agents can programmatically create and work with customized videos in their workflows.

---

## Features

- **MCP-compatible endpoint**  
  - Single static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Usable from any MCP-aware client or agent framework.  
  - Authentication is handled when adding/configuring the server in your application.

- **Personalized video at scale**  
  - Integrates with Gan.AI’s platform to generate personalized videos for many recipients.  
  - Suitable for use cases like marketing, outreach, and CRM workflows where large-scale video personalization is needed.

- **Client-agnostic integration**  
  - Works with multiple chat or agent clients that support MCP.  
  - Configuration guidance available per client (via the linked configuration documentation).

- **Configuration via Pipedream Connect**  
  - Hosted and provided through Pipedream’s “Connect” infrastructure.  
  - Centralized configuration page for setup and management (including adding the server URL and credentials in supported clients).

---

## Setup

1. Use the static MCP server URL: `https://mcp.pipedream.net/v2`.  
2. Add this URL to your MCP-aware chat client or agent environment.  
3. Complete authentication in the client when prompted.  
4. Follow client-specific configuration instructions from the linked configuration page.
