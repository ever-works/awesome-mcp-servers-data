# Acelle Mail MCP Server

**Category:** Messaging MCP Servers  
**Slug:** `acelle-mail-mcp-server`  
**Brand:** Acelle Mail  
**Source URL:** https://mcp.pipedream.com/app/acelle_mail

## Description
Acelle Mail MCP Server is an MCP (Model Context Protocol) server integration for Acelle Mail, a self-hosted, full-featured email marketing web application. It enables MCP-based workflows and automation around email campaigns and subscriber management from compatible chat or agent clients.

## Features
- **MCP integration for Acelle Mail**: Exposes Acelle Mail functionality through the MCP protocol for use in AI or agent-based applications.
- **Self-hosted email marketing backend**: Connects to an existing, self-hosted Acelle Mail instance to leverage its email marketing capabilities.
- **Email campaign workflows**: Designed to support workflows around creating, managing, and automating email campaigns (as supported by Acelle Mail and its exposed tools).
- **Subscriber management automation**: Enables automation related to subscriber lists and audience management, via the MCP tools provided by the integration.
- **Static MCP server endpoint**: Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) that works for every client.
- **Client-agnostic setup**: Can be added to any MCP-compatible chat client or application that supports configuring an external MCP server.
- **Account-based authentication**: Authentication occurs when adding the server to an application, allowing secure access to Acelle Mail resources.

## Setup
1. Configure your Acelle Mail connection in the Pipedream Acelle Mail MCP Server page.  
2. Copy the static MCP server URL: `https://mcp.pipedream.net/v2`.  
3. Add this MCP server URL to your MCP-compatible client or application.  
4. Authenticate when prompted by your client to complete the integration.
