# Ometria MCP Server

**Category:** Business & Commerce · MCP Servers  
**Tags:** retail, marketing automation, customer data

## Overview
Ometria MCP Server is a Model Context Protocol (MCP) server that connects Ometria’s customer marketing platform for retail to MCP-compatible AI clients. It enables AI-powered customer and campaign operations by exposing Ometria’s retail marketing capabilities and data through a standardized server endpoint.

- **Static MCP endpoint:** `https://mcp.pipedream.net/v2`  
- **Provider:** Ometria, delivered via Pipedream Connect  
- **Use case focus:** Retail customer marketing, customer data–driven campaigns

## Features
- **MCP-compatible server** exposing Ometria’s retail customer marketing platform to AI agents and MCP-enabled chat clients.
- **Static server URL** (`https://mcp.pipedream.net/v2`) usable across clients; authentication is handled when adding the server in the client.
- **Client-agnostic integration**: can be added to various MCP-capable chat or AI applications by following client-specific setup instructions.
- **Support for AI-powered operations** on customer and campaign data within Ometria (e.g., customer marketing workflows, campaign-related interactions) through MCP.
- **Central configuration reference** via a dedicated configuration page explaining how to connect and configure the server in supported clients.

## Integration & Setup
- Add the MCP server to your MCP-enabled app using the static URL.
- Authentication is completed during server setup within the client.
- Additional configuration details are available on the linked configuration page from the Pipedream site.
