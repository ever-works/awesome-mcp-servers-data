# Cliento MCP Server

## Overview
Cliento MCP Server provides an MCP (Model Context Protocol) interface to Cliento’s booking system, allowing MCP-compatible tools and chat clients to interact with appointments and scheduling data. It enables you to access and manage booking functionality from within applications that support MCP servers.

- **Category:** Business & Commerce – MCP Servers  
- **Use cases:** booking management, appointments, scheduling integration in MCP-enabled chat / agent environments

## Features
- **MCP server endpoint:** Static, shared MCP server URL for all clients: `https://mcp.pipedream.net/v2`  
- **Booking system integration:** Connects MCP tools to Cliento’s booking system for working with bookings, appointments, and schedules.  
- **Centralized authentication:** Authentication occurs when adding the server to your MCP-compatible application (per-client auth on a shared endpoint).  
- **Chat client support:** Can be added to supported chat clients that implement the MCP standard.  
- **Configuration documentation:** Additional configuration and setup details are available via the linked configuration page on the host site.  
- **Hosted by Pipedream:** Runs on Pipedream’s MCP infrastructure (Pipedream Connect).

## Setup / Usage
1. Use the static MCP server URL: `https://mcp.pipedream.net/v2`.  
2. Add this server to your MCP-compatible chat client or application.  
3. Authenticate when prompted in your client to connect to your Cliento booking data.
