# API.Bible MCP Server

**Category:** Documentation & Learning Resources  
**Brand:** American Bible Society  
**Source:** https://mcp.pipedream.com/app/api_bible

## Overview
API.Bible MCP Server is a Model Context Protocol (MCP) server that exposes API.Bible’s collection of Bible translations and related scriptural data for integration into MCP-enabled AI workflows and tools.

## Features
- Access to API.Bible’s collection of Bible translations and related Bible data.
- Coverage of nearly 2,500 Bible versions.
- Support for over 1,600 languages.
- Standard MCP server endpoint suitable for multiple MCP-compatible clients.
- Static MCP server URL (`https://mcp.pipedream.net/v2`) that works across clients.
- Authentication handled when adding/configuring the server in your application.
- Usable within various MCP-enabled chat and AI applications.

## Technical Details
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`
- **Integration:** Added as a server within MCP-compatible chat or AI clients.
- **Configuration:** Follows generic MCP server configuration flows (via client-specific setup or the central configuration page on Pipedream Connect).

## Use Cases
- Embedding multi-translation Bible access into AI chatbots and assistants.
- Scriptural study tools that require quick access to many versions and languages.
- Educational or research workflows that need Bible texts programmatically within MCP-enabled environments.
