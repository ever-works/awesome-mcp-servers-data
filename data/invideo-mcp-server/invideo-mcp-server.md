# Invideo MCP Server

**Category:** Media Processing MCP Servers  
**Website:** https://invideo.io/ai/mcp/  
**Protocol:** Model Context Protocol (MCP) over SSE (https://mcp.invideo.io/sse) with OAuth 2.1 authentication

## Overview
The Invideo MCP Server exposes Invideo’s AI video creation platform as an MCP-compatible tool for AI agents, developers, and creators. It lets clients generate full-length or short-form videos from text prompts, automating scriptwriting, stock/AI visuals, voiceovers, subtitles, and assembly into ready-to-publish videos for platforms like YouTube, TikTok, and Instagram.

## Features

### MCP & Integration
- Implements the Model Context Protocol (MCP) to connect AI systems to Invideo’s video generation workflows.
- Accessible via SSE endpoint: `https://mcp.invideo.io/sse`.
- Uses OAuth 2.1 for secure authentication and access control.
- Designed for integration into apps, AI agents, and automation pipelines.

### AI Video Generation
- Generate full-length AI videos on virtually any topic and in various styles using text prompts.
- Converts plain-language ideas into complete, ready-to-publish videos.
- End-to-end workflow: from prompt → script → visuals → voiceover → subtitles → final video.

### Script & Content Creation
- Uses AI models to generate video scripts directly from user prompts.
- Suitable for a wide range of subject areas (e.g., archaeology, blockchain, astronomy, branding, finance, tech, travel, and more).

### Visuals & Media Handling
- Connects to a library of 16M+ stock images.
- Can generate relevant AI video clips when appropriate.
- Automatically selects or generates visuals aligned with the generated script.

### Audio & Voiceovers
- Adds AI voiceovers to match the script.
- Designed to produce publish-ready narration without manual recording.

### Subtitles & Localization
- Automatically generates subtitles in 50+ languages.
- Supports multilingual video output for global audiences.

### Output Types & Use Cases
- Long-form videos for YouTube and other platforms.
- Short-form content:
  - YouTube Shorts generation from simple topic prompts.
  - Instagram Reels creation for topics such as finance, tech, travel, and more.
  - TikTok and other vertical-video platforms.
- Suitable for:
  - Social media content production at scale.
  - Creator workflows embedded in apps or tools.
  - Automated marketing, education, or informational video pipelines.

### Developer & AI Agent Focus
- Built for developers, creators, and AI agents using MCP.
- Fits into AI-driven workflows where an assistant or agent can request complete video assets programmatically.
- Designed for secure, scalable video generation in production environments.

## Target Users
- Developers integrating AI video generation into their applications.
- AI assistants/agents needing a video generation backend via MCP.
- Content creators and teams looking to automate or scale video production for YouTube, TikTok, Instagram, and other platforms.
