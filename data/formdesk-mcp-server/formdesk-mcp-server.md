# Formdesk MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Formdesk  
**Source:** https://mcp.pipedream.com/app/formdesk

## Description
Formdesk MCP Server is a Model Context Protocol (MCP) server that connects Formdesk’s online form creation and management platform to AI tools via Pipedream. It allows AI-powered environments to work with Formdesk’s application, registration, order, and survey forms.

## Features
- **MCP integration**: Exposes Formdesk functionality through the Model Context Protocol.
- **Form types**: Supports working with application, registration, order, and survey forms.
- **Online form management**: Integrates Formdesk’s online form creation and management features into MCP-compatible tools.
- **AI tool connectivity**: Connects Formdesk to AI tools such as:
  - Cursor
  - Claude
  - Windsurf
- **Pipedream-based server**: Runs as an MCP server provided through Pipedream’s platform for easy connection and configuration.
