# Process Street MCP Server

Checklist, workflow, and SOP (standard operating procedure) management via an MCP (Model Context Protocol) server.

## Overview
The **Process Street MCP Server** exposes Process Street’s checklist and workflow capabilities through a standard MCP endpoint, allowing AI/chat clients and other MCP-compatible tools to interact with Process Street programmatically.

- **Server URL:** `https://mcp.pipedream.net/v2`
- **Platform:** Powered by Pipedream Connect
- **Category:** Workflow automation MCP server

## Features
- **MCP-compatible server**
  - Provides a static MCP endpoint (`https://mcp.pipedream.net/v2`) usable across different MCP clients.
  - Authentication is performed when adding the server to the client/app.

- **Checklist and workflow access**
  - Designed to work with Process Street’s checklist and workflow system.
  - Intended for managing and interacting with SOPs, checklists, and workflows through MCP-enabled applications.

- **Client integration**
  - Can be added to various chat or MCP clients by configuring the single static URL.
  - Supports configuration via a dedicated configuration flow (referenced Configuration page).
