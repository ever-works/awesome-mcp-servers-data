## Description
Stripe MCP is an MCP (Model Context Protocol) server that integrates with Stripe to enable programmatic handling of payments, customers, and refunds from MCP-compatible clients.

## Features
- MCP server implementation for Stripe
- Operations for handling payments
- Operations for managing customers
- Operations for issuing and managing refunds
- Environment-based configuration via `.env` (example provided)
- Packaged as a Python project (`pyproject.toml`)
- Dockerfile provided for containerized deployment
- Example configuration via `smithery.yaml`
- Version and dependency management via `uv.lock`

## Technical Details
- Language: Python (per `pyproject.toml` and `.python-version`)
- Installable and runnable as a service compatible with the Model Context Protocol
- Open-source with contribution guidelines (`CONTRIBUTING.md`)
