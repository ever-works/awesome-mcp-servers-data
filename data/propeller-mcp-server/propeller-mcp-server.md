# Propeller MCP Server

Digitizes buyer–seller interactions through MCP-compatible workflows and tools.

- **Category:** Business & Commerce MCP Servers  
- **Brand:** Propeller  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Source:** https://mcp.pipedream.com/app/propeller  
- **Tags:** sales, B2B, quotes

## Features

- **MCP-compatible server** for integrating Propeller buyer–seller workflows into MCP-capable chat clients and applications.
- **Static server endpoint** (`https://mcp.pipedream.net/v2`) usable across different clients; authentication is handled when adding the server to each application.
- **Buyer–seller interaction digitization** to manage and streamline sales processes, especially for B2B and quote-related workflows.
- **Multi-client support**: the same MCP server URL can be added to various chat clients that support MCP.
- **Centralized configuration docs** via a dedicated configuration page linked from the app.

## Integration & Configuration

- Add the MCP server to your chat client or MCP-compatible app using the static URL: `https://mcp.pipedream.net/v2`.
- Configuration details and setup instructions are provided through the referenced configuration page on the source site.
