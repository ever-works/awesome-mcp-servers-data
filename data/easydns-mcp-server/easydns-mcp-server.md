# easyDNS MCP Server

**Category:** Cloud & DevOps – MCP Servers  
**Brand:** easyDNS  
**Integration provider:** Pipedream Connect

![easyDNS](https://www.easydns.com/wp-content/uploads/2020/04/easydns-home-hero.png)

## Overview
The easyDNS MCP Server is an integration that exposes easyDNS DNS and domain services through the Model Context Protocol (MCP). It allows MCP-compatible clients (such as AI chat applications) to interact programmatically with easyDNS infrastructure via a standardized server endpoint.

## Features
- MCP-compatible server for easyDNS services
- Access to easyDNS DNS and domain management capabilities via MCP-aware clients
- Static MCP server URL shared across clients
- Authentication handled when adding/configuring the server in each client
- Works with multiple chat / MCP client applications
- Hosted and operated via Pipedream Connect infrastructure

## MCP Server URL
Use this URL when adding the easyDNS MCP Server to your MCP-compatible client:

```text
https://mcp.pipedream.net/v2
```

This is a static URL used by all clients; authentication occurs during setup in your application.

## Getting Started
- Add a new MCP server / tool in your chat or MCP client.
- Use the MCP server URL: `https://mcp.pipedream.net/v2`.
- Follow your client’s instructions to authenticate and complete configuration.
- Refer to the Pipedream MCP Configuration page for detailed setup steps.
