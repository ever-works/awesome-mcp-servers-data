# Sifter MCP Server

**Category:** Project Management MCP Servers  
**Brand:** Sifter  
**Slug:** `sifter-mcp-server`  
**Source:** https://mcp.pipedream.com/app/sifter

## Overview
Sifter MCP Server is an MCP-compatible integration that connects the Sifter issue tracking system with AI tools. It enables agents that support the Model Context Protocol (MCP) to interact directly with Sifter’s simple, streamlined issue tracker.

## Features
- **MCP-compatible server** for Sifter issue tracking.
- **Create issues** in Sifter via MCP-compatible AI agents.
- **Manage existing issues** (e.g., update or organize) through connected AI tools.
- **Query issues** in Sifter from within MCP-enabled environments.
- **Integration with AI tools** such as:
  - Cursor
  - Claude
  - Windsurf
- **Utilizes Pipedream’s MCP server** as the bridge between Sifter and AI tools.
