# Coinbase Commerce MCP Server

**Category:** Blockchain & Crypto MCP Servers  
**Brand:** Coinbase Commerce  
**Slug:** coinbase-commerce-mcp-server

## Overview
The Coinbase Commerce MCP Server exposes Coinbase Commerce cryptocurrency payments and merchant transaction APIs through the Model Context Protocol (MCP). It enables applications and chat-based clients to interact with Coinbase Commerce for crypto payment workflows using a standardized MCP interface.

Supported cryptocurrencies include (as provided):
- Bitcoin (BTC)
- Bitcoin Cash (BCH)
- DAI
- Dogecoin (DOGE)
- Ethereum (ETH)
- Litecoin (LTC)
- USD Coin (USDC)

The server is hosted and powered by Pipedream Connect.

## MCP Server Endpoint
- **Base MCP server URL:** `https://mcp.pipedream.net/v2`
- **Authentication:** Performed when adding the server to your application (details depend on your client and Coinbase Commerce credentials).
