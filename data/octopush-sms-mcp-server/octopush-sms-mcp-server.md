# Octopush SMS MCP Server

**Category:** Messaging MCP Servers  
**Brand:** octopush  
**Slug:** octopush-sms-mcp-server

## Description
Octopush SMS MCP Server exposes the Octopush SMS marketing and messaging platform as an MCP-compatible server, allowing SMS workflows to be triggered and managed from MCP-based tools and chat clients.

## Features
- MCP-compatible server endpoint for Octopush SMS
- Static MCP server URL: `https://mcp.pipedream.net/v2`
- Works with multiple MCP-based chat clients and applications
- Authentication handled when adding the server to an MCP-compatible app
- Supports configuration of an Octopush SMS account within the MCP context
- Designed for SMS marketing and general SMS communication workflows

## Integration & Configuration
- Connect an existing Octopush SMS account through the MCP server
- Add the static MCP server URL to supported chat clients or MCP-based tools
