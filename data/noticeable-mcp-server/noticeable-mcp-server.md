## Description
The Noticeable MCP Server is a Model Context Protocol server that connects MCP-compatible agents and chat clients to Noticeable. It allows agents to manage announcements, changelogs, and newsfeeds so teams can communicate product updates and relevant news to users and improve engagement.

## Features
- MCP server endpoint for Noticeable: `https://mcp.pipedream.net/v2`
- Works as a static URL for all supported MCP clients
- Authentication handled when adding the server to the client/application
- Enables management of:
  - Announcements
  - Changelogs
  - Newsfeeds and product news
- Designed to help teams announce:
  - New features
  - Latest releases
  - Other relevant product or company updates
- Integrates with various MCP-compatible chat clients (setup guided per client)
- Configuration guidance available via a dedicated configuration page

## Integration & Setup
- Connect a Noticeable account through Pipedream and select the relevant client to begin
- Copy and use the provided MCP server URL in any supported MCP client
- Per-client instructions available for adding the server to chat or agent applications
