# Pipedream MCP Server

**Category:** Workflow Automation MCP Servers  
**Brand:** Pipedream  
**Website:** https://mcp.pipedream.com/app/pipedream  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Description
Pipedream MCP Server exposes the Pipedream integration platform to AI agents and MCP-compatible clients, enabling them to search, manage, and generate Pipedream components, interact with apps, and create or manage subscriptions and workflows via MCP.

## Features

- **Static MCP Server Endpoint**
  - Single static URL (`https://mcp.pipedream.net/v2`) used for all compatible clients.
  - Authentication handled when adding the server to each application.

- **Client Integration**
  - Designed to be added as an MCP server inside compatible chat or agent clients.
  - Configuration guidance available via a dedicated configuration page.

- **Available Tools (Actions)**
  - **Search For Registry Components**  
    - Search for components in the Pipedream global registry using a query string.
  - **List Apps**  
    - List all available Pipedream apps.
  - **Get Component**  
    - Retrieve information about a published Pipedream component.
  - **Get Component From Global Registry**  
    - Fetch detailed information for a component from the Pipedream global registry.
  - **Get App**  
    - Get details for a specific Pipedream app.
  - **Generate Component Code (using AI)**  
    - Generate Pipedream component code programmatically with AI assistance.
  - **Create a Subscription**  
    - Create a subscription (e.g., for Pipedream event sources or workflows).
  - **Delete a Subscription**  
    - Delete an existing subscription.
