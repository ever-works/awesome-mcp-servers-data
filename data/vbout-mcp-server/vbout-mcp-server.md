# VBOUT MCP Server

**Website:** https://mcp.pipedream.com/app/vbout  
**Category:** Business & Commerce – MCP Servers  
**Tags:** marketing-automation, email-marketing, analytics

## Overview

VBOUT MCP Server exposes VBOUT’s all‑in‑one marketing automation capabilities—email marketing, social media, landing pages, lead management, automation, and analytics—over the MCP protocol via Pipedream. It allows applications and chat clients to interact with VBOUT programmatically through a static MCP server URL.

## Features

### MCP Server & Connection
- Static MCP server endpoint: `https://mcp.pipedream.net/v2`
- Works for every VBOUT client; authentication occurs when adding the server to your application
- Can be added to supported chat clients or other MCP-compatible apps using Pipedream’s configuration flow
- Connects to a VBOUT account and selected client to manage marketing data and actions

### Marketing Automation Capabilities (via VBOUT)
- Email marketing campaign creation
- Social media posting
- Landing pages (platform capability; not directly exposed as a listed action in this page)
- Lead management / contact management
- Marketing automation workflows (platform capability)
- Analytics for marketing activities

### Available Tools (Actions Exposed via MCP)
The server exposes 8 actions as tools:

1. **Update Contact**
   - Update details of a specific contact in VBOUT.

2. **Unsubscribe Contact**
   - Unsubscribe a specific contact from a selected list.

3. **Create Social Media Message**
   - Create a new social media post through VBOUT’s social channels.

4. **Create Email Marketing Campaign**
   - Create a new email marketing campaign in VBOUT.

5. **Create Email List**
   - Create a new email list to organize contacts.

6. **Add Custom Event to Contact**
   - Add a custom activity/event record to a contact’s history.

7. **Add Contact to List**
   - Move / assign a contact to a specific list.

8. **Add Contact to List by Email**
   - Add contacts to a specific list by matching on email address.
   - All contacts with the same email in the provided list are updated if they exist.
