# Zoho Survey MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Zoho

An MCP Server integration for Zoho Survey that lets AI agents create, distribute, and analyze online surveys and questionnaires directly within MCP-powered workflows.

---

## Features

- **MCP integration:** Exposes Zoho Survey capabilities through a standard MCP Server endpoint, enabling use from compatible chat or agent clients.
- **Survey creation:** Create online surveys and questionnaires for a variety of use cases, including:
  - Customer satisfaction
  - General customer feedback
  - Business surveys
  - Research and data collection
  - Other custom survey scenarios
- **Survey distribution:** Use within MCP workflows to distribute surveys to respondents (e.g., as part of automated conversations or processes).
- **Data collection:** Capture responses to surveys and questionnaires for later analysis.
- **Analysis support:** Enable AI agents and workflows to analyze collected survey data using MCP tools and automations.
- **Static MCP server URL:**
  - Server endpoint: `https://mcp.pipedream.net/v2`
  - Same URL works for all clients; authentication is performed when adding the server to each application.
- **Client-agnostic setup:** Designed to be added to multiple chat or agent clients that support MCP, using the same server URL.

---

## Technical Details

- **MCP server URL:** `https://mcp.pipedream.net/v2`
- **Integration host:** Pipedream Connect MCP infrastructure
- **Usage pattern:** Add the server to an MCP-compatible client, then authenticate and configure Zoho Survey access.
