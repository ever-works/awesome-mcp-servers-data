# actiTIME MCP Server

**Category:** Project Management MCP Servers  
**Brand:** actiTIME  
**Slug:** `actitime-mcp-server`

An MCP Server integration that connects actiTIME’s time tracking and project cost management capabilities into MCP-compatible applications via Pipedream.

---

## Features

- **MCP server integration for actiTIME**  
  - Exposes actiTIME’s time tracking and project cost management functionality as tools within MCP-compatible chat or AI clients.

- **Static MCP server endpoint**  
  - Uses a single static MCP server URL for all clients:  
    `https://mcp.pipedream.net/v2`
  - Authentication is handled when adding the server to your application.

- **Client-agnostic setup**  
  - Can be added to various chat / AI clients that support the MCP protocol.  
  - Configuration guidance available via a dedicated configuration page on Pipedream.

- **Tool-based access**  
  - Exposes actiTIME functionality as “available tools” inside MCP-aware applications (actual tools/actions are dynamically loaded within Pipedream).

---

## Integration & Configuration

- Connect your actiTIME account within Pipedream.
- Copy the static MCP server URL and add it to your MCP-compatible application.
- Select your chat client in Pipedream for client-specific setup instructions or use the general configuration documentation.
