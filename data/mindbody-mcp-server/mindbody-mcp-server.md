# Mindbody MCP Server

**Description**  
Mindbody MCP Server provides an MCP integration layer between Mindbody’s wellness business management platform and AI tools. It enables AI-powered workflows for scheduling, client management, and studio operations automation.

**Source**  
<https://mcp.pipedream.com/app/mindbody>

---

## Features

- **MCP integration with Mindbody**: Connects Mindbody’s wellness business management platform to the Model Context Protocol (MCP) ecosystem.
- **AI tool connectivity**: Works with AI tools such as Cursor, Claude, and Windsurf via Pipedream’s MCP server.
- **Scheduling workflows**: Enables AI-driven scheduling and calendar-related operations using Mindbody data.
- **Client management automation**: Supports automations around client records and interactions within Mindbody.
- **Studio operations support**: Facilitates automated workflows for studio operations (e.g., classes, bookings, and related business processes) by exposing Mindbody functionality through MCP.
