## TPSCheck MCP Server

**Website:** https://mcp.pipedream.com/app/tpscheck  
**Category:** Security & Attestation MCP Servers  
**Brand:** TPSCheck

### Description
TPSCheck MCP Server is a model context protocol (MCP) server that enables MCP-compatible applications to perform real-time checks on phone numbers against TPS/CTPS registries and retrieve detailed phone intelligence, including validity, location, line type, and provider.

### Features
- **Real-time TPS/CTPS checks**  
  - Determine whether a phone number is registered with Telephone Preference Service (TPS) or Corporate Telephone Preference Service (CTPS).

- **Phone number validity**  
  - Verify if a phone number is structurally valid and can be used.

- **Location insights**  
  - Retrieve geographic information associated with the phone number.

- **Line type detection**  
  - Identify the type of phone number (e.g., mobile, landline, etc.).

- **Provider information**  
  - Get the carrier or service provider associated with the number.

- **MCP-compatible server**  
  - Exposes capabilities via a standard MCP Server interface for integration with any MCP-compatible client.

- **Static server URL**  
  - Single endpoint for all clients: `https://mcp.pipedream.net/v2`.

- **Authentication at client setup**  
  - Authentication is handled when adding the server to your MCP-enabled application.
