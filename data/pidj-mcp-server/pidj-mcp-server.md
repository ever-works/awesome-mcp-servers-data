# Pidj MCP Server

## Overview
Pidj MCP Server is an MCP-compatible server that exposes Pidj’s SMS-based business text messaging capabilities to MCP-enabled clients. Once configured, it allows applications and chat clients that support the Model Context Protocol (MCP) to send and manage SMS communications through Pidj.

## Features
- **MCP-compatible SMS integration**: Access Pidj’s business SMS text messaging services from any MCP-compatible client.
- **Static server URL**: Use a single, static MCP server endpoint for all clients:
  - `https://mcp.pipedream.net/v2`
- **Per-client authentication**: Authenticate when adding the server in each client, allowing secure, client-specific access to Pidj.
- **Works across multiple chat clients**: Can be added to various MCP-capable chat applications; setup instructions are provided per client.
- **Centralized configuration**: Configuration options and details are available via a dedicated configuration page (linked from the interface).

## Getting Started
1. Connect or configure your Pidj account via the Pidj MCP Server page.
2. Copy the static MCP server URL: `https://mcp.pipedream.net/v2`.
3. Add this MCP server URL to your preferred MCP-compatible chat client.
4. Authenticate when prompted by your client to start using Pidj SMS features.
