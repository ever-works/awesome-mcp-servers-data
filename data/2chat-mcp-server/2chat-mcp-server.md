# 2Chat MCP Server

**Description**  
2Chat MCP Server is an MCP-compatible integration that brings all-in-one WhatsApp automation into existing workflows, allowing small teams to automate messaging while continuing to use WhatsApp as usual.

**Source**  
<https://mcp.pipedream.com/app/_2chat>

**Category**  
Messaging MCP Servers

**Tags**  
- WhatsApp  
- Messaging  
- Automation

---

## Features

- MCP-compatible WhatsApp integration for use within MCP-capable clients and tooling.
- All-in-one WhatsApp automation designed for small teams.
- Works alongside normal WhatsApp usage (no change to how users use WhatsApp day-to-day).
- Static MCP server endpoint for all clients: `https://mcp.pipedream.net/v2`.
- Centralized configuration via a 2Chat configuration flow.
- Account connection step to link your 2Chat/WhatsApp setup before use.
- Client-agnostic: same server URL works with multiple MCP-compatible chat clients.
- Authentication handled when adding the server to your MCP-compatible application.
- Documentation available via a dedicated configuration page (setup and usage details).
- Dynamic tools/actions loading to expose available WhatsApp-related operations to the client.

---

## Setup & Integration

1. Connect your 2Chat account in the configuration UI.
2. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
3. Add this server URL to your MCP-compatible chat client and complete authentication.
4. Use the loaded tools/actions to automate WhatsApp messaging within your workflows.
