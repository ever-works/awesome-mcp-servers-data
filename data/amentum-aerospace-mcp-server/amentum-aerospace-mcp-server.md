# Amentum Aerospace MCP Server

## Overview
Amentum Aerospace MCP Server exposes Amentum Aerospace’s predictive scientific models for aviation, space, defense, and navigation via the Model Context Protocol (MCP). It is accessible as an MCP server endpoint that AI agents and compatible clients can connect to in order to query and use these models.

- **Category:** AI Integration – MCP Servers  
- **Provider / Brand:** Amentum Aerospace (via Pipedream)  
- **Protocol:** Model Context Protocol (MCP)  
- **Primary domains:** Aviation, space, defense, navigation

## Features
- **Predictive scientific models:** Access to Amentum Aerospace’s domain-specific predictive models across aviation, space, defense, and navigation use cases.
- **MCP server endpoint:** Exposed through a static MCP server URL that can be added to any MCP-compatible application:
  - Endpoint: `https://mcp.pipedream.net/v2`
- **AI agent integration:** Designed for AI agents and tools to query and consume the models through the MCP interface.
- **Client-agnostic URL:** A single static URL is used for all clients, with authentication handled when the server is added to an application.
- **Configurable in Pipedream:** Can be configured and connected via Pipedream’s interface by linking an account and selecting a client.
- **Multi-client support:** Instructions and configuration flows are available for different chat clients that support MCP.

## Integration & Usage
- **Connection flow:**
  1. Connect your Amentum Aerospace integration through Pipedream.
  2. Copy the static MCP server URL: `https://mcp.pipedream.net/v2`.
  3. Add the MCP server URL to your MCP-compatible chat client or application.
  4. Authenticate within your client when prompted.
- **Configuration resources:** Additional setup details are available on Pipedream’s configuration page (linked from the product page).
