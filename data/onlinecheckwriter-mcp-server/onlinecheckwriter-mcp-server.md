# OnlineCheckWriter MCP Server

## Description
The OnlineCheckWriter MCP Server is a Model Context Protocol (MCP) server that exposes Online Check Writer’s capabilities—online check printing, eChecks, and mail checks—to MCP-based AI agents and workflows. It allows applications and chat-based clients to integrate check-related payment operations using a standardized MCP endpoint.

## Features
- MCP-compliant server for Online Check Writer
- Supports online check printing operations
- Supports eCheck-related operations
- Supports mail check-related operations
- Single static MCP server URL usable across compatible clients
- Authentication handled when adding the server to an application

## Technical Details
- MCP server base URL: `https://mcp.pipedream.net/v2`
- Works with MCP-compatible chat clients and applications
- Provided and hosted via Pipedream Connect
