## Open Strategy Partners Marketing Tools MCP Server

### Description
An open-source Model Context Protocol (MCP) server that lets LLMs apply Open Strategy Partners’ writing and product marketing techniques for technical and product marketing content.

### Features
- MCP server interface for integrating with compatible LLM tools
- Tools for technical marketing content creation
- Content editing tools and “codes” aligned with OSP’s methodology
- Value map tools to articulate and structure product value
- Product positioning tools to support messaging and narrative development
- Workflows oriented around product marketing and marketing operations
