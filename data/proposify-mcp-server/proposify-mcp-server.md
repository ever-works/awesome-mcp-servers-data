# Proposify MCP Server

Proposal software MCP server that exposes Proposify’s proposal creation and management functionality via the Model Context Protocol (MCP), enabling programmatic access from compatible chat or AI clients.

- **Name:** Proposify MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Brand:** Proposify  
- **Tags:** proposals, sales, CRM  
- **Source URL:** https://mcp.pipedream.com/app/proposify  
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features

- **MCP-compatible server** that exposes Proposify capabilities over the Model Context Protocol for use in AI/chat clients.
- **Static server URL** (`https://mcp.pipedream.net/v2`) usable across clients, with authentication handled when adding the server to each application.
- **Programmatic access to proposal workflows**, designed to integrate with Proposify’s proposal creation and management features (e.g., generating, updating, and managing sales proposals via MCP-aware tools).
- **Client-agnostic integration**: intended to work with multiple chat/AI clients that support adding custom MCP servers.
- **Configuration guidance** available via a dedicated configuration page (how to add and use the server within supported clients).
