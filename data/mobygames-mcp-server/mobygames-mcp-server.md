# MobyGames MCP Server

**Category:** Search & Discovery MCP Servers  
**Brand:** MobyGames  
**Slug:** `mobygames-mcp-server`

## Overview
The MobyGames MCP Server provides programmatic access to MobyGames’ catalog of electronic games (computer, console, and arcade). It exposes structured data on games, including detailed release information, credits, cover art, and related metadata via the MCP (Model Context Protocol).

**MCP Server URL:**
```text
https://mcp.pipedream.net/v2
```

## Features
- Programmatic access to the MobyGames game catalog via MCP
- Coverage of multiple platforms:
  - Computer games
  - Console games
  - Arcade games
- Game-level structured data, organized on a game-by-game basis
- Access to:
  - Release information
  - Game credits
  - Cover art
  - Other associated game metadata (“and much more”) via the MobyGames database
- Static MCP endpoint URL that works across compatible MCP clients
- Designed for integration into chat / AI applications and other tools that support MCP

## Integration & Usage
- Use the static MCP URL `https://mcp.pipedream.net/v2` when adding the server to your MCP-compatible client or application.
- Authentication is performed when you add the server to your application (client handles auth details).
