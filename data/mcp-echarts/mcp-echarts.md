# mcp-echarts

**Category:** Data Visualization  
**Brand:** apache-echarts

An open-source MCP server that uses Apache ECharts to generate interactive visual charts via AI, enabling MCP-compatible clients to request and render ECharts-based visualizations dynamically within LLM workflows.

![mcp-echarts bar demo](https://raw.githubusercontent.com/hustcc/mcp-echarts/main/assets/demo-bar.png)
![mcp-echarts line demo](https://raw.githubusercontent.com/hustcc/mcp-echarts/main/assets/demo-line.png)

---

## Features

- **MCP Server for Charts**
  - Implements an MCP (Model Context Protocol) server dedicated to chart generation.
  - Integrates directly into MCP-compatible LLM tools and workflows.

- **Apache ECharts Integration**
  - Uses Apache ECharts as the charting engine.
  - Supports rich, interactive, browser-based visualizations.

- **AI-Driven Chart Generation**
  - Allows LLMs/clients to request charts programmatically.
  - Designed for dynamic chart creation within AI conversations or pipelines.

- **Data Visualization & Analysis**
  - Intended for visual chart generation and data analysis scenarios.
  - Suitable for bar, line, and other common chart types supported by ECharts (examples include bar and line charts as shown in the demo images).

- **Open-Source Project**
  - Source available on GitHub.
  - Licensed under the MIT License.

- **Containerization & Tooling**
  - Includes a `Dockerfile` for containerized deployment.
  - Provides configuration and project tooling files (e.g., `.env.example`, `.prettierrc`, `biome.json`, GitHub workflow configs) for development and CI.
