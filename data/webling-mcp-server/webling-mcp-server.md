# Webling MCP Server

**Category:** Business & Commerce – MCP Servers  
**Tags:** CRM, membership, management

## Overview
Webling MCP Server connects MCP tools to Webling’s online club and membership management software. It exposes Webling functionality through a standard MCP endpoint that can be added to compatible chat or MCP-enabled applications.

- **MCP server URL:** `https://mcp.pipedream.net/v2`
- **Provider:** Webling, via Pipedream Connect

## Features
- **Standard MCP Endpoint**  
  - Single static URL (`https://mcp.pipedream.net/v2`) used by all clients.  
  - Authentication handled when adding/configuring the server in your application.

- **Chat / Client Integration**  
  - Can be added to MCP-capable chat clients and applications.  
  - Uses a common configuration flow documented in the associated configuration page.

- **Access to Webling Functions**  
  - Provides MCP tools access to Webling’s online club and membership management capabilities (e.g., handling members, clubs, and related data) via the MCP server.

- **Hosted by Pipedream Connect**  
  - Server infrastructure and connectivity are managed by Pipedream’s Connect platform.

## Setup
1. Use `https://mcp.pipedream.net/v2` as the MCP server URL in your app.  
2. Follow your chat client or application’s instructions to add an MCP server.  
3. Authenticate when prompted to link your Webling account.
