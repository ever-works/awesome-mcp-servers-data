# T2M URL Shortener MCP Server

**Brand:** Pipedream  
**Category:** Business & Commerce MCP Servers  
**Tags:** url-shortening, marketing, links

## Overview
T2M URL Shortener MCP Server provides an MCP-compatible interface to the T2M URL Shortener platform, enabling URL marketing and link-shortening operations from MCP-enabled chat clients and tools.

## Features
- MCP server endpoint for T2M URL Shortener: `https://mcp.pipedream.net/v2`
- Works as a static URL for all compatible MCP clients
- Authentication handled when adding the server to an application
- Designed for URL marketing and link-shortening use cases
- Suitable for a range of organizations, from startups to large enterprises
