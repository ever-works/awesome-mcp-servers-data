# Basecamp MCP Server

Integration that connects MCP-enabled tools to Basecamp’s project management and team communication features.

## Overview
- **Type:** MCP server integration
- **Platform:** Basecamp (via Pipedream Connect)
- **Category:** Project management & team communication
- **Server URL:** `https://mcp.pipedream.net/v2`

## Features
- Provides a static MCP server endpoint (`https://mcp.pipedream.net/v2`) usable from any compatible MCP client.
- Supports authentication at the time you add the server in your MCP-enabled application.
- Integrates MCP tools with Basecamp’s project management and team collaboration capabilities (e.g., projects, communication, coordination workflows) via Pipedream.
- Usable across different chat or MCP clients; setup instructions are available per client.
- Central configuration guidance available through a dedicated configuration page.

## Setup
- Copy and use the MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server in your MCP-enabled or chat-based client.
- Authenticate to Basecamp when prompted during server addition.
- Optionally follow client-specific and advanced instructions from the configuration documentation.
