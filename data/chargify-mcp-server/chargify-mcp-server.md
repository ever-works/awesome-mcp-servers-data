# Chargify MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Chargify  
**Slug:** `chargify-mcp-server`

## Description
Chargify MCP Server provides access to Chargify’s B2B SaaS billing and revenue management capabilities through the Model Context Protocol. It is delivered via Pipedream’s MCP server, enabling AI tools and IDEs to work with Chargify data and operations.

## Features
- Connects a Chargify account through Pipedream’s MCP server
- Exposes Chargify billing and revenue management functions via the Model Context Protocol
- Integrates with AI tools such as:
  - Cursor
  - Claude
  - Windsurf
- Designed for B2B SaaS billing, subscriptions, and revenue workflows
