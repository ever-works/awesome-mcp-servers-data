# wit.ai MCP Server

**Category:** AI Integration – MCP Servers  
**Brand:** meta-witai  
**Website:** https://mcp.pipedream.com/app/wit_ai

## Overview
wit.ai MCP Server exposes wit.ai’s natural language understanding (NLU) APIs through the Model Context Protocol (MCP), allowing MCP-compatible tools and conversational agents to perform intent detection and entity extraction via a standardized server endpoint.

## Features
- **MCP-compatible NLU server**
  - Exposes wit.ai’s natural language understanding APIs via MCP
  - Enables integration with any MCP-capable chat or developer client
- **Intent detection**
  - Detects user intents from natural language input using wit.ai models
- **Entity extraction**
  - Extracts structured entities from user utterances
- **Static server endpoint**
  - Single static MCP server URL usable across clients: `https://mcp.pipedream.net/v2`
  - Authentication handled when adding the server to each application/client
- **Multi-client integration**
  - Designed to be added to various chat clients and tools that support MCP
- **Configuration resources**
  - Central configuration documentation available via the "Configuration" page on the provider site

## Technical Details
- **Protocol:** Model Context Protocol (MCP)
- **Endpoint:** `https://mcp.pipedream.net/v2`
- **Provider:** Pipedream Connect (hosting the MCP server for wit.ai)
