# HumanLoop MCP Server

## Overview
The HumanLoop MCP Server is an MCP (Model Context Protocol) server hosted on Pipedream that connects MCP-compatible chat clients (such as ChatGPT) to HumanLoop’s platform for building and aligning LLM-based applications with human feedback.

- **Category:** AI Integration – MCP Servers  
- **Provider / Brand:** HumanLoop (via Pipedream)  
- **Primary Use Case:** Integrate HumanLoop workflows and feedback-driven LLM alignment into MCP-based agents and applications.

## Features
- **MCP-compatible server endpoint**  
  - Static, shared server URL for all clients:  
    `https://mcp.pipedream.net/v2`  
  - Works across different MCP-enabled chat clients.

- **Account-based configuration**  
  - Connect your HumanLoop account through Pipedream.  
  - Select and manage HumanLoop clients / accounts after sign-in.  
  - Authentication occurs when adding the server to your MCP application.

- **Client integration support**  
  - Guidance available for integrating with ChatGPT (OpenAI) as an MCP client.  
  - Additional setup details via Pipedream’s MCP Configuration page.

- **Hosted by Pipedream**  
  - Operates as a managed MCP server on Pipedream’s infrastructure.  
  - Uses Pipedream Connect for underlying connectivity.

- **Current tool availability**  
  - At the time of the referenced content, no specific MCP tools/actions are published for this app yet.
