# Scholarly MCP Server

An open-source MCP (Model Context Protocol) server that enables LLMs to search for scholarly and academic articles, integrating literature search directly into MCP-based workflows.

## Features

- MCP server for searching accurate academic and scholarly articles
- Designed to integrate literature search into Model Context Protocol workflows used by LLMs
- Supports multiple scholarly content vendors, with additional vendors planned
- Example assets and usage demonstrations provided in the `examples/` directory
- Containerized deployment support via included `Dockerfile`
- Smithery.ai integration via `smithery.yaml` and published Smithery server metadata
- Python-based implementation managed via `pyproject.toml` and `uv.lock`
- Public security assessment badge available via MseeP.ai
