# Product Fruits MCP Server

An MCP server that lets MCP-compatible agents interact with Product Fruits’ user onboarding and in-app guidance platform.

## Overview
- **Type:** MCP Server (Model Context Protocol)
- **Platform:** Product Fruits (user onboarding & in-app guidance)
- **Provider / Host:** Pipedream Connect
- **Category:** Business & Commerce MCP Servers
- **Use Cases:**
  - Manage or integrate user onboarding flows
  - Work with in-app guidance and product adoption experiences via MCP-compatible clients

## MCP Server URL
- **Server URL:** `https://mcp.pipedream.net/v2`
- This is a static URL used for all clients.
- Authentication is performed when adding the server to your MCP-compatible application.

## Setup & Integration
- Add the server URL to your MCP-enabled chat client or application.
- Client-specific setup instructions are available via the provider’s configuration documentation (e.g., “Configuration” page on Pipedream Connect).
