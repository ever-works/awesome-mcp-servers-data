# PostgreSQL MCP Server

PostgreSQL MCP Server grants AI agents direct execution access to PostgreSQL databases via the Model Context Protocol, allowing schema inspection and SELECT statement execution for instant data queries. Key features include Docker self-hosting and natural language to SQL translation support. Essential for AI agent data access and exploration; serves as a baseline for relational DB MCP servers compared to specialized ones like MongoDB MCP for NoSQL.
