# Ikas MCP Server

**Description**  
Ikas MCP Server is an integration that exposes the Ikas new‑generation e‑commerce platform to MCP‑compatible agents and workflows via Pipedream. It provides a static MCP server endpoint that can be added to compatible chat or agent applications to interact with Ikas programmatically.

**Category:** Business & Commerce → MCP Servers  
**Tags:** e-commerce, API integration, workflow automation

---

## Features

- **MCP-compatible server endpoint**  
  - Provides a static MCP server URL: `https://mcp.pipedream.net/v2`.
  - The same URL works for every client; authentication is handled when you add the server to your application.

- **Ikas account connection**  
  - Lets you connect your Ikas account through Pipedream.
  - After connection, you can select the specific client (store/account) to work with.

- **Integration with MCP-based apps**  
  - Designed to be added as a server in MCP-compatible chat clients or agent frameworks.
  - Supports tool-based interactions with the Ikas e‑commerce platform once tools/actions are loaded.

- **Configuration guidance**  
  - Offers client-specific setup instructions for adding the server to different chat clients.
  - Provides a full configuration page (via the Pipedream UI) for detailed setup and management.

- **Tooling / actions exposure**  
  - Exposes Ikas-related tools and actions through the MCP protocol (listed as “Available tools” in the UI).  
  - Tools are dynamically loaded for use in automated workflows and agent interactions.
