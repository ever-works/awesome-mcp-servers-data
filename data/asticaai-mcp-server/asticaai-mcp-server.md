# astica.ai MCP Server

Integrate astica.ai’s AI capabilities into MCP-compatible tools without deploying or managing your own ML infrastructure.

## Overview
- **Type:** MCP server (Model Context Protocol)
- **Provider:** Pipedream / astica.ai
- **Category:** AI integration – MCP servers
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible AI integration**: Exposes astica.ai capabilities via a standard MCP server endpoint for use with MCP-aware clients and tools.
- **No model deployment required**: Use hosted AI capabilities without managing ML models or infrastructure.
- **Static server URL**: Single, static MCP server URL (`https://mcp.pipedream.net/v2`) works across all supported clients.
- **Client-agnostic setup**: Designed to be added to various MCP-compatible chat clients and applications.
- **Configuration documentation**: Dedicated configuration instructions available via the platform’s Configuration page.

## Usage
- Add `https://mcp.pipedream.net/v2` as an MCP server endpoint in your compatible client or application.
- Authenticate during server setup in the client (authentication handled at configuration time, not via changing the URL).
