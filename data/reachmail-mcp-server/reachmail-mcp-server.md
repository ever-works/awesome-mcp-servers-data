## Features
- Programmatic access to ReachMail via MCP
- Create HTML email campaigns
- Create and manage newsletters
- Send email campaigns from applications or chat clients
- Track email campaign performance (e.g., delivery and engagement tracking)
- Web-based email campaign tooling behind the MCP interface
- Single static MCP server URL usable by any compatible client
- Authentication handled when adding the server to an application

## Integration & Usage
- Add the MCP server to compatible chat clients or applications using the static URL.
- Configure authentication within the client when registering the server.
- Additional configuration details are available on the associated configuration page (via the hosting platform).
