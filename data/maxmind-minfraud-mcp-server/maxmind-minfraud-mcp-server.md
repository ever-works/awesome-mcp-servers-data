# MaxMind minFraud MCP Server

## Overview
MaxMind minFraud MCP Server is an MCP-based integration with MaxMind’s minFraud service. It provides access to online fraud risk scoring and risk data for transactions within MCP-compatible workflows and chat clients.

- **Category:** Security & Attestation MCP Servers  
- **Use cases:** Fraud detection, risk analysis, and transaction risk evaluation

## Features
- **MCP server integration** with MaxMind’s minFraud API for use in MCP-enabled tools and chat clients.
- **Online fraud prevention** by returning risk scores and risk-related data for online transactions.
- **Transaction risk scoring** to assess the likelihood of fraud for a given transaction.
- **Risk data enrichment** with additional attributes about transactions to support fraud review and decisioning.
- **Static MCP server URL** (`https://mcp.pipedream.net/v2`) usable by all clients, with authentication handled when adding the server to an app.
- **Works across compatible clients**: can be added to different chat or MCP clients by configuring the server URL and authentication.

## Configuration
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- Authentication is performed when adding the server to the application.
