# ONLYOFFICE DocSpace MCP Server

**Category:** Document Management MCP Servers  
**Brand:** ONLYOFFICE

An MCP server that connects AI tools to ONLYOFFICE DocSpace so they can work with customizable collaboration rooms and documents in the ONLYOFFICE ecosystem.

---

## Features

- Integrates ONLYOFFICE DocSpace with MCP-compatible AI/chat clients via a single static server URL
- Enables AI tools to interact with documents and collaboration rooms in ONLYOFFICE DocSpace
- Supports customizable rooms for different business purposes within the ONLYOFFICE ecosystem
- Uses a common MCP endpoint (`https://mcp.pipedream.net/v2`) that can be added to multiple clients
- Authentication handled when adding the server within each client/application

---

## Technical Details

- **MCP server URL:** `https://mcp.pipedream.net/v2`
- **Integration platform:** Powered by Pipedream Connect
- **Usage:** Add the MCP server URL to any supported MCP client and authenticate within that client
