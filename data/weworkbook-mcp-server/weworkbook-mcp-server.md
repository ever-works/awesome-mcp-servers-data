# Weworkbook MCP Server

**Category:** Business & Commerce MCP Servers  
**Tags:** training, workflow-automation, education

## Description
Weworkbook MCP Server is an MCP server that connects AI tools with Weworkbook, a cloud-based training organization platform. It enables automated workflows for managing training enrollments and helps reduce administrative overhead for education and training providers.

## Features
- **MCP-compatible server endpoint**  
  - Static server URL for all clients: `https://mcp.pipedream.net/v2`  
  - Can be added to any MCP-capable chat or AI application.

- **Integration with Weworkbook**  
  - Connects AI tools to the Weworkbook cloud-based training management system.  
  - Supports workflows around enrollments and administration for training organizations.

- **Account-based configuration**  
  - Connect a Weworkbook account to use the server.  
  - Client selection step after account connection.

- **Client-agnostic setup**  
  - Same URL works for every client; authentication is handled when adding the server to the chosen application.  
  - Compatible with multiple chat clients / AI frontends via MCP.

- **Configuration documentation**  
  - Dedicated configuration page (via Pipedream) describing how to add and use the server in different clients.
