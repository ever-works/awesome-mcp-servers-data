# Wave MCP Server

An MCP server that connects Wave’s small‑business money management and accounting capabilities to MCP-compatible AI tools and workflows.

- **Category:** Finance & Market Data MCP Servers
- **Brand:** Wave
- **Source URL:** https://mcp.pipedream.com/app/wave
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Description
Wave MCP Server is a static MCP endpoint that exposes Wave’s small‑business accounting and money management functionality to any compatible client or workflow. The same server URL can be used across different MCP clients, with authentication handled when the server is added to each application.

## Features
- **Static MCP endpoint**
  - Single server URL (`https://mcp.pipedream.net/v2`) usable across all supported MCP clients.
- **Wave integration**
  - Provides access to Wave’s small‑business money management and accounting capabilities via MCP.
- **Client-agnostic setup**
  - Works with any MCP-compatible chat client or application.
- **Per-client authentication**
  - Authentication occurs when adding the server to each client/application (no per-client URL required).
- **Configuration documentation**
  - Dedicated configuration page available for setup details and client-specific instructions.
- **Pipedream-powered**
  - Hosted and operated via Pipedream Connect infrastructure.

## Use Cases
- Integrating Wave accounting data into AI assistants or chat clients.
- Automating small‑business money management workflows through MCP-compatible tools.
- Centralizing money management operations from Wave inside custom AI-driven workflows.
