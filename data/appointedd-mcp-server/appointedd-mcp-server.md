# Appointedd MCP Server

**Brand:** appointedd  
**Category:** business-commerce-mcp-servers  
**Tags:** scheduling, appointments, booking

## Overview
Appointedd MCP Server exposes Appointedd’s online scheduling capabilities to AI tools and agents via the Model Context Protocol (MCP). It enables programmatic management of bookings, availability, and appointments through a standardized MCP endpoint.

## MCP Server URL
- **Base MCP server URL:** `https://mcp.pipedream.net/v2`  
  - This is a static URL used by all clients.  
  - Authentication is handled when adding the server to your MCP-compatible application.
