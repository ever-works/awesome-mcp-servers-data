# CDR Platform MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** cdr-platform  
**Source:** https://mcp.pipedream.com/app/cdr_platform

## Description
MCP Server for the CDR Platform, providing an API interface over the Model Context Protocol to manage carbon dioxide removal (CDR) operations. It runs on Pipedream’s MCP server to connect a CDR Platform account with AI-powered tools.

## Features
- MCP-based API interface to the CDR Platform
- Designed for managing carbon dioxide removal operations via programmatic access
- Integrates a CDR Platform account with AI tools such as:
  - Cursor
  - Claude
  - Windsurf
- Built to run on Pipedream’s MCP server infrastructure

## Integrations
- Pipedream MCP server
- Cursor
- Claude
- Windsurf
