# SiteSpeakAI MCP Server

**Category:** AI Integration – MCP Servers  
**Slug:** sitespeakai-mcp-server

## Overview
SiteSpeakAI MCP Server is an MCP server integration that exposes SiteSpeakAI’s custom-trained AI support chatbots to MCP-compatible clients. It enables you to integrate AI-powered customer support chatbots, trained on your business or website content, directly into tools that support the Model Context Protocol.

## Features
- **MCP server integration for SiteSpeakAI**: Connects SiteSpeakAI’s AI support chatbots to any MCP-compatible client.
- **Custom-trained AI chatbots**: Uses SiteSpeakAI’s platform to create support chatbots tailored to your website or business content.
- **Customer support automation**: Automates responses to customer queries using ChatGPT-based AI.
- **Static MCP server URL**: Single, static endpoint usable across clients:  
  `https://mcp.pipedream.net/v2`
- **Client-agnostic setup**: Same server URL for all supported MCP clients; authentication is handled when adding the server to each application.
- **Configuration documentation**: Additional configuration details available via a dedicated configuration page on the provider’s site.

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Provider / Infrastructure:** Powered by Pipedream Connect
