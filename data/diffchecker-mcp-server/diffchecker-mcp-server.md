# Diffchecker MCP Server

## Overview
Diffchecker MCP Server is an MCP-compatible integration that exposes Diffchecker’s comparison capabilities (text, images, files, and folders) through a standardized MCP server endpoint. It is provided via Pipedream Connect and can be used with any MCP client that supports configuring a static server URL.

- **Name:** Diffchecker MCP Server  
- **Category:** Development Tools → MCP Servers  
- **Provider / Brand:** Diffchecker (via Pipedream Connect)  
- **Website / Source:** https://mcp.pipedream.com/app/diffchecker  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP server integration for Diffchecker**  
  - Provides a Machine-Control Protocol (MCP) server that exposes Diffchecker’s comparison tools to compatible clients.

- **Static MCP server endpoint**  
  - Uses a single static URL (`https://mcp.pipedream.net/v2`) for all supported clients.  
  - Authentication occurs when adding/configuring the server in the client application.

- **Multi-type comparison capabilities**  
  - **Text comparison:** Compare text content to identify differences.  
  - **Image comparison:** Compare images to highlight visual differences.  
  - **File comparison:** Compare individual files for content changes.  
  - **Folder comparison:** Compare directory structures and contents.

- **Client-agnostic configuration**  
  - Can be added to different MCP-compatible chat or agent clients using the same server URL.  
  - Documentation references per-client setup instructions and a central configuration guide.

- **Pipedream Connect integration**  
  - Hosted and delivered through Pipedream Connect infrastructure.  
  - Uses Pipedream’s standard configuration and policy framework for terms and privacy.

## Setup & Integration
- Use the static MCP server URL: `https://mcp.pipedream.net/v2` when adding the server in your MCP-compatible client.  
- Follow your chat or agent client’s instructions for adding an MCP server and performing authentication.  
- A general configuration guide is available via the “Configuration” page linked from the app listing.
