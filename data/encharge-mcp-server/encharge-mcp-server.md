# Encharge MCP Server

Marketing automation platform built for B2B SaaS, exposed via the Model Context Protocol (MCP) for AI agents.

**Website:** https://mcp.pipedream.com/app/encharge  
**MCP Server URL:** `https://mcp.pipedream.net/v2`  
**Category:** Business & Commerce – MCP Servers  
**Brand:** Encharge  
**Tags:** marketing-automation, email-marketing, workflows

## Overview
Encharge MCP Server is an integration that exposes Encharge’s B2B SaaS marketing automation capabilities—such as workflows and email campaigns—to AI agents and MCP-compatible clients through a single static MCP server endpoint.

## Features
- **MCP Integration for Encharge**
  - Provides access to Encharge’s marketing automation features via the Model Context Protocol.
  - Designed to be used by AI agents and MCP-compatible chat or automation clients.

- **Static MCP Server Endpoint**
  - Uses a single static URL that works for all MCP clients: `https://mcp.pipedream.net/v2`.
  - Authentication is handled when adding the server within your chosen client.

- **B2B SaaS Marketing Automation Focus**
  - Built specifically around B2B SaaS use cases.
  - Intended to expose capabilities such as workflows and email campaigns to automated agents.

- **Client-Agnostic Setup**
  - Can be added to multiple MCP-compatible chat clients.
  - Configuration instructions are available via the platform’s configuration page.

- **Pipedream Connect Integration**
  - Powered by Pipedream Connect infrastructure.
  - Governed by Pipedream’s Terms and Privacy Policy.
