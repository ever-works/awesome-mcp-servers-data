# FullContact MCP Server

**Category:** Business & Commerce – MCP Servers  
**Website:** https://mcp.pipedream.com/app/full_contact

## Overview
The FullContact MCP Server connects MCP-compatible applications to FullContact’s identity resolution platform. It enables enrichment and contact intelligence operations via a static MCP server endpoint hosted by Pipedream.

## Features
- **Identity Resolution Integration**  
  - Connects to FullContact’s identity resolution platform through MCP.
  - Supports CRM and contact-data workflows that rely on identity resolution.

- **Contact & Data Enrichment**  
  - Enables contact intelligence and enrichment operations via MCP tools/actions (tools are dynamically loaded in the interface).

- **Static MCP Server URL**  
  - Single static endpoint for all clients: `https://mcp.pipedream.net/v2`.  
  - Authentication handled when the server is added to an MCP-compatible application.

- **App Integration Workflow**  
  - Copy-and-paste MCP server URL into your chat client or MCP-enabled application.  
  - Guided setup per client type via the Pipedream UI.  
  - Central configuration page available for additional setup details (via Pipedream Configuration page).

- **Account Connection**  
  - Connect your FullContact account within Pipedream.  
  - Select your client to complete configuration and start using identity and enrichment tools.
