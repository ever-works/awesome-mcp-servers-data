# AskNicely MCP Server

**Category:** Business & Commerce · MCP Servers  
**Brand:** AskNicely  
**Slug:** asknicely-mcp-server  
**Source:** https://mcp.pipedream.com/app/asknicely

## Overview
AskNicely MCP Server integrates the AskNicely customer experience and survey platform into MCP-compatible environments, allowing you to connect AskNicely workflows with MCP-based chat or automation clients.

## MCP Server URL
- Static MCP server endpoint (for all clients):
  - `https://mcp.pipedream.net/v2`
- Authentication is performed when adding the server to your application.

## Features
- Integration of AskNicely’s customer experience platform with MCP-compatible tools.
- Access to survey and feedback workflows through the MCP server.
- Single static MCP endpoint usable across different clients.
- Compatible with multiple chat clients; configuration depends on the selected client.
- Central configuration documentation available via the Pipedream configuration page.

## Usage
- Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server to your chat or MCP-enabled application and authenticate during setup.
- Refer to the configuration documentation for client-specific setup steps.
