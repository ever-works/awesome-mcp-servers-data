# Gloria AI MCP Server

MCP Server for Gloria AI, providing conversational AI–driven lead qualification capabilities to MCP clients.

## Overview
The Gloria AI MCP Server connects your Gloria AI account to MCP-compatible AI tools via Pipedream. It focuses on conversational, AI-driven lead qualification for business and sales workflows.

## Features
- **MCP integration via Pipedream**: Connects Gloria AI to Pipedream’s MCP server infrastructure.
- **AI tool connectivity**: Enables use of Gloria AI with compatible AI tools, including:
  - Cursor
  - Claude
  - Windsurf
- **Conversational lead qualification**: Supports AI-driven conversations aimed at qualifying leads for sales and business workflows.
- **MCP client compatibility**: Designed to be used from MCP-enabled clients and developer tools.
