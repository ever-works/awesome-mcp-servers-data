# Nextdoor Ads MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Nextdoor  
**Website:** https://mcp.pipedream.com/app/nextdoor  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Overview
The Nextdoor Ads MCP Server exposes Nextdoor’s neighborhood advertising and related data to MCP-compatible clients. It enables applications and AI agents to interact programmatically with Nextdoor’s local advertising capabilities via a standardized MCP endpoint.

## Features
- **MCP-compatible server** for integrating Nextdoor Ads into any MCP client.
- **Single static endpoint** for all clients: `https://mcp.pipedream.net/v2`.
- **Authentication at client setup time**, handled when adding the server to your MCP-compatible application.
- **Works with multiple chat / MCP clients**, following each client’s standard MCP server configuration flow.
- **Configuration guidance available** via a dedicated configuration page (client-specific setup instructions).
- **Hosted and operated by Pipedream Connect**, removing the need to self-host the MCP server.

## Usage
- Add the MCP server URL `https://mcp.pipedream.net/v2` to your MCP-compatible chat or agent client.
- Complete authentication as prompted by your client during setup.
- Use the client’s MCP tools interface to access Nextdoor Ads–related functionality and data.
