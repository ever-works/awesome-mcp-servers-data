# Larger.io MCP Server

**Description**  
Larger.io MCP Server connects MCP-compatible tools to Larger.io for identifying and reporting the technologies used on websites. It enables in-context tech stack discovery directly from supported chat or agent clients.

**Category**  
- Web Search MCP Servers

**Tags**

**Features**  
- Integrates with Larger.io to detect technologies used on websites.  
- Returns structured information about a siteâ€™s tech stack for use inside MCP-based tools.  
- Exposes a single, static MCP server endpoint that works across clients:  
  - MCP server URL: `https://mcp.pipedream.net/v2`  
- Supports authentication at the time you add the server to your application.  
- Compatible with multiple MCP-aware chat clients and tools.  
- Configuration details and setup guidance available via a dedicated configuration page (for various supported clients).

**Usage**  
- Add the MCP server URL to your MCP-compatible client or application.  
- Authenticate when prompted during server addition.  
- Use the provided tools within your client to query website URLs and retrieve their underlying technology stack.
