# FacturaDirecta MCP Server

Invoice and accounting cloud software for companies and freelancers, exposed via an MCP (Model Context Protocol) server.

- **Name:** FacturaDirecta MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Brand:** FacturaDirecta  
- **Slug:** `facturadirecta-mcp-server`  
- **Source URL:** https://mcp.pipedream.com/app/facturadirecta

## Overview
The FacturaDirecta MCP Server provides MCP-accessible endpoints to FacturaDirecta’s cloud-based invoicing, billing, and accounting features for companies and freelancers. It is delivered via Pipedream Connect and can be integrated into compatible chat or AI clients through a single static MCP endpoint.

## Features
- **MCP-accessible API** exposing FacturaDirecta’s capabilities through the Model Context Protocol.  
- **Invoicing functions** (e.g., support for creating and managing invoices in the underlying FacturaDirecta platform).  
- **Billing features** usable via MCP endpoints tied to FacturaDirecta’s cloud billing tools.  
- **Accounting features** (e.g., access to cloud-based accounting data and workflows for companies and freelancers).  
- **Static MCP server URL** for all clients: `https://mcp.pipedream.net/v2`.  
- **Authentication at connection time**: the URL is the same for all clients; authentication occurs when adding the server to an application.  
- **Client-agnostic integration**: can be added to various chat / AI clients that support MCP.  
- **Configuration documentation** available via a central configuration page (hosted by Pipedream) for setup details.

## Technical Details
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`  
- **Provider Platform:** Pipedream Connect  
- **Usage Pattern:** Add as an MCP server in a compatible client, then authenticate and configure according to the client’s instructions.
