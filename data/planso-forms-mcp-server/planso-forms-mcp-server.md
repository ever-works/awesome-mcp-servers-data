# PlanSo Forms MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** PlanSo Forms  
**Slug:** planso-forms-mcp-server

MCP Server for PlanSo Forms that lets AI agents create and manage professional online forms by interfacing with the PlanSo Forms platform via MCP.

---

## Features

- Connects a PlanSo Forms account to AI-powered tools using Pipedream's MCP server
- Enables AI agents to create and manage professional online forms
- Interfaces directly with the PlanSo Forms platform via the Model Context Protocol (MCP)
- Integrates with AI tools such as:
  - Cursor
  - Claude
  - Windsurf
- Supports use in automated or AI-assisted workflows involving online forms and surveys

---

## Use Cases

- Building and updating online forms through AI assistants
- Managing surveys and form configurations programmatically via MCP
- Integrating PlanSo Forms into AI-driven business processes and tools
