# Copper MCP Server

## Overview
Copper MCP Server is an MCP server integration for Copper CRM that exposes Copperâ€™s CRM objects and workflows to MCP-based assistants and automations. It is provided via Pipedream Connect and accessed through a static MCP server URL.

## Features
- MCP server integration for Copper CRM
- Exposes Copper CRM objects for use in MCP-based tools and automations
- Exposes Copper CRM workflows to assistants and automation platforms that support MCP
- Single static MCP server URL usable by all clients
- Authentication handled when adding the server to an MCP-compatible application

## Technical Details
- MCP server base URL: `https://mcp.pipedream.net/v2`
- Server is added to compatible chat or automation clients as an MCP endpoint
