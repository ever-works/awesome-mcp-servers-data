# Octoparse MCP Server

## Overview
Octoparse MCP Server is an MCP-compatible integration that connects your Octoparse no-code web scraping account to AI tools such as Cursor, Claude, and Windsurf. It enables building, managing, and automating web scraping workflows from within MCP-enabled environments.

- **Category:** Content Extraction & Summarization MCP Servers  
- **Use cases:** Web scraping, no-code data collection, integrating scraping workflows into AI assistants and developer tools.

## Features
- **MCP integration:** Exposes Octoparse functionality through the Model Context Protocol (MCP), allowing AI tools and IDEs that support MCP to control scraping workflows.
- **No-code web scraping workflows:** Leverages Octoparseâ€™s no-code interface to define scraping tasks without writing custom scraper code.
- **Workflow management:** Build and manage scraping workflows centrally in Octoparse while accessing and triggering them via MCP-compatible tools.
- **Automation:** Automate execution of Octoparse scraping tasks from AI environments like Cursor, Claude, and Windsurf.
- **Data collection:** Collect structured data from websites into your existing Octoparse pipelines, then consume that data via MCP-integrated tools.
- **Multi-tool connectivity:** Designed to work with multiple AI / developer tools that implement MCP, enabling a consistent scraping interface across environments.

## Integrations
- **Octoparse** (no-code web scraping platform)
- **Cursor** (AI-augmented IDE)
- **Claude** (AI assistant)
- **Windsurf** (AI-enhanced development environment)
