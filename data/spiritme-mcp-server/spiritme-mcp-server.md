# SpiritMe MCP Server

AI-video platform integration for creating personalized videos with digital avatars via the Model Context Protocol (MCP).

## Overview
SpiritMe MCP Server connects MCP-enabled tools and AI assistants to SpiritMe’s AI video platform, allowing applications to create and manage personalized, avatar-based videos programmatically.

- **Category:** Media-processing MCP server  
- **Provider / Brand:** SpiritMe (via Pipedream Connect)  
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server**
  - Exposes SpiritMe’s AI video capabilities through the Model Context Protocol.
  - Usable from any MCP-enabled chat client or tool.

- **AI video generation with avatars**
  - Supports creation of personalized videos using digital avatars.
  - Designed for automated and programmatic video workflows.

- **Centralized static server URL**
  - Single static MCP server URL: `https://mcp.pipedream.net/v2` works across all supported clients.
  - Authentication is performed when adding/configuring the server in the client or application.

- **Client-agnostic integration**
  - Can be added to various MCP-compatible chat clients and applications.
  - Configuration guidance available via Pipedream’s configuration documentation.

- **Pipedream Connect integration**
  - Hosted and delivered via Pipedream Connect for easier setup and management.

## Usage
- Add the MCP server URL `https://mcp.pipedream.net/v2` to an MCP-enabled client.
- Authenticate within the client when prompted.
- Use the client’s MCP tooling to access SpiritMe’s avatar-based video creation and management features.
