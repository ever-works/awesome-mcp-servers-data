# Evenium MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Evenium  
**Slug:** `evenium-mcp-server`

An MCP (Model Context Protocol) server that connects AI/chat clients to Evenium’s web and mobile technologies for managing and enhancing meetings and events.

---

## MCP Server URL

Use this static MCP server endpoint in compatible clients:

```text
https://mcp.pipedream.net/v2
```

Authentication is handled when you add the server to your application.

---

## Features

- **MCP-compatible server** for integrating Evenium capabilities into AI/chat-based clients.
- **Access to Evenium web and mobile technologies** focused on:
  - Managing meetings and events.
  - Enhancing participant experiences.
  - Driving engagement during events and meetings.
- **Static server URL** usable across clients (`https://mcp.pipedream.net/v2`).
- **Client-agnostic integration**: can be added to various MCP-enabled chat clients via their configuration.
- **Configuration documentation available** via a dedicated configuration page (hosted by Pipedream Connect).
- **Hosted and operated via Pipedream Connect**, inheriting its infrastructure and platform terms/privacy controls.

---

## Usage

1. Copy the MCP server URL:
   ```text
   https://mcp.pipedream.net/v2
   ```
2. Add it to your MCP-compatible chat client.
3. Authenticate when prompted by the client.
4. Configure according to the client’s instructions or the referenced configuration page.
