## Overview

TinyFn MCP Server provides deterministic utility functions for math operations, validation, and data conversions. It gives AI agents reliable tools for tasks where hallucination is dangerous, ensuring accurate results for calculations and format conversions.

## Features

- Deterministic math calculations and unit conversions
- Validation utilities for data formats
- Reliable data conversion functions
- Prevents agent hallucination on basic computational tasks
- Provides fallback tools where LLM reasoning may produce incorrect results

## Use Cases

- Accurate unit conversions (distance, weight, temperature)
- Mathematical calculations agents cannot reliably compute
- Data validation and format checking
- Ensuring deterministic outputs for critical operations
