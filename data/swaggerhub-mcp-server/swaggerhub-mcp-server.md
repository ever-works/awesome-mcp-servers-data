# SwaggerHub MCP Server

MCP Server for SwaggerHub, providing AI agents and MCP tools with programmatic access to API design, documentation, and collaboration workflows based on OpenAPI.

- **Category:** development-tools-mcp-servers  
- **Slug:** `swaggerhub-mcp-server`  
- **Source URL:** https://mcp.pipedream.com/app/swaggerhub  
- **Brand:** SwaggerHub

## Features

- Programmatic access to SwaggerHub resources for AI agents and MCP-compatible tools.  
- Focused on **OpenAPI-based** API design and documentation workflows.  
- Supports collaboration-oriented API workflows (design, documentation, and related processes) via MCP.  
- Exposed via a single static MCP server endpoint usable from any compatible client:  
  - MCP server URL: `https://mcp.pipedream.net/v2`  
- Authentication handled when adding/configuring the server in the client (per-client auth, static URL).

## Getting Started
