# DeTrack MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** DeTrack

An MCP server integration that connects DeTrack’s real-time delivery and vehicle tracking data to AI tools for logistics workflows.

---

## Overview
The DeTrack MCP Server exposes DeTrack’s vehicle tracking and electronic proof of delivery capabilities via Pipedream’s MCP server, enabling AI tools (such as Cursor, Claude, and Windsurf) to access and use this logistics data in real time.

---

## Features
- **MCP Server Integration**
  - Runs as an MCP server via Pipedream.
  - Bridges DeTrack data into MCP-compatible AI and developer tools.

- **DeTrack Account Connectivity**
  - Connects directly to your existing DeTrack account.
  - Uses DeTrack as the source of logistics and delivery data.

- **Real-Time Vehicle Tracking Data**
  - Provides real-time vehicle tracking information from DeTrack.
  - Suitable for monitoring fleets and active deliveries programmatically.

- **Electronic Proof of Delivery (ePOD) Data Access**
  - Exposes electronic proof of delivery records from DeTrack.
  - Enables AI agents or workflows to query delivery status and confirmations.

- **AI Tool Integration**
  - Designed to work with AI tools like:
    - Cursor
    - Claude
    - Windsurf
  - Allows these tools to call DeTrack data through MCP for automation, analysis, and decision support.

- **Global Logistics Workflows**
  - Intended for use in global logistics and delivery operations.
  - Supports building AI-driven workflows around tracking, delivery confirmation, and related logistics data.
