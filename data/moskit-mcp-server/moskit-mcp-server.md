# Moskit MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Moskit  
**Source:** https://mcp.pipedream.com/app/moskit

Moskit MCP Server connects the Moskit sales CRM—designed for small and medium businesses—to the MCP framework so you can perform CRM-related operations via MCP-compatible tools and clients.

---

## Description
Moskit is a sales-focused CRM built for small and medium-sized businesses, emphasizing accessibility, ease of use, and strong customization capabilities. The Moskit MCP Server exposes Moskit’s CRM functionality through the Model Context Protocol (MCP), allowing chat clients and other MCP-compatible applications to interact with Moskit via a unified MCP server endpoint.

---

## Features
- **MCP Integration for Moskit CRM**  
  - Connects Moskit’s sales CRM to the MCP framework.  
  - Enables CRM operations to be performed through MCP tooling and chat clients.

- **Single Static MCP Endpoint**  
  - Uses a shared static MCP server URL: `https://mcp.pipedream.net/v2`.  
  - The same URL works for all clients; authentication is handled when adding the server to your application.

- **Sales-Focused CRM for SMBs**  
  - Built specifically for sales workflows.  
  - Targeted at small and medium businesses.  
  - Designed to be accessible and easy to use.  
  - Offers high customization potential within Moskit CRM (e.g., adaptable sales processes and configurations).

- **Client-Agnostic Setup**  
  - Can be added to different MCP-compatible chat clients.  
  - Configuration guidance available via a dedicated configuration page.

- **Operated via Pipedream Connect**  
  - MCP server is hosted and powered by Pipedream Connect infrastructure.

---

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Authentication:** Performed when adding the server in your MCP-compatible application or chat client.  
- **Integration Model:** Central MCP server that exposes Moskit CRM capabilities as tools/endpoints to MCP clients.

---

## Setup
1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.  
2. Add a new MCP server in your MCP-compatible chat client or application.  
3. Use the copied URL as the server endpoint.  
4. Authenticate when prompted by your client during setup.
