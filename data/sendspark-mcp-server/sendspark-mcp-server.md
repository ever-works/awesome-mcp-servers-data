# Sendspark MCP Server

**Category:** Business & Commerce – MCP Servers  
**Slug:** sendspark-mcp-server  
**Brand:** Sendspark  
**Source:** https://mcp.pipedream.com/app/sendspark

## Overview
Sendspark MCP Server is an MCP integration for Sendspark’s video outreach platform. It allows MCP-compatible agents and chat clients to use personalized video messages as part of outreach and meeting workflows.

The server is hosted via Pipedream and exposes Sendspark capabilities to MCP-based applications through a static MCP server URL.

## Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same URL works for all clients; authentication is handled during server addition to the app.

- **Account configuration**  
  - Connect a Sendspark account through the Pipedream UI.  
  - "Configure Sendspark" flow to link your account and select a client.

- **Client integration**  
  - Instructions to add the MCP server to various chat clients.  
  - Option to consult a full configuration page for detailed setup.

- **Tool exposure to MCP agents**  
  - Server exposes Sendspark “tools” (actions) to MCP agents (listed as “Available tools”).  
  - Tools load dynamically within the Pipedream interface (e.g., “Loading actions…”, “Loading available tools…”), indicating support for multiple Sendspark-related actions once configured.

- **Use cases (implied)**  
  - Use personalized video messages for:  
    - Sales outreach  
    - Scheduling more meetings  
    - Revenue-focused communication workflows
