## Hystruct MCP Server

**Description**  
Hystruct MCP Server exposes Hystruct’s AI-powered web scraping and structured data extraction capabilities through the Model Context Protocol (MCP). It allows you to connect your Hystruct account to compatible AI tools.

**Key Details**
- **Category:** Content Extraction & Summarization MCP Servers  
- **Brand:** Hystruct  
- **Source:** [Pipedream Hystruct MCP App](https://mcp.pipedream.com/app/hystruct)

### Features
- AI-powered web scraping via Hystruct
- Structured data extraction from web pages
- MCP-compatible server implementation
- Connects a Hystruct account to AI tools through Pipedream’s MCP server
- Integration support for AI coding and assistant tools such as:
  - Cursor
  - Claude
  - Windsurf
- Enables AI tools to request and consume structured web data in context

### Integrations
- **Pipedream MCP Server** (host environment)
- **AI Tools:** Cursor, Claude, Windsurf (and other MCP-compatible clients)
