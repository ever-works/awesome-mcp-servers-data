# Dialer MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Dialer  
**Source:** https://metorial.com/integration/dialer

The Dialer MCP Server exposes Dialer’s outbound calling capabilities to MCP-aware agents, allowing AI workflows to initiate and manage phone calls over OAuth 2.1–authenticated connections.

![Dialer](https://getdialer.app/og-image.png)

---

## Features

- **Outbound call initiation**  
  - Start phone calls directly from AI agents and automated workflows.

- **Call management**  
  - Manage active calls through MCP tools within your workflows (e.g., control call lifecycle, integrate into larger automations).

- **Real-time call status monitoring**  
  - Track call progress and status updates during execution of workflows.

- **Call logging**  
  - Automatically log call activity for downstream processing, auditing, or analytics.

- **OAuth 2.1 authentication**  
  - Securely connect AI agents to Dialer using OAuth 2.1.

- **MCP integration**  
  - Exposed as an MCP server at `https://getdialer.app/sse`, usable by MCP-aware agents and platforms such as Metorial.

- **Workflow integration via Metorial**  
  - Deployable on Metorial to plug Dialer into broader AI workflows and combine it with other integrations (e.g., Slack, Gmail, WhatsApp, calendar, search tools).

---

## Technical Details

- **MCP Server Endpoint:** `https://getdialer.app/sse`  
- **Auth:** OAuth 2.1  
- **Intended use:** Programmatic outbound telephony and call management from AI/agent workflows.
