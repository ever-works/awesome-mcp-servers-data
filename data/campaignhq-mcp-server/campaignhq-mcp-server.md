# CampaignHQ MCP Server

**Category:** Business & Commerce – MCP Servers  
**Website:** https://mcp.pipedream.com/app/campaignhq

## Overview
CampaignHQ MCP Server connects the CampaignHQ email marketing platform to Model Context Protocol (MCP)–compatible applications. It exposes CampaignHQ capabilities as tools that can be invoked from chat clients or other MCP-aware environments.

## Features
- **MCP integration endpoint**
  - Static MCP server URL: `https://mcp.pipedream.net/v2`
  - Single URL works for all CampaignHQ clients; authentication is handled when adding the server to an MCP application.
- **CampaignHQ account connection**
  - Connect a CampaignHQ account via the Pipedream-hosted configuration flow.
  - Select a specific client/account after connecting to target the right CampaignHQ workspace.
- **Tool-based access to CampaignHQ**
  - Exposes CampaignHQ functionality as "available tools" within the MCP ecosystem (exact tools are dynamically loaded in the interface).
  - Designed to integrate CampaignHQ’s email marketing and campaign capabilities into chat clients and other MCP-compatible apps.
- **Central configuration page**
  - Dedicated configuration page (via Pipedream) to review setup details and instructions for various chat clients.

## Usage
1. Go to the CampaignHQ MCP Server page on Pipedream.  
2. Connect your CampaignHQ account and choose the appropriate client.  
3. Copy the MCP server URL `https://mcp.pipedream.net/v2`.  
4. Add this URL as an MCP server in your chat client or MCP-compatible application.  
5. Authenticate when prompted and start invoking the exposed CampaignHQ tools.
