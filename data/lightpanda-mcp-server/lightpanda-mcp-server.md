**Category:** Web Search MCP Servers

## Description
Lightpanda MCP Server exposes a headless Chrome API and a tailored browser environment for browser automation through the Model Context Protocol (MCP). It is designed to provide efficient, high-performance headless browsing and automation capabilities to MCP-compatible clients.

## Features
- **Headless Chrome API**: Programmatic control of a headless Chrome instance for automation and scraping.
- **Tailored browser environment**: Custom, purpose-built browser setup optimized for automation workloads.
- **Browser automation**: Support for scripted navigation, interaction, and other automated browser tasks.
- **Web scraping support**: Suitable for extracting data from websites via automated browsing.
- **Headless operation**: Runs without a visible UI, optimized for server-side and background tasks.
- **MCP integration**: Exposes capabilities as an MCP server, enabling use from any compatible MCP client.
- **Static MCP server URL**: A single, static endpoint usable across clients: `https://mcp.pipedream.net/v2`.
- **Client-agnostic setup**: Works with multiple MCP-enabled chat or agent clients via the same URL.

## Configuration
