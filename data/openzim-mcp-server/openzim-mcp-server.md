## Overview

OpenZim MCP Server enables AI agents to query local Wikipedia dumps and ZIM archives without requiring internet access. It provides full-text search over offline knowledge bases, solving the need for agents to access comprehensive reference data without complex infrastructure.

## Features

- Query local Wikipedia dumps and ZIM archives
- Full-text search capabilities over offline content
- No internet connection required after initial download
- Works with a local ~110GB Wikipedia copy
- Pragmatic alternative to building custom RAG retrieval pipelines
- Most upvoted recommendation on r/LocalLLaMA's most useful server thread

## Use Cases

- Offline knowledge retrieval for AI agents
- Privacy-preservised search without sending queries to external APIs
- Alternative to building complex retrieval-augmented generation setups
