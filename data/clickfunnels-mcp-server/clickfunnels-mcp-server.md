# ClickFunnels MCP Server

An MCP Server integration for ClickFunnels, enabling MCP agents to interact with and automate marketing funnel operations within ClickFunnels.

**Website:** https://mcp.pipedream.com/app/clickfunnels  
**Category:** Business & Commerce – MCP Servers  
**Tags:** marketing, sales, automation  
**Provider / Infrastructure:** Powered by Pipedream Connect

---

## Features

- **MCP Server for ClickFunnels**
  - Exposes ClickFunnels functionality via the Model Context Protocol (MCP), allowing compatible agents or chat clients to work with ClickFunnels data and actions.
- **Static MCP Endpoint**
  - Single universal MCP server URL: `https://mcp.pipedream.net/v2` used for all clients.
  - Authentication occurs when adding the server to your MCP-compatible application or chat client.
- **Client-Agnostic Integration**
  - Designed to be added to various MCP-capable chat clients.
  - Central configuration guidance available via the linked configuration page (how to register the server, set credentials, etc.).
- **Infrastructure & Compliance**
  - Runs on Pipedream Connect infrastructure.
  - Governed by Pipedream’s Terms and Privacy Policy.

## Setup & Configuration

1. Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add this server URL to your MCP-compatible app or chat client.
3. Complete authentication when prompted by your client.
4. Optionally refer to the full configuration documentation at the linked Configuration page on Pipedream.
