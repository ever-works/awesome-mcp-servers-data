# GorillaStack MCP Server

**Category:** Cloud & DevOps MCP Servers  
**Brand:** GorillaStack  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

GorillaStack MCP Server is an integration that exposes GorillaStack’s public cloud automation rules engine to MCP-compatible clients via a static Pipedream-hosted endpoint. It focuses on operational automation, cost optimization, and backup management workflows for DevOps teams.

## Features

- **MCP-compatible server endpoint**  
  - Provides a static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Single URL usable across all compatible MCP clients

- **GorillaStack cloud automation integration**  
  - Connects to GorillaStack’s public cloud automation rules engine  
  - Designed around DevOps operational needs  
  - Targets cost optimization use cases in public cloud environments  
  - Supports backup management workflows via GorillaStack

- **Client-agnostic configuration**  
  - Works with multiple chat or MCP-capable applications  
  - Authentication is performed when adding the server within the chosen client  
  - Can be configured following the general MCP configuration guidance (via the referenced configuration page)

- **Pipedream-hosted infrastructure**  
  - MCP server infrastructure is provided by Pipedream Connect  
  - Uses a managed, shared endpoint rather than requiring self-hosting of the MCP server

## Authentication

- Authentication occurs when adding the MCP server to the client application, using the static URL `https://mcp.pipedream.net/v2`.
