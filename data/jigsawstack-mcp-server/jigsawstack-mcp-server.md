# JigsawStack MCP Server

**Category:** AI Integration – MCP Servers  
**Brand:** JigsawStack  
**Source:** https://mcp.pipedream.com/app/jigsawstack

## Overview
JigsawStack MCP Server exposes JigsawStack’s AI SDK capabilities through the Model Context Protocol (MCP), allowing integration with any MCP-compatible client or tech stack.

## Features
- **MCP-compatible AI SDK access**: Provides access to JigsawStack’s AI SDK from any stack that supports MCP.
- **Static MCP server endpoint**: Uses a single, static URL for all clients:
  - `https://mcp.pipedream.net/v2`
- **Client-agnostic integration**: Designed to work with multiple chat clients and MCP-enabled applications.
- **Configuration guidance**: Documentation available via a dedicated Configuration page for setup in different clients.
- **Hosted by Pipedream**: Delivered via Pipedream Connect infrastructure.

## Usage
- Add the MCP server URL `https://mcp.pipedream.net/v2` to your MCP-compatible client or application.
- Authenticate during server addition/configuration as required by your client.
