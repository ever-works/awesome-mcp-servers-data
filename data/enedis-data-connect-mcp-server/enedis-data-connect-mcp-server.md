# Enedis Data-Connect MCP Server

**Description**  
The Enedis Data-Connect MCP Server exposes Enedis energy data services to Model Context Protocol (MCP) clients, enabling integration of Enedis energy and open data into energy-related applications and workflows.

## Features

- **MCP-compatible server**  
  - Implements an MCP server endpoint that can be added to any compatible MCP client or chat application.

- **Static server URL**  
  - Single shared endpoint for all clients:  
    - `https://mcp.pipedream.net/v2`
  - Authentication is handled when the server is added/connected in the client.

- **Access to Enedis energy data services**  
  - Exposes Enedis Data-Connect services intended for energy service providers.  
  - Provides data services designed to support energy-related activities and use cases (e.g., integrating energy usage or grid data into applications and workflows).

- **Integration via Pipedream Connect**  
  - Hosted and delivered through Pipedream’s "Connect" infrastructure.  
  - Can be configured from the Pipedream environment and associated configuration pages.

- **Client-agnostic usage**  
  - The same MCP endpoint can be consumed by multiple MCP-aware chat clients and tools.  
  - Configuration documentation and examples available via the platform’s Configuration page.

---

## Configuration & Usage

- **MCP server URL**: `https://mcp.pipedream.net/v2`  
- **Setup**: Add this URL as an MCP server in your MCP-compatible app or chat client; complete authentication during server addition.  
- **Further configuration details**: Available via the platform’s Configuration page (linked from the app listing).
