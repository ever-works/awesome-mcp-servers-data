## Google Maps MCP Server

**Description**

An MCP (Model Context Protocol) server that wraps Google Maps functionality, providing location search, directions, and place details for geography-aware tasks.

### Features

- **Location search** using Google Maps data.  
- **Directions and routing** between locations.  
- **Place details** lookup (e.g., information about specific locations).  
- **Geography-aware task support** via an MCP server interface, enabling tools to query map-related data programmatically.

---

### Use Cases

- Adding location search and directions to MCP-compatible AI assistants.  
- Retrieving detailed information about places within geography-aware workflows.  
- Building tools that need map-based context or routing information.

---

### Integration

- Exposed as an **MCP server**, suitable for integration into clients and frameworks that support the Model Context Protocol.  
- Accessed programmatically via the MCP interface defined in the `anthropic-mcp-servers` repository.
