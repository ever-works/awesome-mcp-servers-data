# Pingdom MCP Server

**Description**  
An MCP Server integration for SolarWinds Pingdom that lets MCP‑compatible clients access website uptime, performance, real user, page speed, and synthetic monitoring data for web applications.

**Category:** Monitoring  
**Brand:** SolarWinds Pingdom  
**Slug:** `pingdom-mcp-server`  
**Source URL:** https://mcp.pipedream.com/app/pingdom

## Features
- MCP server endpoint for Pingdom: static MCP URL `https://mcp.pipedream.net/v2` usable across compatible clients.
- Access to SolarWinds Pingdom monitoring capabilities via MCP-compatible applications, including:
  - Website uptime monitoring
  - Web performance/page speed monitoring
  - Real user monitoring for web applications
  - Synthetic monitoring from outside the firewall
- Works with multiple chat or MCP clients by adding the same server URL and authenticating per client.

## Integration & Usage
- Add the MCP server URL `https://mcp.pipedream.net/v2` to any supported MCP/chat client.
- Authenticate within the client after adding the server.
- Configuration guidance available via the provider’s configuration documentation (linked from the source page).
