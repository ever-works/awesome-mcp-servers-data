# Bruzu MCP Server

**Category:** Media Processing MCP Servers  
**Brand:** bruzu  
**Source:** https://mcp.pipedream.com/app/bruzu

## Overview
Bruzu MCP Server is an MCP-compatible integration for Bruzu’s Simple Image API. It enables image generation and manipulation workflows through the Model Context Protocol using a single static server URL provided by Pipedream.

## Features
- **MCP-compatible server** for integrating Bruzu’s Simple Image API with MCP-enabled chat clients and applications.
- **Static MCP endpoint** for all clients:
  - Base URL: `https://mcp.pipedream.net/v2`
  - Same URL is used across different clients; authentication is handled when adding the server to an application.
- **Image generation and manipulation support** via Bruzu’s Simple Image API (e.g., creating, editing, or transforming images programmatically through MCP tools).
- **Client-agnostic configuration**:
  - Designed to be added to various chat clients that support MCP.
  - Central configuration reference available via the linked Configuration page on Pipedream.
- **Hosted by Pipedream Connect**:
  - Managed MCP server infrastructure provided by Pipedream.
  - Uses Pipedream’s platform policies (Terms, Privacy, Cookies) for the hosted endpoint.
