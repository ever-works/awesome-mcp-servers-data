# mcp-server-playwright

An MCP server package that exposes Playwright browser automation and testing capabilities as MCP tools, allowing editors like VS Code to run Playwright commands via the Model Context Protocol within a development environment or dev container.

[https://playbooks.com/mcp/vikashloomba/mcp-server-playwright](https://playbooks.com/mcp/vikashloomba/mcp-server-playwright)
