# Eagle Doc MCP Server

**Category:** Content Extraction & Summarization MCP Servers  
**Tags:** OCR, invoices, document-processing

## Overview
Eagle Doc MCP Server exposes Eagle Doc’s AI-powered invoice and receipt OCR and document processing capabilities via the Model Context Protocol (MCP). It is designed for fast, reliable, and accurate extraction of structured data from financial documents, enabling seamless integration into MCP-compatible chat clients and applications.

## Features
- **MCP-compatible server**: Provides an MCP server endpoint that can be added to any compatible client or application.
- **Static server URL**: Uses a single static URL for all clients: `https://mcp.pipedream.net/v2`.
- **Authentication on connect**: Authentication is handled when adding the server to your application or client.
- **Invoice OCR**: AI-powered optical character recognition tailored for invoices.
- **Receipt OCR**: AI-powered OCR optimized for receipts.
- **Document processing APIs**: Exposes APIs for document parsing and processing through MCP.
- **Focus on speed and reliability**: Designed for fast and reliable document OCR and extraction workflows.
- **Structured data extraction**: Extracts key information from financial documents for downstream use in applications.
- **Seamless integration**: Built to integrate into existing apps and chat clients that support MCP, including via Pipedream Connect.

## Integration & Usage
- **MCP Server URL:** `https://mcp.pipedream.net/v2` (shared static URL for all clients).
- **Client setup:** Add the server URL in your MCP-compatible chat client or application and authenticate during setup.
- **Configuration docs:** Additional configuration details are available on the platform’s configuration page (linked from the app).
