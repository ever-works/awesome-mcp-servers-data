# Frappe MCP Server

**Website:** https://mcp.pipedream.com/app/frappe  
**Category:** Business & Commerce – MCP Servers  
**Tags:** erp, framework, workflow

## Overview
Frappe MCP Server is an MCP server hosted on Pipedream that connects MCP-compatible tools and clients to the Frappe framework and its software products, enabling ERP and application workflows via the MCP protocol.

## Features
- **MCP server endpoint**  
  - Static MCP server URL usable across clients: `https://mcp.pipedream.net/v2`.
  - URL can be added to any compatible MCP client or chat application.

- **Frappe integration**  
  - Connects MCP tools to the Frappe framework and its suite of products and services.  
  - Designed to support ERP and app-related workflows via MCP.

- **Client configuration**  
  - “Configure Frappe” flow to connect a Frappe account.  
  - User selects their client (e.g., chat client) to receive specific setup instructions.  
  - Central configuration documentation available via a dedicated configuration page.

- **Authentication**  
  - Authentication occurs when the MCP server is added to an application, rather than per-client URL changes.

- **Tool access**  
  - Exposes “available tools” and actions over MCP (tool list is dynamically loaded at runtime).
