# Pexels MCP Server

Free stock photo search and retrieval via MCP.

**Website:** https://mcp.pipedream.com/app/pexels  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Description
Pexels MCP Server is an MCP-compatible service that lets tools and chat-based clients search and retrieve free stock photos from Pexels programmatically through a single static MCP endpoint.

## Features
- MCP-compatible server for integrating Pexels into MCP-enabled clients
- Programmatic access to free stock photos from Pexels
- Static MCP server URL (`https://mcp.pipedream.net/v2`) usable across supported clients
- Authentication handled when adding the server to your application
- Configuration guidance available via a dedicated configuration page
