# Coassemble MCP Server

**Category:** Documentation & Learning Resources  
**Tags:** LMS, Training, Courses

Coassemble MCP Server is an MCP integration for the Coassemble online training platform, allowing MCP-aware agents and AI tools to programmatically create and manage SaaS-focused training experiences.

## Overview
- Connects a Coassemble account to AI tools via Pipedream’s MCP server.
- Designed for programmatic management of training and course content.
- Focused on SaaS-oriented training experiences.

## Features
- **MCP Integration**: Exposes Coassemble functionality through the Model Context Protocol (MCP) so MCP-aware agents can interact with Coassemble.
- **AI Tool Connectivity**: Works with AI tools such as Cursor, Claude, and Windsurf via Pipedream’s MCP server.
- **Programmatic Course Management**: Intended to enable creation and management of training experiences in Coassemble through automated or AI-driven workflows.
- **SaaS-Focused Training**: Optimized for building and managing training experiences centered on SaaS products and workflows.
