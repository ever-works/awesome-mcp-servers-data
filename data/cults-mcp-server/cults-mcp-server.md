# Cults MCP Server

## Overview
Cults MCP Server is an MCP-compatible integration that exposes the Cults 3D-printing model marketplace to MCP clients. It enables discovery and retrieval of 3D printer files and related metadata from Cults within tools that support the Model Context Protocol.

- **Category:** Content Management MCP Servers  
- **Brand:** Cults  
- **Source URL:** https://mcp.pipedream.com/app/cults  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **Access to Cults 3D model marketplace**  
  - Connects MCP-compatible clients to the Cults marketplace of 3D printer files.  
  - Supports discovery of 3D models for 3D printing projects.

- **Metadata retrieval**  
  - Exposes related metadata for 3D-printing models (such as descriptive information associated with the files) via the MCP interface.

- **Static MCP server endpoint**  
  - Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) that works across supported clients.  
  - Authentication occurs when adding the server to the client/application.

- **Client-agnostic integration**  
  - Designed to be used from multiple MCP-compatible chat or AI clients.  
  - Configuration instructions are available per client via the configuration page.

- **Pipedream-hosted**  
  - Operated and delivered via Pipedream’s MCP infrastructure.

## Configuration
- Connect your Cults account through the Pipedream interface.  
- Copy the MCP server URL and add it to your MCP-compatible client.  
- Authenticate within your client when prompted.
