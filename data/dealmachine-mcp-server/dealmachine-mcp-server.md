# DealMachine MCP Server

**Category:** Business & Commerce – MCP Servers  
**Slug:** dealmachine-mcp-server  
**Tags:** real-estate, marketing, lead-generation

## Overview
DealMachine MCP Server exposes DealMachine’s real estate investor tools to MCP-compatible clients. It focuses on helping single-family real estate investors discover more property deals at lower cost using direct-to-seller marketing capabilities.

## Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL that can be used across supported MCP clients:  
    - `https://mcp.pipedream.net/v2`  
  - Authentication is handled when adding the server within each client.

- **Direct-to-seller marketing tooling (via DealMachine platform)**  
  - Access to DealMachine’s suite of tools oriented around direct outreach to property owners.  
  - Optimized for single-family real estate investors seeking motivated sellers and better deal flow.

- **Client-agnostic integration**  
  - Works with any MCP-compatible chat or agent client.  
  - Configuration guidance available via a central configuration page (per Pipedream Connect docs).

- **Hosted by Pipedream Connect**  
  - Operated as a managed MCP server hosted by Pipedream.  
  - Centralized terms, privacy, and cookie settings via Pipedream.

## Integration & Setup
- Use the universal MCP server URL: `https://mcp.pipedream.net/v2`.  
- Add the server within your chosen MCP client and complete its authentication flow.  
- Optional: Refer to the provider’s full configuration documentation for detailed client-specific setup steps.
