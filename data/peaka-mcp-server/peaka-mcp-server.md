# Peaka MCP Server

Zero-ETL data integration MCP server for accessing unified data via MCP clients.

**Website:** https://mcp.pipedream.com/app/peaka  
**MCP Server URL:** `https://mcp.pipedream.net/v2`  
**Category:** Data Access & Integration – MCP Servers  
**Tags:** data-integration, zero-etl, data-virtualization

---

## Description
Peaka MCP Server is an MCP (Model Context Protocol) server integration for Peaka’s zero-ETL data integration platform. It allows MCP-compatible clients to connect to Peaka and work with unified data across multiple sources without traditional ETL pipelines.

---

## Features
- **MCP-compatible server endpoint**  
  - Single static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Works with any supported MCP client.

- **Zero-ETL data integration**  
  - Connects to Peaka’s data integration platform without requiring conventional ETL workflows.  
  - Enables querying and working with data where it resides (data virtualization approach).

- **Unified data access**  
  - Provides a unified interface to data from multiple underlying sources via Peaka.  
  - Designed for use within chat or agent-style clients that implement MCP.

- **Client-agnostic configuration**  
  - Same static URL used across different MCP clients.  
  - Authentication is handled when adding the server in each client.

- **Hosted by Pipedream Connect**  
  - MCP server delivered via Pipedream’s infrastructure.  
  - Central configuration and management through Pipedream’s MCP configuration pages.

---

## Setup & Integration
- Use the MCP server URL `https://mcp.pipedream.net/v2` when adding a new MCP server in your client.  
- Authenticate within your chosen MCP-compatible application when prompted.  
- Refer to the client-specific instructions or the general configuration page on Pipedream for detailed setup steps.
