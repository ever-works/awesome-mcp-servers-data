# Tally MCP Server

**Category:** Business & Commerce · MCP Servers  
**Tags:** forms, surveys, data-collection

## Overview
Tally MCP Server exposes Tally’s online form and survey capabilities through an MCP-compatible endpoint, allowing you to create and manage forms and surveys programmatically via MCP-enabled clients.

- **Name:** Tally MCP Server  
- **Provider:** Pipedream (powered by Pipedream Connect)  
- **Brand:** Tally  
- **MCP Endpoint URL:** `https://mcp.pipedream.net/v2`

## Features
- MCP-compatible server for integrating Tally with MCP-enabled chat or automation clients.  
- Programmatic access to Tally form and survey functionality (via MCP tools exposed by the server).  
- Single static MCP server URL (`https://mcp.pipedream.net/v2`) usable across different clients.  
- Authentication handled when adding the server to your MCP client or application.  
- Works with configuration documented on the associated Configuration page (for setup in different chat clients / tools).
