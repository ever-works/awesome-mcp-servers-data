# Nicereply MCP Server

**Category:** Business & Commerce – MCP Servers  
**Tags:** customer-feedback, surveys, customer-experience

Nicereply MCP Server integrates Nicereply’s customer satisfaction and feedback tools with MCP-compatible AI agents, enabling them to read and act on customer feedback metrics such as CSAT and NPS.

---

## Features

- **MCP Integration**  
  - Exposes Nicereply data and actions via the Model Context Protocol (MCP).  
  - Allows AI agents and MCP-enabled chat clients to work directly with Nicereply feedback.

- **Customer Feedback Metrics**  
  - Access to CSAT (Customer Satisfaction) scores.  
  - Access to NPS (Net Promoter Score) and other customer feedback metrics (as supported by Nicereply).  
  - Enables workflows that respond to specific customer feedback in the short term and inform longer-term CX improvements.

- **Unified MCP Server Endpoint**  
  - Single static MCP server URL for all clients:  
    - `https://mcp.pipedream.net/v2`  
  - Authentication handled when adding/configuring the server in your MCP client or application.

- **Client-Agnostic Setup**  
  - Works with multiple MCP-compatible chat clients.  
  - Configuration guidance available per client via the MCP configuration page.

- **Pipedream Connect Backend**  
  - Powered by Pipedream Connect’s infrastructure for MCP servers.

---

## Setup

1. Use the MCP server URL: `https://mcp.pipedream.net/v2`.  
2. Add the Nicereply MCP Server to your MCP-compatible app or chat client.  
3. Authenticate when prompted in your client’s configuration flow.  
4. (Optionally) Refer to the full configuration documentation on the Pipedream MCP configuration page.
