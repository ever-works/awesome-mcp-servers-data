# Mailgun MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Mailgun  
**Source:** https://mcp.pipedream.com/app/mailgun

## Overview
Mailgun MCP Server is an MCP (Model Context Protocol) server integration that exposes Mailgun’s email automation and transactional email APIs as tools usable by MCP-based clients and chat applications. It lets you send, receive-related, and manage emails and mailing lists via structured MCP actions.

## Features

### MCP Server & Configuration
- Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`
- Can be added to any compatible chat client or MCP-based application
- Authentication handled when adding the server to your application

### Email Operations
- **Send Email**  
  - Send transactional or automated emails through Mailgun’s API.

- **Verify Email**  
  - Verify email address deliverability using Mailgun’s verification endpoint.

- **Suppress Email**  
  - Add an email address to the Mailgun suppression list to prevent further sends.

### Mailing List Management
- **Create Mailing List Member**  
  - Add an email address to an existing Mailgun mailing list.

- **Get Mailing List Member**  
  - Retrieve a mailing list member by email address.

- **Get Mailing List Members**  
  - List all members of a specified mailing list.

- **Delete Mailing List Member**  
  - Remove a mailing list member by email address.

### Routing & Domains
- **Create Route**  
  - Create a new Mailgun route for handling inbound email.

- **List Domains**  
  - List all domains configured in the connected Mailgun account.
