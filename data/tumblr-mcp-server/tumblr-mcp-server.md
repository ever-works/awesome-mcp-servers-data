# Tumblr MCP Server

An MCP (Model Context Protocol) server that connects to Tumblr, enabling MCP-compatible clients to post, retrieve, and interact with Tumblr content and communities.

## Features

- Connects MCP-compatible clients to Tumblr
- Supports posting Tumblr content via MCP
- Supports retrieving Tumblr content via MCP
- Enables interaction with Tumblr content and communities from MCP clients
- Uses a single static MCP server URL for all clients: `https://mcp.pipedream.net/v2`
- Authentication handled when adding the server to your application

## Setup

- MCP server base URL: `https://mcp.pipedream.net/v2`
