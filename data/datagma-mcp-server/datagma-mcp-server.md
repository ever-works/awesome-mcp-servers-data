## Overview
Datagma MCP Server is an MCP-compatible service that enriches incomplete B2B prospect and company records with more than 50 additional attributes. It is designed to plug into MCP-aware chat clients and workflows to improve the quality and completeness of B2B lead and company data.

## Features
- **B2B data enrichment** for prospects and target companies
- **>50 additional attributes** added to incomplete records (e.g., extended firmographic and contact-related fields)
- **MCP-compatible server** that integrates with MCP-enabled chat clients and tools
- **Single static MCP endpoint** used across clients: `https://mcp.pipedream.net/v2`
- **Works with multiple MCP clients**: server URL can be added to any supported chat client
- **Workflow-ready**: intended for use in automated, MCP-based enrichment workflows

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Authentication:** Performed when adding the server in the client (static URL is shared across clients)
