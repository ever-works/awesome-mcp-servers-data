## Overview
The NextDNS MCP Server integrates NextDNS with AI agents and MCP-compatible clients. It exposes NextDNSâ€™ DNS security, ad-blocking, tracker blocking, and parental control capabilities so they can be managed and queried programmatically through the MCP protocol.

## Features
- **MCP integration for NextDNS**
  - Provides a Machine Control Protocol (MCP) server endpoint for interacting with NextDNS.
  - Uses a single static server URL (`https://mcp.pipedream.net/v2`) that works across supported MCP clients.
  - Authentication is handled when the server is added to an MCP-compatible application.

- **Security protections (via NextDNS)**
  - Protects against a wide range of Internet security threats at the DNS level.
  - Blocks access to known malicious domains.

- **Ad and tracker blocking**
  - Blocks ads across websites and applications.
  - Blocks trackers to enhance privacy on all devices.

- **Parental controls & safe browsing**
  - Provides tools for a safe and supervised Internet experience for kids.
  - Can be applied across devices and networks through DNS-based controls.

- **Device- and network-wide coverage**
  - DNS-based filtering that can protect all devices and all networks configured to use NextDNS.

- **Client-agnostic MCP endpoint**
  - Same MCP server URL for every MCP-compatible client.
  - Configuration performed from within the chosen chat or MCP client by adding this server.

## Usage
- Configure your MCP-compatible chat or AI client to use the MCP server URL: `https://mcp.pipedream.net/v2`.
- Authenticate when prompted by your client to allow it to manage and query your NextDNS configuration.
