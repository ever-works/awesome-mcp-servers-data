# Workiz MCP Server

**Description**  
An MCP (Model Context Protocol) server that connects AI tools to Workiz’s field service management platform, enabling access to scheduling, jobs, and operations data for field service businesses (e.g., HVAC, plumbing, garage door, electrical contracting).

**Category**  
- Business & Commerce MCP Servers

**Provider / Brand**  
- Workiz  
- Delivered via Pipedream Connect

**MCP Server URL**  
- Base MCP Server URL (static for all clients):  
  `https://mcp.pipedream.net/v2`

You authenticate when adding the server to your MCP-compatible application.

## Features
- **Workiz integration for AI tools**: Provides AI agents and MCP-compatible clients with access to Workiz’s field service management capabilities.
- **Scheduling access**: Interface with Workiz’s scheduling features for technicians and jobs (e.g., viewing and managing scheduled work).
- **Jobs and operations access**: Connect to job and operations data in Workiz, enabling workflows around job management and field operations.
- **Field service business focus**: Tailored for businesses such as HVAC, plumbing, garage door, and electrical contracting.
- **Standard MCP endpoint**: Uses a single, static MCP endpoint (`/v2`) that can be reused across clients, with authentication handled at client setup.
