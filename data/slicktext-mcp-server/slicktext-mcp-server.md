# SlickText MCP Server

An MCP server integration for SlickText, enabling SMS and text message marketing features within MCP-compatible applications.

## Overview
- **Type:** MCP server / integration
- **Category:** Messaging MCP Servers
- **Brand:** SlickText
- **Slug:** `slicktext-mcp-server`
- **Base MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Source / Documentation:** https://mcp.pipedream.com/app/slicktext

## Features
- Connects MCP-based applications to SlickText’s SMS marketing platform.
- Supports running SMS campaigns from compatible chat or MCP clients.
- Enables text-message-based customer engagement flows.
- Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) for all clients.
- Authentication handled when adding the server to each client/application.
- Configurable via Pipedream’s MCP configuration flow (see configuration page on source site).

## Integration & Setup
- Add the MCP server URL to your MCP-compatible chat client or application.
- Authenticate within the client as prompted.
- Additional configuration details are available on the Pipedream configuration page (linked from the source URL).
