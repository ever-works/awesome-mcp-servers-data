# Recruitee MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Recruitee (by Tellent)  
**Type:** MCP server integration for collaborative hiring / ATS

## Overview
The Recruitee MCP Server (by Tellent) connects Recruitee’s collaborative hiring and applicant tracking system (ATS) capabilities to MCP-compatible clients and workflows via Pipedream Connect.

## Features
- **MCP-Compatible Server**
  - Exposes Recruitee hiring and ATS functionality through the Model Context Protocol (MCP).
  - Can be added as a server to any compatible chat or AI client.

- **Static MCP Endpoint**
  - Uses a single static server URL for all clients:  
    `https://mcp.pipedream.net/v2`
  - Authentication is performed when adding the server to your application.

- **Integration via Pipedream Connect**
  - Operates as a Pipedream-powered MCP server, enabling integration into workflows built on Pipedream Connect.

- **Collaborative Hiring / ATS Connectivity**
  - Designed to bridge Recruitee’s collaborative hiring and ATS features with MCP-based tools and automations (e.g., AI assistants, chat clients, or other MCP-compatible systems).

## Setup
- Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server in your MCP-compatible client and complete authentication during setup.
- Optionally refer to the provider’s configuration documentation ("Configuration" page) for client-specific setup steps.
