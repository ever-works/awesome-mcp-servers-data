# Northflank MCP Server

## Overview
The Northflank MCP Server is an MCP (Model Context Protocol) server that connects AI agents to Northflank’s self-service developer platform. It enables interaction with deployments, services, and workloads managed on Northflank through a standardized MCP endpoint.

- **Category:** Cloud & DevOps MCP Server  
- **Provider / Brand:** Northflank (via Pipedream)  
- **MCP Endpoint URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible endpoint**  
  - Provides a static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Designed to be added to any compatible chat client or MCP-capable application.

- **Northflank platform integration**  
  - Connects to a Northflank account (through Pipedream) to enable AI-driven operations on deployments, services, and workloads.  
  - Targets infrastructure and cloud automation scenarios.

- **Client-agnostic configuration**  
  - Same static URL works for every supported client.  
  - Authentication is handled when adding the server to the client / application.

- **Self-service setup flow**  
  - Connect a Northflank account via the Pipedream interface.  
  - Choose a chat client and follow client-specific instructions to add the server.  
  - Option to reference a general configuration guide (Pipedream configuration page).

> Note: The page references “Available tools / actions” that are dynamically loaded, but the specific tools and operations are not listed in the provided content.

## Usage
1. Configure your Northflank account connection via Pipedream.  
2. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.  
3. Add this URL as an MCP server in your supported chat client or application.  
4. Authenticate when prompted to allow the MCP server to access your Northflank resources.
