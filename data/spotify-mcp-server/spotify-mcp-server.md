# Spotify MCP Server

**Type:** Open-source MCP server / integration  
**Category:** MCP server directories & lists  
**License:** MIT  
**Source:** https://github.com/varunneal/spotify-mcp

## Description
Spotify MCP Server is an MCP project that connects LLMs (e.g., Claude) with Spotify. It is built on top of the Spotipy API (spotipy-dev, v2.24.0) and enables programmatic control of music playback and library management through the Spotify platform.

## Features
- Control playback
  - Start playback
  - Pause playback
  - Skip to next track
- Search Spotify catalog
  - Search for tracks
  - Search for albums
  - Search for artists
  - Search for playlists
- Retrieve metadata
  - Get information about a track
  - Get information about an album
  - Get information about an artist
  - Get information about a playlist
- Queue management
  - Manage the Spotify queue (e.g., add items)
- Playlist management
  - Create playlists
  - Update playlists
  - Manage existing playlists

## Technology
- MCP (Model Context Protocol) server
- Built on Spotipy (Spotify Web API client) v2.24.0
- Requires Spotify Developer credentials (Spotify Web API keys)
