# Tavily MCP

An MCP server that connects AI agents to Tavily’s real-time web search and extraction engine for research workflows.

---

## Overview
Tavily MCP is a Model Context Protocol (MCP) server that exposes Tavily’s web research capabilities—such as search, extraction, mapping, and crawling—to compatible AI agents and tools. It is designed to be used as an integration layer so agents can perform live web research programmatically.

---

## Features
- **MCP-compatible server**  
  - Implements a Model Context Protocol server for use with MCP-aware AI agents and tooling.

- **Real-time web search**  
  - Provides access to Tavily’s search engine for up-to-date web results.

- **Content extraction**  
  - Enables agents to extract relevant information from web pages via Tavily’s extraction engine.

- **Map-style operations**  
  - Supports mapping operations over sets of URLs or search results (e.g., applying the same extraction or processing across multiple pages).

- **Web crawling**  
  - Offers crawl capabilities to traverse and analyze linked pages starting from one or more URLs.

- **Web research workflow support**  
  - Built to support end-to-end web research tasks (search → crawl/map → extract) inside AI agent workflows.
