# Swiftype MCP Server

**Category:** Web Search MCP Servers  
**Slug:** `swiftype-mcp-server`

An MCP server that integrates Swiftype’s site, app, and enterprise search platform into MCP-compatible tools, enabling agents to run search and indexing operations across websites and applications.

---

## Features

- **Unified Swiftype Integration**  
  - Connects Swiftype’s Site Search, App Search, and Enterprise Search capabilities into MCP tools.
  - Designed for use by agents / chat-based applications that support the Model Context Protocol (MCP).

- **Search Across Properties**  
  - Enables search operations across websites and applications backed by Swiftype.
  - Suitable for enterprise and large-scale content search scenarios.

- **Indexing Operations**  
  - Supports indexing-related operations exposed via Swiftype’s API (e.g., managing searchable content), as made available through the MCP server.

- **Standard MCP Endpoint**  
  - Single static URL for all clients:  
    `https://mcp.pipedream.net/v2`
  - Authentication is handled when adding the server to the client / application.

- **Client-Agnostic Setup**  
  - Works with any MCP-compatible chat client or application.
  - Configuration instructions are provided per client via the associated configuration documentation.
