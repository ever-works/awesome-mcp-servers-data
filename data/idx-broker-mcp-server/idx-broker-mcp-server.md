# IDX Broker MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** IDX Broker

IDX Broker MCP Server exposes IDX Broker’s real estate capabilities—lead management, home search, and website-related functionality—to MCP-based tools via a standardized MCP server endpoint.

---

## Features

- **MCP-based integration**
  - Provides an MCP (Model Context Protocol) server endpoint that MCP-compatible chat clients and tools can connect to.
  - Uses a single static MCP server URL for all clients.

- **IDX Broker real estate capabilities**
  - Connects to IDX Broker’s real estate platform features, including:
    - Lead management tools for real estate inquiries and prospects.
    - Customizable home search functionality.
    - Integration with IDX Broker–powered, fully hosted real estate websites.

- **Static server URL**
  - MCP server base URL: `https://mcp.pipedream.net/v2`
  - Same URL used across clients; individual authentication happens when adding the server to each application.

- **Client setup support**
  - Can be added to various MCP-compatible chat clients.
  - Configuration details available via a dedicated configuration page (outside this summary).

---

## Integration & Usage

- Add the MCP server URL (`https://mcp.pipedream.net/v2`) to your MCP-enabled chat or automation client.
- Authenticate within your client when prompted to connect to IDX Broker services.
