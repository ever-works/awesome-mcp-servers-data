# Remarkety MCP Server

## Overview
The Remarkety MCP Server is a Model Context Protocol (MCP) integration that connects Remarkety’s e‑commerce email marketing and automation capabilities to MCP-compatible applications via a static server endpoint.

## Features
- **MCP-compatible server endpoint**
  - Provides a single static MCP server URL: `https://mcp.pipedream.net/v2`
  - Works for every Remarkety client; authentication is handled when adding the server to your application.

- **Account connection flow**
  - Connect a Remarkety account through the Pipedream-powered configuration UI.
  - Select the appropriate client after authentication to finalize setup.

- **Tool discovery (actions exposure)**
  - Exposes available Remarkety-related tools / actions to MCP clients (displayed as “Available tools” in the UI).
  - Tools are dynamically loaded and made accessible through the MCP protocol.

- **Configuration guidance**
  - Step-by-step guidance for copying the MCP server URL and adding it to supported chat or MCP clients.
  - Links to a full configuration page for more detailed setup instructions.

## Usage
1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add this server URL to your MCP-compatible app or chat client.
3. Authenticate with your Remarkety account when prompted and select the desired client.
4. Use the exposed tools/actions to interact with Remarkety’s e‑commerce email marketing and automation features from within your application.
