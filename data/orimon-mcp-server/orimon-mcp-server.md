# Orimon MCP Server

**Description**  
An MCP server integration that connects Orimon’s AI sales chatbot platform to MCP-compatible AI tools, enabling automated lead qualification and deal closing.

**Website**  
https://mcp.pipedream.com/app/orimon

**Category**  
Business & Commerce · MCP Servers

**Tags**  
- Sales  
- Chatbots  
- Lead generation

---

## Features
- Connects an Orimon account to MCP-compatible AI tools.  
- Works via Pipedream’s MCP server.  
- Integrates with tools such as Cursor, Claude, and Windsurf.  
- Enables use of Orimon’s AI sales chatbot capabilities (e.g., lead qualification and deal closing) from within MCP client environments.
