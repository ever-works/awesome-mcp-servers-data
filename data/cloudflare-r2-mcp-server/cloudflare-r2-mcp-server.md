# Cloudflare R2 MCP Server

**Description**  
Cloudflare R2 MCP Server exposes Cloudflare R2 object storage through an MCP-compatible interface, allowing MCP clients to perform S3-compatible, multi-cloud object storage operations.

**Category**  
File Management MCP Servers

**Brand**  
Cloudflare (via Pipedream Connect)

**MCP Server URL**  
`https://mcp.pipedream.net/v2`

This is a static endpoint used to connect from any compatible MCP client; authentication is handled when adding the server to the client application.

## Features
- MCP-compatible server interface for Cloudflare R2
- Access to Cloudflare R2 object storage from MCP clients
- S3-compatible object storage operations
- Designed for multi-cloud storage architectures
- Single static MCP server URL usable across different clients
- Authentication performed during client-side server configuration
