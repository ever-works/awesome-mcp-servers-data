# Cyfe MCP Server

**Description**  
Cyfe MCP Server connects MCP-based tools and chat clients to Cyfe’s all‑in‑one business dashboard and data visualization platform, enabling custom, real‑time views of performance data aggregated from many different services.

## Features

- **MCP server for Cyfe**  
  - Exposes Cyfe’s dashboard and data visualization capabilities to MCP-compatible tools and chat clients.

- **Static MCP server URL**  
  - Server endpoint: `https://mcp.pipedream.net/v2`  
  - Same URL can be reused across different clients; authentication is handled when adding the server to each application.

- **Custom data visualization dashboards**  
  - Supports building custom dashboards tailored to business or agency performance tracking.

- **Real-time performance view**  
  - Provides a real-time view of key metrics within Cyfe dashboards.

- **Multi-service integrations**  
  - Integrates metrics from 60+ external services into a single Cyfe dashboard environment.

- **Chat client integration**  
  - Designed to be added to various chat clients that support MCP, enabling conversational access to Cyfe data.

- **Configuration documentation**  
  - A dedicated configuration page describes how to add and configure the server in supported applications.

- **Powered by Pipedream Connect**  
  - Operates via Pipedream’s infrastructure for connectivity and server management.

---

## Setup & Usage

- Use the static MCP server URL `https://mcp.pipedream.net/v2` when adding the server to your MCP-compatible app or chat client.
- Authenticate within your client when prompted to link it with your Cyfe account.
- Refer to the provider’s configuration guide ("Configuration" page) for client-specific setup steps.
