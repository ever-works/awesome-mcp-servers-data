# Trainual MCP Server

**Category:** Documentation & Learning Resources  
**Website:** https://mcp.pipedream.com/app/trainual

## Overview
Trainual MCP Server is an MCP (Model Context Protocol) server integration that connects Trainual’s business playbooks, SOPs, onboarding, training, and process documentation into compatible chat or AI clients. It allows automated interaction with a company’s training and knowledge content from within MCP-enabled applications.

## Features
- **MCP Server Endpoint**  
  - Provides a static MCP server URL that works for all clients:  
    - `https://mcp.pipedream.net/v2`
  - Authentication is handled when the server is added to an MCP-compatible application.

- **Trainual Integration**  
  - Designed to work with Trainual content, including:  
    - Business playbooks  
    - Standard operating procedures (SOPs)  
    - Onboarding materials  
    - Training content  
    - Process documentation  
    - General company knowledge resources

- **Chat Client Compatibility**  
  - Can be added as a server to various chat or AI clients that support MCP.  
  - Provides guidance per chat client on how to connect and start using the server (via the linked configuration resources).

- **Centralized Configuration Docs**  
  - A full configuration page (linked from the app) explains how to:  
    - Add the server URL to a client  
    - Authenticate  
    - Complete setup for different client environments.
