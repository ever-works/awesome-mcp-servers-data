# Salestown MCP Server

**Category:** Business & Commerce MCP Servers  
**Tags:** lead-management, sales, CRM

## Description
Salestown MCP Server is an MCP (Model Context Protocol) server integration for Salestown, a 360-degree lead management system focused on business growth. It is provided via Pipedream Connect and can be used from compatible chat clients and applications.

## Features
- MCP server integration for the Salestown lead management platform
- Works via a single static MCP endpoint URL
- Supports authentication at the time you add the server to your application
- Usable from multiple compatible chat clients
- Central configuration and setup through Pipedream’s configuration page

## MCP Server Endpoint
- **Base URL:** `https://mcp.pipedream.net/v2`
- **Usage:**
  - Same static URL for all clients
  - Authentication handled during server addition / configuration in your client or app

## Setup & Configuration
- Add the MCP server in your preferred chat client or MCP-compatible application
- Follow the client-specific instructions provided via Pipedream’s configuration documentation
- Configuration details and additional setup steps are available on the Pipedream **Configuration** page.
