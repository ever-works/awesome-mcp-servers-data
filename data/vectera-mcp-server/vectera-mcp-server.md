# Vectera MCP Server

**Category:** Project Management MCP Servers  
**Brand:** Vectera  
**Source:** https://mcp.pipedream.com/app/vectera

## Overview
Vectera MCP Server is an MCP-compatible integration that lets MCP clients schedule appointments and manage online video meetings and client sessions using Vectera’s platform.

It provides a static MCP server URL that you can add to compatible chat or MCP client applications to interact with Vectera.

## Features
- **MCP-compatible server endpoint**  
  - Static server URL: `https://mcp.pipedream.net/v2`  
  - Works with any MCP-capable client that supports adding custom servers.

- **Vectera integration**  
  - Schedule appointments via Vectera.  
  - Manage and conduct online video meetings.  
  - Manage client sessions using Vectera’s meeting tools.

- **Account connection**  
  - Connect your Vectera account through the Pipedream-hosted MCP server.  
  - Authentication flow occurs when adding the server to your MCP client.

- **Client-agnostic usage**  
  - Designed to work with multiple chat/MCP clients (e.g., different AI/chat applications that support MCP).  
  - Central configuration page for setup instructions (via the Pipedream configuration page).

## Configuration
- Copy and use the MCP server URL: `https://mcp.pipedream.net/v2`.  
- Add the server to your MCP-compatible chat client or application.  
- Authenticate your Vectera account when prompted.
