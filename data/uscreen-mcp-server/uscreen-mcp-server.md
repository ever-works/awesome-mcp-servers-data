# Uscreen MCP Server

**Category:** Media Processing MCP Servers  
**Source:** https://mcp.pipedream.com/app/uscreen

## Overview
Uscreen MCP Server exposes Uscreen’s video membership and streaming platform capabilities as tools that MCP-compatible agents and chat clients can use. It is delivered via Pipedream Connect as a static MCP endpoint.

## Features
- **Uscreen integration via MCP**
  - Provides tools that reflect Uscreen’s video membership and streaming platform features for use by MCP agents.
  - Designed for video creators and membership-based video platforms.

- **Static MCP endpoint**
  - Single MCP server URL for all compatible clients:  
    `https://mcp.pipedream.net/v2`
  - Authentication is handled when the server is added to the client/application.

- **Client-agnostic setup**
  - Same endpoint can be added to different MCP-enabled chat clients.
  - Configuration details available through a central configuration page (Pipedream Configuration).

- **Pipedream Connect integration**
  - Hosted and powered by Pipedream Connect infrastructure.
  - Benefits from Pipedream’s standardized MCP server management and connection model.

## Configuration
- Add the MCP server URL `https://mcp.pipedream.net/v2` to your MCP-compatible app or chat client.
- Follow the client-specific instructions provided in the Pipedream configuration page for authentication and setup.
