# Philips Hue MCP Server

An MCP server that connects MCP-compatible tools and agents to Philips Hue smart lighting, enabling control and automation of lights and scenes.

## Overview
- **Type:** MCP server integration
- **Category:** Home automation / smart lighting control
- **Provider:** Pipedream (Pipedream Connect)
- **Brand:** Philips Hue
- **Primary use:** Control and automate Philips Hue smart lighting systems and scenes via MCP-compatible clients.

## MCP Server URL
Use this static URL for all MCP clients (authentication occurs when you add it to your app):

```text
https://mcp.pipedream.net/v2
```

## Features
- Connects MCP tools and agents to Philips Hue smart lighting.
- Supports control of Philips Hue devices (e.g., turning lights on/off, adjusting settings) via MCP.
- Enables automation of Philips Hue lighting scenes through MCP workflows.
- Uses a single, static MCP server URL compatible with multiple clients.
- Authentication handled at the time of adding the server to your application.

## Usage
- Add the MCP server URL to your preferred MCP-compatible chat client or application.
- Follow your client’s configuration flow to authenticate and start controlling Philips Hue lights and scenes.
- Full configuration details are available on the provider’s configuration page (referenced as the “Configuration page” on the source site).
