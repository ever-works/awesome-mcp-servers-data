## Planview Leankit MCP Server

**Category:** Project Management MCP Servers  
**Slug:** `planview-leankit-mcp-server`  
**Brand:** Planview LeanKit

### Description
Planview Leankit MCP Server exposes Planview LeanKitâ€™s enterprise Kanban and work-tracking capabilities through the MCP protocol. It enables applications and chat clients to access, query, and update LeanKit boards, cards, and tasks, supporting work management from strategy to delivery.

### Connection & Configuration
- Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`
- Authenticate when adding the server to your MCP-compatible application or chat client.
- Configure by connecting a Planview LeanKit account and selecting the desired client.

### Features

#### General Capabilities
- MCP-based access to Planview LeanKit data and actions.
- Work management on enterprise Kanban boards from strategy to delivery.
- Support for distributed teams with shared views of work status and priority.
- 26 available MCP actions (tools) for interacting with LeanKit.

#### Available Tools / Actions
(The following are explicitly listed tools in the integration.)

- **Update Custom Field Value**  
  Update a custom field value on a card.

- **Update Card (Or Task)**  
  Update fields on a card or task (e.g., details, metadata).

- **Unblock Card (Or Task)**  
  Unlock/unblock a card or task.

- **Unassign User**  
  Remove a user assignment from a card or task.

- **Start Task**  
  Move a task to the `inProcess` lane.

- **Remove Tags From Card**  
  Remove one or more tags from a card or task.

- **Remove Parent/Child Connection**  
  Remove a parent or child relationship from a card.

- **Move Cards To Another Board**  
  Move one or more cards between boards.

- **Move Cards Between Lanes**  
  Move cards between lanes on a board.

- **Get Card**  
  Retrieve card details by card ID.

- **Find OR Create Card**  
  Search for a card and, if not found, create a new one.

- **Find First Card**  
  Find the first card that matches specified criteria.

- **Duplicate Task**  
  Create a copy of an existing task.

- **Duplicate Card**  
  Create a copy of an existing card.

- **Delete Custom Field**  
  Delete a custom field (truncated in content, but identified as an action).
