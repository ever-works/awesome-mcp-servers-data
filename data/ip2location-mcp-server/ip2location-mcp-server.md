# IP2Location MCP Server

Identify geographical location and proxy information by IP address via an MCP server powered by Pipedream.

## Overview
- **Type:** MCP server (data access / integration)
- **Provider:** Pipedream
- **Service:** IP2Location IP geolocation lookup
- **Use case:** Enrich MCP-based workflows with IP-based geolocation and proxy detection data
- **Access URL:** `https://mcp.pipedream.net/v2`

- Supports identification of geographical location from IP address
- Supports identification of proxy usage from IP address

## Integration & Usage
- Add the server using the static URL: `https://mcp.pipedream.net/v2`
- Configure within compatible chat clients or applications that support MCP
- Additional configuration details are available on the Pipedream MCP configuration page
