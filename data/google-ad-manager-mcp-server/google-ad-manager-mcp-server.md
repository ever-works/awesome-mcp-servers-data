# Google Ad Manager MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Google / Pipedream  
**Slug:** google-ad-manager-mcp-server

## Overview
The Google Ad Manager MCP Server is an MCP-compatible server that connects applications to Google Ad Manager, Google’s integrated advertising management platform. It is hosted by Pipedream and is accessed via a static MCP server URL.

## Features
- **MCP-compatible server endpoint** for integrating Google Ad Manager with MCP-enabled applications.
- **Static server URL** shared across clients:
  - `https://mcp.pipedream.net/v2`
- **Per-app authentication** performed when you add the server to your application (authentication is not embedded in the URL).
- **Client-agnostic integration**: the same server URL works for all supported chat or MCP clients.
- **Configuration guidance** available via a dedicated configuration page (instructions vary by client type).
- **Tooling exposure**: designed to load and expose available Google Ad Manager–related tools/actions to your MCP client (e.g., “Available tools / Loading actions / Loading available tools” integration points), enabling programmatic interaction with Google Ad Manager once configured.

## Usage
1. Connect your Google Ad Manager account in Pipedream.
2. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
3. Add this server URL to your MCP-enabled application or chat client.
4. Authenticate when prompted by your client.
5. Use the loaded tools/actions to interact with Google Ad Manager through MCP.
