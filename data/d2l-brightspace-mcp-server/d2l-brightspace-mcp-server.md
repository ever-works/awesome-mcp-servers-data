## Overview
D2L Brightspace MCP Server is an MCP-compatible server that integrates the Brightspace LMS with MCP-based tools. It enables AI-driven access to Brightspace courses, content, and learner data via a standardized MCP endpoint.

## Features
- **MCP-compatible server** for connecting Brightspace LMS to MCP-based tools and applications.
- **AI-driven access to LMS data**, including courses, content, and learner-related information (as exposed through the MCP integration).
- **Static MCP server URL** for all clients:
  - Endpoint: `https://mcp.pipedream.net/v2`
- **Authentication at configuration time**, performed when adding the MCP server to an application (the static URL is shared; auth differentiates clients).
- **Works with multiple chat clients** (can be added as a server to supported chat applications that understand MCP).
- **Configuration documentation available** via a dedicated configuration page (linked from the app page) for setup and usage details.
- **Hosted and powered by Pipedream Connect**, leveraging Pipedreamâ€™s infrastructure for MCP connectivity.

## Getting Started
1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.  
2. Add the server to your MCP-compatible chat client or application.  
3. Authenticate as prompted by your client to enable access to Brightspace LMS data.
4. Refer to the configuration documentation (linked from the app page) for client-specific setup steps.
