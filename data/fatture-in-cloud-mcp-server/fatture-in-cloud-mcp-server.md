# Fatture in Cloud MCP Server

## Overview
Fatture in Cloud MCP Server is an MCP Server integration for Fatture in Cloud (Cloud Invoices). It exposes business and VAT management capabilities so MCP-compatible tools and chat clients can interact with invoices and accounting data from mobile or web environments.

## Features
- MCP server integration with Fatture in Cloud (Cloud Invoices)
- Access to invoicing and accounting data through MCP-aware tools
- Supports management of business and individual VAT numbers
- Designed for use from smartphone, tablet, or web-based clients
- Single static MCP server URL (`https://mcp.pipedream.net/v2`) used by all clients
- Authentication handled when adding the server to the client application
- Works with multiple chat clients via MCP configuration

## Technical Details
- MCP Server URL: `https://mcp.pipedream.net/v2`
- Configuration and setup guided through supported chat clients or the main configuration page
