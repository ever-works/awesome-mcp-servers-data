# Clear Books MCP Server

**Description**  
An MCP Server integration that exposes Clear Books online accounting capabilities to MCP-based tools and agents.

**Server URL**  
- MCP server base URL (static for all clients): `https://mcp.pipedream.net/v2`

**Features**  
- Integrates Clear Books online accounting software with MCP-compatible tools and agents.  
- Uses a single, static MCP server URL for all clients; authentication is handled when adding the server to your application.  
- Can be added to various chat or MCP-enabled clients to access Clear Books capabilities programmatically.  
- Configuration details and setup are available via the Pipedream MCP configuration pages.
