# Paykickstart MCP Server

## Overview
Paykickstart MCP Server is an MCP-compatible interface to the Paykickstart platform, exposing shopping cart and affiliate management capabilities for use by MCP-based agents, automations, and chat clients.

- **Type:** MCP server (integration layer)
- **Platform:** Paykickstart (shopping cart & affiliate management)
- **Primary use cases:**
  - Manage online shopping cart operations via MCP
  - Integrate affiliate management features into MCP-based workflows
  - Enable agents and automations to interact with Paykickstart data

## MCP Server Endpoint
- **Base URL:** `https://mcp.pipedream.net/v2`
- This is a static MCP server URL used across clients; authentication is handled when adding the server to your application.
