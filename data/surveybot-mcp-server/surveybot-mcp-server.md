# Surveybot MCP Server

## Overview
Surveybot MCP Server connects MCP-compatible clients to Surveybot, enabling the creation and management of surveys delivered via Facebook Messenger and Workplace.

- **Category:** Messaging MCP Servers
- **Integrations:** Facebook Messenger, Workplace
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP server endpoint** for Surveybot, usable across multiple MCP clients via a single static URL.
- **Facebook Messenger survey delivery**, allowing you to run surveys directly in Messenger.
- **Workplace survey delivery**, enabling surveys within Workplace by Facebook.
- **Account connection flow** to link your Surveybot account with the MCP server.
- **Client-agnostic setup**, where the same server URL works for every supported MCP client.
- **Configuration guidance** via a dedicated configuration page for adding the server to different chat clients.
