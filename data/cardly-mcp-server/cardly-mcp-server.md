# Cardly MCP Server

An MCP Server integration for Cardly that lets MCP-enabled clients automate sending personalized, handwritten-style physical cards.

## Overview
- **Type**: MCP server integration / connector
- **Use case**: Automate direct mail workflows, especially personalized, handwritten-style cards
- **Works with**: Any MCP-compatible chat or automation client via a static MCP endpoint
- **Provider**: Cardly, delivered via Pipedream Connect

## Features
- **MCP-compatible endpoint**  
  - Single static MCP server URL for all supported clients:  
    `https://mcp.pipedream.net/v2`  
  - Authentication handled when adding the server in the client application.

- **Automated card sending**  
  - Supports sending personal, handwritten-style cards programmatically.  
  - Designed for use within chat/automation flows (e.g., triggered from conversations or workflows).

- **Client-agnostic integration**  
  - Usable from multiple MCP clients; the same URL works across clients.  
  - Configuration performed in the client (e.g., adding the server and authenticating).

- **Pipedream Connect integration**  
  - Hosted and powered by Pipedream Connect.  
  - Governed by Pipedream terms and privacy policies.
