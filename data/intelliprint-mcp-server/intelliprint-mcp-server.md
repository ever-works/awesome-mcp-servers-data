MCP Server for Intelliprint, exposing APIs to send online letters and postcards programmatically within MCP-compatible environments.

---

## Features

- MCP-compatible server for Intelliprint
- Programmatic sending of online letters
- Programmatic sending of online postcards
- Works within any MCP-compatible client / environment
- Single static MCP server URL for all clients:  
  `https://mcp.pipedream.net/v2`
- Authentication handled when adding the server to your application

---

## Integration & Configuration

- Add the server URL directly in your MCP-capable chat client or app
- Configuration details available via the Pipedream MCP configuration documentation (linked from the source page)
