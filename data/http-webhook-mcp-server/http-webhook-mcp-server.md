# HTTP / Webhook MCP Server

**Brand:** Pipedream  
**Category:** API Integration MCP Servers  
**Website:** https://mcp.pipedream.com/app/http

## Description
HTTP / Webhook MCP Server provides a static MCP-compatible HTTP endpoint you can use to receive HTTP or webhook requests, enabling HTTP-based integrations from MCP-enabled clients.

## Features
- **Static MCP server URL**  
  - Single, fixed endpoint for all clients: `https://mcp.pipedream.net/v2`  
  - Works across different MCP-compatible chat clients.

- **HTTP / webhook ingestion**  
  - Accepts standard HTTP and webhook requests sent to the unique URL.  
  - Suitable for integrating external systems and services with MCP-enabled applications.

- **Client-agnostic configuration**  
  - Same base URL used regardless of which MCP client you use.  
  - Authentication performed when adding the server to each application/client.

- **Configuration guidance**  
  - Documentation available via the "Configuration" page for step-by-step setup in supported chat clients.

## Integration & Usage
- Add `https://mcp.pipedream.net/v2` as an MCP server in your compatible chat or agent client.
- Authenticate within the client when prompted.
- Use the provided URL to receive HTTP or webhook calls from external services.
