**Description**  
Geoapify MCP Server is an MCP (Model Context Protocol) integration that connects your Geoapify account to AI-powered tools, enabling access to Geoapify map APIs and location-based services directly from MCP-enabled applications.

**Features**
- Integrates Geoapify with AI tools such as Cursor, Claude, and Windsurf.  
- Provides access to Geoapify map APIs through the MCP server.  
- Supports building custom interactive maps within MCP-enabled workflows.  
- Enables use of location intelligence and mapping tools from Geoapify in AI-assisted environments.  
- Facilitates API-based map and geolocation operations (e.g., queries and tools) via a standardized MCP interface.
