## Description
The Hotjar MCP Server is an MCP Server integration that exposes Hotjar’s user behavior analytics, heatmaps, and feedback data to MCP-based tools and chat clients. It enables applications that support the Model Context Protocol (MCP) to query and use Hotjar product experience insights directly within conversational or automated workflows.

## Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Designed to work with any compatible MCP client.

- **Integration with MCP-based tools and chat clients**  
  - Can be added to various chat clients that support MCP.  
  - Central, reusable server configuration across different clients.

- **Access to Hotjar product experience data**  
  - User behavior analytics.  
  - Heatmaps and interaction insights.  
  - User feedback and qualitative data (as exposed via Hotjar’s APIs through MCP).  
  - Intended to help inform product and UX decisions.

- **Static configuration model**  
  - Single, static URL for all clients with authentication handled during client configuration.

- **Hosted by Pipedream Connect**  
  - Managed hosting of the MCP server endpoint via Pipedream’s Connect platform.

## Integration & Setup
- Use the static MCP endpoint: `https://mcp.pipedream.net/v2`.
- Add the server in your MCP-compatible application or chat client.  
- Authentication is completed when registering/configuring the server in the client.
- Additional configuration details are available on the provider’s configuration page (not included in the provided content).
