## Overview
Housecall Pro MCP Server is an MCP-compatible server that exposes Housecall Pro’s home services business management functionality (such as scheduling and operations) to chat clients and other MCP-enabled tools via Pipedream.

## Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Works for all clients; authentication occurs when adding the server to your application.
- **Housecall Pro integration**  
  - Connects a Housecall Pro account through Pipedream.  
  - Designed to surface home services business management capabilities (e.g., scheduling and operations) to MCP tools and chat applications.
- **Client configuration**  
  - Connect your Housecall Pro account and select a client within the Pipedream UI to start using the server.  
  - Documentation available via a configuration page for step-by-step setup.
- **Tool exposure for automations**  
  - Intended to load and expose Housecall Pro-related actions and tools to MCP-compatible clients (specific tools/actions are not listed in the provided content, but are presented within the Pipedream interface as “Available tools”).

## Integration & Setup
- Copy and use the static MCP server URL in any MCP-compatible client.  
- Authenticate when adding the server to your chat client or application.  
- Use Pipedream’s configuration page for detailed setup instructions.
