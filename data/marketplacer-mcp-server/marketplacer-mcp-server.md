# Marketplacer MCP Server

## Overview
The Marketplacer MCP Server exposes the Marketplacer online marketplace platform through the MCP/Pipedream integration, allowing MCP-compatible chat clients and tools to interact with Marketplacer functionality via a single static server endpoint.

- **Type:** MCP Server (integration endpoint)
- **Platform:** Marketplacer (online marketplace / e‑commerce)
- **Category:** business-commerce-mcp-servers
- **Tags:** marketplace, e‑commerce, online-store

## Features
- **MCP-compatible endpoint**: Provides a static MCP server URL that can be used by any compatible client:
  - `https://mcp.pipedream.net/v2`
- **Marketplacer platform access**: Enables access to functionality of the Marketplacer online marketplace platform via the MCP/Pipedream integration (specific tools / methods are defined within the MCP server, not detailed in the provided content).
- **Client-agnostic configuration**:
  - The same server URL is used across different MCP-enabled chat clients.
  - Authentication is handled when adding the server to each client/app.
- **Centralized configuration documentation**:
  - A configuration page (on Pipedream) provides instructions for adding the server to different chat clients.

## Usage
- Add the MCP server to your MCP-capable chat client or application using:
  - **Server URL:** `https://mcp.pipedream.net/v2`
- Complete authentication during the add/configure step in your client.
