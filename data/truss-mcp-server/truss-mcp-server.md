# Truss MCP Server

**Category:** Document Management MCP Servers  
**Brand:** Truss  
**Slug:** `truss-mcp-server`

## Overview
Truss MCP Server is an MCP (Model Context Protocol) server integration that connects your Truss account to AI tools such as Cursor, Claude, and Windsurf. It is designed for accounting professionals to manage secure client document requests and organize documents within MCP-based workflows.

## Features
- **MCP-based integration**: Exposes Truss functionality through the MCP protocol so it can be used by compatible AI tools and IDEs.
- **Truss account connectivity**: Links directly to your existing Truss account for unified access to client documents.
- **Secure document requests**: Supports workflows for requesting documents from clients in a secure, controlled manner (for accounting and compliance use cases).
- **Document organization**: Helps structure and manage client documents for accounting professionals inside AI-assisted workflows.
- **AI tool compatibility**:
  - Cursor
  - Claude
  - Windsurf
- **Workflow automation**: Enables AI-driven or semi-automated document management flows by combining Truss with AI agents via MCP.

## Use Cases
- Accounting firms managing secure client document intake via AI tools.
- Organizing and retrieving client financial documents through MCP-compatible editors or AI assistants.
- Embedding Truss document workflows into AI-powered compliance or accounting processes.
