# Cflow MCP Server

**Category:** Workflow Automation MCP Servers  
**Vendor / Brand:** Cflow  
**Source:** https://mcp.pipedream.com/app/cflow

Cflow MCP Server integrates Cflow’s cloud BPM and workflow automation platform with MCP-compatible clients, enabling automation and management of organizational workflows through standardized MCP tools.

---

## Features

- **Cloud BPM & Workflow Integration**  
  - Connects MCP clients to Cflow’s cloud-based BPM and workflow automation platform.  
  - Supports organizations of various sizes looking to automate daily activities and improve productivity.

- **Static MCP Server Endpoint**  
  - Single static server URL for all clients:  
    - `https://mcp.pipedream.net/v2`  
  - Same URL works across different chat or MCP-enabled applications.

- **Account-Based Configuration**  
  - Requires connecting a Cflow account via the Pipedream MCP configuration flow.  
  - Once connected, users can select their client / chat interface to finalize setup.

- **Client-Agnostic Setup**  
  - Designed to be added to multiple chat clients or MCP-compatible applications.  
  - Documentation via a configuration page to guide per-client setup.

- **Tool / Action Exposure (via MCP)**  
  - Exposes Cflow-related tools and actions into the MCP ecosystem (listed dynamically as “available tools” / “actions”).  
  - Intended for automating and managing organizational workflows from within an MCP client.

---

## Usage

1. Connect your Cflow account via the Pipedream MCP configuration UI.  
2. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.  
3. Add this URL as an MCP server in your chat client or MCP-compatible application.  
4. Authenticate when prompted and use the exposed tools to interact with Cflow workflows.
