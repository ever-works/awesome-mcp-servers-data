# Affinity MCP Server

Affinity MCP Server exposes Affinity’s collective-intelligence CRM data to Model Context Protocol (MCP)–based tools and automations.

## Overview
- **Type:** MCP Server (for Affinity CRM)
- **Category:** Business & Commerce · MCP Servers
- **Use Case:** Access and work with Affinity CRM relationship and intelligence data from MCP-compatible chat clients and automation tools.

## Features
- **MCP-compatible endpoint**
  - Provides a static MCP server URL usable across supported MCP clients:  
    `https://mcp.pipedream.net/v2`
  - Designed to plug into MCP-based chat and automation environments.

- **Affinity CRM integration**
  - Connects to Affinity, a collective-intelligence CRM focused on relationship data.
  - Enables MCP tools to leverage Affinity’s relationship and deal intelligence within conversations and workflows.

- **Client-agnostic configuration**
  - Same server URL works for every supported MCP client.
  - Authentication is performed when you add/configure the server in your chosen application.
  - Guided setup available by selecting your chat client and via a centralized configuration page (referenced as “Configuration page” in the source content).

- **Hosted by Pipedream**
  - MCP server infrastructure provided via Pipedream Connect.
  - Centralized hosting removes the need to self-host the MCP server.

## Getting Started
1. Use the static server URL in your MCP-compatible client:  
   `https://mcp.pipedream.net/v2`
2. Add the server in your app’s MCP / tools configuration and complete authentication with Affinity when prompted.
3. (Optional) Refer to the linked configuration documentation in the original site for client-specific setup steps.
