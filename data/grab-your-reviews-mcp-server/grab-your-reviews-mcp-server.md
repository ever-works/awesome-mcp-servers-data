## Features
- **MCP-compatible server** that can be added to any supported MCP client via a single static URL.
- **Unified reviews and feedback management** for handling online reviews from multiple sites within one platform (via Grab Your Reviews).
- **Customer feedback workflows** support, enabling management of feedback-related tasks from a unified interface.
- **Static server endpoint** (`https://mcp.pipedream.net/v2`) used across all clients; authentication occurs when adding the server.
- **Configuration documentation** available through a dedicated configuration page for setup details (client-specific instructions).
- **Powered by Pipedream Connect**, integrating Grab Your Reviews capabilities into the MCP ecosystem.

## Integration & Setup
- Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server within your MCP-compatible chat / agent client.
- Authenticate during the server-add process in your chosen client.
- Optional: Refer to the full configuration page for client-specific setup steps.
