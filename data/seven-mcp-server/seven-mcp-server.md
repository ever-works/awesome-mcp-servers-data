# Seven MCP Server

**Category:** Messaging MCP Servers  
**Brand:** seven  
**Slug:** seven-mcp-server

## Overview
Seven MCP Server provides integration with Seven’s SMS solutions through the Model Context Protocol (MCP). It enables sending and receiving SMS messages from MCP-compatible tools and applications using Seven’s messaging infrastructure.

## Features
- MCP-based integration with Seven’s SMS platform
- Send and receive SMS messages
- Accessible via:
  - Online interfaces
  - API
  - Email
  - Directly from your app
- Usable from any MCP-compatible chat client
- Static MCP server endpoint for all clients: `https://mcp.pipedream.net/v2`
- Authentication handled when adding the server to your application

## Configuration
- Add the MCP server URL `https://mcp.pipedream.net/v2` to your MCP-compatible client or application
- Follow client-specific setup instructions (via the provider’s configuration documentation)
