# ProfitWell MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** ProfitWell

An MCP (Model Context Protocol) server that connects ProfitWell’s subscription analytics and growth capabilities to MCP-based applications and automations.

---

## Overview
ProfitWell MCP Server exposes ProfitWell subscription and revenue analytics through a standard MCP endpoint, allowing integration with compatible chat clients and automation tools. It centralizes access to ProfitWell data via a static server URL that you can add to any MCP-enabled application.

---

## Features

- **MCP-compatible server endpoint**  
  - Static MCP URL: `https://mcp.pipedream.net/v2`  
  - Same URL works across all clients; authentication occurs when adding the server to your app.

- **ProfitWell account integration**  
  - Connect a ProfitWell account through the Pipedream configuration interface.  
  - Select a client after connection to start using ProfitWell data.

- **Subscription growth & analytics exposure**  
  - Designed to surface ProfitWell’s subscription growth and analytics capabilities to MCP-based automations and chat clients (exact tools/actions not listed in the source content).

- **Multi-client configuration**  
  - Intended for use with multiple chat clients that support MCP.  
  - Client-specific setup instructions available via the configuration page.

- **Configuration UI**  
  - Web-based configuration flow (“Configure ProfitWell”) to connect accounts and verify access.  
  - Status feedback (e.g., “Checking your account…”) during setup.

## Integration & Usage

- **Add to MCP-compatible apps**  
  - Copy the MCP server URL (`https://mcp.pipedream.net/v2`).  
  - Add the server in your chosen MCP client/application.  
  - Authenticate when prompted to link your ProfitWell account.
