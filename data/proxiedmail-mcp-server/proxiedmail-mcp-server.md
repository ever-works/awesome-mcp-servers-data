# ProxiedMail MCP Server

Proxy-email service MCP server for use with MCP-enabled applications.

## Overview
- **Name:** ProxiedMail MCP Server  
- **Category:** Messaging MCP Servers  
- **Functions:** Proxy email operations such as address masking and relaying via MCP-compatible clients  
- **MCP Server Base URL:** `https://mcp.pipedream.net/v2`

## Features
- Enables proxy-email workflows (e.g., address masking and email relaying) from MCP-enabled apps
- Exposes ProxiedMail capabilities through a standardized MCP server endpoint
- Uses a single static MCP server URL that works across all supported MCP clients
- Authentication handled when adding/configuring the server in the client application
- Can be integrated into various chat or agent clients that support MCP

## Integration & Usage
- **Server URL:** Add `https://mcp.pipedream.net/v2` as an MCP server in your client
- **Client setup:** Select or configure your MCP-compatible chat/client app, then authenticate during server addition
