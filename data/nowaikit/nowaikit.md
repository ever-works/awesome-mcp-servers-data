## Overview

NowAIKit is the most comprehensive ServiceNow AI toolkit, providing 400+ MCP tools and 26 AI capabilities for interacting with ServiceNow platforms. It supports both SDK mode and Direct API mode for flexible integration.

## Coverage

- ITSM (IT Service Management)
- ITOM (IT Operations Management)
- CMDB (Configuration Management Database)
- HRSD (Human Resources Service Delivery)
- CSM (Customer Service Management)
- SecOps (Security Operations)
- GRC (Governance, Risk, and Compliance)
- 35+ ServiceNow modules

## Features

- 400+ MCP tools for comprehensive ServiceNow automation
- 26 AI capabilities built-in
- SDK mode for programmatic integration
- Direct API mode for straightforward HTTP-based access
- Broad module coverage across the ServiceNow platform
