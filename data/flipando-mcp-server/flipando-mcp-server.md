# Flipando MCP Server

**Category:** Workflow Automation MCP Servers  
**Brand:** Flipando  
**Source:** https://mcp.pipedream.com/app/flipando

## Overview
Flipando MCP Server is an MCP-compatible integration that connects Flipando with supported chat or MCP client applications so teams can use AI to enhance business workflows and efficiency.

## Features
- **MCP-compatible AI server** – Exposes Flipando’s AI capabilities as an MCP server that can be added to any compatible client.
- **Static server endpoint** – Uses a single static MCP URL that works across all supported clients:  
  `https://mcp.pipedream.net/v2`
- **Per-client authentication** – Authentication occurs when you add the server to your chosen application or client.
- **Chat client integration** – Designed to be added to chat-based tools; setup documentation is provided per supported chat client.
- **Central configuration docs** – A dedicated configuration page is available for detailed setup and usage instructions.

## Setup
1. Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add this server in your MCP-compatible client or chat application.
3. Complete authentication when prompted in your client.
4. Refer to the configuration page for client-specific instructions.
