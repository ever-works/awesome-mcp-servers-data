## Features

- Enables chatting with your own data using AI-powered assistants.
- Built on the **Model Context Protocol (MCP)** for standardized context and tool integration.
- Provides a **static MCP server URL** that works across compatible clients:
  - `https://mcp.pipedream.net/v2`
- Authentication is handled when adding the server to your chosen application/client.
- Compatible with multiple MCP-enabled chat clients (selectable in the UI to get client-specific setup steps).
- Centralized configuration guidance available via a dedicated **Configuration** page.

## Integration & Usage

- Add the provided MCP server URL to any supported MCP client.
- Authenticate during setup within your chosen application.
- Use it to query and interact with your own data sources (e.g., RAG / knowledge-base scenarios) via AI assistants.
