# WordPress.com MCP Server

## Overview
The WordPress.com MCP Server by Pipedream connects MCP-compatible clients (such as ChatGPT) to hosted WordPress.com sites. It enables agents and applications to perform common content management operations—like creating, deleting posts, and uploading media—via a standardized server endpoint.

- **Category:** Content Management MCP Server
- **Provider / Brand:** Pipedream
- **Works with:** Hosted WordPress.com sites
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features

### MCP Server Integration
- Provides a static MCP server URL (`https://mcp.pipedream.net/v2`) that works for all supported clients.
- Authentication occurs when you add and configure the server within your client/application.
- Can be used with MCP-capable chat clients, including ChatGPT (OpenAI), following the client-specific configuration guide.

### WordPress.com Content Management Tools
The server exposes three primary actions as MCP tools:

1. **Create New Post**
   - Creates a new post on a specified WordPress.com site.
   - Allows agents to programmatically publish content via WordPress.com.

2. **Delete Post**
   - Deletes an existing post from a specified WordPress.com site.
   - Enables content cleanup and automated post management.

3. **Upload Media**
   - Uploads a media file from a given URL to a specified WordPress.com site.
   - Supports adding images or other media assets by reference (URL) rather than manual upload.
