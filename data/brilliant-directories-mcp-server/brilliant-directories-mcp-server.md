# Brilliant Directories MCP Server

**Description**  
An MCP (Model Context Protocol) server integration for Brilliant Directories, enabling automation and management of business directory and membership website data via Pipedream.

**Key Details**
- **Type:** MCP server integration
- **Platform:** Powered by Pipedream Connect
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Use Case:** Manage and automate data for business directory and membership websites built with Brilliant Directories

## Features
- Provides a static MCP server endpoint (`https://mcp.pipedream.net/v2`) usable across clients
- Supports authentication at the time of adding the server to an MCP-compatible application or chat client
- Integrates with Brilliant Directories for:
  - Business directory management
  - Membership website data management
- Compatible with multiple chat clients / MCP-capable applications (configured per client)
- Configuration details available via a dedicated configuration page (through Pipedream)
