# SparkPost MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** SparkPost  
**Slug:** `sparkpost-mcp-server`

## Description
SparkPost MCP Server exposes SparkPost’s email optimization and delivery capabilities to Model Context Protocol (MCP) clients. It enables automated, reliable email sending and access to email analytics from compatible chat or AI applications via a standardized MCP interface.

## Features
- **Email delivery via MCP**  
  - Send emails through SparkPost from any MCP-compatible client.  
  - Uses a single static MCP server URL for all clients:  
    - `https://mcp.pipedream.net/v2`

- **Email optimization support**  
  - Integrates with SparkPost’s email optimization platform (e.g., inbox reach, deliverability-focused sending behavior) through the MCP server.

- **Analytics access (via SparkPost)**  
  - Designed to expose SparkPost analytics to MCP clients for reporting and monitoring use cases.

- **Client-agnostic MCP endpoint**  
  - One static URL works across different MCP-enabled chat or AI applications.  
  - Authentication is performed when adding/configuring the server inside each application.

- **Configuration guidance**  
  - Compatible with multiple chat clients that support MCP.  
  - Documentation available via a dedicated configuration page for setup details.

## Integration & Access
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Access method:** Add the server URL to an MCP-compatible application and complete authentication within that client.  
- **Provider:** Delivered and hosted via Pipedream Connect.
