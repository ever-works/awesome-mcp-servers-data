# ProcFu MCP Server

**Category:** Workflow Automation MCP Servers  
**Brand:** ProcFu  
**Source:** https://mcp.pipedream.com/app/procfu

ProcFu MCP Server connects MCP-compatible clients to ProcFu, an add-on for Citrix Podio. It extends Podio Workflow Automation capabilities so you can automate and enhance Podio-based processes via your MCP client.

---

## Features

- **ProcFu–Podio integration**
  - Connects MCP clients to ProcFu, which augments Citrix Podio functionality.
  - Enables MCP-driven automation of Podio workflows and processes.

- **Central MCP server endpoint**
  - Uses a static server URL that is the same for all clients:
    - `https://mcp.pipedream.net/v2`
  - Authentication is performed when adding the server to your MCP-compatible application.

- **Client-agnostic setup**
  - Designed to be added to any supported MCP chat client.
  - Configuration guidance available through a centralized configuration page (not client-specific in the server itself).

- **Pipedream-hosted infrastructure**
  - Powered by Pipedream Connect, leveraging Pipedream’s MCP hosting to expose ProcFu capabilities to clients.

---

## Setup

1. Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add this server URL to your MCP-compatible app or chat client.
3. Authenticate when prompted by your client during server addition.
4. (Optional) Refer to the provider’s configuration documentation for client-specific steps.
