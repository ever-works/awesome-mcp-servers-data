# Amazon MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Amazon  
**Source:** https://mcp.pipedream.com/app/amazon

## Description
Amazon MCP Server provides a Model Context Protocol (MCP) server interface to Amazon’s ecommerce platform, enabling MCP-based tools and chat clients to interact with Amazon product and shopping-related capabilities through this integration.

## Features
- MCP server interface to Amazon’s ecommerce platform
- Supports interaction with product and shopping-related capabilities via MCP-based tools
- Accessible through a static MCP server URL that works across compatible clients
- Authentication handled when adding the server to an application
- Can be added to various chat clients that support MCP

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Integration Platform:** Powered by Pipedream Connect
