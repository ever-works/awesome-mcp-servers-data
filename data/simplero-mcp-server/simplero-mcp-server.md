# Simplero MCP Server

MCP Server integration for Simplero’s all-in-one marketing and course platform, enabling MCP-based tools to interact with courses, automations, and marketing assets.

## Overview
- **Type:** MCP server integration
- **Platform:** Simplero (marketing and course platform)
- **Category:** Business & Commerce – MCP Servers
- **Use Cases:** Course creators, coaches, and online marketers who want to access Simplero resources via MCP-compatible chat or agent clients.

## MCP Server URL
- **Base MCP Server URL:** `https://mcp.pipedream.net/v2`
- Static URL used for all clients
- Authentication occurs when adding the server in your MCP-compatible application

## Features
- **MCP Integration Layer**
  - Exposes Simplero functionality to MCP-based tools and chat clients
  - Central connection point for accessing Simplero data and actions through MCP

- **Support for Marketing & Course Operations** *(implied from platform description)*
  - Designed to work with Simplero’s:
    - Online courses
    - Marketing assets
    - Automations
  - Suitable for course creators, coaches, and online marketers using Simplero as their all-in-one marketing platform

- **Static Server Configuration**
  - Single, static MCP endpoint (`https://mcp.pipedream.net/v2`) for all clients
  - Configuration handled once at the client/app level

- **Client Setup Guidance**
  - Documentation and instructions provided for adding the server to various chat or agent clients
  - Additional configuration details available via a dedicated configuration page (linked from the app page)

## Integration & Configuration
- Add the MCP server URL to your MCP-compatible chat or agent application
- Authenticate within your client when prompted
- Refer to the external configuration documentation ("Configuration" page on Pipedream) for:
  - Client-specific setup steps
  - Any required authentication parameters
