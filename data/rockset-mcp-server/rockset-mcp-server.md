# Rockset MCP Server

## Description
Rockset MCP Server integrates Rockset’s serverless search and analytics engine with MCP-compatible tools, enabling live dashboards and real-time data APIs on data sources like DynamoDB, Kafka, and S3.

## Features
- MCP server interface for Rockset’s search and analytics engine
- Support for live dashboards powered by Rockset
- Real-time data APIs backed by Rockset queries
- Works with common data sources such as:
  - DynamoDB
  - Kafka
  - S3
  - (and other Rockset-supported sources)
- Static MCP server URL usable from any compatible client
- Authentication handled when adding the server to an application

## Technical Details
- MCP server base URL: `https://mcp.pipedream.net/v2`
- Compatible with multiple MCP-enabled chat or agent clients (added as a remote MCP server)
