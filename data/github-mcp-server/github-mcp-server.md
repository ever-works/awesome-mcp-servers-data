# GitHub MCP Server

GitHub MCP Server allows AI to interact with repositories, issues, pull requests, and GitHub features. Supports queries on open issues, stale issues, code references, and file changes. Integrates with VS Code, JetBrains, Eclipse, and Claude Desktop; use scoped tokens for read-only or controlled write access.
