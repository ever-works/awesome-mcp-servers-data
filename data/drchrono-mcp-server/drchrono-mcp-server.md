# DrChrono MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** drchrono  
**Slug:** drchrono-mcp-server

## Description
DrChrono MCP Server is an MCP (Model Context Protocol) server integration for the DrChrono EHR platform. It enables agents, chat clients, or workflows to interact programmatically with DrChrono to manage medical practice data and administration tasks.

## Features
- MCP server integration with DrChrono’s EHR platform
- Programmatic access for agents or workflows to interact with practice data
- Supports medical practice administration tasks through automated tools
- Single static MCP server URL for all clients: `https://mcp.pipedream.net/v2`
- Authentication handled when adding the server to your application or chat client
- Usable across different chat clients that support MCP servers
- Central configuration via Pipedream’s configuration interface

## Configuration
- Connect a DrChrono account via Pipedream
- Use the static MCP server URL: `https://mcp.pipedream.net/v2`
- Add the server to a compatible app or chat client and authenticate there
- Additional configuration details available on Pipedream’s configuration page (not tool-specific in the provided content)
