# CallFire MCP Server

CallFire MCP Server provides access to CallFire’s communication platform for marketing and customer engagement via text and phone through an MCP-compatible client.

---

## Overview
- **Type:** MCP server / integration
- **Platform:** CallFire (SMS and phone communication)
- **Use Cases:** Marketing campaigns, customer engagement, notifications, text and phone-based communication flows
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

---

## Features
- **CallFire integration via MCP**  
  - Connects MCP-compatible chat or agent clients to CallFire’s communication services.
- **Text messaging (SMS) support**  
  - Use CallFire’s SMS capabilities for marketing and customer engagement workflows (e.g., promotions, alerts, reminders).
- **Phone communication support**  
  - Access CallFire’s phone-based communication features for voice campaigns and customer outreach (e.g., calls, phone-based notifications).
- **Static MCP endpoint**  
  - Single static server URL (`https://mcp.pipedream.net/v2`) works across clients.
- **Client-agnostic setup**  
  - Designed to be added to various MCP-compatible chat or agent clients.
- **Authentication at client level**  
  - Authentication is performed when you add the server to your application (supports secure access per client / user).

---

## Setup & Configuration
- Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
- Add this MCP server in your MCP-compatible client, then authenticate with your CallFire / Pipedream-related credentials as prompted.
- For detailed setup steps, refer to the client-specific configuration instructions (via the referenced configuration page in the source site).
