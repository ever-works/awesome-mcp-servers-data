# BrainShop MCP Server

An MCP (Model Context Protocol) server integration for BrainShop that enables conversational AI experiences via a standard MCP endpoint.

## Features
- **MCP-compliant server endpoint**  
  - Single static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Works across all supported MCP clients.

- **Client-agnostic integration**  
  - Can be added to any MCP-compatible chat client or application.  
  - Authentication is performed when adding the server to your app (rather than via per-client URLs).

- **BrainShop conversational AI backend**  
  - Uses BrainShop as the conversational AI provider behind the MCP interface.  
  - Suitable for building chatbot and conversational experiences.

- **Configuration guidance**  
  - Documentation available via a configuration page for setup details (e.g., how to add the server in different clients).
