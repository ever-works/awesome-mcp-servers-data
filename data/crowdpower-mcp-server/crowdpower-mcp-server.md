# CrowdPower MCP Server

Lifecycle automations to keep and convert your customers.

## Overview
CrowdPower MCP Server is an MCP endpoint, powered by Pipedream Connect, that exposes CrowdPower lifecycle automation and customer retention workflows to MCP-compatible clients. It enables applications (such as chat-based clients) to trigger and manage automated customer engagement scenarios through a unified MCP server URL.

## MCP Server URL
- Base MCP server URL (static for all clients):
  - `https://mcp.pipedream.net/v2`
- Authentication is performed when adding the server to your application.

## Features
- **Unified MCP Endpoint**
  - Single static URL that works across compatible MCP clients.
- **Lifecycle Automation Access**
  - Exposes CrowdPower lifecycle automations over MCP for use in client applications.
- **Customer Retention Workflows**
  - Provides access to workflows focused on keeping and converting customers.
- **Integration with Chat Clients**
  - Designed to be added to chat-based or other MCP-aware applications.
- **Configuration Documentation**
  - Dedicated configuration page (via Pipedream) for connection and setup details.
- **Pipedream Connect Integration**
  - Runs on Pipedream Connect infrastructure for MCP server hosting.
