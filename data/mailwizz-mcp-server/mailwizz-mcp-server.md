# Mailwizz MCP Server

Full-featured self-hosted email marketing software exposed as an MCP server.

**Website:** https://mcp.pipedream.com/app/mailwizz  
**MCP Server URL:** `https://mcp.pipedream.net/v2`  
**Category:** Business & Commerce – MCP Servers  
**Tags:** email-marketing, self-hosted, marketing-automation

---

## Overview

Mailwizz MCP Server is an MCP (Model Context Protocol) server that connects Mailwizz’s self-hosted email marketing capabilities to MCP-based automation and orchestration tools. It uses a static MCP endpoint that can be added to compatible chat or automation clients, with authentication handled during client configuration.

---

## Features

- **Self-hosted email marketing integration**  
  - Exposes Mailwizz’s self-hosted email marketing functionality through MCP.
- **Static MCP endpoint**  
  - Uses a single static URL for all clients: `https://mcp.pipedream.net/v2`.
  - Authentication is performed when adding the server to a client.
- **MCP-based automation support**  
  - Designed to plug Mailwizz into MCP-based automation and orchestration tools.
- **Client-agnostic configuration**  
  - Works with multiple MCP-compatible chat clients and tools.  
  - Additional configuration details available via the platform’s configuration page (outside this summary).

---

## Integration

- Add the static MCP server URL to your MCP-compatible app or chat client.
- Authenticate within your client when prompted.
- Optional: reference the platform’s configuration documentation for client-specific setup steps.
