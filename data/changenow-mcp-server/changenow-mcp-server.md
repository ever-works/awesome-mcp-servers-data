# ChangeNOW MCP Server

MCP Server integration for ChangeNOW, enabling AI‑driven access to instant crypto exchange services through the Model Context Protocol.

- **Website / Source**: https://mcp.pipedream.com/app/changenow
- **Category**: Blockchain & Crypto MCP Servers
- **Tags**: crypto, exchange, trading

## Overview
The ChangeNOW MCP Server exposes ChangeNOW’s instant crypto exchange service through a standardized Model Context Protocol (MCP) endpoint. It provides a single, static server URL that can be used by compatible AI/chat clients to interact with ChangeNOW for crypto exchange workflows.

## Features
- **Instant crypto exchange access** via an MCP-compliant server.
- **Static MCP server URL** (`https://mcp.pipedream.net/v2`) that works across all supported clients.
- **Client-agnostic integration**: designed to be added to various chat or AI applications that support MCP.
- **Authentication at client configuration time** (authentication is handled when adding the server to your app, not by changing the URL).
- **Central configuration documentation** via a shared configuration page for setup and usage details.
- **Hosted by Pipedream Connect**, providing managed MCP server infrastructure.

## Integration & Usage
- **Server URL**: `https://mcp.pipedream.net/v2`
- **How to use**:
  - Add the above MCP server URL to your MCP-compatible chat client or application.
  - Complete authentication within your client when prompted.
  - Use the configured server to perform instant crypto exchange operations exposed by ChangeNOW (capabilities depend on your client’s MCP tooling and configuration).
- **Configuration reference**: A full configuration page is available from the app’s source URL for detailed setup steps per client.
