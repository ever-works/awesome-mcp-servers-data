# Vida MCP Server

**Category:** AI Integration – MCP Servers  
**Website:** https://mcp.pipedream.com/app/vida

## Description
Vida MCP Server is a Model Context Protocol (MCP) server that connects your Vida account to MCP-based AI tools, enabling AI phone agent functionality for business workflows.

## Features
- Connects a Vida account to MCP-compatible AI tools.  
- Enables AI phone agent capabilities for business workflows.  
- Works with tools such as Cursor, Claude, and Windsurf via Pipedream’s MCP server.

## Integrations
- Cursor  
- Claude  
- Windsurf  
- Vida (phone AI agent platform)  
- Pipedream MCP server as the integration layer
