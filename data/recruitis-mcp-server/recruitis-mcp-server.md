# Recruitis MCP Server

Recruitis MCP Server provides machine-consumable access to the Recruitis recruiting system via the Pipedream Connect MCP infrastructure, enabling integration of Recruitis into MCP-compatible chat clients and workflows.

## Overview
- **Type:** MCP (Model Context Protocol) server
- **Purpose:** Access and automate Recruitis recruiting / ATS workflows programmatically
- **Base MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Provider:** Recruitis (delivered via Pipedream Connect)

## Features
- **Standard MCP Endpoint**
  - Uses a single static MCP server URL: `https://mcp.pipedream.net/v2`.
  - Same URL works for all clients; authentication is handled when adding the server to your application.

- **Client-Agnostic Integration**
  - Designed to be added to various MCP-compatible chat clients.
  - Users select their chat client and follow client-specific setup instructions.

- **Pipedream Connect Infrastructure**
