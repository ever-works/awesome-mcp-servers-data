# Dropbox Sign MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Dropbox Sign (HelloSign)

## Description
The Dropbox Sign (HelloSign) MCP Server integrates legally binding e‑signature capabilities into MCP-compatible applications, enabling automated preparation, sending, and tracking of electronic signature requests.

## Features
- Provides legally binding electronic signatures.
- Exposes a static MCP server endpoint usable by any compatible client.
- Supports authentication at the time you add the server to your application.
- Integrates with MCP-enabled chat or agent clients for e-signature workflows.
- Enables creation and automation of signature request workflows (preparation, sending, tracking).

## MCP Server URL
- Base MCP server URL (static for all clients):
  - `https://mcp.pipedream.net/v2`

## Integration & Setup
- Add the MCP server URL to any supported MCP/chat client.
- Authentication occurs when configuring/adding the server in your app.
- Additional configuration details are available on the provider’s configuration page.
