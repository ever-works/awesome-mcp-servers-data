# SolarWinds Service Desk MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** SolarWinds  
**Source:** https://mcp.pipedream.com/app/solarwinds_service_desk

## Overview
SolarWinds Service Desk MCP Server is an integration service that connects a SolarWinds Service Desk account to Model Context Protocol (MCP)–compatible AI tools such as Cursor, Claude, and Windsurf. It exposes SolarWinds Service Desk’s IT service management (ITSM) capabilities to AI agents and tools, enabling automated and assisted handling of employee support services.

## Features
- **MCP integration layer**: Implements an MCP server that allows AI tools and agents to interact with SolarWinds Service Desk.
- **SolarWinds Service Desk connectivity**: Connects directly to an existing SolarWinds Service Desk account.
- **AI tool compatibility**:
  - Works with Cursor
  - Works with Claude
  - Works with Windsurf
- **ITSM access for AI agents**: Provides programmatic access to modern ITSM capabilities for employee support workflows (e.g., ticket-based support and related service management operations, as exposed via SolarWinds APIs).

## Use Cases
- Allow AI coding assistants or IDE-integrated tools (e.g., Cursor, Windsurf) to query and act on Service Desk data.
- Let conversational AI (e.g., Claude) assist with creating, updating, or managing support requests in SolarWinds Service Desk.
- Build automated workflows that combine AI reasoning with ITSM operations for employee support.
