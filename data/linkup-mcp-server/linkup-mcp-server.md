# Linkup MCP Server

MCP Server for Linkup, allowing MCP-based AIs to connect to and interact with the web.

- **Website:** https://mcp.pipedream.com/app/linkup  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Category:** Web Search MCP Servers  
- **Tags:** web-search, web, browser-automation

## Features

- **Static MCP Endpoint**  
  - Single static MCP server URL (`https://mcp.pipedream.net/v2`) that works across compatible MCP clients.  
  - Authentication handled when adding the server to an application.

- **AI-to-Web Connectivity**  
  - Designed to let MCP-based AIs connect to and interact with the web.  
  - Suitable for web search and browser-automation style use cases (based on tags).

- **Client-Agnostic Integration**  
  - Server URL usable with multiple chat or MCP client applications.  
  - Configuration guidance available via a dedicated configuration page (per-site link).

- **Pipedream Connect Backend**  
  - Powered by Pipedream Connect, indicating underlying infrastructure for request handling and integration.
