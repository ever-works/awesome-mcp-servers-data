# Endorsal MCP Server

## Overview
Endorsal MCP Server is a Pipedream-hosted Model Context Protocol (MCP) server that connects MCP-compatible agents to the Endorsal platform via Endorsal’s APIs. It enables automated collection and management workflows around customer feedback and review data within compatible chat or agent clients.

- **Type:** MCP server (Pipedream Connect)
- **Category:** Business & Commerce – MCP Servers
- **Brand:** Endorsal
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible endpoint**  
  - Exposes a static MCP server URL (`https://mcp.pipedream.net/v2`) that can be reused across different MCP clients.  
  - Authentication is handled when adding the server to a client application.

- **Integration with MCP agents**  
  - Allows MCP-compatible chat and AI agents to interact with Endorsal via Pipedream Connect.  
  - Designed to be added as a server in supported chat clients and agent frameworks.

- **Endorsal API access layer**  
  - Provides an integration surface for agents to work with customer feedback and review data through Endorsal’s APIs.  
  - Centralizes access so multiple clients can use the same MCP endpoint configuration.

- **Configuration resources**  
  - Option to follow client-specific setup guides by selecting a chat client.  
  - Additional configuration details available via a dedicated configuration page (linked from the app page).

- **Hosted by Pipedream Connect**  
  - Managed and delivered via Pipedream’s integration platform.  
  - Subject to Pipedream’s Terms and Privacy Policy.
