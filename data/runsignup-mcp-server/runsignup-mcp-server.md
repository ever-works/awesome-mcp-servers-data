# RunSignup MCP Server

**Category:** Business & Commerce · MCP Servers  
**Brand:** RunSignup  
**Source:** https://mcp.pipedream.com/app/runsignup

RunSignup MCP Server exposes RunSignup’s endurance event management capabilities through the Model Context Protocol (MCP), allowing AI tools and MCP-compatible clients to interact programmatically with race registrations, participants, and events.

---

## Features

- **MCP-based integration**
  - Provides a static MCP server endpoint: `https://mcp.pipedream.net/v2`.
  - Can be added to any compatible chat client or MCP-enabled application.

- **RunSignup account connection**
  - Connects to a RunSignup account through Pipedream’s configuration flow.
  - Uses your configured RunSignup client once the account is connected.

- **Endurance event management via AI tools**
  - Exposes RunSignup capabilities related to endurance events (e.g., races and similar events).
  - Designed for interacting with race registrations, participants, and event data programmatically through AI or other MCP clients.

- **Central configuration page**
  - Configuration and setup are managed via Pipedream’s Configuration page.
  - Client-specific setup instructions are available for different chat clients / MCP consumers.

> Note: The page references “Available tools” and “Loading actions…”, indicating that specific tools/actions (e.g., for registrations, participants, events) are dynamically loaded, but the exact list is not visible in the provided content.
