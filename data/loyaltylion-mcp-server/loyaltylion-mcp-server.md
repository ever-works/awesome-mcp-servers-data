# LoyaltyLion MCP Server

Customer loyalty and engagement platform MCP server that exposes LoyaltyLion functionality to MCP-compatible tools and chat clients.

## Overview
- **Name:** LoyaltyLion MCP Server  
- **Category:** Business & Commerce MCP Servers  
- **Use Case:** Enable MCP tools and chat applications to interact with LoyaltyLion’s customer loyalty and engagement features and data.  
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- MCP-compatible server interface for LoyaltyLion.  
- Single static MCP server URL that works across clients.  
- Authentication handled when adding the server within your client or application.  
- Designed to integrate LoyaltyLion loyalty and customer engagement capabilities into chat-based and MCP-enabled tools.

## Integration & Usage
- Add the MCP server using the static URL: `https://mcp.pipedream.net/v2`.  
- Configure within your MCP-compatible chat client or application (client-specific steps provided by the client or the configuration documentation).  
- Full configuration details available via the provider’s configuration page (referenced as “Configuration page” in the source content).
