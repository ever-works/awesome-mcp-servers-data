# DataScope MCP Server

**Description**  
DataScope MCP Server connects MCP-compatible clients to the DataScope platform for mobile form-based data collection and field workflows. It enables teams to complete customized mobile forms, including offline, from phones or tablets and integrate that data into automated workflows.

**Category**  
- Workflow Automation MCP Servers

**Tags**  
- Forms  
- Data Collection  
- Field Service

---

## Features

- **MCP-compatible server endpoint**  
  - Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`  
  - Usable with any MCP-supporting chat or automation client.

- **Authentication at client setup**  
  - Authentication handled when adding the server to your MCP client or application.

- **Mobile form interaction**  
  - Connects MCP clients to DataScope’s personalized mobile forms.  
  - Supports answering forms directly from mobile devices (phones or tablets).

- **Offline data collection support**  
  - Designed to work with forms that can be filled even when devices are offline, enabling field data capture without connectivity.

- **Field data collection workflows**  
  - Integrates DataScope’s field data collection into automated workflows via MCP.  
  - Suitable for replacing paper-based processes with digital capture.

- **Configuration guidance**  
  - Works with multiple chat clients; configuration instructions available via the platform’s configuration documentation.
