### MCP Community Discord (Glama)

**Category:** MCP server directories & lists  
**Website:** https://glama.ai/mcp/discord

**Overview**  
A community Discord server for people interested in the Model Context Protocol (MCP), hosted by Frank Fiegel (punkpeye) via Glama. The server focuses on discussion around MCP, sharing MCP servers, and related tools. The community has around **10,799 members**.

#### Features
- Discord-based community centered on the Model Context Protocol (MCP)
- Space for discussion of MCP concepts, implementations, and usage
- Channels for sharing MCP servers
- Channels for sharing tools and integrations related to MCP
- Large and active member base (approx. 10k+ members)
