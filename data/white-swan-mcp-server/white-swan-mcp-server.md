# White Swan MCP Server

**Category:** Business & Commerce MCP Servers  
**Tags:** insurance, finance, advisory  
**Source:** https://mcp.pipedream.com/app/white_swan

## Overview
White Swan MCP Server is an MCP integration that connects to White Swan’s digital life insurance platform and advisory workflows. It exposes life insurance–related actions as tools that can be invoked from MCP-compatible chat or agent applications.

## Features
- **Static MCP Server Endpoint**  
  - Uses a single static URL for all clients: `https://mcp.pipedream.net/v2`  
  - Authentication occurs when adding/configuring the server in your application.

- **Submit Complete Plan Request**  
  - Creates a new comprehensive life insurance quote request.  
  - Uses provided client information to generate a final quotation.  
  - Designed to avoid additional data requirements after submission.

- **Get Referred Clients Info**  
  - Retrieves information about clients referred from the connected White Swan account.  
  - Useful for viewing and managing referred client records within MCP-enabled tools.

- **Create Pre-fill Info**  
  - Imports client data for pre-filling insurance applications.  
  - Helps streamline and enrich the end-user application experience.

- **Client & Account Connection**  
  - Connect a White Swan account through the Pipedream MCP interface.  
  - Select specific clients to operate on when using the tools.

- **Integration with Chat/Agent Apps**  
  - Can be added as a server to MCP-compatible chat clients and applications.  
  - Tools are exposed as actions that can be invoked programmatically or via conversational interfaces.
