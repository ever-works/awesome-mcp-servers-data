# Proxycurl MCP Server

Pull rich people and company data into MCP-compatible clients using Proxycurl’s enrichment APIs.

## Overview

Proxycurl MCP Server exposes Proxycurl’s people and company enrichment APIs to MCP agents, enabling applications and chat clients to retrieve detailed professional profiles and firmographic information through a standardized MCP interface.

- **Type:** MCP server / integration
- **Category:** Business & Commerce – MCP Servers
- **Source URL:** https://mcp.pipedream.com/app/proxycurl
- **MCP Endpoint URL:** `https://mcp.pipedream.net/v2`

## Features

- **MCP-compatible server**
  - Provides a static MCP server URL (`https://mcp.pipedream.net/v2`) usable across compatible MCP clients.
  - Can be added to any supported chat client or MCP-enabled application.

- **Proxycurl data enrichment integration**
  - Connects MCP agents to Proxycurl’s people enrichment APIs for rich professional profiles.
  - Connects MCP agents to Proxycurl’s company enrichment APIs for firmographic data.

- **Centralized configuration**
  - Authentication is handled when adding the server to an application or client.
  - Additional setup and usage details available via a dedicated configuration page (through Pipedream).

- **Pipedream Connect platform**
  - MCP server is hosted and powered by Pipedream Connect.
  - Benefits from Pipedream’s infrastructure and terms / privacy framework.

## Use Cases

- Enriching contact records with detailed professional information.
- Retrieving firmographic data for prospecting or account research.
- Powering MCP-based assistants that need live people/company data.
