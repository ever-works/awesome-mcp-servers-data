## Description
Waboxapp MCP Server is an MCP integration that connects the Waboxapp API to AI tools, enabling access to WhatsApp and WhatsApp Business messaging capabilities within automated workflows.

- Designed to operate within the Model Context Protocol (MCP) ecosystem

- Allowing AI tools to interact with users or systems through WhatsApp channels
- Building automated messaging flows that combine AI reasoning with WhatsApp delivery
