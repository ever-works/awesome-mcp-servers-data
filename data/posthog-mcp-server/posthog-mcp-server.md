# PostHog MCP Server

**Category:** Data Analysis & Exploration MCP Servers  
**Brand:** PostHog  
**Slug:** `posthog-mcp-server`

## Description
PostHog MCP Server is an MCP-compatible server that exposes PostHog’s product analytics, session recording, feature flags, and experimentation capabilities to AI agents and tools. It is available via Pipedream and allows connecting a PostHog account to AI development environments like Cursor, Claude, and Windsurf.

Source: <https://mcp.pipedream.com/app/posthog>

## Features
- **MCP-compatible server**  
  - Implements the Model Context Protocol (MCP) to integrate with MCP-aware tools and agents.

- **PostHog integration**  
  - Connects directly to a PostHog account via Pipedream’s MCP server.

- **Product analytics access**  
  - Exposes PostHog’s product analytics data to AI tools for querying and analysis.

- **Session recording capabilities**  
  - Provides access to session recording data from PostHog for inspection and analysis in compatible tools.

- **Feature flags management**  
  - Surfaces PostHog’s feature flag capabilities so agents/tools can read or act based on feature flag states.

- **Experimentation support**  
  - Exposes PostHog’s experimentation / A/B testing capabilities to AI tools for data-driven experimentation workflows.

- **AI tool connectivity**  
  - Designed to work with AI-driven environments such as:
    - Cursor
    - Claude
    - Windsurf
