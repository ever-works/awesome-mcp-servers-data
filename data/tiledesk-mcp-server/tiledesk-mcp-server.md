# Tiledesk MCP Server

## Overview
Tiledesk MCP Server is an MCP server that connects MCP-compatible agents with Tiledesk, allowing them to build and interact with chatbots and conversational applications through the Tiledesk platform.

- **Name:** Tiledesk MCP Server  
- **Category:** Messaging MCP Servers  
- **Use cases:** Chatbots, conversational experiences, customer support automation

## Features
- Provides an MCP endpoint for integrating Tiledesk with MCP-compatible clients and agents.
- Enables agents to build and interact with Tiledesk-powered chatbots and conversational apps.
- Uses a single static MCP server URL for all supported clients:  
  - `https://mcp.pipedream.net/v2`
- Supports authentication at the time you add the server within your MCP client or application.
- Documentation and configuration guidance available via a dedicated configuration page (linked from the app page).

## Setup
1. Copy the MCP server URL:  
   `https://mcp.pipedream.net/v2`
2. Add the MCP server to your preferred MCP-compatible chat client or application.
3. Authenticate within your client when prompted.
4. Follow the configuration documentation provided via the linked configuration page.
