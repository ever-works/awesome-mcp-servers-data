# PayPro MCP Server

Online payment tools MCP server exposing PayPro’s payment and commerce capabilities through the Model Context Protocol (MCP).

## Overview
- **Type:** MCP server (PayPro integration)
- **Category:** Business & Commerce / MCP Servers
- **Provider:** PayPro (via Pipedream Connect)
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- **Online payment processing** via PayPro through a unified MCP interface
- **Multiple payment methods** support (through PayPro’s payment tools)
- **Recurring payments** for subscriptions or ongoing billing
- **Upsell flows** to add additional products/services during or after checkout
- **Affiliate marketing integration** to track and manage affiliate-driven sales
- **MCP-based access** so chat clients and MCP-compatible applications can call PayPro tools programmatically
- **Static server URL** usable across different MCP clients, with authentication handled on connection

## Integration & Usage
- Use the static MCP server URL: `https://mcp.pipedream.net/v2`
- Add as an MCP server in compatible chat clients or applications
- Configuration details are available on the referenced configuration page of the provider
