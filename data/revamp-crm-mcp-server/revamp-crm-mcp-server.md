# Revamp CRM MCP Server

Sales & Marketing Automation CRM for small business, exposed as an MCP Server.

## Overview
Revamp CRM MCP Server integrates Revamp CRM into MCP-compatible clients via a static server endpoint. It provides access to Revamp CRM’s sales and marketing automation capabilities for small businesses directly within MCP-enabled applications.

## MCP Server URL
- Base MCP server URL (static for all clients):
  - `https://mcp.pipedream.net/v2`
- Authentication is performed when adding the server to your application.

## Features
- **MCP Integration**
  - Exposes Revamp CRM as an MCP Server endpoint.
  - Uses a single static URL for all clients.
  - Compatible with multiple MCP-capable chat clients and tools.

- **Sales & Marketing CRM Focus**
  - Designed around sales and marketing automation workflows for small businesses (as supported by Revamp CRM’s platform).

- **Configuration Support**
  - Can be added to supported clients following their specific MCP configuration instructions.
  - A general configuration guide is available via the platform’s configuration page (outside this summary).
