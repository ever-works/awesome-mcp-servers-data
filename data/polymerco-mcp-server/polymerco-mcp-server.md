# Polymer.co MCP Server

## Overview
The Polymer.co MCP Server is a Model Context Protocol (MCP) server that connects Polymer’s job board and applicant tracking system (ATS) to MCP-compatible applications. It allows chat and AI clients to access and work with job postings and candidate management features via a standardized MCP endpoint.

- **Category:** Business & Commerce – MCP Servers
- **Use cases:** Recruiting, HR, applicant tracking, candidate relationship management via MCP
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-based access to Polymer.co**  
  Provides a static MCP server endpoint (`https://mcp.pipedream.net/v2`) that can be added to MCP-compatible applications.

- **Job board integration**  
  Connects to Polymer’s job board functionality, which is designed to host and display job openings on a dedicated site.

- **Applicant Tracking System (ATS)**  
  Integrates with Polymer’s applicant tracking capabilities, enabling structured handling of candidates and their applications.

- **Candidate Relationship Management**  
  Supports features related to candidate relationship management (CRM) within the ATS context, such as tracking candidate interactions and status over time.

- **Client selection and configuration**  
  Once an account is connected, users can select the appropriate client (e.g., specific Polymer site or instance) within the configuration flow.

- **Static server URL with per-app authentication**  
  The same MCP server URL is used for all clients; authentication is handled when adding the server to each application, enabling multi-client use with a single endpoint.

- **Compatibility with multiple chat clients**  
  Provides instructions (via the host platform) for adding the server to different chat or AI clients that support MCP.
