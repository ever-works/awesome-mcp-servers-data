# Amplitude MCP Server

**Category:** Business & Commerce MCP Servers  
**Slug:** amplitude-mcp-server  
**Provider / Brand:** Amplitude (via Pipedream Connect)

## Description
The Amplitude MCP Server exposes Amplitude’s product analytics and experimentation data through the MCP protocol, allowing compatible clients (such as AI/chat applications and other MCP-enabled tools) to query user behavior and experimentation insights programmatically.

## MCP Server URL
- Static MCP server endpoint (for all clients):  
  `https://mcp.pipedream.net/v2`  
  Authentication is handled when you add the server to your application.

## Features
- **Access to Amplitude analytics data**  
  - Programmatic access to Amplitude’s digital analytics data via MCP.  
  - Enables querying of user behavior and product usage insights from MCP-compatible clients.

- **Experimentation data exposure**  
  - Surfaces Amplitude experimentation data (e.g., experiment metrics and results) for analysis or decision support within MCP clients.

- **MCP client integration**  
  - Designed to be added as an MCP server to any compatible chat or tooling client.  
  - Uses a single static base URL for easier configuration across multiple clients.

- **Pipedream Connect integration**  
  - Implemented and hosted via Pipedream Connect’s infrastructure.  
  - Uses Pipedream’s platform for connectivity and management of the MCP server.

## Configuration & Setup
- Add the MCP server using the static URL: `https://mcp.pipedream.net/v2`.  
- Authentication occurs during server addition within your MCP-enabled application.  
- Additional setup and configuration details are available on Pipedream’s configuration documentation (not specific to this summary).
