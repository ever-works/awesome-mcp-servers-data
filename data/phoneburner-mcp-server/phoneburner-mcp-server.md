# PhoneBurner MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** PhoneBurner  
**Slug:** phoneburner-mcp-server

## Description
PhoneBurner MCP Server is an MCP (Model Context Protocol) server integration that connects AI agents or tools to PhoneBurner’s outbound dialing platform. It enables automated and assisted interactions with PhoneBurner to support outbound calling, improve contact rates, and streamline sales workflows.

## Features
- **MCP Server Integration**: Exposes PhoneBurner’s outbound dialing capabilities via a standard MCP server endpoint.
- **Static MCP Endpoint**: Uses a single, static server URL for all clients:
  - `https://mcp.pipedream.net/v2`
- **Authentication at Client Setup**: Authentication is performed when adding the server to an MCP-compatible application or chat client.
- **AI / Tool Connectivity**: Designed so AI agents or tools can interact programmatically with PhoneBurner’s outbound dialing platform (e.g., for sales or contact-center workflows).
- **Outbound Dialing Platform Access**: Integrates with PhoneBurner’s dialer to support higher contact rates and sales-focused outbound call operations.
- **Client-Agnostic Setup**: Can be added to different MCP-compatible chat clients or applications using the same server URL.
- **Configuration Documentation**: Accompanied by a configuration guide (via the “Configuration” page) for setup and integration details.

## Technical Details
- **MCP Server URL**: `https://mcp.pipedream.net/v2`
- **Integration Provider**: Built and hosted via Pipedream Connect.
