# Rejoiner MCP Server

**Category:** Business & Commerce · MCP Servers  
**Tags:** e-commerce, marketing, automation

Rejoiner MCP Server is an MCP (Model Context Protocol) server integration provided via Pipedream that allows MCP-compatible agents or chat clients to interact with Rejoiner’s ecommerce retention marketing software. It enables configuration and control of Rejoiner flows through a standard MCP endpoint.

---

## Features

- **MCP-compatible server endpoint**  
  - Provides a static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Usable across all Rejoiner clients; authentication occurs when adding the server to an application.

- **Rejoiner account connection**  
  - Connect a Rejoiner account through the Pipedream interface.  
  - Select the relevant client within the connected Rejoiner account to begin using tools.

- **Integration with chat clients / agents**  
  - Designed to be added to MCP-compatible chat clients or agent frameworks.  
  - Lets those agents drive and analyze ecommerce retention marketing flows via Rejoiner.

- **Tooling exposure via MCP**  
  - Exposes Rejoiner-related “tools” (actions) to MCP clients (e.g., for automation and analysis).  
  - Tools are discovered and loaded dynamically by the MCP client (e.g., “Loading actions…”, “Loading available tools…” in the interface).

- **Centralized configuration page**  
  - Configuration and setup details available via a dedicated configuration page within Pipedream (referenced as a “full Configuration page”).

---

## Setup & Configuration

1. Connect your Rejoiner account in the Pipedream-based UI.  
2. Select the specific Rejoiner client you want to work with.  
3. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.  
4. Add this MCP server URL to your MCP-compatible application or chat client.  
5. Authenticate when prompted by your MCP client to begin using Rejoiner tools.
