# SendPulse MCP Server

**Category:** Business & Commerce · MCP Servers  
**Slug:** sendpulse-mcp-server  
**Brand:** SendPulse

An MCP (Model Context Protocol) server that connects to SendPulse, a multi‑channel marketing platform for email, SMS, and other communication channels.

---

## Features

- Provides an MCP server endpoint for integrating SendPulse into compatible chat or AI clients.
- Uses a static base URL for all clients: `https://mcp.pipedream.net/v2`.
- Authentication is performed when adding/configuring the server in the client application (per‑client auth, not per‑URL).
- Built and hosted via Pipedream Connect infrastructure.
- Supports interaction with SendPulse’s multi‑channel marketing capabilities (email, SMS, and other channels) through the MCP interface.

---

## Technical Details

- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Integration Flow:**
  - Add the MCP server URL to a compatible chat/AI client.
  - Authenticate to SendPulse during configuration in that client.
