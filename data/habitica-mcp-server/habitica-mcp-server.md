# Habitica MCP Server

**Category:** Project Management MCP Servers  
**Brand:** Habitica  
**Slug:** `habitica-mcp-server`

## Overview
The Habitica MCP Server connects MCP-compatible clients to Habitica, a gamified habit and productivity platform. It enables task and habit management through Habitica directly within MCP workflows via a single shared server endpoint.

## Server URL
- **MCP Server Endpoint:** `https://mcp.pipedream.net/v2`
- This is a static URL used by all compatible clients.
- Authentication is performed when adding the server to your application.

## Features
- **Habitica Integration**
  - Connects your MCP client to Habitica, a habit and productivity app that treats real life like a game.
  - Intended for managing tasks, habits, and goals within Habitica from MCP-based tools.

- **Single Shared Endpoint**
  - Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) for all supported clients.
  - Simplifies configuration across different chat or MCP clients.

- **Client-Agnostic Setup**
  - Works with multiple MCP-compatible chat clients.
  - Setup instructions are available per client (via “Select your chat client to learn how to get started”).

- **Central Configuration Docs**
  - Additional configuration details are available via a dedicated configuration page (linked from the app page).
