# Genderize MCP Server

An MCP (Model Context Protocol) server that exposes the Genderize API for determining likely gender from a given name.

---

## Basic Info

- **Name:** Genderize MCP Server  
- **Category:** AI Integration – MCP Servers  
- **Brand:** Genderize  
- **Source URL:** https://mcp.pipedream.com/app/genderize  
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`  

---

## Features

- **Gender classification from names**  
  - Uses the Genderize API to infer gender based on a provided name.
  - Designed to return an accurate gender classification where possible.

- **Model Context Protocol (MCP) compatible**  
  - Exposed as an MCP server so it can be integrated into MCP-capable chat clients and tools.

- **Static MCP server URL**  
  - Single endpoint (`https://mcp.pipedream.net/v2`) works across supported clients.
  - Authentication is handled when adding the server within the client or application.

- **Client-agnostic integration**  
  - Can be added to multiple chat clients that support MCP.  
  - Configuration guidance available via a generic “Configuration” page (no client lock-in implied).

- **Hosted by Pipedream Connect**  
  - The MCP server is provided and hosted via Pipedream’s Connect infrastructure.

---

## Integration & Usage

- **How to use**  
  1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.  
  2. Add this server to your MCP-compatible chat client or application.  
  3. Authenticate within your client when prompted.  
  4. Use the provided tools/endpoints to classify gender from input names.

- **Configuration Documentation**  
  - Additional configuration and setup details are available on the Pipedream Connect configuration page (linked from the app page).
