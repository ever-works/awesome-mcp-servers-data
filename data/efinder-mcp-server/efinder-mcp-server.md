# efinder MCP Server

Free email finder and enrichment MCP server that runs on Pipedream Connect.

## Overview
The efinder MCP Server integrates with efinder to perform free email discovery and enrichment within automated workflows via the Model Context Protocol (MCP). It is exposed through a static MCP server endpoint that can be added to compatible chat or AI clients.

- **Category:** Business & Commerce – MCP Servers  
- **Provider / Brand:** Pipedream  
- **Integration:** efinder (email discovery and enrichment)

## MCP Server URL
- **Base URL:** `https://mcp.pipedream.net/v2`  
- This is a **static URL** shared across clients; authentication is handled when adding the server to your specific application or client.

## Features
- **Free email discovery**: Find email addresses associated with people or leads.  
- **Email enrichment**: Enhance lead or contact data using efinder (e.g., for marketing and lead-enrichment workflows).  
- **MCP-compatible server**: Exposes efinder functionality as an MCP Server for use in MCP-aware apps and chat clients.  
- **Static server endpoint**: Single, reusable URL (`https://mcp.pipedream.net/v2`) for all supported clients.  
- **Authentication at client setup**: Credentials are applied when adding the server to your chosen app, keeping the URL itself static.  
- **Works within automated workflows**: Designed to plug into automated processes (e.g., lead generation, marketing automation) via Pipedream Connect.

## Usage
- Add `https://mcp.pipedream.net/v2` as an MCP server in your compatible chat/AI client.  
- Follow that client’s specific instructions for MCP configuration and authentication.  
- Optionally, refer to the full **Configuration** documentation on Pipedream for detailed setup steps.
