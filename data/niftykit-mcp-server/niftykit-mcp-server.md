# NiftyKit MCP Server

## Overview
NiftyKit MCP Server is an MCP-based integration that connects NiftyKit’s no-code platform for building and launching projects (such as NFT and web3 initiatives) to MCP-compatible chat clients and applications. It exposes NiftyKit’s tools through a standardized MCP server endpoint so they can be accessed programmatically via supported clients.

- **Category:** Blockchain & Crypto MCP Servers  
- **Provider / Brand:** NiftyKit (hosted via Pipedream)  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server endpoint**  
  - Static MCP URL (`https://mcp.pipedream.net/v2`) usable across supported clients.  
  - Authentication handled when adding the server to your application.

- **Access to NiftyKit’s no-code tools**  
  - Interfaces with NiftyKit’s platform for building, collaborating on, and launching projects without coding.  
  - Suitable for NFT and broader web3 project workflows.

- **Multi-client support**  
  - Designed to be added to various MCP-capable chat or AI clients.  
  - Client-specific setup instructions available via the configuration flow.

- **Central configuration flow**  
  - Connect a NiftyKit account and select a client to begin using the server.  
  - Single configuration process that then works across chosen MCP clients.

## Installation / Usage
1. **Connect NiftyKit account** within the provided configuration interface.  
2. **Copy the MCP server URL:** `https://mcp.pipedream.net/v2`.  
3. **Add the server to your MCP-compatible app or chat client** using its “add server” or equivalent configuration option.  
4. **Authenticate** when prompted in your application.  
5. **Use exposed tools** once the actions/tools list loads in the client.
