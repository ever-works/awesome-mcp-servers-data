## Overview

MineRL provides environments and datasets for MCP-based AI research.

## Features

- Gym-style APIs
- Mission tasks
- Datasets for RL and imitation learning

## Installation

pip install minerl

## Usage

```python
import minerl
env = minerl.make('MineRLNavigateDense-v0')
obs = env.reset()
```
