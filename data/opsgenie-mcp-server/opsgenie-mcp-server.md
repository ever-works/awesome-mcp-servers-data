# Opsgenie MCP Server

**Category:** Monitoring  
**Tags:** incident-management, alerting

## Overview
Opsgenie MCP Server is an MCP (Model Context Protocol) server integration that lets AI agents interact with the Atlassian Opsgenie incident management platform. It is designed to support alerting and incident operations by exposing Opsgenie functionality through MCP tools.

The server is provided via a static MCP endpoint hosted by Pipedream.

## Features
- **MCP Server for Opsgenie**  
  - Exposes Opsgenie incident management capabilities to AI agents via MCP tools.
  - Intended for working with alerts and incident operations within Opsgenie.

- **Static MCP Endpoint**  
  - Single, static URL for all compatible MCP clients:  
    `https://mcp.pipedream.net/v2`
  - Authentication is handled when adding the server to your application or chat client.

- **Client-Agnostic Integration**  
  - Can be added to different MCP-compatible chat clients / applications using the same URL.
  - Central configuration guidance available via a dedicated Configuration page.

- **Hosted by Pipedream**  
  - Delivered through Pipedream Connect infrastructure.
  - Centralized terms, privacy, and cookie management via Pipedream policies.

## Setup & Configuration
- Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server to your MCP-compatible app or chat client.
- Follow client-specific instructions or refer to the full configuration page linked from the product page.
