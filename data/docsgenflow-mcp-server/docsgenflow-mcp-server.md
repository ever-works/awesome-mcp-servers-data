# DocsGenFlow MCP Server

Document Generation & Automation Software exposed via the Model Context Protocol (MCP).

**Website:** https://mcp.pipedream.com/app/docsgenflow  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

---

## Overview

DocsGenFlow MCP Server makes DocsGenFlow’s document generation and automation capabilities available to MCP-compatible chat clients and applications through a single static MCP endpoint. Authentication is handled when adding the server in the client.

---

## Features

- **MCP Integration**
  - Exposes DocsGenFlow’s capabilities as an MCP server.
  - Uses a single, static MCP URL that works across compatible clients.
  - Authentication performed when the server is added to the application.

- **Document Generation**
  - Supports templated document generation workflows (per item description).
  - Supports dynamic document generation workflows (per item description).

- **Document Automation & Workflows**
  - Enables automated document workflows via DocsGenFlow (per item description).
  - Designed to be orchestrated within MCP-aware chat clients and tools.

- **Client-Agnostic Endpoint**
  - One MCP server URL (`https://mcp.pipedream.net/v2`) works for every supported client.
  - Configuration handled via each client’s MCP settings.
  - Additional configuration details available on a central configuration page (linked from the app page).

- **Pipedream Integration**
  - Hosted and provided through Pipedream Connect.
  - Inherits general platform terms, privacy, and cookie policies from Pipedream.

---

## Configuration

- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Setup Flow:**
  - Copy the MCP server URL.
  - Add it inside your MCP-capable chat client or application.
  - Authenticate when prompted by the client.
  - Optionally refer to the full configuration page linked from the app entry.
