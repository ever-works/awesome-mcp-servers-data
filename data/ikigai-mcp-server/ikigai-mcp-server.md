## Description
Ikigai MCP Server exposes generative AI capabilities over enterprise data through MCP (Model Context Protocol) endpoints, enabling integration with Pipedream-based workflows and compatible chat clients.

## Features
- Provides generative AI access over enterprise data.  
- Exposes capabilities via MCP endpoints compatible with MCP clients.  
- Delivered through Pipedream Connect infrastructure.  
- Uses a static MCP server URL that works with any supported client.  
- Authentication occurs when adding the server to your application.  
- Can be integrated into chat-based applications and workflows.  
- Configuration details available via a dedicated configuration page (client-specific setup instructions).

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Integration Context:** Designed to be added as an MCP server in supported chat clients and Pipedream workflows.
