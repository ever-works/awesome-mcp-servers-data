# Act! MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Act! 365  

An MCP (Model Context Protocol) server for Act! 365 that connects its sales and marketing tools to automated workflows and other MCP-compatible services.

---

## Overview

Act! 365 MCP Server exposes Act! 365 CRM, sales, and marketing capabilities to MCP-compatible clients, allowing you to integrate Act! 365 data and actions into automated workflows and chat-based tools. It is designed as a companion to Microsoft Office 365 so you can work from your inbox, the Act! app, or other connected interfaces.

---

## Features

- **MCP-compatible API endpoint**  
  - Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`  
  - Can be added to any MCP-aware chat client or application.

- **Act! 365 integration**  
  - Connects Act! 365 sales tools (e.g., deal management, pipeline activities) into automated workflows.  
  - Connects Act! 365 marketing tools (e.g., campaigns, lead management) into MCP-compatible services.

- **Workflow automation**  
  - Enables using Act! 365 data and actions within automated workflows orchestrated via MCP.  
  - Supports integration with other MCP-compatible services for cross-app workflows (e.g., CRM + email, CRM + other business tools).

- **Office 365 companion usage**  
  - Intended to work alongside Microsoft Office 365 so users can stay productive from their inbox or other Office 365 contexts.

- **Client-agnostic configuration**  
  - Single, static URL works for every supported MCP client; authentication is handled when adding the server to the chosen application.  
  - Supports configuration via a general MCP configuration flow (referenced "Configuration" page).

---

## Integration & Usage

- Add the MCP server URL (`https://mcp.pipedream.net/v2`) to your MCP-compatible chat client or application.
- Authenticate with your Act! 365 account when prompted by the client.
- Use the client’s interface to invoke Act! 365-related tools and workflows exposed through the MCP server.
