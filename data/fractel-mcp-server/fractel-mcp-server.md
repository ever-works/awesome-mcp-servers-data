# FracTEL MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Pipedream  
**Slug:** `fractel-mcp-server`

## Overview
FracTEL MCP Server is an MCP (Model Context Protocol) server integration that connects Pipedream-based workflows with FracTEL’s telephony and communication services. It provides a static MCP endpoint that can be added to compatible chat or AI clients to enable telephony and communications features via FracTEL.

- **Description:** An MCP Server integration for FracTEL on Pipedream to connect workflows with FracTEL’s telephony and communication services.
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP integration for FracTEL**: Exposes FracTEL’s telephony and communications capabilities through the MCP protocol.
- **Static MCP endpoint**: Uses a single, static server URL (`https://mcp.pipedream.net/v2`) that works for all clients.
- **Per-client authentication**: Authentication is handled when adding the server to each application / client (client-specific auth rather than URL-specific).
- **Pipedream-powered integration**: Runs on Pipedream Connect infrastructure for integrating with workflows.
- **Multi-client compatibility**: Designed to be added to various chat or AI clients that support MCP servers.
- **Centralized configuration docs**: Can be configured using the general Pipedream MCP configuration guidance (via the linked Configuration page).

## Usage
- Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server to a supported chat or AI client as an MCP server.
- Complete authentication within the client when prompted.
