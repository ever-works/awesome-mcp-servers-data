# RiskAdvisor MCP Server

**Category:** Business & Commerce – MCP Servers  
**Website / Source:** https://mcp.pipedream.com/app/riskadvisor  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Overview
RiskAdvisor MCP Server connects RiskAdvisor’s smart-form insurance workflows—covering data collection, risk assessment, and documentation—to MCP‑compatible agents and chat-based applications. It allows insurance agencies to integrate RiskAdvisor’s structured workflows into conversational interfaces used by producers, virtual assistants, or internal tools.

## Features
- **MCP-compatible server endpoint**  
  - Provides a static MCP server URL (`https://mcp.pipedream.net/v2`) usable across clients.  
  - Authentication handled when adding the server to the client application.

- **Integration with RiskAdvisor smart-form workflows**  
  - Connects to RiskAdvisor’s insurance smart forms for structured **data collection**.  
  - Enables **risk assessment** flows within MCP-compatible agents.  
  - Supports **documentation** workflows for insurance use cases.

- **Chat and agent integration**  
  - Designed to be added to chat clients and agent frameworks that support MCP.  
  - Lets conversational agents leverage RiskAdvisor workflows to gather insurance information and perform risk-related tasks.

- **Pipedream Connect platform**  
  - Hosted and powered by Pipedream Connect, allowing configuration through Pipedream’s MCP infrastructure.  
  - Central configuration page available for setup and management (via the Pipedream configuration interface).

## Use Cases
- Embedding RiskAdvisor-driven insurance intake forms into chatbots used by producers or virtual assistants.  
- Running risk assessment workflows from within agentic AI tools.  
- Generating and managing insurance documentation via conversational interfaces.
