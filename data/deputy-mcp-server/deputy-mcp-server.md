# Deputy MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Deputy  
**Slug:** deputy-mcp-server

Deputy MCP Server connects MCP-compatible tools to Deputy’s employee scheduling, timesheets, tasking, and communication platform to support automated workforce operations.

---

## Features

- **MCP-compatible server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Designed to work with any MCP-capable client; authentication is handled when adding the server to the client.

- **Integration with Deputy platform**  
  - Connects to Deputy’s employee scheduling features.  
  - Connects to Deputy’s timesheets and time-tracking capabilities.  
  - Connects to Deputy’s task management features.  
  - Connects to Deputy’s communication tools for workforce coordination.

- **Client-agnostic setup**  
  - Usable from multiple chat or MCP clients.  
  - Configuration guidance available via a general configuration page (client-specific setup steps are referenced but not detailed in the provided content).

- **Built on Pipedream Connect**  
  - Server is hosted and provided via Pipedream’s integration infrastructure.
