## Overview

The MCP (Model Context Protocol) Server Plugin for Jenkins implements the server-side component of the Model Context Protocol, enabling Jenkins to act as an MCP server. This plugin provides context, tools, and capabilities to MCP clients such as LLM-powered applications or IDEs, allowing natural language interaction with Jenkins jobs and builds.

## Features

- **Job Management**: Create, configure, and manage Jenkins jobs through AI assistants
- **Build Operations**: Trigger, monitor, and control build executions
- **Pipeline Interaction**: Work with Jenkins pipelines using natural language
- **Build History**: Query build history and analyze results
- **Natural Language Queries**: Ask questions about job status and build outcomes
- **GitHub Copilot Integration**: Interact with Jenkins using GitHub Copilot
- **LLM Integration**: Connect to various LLM-powered tools and IDEs

## Technical Implementation

Implements the Model Context Protocol server specification, making Jenkins tools and data accessible to MCP clients while maintaining data privacy and security.

## Supported Clients

GitHub Copilot, Claude, Cursor, Windsurf, and any MCP-compatible AI assistant or IDE.

## Authentication

Uses Jenkins authentication and authorization mechanisms for secure access control.
