## Description
An MCP server that integrates WhatsApp messaging via the Online Live Support platform, enabling WhatsApp message automation and handling inside MCP-compatible workflows and chat clients.

## Features
- **WhatsApp integration via Online Live Support**: Connects WhatsApp messaging to MCP-enabled applications using the Online Live Support platform.
- **Static MCP server endpoint**: Uses a single server URL for all clients:
  - `https://mcp.pipedream.net/v2`
- **Per-client authentication**: Authentication occurs when adding the server to an MCP-compatible application or chat client.
- **Multi-client support**: Can be added to different chat clients; configuration guidance is available per client.
- **Workflow integration**: Designed to be used as an MCP server within broader MCP workflows for messaging automation and handling.
