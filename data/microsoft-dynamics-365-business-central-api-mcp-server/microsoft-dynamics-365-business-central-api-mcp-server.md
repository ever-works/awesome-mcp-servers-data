# Microsoft Dynamics 365 Business Central API MCP Server

## Overview
The Microsoft Dynamics 365 Business Central API MCP Server exposes Dynamics 365 Business Central ERP capabilities to MCP-compatible clients, allowing agents and chat-based applications to work with finance, sales, and operations data through a unified interface.

## Features
- **MCP server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Single endpoint used across clients; authentication handled when adding the server to your application.

- **Microsoft Dynamics 365 Business Central integration**  
  - Connects to a Microsoft Dynamics 365 Business Central account.  
  - Enables interaction with Business Central ERP data (finance, sales, and operations) from MCP-enabled clients such as ChatGPT.

- **Available tools (actions)**  
  - **Create Customer**  
    - Creates a new customer in Dynamics 365 Business Central.
  - **Update Customer**  
    - Updates an existing customer record.
  - **Get Sales Order**  
    - Retrieves a sales order by ID.

- **Client integration**  
  - Can be added to chat clients (e.g., ChatGPT / OpenAI) as an MCP server.  
  - Configuration guidance available via the Pipedream MCP configuration page.

## Setup
- Sign in with a compatible account on Pipedream MCP.  
- Connect your Microsoft Dynamics 365 Business Central account.  
- Add the MCP server URL (`https://mcp.pipedream.net/v2`) to your client and authenticate.
