# Botpress MCP Server

## Overview
The **Botpress MCP Server** is an MCP-compatible server that exposes Botpress’s conversational automation and workflow capabilities to MCP agents. It enables agents and applications to build, orchestrate, and manage complex chat experiences using Botpress as an underlying automation and workflow engine.

- **Category:** AI Integration – MCP Servers  
- **Use cases:** Chatbots, conversational interfaces, workflow automation within MCP-enabled clients  
- **Static server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server**  
  - Implements an MCP Server endpoint that can be added to any compatible MCP client or application.  
  - Uses a static base URL (`https://mcp.pipedream.net/v2`) for all clients.

- **Botpress integration**  
  - Exposes Botpress’s conversational automation capabilities to MCP agents.  
  - Allows use of Botpress workflows and logic as tools/actions within MCP-based environments.

- **Complex conversation and workflow automation**  
  - Designed to automate complex conversations, tasks, and workflows using Botpress as a backend.  
  - Suitable for building and managing multi-step chat experiences and task flows.

- **Client-agnostic configuration**  
  - Same MCP server URL for every supported client; authentication is handled when the server is added to the application.  
  - Supports selecting a chat client and following client-specific instructions for adding the MCP server.

- **Configuration guidance**  
  - Provides a configuration flow where you connect your account and select a Botpress client to start using the server.  
  - References a full configuration page for more detailed setup instructions (not included in the provided content).
