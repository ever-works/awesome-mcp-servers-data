# Acumbamail MCP Server

**Category:** Business & Commerce MCP Servers  
**Slug:** acumbamail-mcp-server  
**Brand:** Acumbamail  
**Source URL:** https://mcp.pipedream.com/app/acumbamail

## Description
Acumbamail MCP Server is an integration that allows MCP-compatible clients to interact with Acumbamail’s email and SMS marketing capabilities, subscription forms, and landing pages via Pipedream.

It exposes Acumbamail functionality through a single MCP server endpoint so that chat-based or other MCP-aware applications can use Acumbamail tools programmatically.

## Features
- **Email marketing integration**  
  - Access Acumbamail’s email marketing functionality from MCP clients.  
  - Suitable for managing email-based marketing campaigns and related workflows.

- **SMS marketing integration**  
  - Connect to Acumbamail’s SMS marketing tools through MCP.  
  - Enables MCP clients to interact with SMS-based marketing activities.

- **Subscription forms management**  
  - Integrate subscription form handling into MCP-aware applications.  
  - Use MCP to work with forms used to collect and manage subscribers.

- **Landing page editor access**  
  - Connect to Acumbamail’s landing page editing capabilities.  
  - Allows MCP clients to incorporate landing page workflows into marketing processes.

- **Unified MCP server endpoint**  
  - Static server URL for all clients: `https://mcp.pipedream.net/v2`.  
  - Authentication is performed when adding the server to an application.  
  - Can be used by any compatible MCP client or chat application.

## Technical Details
- **MCP endpoint:** `https://mcp.pipedream.net/v2` (shared static URL used for Acumbamail via Pipedream Connect).  
- **Integration host:** Pipedream (Pipedream Connect).
