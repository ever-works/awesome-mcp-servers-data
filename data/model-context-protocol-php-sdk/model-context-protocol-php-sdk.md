## Overview
The Model Context Protocol PHP SDK is the official PHP implementation of the Model Context Protocol (MCP), enabling PHP applications to implement and integrate with MCP servers and clients.

## Features
- **Official MCP PHP SDK** – Canonical SDK for working with Model Context Protocol in PHP.
- **Server and client support** – Designed for building both MCP servers and MCP clients in PHP applications.
- **Integration in existing PHP apps** – Intended to be embedded in PHP projects to communicate with MCP-compatible services.
- **Structured codebase**
  - `src/` – Core SDK source code.
  - `docs/` – Documentation resources for using the SDK.
  - `examples/` – Example implementations and usage patterns.
  - `tests/` – Test suite for validating SDK behavior.
- **Open-source development workflow** – Public repository with changelog, makefile, and configuration for code style tools.
- **MIT-licensed** – Permissive license suitable for commercial and open-source projects.
- **Community & ecosystem alignment** – Maintained in collaboration with The PHP Foundation, aligning with modern PHP practices.
