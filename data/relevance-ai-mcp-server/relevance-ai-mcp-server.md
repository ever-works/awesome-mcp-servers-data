# Relevance AI MCP Server

**Category:** AI Integration – MCP Servers  
**Brand:** Relevance AI  
**Source:** https://mcp.pipedream.com/app/relevance_ai

## Description
The Relevance AI MCP Server is a Model Context Protocol (MCP) integration that connects MCP-compatible tools and chat clients to Relevance AI’s platform for building and deploying AI apps, agents, and AI workforce solutions. It is provided and hosted via Pipedream Connect.

## Features
- **Static MCP Server URL**  
  - Single, static endpoint for all clients:  
    `https://mcp.pipedream.net/v2`  
  - Authentication occurs when adding the server to your application or account.

- **Chat Client Integration**  
  - Instructions and guides available for integrating with supported chat clients, including:  
    - ChatGPT (OpenAI) via MCP.  
  - Additional configuration details available on Pipedream’s Configuration page.

- **Relevance AI Account Connection**  
  - Connect a Relevance AI account through Pipedream.  
  - Select and manage clients/accounts after signing in.  
  - Enables use of Relevance AI’s platform (AI apps, agents, AI workforce) via MCP tools.

- **Available Tools (Actions)**  
  - **Run Tool**  
    - Executes a specific tool within Relevance AI.  
    - Waits for a response for up to 60 seconds.  
    - Mapped to a Pipedream action: `run-tool`.

  - **Send Message to Agent**  
    - Sends a message directly to an agent in Relevance AI.  
    - Does not wait for an agent response (fire-and-forget style).  
    - Mapped to a Pipedream action: `message-agent`.

- **Platform Context**  
  - Integrates with Relevance AI, a platform for:  
    - Building and deploying AI applications.  
    - Creating and orchestrating AI agents.  
    - Implementing AI workforce solutions.
