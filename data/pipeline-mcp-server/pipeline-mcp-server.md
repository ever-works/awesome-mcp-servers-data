# Pipeline MCP Server

## Overview
Pipeline MCP Server exposes Pipeline CRM’s sales engagement and CRM functionality (contacts, companies, deals, tasks, events, activities, and users) as tools that can be called from MCP-compatible applications and chat clients. It lets you connect a Pipeline account and drive sales workflows programmatically.

- **Category:** Business & Commerce – MCP Servers
- **Vendor / Brand:** Pipeline
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features

### MCP Integration
- Static MCP server URL usable across clients: `https://mcp.pipedream.net/v2`.
- Authentication handled when adding the server to the MCP-enabled application.
- Compatible with multiple chat / MCP clients (instructions provided per client via configuration page).
- Central configuration flow: connect a Pipeline account and select the client to start using tools.

### CRM & Sales Data Management Tools (14 Actions)

#### People / Contacts
- **Create Person**  
  - Create a new person record in a Pipeline account.
- **Update Person**  
  - Update an existing person’s details.
- **Find Person**  
  - Look up an existing person in a Pipeline account.

#### Companies / Accounts
- **Create Company**  
  - Create a new company record.
- **Update Company**  
  - Update an existing company’s details.
- **Find Company**  
  - Find an existing company in a Pipeline account.

#### Deals & Pipeline Management
- **Create Deal**  
  - Create a new deal in a Pipeline pipeline.
- **Update Deal**  
  - Update an existing deal’s information.
- **Find Deal**  
  - Search for and retrieve an existing deal.
- **Move Deal Stage**  
  - Move a deal to another stage within the Pipeline account’s sales pipeline.

#### Activities, Tasks & Events
- **Create Activity**  
  - Create a new activity associated with an existing person, company, or deal.
- **Create Task**  
  - Create a new calendar task in a Pipeline account.
- **Create Event**  
  - Create a new calendar event in a Pipeline account.

#### Users
- **Find User**  
  - Find an existing user associated with a Pipeline account.

## Typical Use Cases
- Drive sales workflows from chat / MCP-enabled agents using Pipeline CRM data.
- Automate creation and updating of contacts, companies, and deals from conversational interfaces.
- Manage tasks, events, and activities tied to deals, people, or companies from within MCP-driven tools.
- Move deals between stages programmatically as part of guided sales processes.
