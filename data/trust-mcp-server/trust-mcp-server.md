# Trust MCP Server

## Overview
Trust MCP Server is an MCP-compatible server endpoint, provided via Pipedream, that connects Trust to MCP-capable chat or AI clients. It allows those clients to interact with Trust through a unified MCP interface.

- **Provider/Brand:** Trust (hosted via Pipedream)
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-Compatible Endpoint**  
  - Exposes Trust functionality via a static MCP server URL usable across clients.
  - Same URL can be reused for every client; authentication is handled when adding the server to each application.

- **Account Connection & Configuration**  
  - Flow to connect a Trust account from within Pipedream.  
  - "Configure Trust" step to associate an account and select a client before use.

- **Client Selection & Setup**  
  - Instructions vary by chat / MCP client; users select their chat client to see how to add the server.  
  - Links to a full configuration page for detailed setup steps.

- **Tools Exposure**  
  - Supports listing and loading of available MCP tools/actions once the server is added (shown as "Loading actions" / "Loading available tools" in the UI).

- **Static, Reusable Integration**  
  - The server URL does not change per customer or client, simplifying configuration across multiple MCP-enabled apps.
