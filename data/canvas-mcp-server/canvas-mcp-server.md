# Canvas MCP Server

**Category:** Documentation & Learning Resources  
**Brand:** Instructure  
**Type:** Model Context Protocol (MCP) server for Canvas LMS  
**Source:** https://mcp.pipedream.com/app/canvas

## Overview
Canvas MCP Server is a Model Context Protocol server that connects MCP-compatible tools and chat clients with the Canvas learning management system. It enables applications to interact with Canvas courses, users, and learning data via a standardized MCP endpoint.

## Features
- **MCP-compatible endpoint**  
  - Provides a static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same URL works across all supported clients.

- **Canvas LMS integration**  
  - Designed to integrate with the Canvas learning management system.  
  - Enables MCP-based tools to access and interact with Canvas resources such as courses, users, and learning data (as supported by the underlying Canvas/Pipedream integration).

- **Client-agnostic usage**  
  - Can be added to multiple chat clients and MCP-capable applications.  
  - Authentication is handled when you add the server to your application.

- **Configuration documentation**  
  - Dedicated configuration instructions available via the “Configuration” page for step-by-step setup with different clients.

- **Hosted by Pipedream Connect**  
  - MCP server is hosted and operated through Pipedream’s Connect platform.

## Usage
- Use the static MCP server URL `https://mcp.pipedream.net/v2` in your MCP-capable client.  
- Authenticate during the setup process in your application or chat client.  
- Follow client-specific steps described on the linked Configuration page.
