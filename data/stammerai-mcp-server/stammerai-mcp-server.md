# Stammer.ai MCP Server

**Category:** AI Integration – MCP Servers  
**Website:** https://mcp.pipedream.com/app/stammer_ai  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

White-label AI platform MCP server for agencies, exposed via Pipedream’s MCP infrastructure.

## Features

- **White-label AI platform access**
  - Connects to Stammer.ai’s white-label AI platform designed for agencies.
  - Enables agencies to integrate AI capabilities into their own tools/offerings under their own branding.

- **MCP server integration**
  - Provides an MCP-compliant server endpoint for use with MCP-based tools and chat clients.
  - Single static MCP server URL (`https://mcp.pipedream.net/v2`) used across all clients.

- **Client-agnostic configuration**
  - Same server URL works for every client; configuration is handled per client in the chat or MCP-compatible application.
  - Detailed setup available via a dedicated configuration page (per Pipedream Connect docs).

- **Authentication at connection time**
  - Authentication occurs when adding the server to an application, allowing per-user or per-client authorization while keeping the URL static.

- **Pipedream Connect infrastructure**
  - Hosted and powered by Pipedream Connect, inheriting its MCP server infrastructure and policies.
