# DailyBot MCP Server

An MCP Server integration for DailyBot that connects async meeting automation, milestone tracking, and routine workflows with AI-assisted developer and productivity tools.

## Overview
- **Type:** MCP server integration
- **Category:** Workflow automation MCP servers
- **Works with:** DailyBot + AI tools (e.g., Cursor, Claude, Windsurf via Pipedream MCP)
- **Use cases:** Automate async meetings, track milestones, streamline recurring workflows directly from work chat environments and AI tools.
