# Mission Mobile MCP Server

Mission Mobile MCP Server is an MCP integration that connects Mission Mobile’s SMS marketing solutions with compatible MCP clients, enabling SMS campaign workflows for media companies.

## Overview
- **Type:** MCP server / integration
- **Category:** Messaging MCP servers
- **Use case:** SMS marketing and campaign execution for media companies through MCP-compatible chat or agent clients
- **Provider:** Mission Mobile (via Pipedream Connect)

## Features
- **Standard MCP server endpoint**  
  - Uses a single static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same URL works for all clients; authentication is handled when adding the server in your MCP-compatible application.

- **MCP client integration**  
  - Designed to be added to MCP-compatible chat or agent applications as an external server.  
  - Follows common MCP configuration patterns documented in the Pipedream Configuration page.

- **SMS marketing support for media companies**  
  - Intended to connect MCP clients with Mission Mobile’s SMS marketing capabilities.  
  - Enables workflows for sending and managing SMS campaigns (details of individual actions, triggers, or objects are not listed in the provided content, but the integration is focused on SMS marketing use cases).

## Configuration
- **Server URL:** `https://mcp.pipedream.net/v2`  
- **Authentication:** Performed when the server is added to your MCP client (exact auth method not detailed in the provided content).  
- **Documentation:** Additional setup and configuration details are available via the general MCP server Configuration page on Pipedream.
