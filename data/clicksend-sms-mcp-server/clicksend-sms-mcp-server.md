## ClickSend SMS MCP Server

**Category:** Messaging MCP Servers  
**Brand:** ClickSend  
**Source:** https://mcp.pipedream.com/app/clicksend

### Overview
ClickSend SMS MCP Server is an MCP Server integration that provides programmable access to ClickSend’s SMS and broader business communications services through a static MCP endpoint managed by Pipedream.

### Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Can be added to any compatible chat or MCP-enabled client.

- **ClickSend SMS integration**  
  - Connects to a ClickSend account to enable SMS and business communications actions via MCP.  
  - Uses ClickSend’s APIs behind the scenes for sending and managing SMS and related business communication workflows.

- **Account-based configuration**  
  - Connect your ClickSend account within the Pipedream MCP interface.  
  - Once connected, the server exposes tools/actions associated with your ClickSend client.

- **Tool-based interaction model**  
  - Provides a set of MCP tools (actions) for interacting with ClickSend (e.g., sending SMS or related operations).  
  - Tools load dynamically based on your connected account and configuration.

- **Client-agnostic usage**  
  - Designed to work with multiple chat or MCP-aware clients.  
  - Configuration instructions are provided per client type via the Pipedream configuration page.

### Configuration
- Copy the static MCP server URL: `https://mcp.pipedream.net/v2`.  
- Add the server URL to your MCP-compatible application or chat client.  
- Authenticate when adding the server, then connect your ClickSend account and select the appropriate client configuration within Pipedream.
