## Overview
The Fullstory MCP Server is an MCP (Model Context Protocol) server, provided via Pipedream, that exposes Fullstory customer experience and session data to MCP-compatible chat clients and agents. This allows LLM-based tools to query and analyze how users interact with digital products.

## Features
- **MCP-compatible server endpoint**  
  - Static MCP URL: `https://mcp.pipedream.net/v2`  
  - Works with any compatible client; authentication is handled when adding the server to the application.

- **Fullstory data access** *(implied from description)*  
  - Exposes customer experience and session data from Fullstory.  
  - Enables querying and analysis of user behavior across digital products through MCP agents.

- **Client-agnostic setup**  
  - Designed to be added to various chat or agent clients that support MCP.  
  - Setup instructions are available per client type via the configuration documentation.

- **Configuration documentation**  
  - Central configuration resource ("Configuration" page on Pipedream) for details on connecting the MCP server to different apps.

## Usage
1. Use the static MCP URL `https://mcp.pipedream.net/v2` as the server endpoint in your MCP-enabled client.  
2. Authenticate when prompted by your application during setup.  
3. Configure tools/agents within your client to query and analyze Fullstory customer experience and session data.
