# Metatext.AI Pre-build AI Models API MCP Server

**Description**

Metatext.AI Pre-build AI Models API MCP Server exposes Metatext.AI’s pre-built AI models through the Model Context Protocol (MCP), allowing applications and chat clients to integrate common inference tasks (such as machines that read and write) via a standardized server endpoint.

**Category**
- AI Integration – MCP Servers

**Provider / Brand**
- Brand: Metatext.AI
- Delivered via: Pipedream Connect

**Integration Endpoint**
- MCP server base URL (static across clients):
  - `https://mcp.pipedream.net/v2`

**Features**
- Provides access to Metatext.AI’s pre-built AI models through an MCP-compliant server.
- Supports “machines that read and write,” enabling common inference tasks (e.g., text understanding and generation) via API.
- Single static MCP server URL usable across different MCP-compatible clients.
- Authentication performed when adding the server to your application or chat client.
- Compatible with multiple chat clients that support MCP, with client-specific setup instructions available.
- Central configuration and management via a dedicated Configuration page on Pipedream.

**Usage / Setup**
- Copy and use the static MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the MCP server to your chat client or application and authenticate during setup.
- Optional: refer to the full Configuration page on Pipedream for detailed integration steps.
