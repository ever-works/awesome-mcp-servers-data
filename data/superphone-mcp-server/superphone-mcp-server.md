# SuperPhone MCP Server

SuperPhone MCP Server integrates the SuperPhone communication platform into the MCP ecosystem so AI agents can manage creator and coaching communications via standardized MCP endpoints.

---

## Overview
- **Name:** SuperPhone MCP Server  
- **Category:** Messaging MCP Servers  
- **Use cases:** SMS-based communication, audience engagement for coaches and creators, creator / coaching workflows inside AI chat clients  
- **MCP Base URL:** `https://mcp.pipedream.net/v2`

---

## Features
- **MCP integration for SuperPhone**  
  - Exposes SuperPhone capabilities through standardized MCP endpoints so AI agents can interact with the SuperPhone platform programmatically.
- **Static MCP server URL**  
  - Uses a single static URL (`https://mcp.pipedream.net/v2`) that works across compatible MCP clients.  
  - Authentication is performed when adding the server to a specific application or client.
- **Client-agnostic setup**  
  - Designed to be added to a variety of chat clients that support MCP.  
  - Configuration guidance is available via a central configuration page (per the linked documentation).
- **Creator & coach–focused communication**  
  - Intended for coaches and creators who need to manage audience engagement and communication through AI agents using SuperPhone.

---

## Setup
- Add the following MCP server URL to your MCP-compatible application or chat client:  
  `https://mcp.pipedream.net/v2`  
- Authenticate within your client when prompted.  
- Refer to the platform’s configuration page for client-specific setup steps.
