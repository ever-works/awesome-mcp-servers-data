# OrbisX MCP Server

An MCP server that connects OrbisX—an AI-enabled CRM optimized for auto shops—to compatible chat clients and applications.

- **Category:** Business & Commerce MCP Servers  
- **Use cases:** Auto shop CRM access, AI-powered booking workflows, customer management via MCP-compatible chat clients.

## MCP Server URL

Use this static MCP server endpoint in your client configuration:

```text
https://mcp.pipedream.net/v2
```

Authentication is performed when you add the server to your application.

## Features

- **Static MCP endpoint** – Single, reusable MCP server URL (`https://mcp.pipedream.net/v2`) for all clients.
- **OrbisX integration** – Connects to OrbisX, a CRM tailored for auto shops.
- **AI-enabled workflows** – Designed for AI-assisted bookings and customer management within OrbisX.
- **Chat client compatibility** – Can be added to supported chat clients that implement the MCP protocol.
- **Central configuration** – Setup and usage documented via a configuration page (referenced by the provider) for adding the server to various apps.
