# Elastic Cloud MCP Server

**Category:** Search & Discovery MCP Servers  
**Brand:** Elastic  
**Slug:** elastic-cloud-mcp-server

An MCP server for Elastic Cloud that lets MCP-compatible tools interact with hosted Elasticsearch and hosted search services running on Elastic Cloud.

![Elastic Cloud](https://static-www.elastic.co/v3/assets/bltefdd0b53724fa2ce/blt6a5c45e26dfec353/6605b3f440408910f71ece3b/illustrated-screenshot-elastic-cloud.png)

---

## Features

- MCP server endpoint for Elastic Cloud resources.
- Supports interaction with hosted Elasticsearch.
- Supports interaction with hosted search services on Elastic Cloud.
- Static MCP server URL that works across supported clients.
- Authentication handled when adding the server in your MCP-compatible application.

**MCP Server URL**  
`https://mcp.pipedream.net/v2`
