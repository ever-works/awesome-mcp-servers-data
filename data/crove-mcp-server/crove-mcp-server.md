## Crove MCP Server

**Category:** Document Management MCP Servers  
**Tags:** document-automation, templates, workflow

### Overview
Crove MCP Server is an MCP integration for Crove.app that lets you automate repetitive business documents. It works with intelligent templates that use variables and logical conditions, enabling programmatic creation, updating, submission, and distribution of documents through MCP-compatible clients.

### Features
- **Intelligent document templates**
  - Create business documents as reusable templates.
  - Use variables and logical conditions within templates to generate dynamic content.

- **MCP endpoint**
  - Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`.
  - Authentication handled when adding the server to your MCP-compatible application.

- **Account configuration**
  - Connect a Crove account to manage documents via MCP.
  - Select and work with specific Crove clients / accounts after connection.

- **Available tools (actions)**
  - **Create Document**  
    Create a new document from Crove templates.
  - **Update Document**  
    Update values of variables in an existing document.
  - **Update Document Details**  
    Modify document metadata such as name or current status.
  - **Get Document Details**  
    Retrieve details of a document (e.g., name, current status).
  - **Submit Document**  
    Submit a document as if it were submitted via a Crove form.
  - **Send Document**  
    Send an email invitation link for recipients to fill and sign the document.
  - **Create Invitation Link**  
    Generate an invitation link to fill or sign the document without directly sending the email.
  - **Generate Document PDF From Template**  
    Generate a PDF for a document created from a template.
  - **Generate PDF from Document**  
    Generate a PDF of a document and return a URL to the PDF.
  - **Complete Document**  
    Mark a document as completed.

### Integration & Usage
- Add the Crove MCP server URL to any supported MCP/chat client.
- Configure and authenticate your Crove account in the client.
- Use the provided tools to automate document workflows end-to-end: creation, updating, submission, signature flows, and PDF generation.
