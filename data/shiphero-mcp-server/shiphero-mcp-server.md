# ShipHero MCP Server

An MCP Server integration with ShipHero that lets MCP-powered tools interact with ShipHero’s warehouse and fulfillment platform for managing inventory, orders, and shipping operations.

## Overview
- **Type:** MCP server integration
- **Category:** Business & Commerce – MCP Servers
- **Provider / Brand:** ShipHero (via Pipedream Connect)
- **Slug:** `shiphero-mcp-server`
- **Base MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **ShipHero integration for MCP tools**
  - Connects MCP-compatible applications to ShipHero’s warehouse and fulfillment APIs.
  - Enables automated workflows around inventory, orders, and shipping.

- **Warehouse and fulfillment workflows**
  - Supports scenarios for managing warehouse operations using ShipHero.
  - Can be used whether you outsource warehousing to ShipHero or run your own warehouse on ShipHero.

- **Inventory management use cases**
  - Designed to power tools that need to read or manage inventory data in ShipHero.

- **Order and shipping operations**
  - Intended for workflows involving order processing and shipping management within ShipHero.

- **Static MCP server endpoint**
  - Single static URL (`https://mcp.pipedream.net/v2`) works for all MCP clients.
  - Authentication is handled when adding / configuring the server in the client.

- **Works with multiple chat / MCP clients**
  - Can be added to different chat clients that support MCP.
  - Configuration guidance is available via the referenced configuration page (client-specific setup not detailed here).
