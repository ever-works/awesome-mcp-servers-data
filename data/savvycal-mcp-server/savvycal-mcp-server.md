# SavvyCal MCP Server

## Overview
SavvyCal MCP Server is a Model Context Protocol (MCP) server that allows MCP-compatible clients to interact with SavvyCal for scheduling, calendar availability, and meeting coordination.

## Features
- MCP-compatible server for SavvyCal
- Supports interaction with:
  - Scheduling workflows
  - Calendar availability
  - Meeting coordination and appointments
- Uses a single static MCP server URL that works across all compatible clients
- Authentication is handled when adding the server to an MCP-capable application
- Documentation and setup details available via a configuration page (client-specific setup instructions)

## Technical Details
- **Server type:** Static MCP server endpoint
- **Endpoint:** `https://mcp.pipedream.net/v2`
- **Client support:** Designed to be added to various MCP-capable chat or agent clients (configured per client)
