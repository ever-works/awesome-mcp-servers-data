# Launch27 MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Launch27  
**Slug:** `launch27-mcp-server`

## Overview
The Launch27 MCP Server connects a Launch27 account to MCP-compatible AI tools, enabling management of online booking for local service businesses in a Shopify-style experience. It runs on Pipedream’s MCP server infrastructure to integrate Launch27 with AI assistants and developer tools.

## Features
- **MCP integration for Launch27**: Exposes Launch27 functionality through the Model Context Protocol (MCP), making it accessible to compatible AI tools.
- **AI tool connectivity**: Connects Launch27 to AI tools such as Cursor, Claude, and Windsurf via Pipedream’s MCP server.
- **Online booking management**: Designed to let AI-driven workflows help manage online booking for local service businesses.
- **Local service business focus**: Tailored for businesses that offer local services and need an online, Shopify-style booking and commerce experience.
- **Runs on Pipedream MCP server**: Uses Pipedream as the MCP server backend for connection, configuration, and communication.

## Use Cases
- Allow AI coding tools (e.g., Cursor) to interact programmatically with a Launch27 account.
- Use conversational AI (e.g., Claude) to help manage or automate booking-related tasks.
- Integrate Launch27 booking flows into AI-powered development environments (e.g., Windsurf).
