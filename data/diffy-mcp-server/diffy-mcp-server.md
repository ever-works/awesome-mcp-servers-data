# Diffy MCP Server

**Category:** Testing & Debugging Tools  
**Slug:** diffy-mcp-server  
**Brand:** Diffy

Screenshots-based tool, exposed as an MCP server, to keep visual design consistent during code changes via screenshot comparison.

---

## Features

- **MCP-compatible server**  
  - Exposed via a static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Works with any MCP-compatible client or automation pipeline.

- **Screenshot-based visual regression testing**  
  - Uses screenshots to detect visual changes between versions of your application.  
  - Helps maintain design consistency as code changes.

- **Design consistency checks**  
  - Identifies unexpected visual differences across builds or deployments.  
  - Useful for UI/UX and frontend regression monitoring.

- **Automation / pipeline integration**  
  - Designed to plug into MCP-compatible automation pipelines.  
  - Can be used from chat clients or tools that support MCP servers.

- **Central configuration**  
  - Single, static URL for all clients; authentication handled when adding the server.  
  - Configuration guidance available via a dedicated configuration page (MCP client setup).
