# Blink MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Pipedream  
**Source:** https://mcp.pipedream.com/app/blink

## Overview
Blink MCP Server is an MCP (Model Context Protocol) server integration for Blink that enables real-time communication and engagement between frontline workers and leadership. It is designed to connect chat clients and automations so organizations can facilitate ongoing dialogue and trust across teams.

## MCP Server URL
- Base MCP server URL (static for all clients):
  - `https://mcp.pipedream.net/v2`
- Authentication occurs when adding the server to your specific application / client.

## Features
- **Real-time communication channel** between frontline workers and executive / leadership teams.
- **Supports MCP automations** to facilitate workflows and interactions within Blink.
- **Static, client-agnostic MCP endpoint** (`https://mcp.pipedream.net/v2`) usable across different MCP-compatible clients.
- **Works with multiple chat clients** (add the server to your preferred MCP-capable chat application).
- **Configuration guidance available** via a dedicated configuration page on Pipedream (setup and integration instructions).
- **Operates via Pipedream Connect** infrastructure.

## Integration & Setup
- Copy and use the static MCP server URL in your MCP-compatible client.
- Add the server to your chat app and authenticate during setup.
- Additional configuration steps and options are available on the Pipedream configuration page.
