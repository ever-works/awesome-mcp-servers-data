# Capsule MCP Server

**Category:** Business & Commerce – MCP Servers  
**Tags:** CRM, customer data, sales

## Overview
The Capsule MCP Server is an MCP (Model Context Protocol) server integration for Capsule CRM. It allows AI agents or MCP-compatible tools and chat clients to interact with Capsule’s simple CRM features and customer data via a standardized server endpoint.

- **Name:** Capsule MCP Server  
- **Brand:** Capsule  
- **Source URL:** https://mcp.pipedream.com/app/capsule  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server endpoint**  
  - Provides a static MCP server URL (`https://mcp.pipedream.net/v2`) that works across supported MCP clients.
  - Authentication is handled when you add the server to your MCP-compatible application or chat client.

- **Capsule CRM integration**  
  - Connects MCP-based tools and AI agents to Capsule CRM, enabling interaction with Capsule’s simple CRM features and customer data (as exposed through the integration).

- **Client-agnostic setup**  
  - Same server URL for every supported client.  
  - Can be added to various chat clients or applications that support MCP.

- **Configuration documentation**  
  - A dedicated Configuration page is available from the source site to guide setup and usage details.
