# Craftboxx MCP Server

## Description
Craftboxx MCP Server is an MCP integration that connects your Craftboxx account to AI-powered developer and productivity tools. It enables MCP-based tools to work with Craftboxx data for craftsmen-focused workflows such as job scheduling, documentation, and time recording.

## Features
- Connects Craftboxx to Pipedream’s MCP server
- Enables integration with AI tools that support MCP (e.g., Cursor, Claude, Windsurf)
- Access to job scheduling data from Craftboxx
- Access to documentation and records stored in Craftboxx
- Access to time recording data for field and craftsmen workflows
- Designed for craftsmen and field-service–oriented job management scenarios
