# VivifyScrum MCP Server

Multiplatform Agile project management MCP server integrating VivifyScrum with compatible MCP clients.

## Overview
- Connects MCP clients to VivifyScrum, an Agile project management tool
- Supports Scrum and Kanban boards
- Designed to help manage, track, and deliver projects within chat or MCP-enabled interfaces

## MCP Server URL
- Base MCP server URL (static for all clients):
  - `https://mcp.pipedream.net/v2`
- Authentication is performed when adding the server to your MCP-compatible application

## Features
- Integration with VivifyScrum for Agile project management
- Supports Scrum workflows
- Supports Kanban workflows
- Enables organization of project-related data for management and tracking
- Works with multiple MCP client applications (multiplatform via MCP)
- Central static endpoint for easier configuration across different clients
