## Overview
claude-skills-mcp is an MCP (Model Context Protocol) server that makes Claude Agent Skills available as searchable, intelligent tools. It enables MCP-compatible models and clients to discover and invoke Claude Agent Skills natively via vector search.

## Features
- **MCP server for Claude Agent Skills**  
  - Exposes Claude Agent Skills as tools that can be consumed by any MCP-enabled model or client.

- **Vector-based skill discovery**  
  - Uses vector search to find and retrieve relevant Claude Agent Skills based on semantic similarity.

- **Search and retrieval of skills**  
  - Allows searching for skills and returning detailed skill definitions for use in agent workflows.

- **MCP-native integration**  
  - Designed to plug into MCP ecosystems so models/clients can call skills as first-class tools.

- **Repository structure (implementation-related)**  
  - `docs/` for documentation.  
  - `packages/` for core implementation packages.  
  - `scripts/` for automation/dev scripts.  
  - `tests/integration/` for integration tests.  
  - Example configuration file: `config.example.json`.
