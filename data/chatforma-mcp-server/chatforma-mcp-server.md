# Chatforma MCP Server

An MCP server integration for Chatforma, a CIS-focused no-code platform to build, connect, and manage chatbots for Telegram, Facebook, VK, and Viber.

- **Category:** Messaging MCP Servers  
- **Brand:** Chatforma  
- **Slug:** `chatforma-mcp-server`  
- **Source URL:** https://mcp.pipedream.com/app/chatforma

## Features

- **MCP Server Integration**
  - Exposes Chatforma as an MCP (Model Context Protocol) server.
  - Provides a single static MCP server URL usable by any compatible client:  
    `https://mcp.pipedream.net/v2`.
  - Authentication is handled when adding the server to your MCP-compatible application.

- **No-Code Chatbot Platform (via Chatforma)**
  - Build chatbots without coding.
  - Focused on CIS (Commonwealth of Independent States) markets.
  - Supports major messaging platforms:
    - Telegram
    - Facebook
    - VK (VKontakte)
    - Viber

- **Client-Agnostic Setup**
  - The same MCP server URL works across different MCP clients.
  - Configuration guidance is available via the linked configuration page.
