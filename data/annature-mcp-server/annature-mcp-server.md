## Annature MCP Server

**Category:** Business & Commerce – MCP Servers  
**Tags:** e-signature, document automation

Annature MCP Server exposes Annature’s ISO 27001-certified eSignature and digital transaction management (DTM) capabilities via an MCP endpoint, enabling integration of Annature workflows into MCP-compatible applications.

---

### Features

- **MCP integration endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same URL usable for all clients; authentication handled when adding the server to an application.

- **eSignature and digital transaction management**  
  - Access to Annature’s eSignature functionality.  
  - Digital Transaction Management (DTM) solution, built on blockchain technology.  
  - Designed for use by small businesses and enterprises in Australia and New Zealand.

- **Security and compliance**  
  - ISO 27001-certified DTM solution.

- **Client-agnostic setup**  
  - Can be added to various chat or MCP-compatible clients.  
  - Central configuration via an application “Configuration” page (external setup guidance).

---

### Setup

1. Connect / authenticate your Annature account.  
2. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.  
3. Add the server URL and authenticate within your chosen MCP-compatible app or chat client.
