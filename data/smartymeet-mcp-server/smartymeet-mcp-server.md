# SmartyMeet MCP Server

**Description**  
MCP Server integration for SmartyMeet that connects AI-powered team video meeting capabilities into MCP-compatible workflows and tools.

**Category**

**Tags**

## Features
- Provides an MCP-compatible server endpoint for SmartyMeet.  
- Connects SmartyMeetâ€™s AI-enhanced team video meeting capabilities into tools and workflows that support the Model Context Protocol (MCP).  
- Uses a single static MCP server URL that can be added to any compatible client:  
  - MCP server URL: `https://mcp.pipedream.net/v2`  
- Authentication is handled when adding the server in the client or application.  
- Can be used across multiple chat or MCP clients that support adding custom MCP servers.  
- Configuration details and client-specific setup instructions are available via the associated configuration documentation.

---

## Integration & Setup
- **MCP Server URL:** `https://mcp.pipedream.net/v2` (static URL for all clients).  
- **Setup flow:**  
  - Add the server URL to your MCP-compatible or chat client.  
  - Authenticate as prompted by the client.  
  - Optionally refer to the full configuration page for detailed setup.
