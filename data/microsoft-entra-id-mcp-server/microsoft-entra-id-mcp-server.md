# Microsoft Entra ID MCP Server

## Overview
The Microsoft Entra ID MCP Server is an MCP-compatible endpoint that exposes Microsoft Entra ID (Azure AD) identity and access management capabilities to agent and chat-based applications. It enables agents to interact with Entra ID for tasks related to user, application, and access control within a cloud identity environment.

## Features
- **MCP-compatible server endpoint**  
  - Provides a static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Works with multiple MCP-compatible chat and agent clients.

- **Microsoft Entra ID integration**  
  - Connects to a Microsoft Entra ID (Azure AD) tenant.  
  - Exposes identity and access management capabilities to tools and agents (e.g., user and app access control operations).  
  - Designed to safeguard organizational access to apps, devices, and data via Entra ID.

- **Authentication and configuration**  
  - Authentication occurs when adding the MCP server to the application.  
  - Centralized configuration flow via a dedicated configuration page (external) for setting up the connection and client integration.

- **Multi-client support**  
  - Can be added to different chat or MCP clients, each using the same static URL.  
  - Client-specific setup instructions available via “Select your chat client” workflow (external to this content).

## Setup
1. Connect your Microsoft Entra ID account in Pipedream.  
2. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.  
3. Add the server to your MCP-compatible application or chat client.  
4. Complete authentication when prompted by your client.
