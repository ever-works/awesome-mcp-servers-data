# SalesLens MCP Server

SalesLens MCP Server provides MCP access to SalesLens’s AI-driven call center quality monitoring, enabling agents and teams to analyze and optimize call performance via MCP-compatible tools.

## Overview
- **Type:** MCP server / integration endpoint
- **Category:** Monitoring, Call Center Analytics, Quality Assurance
- **Base MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Provider:** SalesLens (via Pipedream Connect)

## Features
- **MCP-compatible server endpoint**  
  - Single static URL (`https://mcp.pipedream.net/v2`) used by all clients.  
  - Authentication handled when adding the server in your MCP-enabled application.

- **Access to SalesLens capabilities via MCP tools**  
  - Connects MCP clients to SalesLens’s AI-driven call center quality monitoring.  
  - Designed to help analyze and optimize call performance from within MCP-based workflows and chat clients.

- **Client-agnostic integration**  
  - Works with multiple MCP-enabled chat clients.  
  - Configuration is performed in the client by adding the server URL and authenticating.

- **Central configuration reference**  
  - Additional setup and configuration details are available on a dedicated configuration page (linked from the app page).
