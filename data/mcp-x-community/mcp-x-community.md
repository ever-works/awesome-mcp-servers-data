# MCP X Community

## Overview
MCP X Community (via the `modelcontextprotocol` GitHub repository) is a curated collection of Model Context Protocol (MCP) servers and related resources. It includes reference implementations and links to community-built servers, illustrating how MCP can be used to give large language models secure, controlled access to tools and data sources.

## Features
- **Reference MCP servers**
  - A set of reference implementations demonstrating core MCP features.
  - Examples designed to show how to connect LLMs to external tools and data through MCP.

- **Community-built servers and resources**
  - References to MCP servers created by the wider community.
  - Additional resources collected alongside the server list.

- **SDK-based implementations**
  - Each listed MCP server is implemented using one of the official SDKs:
    - Typescript MCP SDK
    - Python MCP SDK

- **Server categories**
  - **Reference Servers**: Focused on demonstrating MCP capabilities and SDK usage.
  - **Third-Party Servers**: Community or external implementations.
  - **Official Integrations**: A subset of third-party servers classified as official integrations (category exists; specific integrations not detailed in the provided content).
