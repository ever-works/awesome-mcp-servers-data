# ERPNext MCP Server

**Category:** Business & Commerce MCP Servers  
**Slug:** `erpnext-mcp-server`  
**Brand:** ERPNext  
**Source URL:** https://mcp.pipedream.com/app/erpnext

## Description
ERPNext MCP Server exposes machine-accessible interfaces to ERPNext, the open‑source ERP platform used for enterprise resource planning and business operations. It is hosted via Pipedream and accessed through a common MCP endpoint for integration with MCP-compatible clients.

## Features
- **MCP-compatible server** for ERPNext, usable from any MCP-capable client or application.
- **Single static endpoint** for all clients:
  - MCP server base URL: `https://mcp.pipedream.net/v2`
- **Authentication handled at client setup time**, allowing the same URL to be reused across different applications.
- **Integration with ERPNext modules** to enable programmatic access to ERP-related data and operations (e.g., business operations, enterprise resource planning workflows).
- **Hosted via Pipedream Connect**, benefiting from Pipedream’s infrastructure for connectivity and configuration.
- **Configuration documentation available** via the general MCP Configuration page on Pipedream.

## Usage
- Use the static MCP server URL `https://mcp.pipedream.net/v2` when adding the ERPNext MCP Server to your chat or automation client.
- Authenticate within your client or application according to its MCP configuration flow.
