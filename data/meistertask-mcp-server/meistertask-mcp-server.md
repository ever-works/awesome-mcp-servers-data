# MeisterTask MCP Server

## Overview
The MeisterTask MCP Server is an MCP-compatible integration that lets MCP-based agents and chat clients manage tasks and related resources within the MeisterTask ecosystem via Pipedream.

- **Category:** Project Management MCP Servers  
- **Vendor / Brand:** Meister (via Pipedream)  
- **Endpoint (MCP server URL):** `https://mcp.pipedream.net/v2`

## Features

### MCP Server & Configuration
- Static MCP server URL that works for all supported clients.
- Authentication occurs when adding the server to your MCP-compatible application or chat client.
- Usable across multiple chat clients; configuration guidance available via the Pipedream configuration page.

### Available Tools (Actions)
There are 9 actions exposed as MCP tools for MeisterTask:

1. **Update Task**  
   - Update an existing task within a specific project section.

2. **Get Task**  
   - Retrieve detailed information about a specific task.

3. **Get Person**  
   - Retrieve information about a person (user) in MeisterTask.

4. **Get Label**  
   - Retrieve information about a specific label.

5. **Get Attachment**  
   - Retrieve information about an attachment associated with tasks.

6. **Create Task**  
   - Create a new task in a specified project section.

7. **Create Label**  
   - Create a new label within a project.

8. **Create Attachment**  
   - Create a new attachment (e.g., file or resource) associated with MeisterTask items.

9. **Add Label To Task**  
   - Add an existing project label to a specific task.
