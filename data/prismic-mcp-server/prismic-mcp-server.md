# Prismic MCP Server

**Category:** Content Management MCP Servers  
**Slug:** prismic-mcp-server  
**Source:** https://mcp.pipedream.com/app/prismic

## Overview
Prismic MCP Server is an MCP (Model Context Protocol) server that lets MCP-based agents work with Prismic’s headless page builder and CMS. It is designed for sites built with modern frameworks including Next.js, Nuxt, and SvelteKit.

## Features
- MCP server endpoint for integrating Prismic with MCP-compatible clients and agents
- Works with Prismic’s headless CMS and page builder
- Supports sites built with:
  - Next.js
  - Nuxt
  - SvelteKit
- Single static server URL usable across MCP clients
- Authentication handled when adding the server in the client/application

## Technical Details
- MCP server base URL: `https://mcp.pipedream.net/v2`

## Setup
- Copy the MCP server URL: `https://mcp.pipedream.net/v2`
- Add the server in your MCP-compatible chat client or application using its configuration for external MCP servers
- Refer to the provider’s configuration documentation for client-specific setup steps
