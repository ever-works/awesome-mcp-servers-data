# Cliniko MCP Server

An MCP server that connects Cliniko health practice management software with MCP-aware AI tools.

## Overview

- **Category:** Business & Commerce – MCP Servers  
- **Use case:** Integrate Cliniko’s practice management data and workflows into AI-enabled applications via the Model Context Protocol (MCP).
- **Provider:** Pipedream (powered by Pipedream Connect)

## MCP Server URL

Use this static URL for all clients (authentication is handled when you add the server to your app):

```text
https://mcp.pipedream.net/v2
```

## Features

- **MCP-compatible endpoint**  
  - Single, static MCP server URL usable across different MCP-aware chat and AI clients.
  - Authentication occurs during server setup in the client.

- **Cliniko integration**  
  - Designed to connect to Cliniko’s health practice management platform (appointments, practice management workflows, etc.) so AI tools can interact with Cliniko data and actions via MCP.

- **Multi-client support**  
  - Can be added to various MCP-enabled chat applications or AI assistants using the same server URL.

- **Configuration resources**  
  - Full configuration instructions available via a dedicated Configuration page (client-specific setup guides).
