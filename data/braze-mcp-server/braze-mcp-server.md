# Braze MCP Server

Customer engagement platform MCP server exposing Braze’s capabilities to MCP-based agents and workflows.

- **Website**: https://mcp.pipedream.com/app/braze
- **Category**: Business & Commerce · MCP Servers
- **Tags**: customer-engagement, marketing-automation, messaging
- **MCP Server URL**: `https://mcp.pipedream.net/v2`

## Overview
The Braze MCP Server connects MCP-compatible clients and agents to Braze, enabling programmatic access to Braze’s customer engagement and messaging capabilities via a standardized MCP endpoint.

## Features
- **Standard MCP Endpoint**
  - Single static MCP server URL: `https://mcp.pipedream.net/v2`
  - Works with any compatible MCP client.
  - Authentication handled when adding the server to your application.

- **Braze Integration**
  - Access to Braze’s customer engagement platform through MCP.
  - Supports customer-centric, real-time interactions between brands and consumers (via Braze’s APIs exposed through MCP).

- **Client-Agnostic Setup**
  - Can be added to multiple chat or agent clients.
  - Compatible with MCP-based workflows and tools.

- **Configuration Documentation**
  - Central configuration reference available via a “Configuration” page (for setup and integration details with various clients).

## Usage
- Add the MCP server URL `https://mcp.pipedream.net/v2` to your MCP-compatible client or agent environment.
- Authenticate within the client as prompted during setup.
