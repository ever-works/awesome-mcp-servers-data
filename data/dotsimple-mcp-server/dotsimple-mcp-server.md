# DotSimple MCP Server

## Overview
DotSimple MCP Server provides Model Context Protocol (MCP) access to DotSimple’s social media management platform, enabling scheduling, posting, and analytics workflows from compatible MCP-based applications.

- **Category:** Business & Commerce – MCP Servers
- **Use cases:** Social media scheduling, publishing, and performance tracking via MCP tools
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-based integration**
  - Exposes DotSimple’s social media management capabilities as MCP tools.
  - Uses a static MCP server URL (`https://mcp.pipedream.net/v2`) that works for every client.
  - Authentication occurs when adding the server to an MCP-compatible application.

- **Social media management functions** (via DotSimple platform)
  - All-in-one social media management, including:
    - Content scheduling.
    - Posting to connected social profiles.
    - Access to analytics and performance data.

- **Client-agnostic setup**
  - Same server URL usable across multiple chat or MCP clients.
  - Configuration guidance available via the platform’s configuration page.

## Configuration
- Copy and use the MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server to your preferred MCP-compatible app or chat client and authenticate when prompted.
