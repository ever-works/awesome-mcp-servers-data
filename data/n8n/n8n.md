## Overview

n8n is a workflow automation tool with extensive integrations for popular services like Slack, Google Sheets, and APIs.

## Features

- Open-Source Flexibility: No vendor lock-in.
- Extensive Integrations: Hundreds of applications.
- Visual Editor: Drag-and-drop workflow building.
- Custom Code Support: JavaScript, Python, Docker.
- Self-Hosting Option: Enhanced security.

## Use Cases

Automating bug tracking across Jira and GitHub, notifying teams via email.
