# EdApp MCP Server

**Category:** Documentation & Learning Resources  
**Website:** https://mcp.pipedream.com/app/edapp  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Overview
EdApp MCP Server is an MCP server, provided via Pipedream Connect, that links AI agents and chat-based clients to EdApp’s mobile-first training platform. It is intended to support the creation and management of bite-sized learning and training content in EdApp.

## Features
- **MCP server endpoint**
  - Static MCP server URL that works for all compatible MCP clients: `https://mcp.pipedream.net/v2`.
  - Authentication is handled when adding the server to a client application.

- **AI–EdApp integration**
  - Connects AI agents to EdApp’s mobile-first learning management system (LMS).
  - Supports workflows around creating and managing bite-sized training content in EdApp from within MCP-compatible tools.

- **Client-agnostic setup**
  - Designed to be added to various chat or AI clients that support MCP.
  - Central configuration guidance available via a dedicated configuration page (through Pipedream Connect).

- **Mobile-first training platform support**
  - Targets EdApp’s mobile training environment, enabling AI-assisted interaction with course and training content suitable for on-the-go learning.

## Configuration
- Use the static server URL `https://mcp.pipedream.net/v2` when configuring your MCP-compatible client.
- Follow client-specific steps in the “Add the server to your app” and configuration documentation (linked from the Pipedream EdApp MCP page).
