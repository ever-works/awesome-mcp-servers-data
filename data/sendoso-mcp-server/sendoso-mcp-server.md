# Sendoso MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Sendoso  
**Source:** https://mcp.pipedream.com/app/sendoso

## Overview
The Sendoso MCP Server exposes Sendoso’s sending platform as MCP tools, allowing applications and chat clients to trigger gifting, direct mail, and outbound engagement workflows via a static MCP endpoint.

- **MCP server URL:** `https://mcp.pipedream.net/v2`
- **Tools available:** 14 actions

## Features

### Connection & Configuration
- Connect a Sendoso account via MCP.
- Use a single static MCP server URL for all clients; authenticate per application.
- Add the server to different chat clients and apps (configuration documentation available via Pipedream).

### Gifting & Sending Actions
- **Send a Physical Gift**  
  Trigger sending of a physical gift using Sendoso’s platform.

- **Send a Physical Gift with Address Confirmation**  
  Send a physical gift while collecting or confirming the recipient’s address before fulfillment.

- **Send an eGift**  
  Initiate sending of a digital gift (eGift) through Sendoso.

- **Create eGift Links**  
  Generate multiple eGift links for distribution through external channels.

- **Generate eGift Link**  
  Generate a single new eGift link programmatically.

### Reporting & Tracking
- **List Sends**  
  Retrieve a list of all sends/gifts with optional filters (e.g., by status, date, campaign, or other criteria as supported by the API).

- **Get Send Status**  
  Track previously sent gifts and retrieve analytics information about those sends (e.g., delivery status, engagement details where available).

### Campaign & Group Management
- **List Campaigns**  
  Retrieve a list of all campaigns with optional filters.

- **Get Campaign**  
  Retrieve detailed information about a specific campaign.

- **List Groups (Teams)**  
  List all groups/teams configured in the Sendoso account.

- **List Group Members**  
  List all members/users belonging to a specific group.

### User & Account Management
- **List All Users**  
  Retrieve a list of all users in the Sendoso account.

- **Invite New User**  
  Invite a new user to the Sendoso account via the MCP server.

- **Get Current User**  
  Retrieve information about the currently authenticated Sendoso user.

## Use Cases
- Automate sending of physical and digital gifts from within MCP-enabled chat clients or applications.
- Build workflows that trigger gifts based on CRM events, marketing automation, or support interactions.
- Monitor send status and engagement data in conversational interfaces.
- Manage campaigns, groups, and users programmatically.
