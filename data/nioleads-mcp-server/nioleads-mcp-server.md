# NioLeads MCP Server

## Overview
NioLeads MCP Server is an MCP (Model Context Protocol) server integration on Pipedream that supports LinkedIn-focused lead generation workflows. It works with the NioLeads Email Finder to assist with outreach and networking activities.

## Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Can be added to any compatible chat client or application that supports MCP.

- **LinkedIn outreach support**  
  - Designed to simplify LinkedIn outreach workflows.  
  - Intended for prospecting and networking use cases.

- **Email finding (NioLeads Email Finder)**  
  - Integrates with NioLeads’ email-finding capabilities to help locate contact emails for leads and prospects.

- **Client-based configuration**  
  - Connect a NioLeads account and select a client within the Pipedream interface to begin using tools.  
  - Central configuration via Pipedream’s “Configure NioLeads” flow.

- **Tool-based workflow execution**  
  - Exposes “available tools” (actions) through the MCP server (loaded dynamically in Pipedream) for use inside chat clients or other MCP-compatible apps.

## Technical Details
- **Server URL:** `https://mcp.pipedream.net/v2`  
- **Hosting / Integration Platform:** Pipedream  
- **Category:** Business & Commerce – MCP Servers  
- **Primary Use Cases:** Lead generation, prospecting, email discovery for networking.
