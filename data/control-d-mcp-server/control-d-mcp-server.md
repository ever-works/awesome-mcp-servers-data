# Control D MCP Server

**Category:** Security & Attestation MCP Servers  
**Brand:** Control D  
**Source:** https://mcp.pipedream.com/app/control_d

## Overview
Control D MCP Server is an MCP (Model Context Protocol) integration for the Control D DNS service. It exposes Control D’s programmable, customizable DNS capabilities to MCP-compatible chat clients and applications, enabling DNS-based blocking of threats, unwanted content, and ads across devices.

## Features
- **MCP integration endpoint**  
  - Static MCP server URL usable by any compatible client:  
    - `https://mcp.pipedream.net/v2`  
  - Authentication performed when adding the server to an application.

- **Programmable DNS control**  
  - Integrates with Control D’s customizable DNS service.  
  - Designed to manage blocking of:  
    - Security threats  
    - Unwanted or inappropriate content  
    - Advertisements  
  - Works across all devices supported by Control D’s DNS.

- **Client-agnostic setup**  
  - Single static URL works for every supported MCP client.  
  - Can be added to various chat clients and apps that support MCP.

- **Configuration resources**  
  - Documentation and setup examples available via a central configuration page (linked from the app page).

## Integration & Usage
- Add the MCP server to an MCP-compatible chat client or application using:  
  `https://mcp.pipedream.net/v2`  
- Authenticate within the client/app during setup.  
- Use MCP capabilities to programmatically control DNS filtering policies exposed by Control D.
