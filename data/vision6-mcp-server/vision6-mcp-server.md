# Vision6 MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Vision6  
**Source URL:** https://mcp.pipedream.com/app/vision6

## Overview
The Vision6 MCP Server is an integration that exposes Vision6 (email and SMS marketing platform) to MCP-compatible agents. It allows agents and MCP-enabled applications to manage email and SMS marketing campaigns through a standardized MCP server endpoint.

## Features
- **MCP server integration for Vision6**
  - Provides an MCP-compliant interface to the Vision6 email and SMS marketing platform.
- **Static MCP server URL**
  - Single shared endpoint for all clients: `https://mcp.pipedream.net/v2`.
  - Same URL is used across different chat clients and MCP-compatible applications.
- **Authentication at configuration time**
  - Authentication is performed when adding the server to your MCP-compatible application or chat client (credentials / auth details are not embedded in the URL).
- **Multi-client compatibility**
  - Can be added to various MCP-enabled chat clients and tools.
  - Documentation references client-specific setup flows (“Select your chat client to learn how to get started”).
- **Centralized configuration documentation**
  - A full configuration page is available for detailed setup instructions (via the linked “Configuration page”).
- **Backed by Pipedream Connect**
  - Runs on Pipedream’s Connect infrastructure to handle MCP server hosting and connectivity.
- **Campaign management via MCP (from description)**
  - Enables MCP agents to manage Vision6 email and SMS marketing campaigns.
  - Intended for operations related to sending and managing marketing communications over email and SMS.

## Usage
- Use the static MCP URL `https://mcp.pipedream.net/v2` when adding the Vision6 MCP Server to your MCP-compatible client.
- Complete authentication during server setup in your chosen application or chat client.
- Refer to the provider’s configuration documentation for client-specific instructions.
