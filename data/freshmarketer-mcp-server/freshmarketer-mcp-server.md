# Freshmarketer MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Freshmarketer  
**Slug:** `freshmarketer-mcp-server`

## Overview
The Freshmarketer MCP Server is an integration that connects Freshmarketer—a marketing automation suite—to MCP-compatible applications (such as chat clients). It enables you to work with contacts, lists, and marketing campaigns from within tools that support MCP servers.

Freshmarketer itself is designed to help manage contacts and lists, design and deliver marketing campaigns, and automate email marketing workflows.

## Features
- **MCP server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Single URL works for all clients; authentication occurs when adding the server to your app.

- **Freshmarketer integration**  
  - Connects your Freshmarketer account via MCP.  
  - Uses your account authentication when added to a supported client or application.

- **Contact and list management (via Freshmarketer)**  
  - Manage contacts and contact lists within Freshmarketer.  
  - Intended for organizing recipients for marketing campaigns and workflows.

- **Campaign design and delivery (via Freshmarketer)**  
  - Design marketing campaigns in Freshmarketer.  
  - Deliver email campaigns to selected contacts and lists.

- **Email marketing automation (via Freshmarketer)**  
  - Automate email marketing workflows such as sequences, follow-ups, or other triggered campaigns.

- **Client compatibility**  
  - Can be added to supported chat clients or other MCP-capable applications.  
  - Configuration docs are available on a dedicated configuration page (not reproduced here).

## Configuration
- Copy the MCP server URL: `https://mcp.pipedream.net/v2`.  
- Add the MCP server URL to your MCP-compatible app or chat client.  
- Authenticate with your Freshmarketer account when prompted by the client.
