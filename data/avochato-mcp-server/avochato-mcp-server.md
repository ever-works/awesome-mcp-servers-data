# Avochato MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Pipedream  
**Slug:** avochato-mcp-server

## Overview
The Avochato MCP Server exposes Avochato’s SMS and team messaging automation capabilities—such as templates, auto-assignment, tags, and collaboration tools—to MCP-based workflows and applications via a static MCP endpoint.

## Features
- **MCP server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Works for all Avochato clients; authentication handled when adding the server to your MCP-compatible app

- **Customer and team messaging**  
  - Send and manage SMS conversations with customers  
  - Coordinate internal team communications through the same messaging workflows

- **Automation features**  
  - Message templates to standardize and speed up responses  
  - Auto-assignment of conversations to team members to balance workload and maintain ownership  
  - Tagging to categorize and organize conversations at scale

- **Collaboration and context**  
  - Shared notes on conversations for internal context  
  - Conversation history available to support continuity and better responses

- **Configuration & integration**  
  - Connect an Avochato account and select a client within the Pipedream interface  
  - Can be added to different chat clients / MCP-compatible apps by following client-specific instructions  
  - Central configuration reference via a dedicated configuration page on Pipedream
