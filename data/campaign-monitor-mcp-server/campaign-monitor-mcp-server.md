# Campaign Monitor MCP Server

Email Marketing Platform & Services MCP server for Campaign Monitor, enabling MCP clients to create, send, and track email marketing campaigns programmatically.

## Overview
The Campaign Monitor MCP Server is an MCP-compatible endpoint (powered by Pipedream Connect) that allows MCP clients to integrate with Campaign Monitor to manage and automate email marketing workflows.

- **Server type:** MCP Server
- **Category:** Business & Commerce MCP Servers
- **Tags:** email-marketing, marketing-automation, campaigns
- **Base MCP URL:** `https://mcp.pipedream.net/v2`

## Features
Based on the available description, this MCP server is intended to:

- Connect MCP-compatible clients to Campaign Monitor.
- Programmatically **create** email marketing campaigns.
- Programmatically **send** email campaigns.
- **Track** performance and other metrics of sent campaigns.
- Use a static MCP server URL (`https://mcp.pipedream.net/v2`) for all clients, with authentication handled during server setup in the client.

> Note: The underlying website content does not list individual API methods or detailed capabilities beyond creating, sending, and tracking campaigns.

## Configuration
