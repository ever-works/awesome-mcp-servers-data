# Security Reporter MCP Server

**Category:** Security Attestation MCP Servers  
**Tags:** security, pentesting, reporting

An MCP (Model Context Protocol) server that connects to Security Reporter, simplifying security reporting and enhancing pentesting workflows.

---

## MCP Server URL

Use this static URL for all MCP-compatible clients:

```text
https://mcp.pipedream.net/v2
```

Authentication is handled when you add the server to your application.

---

## Features

- **Model Context Protocol server** for integrating Security Reporter into MCP-compatible chat and AI clients.
- **Single static endpoint** (`https://mcp.pipedream.net/v2`) usable across all supported clients.
- **Client-agnostic setup**: same URL for every client; configuration handled per application.
- **Authentication on connect**: you authenticate when adding the server to your app (no per-client URL changes required).
- **Hosted by Pipedream** via Pipedream Connect infrastructure.
