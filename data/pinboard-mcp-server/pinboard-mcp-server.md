# Pinboard MCP Server

Fast, ad‑free bookmarking MCP server that integrates Pinboard’s social bookmarking features (saving, organizing, and retrieving bookmarks) into MCP-compatible workflows.

- **Website / Source**: https://mcp.pipedream.com/app/pinboard
- **Category**: Content Management MCP Servers
- **Tags**: bookmark-management, content-search, knowledge-management
- **MCP Server URL**: `https://mcp.pipedream.net/v2`

## Features

- **MCP integration**
  - Exposes Pinboard functionality as an MCP server for use in MCP-compatible chat clients and tools.
  - Uses a single static MCP server endpoint (`https://mcp.pipedream.net/v2`) for all clients.
  - Authentication handled when adding the server to the client/application.

- **Bookmark management**
  - Save bookmarks from within MCP-powered workflows.
  - Organize bookmarks for later retrieval (e.g., as part of knowledge management flows).
  - Retrieve and reference saved bookmarks directly in conversations or automations.

- **Search and knowledge management**
  - Supports content/bookmark search to quickly find saved links.
  - Enables use of bookmarks as a lightweight knowledge base inside MCP workflows.

- **Privacy-focused bookmarking** (inherited from Pinboard behavior)
  - Fast, no‑nonsense interface.
  - No ads.
  - No third‑party tracking.

## Integration & Usage

- Add the server URL `https://mcp.pipedream.net/v2` to any supported MCP chat client.
- Configure and authenticate the Pinboard MCP Server following the client’s MCP configuration instructions or the provider’s configuration page.
