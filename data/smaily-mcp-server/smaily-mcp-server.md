# Smaily MCP Server

**Description**

Smaily MCP Server is an MCP (Model Context Protocol) server integration for Smaily’s email marketing platform. It enables MCP-compatible clients to create and send newsletters and manage email campaigns via a unified MCP endpoint.

---

## Features

- MCP server integration with Smaily’s email marketing platform
- Tools for creating and sending newsletters
- Campaign management capabilities for email marketing
- Single static MCP server URL usable across compatible clients
- Authentication handled when adding the server to an MCP-enabled application

---

## Technical Details

- **MCP Server URL**: `https://mcp.pipedream.net/v2`
- **Integration layer**: Powered by Pipedream Connect
- **Usage pattern**:
  - Add the static MCP server URL to any supported MCP client
  - Authenticate within the client when registering the server

---

## Setup

1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`
2. Add the server to your MCP-compatible chat client or application.
3. Authenticate when prompted within your client.
4. Use the Smaily MCP tools in your client to create and send newsletters and manage email campaigns.
