# Outscraper MCP Server

MCP server for accessing Outscraper’s data scraping and lead-generation APIs directly from MCP-compatible AI and chat clients.

- **Category:** Content Extraction & Summarization MCP Servers  
- **Provider / Brand:** Outscraper (via Pipedream)  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

---

## Features

### MCP Integration
- Static MCP server URL that works for all clients (`https://mcp.pipedream.net/v2`).  
- Authentication handled when adding the server to your MCP-compatible application.  
- Setup guides available for ChatGPT (OpenAI) and other clients via the Pipedream Configuration page.

### Available Tools / Actions

1. **Search Places on Google Maps**  
   - Searches for places on Google Maps using text queries.  
   - Useful for location discovery, local business lookup, and place-based lead lists.  
   - Backed by Outscraper’s Google Maps data extraction API.

2. **Scrape Emails and Contacts**  
   - Extracts email addresses, social links, and phone numbers from provided domains.  
   - Supports lead-generation workflows and contact enrichment for websites.

3. **Reverse Geocoding**  
   - Converts geographic coordinates / locations into human-readable addresses.  
   - Enables address enrichment and location normalization from lat/long data.

---

## Use Cases
- Lead generation from websites and Google Maps.  
- Enriching datasets with contact details (emails, phones, social links).  
- Converting coordinates into full postal addresses for mapping or CRM records.  
- Integrating web data extraction directly into AI agents or chat-based tools.
