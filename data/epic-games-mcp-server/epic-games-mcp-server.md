# Epic Games MCP Server

An MCP Server that integrates with Epic Games, enabling access to services and data from the Epic Games digital storefront and ecosystem.

---

## Overview
- Connects applications to the Epic Games digital storefront via the MCP (Model Context Protocol).
- Designed for both players and creators to interact with Epic Games data and services.
- Hosted and powered by Pipedream Connect.

---

## MCP Server URL
- Static MCP endpoint (works for all compatible clients):
  - `https://mcp.pipedream.net/v2`
- Authentication is performed when adding the server to your client/application.

---

## Features
- **Epic Games storefront integration**: Access services and data from the Epic Games digital storefront and ecosystem.
- **MCP-compatible server**: Exposes Epic Games-related functionality through a standard MCP server endpoint.
- **Static server URL**: Single, reusable URL for all MCP-compatible clients.
- **Client-agnostic**: Designed to be added to various chat or MCP-enabled clients.
