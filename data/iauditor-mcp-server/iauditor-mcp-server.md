# iAuditor MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** SafetyCulture  
**Slug:** `iauditor-mcp-server`

## Description
The iAuditor MCP Server connects SafetyCulture’s iAuditor platform to AI tools via the Model Context Protocol (MCP). It enables AI assistants and compatible developer tools to interact with your iAuditor account for inspection and issue workflows.

## Features
- Connects an iAuditor by SafetyCulture account to AI tools through Pipedream’s MCP server
- Supports integration with AI tools such as:
  - Cursor
  - Claude
  - Windsurf
- Enables inspection-related workflows via MCP
- Supports issue capture and corrective action workflows through MCP-enabled tools
