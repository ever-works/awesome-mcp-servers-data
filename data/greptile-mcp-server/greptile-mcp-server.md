# Greptile MCP Server

**Category:** Repository & Code Analysis MCP Servers  
**Brand:** Pipedream  
**Source:** https://mcp.pipedream.com/app/greptile

## Description
Greptile MCP Server is a Model Context Protocol (MCP) server provided by Pipedream that connects your Greptile account to AI tools. It enables chatting with code repositories and using Greptile’s API to build AI-powered developer tools.

## Features
- Connects a Greptile account to compatible AI tools.  
- Supports integration with AI code assistants such as:
  - Cursor
  - Claude
  - Windsurf
- Allows conversational interaction with code repositories via Greptile.  
- Exposes Greptile’s API capabilities through an MCP server for building AI-powered developer tooling on Pipedream.

## Use Cases
- Chatting with and querying code repositories through AI assistants.  
- Building custom AI-powered developer tools that leverage Greptile’s repository analysis via Pipedream.
