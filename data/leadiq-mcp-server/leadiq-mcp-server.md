# LeadIQ MCP Server

MCP Server for LeadIQ, exposing B2B prospecting and lead-enrichment functionalities through the MCP framework.

- **Category:** Business & Commerce – MCP Servers  
- **Brand:** LeadIQ  
- **Source URL:** https://mcp.pipedream.com/app/leadiq  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Overview
LeadIQ MCP Server integrates LeadIQ’s B2B prospecting and lead-enrichment capabilities into MCP-compatible clients. It allows applications and chat-based tools to interact with LeadIQ for prospecting workflows via a standardized MCP endpoint.

## Features
- **B2B Prospecting Integration**  
  - Access LeadIQ’s prospecting functionality through the MCP framework.  
  - Designed for B2B lead identification and outreach workflows.

- **Lead Enrichment**  
  - Exposes LeadIQ’s enrichment capabilities so tools can retrieve richer lead data (e.g., for sales and marketing use cases).

- **Standard MCP Endpoint**  
  - Uses a single, static MCP server URL: `https://mcp.pipedream.net/v2`.  
  - Same URL works for all compatible MCP clients.

- **Per‑Client Authentication**  
  - Authentication is performed when adding the server to each client, allowing secure access from multiple applications.

- **Multi‑Client Compatibility**  
  - Intended to be added to various MCP‑aware chat clients and applications.  
  - Central configuration via a common server endpoint.

- **Hosted by Pipedream Connect**  
  - Server is powered and hosted by Pipedream’s Connect infrastructure.

## Configuration
- Add the MCP server to your preferred MCP-compatible client using:  
  `https://mcp.pipedream.net/v2`  
- Additional configuration details are available on the product’s configuration page (linked from the source URL / interface).
