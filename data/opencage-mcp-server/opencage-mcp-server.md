# OpenCage MCP Server

**Category:** Data Access & Integration – MCP Servers  
**Brand:** OpenCage  
**Slug:** opencage-mcp-server

## Description
OpenCage MCP Server is an MCP-compatible integration that connects the OpenCage geocoding platform to AI tools. It exposes OpenCage’s geocoding and reverse geocoding capabilities through the Model Context Protocol (MCP), enabling location-based functionality inside AI-powered development and chat environments.

## Features
- **MCP server integration** for the OpenCage geocoding platform.
- **Geocoding**: Convert place names or addresses into geographic coordinates via the OpenCage API.
- **Reverse geocoding**: Convert coordinates into human-readable place information via the OpenCage API.
- **AI tool connectivity**: Designed to be used from AI tools that support MCP, including:
  - Cursor
  - Claude
  - Windsurf
- **Pipedream-based deployment**: Runs as a Pipedream MCP server to connect your OpenCage account with compatible AI clients.
- **API integration**: Uses your OpenCage account and API key to access location data programmatically through MCP.
