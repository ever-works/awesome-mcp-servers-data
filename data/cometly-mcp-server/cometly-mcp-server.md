# Cometly MCP Server

**Category:** Business & Commerce – MCP Servers  
**Website / Source:** https://mcp.pipedream.com/app/cometly

## Overview
The Cometly MCP Server is an MCP-compatible backend, provided via Pipedream, that integrates Cometly’s revenue attribution capabilities into MCP clients. It enables marketing and analytics teams to access Cometly attribution data directly from MCP-based chat or agent applications.

## Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL for all supported clients:  
    `https://mcp.pipedream.net/v2`  
  - Designed to be added as an MCP server in compatible chat or agent applications.

- **Cometly integration**  
  - Connects MCP clients to Cometly’s revenue attribution data for marketing teams.  
  - Intended for use in marketing analytics, attribution, and campaign-related workflows (per tags and description).

- **Client-agnostic configuration**  
  - Same base URL works across different MCP clients.  
  - Authentication is performed when adding/configuring the server in the client.

- **Documentation & configuration resources**  
  - Setup guidance available via Pipedream’s "Configuration" page (general MCP server configuration instructions).  
  - Hosted and operated by Pipedream (Pipedream Connect infrastructure).

## Authentication
- Authentication occurs when adding the MCP server to an application/client.  
- The server URL itself is static; credentials or tokens are provided during client-side configuration (details in the external configuration docs, not in the provided content).

## Technical Details
- **Server URL:** `https://mcp.pipedream.net/v2`  
- **Protocol:** MCP server (Model Context Protocol) endpoint suitable for MCP clients.  
- **Integration target:** Cometly revenue attribution data for marketing and campaign analytics.
