# Omnivore MCP Server

**Category:** Content Management MCP Servers  
**Brand:** Omnivore  
**Website / Source:** https://mcp.pipedream.com/app/omnivore  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Description
Omnivore MCP Server is an MCP-compatible integration for Omnivore, a free and open-source read-it-later application. It allows tools and clients that support the Model Context Protocol (MCP) to save, retrieve, and manage read-it-later content, including articles, highlights, and notes.

## Features
- **Read-it-later integration**: Connects MCP-compatible tools with Omnivore’s read-it-later functionality.
- **Save web articles**: Store any web article for later reading through MCP-compatible clients.
- **Newsletter subscription support**: Subscribe to and manage newsletters within Omnivore’s read-it-later environment.
- **Distraction-free reading**: Access cleaned, distraction-free versions of saved content (as provided by Omnivore).
- **Highlights management**: Create, store, and manage highlights from saved content.
- **Notes management**: Add and manage notes associated with articles and other saved content.
- **Static MCP server URL**: Uses a single static endpoint (`https://mcp.pipedream.net/v2`) that works across compatible clients.
- **Client-agnostic usage**: Designed to be added to various chat or MCP-enabled clients via configuration.
- **Authentication at setup**: Authentication is handled when the server is added to the client/application.
- **Open source backing (Omnivore)**: Built on top of Omnivore, a free and open source read-it-later app for advanced/serious readers.
