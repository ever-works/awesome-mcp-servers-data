# LifterLMS MCP Server

An MCP (Model Context Protocol) server integration for LifterLMS, an all‑in‑one WordPress LMS platform for courses and memberships.

## Overview
- **Name:** LifterLMS MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Use Case:** Connects LifterLMS (WordPress LMS for courses and memberships) to MCP‑compatible chat or AI clients via Pipedream Connect.

## MCP Server URL
- **Base URL:** `https://mcp.pipedream.net/v2`  
- This is a static URL used for all clients; authentication is handled when adding the server in your application.

## Features
- **Static MCP Endpoint**  
  - Single, shared MCP server URL (`https://mcp.pipedream.net/v2`) for all clients.

- **Client Integration**  
  - Add the MCP server to compatible chat or AI applications.  
  - Supports configuration through a “Configuration” page (general MCP setup guidance via Pipedream Connect).

- **Pipedream Connect Integration**  
  - MCP server is powered and hosted by Pipedream Connect.  
  - Uses Pipedream’s infrastructure, terms of service, and privacy controls.

- **LifterLMS Context**  
  - Intended for use with LifterLMS, a WordPress LMS platform for:  
    - Creating and managing online courses.  
    - Handling memberships.  
  - Designed to bring LifterLMS data and actions into MCP‑enabled tools (where supported by the client and configuration).

## Setup
1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.  
2. Add the server to your MCP‑compatible chat or AI client.  
3. Authenticate within your client when prompted.  
4. Optionally, review the full configuration instructions on the Pipedream Configuration page.
