## y.gy MCP Server

**Category:** Business & Commerce – MCP Servers  
**Slug:** ygy-mcp-server  
**Brand:** ygy  
**Source:** https://mcp.pipedream.com/app/y_gy

### Description
The y.gy MCP Server exposes y.gy’s free QR code generation and URL shortening services via the Model Context Protocol (MCP), enabling agents and tools to programmatically create and manage QR codes and short links.

### Features
- MCP-enabled access to y.gy services
- Programmatic QR code creation
- Programmatic short URL (link) generation
- Uses a single static MCP server endpoint usable across compatible clients
- Designed for integration with chat clients and other MCP-aware applications

### Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Integration:** Add as an MCP server in supported chat clients or tools

- Uses y.gy’s **free** QR code and link shortening services (no pricing tiers listed).
