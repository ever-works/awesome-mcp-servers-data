## Overview

Unified's MCP server connects Cohere to customer integrations through tool-calling, allowing Cohere to connect intelligence to structured SaaS data and authorized actions. Cohere provides state-of-the-art embedding models and reranking capabilities optimized for RAG (Retrieval Augmented Generation) pipelines.

## Features

### Embed Models (Embeddings)

**Embed v4** - Latest multimodal embedding model:
- Converts text and images into numerical representations
- Captures semantic meaning for sophisticated applications
- **Pricing**: Embed v3 at $0.10/1M tokens
- Enables semantic search, clustering, and classification

### Rerank Models

**Rerank 4.0** - Newest and most performant foundational model:
- Sorts text inputs by semantic relevance to a specified query
- **Pricing**: Rerank 3.5 at $2/1K searches
- Dramatically improves RAG accuracy by filtering irrelevant documents
- Often used to sort search results from existing search solutions

### Command Models

Command R+ for production-grade LLM capabilities:
- Tool-calling integration
- Multi-language support
- Enterprise features
