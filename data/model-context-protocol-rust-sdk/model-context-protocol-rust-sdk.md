# Model Context Protocol Rust SDK

## Overview
Official Rust SDK for the Model Context Protocol (MCP), providing Rust crates and tooling to build MCP servers and clients.

## Features
- **Official MCP SDK for Rust**
  - Implements the Model Context Protocol for the Rust ecosystem.
  - Suitable for building MCP-compatible servers and clients.

- **Rust crates workspace**
  - `crates/` directory containing multiple Rust crates managed via Cargo.
  - Central `Cargo.toml` configured as a workspace to manage the SDK components.

- **Server and client development**
  - Provides building blocks for highâ€‘performance MCP servers.
  - Provides building blocks for MCP clients that can talk to MCP servers.

- **Documentation and guides**
  - `docs/` directory for SDK and protocol documentation.
  - `README.md` in the repository root describing usage and setup (GitHub-hosted).

- **Examples**
  - `examples/` directory with example projects showing how to use the SDK to implement MCP servers/clients.

- **Development environment & tooling**
  - `.devcontainer/` configuration for container-based / VS Code Dev Containers development.
  - `.github/` directory for GitHub workflows and repository automation.
  - `clippy.toml` for linting configuration with `cargo clippy`.
  - `.gitignore` tailored for Rust/Cargo and project-specific artifacts.

- **Metadata & configuration**
  - `client-metadata.json` for describing client metadata in the MCP context.

- **Open source**
  - Source code hosted publicly on GitHub.
  - Community contributions via standard GitHub workflows (issues, pull requests, forks).
