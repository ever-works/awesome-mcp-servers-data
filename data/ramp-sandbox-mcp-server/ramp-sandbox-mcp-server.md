# Ramp (Sandbox) MCP Server

**Category:** Finance & Market Data MCP Servers  
**Tags:** sandbox, cards, spend-management

## Overview
Ramp (Sandbox) MCP Server provides a full test environment for exploring Ramp API endpoints and validating MCP-based applications without affecting production data. It is powered by Pipedream Connect and is intended for safe development, integration testing, and experimentation.

## Features
- **Full sandbox environment** for Ramp APIs, isolated from production data.
- **Supports exploration of multiple API endpoints** to prototype and test integrations.
- **Safe testing of MCP-based applications**, workflows, and clients.
- **Static MCP server URL** for all clients: `https://mcp.pipedream.net/v2`.
- **Authentication on client setup**, allowing secure access per application.
- **Compatible with multiple chat clients** that support MCP servers.
- **Centralized configuration guidance** via the referenced configuration page (for setup and connection details).
- **Powered by Pipedream Connect**, integrating with the broader Pipedream ecosystem.

## How to Use
1. Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add the MCP server to your chat client or application.
3. Authenticate when prompted during server setup.
4. Use the sandbox endpoints to test and validate your Ramp-related workflows.
