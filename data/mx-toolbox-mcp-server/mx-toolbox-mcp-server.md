## Overview
Mx Toolbox MCP Server is an MCP (Model Context Protocol) server that exposes MxToolboxâ€™s DNS, email, and network diagnostic tools to AI agents. It enables automated checks such as MX lookups, blacklist checks, and DNS troubleshooting through a unified endpoint.

## Features
- Provides access to MxToolbox DNS, email, and network diagnostic tools via MCP
- Supports automated checks for:
  - MX (Mail Exchange) record lookups
  - Blacklist / blocklist checks
  - General DNS troubleshooting and diagnostics
- Single static MCP server endpoint usable across compatible clients
- Designed for integration with AI agents and chat-based applications
- Hosted and powered by Pipedream Connect

## Technical Details
- **MCP server URL:** `https://mcp.pipedream.net/v2`  
  - Static URL that works for all supported clients
  - Authentication is handled when adding the server to your application
- Can be added to various chat clients that support MCP by configuring this endpoint
