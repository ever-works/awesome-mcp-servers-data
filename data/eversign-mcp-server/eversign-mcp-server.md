## Features

- **Legally binding e-signatures**
  - Supports online signatures that are intended to be legally binding.
- **Document sending for signature**
  - Initiate and send documents for signing via MCP-compatible tools.
- **Signature tracking**
  - Track the status and progress of documents sent for signing.
- **MCP-compatible endpoint**
  - Single static MCP server URL that works across clients: `https://mcp.pipedream.net/v2`.
  - Can be added to any MCP-enabled chat client or application.
- **Authentication at client level**
  - Authentication occurs when adding the server to an application/client (supports per-client auth flows).
- **Configuration documentation**
  - Additional technical setup details available via the Pipedream MCP Configuration page.
