# Apex MCP Server

Estate agency-focused MCP (Model Context Protocol) server integration for Apex 27, enabling estate agency professionals to access Apex 27 estate agency software via MCP-compatible applications.

---

## Overview
- **Name:** Apex MCP Server (Apex 27 MCP Server)
- **Category:** Business & Commerce – MCP Servers
- **Use Case:** Real estate CRM and property management access via MCP
- **Brand:** Apex 27
- **Source URL:** https://mcp.pipedream.com/app/apex_27

---

## Features
- **MCP integration for Apex 27**
  - Exposes Apex 27 estate agency software capabilities through the MCP protocol.
  - Designed for estate agency professionals using Apex 27.

- **Static MCP server URL**
  - Uses a single static MCP server endpoint for all clients: `https://mcp.pipedream.net/v2`.
  - Authentication occurs when adding the server to your MCP-compatible application.

- **Configuration via Pipedream**
  - Configure Apex 27 within the Pipedream MCP environment.
  - Connect your Apex 27 account and select a client to start using the tools.

- **Client-agnostic setup**
  - Can be added to different chat or MCP-compatible clients.
  - Guidance available for adding the server to a chosen chat client.

- **Tool-based actions (MCP tools)**
  - Provides “Available tools” (actions) exposed through MCP for interacting with Apex 27 (specific tools are loaded dynamically and not listed in the content).

- **Central configuration reference**
  - Compatible with the general Pipedream MCP configuration flow (via a separate configuration page).
