# Fidel API MCP Server

**Category:** Finance & Market Data MCP Servers  
**Brand:** Fidel API  
**Source:** https://mcp.pipedream.com/app/fidel_api

## Description
Fidel API MCP Server exposes Fidel’s programmable payment card APIs via the MCP (Model Context Protocol). It allows applications and AI agents to access card-linked experiences and payment transaction data through a standardized MCP server interface, enabling integrations built on top of payment cards.

## Features
- **MCP Endpoint Access**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same URL usable across compatible MCP clients
- **Authentication at Client Integration**  
  - Authentication is performed when adding the server to an application or chat client
- **Programmable Payment Card Integration**  
  - Surfaces Fidel’s programmable payment card APIs through MCP  
  - Supports building card-linked experiences (e.g., card-linked offers and rewards)  
  - Enables access to transaction data integrations based on payment card activity
- **Client / App Integration Support**  
  - Designed to be added as a server to MCP-compatible chat clients and applications  
  - Configuration instructions available via a central configuration page (outside this summary)
