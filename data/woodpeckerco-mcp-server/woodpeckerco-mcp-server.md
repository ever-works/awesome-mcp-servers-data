# Woodpecker.co MCP Server

**Description**  
An MCP Server integration for Woodpecker.co that enables automated cold email outreach and follow‑up workflows from compatible MCP clients.

**Category:** Business & Commerce MCP Servers  
**Tags:** email-marketing, outreach, automation  
**Source:** https://mcp.pipedream.com/app/woodpecker_co

---

## Features

### MCP Server Configuration
- Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`  
- Authenticate when adding the server to your MCP-compatible chat or agent application.
- Can be added to various chat clients; detailed setup steps available via the Configuration page (per client instructions not listed here).

### Woodpecker.co Integration
- Connect a Woodpecker.co account to manage prospects and campaigns via MCP.
- Use MCP tools (actions) to control outreach and follow‑up behavior programmatically.

### Available Tools (Actions)
1. **Stop Follow-Ups**  
   - Updates a prospect so they stop receiving follow‑up emails.
   - Intended to manage and halt ongoing automated follow‑up sequences for a specific prospect.

2. **Find Prospect's Email**  
   - Searches for a specific prospect within Woodpecker.co.  
   - Returns prospect email details to support targeted outreach workflows.

3. **Create or Update Prospect**  
   - Creates a new prospect record in Woodpecker.co or updates an existing one.  
   - Useful for keeping lead data synchronized from external systems or chat-based workflows.

4. **Create or Update Prospect in Campaign**  
   - Creates or updates a prospect and associates them with a specific campaign.  
   - Allows programmatic enrollment of prospects into campaigns and updating their details within that campaign context.
