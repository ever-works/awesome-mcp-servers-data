# Tookan MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Tookan  
**Source:** https://mcp.pipedream.com/app/tookan

## Overview
Tookan MCP Server exposes Tookan’s field workforce and delivery management capabilities to MCP-based tools and chat clients. It is a static MCP endpoint provided via Pipedream Connect, allowing applications to integrate with Tookan for managing and tracking on‑demand deliveries and field service operations.

## Features
- **MCP integration endpoint**  
  - Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`  
  - Authentication handled when adding the server to an MCP-compatible application.

- **Access to Tookan capabilities via MCP**  
  - Interfaces with Tookan, an off‑the‑shelf solution for managing and tracking field workforces.  
  - Supports use cases involving on‑demand deliveries and field service logistics.

- **Client and configuration support**  
  - Can be added to multiple MCP‑compatible chat clients.  
  - Central configuration information available through a dedicated configuration page (linked from the app page).

- **Hosted by Pipedream Connect**  
  - MCP server infrastructure and hosting provided by Pipedream.  
  - Governed by Pipedream’s terms and privacy policy.

## Setup
1. Use the static MCP server URL: `https://mcp.pipedream.net/v2`.  
2. Add the server in your MCP‑compatible app or chat client.  
3. Authenticate within the client as prompted.
