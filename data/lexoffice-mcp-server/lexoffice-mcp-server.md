# Lexoffice MCP Server

**Description**  
Lexoffice MCP Server exposes Lexoffice’s online accounting and invoicing capabilities to MCP-compatible agents and automations via Pipedream Connect. It allows applications and chat-based agents to interact with Lexoffice accounting data over a standard MCP server endpoint.

**Category**  
- Finance & Market Data MCP Servers

**Use Cases**  
- Automating creation and management of invoices through MCP-compatible agents
- Integrating Lexoffice accounting workflows into chatbots or custom apps
- Accessing and working with accounting and invoicing data programmatically

---

## Features

- **MCP-Compatible Endpoint**  
  - Provides a single static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Can be added to any MCP-compatible client or application.

- **Lexoffice Integration**  
  - Connects to Lexoffice, the online accounting software from Lexware.
  - Exposes accounting and invoicing functionality to agents / automations (e.g., writing invoices, creating offers/quotes, accessing accounting data).

- **Web-Based & App Access (via Lexoffice)**  
  - Works with Lexoffice’s web-based and app-based access model, enabling agents to interact with numbers and documents available in Lexoffice from anywhere.

- **Authentication Flow**  
  - Uses a static MCP URL; authentication occurs when adding the server to the client, allowing per-client secure access.

- **Configuration Guidance**  
  - Documentation available via a dedicated configuration page (how to add and configure the server in supported clients).

---

## Getting Started

- Use the MCP server URL: `https://mcp.pipedream.net/v2` when adding a new MCP server in your client.
- Follow your chat client’s instructions or the Lexoffice MCP Server configuration page for setup and authentication.
