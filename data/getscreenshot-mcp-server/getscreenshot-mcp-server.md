# GetScreenshot MCP Server

**Category:** Media Processing MCP Servers

## Overview
GetScreenshot MCP Server exposes the GetScreenshot programmable screenshot API inside MCP-compatible clients. It enables automated, large-scale screenshot capture and processing as part of MCP workflows using a single static MCP server endpoint.

## Features
- **Programmable screenshot API integration**: Access the GetScreenshot API through MCP to perform screenshot operations programmatically.
- **Scale-oriented design**: Built to handle screenshot operations at scale for automation and workflow use cases.
- **Static MCP server URL**: Uses a single, static MCP server endpoint for all clients:
  - `https://mcp.pipedream.net/v2`
- **Client-agnostic usage**: Works with any MCP-compatible chat client or application; you authenticate when adding the server to your app.
- **MCP workflow integration**: Designed to fit into broader MCP workflows for automated capture and processing of screenshots.
