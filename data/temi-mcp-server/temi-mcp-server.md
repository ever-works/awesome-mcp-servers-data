# Temi MCP Server

**Category:** Media Processing MCP Servers  
**Brand:** Temi

Temi MCP Server integrates Temi’s automated English transcription service into MCP-compatible clients, enabling audio and video transcription plus transcript editing workflows.

---

## Description
Temi is an automated audio transcription service that converts English audio or video into text. Through the Temi MCP Server endpoint, MCP clients can access Temi’s transcription and editing capabilities within their own applications.

Server base URL (for MCP clients):

```text
https://mcp.pipedream.net/v2
```

Authentication is handled when adding the server to a supported MCP client.

---

## Features
- **Automated English Transcription**
  - Convert English audio into text.
  - Convert English video audio tracks into text.
- **Transcript Editing Tools**
  - Edit generated transcripts.
  - Annotate transcripts with additional notes or corrections.
- **MCP Integration**
  - Accessible via a static MCP server URL usable by multiple clients.
  - Designed to be added to MCP-based chat or tooling clients.
- **Video & Audio Support**
  - Works with both audio-only and video inputs (audio track transcription).
