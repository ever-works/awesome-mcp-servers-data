# figma-flutter-mcp

**Category:** development-tools-mcp-servers  
**Tags:** figma, flutter, code-generation  
**Source:** https://github.com/mhmzdev/figma-flutter-mcp

## Overview
figma-flutter-mcp is an MCP (Model Context Protocol) server that gives coding agents direct access to Figma design data and design tokens so they can generate Flutter code for building apps, including handling assets and layout details.

## Features
- MCP server interface for integrating with coding/AI agents.
- Direct access to Figma design data from agents.
- Exposure of Figma design tokens to guide code generation (e.g., colors, typography, spacing).
- Generation of Flutter UI code from Figma designs.
- Support for handling layout details extracted from Figma.
- Support for handling design assets (such as images/icons) referenced in Figma when generating Flutter code.
- Multi-language documentation (Korean, Japanese, Simplified Chinese, Traditional Chinese) via separate README files.
