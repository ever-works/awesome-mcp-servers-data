# eGestor MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** eGestor  
**Type:** MCP server integration for an online ERP system

## Overview
The eGestor MCP Server exposes the eGestor online ERP (business management system) through the Model Context Protocol (MCP), enabling automated access to ERP and business management capabilities from compatible chat or agent clients.

Server base URL (static for all clients):
```text
https://mcp.pipedream.net/v2
```

## Features
- **MCP-compatible ERP access**: Connects the eGestor online ERP system to MCP-capable applications and chat clients.
- **Business management integration**: Provides programmatic access to core business management and ERP-related functions of eGestor (e.g., typical ERP areas like accounting and small-business management, as indicated by tags), via MCP.
- **Static server URL**: Uses a single, static MCP server endpoint (`https://mcp.pipedream.net/v2`) for all clients.
- **Per-client authentication**: Authentication is handled when adding/configuring the server in the client application, allowing the same URL to be reused across different users and environments.
- **Chat client compatibility**: Designed to be added to various chat clients that support MCP, enabling conversational or agent-based workflows against eGestor ERP.
