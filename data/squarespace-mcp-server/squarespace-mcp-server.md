# Squarespace MCP Server

MCP Server for Squarespace’s website builder platform, allowing MCP-compatible tools and agents to manage sites, content, and e‑commerce resources via Squarespace APIs.

- **Category:** Content Management MCP Servers  
- **Brand:** Squarespace  
- **Website / Source:** https://mcp.pipedream.com/app/squarespace  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features

- Integrates Squarespace’s website builder platform with MCP-compatible tools and agents.  
- Provides access to Squarespace APIs for managing:  
  - Website content (pages, sections, assets, etc.)  
  - Site configuration and structure  
  - E‑commerce resources (products, orders, related entities)  
- Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) for all clients.  
- Authentication occurs when adding the server to an MCP-compatible application.  
- Built and hosted via Pipedream Connect’s MCP infrastructure.
