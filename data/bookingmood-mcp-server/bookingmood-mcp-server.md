# Bookingmood MCP Server

## Overview
Bookingmood MCP Server provides a Machine Control Protocol (MCP) interface for Bookingmood, a commission‑free booking platform used in travel and hospitality. It allows applications (such as chat clients) to programmatically access availability and booking‑related data via a static MCP server URL.

- **Type:** MCP server / integration endpoint
- **Category:** Business & Commerce – MCP Servers
- **Use cases:** Booking, reservations, hospitality, travel
- **Base URL:** `https://mcp.pipedream.net/v2`

## Features
- **Commission‑free booking platform integration**  
  Integrates with Bookingmood, enabling use of its commission‑free booking capabilities from compatible MCP clients.

- **Static MCP server URL**  
  Uses a single static server URL (`https://mcp.pipedream.net/v2`) that works across clients; authentication is handled when adding the server to your application.

- **Account connection & client selection**  
  Allows you to connect your Bookingmood account and select a specific client/profile before using the tools.

- **Chat client integration**  
  Designed to be added to chat clients that support MCP, enabling conversational access to availability and booking‑related actions.

- **Configuration documentation**  
  Provides a configuration page (via Pipedream) detailing how to add and use the server within supported clients.

- **Tool discovery (actions)**  
  Exposes “available tools” (actions) through the MCP interface for working with booking and availability data. (Specific tools are not listed in the provided content.)

## Setup
1. Connect your Bookingmood account in the Pipedream UI.  
2. Select the appropriate client/profile.  
3. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.  
4. Add this MCP server to your chat client or MCP‑compatible application and authenticate when prompted.
