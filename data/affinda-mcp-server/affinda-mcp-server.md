# Affinda MCP Server

**Category:** Content Extraction & Summarization MCP Servers  
**Brand:** Affinda  
**Website:** https://mcp.pipedream.com/app/affinda

## Overview
Affinda MCP Server connects Affinda’s AI-powered document processing platform to Model Context Protocol (MCP) clients. It enables AI tools to automate document extraction and processing tasks using Affinda’s APIs.

## Features
- **MCP Integration**: Implements an MCP server compatible with tools that support the Model Context Protocol.
- **Affinda Account Connectivity**: Connects directly to your Affinda account for authenticated document processing.
- **AI Tool Compatibility**: Designed to work with AI development and IDE-assistant tools such as:
  - Cursor
  - Claude
  - Windsurf
- **Document Processing Automation**: Allows AI tools to trigger and orchestrate document extraction and processing operations via Affinda.
- **Content Extraction & Insights**: Supports use cases around document processing, content extraction, and AI-driven insights on documents.
