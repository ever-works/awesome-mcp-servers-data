# Go MCP Server

**Brand:** Pipedream  
**Category:** Code Execution & Automation MCP Servers  
**Website:** https://mcp.pipedream.com/app/go  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Description
Go MCP Server is an MCP server that lets you run Go code inside Pipedream workflows. It allows you to use any available Go packages via standard imports, exposing Go-powered functionality to MCP-compatible chat clients and applications.

## Features
- Run arbitrary Go code within Pipedream workflows
- Import and use any available Go packages with standard `import` statements
- Expose Go-based logic and automation via the MCP protocol
- Single static MCP server endpoint (`https://mcp.pipedream.net/v2`) usable across all compatible clients
- Authentication handled when adding the server to your MCP-enabled application
- Works with multiple chat clients through MCP configuration
- Central configuration and setup guidance via the Pipedream MCP Configuration page

## Integration & Usage
- Add the MCP server URL (`https://mcp.pipedream.net/v2`) to your MCP-compatible app or chat client
- Authenticate during the add/connect flow in the client
- Use Go code and Go packages inside Pipedream workflows to power tools, automations, and actions exposed through MCP
