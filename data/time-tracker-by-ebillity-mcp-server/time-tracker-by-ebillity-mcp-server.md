# Time Tracker by eBillity MCP Server

Simple time and billing software exposed as an MCP Server for use with MCP-compatible clients.

**Website:** https://mcp.pipedream.com/app/time_tracker_by_ebillity  
**Category:** Business & Commerce – MCP Servers  
**Brand:** eBillity  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Overview
Time Tracker by eBillity MCP Server provides access to eBillity’s time tracking and billing capabilities through the Model Context Protocol (MCP). It enables automated time tracking and billing workflows from compatible chat and automation clients.

## Features
- MCP-compatible server exposing Time Tracker by eBillity
- Supports automated time tracking operations
- Supports automated billing operations
- Single static MCP endpoint usable across different clients
- Works with multiple MCP-capable chat or automation applications

## Integration & Configuration
- **Static MCP URL:** `https://mcp.pipedream.net/v2` (same for all clients)
- Authentication is performed when adding the server within each client
- Configuration guidance available via the provider’s configuration documentation (per client)
