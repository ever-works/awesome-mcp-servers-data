# Tomba MCP Server

**Category:** Business & Commerce – MCP Servers  
**Website:** https://mcp.pipedream.com/app/tomba

## Overview
Tomba MCP Server is an MCP endpoint that connects Tomba’s professional email search and verification capabilities to AI tools. It enables AI agents and developer workflows to programmatically find and verify professional email addresses for business outreach and lead generation.

## Features
- MCP endpoint for Tomba’s email search and verification API
- Programmatic access to professional email addresses for business outreach
- Email verification to validate professional contacts
- Designed for lead generation and sales prospecting workflows
- Integrates via Pipedream’s MCP server
- Compatible with AI tools such as:
  - Cursor
  - Claude
  - Windsurf

## Use Cases
- Building AI-powered lead generation and prospecting assistants
- Automating professional email discovery for sales and marketing teams
- Verifying business email addresses before outreach
