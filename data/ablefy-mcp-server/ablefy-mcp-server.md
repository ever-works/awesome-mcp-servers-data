# Ablefy MCP Server

**Category:** Business & Commerce · MCP Servers  
**Brand:** Ablefy (formerly elopage)  
**Slug:** ablefy-mcp-server

## Overview
Ablefy MCP Server is an MCP (Model Context Protocol) server integration that connects AI agents and MCP-compatible tools to Ablefy’s sales platform and payment processing for online courses, digital products, memberships, and subscriptions.

It exposes Ablefy (formerly elopage) commerce capabilities to MCP-enabled clients through a standardized server endpoint.

## Features
- **MCP-compatible server**  
  - Implements the Model Context Protocol to integrate with MCP-capable AI agents and tooling.
- **Static server URL**  
  - Single shared endpoint for all clients:  
    `https://mcp.pipedream.net/v2`  
  - Authentication is handled when the server is added to the client/application.
- **Sales platform integration**  
  - Connects to Ablefy’s platform for selling:  
    - Online courses  
    - Digital products  
    - Memberships  
    - Subscriptions  
    - Other similar digital offerings
- **Payment processing**  
  - Provides access to Ablefy’s payment processing for the above products and services.
- **Client-agnostic usage**  
  - Designed to work with any compatible MCP chat client or application via the same URL.
- **Configuration documentation**  
  - Can be configured via a dedicated configuration page (through Pipedream Connect) with per-client setup instructions.

## Technical Details
- **MCP server URL:** `https://mcp.pipedream.net/v2`  
- **Integration host:** Pipedream Connect  
- **Intended clients:** MCP-compatible chat clients and applications that can add external MCP servers.
