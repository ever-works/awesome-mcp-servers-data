# McTime MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** McTime  
**Tags:** time-tracking, workflow-automation

## Description
McTime MCP Server connects McTime’s project time tracking and absence recording system with MCP-compatible applications. It enables automated time entry, reporting, and analysis by exposing McTime’s time recording capabilities through a standard MCP server endpoint.

## Features
- **Project time tracking** – Record working hours against projects and tasks for individuals and teams.
- **Absence recording** – Track absences (such as vacation or leave) alongside project time.
- **Support for small and large teams** – Designed to handle usage from small groups up to larger organizations.
- **Automatic time recording** – Includes fully automatic time recording to reduce manual input.
- **MCP integration** – Available as an MCP server that can be added to compatible chat or agent clients.
- **Standard MCP endpoint** – Uses a static MCP server URL (`https://mcp.pipedream.net/v2`) for all clients; authentication is handled when adding the server to the application.
- **Workflow automation support** – Enables building automated flows for time entry, reporting, and analysis through MCP-aware tools.

## Technical Details
- **MCP server URL:** `https://mcp.pipedream.net/v2`  
- **Integration host:** Provided via Pipedream Connect as the MCP infrastructure layer.
