# Bitbucket Data Center MCP Server

**Description**  
An MCP Server that connects to Bitbucket Data Center, enabling MCP-compatible agents to interact with self-managed Bitbucket repositories for source code collaboration and automation.

- **Website / Source**: https://mcp.pipedream.com/app/bitbucket_data_center  
- **Category**: Repository & Code Analysis MCP Servers  
- **Brand**: Atlassian  
- **Tags**: bitbucket, repository-management, devops

## Features

- Connects MCP-compatible agents to Bitbucket Data Center (self-managed Bitbucket instances).  
- Enables interaction with self-managed Bitbucket repositories for:
  - Source code collaboration workflows.
  - Automation scenarios leveraging Bitbucket as a code host.
- Uses a static MCP server URL for all clients: `https://mcp.pipedream.net/v2`.
- Supports authentication during server addition/registration in clients (authentication handled when you add the server to your app).  
- Designed to work with multiple chat or agent clients (add the same server URL to different clients).  
- Provided and hosted via Pipedream Connect infrastructure.

## Integration & Setup

- **MCP server URL**: `https://mcp.pipedream.net/v2` (static, same for every client).
