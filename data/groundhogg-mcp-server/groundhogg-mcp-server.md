# GroundHogg MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Groundhogg Inc.  
**Website / Source:** https://mcp.pipedream.com/app/groundhogg

## Description
GroundHogg MCP Server is an MCP (Model Context Protocol) server integration provided via Pipedream that connects MCP-based tools and chat clients to the Groundhogg WordPress CRM and marketing automation plugin. It exposes Groundhogg’s CRM and automation capabilities to compatible MCP clients through a static server endpoint.

## Features
- **MCP-compatible server endpoint**  
  - Provides a static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same URL is used across clients; authentication is configured when adding the server to an application.

- **Integration with Groundhogg (WordPress CRM & Marketing Automation)**  
  - Connects MCP-based tools and chat clients to the Groundhogg CRM plugin running on WordPress.  
  - Intended to allow programmatic or conversational access to CRM and marketing automation capabilities exposed via MCP.

- **Client-agnostic usage**  
  - Designed to be added to various chat clients that support MCP.  
  - Setup instructions are available per client via the Pipedream configuration resources.

- **Configuration resources**  
  - Linked “Configuration” page on Pipedream provides full setup and usage details for the MCP server.  
  - Authentication and connection steps are handled during server addition inside the target application.
