# 21RISK MCP Server

MCP Server for 21RISK, enabling access to its compliance, audit, and risk management features via the Model Context Protocol (MCP).

- **Name:** 21RISK MCP Server  
- **Category:** Security Attestation MCP Servers  
- **Slug:** `21risk-mcp-server`  
- **Source URL:** https://mcp.pipedream.com/app/_21risk  
- **Brand:** 21RISK

---

## Features

- Provides an MCP-compliant server interface for 21RISK.  
- Enables access to 21RISK compliance, audit, and risk management capabilities from MCP-compatible chat clients and applications.  
- Uses a single static MCP server URL for all clients:
  - `https://mcp.pipedream.net/v2`  
- Authentication is performed when adding the server to your MCP-enabled application.  
- Hosted and delivered via Pipedream Connect infrastructure.

---

## Integration

- Add the MCP server URL to your MCP-compatible chat client or application configuration.  
- A general configuration guide is available via the platform’s configuration page (referenced as “Configuration page” on the source site).
