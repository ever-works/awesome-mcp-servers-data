# Helcim MCP Server

**Category:** Finance & Market Data MCP Servers  
**Tags:** payments, credit-card-processing

## Overview
The Helcim MCP Server is a Model Context Protocol (MCP) server that connects MCP-compatible clients to Helcim’s payment platform. It enables applications to work with Helcim’s payment and credit card processing services for businesses operating in Canada and the United States.

## Features
- **MCP-compatible server endpoint**  
  - Provides a static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Can be added to any supported MCP/chat client.

- **Helcim payments integration**  
  - Connects to Helcim, a payments company for accepting credit cards.  
  - Supports payment and credit card processing for businesses in Canada and the US.

- **Account-based configuration**  
  - Requires connecting a Helcim account.  
  - Once connected, you can select a client within the interface to start using tools.

- **Tooling via MCP**  
  - Exposes Helcim-related “tools” (actions) to MCP clients (listed as available tools/actions in the UI, loaded dynamically).

- **Centralized configuration documentation**  
  - Additional setup instructions are available through a configuration page linked from the interface.
