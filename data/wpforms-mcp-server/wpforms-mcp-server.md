# WPForms MCP Server

**Description**  
WPForms MCP Server is an integration that connects your WPForms account to AI tools via Pipedream’s MCP server, enabling AI agents and automated workflows to create and manage WordPress forms.

**Category**

**Brand**

**Tags**

## Features
- Connects a WPForms account to AI tools through Pipedream’s MCP server
- Enables AI agents to create and manage WordPress forms
- Supports automated workflows involving WPForms
- Integrates with AI tools such as Cursor, Claude, and Windsurf

## Integrations
- WPForms (WordPress forms plugin)
