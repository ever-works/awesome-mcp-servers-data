# Basin MCP Server

## Overview
Basin MCP Server is an MCP-compatible server that connects Basin’s form backend services to MCP-enabled applications. It lets you deploy and use a form backend for surveys and forms without writing any server-side code.

- **Item type:** MCP server / workflow automation backend
- **Category:** Workflow Automation – MCP Servers
- **Use cases:** Process form and survey submissions, integrate Basin forms into MCP-based chat clients or tools.

## Features

- **No server-side code required**
  - Deploy and use a form backend for forms and surveys without implementing custom backend infrastructure.

- **MCP-compatible server endpoint**
  - Static MCP server URL: `https://mcp.pipedream.net/v2`
  - Works across all supported MCP clients.

- **Basin integration**
  - Connects to a Basin account to use Basin as the form backend.
  - Intended for processing form submissions via Basin’s services.

- **Client-agnostic configuration**
  - Same server URL for every MCP client; authentication happens when you add the server to your application.
  - Supports adding the server to multiple chat or MCP-enabled applications.

- **Configuration guidance**
  - Provides instructions per chat client (via configuration documentation) on how to add and configure the MCP server.
