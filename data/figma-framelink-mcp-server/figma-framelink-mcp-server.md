## Overview

Figma (Framelink) MCP Server bridges the design-to-code gap by allowing AI agents to access Figma designs directly. Instead of generating code from verbal descriptions, agents can reference actual Figma files, inspect layouts, extract assets, and understand component structures.

## Features

- Read design tokens directly from Figma files
- Inspect layouts and extract styling information
- Access and extract design assets
- Understand component structures and hierarchies
- Enables agents to generate UI code based on actual design specifications
- Listed in the original "11 most useful" thread

## Use Cases

- Design-to-code workflows where agents reference actual Figma files
- Automatic UI generation from design specifications
- Extracting design tokens for development handoff
