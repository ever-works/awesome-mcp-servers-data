# Superdocu MCP Server

Superdocu MCP Server is an MCP integration for Superdocu’s document collection platform, enabling AI agents and chat-based tools to orchestrate and track document requests from clients, employees, or partners via a standardized MCP endpoint.

- **Category:** Document Management MCP Servers  
- **Brand:** Superdocu  
- **Source URL:** https://mcp.pipedream.com/app/superdocu  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features

- **Document collection orchestration**  
  - Connects to Superdocu’s document collection platform through MCP.  
  - Allows AI agents to request and track documents from clients, employees, or partners.

- **Standard MCP integration**  
  - Exposes Superdocu functionality via a single static MCP server URL.  
  - The same endpoint works across different MCP-compatible clients.

- **Authentication at client level**  
  - Authentication is performed when adding the server to an MCP-compatible application or chat client.  
  - Supports secure access to Superdocu resources through the provided URL.

- **Client-agnostic setup**  
  - Designed to work with multiple chat clients and MCP-enabled tools.  
  - Configuration instructions available via the platform’s configuration page.

- **Hosted by Pipedream Connect**  
  - MCP server is provided and hosted through Pipedream’s infrastructure.  
  - Uses Pipedream’s terms, privacy, and cookie policies.

## Usage

1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.  
2. Add this URL as an MCP server in your chat client or MCP-compatible app.  
3. Authenticate within your client when prompted.  
4. Use the integration to coordinate and monitor document requests through Superdocu.
