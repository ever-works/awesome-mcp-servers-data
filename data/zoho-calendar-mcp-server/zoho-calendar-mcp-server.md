# Zoho Calendar MCP Server

An MCP (Model Context Protocol) server that exposes Zoho Calendar’s scheduling and calendar management capabilities for integration with MCP-compatible AI assistants and clients.

- **Category:** Project Management MCP Servers  
- **Brand:** Zoho  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features

- Integrates Zoho Calendar with MCP-compatible AI assistants and chat clients.
- Provides access to Zoho Calendar’s online business calendar capabilities for scheduling and calendar management.  
- Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) for all supported clients.  
- Authentication occurs when adding the server to an MCP-compatible application.  
- Configuration documentation available via the platform’s configuration page (client-specific setup guidance).
