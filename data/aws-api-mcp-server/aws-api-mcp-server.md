# AWS API MCP Server

**Category:** Cloud & DevOps MCP Servers  
**Brand:** Amazon Web Services (AWS)  
**Website:** https://modelcontextprotocol.io/mcp/servers/aws-api-mcp-server

## Overview
AWS API MCP Server is a core Model Context Protocol (MCP) server that enables AWS CLI–style operations against AWS services and resources. It exposes AWS capabilities to MCP-compatible clients so they can programmatically interact with AWS infrastructure.

## Features
- Executes AWS CLI–style operations via MCP
- Provides broad interaction with AWS services and resources through the Model Context Protocol
- Acts as a core AWS-focused MCP server for integrating AWS workflows into MCP clients

## Integrations
- Designed to work with AWS services and resources using AWS-style (CLI-like) operations
- Accessible from MCP-compatible clients and tools
