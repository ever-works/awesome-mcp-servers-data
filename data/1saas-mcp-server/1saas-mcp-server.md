# 1SaaS MCP Server

**Category:** Workflow Automation MCP Servers  
**Brand:** 1SaaS  
**Slug:** `1saas-mcp-server`

1SaaS MCP Server is an MCP server integration that connects the 1SaaS low-code / no-code tooling platform with MCP-based workflows, allowing you to automate work across APIs and different projects.

---

## Features

- **MCP Server Integration**
  - Exposes 1SaaS capabilities through the Model Context Protocol (MCP) for use in compatible clients.
  - Integrates low-code and no-code workflows into MCP-based environments.

- **Automation Across APIs and Projects**
  - Designed to orchestrate automation that spans multiple APIs.
  - Supports use across various projects built on the 1SaaS platform.

- **Static MCP Server URL**
  - Single static endpoint usable from any supported MCP client:
    - `https://mcp.pipedream.net/v2`
  - Authentication is handled when adding the server to your application.

- **Client-Agnostic Usage**
  - Works with multiple chat or MCP-compatible clients.
  - Configuration instructions are available via the platform’s configuration page (e.g., “Add the server to your app” flows per client).

- **Pipedream Connect Integration**
  - Hosted and powered by Pipedream Connect.
  - Inherits standard Pipedream terms, privacy, and cookie policy framework.

---

## Setup

1. Use the MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add this server URL in your MCP-compatible client or application.
3. Authenticate when prompted within your client.
4. Configure workflows to leverage 1SaaS low-code / no-code and API automation capabilities.
