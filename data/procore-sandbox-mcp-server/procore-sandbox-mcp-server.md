# Procore Sandbox MCP Server

An MCP Server for the Procore Sandbox environment, used to test and develop integrations with Procore’s construction management platform.

- **Website:** https://mcp.pipedream.com/app/procore_sandbox
- **Category:** Business & Commerce MCP Servers
- **Brand:** Procore
- **Slug:** `procore-sandbox-mcp-server`
- **Tags:** construction, project-management, sandbox

## Features

- **Sandbox Environment Integration**  
  - Connects to the Procore Sandbox environment for safe testing and development.  
  - Intended for building and validating integrations with Procore’s construction management platform without impacting production data.

- **Static MCP Server URL**  
  - Single, static MCP endpoint for all clients:  
    - `https://mcp.pipedream.net/v2`  
  - Same URL works across compatible chat or MCP clients.

- **Client-Agnostic Setup**  
  - Designed to be added to various chat/MCP-compatible applications.  
  - Authentication occurs when adding/configuring the server in the client.

- **Configuration Guidance (via host platform)**  
  - Configuration instructions available through the host platform’s configuration page (e.g., how to add the server to supported chat clients).
