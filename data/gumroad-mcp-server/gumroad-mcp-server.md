# Gumroad MCP Server

**Category:** Business & Commerce MCP Servers  
**Tags:** e-commerce, creators, digital-products

## Overview
Gumroad MCP Server is an MCP-compatible integration that connects your Gumroad account to AI tools via Pipedream. It enables creators and businesses using Gumroad to expose their ecommerce and audience-building capabilities to AI-driven workflows and developer tools.
