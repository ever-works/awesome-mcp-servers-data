# Roll MCP Server

**Category:** Business & Commerce / MCP Servers  
**Brand:** Roll  
**Source:** https://mcp.pipedream.com/app/roll

## Overview
Roll MCP Server exposes Roll’s business management platform to AI agents and MCP-compatible clients, allowing programmatic access to project, sales, and business data.

## Features
- **MCP-compatible server**
  - Provides a Machine Control Protocol (MCP) server endpoint for integration with MCP-capable chat clients and AI agents.
- **Static server URL**
  - Single endpoint for all clients: `https://mcp.pipedream.net/v2`.
  - URL is the same regardless of client; authentication is handled when adding the server to each application.
- **Roll platform integration**
  - Connects to Roll’s business management platform, which includes:
    - Project management capabilities.
    - Sales tracking capabilities.
    - Business performance visibility to understand and grow the business.
- **Programmatic data access**
  - Designed so AI agents and tools can access Roll project, sales, and business data programmatically (e.g., for automations, workflows, and assistants).
- **Client setup guidance**
  - Documentation and configuration instructions available via the Pipedream configuration page to help add the server to compatible chat or AI apps.
- **Pipedream Connect infrastructure**
  - Runs on Pipedream Connect, leveraging Pipedream’s infrastructure for hosting and connectivity.

## Integration & Configuration
- MCP server base URL: `https://mcp.pipedream.net/v2`
- Added to applications via MCP configuration in compatible chat clients.
- Full configuration documentation available on the Pipedream configuration page (linked from the app page).
