# Getty Images MCP Server

![Getty Images Logo](https://www.gettyimages.com/gi-resources/images/LogoShare.jpg)

An MCP server integration that connects a Getty Images account to MCP-based AI tools, enabling search and retrieval of professional news, fashion, and entertainment imagery.

![Getty Images MCP Screenshot](https://media.gettyimages.com/id/1170940259/photo/getty-images-homepage-sample.jpg?s=2048x2048)

## Category
- Media Processing MCP Servers

## Features
- Connects a Getty Images account via Pipedream's MCP server
- Enables MCP-based tools (e.g., Cursor, Claude, Windsurf) to access Getty Images
- Supports search of professional imagery
- Covers news, fashion, and entertainment image content
- Retrieval of stock and editorial media suitable for AI-assisted workflows

## Integrations
- Cursor
- Claude
- Windsurf
- Other MCP-compatible tools
