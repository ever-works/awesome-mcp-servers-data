## Overview

The Pulumi Model Context Protocol (MCP) server enables AI-powered coding assistants to access your Pulumi Cloud resources and delegate infrastructure tasks to Pulumi Neo. This integration allows you to query your cloud infrastructure, search for resources, and launch automated infrastructure workflows.

## Features

- **Automated Deployment**: deploy-to-aws automatically analyzes application files and generates Pulumi infrastructure code for AWS deployment
- **CLI Integration**: Run pulumi preview and pulumi up commands to deploy changes
- **Stack Management**: Retrieve outputs from deployed stacks and manage infrastructure state
- **Resource Querying**: Query and search for cloud infrastructure resources
- **Neo AI Integration**: Delegate infrastructure tasks to Pulumi Neo for automated workflows

## Deployment Options

- **Remote MCP Server**: Hosted service available to all Pulumi users with no installation required
- **Local Server**: Can be installed and run locally for development workflows

## Supported AI Assistants

Cursor, Claude Code, Windsurf, Claude Desktop, and any MCP-compatible AI assistant.

## Authentication

Requires Pulumi access token for authentication with Pulumi Cloud.
