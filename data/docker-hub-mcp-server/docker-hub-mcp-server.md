# Docker Hub MCP Server

## Overview
Docker Hub MCP Server is an MCP (Model Context Protocol) server that connects to Docker Hub, enabling MCP-based workflows to search, pull, and manage container images from the Docker Hub registry.

- **Category:** Cloud & DevOps MCP Servers
- **Brand:** Docker
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- Connects to Docker Hub, the Docker container image registry
- Supports MCP-based workflows for interacting with Docker Hub
- Enables searching for container images in Docker Hub
- Enables pulling and managing container images from Docker Hub via MCP clients
- Uses a single static MCP server URL that works across supported clients
- Authentication performed when adding/configuring the server in the client application

## Usage
- Add the MCP server to a compatible chat client or MCP-enabled application using:
  - **Server URL:** `https://mcp.pipedream.net/v2`
- Configure authentication within your chosen client when prompted.
- Refer to the provider’s configuration documentation for step-by-step setup.
