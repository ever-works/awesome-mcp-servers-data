# Zoho People MCP Server

## Overview
Zoho People MCP Server is a Model Context Protocol server that connects AI systems to Zoho People, a cloud-based HR management platform. It enables interaction with Zoho People’s HR features—such as employee data, attendance, and HR workflows—via MCP tools using a single static server URL.

- **Category:** Business & Commerce MCP Servers
- **Brand:** Zoho
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible HR integration**
  - Exposes Zoho People functionality through the Model Context Protocol.
  - Allows AI agents and MCP-capable clients to call Zoho People tools.

- **Access to HR management data and operations**
  - Integration with cloud-based HR features like:
    - Employee data and records (e.g., profiles, basic HR information).
    - Attendance-related information and actions.
    - HR workflows and process automation exposed via Zoho People.

- **Static server URL for all clients**
  - Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) that works across compatible clients.
  - Authentication occurs when adding/configuring the server in your application.

- **Client-agnostic setup**
  - Can be added to different MCP-supporting chat or agent clients.
  - Configuration details are provided via the associated configuration documentation (e.g., Pipedream Connect configuration page).
