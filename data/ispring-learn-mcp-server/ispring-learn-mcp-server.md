# iSpring Learn MCP Server

**Category:** Documentation & Learning Resources  
**Tags:** education, course-management, LMS

## Overview
The iSpring Learn MCP Server is a Model Context Protocol (MCP) server integration for iSpring Learn, a cloud‑based learning management system used for onboarding, upskilling, and certifying teams. It is hosted via Pipedream and provides a static MCP endpoint that can be added to compatible chat or MCP-enabled applications.

## Features
- **Static MCP Endpoint**  
  - Single, static MCP server URL for all clients: `https://mcp.pipedream.net/v2`  
  - Authentication is handled when adding the server to your MCP-compatible application.

- **iSpring Learn LMS Integration**  
  - Designed to work with iSpring Learn, a cloud-based LMS.  
  - Intended for use cases such as onboarding, employee upskilling, and certification workflows.

- **Client-Agnostic Setup**  
  - Usable with multiple MCP-capable chat clients or applications.  
  - Configuration steps are provided per client (via the Pipedream configuration page).

- **Configuration Guidance**  
  - Central configuration page (on Pipedream) with instructions on how to:  
    - Connect your iSpring Learn account.  
    - Select and configure a specific client.  
  - “Available tools” section (actions and tools are loaded dynamically in the Pipedream UI).

## Integration & Setup
- **MCP Server URL**: `https://mcp.pipedream.net/v2`  
- **Setup Steps (high level)**:  
  1. Connect your iSpring Learn account in Pipedream.  
  2. Copy the static MCP server URL.  
  3. Add the MCP server to your chosen client or application.  
  4. Follow client-specific setup guidance from the Pipedream configuration page.
