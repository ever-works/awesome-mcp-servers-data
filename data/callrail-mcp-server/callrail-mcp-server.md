# CallRail MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** CallRail  
**Source:** https://mcp.pipedream.com/app/callrail

## Overview
CallRail MCP Server connects MCP-compatible clients to CallRail’s lead intelligence and call tracking platform via a static MCP endpoint provided by Pipedream Connect.

## Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL usable across compatible clients:  
    `https://mcp.pipedream.net/v2`
  - Server is added at the client level; authentication is handled when configuring the server in the MCP client.

- **Integration with CallRail**  
  - Acts as a bridge between MCP clients and CallRail’s lead intelligence and call tracking platform.  
  - Intended to expose CallRail capabilities (e.g., lead intelligence and call tracking data) to applications that support MCP.

- **Client-agnostic setup**  
  - The same static URL is used for all supported chat or MCP clients.  
  - Configuration is managed in each client using that URL and appropriate authentication.

- **Hosted and managed by Pipedream**  
  - Delivered via Pipedream Connect infrastructure.  
  - Uses Pipedream’s platform terms and privacy policies.

## Setup
- Use the static MCP server URL: `https://mcp.pipedream.net/v2` when adding a new MCP server in your client.  
- Authenticate within your MCP or chat client as prompted during server configuration.
