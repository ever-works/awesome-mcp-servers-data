## Verodat MCP Server

### Overview
Verodat MCP Server is an open-source Model Context Protocol (MCP) server that enables AI tools and agents to interact with the Verodat AI Ready Data platform.

### Features
- **MCP-compliant server**: Implements the Model Context Protocol to expose Verodat platform capabilities to compatible AI clients and runtimes.
- **Verodat platform integration**: Designed specifically to connect with the Verodat AI Ready Data platform for data-related operations (as supported by the platform and server implementation).
- **Node.js/TypeScript implementation**: Project structure includes `package.json` and `tsconfig.json`, indicating a TypeScript-based implementation running on Node.js.
- **Containerization support**: Includes a `Dockerfile` for building and running the MCP server in containerized environments.
- **Configuration & metadata files**:
  - `smithery.yaml` for integration or deployment via Smithery.
  - `glama.json` for MCP server metadata/registration in compatible tools.
- **Testing setup**: Contains a `tests` directory for automated tests of the server behavior.
- **Open-source licensing**: Includes a `LICENSE` file in the repository (exact license type specified in that file).
- **Security assessment badge**: Public MseeP.ai security assessment badge linked from the README, indicating the codebase is scanned and has a published assessment.
