# MoonMail MCP Server

Simple email marketing MCP server that exposes MoonMail’s capabilities via the Model Context Protocol, enabling automated email campaign operations from compatible clients.

- **Name:** MoonMail MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Brand:** MoonMail  
- **Source URL:** https://mcp.pipedream.com/app/moonmail  
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features

- **MCP-compatible email marketing server**  
  - Provides Model Context Protocol access to MoonMail’s email marketing features.  
  - Can be added as a server to any compatible chat or MCP client.

- **Static MCP server URL**  
  - Uses a fixed endpoint: `https://mcp.pipedream.net/v2`.  
  - Same URL works across all supported clients; configuration focuses on client-side setup and authentication.

- **Authentication via client configuration**  
  - Authentication is performed when adding the server to your application / client.  
  - Supports centralized configuration through the client’s MCP server settings.

- **Integration with MoonMail email workflows**  
  - Designed to control simple email marketing and campaign operations (e.g., sending campaigns, automating basic email actions) from within your MCP-enabled tools.  
  - Enables automation of email-related tasks through conversational or tool-based interfaces.

- **Pipedream Connect integration**  
  - Hosted and powered by Pipedream Connect’s MCP infrastructure.  
  - Configuration details and additional setup guidance available via the Pipedream configuration page.

- **Multi-client support**  
  - Documentation on adding the server is organized by chat / MCP client.  
  - Intended to work with any MCP-compliant client using the static URL.

## Configuration

- **Server URL:** `https://mcp.pipedream.net/v2`  
- **Setup steps (high level):**  
  1. Open your MCP- or chat-compatible client.  
  2. Add a new MCP server using the URL above.  
  3. Provide any required credentials or tokens during the add-server / authentication step.  
  4. Save and enable the server to start invoking MoonMail email operations.

- **Additional configuration docs:**  
  - Full configuration guidance is available on the Pipedream **Configuration** page linked from the app.
