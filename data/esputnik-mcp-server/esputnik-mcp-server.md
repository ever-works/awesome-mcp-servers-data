# eSputnik MCP Server

**Category:** Messaging MCP Servers  
**Brand:** eSputnik  
**Slug:** esputnik-mcp-server

## Description
The eSputnik MCP Server is an MCP-compatible server for the eSputnik omnichannel messaging platform. It enables programmatic access to multiple customer communication channels—email, SMS, Viber, and web/mobile push notifications—through a unified MCP endpoint.

## Features
- **MCP-compatible server**
  - Exposes eSputnik functionality via the Model Context Protocol (MCP)
  - Can be added as a server to compatible chat or AI clients
- **Single static server URL**
  - MCP server endpoint: `https://mcp.pipedream.net/v2`
  - Same URL for all clients; authentication handled when you add the server to your application
- **Omnichannel messaging**
  - Email messaging
  - SMS messaging
  - Viber messaging
  - Web push notifications
  - Mobile push notifications
  - Ability to combine multiple channels in a workflow or campaign
- **Integration via client configuration**
  - Designed to be added to chat or AI applications as an MCP server
  - Configuration details provided via a separate configuration page (not included here)
