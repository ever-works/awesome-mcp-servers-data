# Canva MCP Server

## Overview

The Canva MCP Server is a remote Model Context Protocol (MCP) server that connects AI assistants (such as ChatGPT and Claude) to Canva’s design platform via OAuth 2.1. It exposes Canva’s design, export, and collaboration capabilities as tools that can be invoked from within conversational AI workflows.

## Features

### Design Discovery & Content Management
- Search designs (docs, presentations, videos, whiteboards, sheets) by title or content
- Retrieve detailed design metadata (owner, URLs, thumbnails, other metadata)
- Create folders for organizing designs
- Move designs and assets between folders
- List folder contents for project and asset management

### AI-Powered Design Creation
- Generate new designs (e.g., presentations, decks, visual content) from natural language prompts
- Create context-aware decks and visual content based on the current chat or conversation context
- Import content from a URL and convert it into editable Canva designs
- Generate multiple design variations using templates and AI assistance

### Export & Sharing
- Export designs in multiple formats: PDF, JPG, PNG, PPTX, GIF, MP4
- Track export job status and completion
- Retrieve download URLs for completed exports
- Generate public sharing links for designs (controlled via Canva sharing settings)

### Collaboration & Communication
- Add comments to designs
- List existing comments and discussion threads on designs
- Reply to comments and participate in ongoing feedback threads
- Support team collaboration workflows around design review and iteration
- Streamline feedback loops by handling comments and changes directly from AI chat interfaces

### ChatGPT Connector Capabilities
- Search and surface Canva designs directly in ChatGPT
- Summarize and reference existing Canva designs in conversation
- Enable context-aware responses by giving ChatGPT access to relevant design content
- Provide a seamless workflow for finding and using the latest versions of designs without manual folder browsing

### Claude Connector Capabilities
- Generate new Canva designs directly from within the Claude chat interface
- Edit and manipulate existing Canva designs through Claude
- Perform real-time design operations: create decks, resize assets, import assets, and export assets via chat
- Discuss Canva Apps SDK and Connect APIs with AI assistance, leveraging Canva-specific knowledge for development workflows

### Integration & Protocol
- Remote MCP server accessible at `https://mcp.canva.com/mcp`
- Integrates Canva’s design capabilities into MCP-compatible clients (e.g., AI IDEs, chat tools)
- Uses OAuth 2.1 for user authentication and secure access to Canva accounts

### Intended Use Cases
- **Marketing and content teams:** discover, generate, and export campaign assets and presentations from within AI tools
- **Sales teams:** create and customize decks and proposals on the fly based on live client conversations
- **Educators and content creators:** generate teaching materials, course content, and templates programmatically
- **Developers and technical teams:** explore, test, and integrate Canva APIs and apps with AI-assisted workflows
