# Xendit MCP Server

**Category:** Finance & Market Data MCP Servers  
**Tags:** payments, payouts

## Overview
The Xendit MCP Server is an MCP-compatible server that connects MCP-enabled agents or chat clients to the Xendit platform. It enables agents to work with Xendit’s payment acceptance and disbursement (payout) capabilities through a standardized MCP endpoint.

## Features
- **MCP-compatible server endpoint**  
  - Provides a single static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Can be added to any supported MCP-compatible chat client or application.

- **Connection to Xendit platform**  
  - Integrates MCP agents with Xendit’s payment acceptance features.  
  - Enables access to payout / disbursement capabilities.

- **Client-agnostic setup**  
  - The same MCP server URL works for every compatible client.  
  - Authentication is performed when adding the server to an application or client.

- **Configuration workflow**  
  - Connect a Xendit account and select a client to start using the MCP server.  
  - Guided configuration available via the platform’s configuration page.

- **Tooling integration**  
  - Designed to expose Xendit actions and tools to MCP agents (payment and payout operations), which load dynamically for the client.
