# Benzinga MCP Server

Benzinga MCP Server provides Model Context Protocol (MCP) based access to Benzinga’s actionable trading ideas and real-time financial news, allowing AI agents to work with market data through an MCP server interface.

## Overview
- **Type:** MCP server / integration
- **Category:** Finance, Market Data, MCP Servers
- **Provider:** Benzinga
- **Slug:** `benzinga-mcp-server`
- **Source URL:** https://mcp.pipedream.com/app/benzinga

## Features
- **MCP-based integration**
  - Exposes Benzinga data via the Model Context Protocol (MCP) for compatibility with MCP-enabled AI tools.
- **Actionable trading ideas access**
  - Provides programmatic access to Benzinga’s actionable trading ideas (strategy and signal-oriented market content).
- **Real-time financial news**
  - Streams or retrieves Benzinga real-time financial news for use inside AI workflows.
- **AI tool connectivity via Pipedream**
  - Uses Pipedream’s MCP server to bridge a Benzinga account with supported AI tools.
  - Mentioned supported tools include:
    - Cursor
    - Claude
    - Windsurf
- **Account-based data access**
  - Connects to a user’s Benzinga account to pull associated market data via MCP.

## Integrations
- **Pipedream MCP Server** – hosts the MCP implementation that connects Benzinga to AI tools.
- **AI Clients** (via MCP):
  - Cursor
  - Claude
  - Windsurf
