# Adhook MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** adhook  
**Slug:** adhook-mcp-server

## Description
Adhook MCP Server is an MCP (Model Context Protocol) server implementation for Adhook that enables automation and management of Google and social media brand campaigns through Pipedream. It provides a unified MCP endpoint that can be added to compatible chat or AI applications for campaign-related workflows.

## Features
- **MCP-based integration**
  - Implements an MCP server compatible with MCP-enabled clients.
  - Exposes Adhook’s campaign management capabilities via MCP.

- **Campaign automation and management**
  - Designed for automation of Google and social media brand campaigns.
  - Supports management of advertising and marketing workflows through Pipedream.

- **Single static MCP URL**
  - Uses a static MCP server URL for all clients:  
    `https://mcp.pipedream.net/v2`
  - Same endpoint for every client; authentication is handled when adding the server to your application.

- **Pipedream Connect integration**
  - Runs on top of Pipedream Connect infrastructure.
  - Can be used within Pipedream-based automation setups.

- **Multi-client compatibility**
  - Can be added to different MCP-compatible chat or AI applications.
  - Configuration guidance available via a dedicated configuration page (on the provider’s site).

## Technical Details
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`  
- **Authentication:** Performed when you add the server to your application (client-specific auth flow).
