# Recurly MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Pipedream  
**Source:** https://mcp.pipedream.com/app/recurly

## Description
Recurly MCP Server is a Pipedream-based MCP server that connects Recurly’s subscription management and billing capabilities to MCP-enabled tools and chat clients. It exposes Recurly functionality through a unified MCP endpoint so applications can interact with subscription and billing data via the MCP protocol.

## Features
- MCP-based server for Recurly subscription and billing functionality
- Single static MCP endpoint usable across compatible clients
- Works with multiple MCP-enabled chat clients and applications
- Authentication handled when adding the server to each application
- Built and hosted on Pipedream’s infrastructure
- Configuration guidance available via a dedicated configuration page

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Integration model:** Connects Recurly APIs to MCP-enabled tools through Pipedream
