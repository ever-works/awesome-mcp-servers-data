# RocketReach MCP Server

Accurate, up-to-date professional contact information via an MCP Server integration with RocketReach.

- **Website:** https://mcp.pipedream.com/app/rocketreach
- **Category:** Business & Commerce – MCP Servers
- **Brand:** RocketReach
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Overview
The RocketReach MCP Server provides programmatic access to RocketReach’s professional contact database through the Model Context Protocol (MCP). It is designed to be added as a server to compatible chat or AI clients, enabling them to look up accurate, current professional contact details from within the client.

## Features
- **MCP-compatible server**
  - Exposes RocketReach functionality over a standard MCP server endpoint.
  - Uses a single static base URL for all clients: `https://mcp.pipedream.net/v2`.

- **Programmatic contact data access**
  - Integrates with RocketReach to access professional contact information.
  - Intended for use in workflows such as prospecting and sales automation (as indicated by tags).

- **Client-agnostic configuration**
  - Same server URL for every supported MCP client.
  - Authentication is performed when adding the server to your chosen application.

- **Pipedream Connect integration**
  - Operates via Pipedream’s “Connect” infrastructure for MCP servers.
  - Centralized configuration and management through Pipedream’s configuration page.

## Use Cases
- **Contact data enrichment** for sales or business development workflows.
- **Prospecting** by pulling professional contact details into chat or AI tools.
- **Sales automation** where an AI agent can query contact information as part of automated outreach or CRM workflows.

## Access & Configuration
- **MCP server URL:** `https://mcp.pipedream.net/v2`
- **Setup:**
  - Add this URL as an MCP server in your supported chat/AI client.
  - Complete authentication during the add-server flow within your client.
  - Refer to the Pipedream configuration page (linked from the app page) for client-specific setup steps.
