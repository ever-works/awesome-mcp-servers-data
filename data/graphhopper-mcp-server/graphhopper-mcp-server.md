# GraphHopper MCP Server

**Category:** API Integration – MCP Servers  
**Brand:** GraphHopper

An MCP server that exposes GraphHopper’s route planning and route optimization capabilities for integration into applications via the Model Context Protocol (MCP).

## Features

- **Route planning for applications (SaaS)**  
  - Provides routing functionality that can be embedded into your own applications.

- **Route Optimization API**  
  - Offers a route optimization interface to compute efficient routes (e.g., for logistics and delivery scenarios).

- **MCP-compatible server**  
  - Exposed as an MCP server so it can be added to compatible chat clients and tools.

- **Static MCP server URL**  
  - Server base URL: `https://mcp.pipedream.net/v2`  
  - Same URL works across all supported MCP clients; authentication is handled when you add the server to your application.

- **Pipedream-hosted integration**  
  - Provided via Pipedream Connect infrastructure.  
  - Central configuration and setup guidance available through Pipedream’s configuration page.

- **Use cases (implied by feature set)**  
  - Routing and location-intelligence workflows.  
  - API-based integration for route planning and optimization inside AI/assistant or other MCP-enabled environments.
