# Miestro MCP Server

An MCP (Model Context Protocol) server integration for Miestro, an online course platform that simplifies building and launching online courses.

## Overview
- **Type:** MCP Server / Integration
- **Platform:** Miestro (online course platform)
- **Purpose:** Connect Miestro with MCP-compatible chat clients and applications via Pipedream Connect.
- **Use Case:** Manage or interact with Miestro-based online courses through tools that support MCP servers.

## Features
- **MCP Server Endpoint**
  - Static MCP server URL: `https://mcp.pipedream.net/v2`
  - Single URL works for all compatible clients.

- **Client Integration**
  - Can be added to various MCP-capable chat clients and applications.
  - Authentication occurs when adding the server to the client/app.

- **Configuration Resources**
  - Dedicated configuration/documentation page available via Pipedream for setup details.

- **Platform Context**
  - Built for Miestro, an online course platform designed to help users create and launch courses in a small number of steps.
  - Delivered and maintained via Pipedream Connect.
