# LearnDash MCP Server

An MCP (Model Context Protocol) server for LearnDash, the WordPress LMS plugin that turns WordPress sites into a full learning management system.

## Overview
- Integrates LearnDash (WordPress LMS) with MCP-compatible chat or AI clients.
- Uses a single static MCP server URL for all clients.
- Authentication is handled when adding the server inside your client/application.
- Provided and hosted via Pipedream Connect.

## Features
- **Static MCP Endpoint**  
  - Universal MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same URL works across different MCP-compatible clients.

- **LearnDash Integration**  
  - Designed specifically for LearnDash, the LMS plugin that turns WordPress into a full learning management system.
  - Intended to expose LearnDash-related functionality to chat/AI tools via MCP.

- **Client-Agnostic Setup**  
  - Can be added to various chat or AI clients that support MCP.  
  - Authentication occurs when configuring the server inside your chosen client.

- **Configuration Resources**  
  - Dedicated configuration documentation via a “Configuration” page (not included here) for setup guidance.
