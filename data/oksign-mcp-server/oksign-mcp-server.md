## Features

- **Online document signing workflows**  
  - Supports signing of contracts, purchase orders, forms, and other documents online.  
  - Designed to provide simple and legally compliant e-signature flows.

- **MCP server integration**  
  - Exposed as a Model Context Protocol (MCP) server that AI assistants or MCP-compatible clients can connect to.  
  - Enables AI-driven initiation and management of e-signature workflows from within compatible chat or agent environments.

- **Static MCP server URL**  
  - Single endpoint used by all clients:  
    - `https://mcp.pipedream.net/v2`  
  - URL is static across clients; authentication is handled when adding the server to an application.

- **Client-agnostic usage**  
  - Intended to be added to various chat or MCP-enabled clients.  
  - Configuration instructions are available via the associated configuration page (per chat client).

- **Pipedream Connect integration**  
  - Provided and operated via Pipedream Connect infrastructure.

## Configuration

- Use the static MCP URL `https://mcp.pipedream.net/v2` when adding the server to your MCP-compatible application or chat client.
- Additional per-client setup details are available on the linked configuration page from the source URL.
