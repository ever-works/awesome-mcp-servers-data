# Aha MCP Server

**Category:** Project Management MCP Servers  
**Brand:** Aha!

MCP Server for Aha! that exposes Aha! product roadmap and planning capabilities to MCP-compatible clients via a static server URL.

---

## Features

- **Static MCP Server Endpoint**  
  - Single static MCP server URL for all clients: `https://mcp.pipedream.net/v2`  
  - Authentication handled when adding the server to each MCP-compatible application.

- **Client-Agnostic Integration**  
  - Designed to work with multiple MCP-compatible chat or agent clients.  
  - Setup instructions available per client type via the “Select your chat client” flow.

- **Aha! Account Connection**  
  - Connects to an existing Aha! account to surface product roadmap and planning features through MCP.  
  - Uses Pipedream’s connection flow to authorize access.

- **Tooling / Actions Exposure**  
  - Exposes Aha!-related tools and actions as MCP tools (listed as “Available tools” in the interface).  
  - Tools are dynamically loaded once the server is configured and the account is connected.

- **Configuration Guidance**  
  - Additional setup and configuration details available via a dedicated configuration page on Pipedream.
