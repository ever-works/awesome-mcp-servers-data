# Gladia MCP Server

Audio Intelligence API integration for MCP clients, provided via Pipedream Connect.

## Overview
The Gladia MCP Server exposes Gladia’s Audio Intelligence API to MCP-compatible chat and agent clients, enabling speech-related and audio analysis capabilities through a standardized MCP server endpoint.

## Features
- MCP server integration for Gladia’s Audio Intelligence API
- Provides speech-related and audio analysis capabilities to MCP clients
- Single static MCP server URL usable across different MCP-compatible clients
- Authentication handled when adding the server to each application/client

## MCP Server URL
- **Base MCP Server URL:** `https://mcp.pipedream.net/v2`

This static URL works for all supported MCP clients; authentication is configured within the client when you add the server.

## Integration & Configuration
- Can be added to various MCP-compatible chat or agent clients
- Full configuration details available via the linked configuration documentation (client-specific setup steps, environment variables, etc., are handled there)
