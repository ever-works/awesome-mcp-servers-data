# Pendo MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Pendo  
**Website:** https://mcp.pipedream.com/app/pendo

## Overview
Pendo MCP Server is an MCP (Model Context Protocol) server integration for Pendo’s product experience platform. It exposes Pendo’s product analytics and product experience capabilities to MCP-based clients, enabling applications (such as chat clients or other tools that support MCP) to access and act on Pendo data.

## Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Designed to work with any MCP-capable client.

- **Pendo account integration**  
  - Connect a Pendo account through the Pipedream configuration UI.  
  - Uses your authenticated Pendo account when accessed from MCP clients.

- **Client-agnostic configuration**  
  - Single server URL for all supported clients.  
  - Per-client instructions available (e.g., for different chat or MCP clients).

- **Product analytics & experience exposure**  
  - Provides access to Pendo’s product analytics and product experience capabilities via MCP tools / actions (tools list is dynamically loaded from the server).

- **Central configuration page**  
  - Dedicated configuration page on Pipedream to manage connection, authentication, and setup.

## Usage
1. Connect your Pendo account in the Pipedream Pendo MCP Server app.  
2. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.  
3. Add the server URL to your MCP-compatible application or chat client following that client’s MCP configuration instructions.
