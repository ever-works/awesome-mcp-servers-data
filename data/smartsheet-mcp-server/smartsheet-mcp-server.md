# Smartsheet MCP Server

Smartsheet MCP Server integrates Smartsheet’s dynamic work and collaboration platform with AI agents via the Model Context Protocol (MCP), enabling AI tools and chat clients to interact with Smartsheet data and workflows.

---

## Overview
- **Type:** MCP (Model Context Protocol) server
- **Platform:** Smartsheet (work & collaboration / spreadsheets / work management)
- **Provider:** Pipedream (Pipedream Connect)
- **Category:** Project management MCP servers

---

## MCP Server URL
- **Base URL (static for all clients):**
  ```
  https://mcp.pipedream.net/v2
  ```
- Authentication is performed when adding the server to your MCP-compatible application or chat client.

---

## Features
- **MCP Integration**
  - Exposes Smartsheet capabilities to AI agents via the Model Context Protocol.
  - Designed to be added as a server in MCP-compatible chat clients and tools.

- **Static Server Endpoint**
  - Uses a single, static MCP server URL (`https://mcp.pipedream.net/v2`) for all clients.
  - Authentication handled at the time of adding/configuring the server in the client.

- **Smartsheet Platform Connectivity**
  - Connects to Smartsheet’s dynamic work and collaboration platform.
  - Suitable for work management and spreadsheet-style workflows.

- **Client-Agnostic Setup**
  - Can be used with multiple MCP-enabled chat clients.
  - Configuration instructions are available per client via the platform’s configuration documentation.

- **Pipedream Infrastructure**
  - Hosted and operated via Pipedream Connect.
  - Benefits from Pipedream’s integration tooling and infrastructure.

---

## Getting Started
- Copy the static MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server to your MCP-compatible chat client or application.
- Authenticate with Smartsheet / Pipedream when prompted during setup.
- Optional: Refer to the provider’s **Configuration** page for detailed setup steps by client.
