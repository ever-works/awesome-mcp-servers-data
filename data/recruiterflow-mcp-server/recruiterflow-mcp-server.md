## Features

### MCP Server & Configuration
- Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`  
- Authentication handled when adding the server to your MCP-compatible application
- Usable from multiple chat clients that support MCP
- Central configuration via the Pipedream configuration page

### Available Tools / Actions (ATS & CRM)
1. **Search Candidates**
   - Search for candidates in Recruiterflow
   - Supports simple search queries
   - Supports advanced filter arrays for complex criteria

2. **Move Candidate to Stage**
   - Move a candidate to a different stage in a job pipeline
   - Stage can be specified by **stage ID** or **stage name**

3. **Create Placement**
   - Record a successful candidate placement
   - Store placement data directly in Recruiterflow via the MCP server

4. **Create Job**
   - Create new job postings in Recruiterflow
   - Trigger job creation from within MCP-compatible chat clients

5. **Create Candidate**
   - Create new candidate records in Recruiterflow
   - Useful for quickly adding prospects discovered in conversations

6. **Add Candidate to Job**
   - Associate an existing candidate with a specific job posting
   - Link candidates to open roles as part of automated or conversational workflows
