# Eventzilla MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Eventzilla  
**Source:** https://mcp.pipedream.com/app/eventzilla

## Overview
The Eventzilla MCP Server connects MCP-compatible clients to Eventzilla’s cloud-based event registration and marketing platform. It enables creating and managing virtual or live events from within tools that support the Model Context Protocol.

## Features
- **MCP-compatible endpoint**
  - Static server URL for all clients: `https://mcp.pipedream.net/v2`
  - Can be added to any MCP-enabled chat or agent client.

- **Eventzilla platform integration**
  - Connects to Eventzilla’s cloud-based event registration system.
  - Supports working with both virtual and in-person (live) events.
  - Uses Eventzilla’s event marketing capabilities via the MCP server connection.

- **Multi-client configuration support**
  - Single URL works across different MCP clients.
  - Configuration instructions available via the provider’s configuration page (per client).

## Integration & Configuration
- Add the MCP server to your chosen MCP-compatible chat client using the static URL: `https://mcp.pipedream.net/v2`.
- Detailed configuration steps are available on the linked configuration page for various clients.
