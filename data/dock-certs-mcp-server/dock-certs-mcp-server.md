# Dock Certs MCP Server

## Overview
Dock Certs MCP Server is an MCP (Model Context Protocol) server that integrates with Dock Certs to support MCP-based workflows for issuing, managing, and verifying fraud‑proof digital documents and certificates.

- **Category:** Security & Attestation MCP Servers
- **Brand:** Dock
- **Slug:** dock-certs-mcp-server

## Features
- Integrates Dock Certs with MCP-compatible clients and applications
- Supports issuing digital documents and certificates
- Supports managing existing digital documents and certificates
- Supports verification of digital documents and certificates
- Enables creation and use of fraud‑proof, instantly verifiable documents
- Accessible via a single static MCP server URL compatible with all clients

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Authentication:** Performed when adding the server to your MCP‑capable application or chat client
- **Deployment:** Hosted and powered by Pipedream Connect
