# Aidaform MCP Server

## Overview
The Aidaform MCP Server provides access to Aidaform’s online form creation and data collection capabilities via the Model Context Protocol (MCP), allowing MCP-compatible clients to interact with Aidaform.

- **Category:** Business & Commerce – MCP Servers
- **Use cases:** Creating online forms, collecting responses, running surveys and data collection workflows from MCP-enabled chat or development environments.

## Features
- **MCP-based access:** Exposes Aidaform functionality through the standard MCP interface for compatible clients.
- **Online form creation:** Integrates with Aidaform’s online form builder for creating and managing forms and surveys.
- **Data collection:** Supports collecting and accessing form submission data via MCP-connected tools.
- **Static server endpoint:**
  - MCP server URL: `https://mcp.pipedream.net/v2`
  - Same static URL for all clients; authentication is handled when adding the server to your application.
- **Client-agnostic setup:** Designed to work with multiple MCP-capable chat clients and applications.

## Integration & Setup
- Add the MCP server to your MCP-compatible app using the static endpoint: `https://mcp.pipedream.net/v2`.
- Authentication occurs during the server-add process in your chosen client.
- Additional configuration details are available on the provider’s configuration page (linked from the source site).
