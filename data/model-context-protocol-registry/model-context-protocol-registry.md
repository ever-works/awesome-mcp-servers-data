# Model Context Protocol Registry

**Website:** https://github.com/modelcontextprotocol/registry  
**Category:** MCP server directories & lists  
**License:** MIT

## Description
A community-driven registry service for Model Context Protocol (MCP) servers. It enables ecosystem participants to discover, publish, and manage MCP server implementations in a shared, open registry.

## Features
- **MCP server registry** – Central place to list and discover MCP server implementations.
- **Publishing of MCP servers** – Server authors can add their MCP servers to the registry.
- **Discovery across the ecosystem** – Clients and tools can find available MCP servers from one shared source.
- **Management of server entries** – Designed to support organizing and maintaining MCP server metadata.
- **Open source implementation** – Source code, docs, and deployment artifacts are available in the repository.
- **Documentation** – Project documentation is maintained under the `docs/` directory in the repo.
- **Deployment resources** – Configuration and deployment-related files are provided under the `deploy/` directory.
- **Tested project** – Integration tests are included under `tests/integration/` to help ensure correctness.

## Pricing
