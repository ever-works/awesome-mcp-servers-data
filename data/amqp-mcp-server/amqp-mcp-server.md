# AMQP MCP Server

**Category:** Messaging MCP Servers  
**Brand:** amqp  
**Source:** https://mcp.pipedream.com/app/amqp

## Description
AMQP MCP Server is an MCP server that exposes AMQP (Advanced Message Queuing Protocol) messaging capabilities to MCP-based tools. It allows AI tools to publish and consume messages using the standard AMQP protocol.

- Integrates with AI tools such as Cursor, Claude, and Windsurf
