## Features
- **AI-powered chatbots** for automated customer conversations.

- **Social CRM** features to manage and track customer interactions from messaging channels.
- **Machine learning and natural language processing** capabilities to better understand and respond to user messages.
- **Phone automation** features for automating phone-based interactions.
- **Static MCP server URL** (`https://mcp.pipedream.net/v2`) usable across clients, with authentication handled when adding the server to an app.
- **Multi-client compatibility**: can be added to different chat clients that support MCP.

## Technical Details

- **Integration host:** Pipedream MCP
- **Authentication:** Performed when adding the MCP server to the client application.
