Shadertoy MCP Server exposes Shadertoy via the Model Context Protocol (MCP), allowing tools and applications to search, retrieve, and interact with shaders and related resources.

## Features
- MCP-based access to Shadertoy resources (shaders and related assets).
- Static MCP server endpoint for all compatible clients:
  - `https://mcp.pipedream.net/v2`
- Works with multiple MCP-capable chat or AI clients.
- Authentication handled when adding the server to your application.
- Configuration guidance available via a dedicated configuration page (per-client setup instructions).
- Operated and hosted via Pipedream Connect.

## Getting Started
- Use the static MCP server URL `https://mcp.pipedream.net/v2` when configuring your MCP-compatible client.
- Follow client-specific instructions from the configuration documentation to add and authenticate the server.
