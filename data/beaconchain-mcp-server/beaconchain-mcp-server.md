# Beaconchain MCP Server

Open-source Ethereum beacon chain explorer MCP server for accessing blockchain data from Beaconcha.in within MCP-compatible clients.

## Overview
- **Name:** Beaconchain MCP Server  
- **Category:** Blockchain / Crypto MCP Servers  
- **Brand:** beaconchain  
- **Description:** MCP server that exposes Beaconcha.in (an open-source Ethereum beacon chain explorer) for blockchain data queries in MCP-enabled applications.  
- **Server URL:** `https://mcp.pipedream.net/v2`  
- **Hosting / Platform:** Powered by Pipedream Connect

## Features
- Provides access to Beaconcha.in, an open-source Ethereum beacon chain explorer.  
- Enables Ethereum beacon chain data queries through the MCP protocol.  
- Static MCP server URL (`https://mcp.pipedream.net/v2`) usable across compatible clients.  
- Authentication handled when adding the server to the client/application.  
- Can be integrated with various chat or MCP-enabled clients.  
- Configuration instructions available via a dedicated configuration page on Pipedream.

## Integration
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`  
- Add the server to your MCP-compatible app or chat client using the static URL, then authenticate within the client.
