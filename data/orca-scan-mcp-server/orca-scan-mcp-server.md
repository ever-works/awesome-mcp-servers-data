# Orca Scan MCP Server

Build a barcode system without code.

## Overview
The Orca Scan MCP Server is a Model Context Protocol (MCP) server that connects MCP-compatible agents to Orca Scan. It enables programmatic interaction with barcode data and Orca Scan’s no-code barcode systems.

- **Type:** MCP server / integration
- **Category:** Business & Commerce – MCP Servers
- **Primary Use Cases:** Barcode scanning workflows, inventory and asset tracking, no-code barcode system integration via MCP-compatible agents
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server endpoint**  
  - Single static MCP server URL that can be used across different MCP clients.
  - Authentication is handled when adding the server to the client application.

- **Integration with Orca Scan**  
  - Connects to Orca Scan to work with barcode data.  
  - Supports building and managing barcode-based workflows via MCP-compatible agents.

- **No-code barcode system support**  
  - Designed for creating and using barcode systems without writing code, orchestrated through MCP clients.

- **Client-agnostic setup**  
  - Works with any MCP-capable chat or agent client that supports adding custom MCP servers.

- **Configuration resources**  
  - Central configuration guidance available via a dedicated configuration page (for step-by-step setup in different clients).

## Technical Details
- **Server URL:** `https://mcp.pipedream.net/v2`  
- **Protocol:** Model Context Protocol (MCP)
