# Graphy MCP Server

An MCP (Model Context Protocol) server integration for Graphy, a collaborative data visualization platform focused on fast, visually appealing dashboards and charts.

## Overview
- **Type:** MCP server integration
- **Category:** Data visualization
- **Use Cases:** Building and interacting with dashboards and charts via MCP-compatible chat or agent clients; collaborative data visualization workflows.
- **Provider:** Pipedream / Graphy

## Features
- **Standard MCP Endpoint**
  - Single static MCP server URL usable across compatible clients:
    - `https://mcp.pipedream.net/v2`
  - Designed to be added to any supported MCP client or chat application.

- **Authentication at Client Setup**
  - Authentication is performed when adding the server to your application or client (credentials and flow depend on the client configuration).

- **Client-Agnostic Integration**
  - Works with multiple MCP-enabled chat or agent clients.
  - Central configuration via a shared server URL.

- **Graphy Integration**
  - Connects MCP clients to Graphy’s data visualization environment.
  - Intended for working with dashboards, charts, and collaborative visualizations.

- **Configuration Documentation**
  - Detailed setup and configuration instructions available via the linked Configuration page (for supported MCP clients and environments).

## Getting Started
1. Use the MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add the server in your MCP-compatible client’s settings.
3. Authenticate when prompted during the add/connect flow.
4. Follow the Configuration documentation for client-specific setup.
