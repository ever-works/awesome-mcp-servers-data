# Web3 Research MCP

**Category:** blockchain-crypto-mcp-servers  
**Tags:** crypto, web3, analysis  
**Source:** https://github.com/aaronjmars/web3-research-mcp

## Description
Web3 Research MCP is a fully local Model Context Protocol (MCP) server designed for deep research on cryptocurrencies and Web3 projects. It enables AI agents to perform in-depth blockchain and token analysis without relying on external services.

## Features
- Fully local MCP server (no external dependencies required at runtime)
- Designed for deep research on crypto and Web3 projects
- Supports in-depth blockchain and token analysis workflows
- Integrates as an MCP tool/server for AI agents
- Distributed as an open repository with Dockerfile and npm configuration for installation and deployment
