# Wonderchat MCP Server

Instantly build and use AI chatbots grounded in your own knowledge base inside MCP-compatible tools.

## Overview
- **Type:** MCP (Model Context Protocol) server
- **Brand:** Wonderchat
- **Category:** AI Integration – MCP Servers
- **Primary use case:** Create and interact with AI chatbots that use your own knowledge base, integrated into MCP workflows and clients.
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server**
  - Provides a static MCP server endpoint that can be used across different MCP clients.
  - Designed to plug into tools that support the Model Context Protocol.

- **Knowledge‑base grounded chatbots**
  - Enables creation of AI chatbots that reference and operate on your own knowledge base (as described in the item metadata).
  - Supports using these chatbots within broader MCP workflows and automations.

- **Static URL for all clients**
  - Single URL (`https://mcp.pipedream.net/v2`) works for every supported client.
  - Simplifies setup across multiple tools and environments.

- **Client-agnostic integration**
  - Can be added to different chat clients; documentation is organized by client type.
  - A dedicated configuration page (referenced as “Configuration page”) provides setup details per client.

- **Authentication at connection time**
  - Authentication happens when you add the MCP server to your application / client (no per-client URL needed).

- **Pipedream Connect integration**
  - Hosted and delivered via Pipedream Connect infrastructure.
  - Inherits Pipedream’s platform policies (Terms, Privacy, Cookies) for usage.

## Setup & Usage
- Use the static MCP endpoint: `https://mcp.pipedream.net/v2`.
- Add the server in your MCP-compatible chat client and authenticate when prompted.
- Optionally follow the client-specific instructions and full configuration guidance on the linked configuration page (not included here).
