# Relink - URL Shortener MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** relink  
**Website:** https://mcp.pipedream.com/app/relink_url_shortener

## Description
Relink - URL Shortener MCP Server is a Pipedream-hosted Model Context Protocol (MCP) server that lets MCP-compatible clients programmatically create and manage shortened URLs using the Relink URL Shortener service.

## Features
- MCP-compatible server for URL shortening
- Programmatic creation of shortened URLs
- Programmatic management of existing shortened URLs (e.g., updating or organizing links, where supported by Relink)
- Single static MCP server endpoint usable across multiple MCP clients
- Hosted and operated via Pipedream Connect infrastructure

## Technical Details
- **MCP Server Endpoint:** `https://mcp.pipedream.net/v2`
- **Authentication:** Performed when adding/configuring the server in the client application
- **Client Compatibility:** Designed for MCP-enabled chat or agent clients that can consume MCP servers

## Getting Started
- Use the static MCP server URL `https://mcp.pipedream.net/v2` when adding this server to your MCP-compatible application.
- Configure authentication and connection details in your chat or MCP client according to its setup process.
- Refer to the Pipedream MCP **Configuration** documentation for full setup instructions.
