# FormCan MCP Server

**Category:** Business & Commerce · MCP Servers  
**Brand:** FormCan  
**Slug:** `formcan-mcp-server`

FormCan MCP Server is an MCP Server integration that lets MCP‑aware applications interact with FormCan’s pay‑as‑you‑go, reimagined form‑building platform.

---

## Features

- MCP Server integration for FormCan, accessible from MCP‑aware applications.
- Static MCP server URL for all clients:  
  `https://mcp.pipedream.net/v2`
- Authentication occurs when adding the server to your application.
- Works with compatible chat clients and other MCP‑enabled tools.
- Configuration details available via a dedicated configuration page (client‑specific setup instructions).
