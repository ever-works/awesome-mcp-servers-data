# Felt MCP Server

**Category:** Data Visualization  
**Website:** https://mcp.pipedream.com/app/felt  
**Brand:** Felt

## Description
Felt MCP Server connects AI tools to Felt’s cloud-native GIS platform, enabling mapping and geospatial workflows through the Model Context Protocol (MCP).

## Features
- Connects a Felt account to AI tools via Pipedream’s MCP server
- Supports integration with AI coding and assistant tools such as Cursor, Claude, and Windsurf
- Provides access to Felt’s cloud-native GIS and mapping capabilities from within MCP-compatible tools
- Enables geospatial and mapping workflows (e.g., working with maps, locations, and spatial data) through AI-driven interfaces

## Use Cases
- Integrating geospatial context into AI assistants and coding environments
- Building or editing maps from within AI tools using Felt’s GIS platform
- Incorporating location intelligence into AI-powered workflows and automations
