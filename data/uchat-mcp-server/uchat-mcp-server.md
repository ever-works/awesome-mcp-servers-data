# Uchat MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Uchat  
**Slug:** `uchat-mcp-server`

## Description
Uchat MCP Server is an MCP integration that connects a Uchat account into MCP-based tools, enabling multi-channel chatbot and automation workflows across platforms such as Messenger, WhatsApp, Google Business Messenger, SMS, Voice, Slack, Telegram, WeChat, and others. It is available via Pipedream’s MCP server to work with AI tools like Cursor, Claude, and Windsurf.

## Features
- Connects a Uchat account to MCP-compatible environments
- Provides multi-channel chatbot integration for:
  - Facebook Messenger
  - WhatsApp
  - Google Business Messenger
  - SMS
  - Voice
  - Slack
  - Telegram
  - WeChat
  - Other supported messaging channels
- Enables automation workflows across multiple messaging platforms
- Integrates with AI tools via Pipedream’s MCP server, including:
  - Cursor
  - Claude
  - Windsurf
- Operates within MCP ecosystems for standardized tool access and orchestration
