# Everhour MCP Server

**Category:** Project Management MCP Servers  
**Brand:** Everhour  
**Source:** https://mcp.pipedream.com/app/everhour

Everhour MCP Server exposes Everhour’s time tracking, budgeting, and reporting capabilities to MCP-based workflows and tools via a static MCP endpoint.

---

## Features

### MCP Server Endpoint
- Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`
- Authentication performed when adding the server to the client / application

### Time Tracking
- Track time for individuals and teams
- View who is actively tracking time
- Monitor workload distribution across team members

### Workload & Resource Management
- Identify overworked team members
- See which team members have additional capacity
- Forward-looking resource planning (forward resource planning)

### Budgeting & Alerts
- Track project budgets in real time
- Configure threshold alerts for budgets / time limits

### Reporting & Invoicing
- Build custom / arbitrary reports on time and budget data
- Generate professional-looking invoices from tracked time and project data

### Integration with MCP Workflows
- Exposes Everhour capabilities (time tracking, budgeting, reporting) to any MCP-compatible tools or chat clients
- Can be added to various chat clients via MCP configuration

---

## Setup

1. Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add the server to your MCP-compatible app or chat client.
3. Authenticate when prompted by your client.
4. (Optional) Refer to the platform’s Configuration page for detailed setup steps.
