# SurveyCTO MCP Server

**Description**  
MCP server integration for SurveyCTO, enabling reliable, offline-capable mobile data collection capabilities inside MCP-based research and field-data workflows.

**Source**  
<https://mcp.pipedream.com/app/surveycto>

**Category**  
Business & Commerce · MCP Servers

**Tags**  
- data-collection  
- surveys  
- offline

---

## Features
- Provides an MCP server endpoint for integrating SurveyCTO with MCP-compatible chat clients and tools.
- Supports offline-capable mobile data collection workflows via SurveyCTO within MCP-based environments.
- Uses a single static MCP server URL for all clients:  
  `https://mcp.pipedream.net/v2`
- Authentication occurs when adding/configuring the MCP server within your application or chat client.
- Compatible with multiple chat clients (connect by selecting your client and following its specific setup instructions).
- Central configuration managed via a dedicated configuration page (for detailed setup and options).

---

## Setup
1. Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add this server URL in your MCP-compatible app or chat client.
3. Authenticate when prompted during server addition.
4. Optionally, review the full configuration documentation via the provider’s configuration page.
