# Blockchain Exchange MCP Server

MCP integration for Blockchain.com Exchange, exposing crypto APIs for wallets, tx querying, NFTs via standardized protocol.

## Features
- Transaction history and details
- Wallet balances and transfers
- Order placement/monitoring
- NFT metadata retrieval

## Use Cases
DeFi agents for trading automation.

Superior to Web3.js for AI due to tool-calling interface.

## Pricing
Free MCP access; exchange fees apply.
