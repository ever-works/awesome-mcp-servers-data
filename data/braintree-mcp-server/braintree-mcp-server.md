# Braintree MCP Server

**Category:** Finance & Market Data MCP Servers  
**Tags:** payments, e-commerce, transactions

## Overview
Braintree MCP Server is an MCP (Model Context Protocol) server that connects your Braintree account to AI tools such as Cursor, Claude, and Windsurf, enabling AI agents to access global commerce and payment processing capabilities.

## Features
- Connects a Braintree account to MCP-compatible AI tools (e.g., Cursor, Claude, Windsurf)
- Provides AI agents access to Braintree’s payment processing capabilities
- Supports use cases around payments, e-commerce, and transactions via MCP
- Integrates Braintree’s global commerce tools into AI-driven workflows
