# EngageBay MCP Server

## Overview
The EngageBay MCP Server exposes EngageBay’s all‑in‑one CRM, marketing, and sales capabilities to MCP‑enabled applications via Pipedream. It provides a static MCP server endpoint that can be added to compatible chat clients or tools, enabling them to interact programmatically with EngageBay.

- **Type:** MCP server / integration
- **Category:** Business & Commerce – MCP Servers
- **Integrates with:** EngageBay CRM (marketing, sales, support)
- **Host platform:** Pipedream

## Features
- **Static MCP Endpoint**  
  - Single, reusable MCP server URL: `https://mcp.pipedream.net/v2`  
  - Works across all EngageBay clients configured via Pipedream.

- **Account Connection & Configuration**  
  - Connect an EngageBay account through Pipedream.  
  - Select the associated client within the Pipedream UI to finalize configuration.  
  - Authentication handled when adding the MCP server to your application.

- **Chat Client Integration**  
  - Instructions available for adding the MCP server to supported chat clients.  
  - Can be used by MCP‑enabled agents to access EngageBay CRM, marketing, and sales functions.

- **Tooling & Actions (via Pipedream)**  
  - Exposes EngageBay actions and tools as MCP tools (listed within Pipedream as "Available tools").  
  - Tools are dynamically loaded from Pipedream, enabling operations against EngageBay (specific tools/actions are not enumerated in the provided content).

- **Centralized Configuration Docs**  
  - Link to a full configuration page on Pipedream for more detailed setup instructions.
