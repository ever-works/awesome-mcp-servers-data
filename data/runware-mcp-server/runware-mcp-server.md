# Runware MCP Server

**Category:** Media Processing MCP Servers  
**Brand:** Runware  
**Slug:** `runware-mcp-server`

## Overview
Runware MCP Server is an MCP-compatible server integration that exposes Runware’s low-cost, ultra-fast Stable Diffusion image generation capabilities through any MCP-enabled client or application, via a single static MCP server URL hosted on Pipedream.

## Features
- **MCP-compatible Stable Diffusion API** – Provides Stable Diffusion image generation as tools accessible through the Model Context Protocol (MCP).
- **Static MCP server endpoint** – Uses a single, static MCP server URL (`https://mcp.pipedream.net/v2`) for all compatible clients.
- **Client-agnostic integration** – Can be added to multiple MCP-enabled chat clients or applications using the same server URL.
- **Authentication via client configuration** – Authentication and account connection are handled when adding the MCP server to your app/client.
- **Hosted by Pipedream** – Operates as a managed MCP server instance on Pipedream’s infrastructure.
- **Configuration guidance** – Provides instructions and a configuration page for connecting your account and adding the server to supported clients.

## Technology
- **Model type:** Stable Diffusion (image generation)
- **Use cases:** Image generation, generative AI media workflows via MCP tools.

## Integration Details
- **MCP server URL:** `https://mcp.pipedream.net/v2`
- **Access method:** Add the server URL to your MCP-compatible chat client or application, then authenticate.
