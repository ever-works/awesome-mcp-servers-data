# SMSlink nc MCP Server

MCP Server for SMSlink (Nouvelle-Calédonie), enabling SMS sending in New Caledonia through MCP-based integrations.

## Overview
The **SMSlink nc MCP Server** provides an MCP-compatible interface for sending SMS messages via SMSlink in New Caledonia. It is powered by Pipedream Connect and can be integrated with any MCP-capable chat client or application.

- **Category:** Messaging MCP Servers  
- **Use case:** Programmatic SMS sending in New Caledonia via MCP  
- **Provider / Brand:** SMSlink  
- **Integration host:** Pipedream Connect  

## MCP Server URL
Use this static MCP server URL for all clients (authentication is handled when you add the server to your app):

```text
https://mcp.pipedream.net/v2
```

## Features
- **MCP-based SMS integration** for SMSlink in New Caledonia (Nouvelle-Calédonie)
- **Static MCP endpoint** (`https://mcp.pipedream.net/v2`) usable across different MCP-capable clients
- **Client-agnostic setup**: same URL works for every compatible chat client; authentication occurs during server addition
- **Pipedream Connect integration** for hosting and managing the MCP server
- **Configuration documentation** available via a dedicated configuration page on Pipedream
