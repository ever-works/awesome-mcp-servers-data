# LinkedIn Ads MCP Server

## Overview
The LinkedIn Ads MCP Server exposes LinkedIn self-service advertising capabilities to MCP-enabled tools, enabling you to manage and automate LinkedIn ad campaigns programmatically.

- **Category:** Business & Commerce – MCP Servers
- **Vendor / Brand:** LinkedIn (via Pipedream Connect)
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server**
  - Static MCP server URL that can be added to any compatible client.
  - Authentication handled at the time you add/configure the server in your application.

- **LinkedIn Ads integration**
  - Access to LinkedIn’s self-service advertising platform via MCP.
  - Designed to work with LinkedIn Ads accounts and campaign management tools.

- **Marketing automation support**
  - Can be used by MCP-enabled tools and chat clients to automate advertising workflows.
  - Suitable for integrating LinkedIn campaign operations into broader marketing automation stacks.

- **Multi-client support**
  - Single static URL works across different MCP-compatible chat clients and applications.
