# SMS Alert MCP Server

**Category:** Messaging MCP Servers  
**Brand:** SMS Alert  
**Slug:** `sms-alert-mcp-server`

SMS Alert MCP Server exposes SMS Alert’s enterprise-grade transactional and promotional SMS capabilities to MCP-compatible clients via a static MCP server endpoint.

---

## Features

- **Enterprise SMS platform integration**  
  - Connects MCP-based clients to SMS Alert, an enterprise-grade platform for transactional and promotional SMS messaging.

- **Static MCP server URL**  
  - Single endpoint for all compatible clients:  
    `https://mcp.pipedream.net/v2`
  - URL is the same across clients; authentication is handled when adding the server to each application.

- **Client-agnostic usage**  
  - Designed to work with multiple MCP-based chat or agent clients.  
  - Documentation available per client type (via “Select your chat client to learn how to get started”).

- **Configuration documentation**  
  - Full configuration details available through a dedicated configuration page (external link from the listing).

- **Transactional & promotional SMS focus**  
  - Suitable for sending one-time, event-driven messages (transactional).  
  - Also supports campaign-style or marketing/promotional SMS use cases.

---

## Integration Details

- **MCP endpoint:** `https://mcp.pipedream.net/v2`
- **Authentication:** Performed when you add the MCP server to your application (client-specific flow).
