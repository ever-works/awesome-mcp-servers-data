# Tettra MCP Server

## Overview
Tettra MCP Server connects MCP-based agents to Tettra’s team knowledge base, enabling them to organize, retrieve, and share documentation programmatically.

- **Category:** Content Management MCP Servers  
- **Use cases:** Knowledge base integration, documentation retrieval, team collaboration automation
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible endpoint**: Exposes Tettra’s knowledge base via a static MCP server URL that works across MCP-compatible clients.
- **Programmatic knowledge access**: Allows agents and applications to retrieve, organize, and share Tettra documentation programmatically.
- **Team knowledge integration**: Connects directly to Tettra’s team knowledge base, making internal documentation accessible from chat clients and other MCP-based tools.
- **Multi-client support**: Designed to be added to various chat clients and MCP-aware applications using the same server URL.
- **Authentication support**: Authentication is performed when adding the server to an application, ensuring controlled access to the Tettra knowledge base.
- **Configuration guidance**: Can be configured through a dedicated configuration flow/documentation (via the referenced configuration page in the source site).

## Integration Details
- **Server URL:** `https://mcp.pipedream.net/v2` (static across clients)
- **Setup:** Add the server URL to your MCP-compatible chat client or application, then authenticate when prompted.
