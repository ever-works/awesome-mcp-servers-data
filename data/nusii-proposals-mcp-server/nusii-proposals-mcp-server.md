# Nusii Proposals MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Nusii  
**Slug:** `nusii-proposals-mcp-server`

## Overview
The Nusii Proposals MCP Server connects your Nusii proposal software account to compatible AI tools via Pipedream’s MCP server, enabling AI agents to create and manage client proposals.

Source: <https://mcp.pipedream.com/app/nusii_proposals>

## Features
- Connects a Nusii Proposals account to AI tools through Pipedream’s MCP server
- Integrates with AI assistants/editors such as Cursor, Claude, and Windsurf
- Provides MCP tools for creating client proposals
- Enables AI-driven management of existing client proposals
- Designed for use within MCP-compatible development and workflow environments

## Use Cases
- Automating creation of client proposals from within AI coding or writing environments
- Letting AI agents update or manage proposals stored in a Nusii Proposals account
- Integrating proposal workflows into broader automated business or sales processes
