# Mezmo MCP Server

**Category:** Cloud & DevOps MCP Servers  
**Brand:** Mezmo  
**Source:** https://mcp.pipedream.com/app/mezmo

## Overview
The Mezmo MCP Server is an MCP-compatible endpoint for Mezmo’s observability pipeline platform. It allows MCP-based workflows and chat clients to control, enrich, and correlate observability data across domains, supporting more actionable monitoring and operations.

- **MCP server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-Compatible Server**: Exposes Mezmo’s observability capabilities via the Model Context Protocol (MCP), enabling integration with MCP-aware clients and tools.
- **Single Static Endpoint**: Uses a static MCP server URL (`https://mcp.pipedream.net/v2`) that works across supported clients.
- **Client-Agnostic Setup**: Can be added to various chat clients and MCP-compatible applications, with configuration handled per client.
- **Observability Pipeline Integration**: Connects to Mezmo’s observability pipeline to:
  - Control observability data flows
  - Enrich telemetry and logs
  - Correlate data across domains
- **Actionability Focus**: Designed to help turn observability data into actionable insights by centralizing control and enrichment within MCP workflows.
- **Hosted by Pipedream**: Delivered via Pipedream Connect infrastructure, which provides the hosted MCP server endpoint.

## Usage
- Configure the MCP server in your MCP-compatible client using the static URL: `https://mcp.pipedream.net/v2`.
- Authenticate within your client when prompted during server setup.
- Refer to the provider’s configuration documentation ("Configuration" page referenced on the source site) for client-specific setup details.
