# Twist MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Doist

An MCP (Model Context Protocol) server that integrates with Twist, allowing MCP-compatible clients to access and manage Twist’s team communication spaces.

---

## Features

- **Twist integration for MCP clients**  
  Connects Twist with any MCP-compatible client so you can work with Twist data from your AI/chat tools.

- **Access to team communication structures**  
  Designed to expose Twist’s organized communication model, including:
  - Channels
  - Threads
  - Messages

- **Central MCP endpoint**  
  Uses a static MCP server URL that can be shared across different clients:  
  `https://mcp.pipedream.net/v2`

- **Multi-client support**  
  Can be added to multiple chat or AI clients; configuration is done per client while using the same server URL.

- **Account-based authentication**  
  Requires connecting a Twist account; authentication occurs when adding the server to a client application.

---

## Usage

1. Connect your Twist account in the Twist MCP Server configuration page.
2. Copy the static MCP server URL: `https://mcp.pipedream.net/v2`.
3. Add the server to your chosen MCP-compatible client following that client’s setup instructions.
