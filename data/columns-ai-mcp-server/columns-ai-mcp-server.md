# Columns Ai MCP Server

**Category:** Data Visualization  
**Brand:** Columns Ai  
**Slug:** `columns-ai-mcp-server`

MCP server integration that connects your Columns Ai account to AI-powered developer and productivity tools.

---

## Description
The Columns Ai MCP Server is an integration layer (via Pipedream) that lets you use your Columns Ai data stories and visual narratives from within AI tools such as Cursor, Claude, and Windsurf.

---

## Features
- **MCP Server Integration**: Implements a Model Context Protocol (MCP) server for Columns Ai.  
- **Pipedream-Based Connector**: Runs on Pipedream, enabling low-friction setup and orchestration.  
- **Columns Ai Account Linking**: Connects directly to your Columns Ai account to access stored data stories and visual narratives.  
- **AI Tool Compatibility**: Designed to work with AI tools including:
  - Cursor  
  - Claude  
  - Windsurf  
- **Data Story Access**: Exposes Columns Ai content to AI tools so they can reference, analyze, or build on your data stories and visual narratives.  
- **Analytics & Visualization Context**: Brings data visualization and analytics context from Columns Ai into AI workflows.

---

## Use Cases
- Access and manipulate Columns Ai data stories from within AI coding tools (e.g., Cursor).  
- Allow AI assistants (e.g., Claude) to reference and reason about your Columns Ai dashboards and visual narratives.  
- Integrate Columns Ai visualizations into AI-powered workflows for analytics or reporting.
