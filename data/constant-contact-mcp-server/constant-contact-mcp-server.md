# Constant Contact MCP Server

## Overview
The Constant Contact MCP Server is an MCP server integration (powered by Pipedream Connect) that allows MCP-compatible clients to interact programmatically with Constant Contact’s email and digital marketing platform.

## Key Information
- **Type:** MCP server integration
- **Category:** Business & Commerce – MCP Servers
- **Provider/Brand:** Constant Contact (via Pipedream)
- **Base MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Use Case:** Enable chat clients or other MCP-enabled applications to trigger and manage Constant Contact email and digital marketing capabilities through a unified MCP interface.

## Features
Based on the provided content and item description, the Constant Contact MCP Server enables:

- **Programmatic access to Constant Contact**
  - Connect MCP clients to Constant Contact’s email and digital marketing tools.
  - Use a single MCP server URL across different clients.

- **Static MCP server endpoint**
  - One static URL (`https://mcp.pipedream.net/v2`) that works for all supported MCP clients.
  - Authentication handled when adding the server to the client/application.

- **MCP client integration**
  - Can be added to compatible chat clients and other applications that support MCP.
  - Central configuration available via the Pipedream Configuration page (referenced in the content).

- **Email and digital marketing tooling (via Constant Contact)**
  - Access to Constant Contact’s email marketing tools.
  - Access to broader digital marketing capabilities (e.g., campaigns, automation) through the MCP interface, as supported by the underlying Constant Contact account and API.

> Note: The source content does not list specific individual API operations (e.g., "create campaign", "list contacts"), only that Constant Contact’s email and digital marketing tools can be leveraged programmatically.

## Setup
- **MCP Server URL:** Use `https://mcp.pipedream.net/v2` as the server endpoint in your MCP-compatible client.
- **Authentication:** Performed when adding the MCP server to your application/client (details are handled in the client’s configuration flow and the Pipedream configuration page).
