# SheetDB MCP Server

**Category:** Data Access & Integration – MCP Servers  
**Brand:** SheetDB  
**Source:** https://mcp.pipedream.com/app/sheetdb

## Overview
SheetDB MCP Server exposes Google Sheets as a JSON API so MCP-based tools and chat clients can read and write spreadsheet data as if they were interacting with a web service. It connects Google Sheets to other applications such as CRMs, websites, WordPress, and other tools.

## Features
- **Google Sheets → JSON API**
  - Turns a Google Spreadsheet into a JSON API endpoint.
  - Enables programmatic access to sheet data for MCP-compatible applications.

- **MCP Server Endpoint**
  - Static MCP server URL: `https://mcp.pipedream.net/v2`.
  - Authentication handled when adding the server to your MCP-enabled application or chat client.

- **Integration with MCP-based Tools and Chat Clients**
  - Can be added as a server to supported chat clients.
  - Works as a data access layer to read and modify spreadsheet content via MCP tools.

- **Available Tools / Actions (5 actions)**
  1. **Update Rows**  
     - Updates the content for specified row(s) in a Google Sheet via SheetDB.
  2. **Search Content**  
     - Searches for content in a Google Sheet using the SheetDB API.
  3. **Get Column Names**  
     - Retrieves column names from a Google Sheet, useful for mapping and dynamic operations.
  4. **Delete Rows**  
     - Deletes specified row(s) by matching a column name and value.
  5. **Create Rows**  
     - Creates new rows in a Google Sheet using the SheetDB API.

## Typical Use Cases
- Use a Google Sheet as a lightweight database behind web or internal tools.
- Integrate spreadsheet data with CRMs, websites, or other APIs.
- Enable chat-based agents to search, read, update, and manage spreadsheet data.
