# DaData.ru MCP Server

**Category:** Data Access & Integration MCP Servers  
**Slug:** `dadataru-mcp-server`

An MCP (Model Context Protocol) server that connects DaData.ru’s automated data cleansing services to AI tools, enabling validation and normalization of Russian postal addresses, phone numbers, and customer names.

![DaData.ru Screenshot](https://dadata.ru/static/img/screenshots/address-cleaner.png)

---

## Description
The DaData.ru MCP Server integrates DaData.ru’s data cleansing APIs with AI environments via Pipedream’s MCP server. It allows AI tools such as Cursor, Claude, and Windsurf to access DaData.ru for automated cleaning, standardization, and verification of Russian customer data.

---

## Features
- **DaData.ru Integration**  
  - Uses Pipedream’s MCP server to connect to a DaData.ru account.  
  - Exposes DaData.ru data cleansing capabilities as MCP tools.

- **Automated Data Cleansing**  
  - Cleans and normalizes Russian postal addresses.  
  - Validates and formats phone numbers.  
  - Standardizes and corrects customer names.

- **AI Tool Connectivity**  
  - Can be used from AI-focused tools such as:  
    - Cursor  
    - Claude  
    - Windsurf  
  - Designed to support AI workflows that need reliable structured contact and address data.

- **Use Cases**  
  - Enriching and correcting customer records within AI-assisted workflows.  
  - Verifying user input (addresses, phones, names) in AI-driven assistants and internal tools.  
  - Maintaining consistent, standardized Russian contact data across systems.
