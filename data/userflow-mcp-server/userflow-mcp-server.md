# Userflow MCP Server

MCP Server that connects to Userflow so agents can manage user onboarding flows, product tours, and checklists via MCP tools.

- **Website / Source**: https://mcp.pipedream.com/app/userflow
- **Category**: Business & Commerce – MCP Servers
- **Brand**: Userflow
- **Slug**: `userflow-mcp-server`
- **Static MCP Server URL**: `https://mcp.pipedream.net/v2`

## Overview
The Userflow MCP Server is an MCP-compatible server (hosted by Pipedream) that integrates with Userflow, enabling agents and applications to manage product onboarding experiences—such as tours and checklists—directly via MCP tools. It is intended for use in web and app development contexts and allows configuration by connecting a Userflow account and selecting a client.

## Features
- **MCP Integration**
  - Provides a static MCP server endpoint (`https://mcp.pipedream.net/v2`) usable across clients.
  - Authenticates at the time of adding the server to an MCP-compatible application.
- **Userflow Connectivity**
  - Connects to a Userflow account to manage user onboarding assets.
  - Supports managing product tours, checklists, and other onboarding flows via MCP tools.
- **Client-Agnostic Setup**
  - Same server URL works for every supported client.
  - Workflow: connect Userflow account → select client → start using tools.
- **No-Code Onboarding Management**
  - Underlying Userflow platform is designed to build product tours, checklists, and onboarding flows without coding.
- **Web & App Development Focus**
  - Positioned for integration into web and app development environments.
- **Configuration Resources**
  - Option to select a chat client for step-by-step connection instructions.
  - Additional details available via a central configuration page (`/configuration`).
