## Overview
Cutt.ly MCP Server integrates the Cutt.ly URL shortening and link management platform into the MCP ecosystem, enabling automated link workflows within compatible chat clients and applications.

## Features
- URL shortening for long links
- Link management capabilities
- Custom URL shortener support
- Branded short links
- Link-in-bio creation
- QR code generation for links
- Works via a static MCP server endpoint (`https://mcp.pipedream.net/v2`) usable across clients
- Designed for integration into MCP-enabled applications and chat clients

## Technical Details
- MCP server base URL: `https://mcp.pipedream.net/v2`
- Added to applications via MCP-compatible client configuration
