# Contentstack MCP Server

**Category:** Content Management MCP Servers  
**Brand:** Contentstack  
**Source:** https://mcp.pipedream.com/app/contentstack

## Description
Contentstack MCP Server is an MCP-compatible server for Contentstack’s headless CMS. It allows MCP clients (such as AI chat or automation tools) to access, manage, and automate content operations using Contentstack’s APIs.

## MCP Server URL
- Static MCP server URL (for all clients):
  - `https://mcp.pipedream.net/v2`
- Authentication is performed when adding the server to your application.

## Features
- **MCP-compatible endpoint** for integrating Contentstack with any MCP client.
- **Headless CMS integration** with Contentstack’s enterprise CMS via MCP.
- **Content operations via API** (access, management, and automation of content through Contentstack APIs, as supported by the MCP server).
- **Static server URL** usable across different clients, simplifying configuration.
- **Client-agnostic setup**: Works with multiple chat or MCP-capable applications.

## Getting Started
- Add the MCP server to your app using the static URL.
- Follow client-specific instructions (via your chat client’s MCP configuration) or refer to the full configuration documentation on the host platform.
