## Features
- **Email parsing**: Automatically extract text and structured data from incoming emails.
- **PDF and document parsing**: Extract data from PDFs and other supported document formats.
- **Structured data output**: Convert unstructured content into structured data suitable for further automation and workflows.
- **MCP integration**: Exposes Parseur capabilities as an MCP server that can be added to any compatible chat or agent client.
- **Static MCP endpoint**: Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) that works across clients.
- **Per-client authentication**: Authentication is handled when you add the server to each application/client.
- **Pipedream Connect hosting**: MCP server is hosted and provided via Pipedream Connect infrastructure.
