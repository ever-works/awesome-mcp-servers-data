# Adobe Document Generation API MCP Server

MCP Server that connects to Adobe’s Document Generation API, enabling automated creation of business documents such as invoices, contracts, proposals, and work orders from MCP-compatible clients.

- **Category:** Document Management MCP Servers  
- **Brand:** Adobe  
- **Source:** https://mcp.pipedream.com/app/adobe_document_generation_api  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features

- **MCP-compatible server**
  - Exposes Adobe Document Generation API capabilities through a standard MCP server endpoint.
  - Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) that works across compatible clients.

- **Automated document creation**
  - Generate business-critical documents programmatically, including:
    - Invoices
    - Contracts
    - Sales proposals
    - Work orders
    - Other similar business documents (PDF and related formats, depending on Adobe’s API capabilities).

- **Client integration**
  - Designed to be added directly into MCP-compatible chat and AI clients.
  - Authentication is performed when adding/configuring the server within your application.

- **Pipedream-hosted**
  - MCP server is provided and hosted via Pipedream Connect.
  - Centralized configuration reference available through the Pipedream Configuration page.
