# Blizzard Battle.net MCP Server

**Brand:** Blizzard Entertainment  
**Category:** Game Server MCP Implementations  
**Slug:** `blizzard-battlenet-mcp-server`

An MCP server for interacting with Blizzard Battle.net services, enabling programmatic access to Blizzard Entertainment’s gaming platform data and features.

---

## Connection Details
- **MCP Server URL (static for all clients):** `https://mcp.pipedream.net/v2`
- Authentication is performed when adding the server to your application.

---

## Features
- Provides an MCP-compatible server interface to Blizzard Battle.net services.
- Enables programmatic access to Blizzard Entertainment platform data and features (e.g., game-related API interactions via Battle.net).
- Works with multiple chat or MCP-enabled clients through a single static server URL.
- Managed and hosted via Pipedream Connect infrastructure.

---

## Usage
- Add the MCP server URL to your MCP-compatible client or application.
- Authenticate within your client during setup.
- Use according to your client’s configuration options (e.g., via a configuration or settings page).
