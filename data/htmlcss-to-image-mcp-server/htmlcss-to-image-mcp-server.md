## Description
HTML/CSS to Image MCP Server is an MCP-compatible wrapper around the HTML/CSS to Image API. It allows AI tools and MCP clients to generate high-quality images from HTML and CSS markup via a standardized MCP server endpoint.

## Features
- Wraps the HTML/CSS to Image API as an MCP server
- Accepts HTML and CSS markup to render images
- Produces high-quality rendered images suitable for UI previews, social cards, and other visual assets
- Exposed via a single static MCP endpoint: `https://mcp.pipedream.net/v2`
- Can be added to any compatible MCP / chat client
- Authentication handled when adding the server to your application
- Hosted and operated via Pipedream Connect infrastructure

## Integration & Usage
- Use the static MCP server URL: `https://mcp.pipedream.net/v2`
- Configure it in your MCP-enabled chat client or application following that client’s MCP configuration instructions
- Refer to the platform’s configuration page for detailed setup steps (e.g., adding the server, setting authentication, and enabling tools within the client)
