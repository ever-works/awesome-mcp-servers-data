# Vend MCP Server

**Category:** Business & Commerce · MCP Servers  
**Tags:** e-commerce, retail, payments

## Overview
Vend MCP Server exposes Vend’s retail point-of-sale and store management functionality through the Model Context Protocol (MCP), enabling MCP-compatible clients (such as chat applications or agents) to interact with Vend as a backend retail POS system.

The server is hosted and provided via Pipedream Connect.

## MCP Server Endpoint
- **Base MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **URL type:** Static (same for all clients)  
- **Authentication:** Performed when adding/configuring the server in your MCP-compatible application

## Features
- Provides MCP-based access to Vend retail point-of-sale capabilities
- Provides MCP-based access to Vend store management functionality
- Compatible with MCP-enabled/chat-based clients and agent frameworks
- Single static server URL for all clients (simplifies configuration)
- Authentication handled at the time of adding/configuring the server in your app

## Integration & Configuration
- Add the static MCP server URL to your MCP-compatible client or application
- Follow your client’s MCP server setup flow to authenticate against the Vend MCP Server
- Additional configuration details are available from the provider’s configuration documentation (Pipedream Connect configuration page)
