# Aircall MCP Server

Call center software integration for customer-centric teams via an MCP server.

- **Name:** Aircall MCP Server  
- **Provider / Brand:** Pipedream  
- **Category:** Messaging MCP Servers  
- **Tags:** call-center, VOIP, customer-support  
- **Source URL:** https://mcp.pipedream.com/app/aircall

## Overview

Aircall MCP Server is an MCP server integration for Aircall that allows MCP-compatible tools and clients to interact with Aircall’s cloud call center features. It provides a unified, static MCP endpoint that can be added to supported chat or agent clients to enable call-center–related workflows.

## Features

- **Static MCP Server Endpoint**  
  - Single static URL for all clients:  
    - `https://mcp.pipedream.net/v2`  
  - Works across different MCP-compatible chat and agent applications.

- **Aircall Integration**  
  - Connects MCP tools to Aircall’s cloud call center platform.  
  - Designed for customer-centric and support-focused teams.

- **Client-Agnostic Configuration**  
  - Can be added to multiple chat clients that support MCP servers.  
  - Configuration guidance available via a dedicated configuration page (per-client setup instructions).

- **Authentication at Client Setup**  
  - Authentication is performed when adding the server to the application (not tied to a per-client URL).

- **Operated by Pipedream**  
  - MCP server hosted and managed by Pipedream Connect infrastructure.

## Usage

- Add `https://mcp.pipedream.net/v2` as an MCP server in your compatible chat or agent client.  
- Authenticate within your client when prompted.  
- Follow client-specific steps described on the configuration page (linked from the Aircall MCP Server app page on Pipedream).
