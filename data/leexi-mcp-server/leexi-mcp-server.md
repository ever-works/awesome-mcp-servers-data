# Leexi MCP Server

AI notetaker MCP server for transcribing, analyzing, and summarizing calls and meetings via Leexi, provided through Pipedream Connect.

- **Website**: https://mcp.pipedream.com/app/leexi
- **Category**: Content Extraction & Summarization MCP Servers
- **Tags**: transcript, summarization, meetings
- **MCP Server URL**: `https://mcp.pipedream.net/v2`

## Description
Leexi MCP Server is an MCP (Model Context Protocol) server integration that connects MCP-based tools and chat clients to Leexi’s AI notetaker capabilities. It enables automated transcription, analysis, and summarization of calls and meetings through a unified MCP endpoint.

## Features
- **Unified MCP Endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same URL works across compatible MCP clients

- **Transcription**  
  - Capture call and meeting audio as text (transcripts) for downstream processing

- **Analysis**  
  - Analyze transcribed calls and meetings using Leexi’s AI notetaker capabilities (e.g., extracting key points or insights, depending on client implementation)

- **Summarization**  
  - Generate summarized versions of calls and meetings for quick review

- **MCP-based Tool Integration**  
  - Designed to be used by MCP-compatible applications and chat clients  
  - Can be added as a server in supported chat clients via MCP configuration

- **Configuration Documentation**  
  - Central configuration guide available via Pipedream’s Configuration page (for steps to add the server to different clients)

## Integration & Usage
- Add the MCP server URL `https://mcp.pipedream.net/v2` in your MCP-enabled chat client or application.
- Authenticate within your client when prompted (authentication flow handled at client side).
- Use the integration to trigger transcription, analysis, and summarization of calls and meetings through Leexi.
