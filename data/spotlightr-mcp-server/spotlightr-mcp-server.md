# Spotlightr MCP Server

Video hosting platform MCP server designed for course creators and e-learning.

- **Brand:** pipedream  
- **Category:** media-processing-mcp-servers  
- **Tags:** video-hosting, courses, media-management  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Source URL:** https://mcp.pipedream.com/app/spotlightr

## Features

- Exposes Spotlightr video hosting functionality through the MCP protocol.
- Provides course-focused video capabilities for e-learning use cases.
- Designed for integration with AI-driven tools and automations via MCP.
- Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) that works across clients.
- Authentication handled when adding the server to compatible MCP-enabled applications.

## Integration & Usage

- Configuration details and client-specific setup are available via the linked configuration documentation on the source site.
