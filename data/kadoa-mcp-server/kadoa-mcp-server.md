# Kadoa MCP Server

**Category:** Workflow Automation MCP Servers  
**Website:** https://mcp.pipedream.com/app/kadoa

## Overview
Kadoa MCP Server is an integration that connects Kadoa—an AI-powered no-code platform for building data workflows—with MCP-compatible chat or agent clients. It focuses on creating and running data extraction and transformation workflows without writing code.

## Features
- **MCP server endpoint**: Static server URL usable across compatible clients:
  - `https://mcp.pipedream.net/v2`
- **No-code data workflows**: Build workflows for data extraction and transformation without coding.
- **AI-powered workflow building**: Uses AI assistance to help create and manage data workflows.
- **Data extraction**: Designed to pull data from sources as part of automated workflows.
- **Data transformation**: Supports ETL-style transformations on extracted data.
- **Client-agnostic setup**: Same MCP URL works for any supported chat or agent client; authentication handled when adding the server.
- **Configuration docs**: Dedicated configuration page for setup details via Pipedream.

## Use Cases
- Automating data extraction from various sources via MCP-compatible clients.
- Running ETL-style workflows for cleaning and transforming data.
- Integrating Kadoa’s no-code workflows into AI chat or agent environments.

## MCP Connection Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Authentication:** Performed when adding the server inside the target application/client.
