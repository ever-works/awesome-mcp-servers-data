# Zoho Cliq MCP Server

An MCP server integration for Zoho Cliq that connects work chat with MCP workflows to automate messaging, collaboration, and task-related operations.

## Overview
- Product type: MCP server (MCP endpoint) for Zoho Cliq
- Category: Messaging MCP servers
- Brand: Zoho
- MCP base URL: `https://mcp.pipedream.net/v2`
- Purpose: Enable automation of Zoho Cliq work chat, messaging, video collaboration, and task operations via MCP-compatible clients.

## Features
- **MCP-compatible server endpoint**
  - Static MCP server URL: `https://mcp.pipedream.net/v2`
  - Same endpoint can be used across different MCP clients.
  - Authentication occurs when adding the server in your client or application.

- **Zoho Cliq integration focus**
  - Designed for Zoho Cliq, a work-focused chat app.
  - Targets use cases around:
    - Team messaging and work chat
    - Video collaboration sessions
    - Task management workflows
    - General productivity and collaboration operations

- **Client-agnostic setup**
  - Can be added to any compatible MCP client.
  - Setup instructions depend on your specific chat or MCP client.
  - Central configuration reference available through the “Configuration” page on the provider site.

- **Workflow automation enablement**
  - Intended to be used in MCP workflows that:
    - Send or manage messages in Zoho Cliq.
    - Trigger or coordinate collaboration activities.
    - Integrate tasks and productivity tools into chat workflows.

## Configuration & Usage
- Use the static MCP URL in your MCP-compatible client.
- Authenticate when prompted by your client during server addition.
- Refer to the provider’s “Configuration” page for client-specific connection steps.
