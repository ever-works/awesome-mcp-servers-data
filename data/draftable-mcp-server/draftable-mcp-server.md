# Draftable MCP Server

**Category:** Document Management MCP Servers  
**Brand:** Draftable  
**Slug:** `draftable-mcp-server`

Draftable MCP Server exposes Draftable’s document comparison capabilities via the Model Context Protocol (MCP), allowing AI tools and developer environments to automatically compare document versions and highlight differences.

## Features

- **MCP-based integration**: Exposes Draftable’s document comparison features through an MCP server interface.  
- **Automated document comparison**: Compares different versions of documents and highlights differences.  
- **AI tool connectivity**: Designed to connect a Draftable account to AI tools such as:  
  - Cursor  
  - Claude  
  - Windsurf  
- **Document diff highlighting**: Supports diff-style outputs to identify changes between document versions.  
- **Document processing focus**: Tailored for comparison and diff workflows within broader document-processing pipelines.
