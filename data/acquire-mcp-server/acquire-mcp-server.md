# Acquire MCP Server

Unified customer service platform MCP server integration for Acquire, provided via Pipedream Connect.

## Overview
- **Type:** MCP server integration
- **Provider / Brand:** Pipedream
- **Category:** Business & Commerce – MCP Servers
- **Slug:** `acquire-mcp-server`
- **Purpose:** Expose Acquire’s unified customer service platform capabilities to MCP-compatible clients so they can be used in automated and conversational support workflows.

## MCP Server Endpoint
- **Base MCP Server URL (static for all clients):**
  ```text
  https://mcp.pipedream.net/v2
  ```
- Authentication is performed when adding the server to your MCP-compatible application.

## Features
- **Unified customer service integration**
  - Connects MCP clients (e.g., chat-based tools) with Acquire’s unified customer service platform.
  - Designed to support automated support workflows and customer engagement scenarios.

- **Static, reusable endpoint**
  - Single MCP server URL works across all compatible clients.
  - Centralized configuration point for integrating Acquire into multiple tools.

- **Client-agnostic setup**
  - Can be added to different chat or MCP-compatible applications.
  - Configuration guidance available via a dedicated configuration page (not app-specific in this summary).

## Integration & Setup
- Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server to your MCP-compatible chat or workflow client.
- Authenticate during the add/connect process within your application.
