# Mailercloud MCP Server

MCP Server for Mailercloud, providing AI and MCP tools with programmatic access to Mailercloud’s all‑in‑one email marketing features.

- **Website / Source**: https://mcp.pipedream.com/app/mailercloud
- **Category**: Business & Commerce – MCP Servers
- **Tags**: email-marketing, campaigns, marketing-automation

## Features

- Programmatic access to Mailercloud’s email marketing platform via MCP
- Integration endpoint exposed as a static MCP server URL
- Works with multiple MCP-compatible chat or AI clients
- Authentication handled when adding the server to your application
- Central configuration and setup documentation available via Pipedream’s configuration page
- Built on Pipedream Connect infrastructure

## Technical Details

- **MCP Server URL**: `https://mcp.pipedream.net/v2`
- **Integration model**: Add as an MCP server in supported chat/AI clients, then authenticate within the client
