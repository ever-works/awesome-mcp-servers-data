# Droxy MCP Server

**Category:** AI Integration – MCP Servers  
**Brand:** Droxy  
**Source:** https://mcp.pipedream.com/app/droxy

An MCP (Model Context Protocol) server for Droxy that exposes Droxy’s AI chatbot-building capabilities programmatically via a standard MCP endpoint.

---

## Features

- **MCP-compatible endpoint**  
  - Provides a static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Designed to work with any MCP-capable client.

- **Programmatic access to Droxy chatbots**  
  - Enables AI chatbot-building functionality to be accessed via the Model Context Protocol from external tools and applications.

- **Client-agnostic integration**  
  - Usable with multiple chat clients that support MCP.  
  - Clients authenticate when adding/connecting to the server.

- **Configuration documentation**  
  - Step-by-step configuration details available via the linked Configuration page (how to add the server to specific chat clients).

- **Pipedream Connect integration**  
  - MCP server is powered by Pipedream Connect infrastructure.

---

## Usage

- Use the static MCP server URL `https://mcp.pipedream.net/v2` in your MCP-compatible chat client or application.  
- Authenticate within the client when prompted after adding the server.  
- Refer to the Configuration page for client-specific setup instructions.
