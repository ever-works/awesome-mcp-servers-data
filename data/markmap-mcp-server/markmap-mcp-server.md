## Overview
Markmap MCP Server is an MCP-compatible server that converts Markdown content into interactive mind maps using Markmap. It supports exporting visualizations to multiple image formats, making it useful for visually exploring and sharing Markdown-based knowledge structures within MCP clients.

## Features
- **Markdown to mind map conversion**: Transforms standard Markdown documents into interactive Markmap-based mind maps.
- **Interactive visualization**: Produces navigable, expandable/collapsible mind maps suitable for exploring document structure.
- **Export to multiple formats**: Supports exporting generated mind maps as PNG, JPG, and SVG files.
- **MCP integration**: Designed as an MCP (Model Context Protocol) server, enabling use as a tool within MCP-compatible clients.
- **NPM package**: Distributed as an npm package (`@jinzcdev/markmap-mcp-server`) for easy installation and integration into JavaScript/TypeScript projects.
- **Docker support**: Includes a Dockerfile for containerized deployment.
- **Documentation**: Repository includes documentation (e.g., `docs/`, multilingual READMEs such as `README.md` and `README_zh-CN.md`).
- **Test suite**: Contains tests (e.g., under `tests/services`) for service behavior and reliability.
