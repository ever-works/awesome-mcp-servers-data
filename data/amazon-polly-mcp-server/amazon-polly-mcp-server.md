# Amazon Polly MCP Server

## Overview
The Amazon Polly MCP Server is an MCP server wrapper around Amazon Polly that allows MCP-compatible clients to generate high-quality, natural-sounding human speech in many languages.

- **Category:** Media Processing MCP Servers
- **Brand:** Amazon Web Services
- **Source URL:** https://mcp.pipedream.com/app/amazon_polly
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- Integrates Amazon Polly as an MCP server, accessible from any MCP-compatible client.
- Generates natural-sounding human speech.
- Supports many languages and voices (via Amazon Polly).
- Single, static MCP server URL that works across clients.
- Authentication handled when adding the server to the client/application.
- Usable with multiple chat or MCP clients via configuration.
