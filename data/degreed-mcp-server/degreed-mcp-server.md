# Degreed MCP Server

**Category:** Documentation & Learning Resources  
**Tags:** learning, LMS, skills

Degreed MCP Server is a Model Context Protocol (MCP) server that exposes Degreed’s learning experience platform to MCP-compatible clients. It enables programmatic access to skills data, learning paths, and career growth workflows from within MCP-based tools.

---

## Features

- **MCP-Compatible Server**  
  - Implements the Model Context Protocol so it can be used by any MCP-capable client.

- **Programmatic Access to Degreed**  
  - Provides an integration point to interact with Degreed’s learning experience platform via MCP.
  - Supports use cases around skills, learning paths, and career development workflows (as described in the item metadata).

- **Static MCP Server URL**  
  - Single, reusable endpoint for all supported clients:  
    - `https://mcp.pipedream.net/v2`
  - Authentication is handled when the server is added/configured in the client application.

- **Client-Agnostic Setup**  
  - Designed to work with multiple MCP chat or agent clients.  
  - Users select their chat client and follow specific instructions for connecting to this server.

- **Configuration Documentation**  
  - Additional technical configuration details available through a dedicated configuration page (linked from the app page).

---

## Integration

- **Platform**: Runs as part of the Pipedream Connect ecosystem.  
- **Usage**: Added as an MCP server endpoint to compatible chat or agent applications to bring Degreed learning and skills context into conversations and workflows.
