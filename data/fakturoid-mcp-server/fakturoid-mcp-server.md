# Fakturoid MCP Server

**Category:** Business & Commerce MCP Servers  
**Tags:** invoicing, billing, accounting

## Overview
Fakturoid MCP Server exposes Fakturoid’s invoicing and billing capabilities to MCP-compatible clients, enabling automated billing and invoice workflows directly from chat or other MCP applications.

## Features
- Connects a Fakturoid account to MCP-compatible clients.
- Surfaces Fakturoid’s invoicing and billing functionality for use in automated workflows.
- Supports configuration by selecting and linking a specific Fakturoid client/account.
- Uses a static MCP server URL (`https://mcp.pipedream.net/v2`) that works for all clients, with authentication handled when adding the server to the application.
- Integrates with multiple chat or MCP client applications (addable as an MCP server in supported apps).

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Provider:** Pipedream (Fakturoid MCP integration)
