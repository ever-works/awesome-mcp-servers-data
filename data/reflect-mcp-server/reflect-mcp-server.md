# Reflect MCP Server

**Category:** Document Management MCP Servers  
**Brand:** Reflect

An MCP (Model Context Protocol) server that connects AI clients to your Reflect note-taking account, allowing AI-driven access to your notes and knowledge base.

---

## Features

- **Reflect Integration**  
  - Connects directly to your Reflect note-taking app.  
  - Enables AI tools and chat clients to access notes and stored knowledge from Reflect.

- **MCP Compatibility**  
  - Exposes Reflect as an MCP server, usable from any MCP-compatible client.

- **Static Server URL**  
  - Uses a single static base URL for all clients:  
    - `https://mcp.pipedream.net/v2`
  - Same URL works across different applications and chat clients.

- **Authentication at Client Setup**  
  - Authentication occurs when you add/configure the server in your chosen application.

- **Configuration Guidance**  
  - Instructions available for adding the server to different chat clients.  
  - Additional setup details provided via a dedicated configuration page (referred to as the “Configuration” page).

- **Built on Pipedream Connect**  
  - Hosted and powered by Pipedream’s infrastructure for MCP servers.

- **Use Cases**  
  - Use AI to query, summarize, or reference information in your Reflect notes.  
  - Treat your Reflect workspace as a knowledge base accessible from within MCP-aware chat tools.
