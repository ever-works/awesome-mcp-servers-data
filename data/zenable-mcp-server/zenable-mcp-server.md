## Overview
Zenable MCP Server is a security-oriented Model Context Protocol (MCP) server at `https://mcp.zenable.app/` that lets MCP-compatible AI agents integrate with Zenable’s security services using OAuth 2.1. It is designed to enable AI coding assistants to automatically check and improve code against an organization’s internal requirements.

## Features
- **Security-focused MCP endpoint** at `https://mcp.zenable.app/` for integrating Zenable’s security capabilities into MCP agents.
- **OAuth 2.1 authentication** for secure, standards-based access to Zenable’s services from compatible clients.
- **Organizational policy enforcement**: allows AI coding assistants to validate and improve code against an organization’s specified requirements and rules.
- **Code quality and compliance checks** via Zenable’s backend security/compliance services (as exposed through the MCP server).
- **Standardized MCP configuration**:
  - Provides an official `server.json` definition for this MCP server.
  - `server.json` includes installation instructions, configuration options, and usage guidelines.
  - The file is maintained and kept up to date by the owner of `zenable.app`.
