# Happy Scribe MCP Server

**Category:** Media Processing MCP Servers  
**Brand:** Happy Scribe  
**Source:** https://mcp.pipedream.com/app/happy_scribe

## Overview
The Happy Scribe MCP Server is an MCP (Model Context Protocol) server that connects applications to Happy Scribe’s transcription and subtitling platform. It enables automated generation of subtitles and transcripts from audio and video content through a standardized MCP endpoint.

## Features
- Connects to Happy Scribe’s transcription and subtitling platform
- Supports automated subtitle generation
- Supports automated transcript generation
- Designed for audio and video content (transcription and subtitles)
- Exposed via a static MCP server URL usable by any compatible client
- Authentication performed when adding the server to a client application

## MCP Server URL
- Static MCP server endpoint (works for every client):
  - `https://mcp.pipedream.net/v2`

## Integration & Configuration
- Add the MCP server to any compatible chat client or MCP-aware application
- Authentication is handled at the time you add the server to your application
- Additional setup and client-specific instructions are available via the general Configuration page on Pipedream
