## Overview

A Model Context Protocol (MCP) server for interacting with Kong Konnect APIs, allowing AI assistants to query and analyze Kong Gateway configurations, traffic, and analytics. Kong's AI Gateway enables teams to manage remote MCP traffic with enterprise-grade security and performance.

## Features

- **API to MCP Transformation**: Convert RESTful APIs into MCP tools
- **Enterprise Security**: Authentication, authorization, and access control
- **Traffic Management**: Load balancing and rate limiting
- **Context Propagation**: Maintain context across distributed systems
- **Observability**: Comprehensive monitoring and analytics
- **Protocol Bridge**: AI MCP Proxy plugin translates between MCP and HTTP
- **OAuth Integration**: AI MCP OAuth2 plugin for secure authentication

## Kong Gateway 3.12 MCP Capabilities

### AI MCP Proxy Plugin
- Protocol bridge between MCP and HTTP
- Convert regular RESTful APIs into MCP tools
- Expose grouped MCP Tools as an MCP Server
- Proxy MCP requests to existing MCP Servers

### AI MCP OAuth2 Plugin
- Secure OAuth 2.0 authentication
- Token management
- Authorization flows
