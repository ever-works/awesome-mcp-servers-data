# quentn MCP Server

## Overview
The **quentn MCP Server** exposes Quentn’s marketing operating system and customer communication portal through the Model Context Protocol (MCP), enabling applications and AI clients to interact with Quentn for CRM and marketing automation workflows.

- **Category:** Business & Commerce – MCP Servers  
- **Use cases:** Integrate Quentn with AI/chat clients, orchestrate customer journeys, and automate CRM-related tasks via MCP.

## Features
- **MCP-compatible server endpoint**  
  - Static base URL for all clients: `https://mcp.pipedream.net/v2`  
  - Can be added as a server in any MCP-capable application or chat client.

- **Quentn integration**  
  - Connects to a Quentn account (marketing operating system and customer communication portal).  
  - Uses your authenticated Quentn client/account after connection.

- **CRM-oriented capabilities**  
  - Designed to work with Quentn’s CRM and customer communication features (e.g., customer data, journeys, and automation) via MCP tools and actions.

- **Tooling exposure**  
  - Exposes Quentn-related tools / actions to MCP clients (listed in the interface as “Available tools” once loaded), enabling automation and orchestration from within your MCP-compatible app.

- **Configuration guidance**  
  - Simple flow: configure Quentn, connect your account, then copy and use the MCP server URL in your app.  
  - Additional configuration details available via a dedicated configuration page.

## Integration & Setup
1. **Configure Quentn** inside the Pipedream MCP app.  
2. **Connect your Quentn account** and select the client to use.  
3. **Copy the MCP server URL**: `https://mcp.pipedream.net/v2`.  
4. **Add the server to your MCP-compatible client** following that client’s instructions (e.g., chat application configuration).  
5. **Use exposed tools** to interact with Quentn CRM and marketing operations.
