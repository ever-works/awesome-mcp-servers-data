# Harvest MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Harvest  
**Slug:** `harvest-mcp-server`

MCP server that exposes Harvest’s online time tracking and invoicing capabilities so MCP-based agents and applications can log time and manage billing data.

---

## Overview
The Harvest MCP Server connects MCP-compatible agents or chat clients to a Harvest account, enabling programmatic access to time tracking and invoicing features. It is provided via Pipedream Connect as a static MCP endpoint that you can add to any supported MCP client.

---

## MCP Server URL
- Base MCP server URL (static for all clients):
  - `https://mcp.pipedream.net/v2`
- Authentication is performed when you add/configure the server in your application or client.

---

## Features
- **Time Tracking Integration**
  - Designed to expose Harvest time tracking capabilities to MCP-based agents.
  - Allows agents and apps to log time entries via the MCP server.

- **Invoicing & Billing Integration**
  - Provides access to Harvest invoicing and billing-related capabilities.
  - Intended for managing billing data (e.g., invoices, billable time) through MCP-compatible clients.

- **Static MCP Endpoint**
  - Single static URL for all clients: `https://mcp.pipedream.net/v2`.
  - Configuration handled at the client level, with per-client authentication.

- **MCP Client Compatibility**
  - Can be added to supported chat clients or MCP-based applications.
  - Documentation and examples are available via the Pipedream configuration page.

- **Hosted by Pipedream Connect**
  - Server is hosted and managed by Pipedream Connect.
  - No separate server deployment required by the user.

---

## Setup & Configuration
- **Add to Chat Client / App**
  - Use the static URL `https://mcp.pipedream.net/v2` in your MCP-compatible chat client or application.
  - Authenticate your Harvest account during setup.
  - Additional configuration steps vary by client and are documented on Pipedream’s configuration page.
