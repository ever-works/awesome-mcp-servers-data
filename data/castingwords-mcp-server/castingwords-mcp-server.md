# CastingWords MCP Server

**Category:** Media Processing MCP Servers  
**Brand:** CastingWords  
**Slug:** `castingwords-mcp-server`

High-quality audio transcription exposed via the Model Context Protocol (MCP), allowing chat clients and MCP-compatible tools to access CastingWords’ speech-to-text services.

---

## Features

- **MCP-compatible transcription server**  
  - Accessible via a static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Works with any MCP-enabled client or application.

- **Audio transcription services**  
  - Converts spoken audio to text (speech-to-text).  
  - Suitable for a wide range of audio lengths and volumes:
    - Short clips (e.g., ~30 seconds).  
    - Long-form content such as focus-group recordings.  
    - Large-scale or batch workloads.

- **Integration-focused design**  
  - Designed to be added as an MCP server within chat clients and applications.  
  - Supports use in transcription API–style integrations via MCP.

- **General characteristics**  
  - Emphasis on high-quality transcription output.  
  - Intended to be easy to use and reliable for ongoing transcription needs.

- **Authentication**  
  - Authentication occurs when adding the MCP server to your client or application (details handled during configuration).

---

## Configuration & Usage

- Use the static MCP URL `https://mcp.pipedream.net/v2` when adding the server to your MCP-compatible client.  
- Follow your chat client’s instructions or the platform’s configuration documentation to:
  - Register the CastingWords MCP Server.  
  - Provide any required authentication credentials.
