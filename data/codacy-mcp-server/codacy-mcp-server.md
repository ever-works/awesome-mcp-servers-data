# Codacy MCP Server

**Category:** Testing & Debugging Tools  
**Tags:** code-quality, devops, analytics

An MCP-compatible server that exposes Codacy’s DevOps intelligence and code quality metrics for use in automation, analysis workflows, and chat-based tools.

---

## Features

- **MCP-compatible endpoint**  
  - Static server URL for all compatible clients: `https://mcp.pipedream.net/v2`  
  - Designed to plug into MCP-enabled chat or automation applications.

- **Codacy integration**  
  - Provides access to Codacy’s DevOps intelligence data.  
  - Surfaces code quality and related analytics for use in workflows and tools.

- **Client-agnostic configuration**  
  - Same base URL works across different MCP clients.  
  - Authentication handled when adding the server to the client/application.

- **Pipedream Connect platform**  
  - Hosted and powered by Pipedream Connect infrastructure.

---

## Usage

- Add the MCP server URL `https://mcp.pipedream.net/v2` to your MCP-compatible chat client or application.  
- Authenticate within your client when prompted.  
- Use the client’s configuration or documentation to explore available Codacy-backed tools and actions.
