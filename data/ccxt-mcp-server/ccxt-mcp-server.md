## Overview

CCXT-based MCP server for crypto exchange data and trading.

## Features

- Real-time tickers, OHLCV
- Order management
- Multi-exchange support
- Balances and positions

## Use Cases

- Cross-exchange arbitrage agents
- Trading bots

## Comparisons

Direct CCXT wrapper; standardized MCP interface.

## Pricing

Free; exchange APIs may have fees.
