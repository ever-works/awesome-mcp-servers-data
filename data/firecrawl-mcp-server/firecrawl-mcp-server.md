## Overview

A Model Context Protocol (MCP) server implementation that integrates with Firecrawl for web scraping capabilities. Works with Claude Code, Cursor, VS Code, Windsurf, and any MCP-compatible client. Install once and use across all your editors.

## Key Capabilities

- **Scraping**: Return clean markdown or structured JSON your agent can use immediately
- **Search**: Let your agent use Search to discover sources, then Scrape to pull fresh context
- **Error Handling**: Automatic retries for transient errors, rate limit handling with backoff, detailed error messages, credit usage warnings, and network resilience

## Installation

Run one command (`npx -y firecrawl-mcp`) to install the official server. Add your API key once and use it in Cursor, Claude Code, Windsurf, and more. Most developers are set up in under 3 minutes.
