## Features

- **firecrawl_scrape**: Scrape URL to clean markdown or structured JSON with mobile rendering, tag filtering, JS waiting.
- **firecrawl_search**: Web search with content extraction and time-based filtering.
- **firecrawl_crawl / firecrawl_check_crawl_status**: Asynchronous site crawling with depth, deduplication, domain limits.
- **firecrawl_map**: Discover indexed URLs on a site.
- **firecrawl_extract**: LLM-based structured data extraction using JSON schema.
- **firecrawl_agent / firecrawl_agent_status**: Autonomous agent for browsing, searching, compiling reports.
- **firecrawl_interact**: Interactive page actions (click, form fill, navigate) with persistent sessions.
- **firecrawl_browser_* tools**: Manage persistent browser sessions via CDP.
