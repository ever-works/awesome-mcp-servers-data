# DoneDone MCP Server

DoneDone MCP Server integrates DoneDone’s task tracking and shared inbox capabilities into the Model Context Protocol (MCP) so you can manage tasks and conversations programmatically from compatible MCP clients.

---

## Overview
- **Name:** DoneDone MCP Server  
- **Category:** Project Management MCP Server  
- **Tags:** task-management, shared-inbox, support  
- **Provider:** Pipedream Connect  
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

---

## Features
- **MCP integration**: Exposes DoneDone functionality through a standard MCP server endpoint usable by any compatible MCP client.
- **Task tracking**: Access and manage DoneDone’s simple task tracking features programmatically (e.g., create, update, and organize tasks).*  
- **Shared inbox integration**: Integrate DoneDone’s shared inbox for handling conversations via MCP.*  
- **Static server URL**: Uses a single static URL (`https://mcp.pipedream.net/v2`) for all clients; configuration is the same across tools.
- **Per-client authentication**: Authentication is performed when you add the MCP server to your specific application or chat client.
- **Client-agnostic setup**: Designed to work with multiple chat clients; configuration instructions are available via the full configuration page.

\*The underlying DoneDone product provides task tracking and shared inbox capabilities; specific MCP tools / methods are implied but not detailed in the provided content.

---

## Setup & Configuration
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **How to add to your app:**
  - Use the static MCP endpoint above in your MCP-compatible chat client or application.
  - Authenticate within your client when prompted during server addition.
  - Refer to the platform’s configuration documentation ("Configuration" page) for client-specific steps.
