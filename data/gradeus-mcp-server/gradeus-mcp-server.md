# Grade.us MCP Server

**Category:** Business & Commerce · MCP Servers  
**Slug:** `gradeus-mcp-server`

A Model Context Protocol (MCP) server integration for Grade.us that connects white‑label review management capabilities into MCP‑enabled applications.

---

## Description

Grade.us MCP Server integrates Grade.us’s white‑label review management platform with MCP-compatible chat or agent applications. It allows you to connect a Grade.us account, select a specific client, and then use Grade.us review workflows programmatically from within your MCP environment.

---

## Features

- **MCP Integration**  
  - Exposes Grade.us functionality as an MCP server endpoint for use in MCP-compatible tools and chat clients.

- **Static MCP Server URL**  
  - Uses a single static server URL for all clients:  
    - `https://mcp.pipedream.net/v2`  
  - Authentication is handled when you add the server to your application.

- **Account & Client Selection**  
  - Connect a Grade.us account via the Pipedream MCP integration.  
  - Select an individual client profile to scope review management operations.

- **White‑Label Review Management Workflows**  
  - Supports workflows aligned with Grade.us’s core capabilities, including:  
    - Generating online reviews (review request workflows).  
    - Monitoring reviews across supported platforms.  
    - Using reviews for marketing and reputation management for agency clients.

- **App Integration Instructions**  
  - Guided steps to add the MCP server to various supported chat clients.  
  - Reference to a configuration page with additional setup details.

- **Tooling Interface**  
  - Provides MCP tools / actions (listed dynamically as “available tools”) once configured, enabling automated or assisted workflows from within the MCP client.
