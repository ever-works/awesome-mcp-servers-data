# Model Context Protocol Kotlin SDK

## Overview
Official Kotlin SDK for the Model Context Protocol (MCP), providing idiomatic Kotlin support for building MCP servers and clients. The project is maintained in collaboration with JetBrains and is available as an open-source library on GitHub.

- **Repository:** https://github.com/modelcontextprotocol/kotlin-sdk
- **Documentation:** https://modelcontextprotocol.github.io/kotlin-sdk/
- **Brand:** Model Context Protocol

## Features
- **Idiomatic Kotlin API** for implementing MCP servers and clients.
- **Dedicated modules** within the repository:
  - `kotlin-sdk-core` – core shared logic for the SDK.
  - `kotlin-sdk-client` – client-side components for interacting with MCP servers.
  - `kotlin-sdk-server` – server-side components for exposing MCP-compatible services.
  - `kotlin-sdk-test` – testing-related utilities or structures (as indicated by module naming).
  - `conformance-test` – conformance testing module to validate behavior against the protocol specification.
  - `docs` – documentation sources for the SDK.
- **Open-source development model**, with active collaboration and contributions via GitHub.
