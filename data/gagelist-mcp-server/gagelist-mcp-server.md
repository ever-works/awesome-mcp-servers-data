# GageList MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** GageList  
**Source:** https://mcp.pipedream.com/app/gagelist

Calibration management and asset tracking MCP server that connects MCP-compatible tools and automations to GageList.

---

## Overview

GageList MCP Server exposes GageList’s calibration management and asset tracking capabilities to any MCP-compatible client via a static MCP endpoint hosted on Pipedream Connect.

---

## Features

- **MCP-Compatible Endpoint**  
  - Single static MCP server URL for all clients: `https://mcp.pipedream.net/v2`  
  - Usable by any MCP-compatible chat client or automation tool.

- **GageList Integration**  
  - Connects to GageList calibration software.  
  - Exposes calibration management capabilities through MCP.  
  - Exposes asset tracking capabilities through MCP.

- **Client-Agnostic Setup**  
  - Same server URL works across supported clients.  
  - Authentication performed when adding the server in each client.

- **Pipedream Connect Hosting**  
  - Server is provided and hosted via Pipedream Connect infrastructure.

---

## Configuration & Usage

- Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server to your MCP-compatible app or chat client, then authenticate within that client.
- Additional configuration details are available on the product’s Configuration page (linked from the source page).
