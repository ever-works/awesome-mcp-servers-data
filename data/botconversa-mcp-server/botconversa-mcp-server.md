# BotConversa MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** BotConversa  
**Website:** https://mcp.pipedream.com/app/botconversa

## Overview
BotConversa MCP Server is an MCP (Model Context Protocol) integration that connects BotConversa’s no-code chatbot platform with MCP-compatible chat clients. It allows tools and applications using MCP to create and manage chatbots for marketing, lead generation, fulfillment, and other automation workflows.

The integration is powered by Pipedream Connect and is accessible through a single static MCP server URL.

## Features
- **MCP-compatible server**  
  - Exposes BotConversa functionality via the MCP protocol for use in compatible chat clients and tools.

- **Static MCP server endpoint**  
  - Single shared URL for all clients:  
    `https://mcp.pipedream.net/v2`  
  - Authentication is performed when adding the server to each client.

- **No-code chatbot creation (via BotConversa)**  
  - Connects to BotConversa’s no-code builder to create and manage chatbots without programming.

- **Marketing and automation use cases**  
  - Supports automation of:  
    - Marketing campaigns  
    - Lead generation  
    - Fulfillment flows  
    - Other conversational automation scenarios

- **Client-agnostic integration**  
  - Designed to be added to various MCP-compatible chat clients.  
  - Documentation available via a configuration page for specific clients (linked from the app page).

- **Hosted by Pipedream Connect**  
  - Infrastructure and connectivity managed through Pipedream’s integration platform.

## Setup
- Use the MCP server URL: `https://mcp.pipedream.net/v2` when configuring your MCP client.
- Authenticate during the server-adding flow in your client (per its configuration instructions).
- Optional: Refer to the app’s configuration page on Pipedream for client-specific setup steps.
