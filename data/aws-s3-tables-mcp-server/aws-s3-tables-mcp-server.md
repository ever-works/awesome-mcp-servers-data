# AWS S3 Tables MCP Server

**Category:** Data Access & Integration MCP Servers  
**Brand:** AWS  
**Source:** https://modelcontextprotocol.io/mcp/servers/s3-tables-mcp-server

## Overview
AWS S3 Tables MCP Server is an MCP-compatible server for working with table-like data stored in S3 or S3-compatible object storage. It focuses on managing, querying, and ingesting S3-based tables using SQL and related table operations.

## Features
- **S3 table management**
  - Manage tables stored in AWS S3 or S3-compatible storage backends.
- **SQL querying**
  - Run SQL queries against S3-backed tables.
- **CSV-to-table conversion**
  - Ingest CSV files from S3 and convert them into queryable tables.
- **Metadata discovery**
  - Discover and expose metadata about available S3-based tables.

## Integrations & Compatibility
- Works with **AWS S3**.
- Supports **S3-compatible** storage systems.
- Exposed as an **MCP server**, suitable for use with MCP-compatible clients.
