# Microsoft Teams MCP Server

## Overview
Microsoft Teams MCP Server exposes Microsoft Teams collaboration features—such as communities, events, chats, channels, meetings, storage, tasks, and calendars—through the Model Context Protocol (MCP). This allows agents and MCP-compatible clients to coordinate, message, and manage Teams resources programmatically.

- **Category:** Messaging MCP Servers  
- **Brand:** Microsoft  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **Unified collaboration access** via MCP to:
  - Communities
  - Events
  - Chats
  - Channels
  - Meetings
  - File storage
  - Tasks
  - Calendars
- **Programmatic control** of Microsoft Teams resources from MCP-compatible clients and agents.
- **Static MCP endpoint** (`https://mcp.pipedream.net/v2`) usable across different clients.
- **Client-agnostic integration**: same server URL works for any supported MCP client; authentication occurs during server addition.

## Integration & Usage
- Add the MCP server to your MCP-enabled chat or agent client using:
  - **Server URL:** `https://mcp.pipedream.net/v2`
- Configure authentication and other settings in your client as prompted.
- Refer to the platform’s **Configuration** documentation for detailed setup steps.
