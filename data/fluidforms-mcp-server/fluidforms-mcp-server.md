# FluidForms MCP Server

**Category:** Business & Commerce – MCP Servers  
**Tags:** forms, website, lead-generation

## Overview
FluidForms MCP Server is an MCP-based integration that exposes FluidForms’ form creation tools for use with chat-based clients, specifically tailored for Canva websites. It lets applications connect to FluidForms through a standard MCP server endpoint.

## Features
- MCP server interface for FluidForms.
- Access to FluidForms form creation tools via MCP.
- Designed for creating forms on Canva websites.
- Static MCP server URL that works across compatible clients.
- Authentication performed when adding the server to an application.

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Integration Provider:** Pipedream Connect
- **Usage:** Add the MCP server URL to any supported MCP client, then authenticate within the client.
