# Warp

Modern, AI-native terminal with built-in agents and first-class Model Context Protocol (MCP) client support, designed to help developers build, debug, and understand code from a single interface.

## Features

- **AI-native terminal**
  - Modern terminal environment optimized for speed and developer workflows
  - Integrated AI assistance directly in the terminal
- **Agents 3.0 integration**
  - Agents with full terminal capabilities
  - Use agents to build complex features in production codebases
  - Deploy multiple agents and track them centrally
  - Step in and take over from agents when needed
- **MCP client support**
  - Built-in Model Context Protocol (MCP) client
  - Dedicated **MCP Servers** page to:
    - View available MCP servers
    - Configure server settings
    - Start and stop MCP servers
  - Manage and orchestrate multiple MCP servers from a single interface
  - MCP controls integrated into:
    - Global settings
    - Warp Drive
    - Command palette
    - AI settings
- **Terminal features**
  - Modern, intelligent terminal focused on speed and productivity
- **Warp Drive**
  - Centralizes knowledge and context for agents and teammates
- **Code-focused workflows**
  - Support for building features, fixing bugs, debugging production, and understanding large codebases
- **Platform availability**
  - Download available for Windows
