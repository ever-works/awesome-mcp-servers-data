# BugHerd MCP Server

**Category:** Project Management MCP Servers  
**Brand:** BugHerd  
**Slug:** bugherd-mcp-server

## Description
BugHerd MCP Server is an integration that connects BugHerd’s visual bug tracking and website feedback platform to MCP-compatible AI tools. It enables these tools to collect website feedback and manage bug tracking tasks through BugHerd.

## Features
- Connects a BugHerd account to MCP-compatible AI tools.
- Supports integration with tools such as Cursor, Claude, and Windsurf.
- Enables collection of website feedback via AI-assisted workflows.
- Facilitates management of visual bug tracking tasks through BugHerd’s platform.

## Integrations
- Cursor
- Claude
- Windsurf
- Other MCP-compatible tools
