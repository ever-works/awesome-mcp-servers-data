# Linkly MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Linkly  
**Source:** https://mcp.pipedream.com/app/linkly

## Overview
The Linkly MCP Server is an MCP-compatible service that integrates Linklyâ€™s link tracking capabilities into MCP-enabled clients. It enables users to create and manage tracking links, apply retargeting tags, and configure smart redirects through tools that support the MCP protocol.

## Features
- **Tracking link creation**
  - Create tracking links through MCP-compatible clients using a static MCP server endpoint.
- **Retargeting tags**
  - Add retargeting tags to links managed via the MCP server.
- **Smart redirects**
  - Configure smart redirects for links (e.g., redirect behavior based on predefined rules or conditions).
- **MCP integration**
  - Exposes a static server URL (`https://mcp.pipedream.net/v2`) that works across compatible chat and MCP clients.
  - Authentication occurs when adding the server to an application.
- **Multi-client compatibility**
  - Designed to be added to various chat clients or MCP-enabled applications.

## MCP Server URL
- Static MCP Server URL: `https://mcp.pipedream.net/v2`
- The same URL is used for all clients; authentication is handled during server setup in each app.
