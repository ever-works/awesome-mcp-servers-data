# Appcircle MCP Server

**Category:** Cloud DevOps MCP Servers  
**Tags:** CI/CD, mobile, DevOps

## Overview
Appcircle MCP Server is an MCP-compatible endpoint that connects MCP clients (such as chat-based developer tools) to Appcircle’s automated mobile DevOps and CI/CD platform. It is provided via Pipedream Connect as a static MCP server URL that can be added to any supported MCP client.

## MCP Server URL
- Base MCP server URL (static for all clients):
  ```
  https://mcp.pipedream.net/v2
  ```
- Authentication occurs when adding/configuring the server in the MCP-compatible application.

## Features
- **MCP integration endpoint**: Exposes Appcircle’s mobile DevOps / CI/CD capabilities through the Model Context Protocol (MCP) for use in compatible chat clients and tools.
- **Static server URL**: Single, fixed MCP server URL that works across different MCP clients.
- **Client-agnostic setup**: Designed to be added to various MCP-capable chat or AI clients via their MCP configuration flow.
- **Central configuration docs**: Full configuration details are available via the linked configuration page on Pipedream Connect.
- **Enterprise mobile DevOps backend**: Connects to Appcircle, an enterprise-grade automated mobile DevOps and CI/CD platform (builds, testing, deployment workflows for mobile apps—details implied by the platform description, though not enumerated on this page).

## Getting Started
1. Copy the MCP server URL:
   ```
   https://mcp.pipedream.net/v2
   ```
2. Add the server URL to your MCP-compatible chat client or application.
3. Authenticate within the client when prompted.
4. (Optional) Refer to the Pipedream Connect configuration page for detailed setup steps for specific clients.
