# Qwilr MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Qwilr  
**Slug:** qwilr-mcp-server

## Description
Qwilr MCP Server is an MCP Server integration for Qwilr that enables programmatic creation and management of smart sales documents—such as proposals and quotes—within MCP-enabled workflows.

## Features
- MCP-compatible server integration for Qwilr
- Supports creation and management of smart sales documents
- Suitable for proposals, quotes, and similar sales documents
- Single static MCP server URL for all clients: `https://mcp.pipedream.net/v2`
- Authentication performed when adding the server to an MCP-compatible application
- Hosted and powered by Pipedream Connect

## Technical Details
- MCP server base URL: `https://mcp.pipedream.net/v2`
- Can be added to any compatible chat or MCP client via server configuration
