# Mailtrap MCP Server

**Category:** Testing & Debugging Tools  
**Tags:** email, testing, sandbox

## Overview
Mailtrap MCP Server exposes the Mailtrap email testing and delivery platform via the MCP protocol, allowing tools and agents to send, capture, and inspect emails in sandboxed or production environments.

## Features
- **MCP Protocol Integration**: Access Mailtrapâ€™s functionality (email testing and delivery) through the Model Context Protocol.
- **Email Sending**: Enable tools and agents to send emails via Mailtrap programmatically.
- **Email Capture & Inspection**: Capture and inspect emails for testing, debugging, or monitoring purposes.
- **Sandbox & Production Support**: Use sandbox environments for safe email testing or connect to production for live delivery workflows.
- **Static MCP Server URL**: Single static endpoint usable across MCP-compatible clients:
  - `https://mcp.pipedream.net/v2`
- **Client-Agnostic Setup**: Designed to be added to different chat or agent clients that support MCP.

## Setup
- Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
- Authentication is performed when adding the server in your MCP-compatible application or client configuration.
