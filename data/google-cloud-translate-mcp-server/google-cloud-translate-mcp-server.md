# Google Cloud Translate MCP Server

MCP server for Google Cloud Translate, providing dynamic machine translation between languages via the MCP ecosystem.

- **Category:** AI Integration – MCP Servers  
- **Slug:** `google-cloud-translate-mcp-server`  
- **Brand:** Google Cloud  
- **Source URL:** https://mcp.pipedream.com/app/google_cloud_translate

## Features

- **Dynamic machine translation** between languages using Google Cloud’s machine learning–based translation services.
- **MCP-compatible server** that plugs into tools and clients supporting the Model Context Protocol (MCP).
- **Single static MCP endpoint** for all clients:
  - Server URL: `https://mcp.pipedream.net/v2`
- **Client-agnostic setup**: works with any MCP-capable chat or development client; you authenticate when adding the server to your application.
- **Configuration documentation** available via a central configuration page (on the provider site) for setup details.

## Authentication
