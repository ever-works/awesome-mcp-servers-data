# Dynalist MCP Server

## Overview
Dynalist MCP Server connects the Dynalist outlining and note‑taking platform to MCP, enabling agents and applications to interact with Dynalist programmatically for productivity workflows.

- **Category:** Content Management MCP Servers
- **Brand:** Dynalist
- **Slug:** `dynalist-mcp-server`
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- Programmatic access to Dynalist outlines and notes via MCP.
- Central MCP server endpoint (`https://mcp.pipedream.net/v2`) usable across compatible clients.
- Works with multiple chat or MCP-compatible clients (add the server to your chosen app/client).
- Authentication occurs when adding the server to your application, keeping the server URL static.
- Integration managed and hosted through Pipedream’s MCP infrastructure.

## Usage
1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add the server to your MCP-compatible application or chat client.
3. Authenticate your Dynalist/Pipedream account when prompted.
4. Use the exposed tools/actions to create, organize, and query Dynalist outlines.
