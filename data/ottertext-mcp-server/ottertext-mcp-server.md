# OtterText MCP Server

**Category:** Messaging MCP Servers  
**Website:** https://mcp.pipedream.com/app/ottertext

## Description
OtterText MCP Server connects the Otter Text SMS marketing platform to the Model Context Protocol (MCP), enabling AI agents and developer tools to send and manage SMS marketing campaigns programmatically.

## Features
- Connects an OtterText SMS marketing account to MCP-compatible tools.
- Works with AI tools such as Cursor, Claude, and Windsurf via Pipedreamâ€™s MCP server.
- Allows programmatic sending of SMS messages through OtterText.
- Enables management of text message marketing campaigns via AI agents and workflows.
