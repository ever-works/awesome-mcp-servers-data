# KnowBe4 MCP Server

Integrated MCP server that connects KnowBe4’s security awareness training platform with MCP-compatible tools and workflows.

## Overview
- **Type:** MCP server / integration
- **Platform:** Pipedream Connect
- **Purpose:** Enable applications (e.g., chat clients) to integrate with KnowBe4 for security awareness training via MCP.

## Features
- **Static MCP Server Endpoint**  
  - Single static URL used by all clients:  
    `https://mcp.pipedream.net/v2`  
  - Authentication performed when adding the server to your application.

- **Chat Client Integration**  
  - Designed to be added to various chat clients that support MCP.  
  - Documentation available per chat client to guide setup.

- **MCP-Based Workflow Integration**  
  - Intended to integrate KnowBe4’s security awareness training capabilities into MCP-based tools and workflows (per item description).  
  - Operates as part of the Pipedream Connect ecosystem.

- **Central Configuration Resources**  
  - References a full configuration page for detailed setup steps.  
  - Uses Pipedream’s platform for hosting, terms, privacy, and cookie settings.

## Setup
1. Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add the server to your MCP-compatible app or chat client.
3. Authenticate during the add-server process as prompted by your client.
4. Optionally consult the full configuration page on Pipedream for client-specific instructions.
