# Bigpicture.io MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Bigpicture.io  
**Slug:** bigpictureio-mcp-server

Bigpicture.io MCP Server is an integration that exposes Bigpicture.io’s company data APIs to MCP-compatible tools and chat clients via a single, static MCP endpoint.

---

## Features

- **MCP-compatible server**  
  - Provides an MCP Server integration for Bigpicture.io.  
  - Designed to work with any MCP-capable client or chat application.

- **Static server endpoint**  
  - Single static MCP server URL for all clients:  
    - `https://mcp.pipedream.net/v2`  
  - Authentication is handled when adding the server inside the client/application.

- **Company and firmographic data access**  
  - Connects to Bigpicture.io’s company APIs.  
  - Enables retrieval of company and firmographic data suitable for data-driven B2B workflows (e.g., lead enrichment, company data lookups).

- **Pipedream Connect integration**  
  - Powered by Pipedream Connect infrastructure.  
  - Centralized configuration via Pipedream’s configuration page.

- **Multi-client setup**  
  - Can be added to various chat clients that support MCP.  
  - Uses the same URL across clients, simplifying setup.

- **Intended use cases**  
  - Company data enrichment for B2B workflows.  
  - Lead and account research inside MCP-enabled chat tools.  
  - API-based integration of firmographic data into custom agents and tools.

---

## Setup & Configuration

- **Server URL:** `https://mcp.pipedream.net/v2`  
- Add this URL as an MCP server in your chosen MCP-compatible client.  
- Authentication is performed during server addition within the client.

For detailed setup per client, refer to the configuration instructions provided by Pipedream Connect.
