# DNSFilter MCP Server

**Category:** Security / Attestation MCP Servers  
**Tags:** security, DNS

## Overview
DNSFilter MCP Server integrates DNSFilter’s DNS-layer threat protection into MCP-based automations. It allows IT professionals and managed service providers to apply DNS security policies and filtering operations programmatically through MCP-compatible chat clients and tools.

## Features
- Integrates DNSFilter DNS-layer threat protection with MCP-based automations
- Supports security policy and filtering operations via MCP tools
- Designed for IT professionals and managed service providers
- Machine learning–driven DNS threat protection
- Single static MCP server endpoint usable across clients: `https://mcp.pipedream.net/v2`
- Compatible with multiple MCP-capable chat clients (configured per client)

## Technical Details
- MCP server URL (static for all clients):
  - `https://mcp.pipedream.net/v2`
- Authentication occurs when adding the server within each supported client or application
