# YouCanBook.Me MCP Server

## Overview
The YouCanBook.Me MCP Server connects a YouCanBook.Me scheduling account to MCP-compatible AI tools (such as Cursor, Claude, and Windsurf), enabling automated access to online scheduling and booking functionality.

- **Category:** Workflow Automation MCP Servers  
- **Provider / Brand:** Pipedream  
- **Slug:** youcanbookme-mcp-server  
- **Tags:** scheduling, booking, calendar-integration

## Features
- Connects a YouCanBook.Me account to MCP-compatible AI agents.  
- Exposes online scheduling and booking capabilities to AI tools.  
- Supports custom availability configuration via the MCP interface.  
- Enables automatic meeting link creation through YouCanBook.Me.  
- Integrates with AI tools such as Cursor, Claude, and Windsurf for AI-driven scheduling workflows.

## Integrations
- **YouCanBook.Me** for scheduling and booking.  
- **AI Tools:** Cursor, Claude, Windsurf (and other MCP-compatible agents).
