Formcarry MCP Server lets MCP-compatible tools collect and process submissions from custom HTML forms without requiring any backend code. It connects your Formcarry account to MCP-based chat or agent clients via a static MCP server endpoint.

---

## Features

- **Backend-free form handling**  
  - Collect submissions from your own custom HTML forms.  
  - No need to implement or host your own server-side form handling logic.

- **MCP server integration**  
  - Exposes Formcarry as an MCP (Model Context Protocol) server for use with MCP-enabled clients and tools.  
  - Allows chat/agent applications to access and work with form submission data through MCP tools.

- **Static MCP endpoint**  
  - Single static server URL for all clients: `https://mcp.pipedream.net/v2`.  
  - Authentication handled when adding the server to your MCP-compatible application.

- **Account-based configuration**  
  - Connect your Formcarry account to configure access.  
  - Select from available clients within the interface once connected.

- **Client setup guidance**  
  - Per-chat-client instructions on how to add the MCP server to different applications.  
  - Link to a full configuration page with more detailed setup steps.

- **Surveys and general-purpose forms**  
  - Suitable for surveys, contact forms, and other data-collection use cases.  
  - Designed to work with any custom HTML form that can submit to Formcarry.
