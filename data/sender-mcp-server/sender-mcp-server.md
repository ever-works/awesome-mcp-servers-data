# Sender MCP Server

## Overview
Sender MCP Server is an integration that connects the Sender email marketing platform with MCP-compatible clients, allowing you to work with email marketing campaigns through a unified MCP server endpoint.

## Features
- **Unified MCP Endpoint**  
  - Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) for all compatible clients.  
  - Authentication is handled when adding the server to your application.

- **Sender Email Marketing Integration**  
  - Connects to Sender (Sender.net), an email marketing platform.  
  - Intended to support:  
    - Creating email campaign designs.  
    - Automating email campaigns.  
    - Tracking email marketing results.

- **Client Integration**  
  - Can be added to various chat or MCP-compatible clients.  
  - Configuration guidance available via a dedicated configuration page (not client-specific in this summary).

## Usage
1. Use the static MCP URL: `https://mcp.pipedream.net/v2`.  
2. Add this server URL to your MCP-compatible or chat client.  
3. Authenticate within your client when prompted.
