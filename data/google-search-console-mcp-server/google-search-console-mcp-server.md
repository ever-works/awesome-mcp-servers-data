# Google Search Console MCP Server

**Brand:** Google  
**Category:** Web Search MCP Servers  
**Source:** https://mcp.pipedream.com/app/google_search_console

An MCP server that exposes Google Search Console data and tools to MCP-compatible clients, enabling workflows that use search traffic metrics, reports, and related SEO data.

---

## Features

- Provides access to Google Search Console tools and reports.  
- Enables measurement and analysis of website search traffic via MCP workflows.  
- Exposes search-related SEO data from Google Search Console to MCP-compatible chat or automation clients.  
- Uses a single static MCP server URL that works for all clients:  
  - `https://mcp.pipedream.net/v2`
- Authentication is performed when adding the server to your application (per-client auth flow).  
- Can be integrated into supported chat clients and applications as an MCP server endpoint.  
- Configuration details and client-specific setup are provided on the associated configuration page.

---

## Usage

1. Use the static MCP server URL: `https://mcp.pipedream.net/v2`.  
2. Add this MCP server endpoint to your MCP-compatible chat client or application.  
3. Authenticate with your Google Search Console account during setup.  
4. Use the exposed tools and reports to query search traffic metrics and SEO-related data.
