# CallMiner MCP Server

## Overview
The CallMiner MCP Server is an MCP server, hosted by Pipedream, that exposes CallMiner’s AI-powered conversation intelligence capabilities—such as contact center analytics and customer experience (CX) insights—to MCP-compatible clients.

- **Category:** Business & Commerce – MCP Servers  
- **Vendor / Brand:** CallMiner  
- **Integration host:** Pipedream  
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- **AI-powered conversation intelligence access**  
  Provides access to CallMiner’s conversation intelligence capabilities via the MCP protocol, enabling use within MCP-compatible chat or agent clients.

- **Contact center analytics**  
  Surfaces analytics derived from customer interactions handled by CallMiner, intended to help analyze and optimize contact center performance.

- **Customer experience (CX) insights**  
  Exposes insights about customer experience generated by CallMiner’s AI models to downstream MCP clients.

- **Standard MCP server endpoint**  
  Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) that works for all clients; authentication is handled when adding the server to an application.

- **Client-agnostic integration**  
  Can be connected to different MCP-compatible chat clients, which then gain access to the tools and actions exposed by CallMiner via Pipedream.

- **Tooling exposure (actions)**  
  Designed to expose CallMiner tools and actions (e.g., conversation intelligence operations) as MCP tools that can be invoked from supported clients. (Specific tools are not listed in the provided content.)

## Setup
- Connect a CallMiner account in Pipedream.
- Use the static MCP server URL `https://mcp.pipedream.net/v2` when configuring the MCP server in your client.
- Select and configure your chat client according to Pipedream’s MCP configuration guidance.
