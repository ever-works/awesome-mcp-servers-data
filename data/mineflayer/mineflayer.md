## Overview

Mineflayer is a JavaScript library for creating MCP bots.

## Features

- Real-time navigation
- Crafting and block interaction
- Chat parsing

## Installation

npm install mineflayer

## Usage

```javascript
const mineflayer = require('mineflayer');
const bot = mineflayer.createBot({
  host: 'localhost',
  port: 25565,
  username: 'AI_Bot'
});
```
