# Bandwidth MCP Server

Enterprise-grade MCP server integration for Bandwidth, enabling tools to access voice, messaging, and 911 services.

- **Category:** Messaging MCP Servers  
- **Provider:** Pipedream  
- **Source URL:** https://mcp.pipedream.com/app/bandwidth

## Features

- **Enterprise voice access** via Bandwidth through a standardized MCP server.  
- **Messaging integration** (SMS/messaging) exposed as tools to MCP-compatible clients.  
- **911 access** integration for emergency services through Bandwidth.  
- **Static MCP server endpoint** for all clients:  
  - Base URL: `https://mcp.pipedream.net/v2`  
- **Client-agnostic configuration**: the same static URL works for every MCP client; authentication is handled when adding the server to the application.  
- **Configurable via MCP client settings**: add the server URL into your chat or MCP-compatible application configuration.  
- **Central configuration reference** via a dedicated configuration page (instructions per client type).

## Configuration

- **MCP Server URL:**
  - `https://mcp.pipedream.net/v2`
- **Authentication:**
  - Performed when adding the server to your MCP-compatible application (details depend on the client and Bandwidth/Pipedream setup).
