# PolygonScan MCP Server

**Category:** Blockchain & Crypto MCP Servers  
**Brand:** PolygonScan  
**Slug:** polygonscan-mcp-server

## Overview
PolygonScan MCP Server is an MCP server that connects to the PolygonScan blockchain explorer, allowing MCP-compatible agents and applications to query data from the Polygon (MATIC) network.

## Features
- Connects to PolygonScan, the blockchain explorer for the Polygon (MATIC) network.
- Provides blockchain query capabilities to MCP agents and clients via a standardized MCP server interface.
- Uses a static MCP server endpoint that can be shared across compatible clients.
- Supports authentication at the time of adding the server to an application (client-side auth flow).

## MCP Server URL
- Static MCP server URL (used by all clients):
  - `https://mcp.pipedream.net/v2`

## Usage
- Add the MCP server URL to any compatible MCP/chat client.
- Authenticate within your client when prompted after adding the server.
