# Piggy MCP Server

An MCP server integration for Piggy’s loyalty engine, providing programmatic access to loyalty, rewards, and customer engagement capabilities through the MCP ecosystem.

## Overview
- **Type:** MCP server integration
- **Category:** Business & Commerce – MCP Servers
- **Provider / Brand:** Piggy (via Pipedream Connect)
- **Purpose:** Expose Piggy’s loyalty and rewards engine to MCP-compatible clients for use in applications and chat environments.
- **Base MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **Loyalty Engine Access**  
  - Connects to Piggy’s loyalty platform through the MCP protocol.  
  - Enables programmatic interaction with loyalty, rewards, and engagement functionality (as supported by Piggy’s backend and MCP tools).

- **Rewards & Customer Engagement**  
  - Designed to work with features related to rewards programs and customer engagement workflows within Piggy.  
  - Suitable for building or automating loyalty and rewards experiences in MCP-compatible clients.

- **Standard MCP Endpoint**  
  - Uses a single static MCP server URL: `https://mcp.pipedream.net/v2`.  
  - The same URL is used across clients; authentication is handled when adding the server to each application.

- **Client-Agnostic Integration**  
  - Can be added to multiple MCP-capable chat or application clients.  
  - Configuration is done per client, using the same base server URL.

- **Configuration Documentation**  
  - Detailed setup and configuration instructions are available via the linked configuration page (Pipedream / MCP configuration docs).

## Integration & Usage
- Add the server to an MCP-compatible chat or application client using:  
  `https://mcp.pipedream.net/v2`
- Authenticate within your chosen client during setup.  
- Follow the MCP configuration guide (via the “Configuration page” link) for client-specific steps.
