## Overview
AWS Knowledge MCP is a fully managed, remote Model Context Protocol (MCP) server provided by AWS. It exposes AWS knowledge and capabilities to AI assistants without requiring users to operate or maintain any infrastructure. This enables quick integration of AWS expertise into MCP-based workflows and tools.

## Features
- **Fully managed MCP server**  
  - AWS hosts and operates the MCP server for you.  
  - No infrastructure provisioning, scaling, or maintenance required.

- **Remote MCP endpoint**  
  - Accessible as a remote MCP server over the network.  
  - Designed to plug into MCP-compatible AI assistants and developer tools.

- **Access to AWS knowledge and capabilities**  
  - Surfaces AWS domain knowledge (services, concepts, and best practices) to AI assistants.  
  - Enables assistants and tools to leverage AWS capabilities through the MCP interface.

- **Integration into MCP-based workflows**  
  - Acts as a backend knowledge/capability provider inside MCP ecosystems.  
  - Supports building workflows where AI agents can query and use AWS-related information programmatically.

- **No infrastructure management for users**  
  - Users do not need to deploy, secure, or operate servers.  
  - Reduces operational overhead when adding AWS expertise to AI solutions.
