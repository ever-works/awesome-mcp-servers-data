# HoneyHive MCP Server

**Category:** Monitoring  
**Brand:** HoneyHive  
**Slug:** honeyhive-mcp-server

## Overview
The HoneyHive MCP Server is an MCP (Model Context Protocol) server that connects HoneyHive’s AI observability and evaluation platform with MCP-compatible agents and chat clients. It enables monitoring and evaluation workflows for AI systems directly through MCP.

- **Description:** An MCP server for HoneyHive that integrates AI observability and evaluation tooling with MCP agents, allowing monitoring and evaluation workflows via MCP.
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL that works across supported clients: `https://mcp.pipedream.net/v2`.
  - Authentication handled when the server is added to an application.

- **Integration with HoneyHive AI observability & evaluation**  
  - Connects AI observability tooling to MCP agents.  
  - Supports AI monitoring and evaluation workflows through MCP-based clients.

- **Multi-client support**  
  - Designed to be added to various MCP-capable chat clients and applications.
  - Configuration guidance available via a dedicated configuration page (client-specific setup instructions not detailed in the provided content).

- **Pipedream Connect-powered**  
  - Hosted and powered by Pipedream Connect infrastructure.
  - Uses a static, managed endpoint, simplifying client configuration.

## Configuration
- **Server URL:** Add `https://mcp.pipedream.net/v2` as an MCP server in your compatible chat client or application.
- **Authentication:** Performed during server addition/connection in the client (details depend on the specific client’s flow).
