# Icypeas MCP Server

Icypeas MCP Server is an MCP (Model Context Protocol) server, provided via Pipedream, that lets applications and chat clients programmatically access Icypeas contact data capabilities.

## Overview
- **Name:** Icypeas MCP Server  
- **Provider / Brand:** Pipedream  
- **Category:** Business & Commerce â€“ MCP Servers  
- **Primary Function:** Enable clients to find and access contact data through an MCP-compatible interface, suitable for integration into chat applications and workflows.

## Features
- **Programmatic contact data access**  
  - Provides access to Icypeas contact data for business use cases (e.g., sales and lead workflows).

- **Static MCP server endpoint**  
  - Single shared MCP server URL for all clients:  
    `https://mcp.pipedream.net/v2`  
  - Authentication is handled when adding the server to your specific application or client.

- **Chat client integration**  
  - Designed to be added to compatible chat clients as an MCP server.  
  - Supports configuration via a general "Configuration" guide (linked from the product page) to help integrate with different tools.

- **Workflow-friendly**  
  - Built to be used within Pipedream-powered environments and workflows, enabling automated retrieval and use of contact data.

## Usage
1. Use the static MCP server URL: `https://mcp.pipedream.net/v2`.  
2. Add this URL as an MCP server in your chat client or application.  
3. Authenticate within your app/client when prompted.  
4. Configure additional options following the general configuration documentation if needed.
