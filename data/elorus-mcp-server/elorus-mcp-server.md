# Elorus MCP Server

## Overview
Elorus MCP Server is an integration that exposes Elorus’s online business management capabilities—such as time tracking, invoicing, billing, and project and expense management—through an MCP (Model Context Protocol) server. It is accessed via a static MCP server URL and can be used from compatible chat or AI clients.

- **Category:** Business & Commerce MCP Servers  
- **Provider / Brand:** Elorus (via Pipedream)  
- **Primary Use Cases:** Time tracking, invoicing, billing, project and expense management via programmatic tools

## Features
- **MCP Server Integration**  
  - Provides a static MCP server endpoint: `https://mcp.pipedream.net/v2`  
  - Can be added to compatible chat / AI clients as an MCP server for tool-based interactions.

- **Account Connection**  
  - Connects to an existing Elorus account.  
  - Uses authentication at the time you add the server to your application.  
  - Supports selecting a client/account context after connection.

- **Business Management Capabilities (via Elorus)**  
  - **Time Tracking:** Online time-tracking functionality for businesses.  
  - **Invoicing:** Creation and management of invoices.  
  - **Billing:** General billing operations for clients and projects.  
  - **Project Management:** Project-related tracking in the context of billing and time tracking.  
  - **Expense Management:** Recording and managing business expenses.

- **Tool-Based Operations**  
  - Provides “Available tools” (actions) through MCP to interact programmatically with Elorus (exact tools are loaded dynamically and not listed in the provided content).

- **Configuration Guidance**  
  - Instructions to copy and use the MCP server URL.  
  - Option to select a chat client and view specific setup instructions.  
  - Reference to a full configuration page for advanced setup.
