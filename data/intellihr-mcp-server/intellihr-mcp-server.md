# IntelliHR MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** intellihr  
**Tags:** hr, performance-management

## Overview
The IntelliHR MCP Server exposes IntelliHR’s intelligent people management and real-time HR analytics capabilities to MCP-compatible chat clients and tools. It focuses on performance, engagement, wellbeing, compliance, and automation for HR teams, managers, and organizational leaders.

## MCP Server URL
- Base MCP server URL (static for all clients):  
  `https://mcp.pipedream.net/v2`

You'll authenticate when you add the server to your MCP-compatible application or chat client.

## Features
- **MCP integration endpoint**
  - Single static URL (`https://mcp.pipedream.net/v2`) usable by any client that supports the Model Context Protocol (MCP).
  - Designed to be added as a server within MCP-compatible chat applications.

- **Intelligent people management**
  - Interfaces with IntelliHR’s core HR and people management capabilities (as exposed via MCP).
  - Supports use cases for HR teams, managers, and organizational leaders.

- **Real-time HR analytics**
  - Access to real-time analytics around people data (subject to what is exposed by IntelliHR through the MCP integration).

- **Performance management focus**
  - Oriented toward performance-related workflows and insights (e.g., performance metrics, reviews, or goals, where available through IntelliHR’s API/MCP tools).

- **Engagement and wellbeing**
  - Targets scenarios involving employee engagement and wellbeing data and insights.

- **Compliance-related workflows**
  - Supports compliance-related HR scenarios (policies, records, or other compliance signals exposed via IntelliHR).

- **Automation support**
  - Intended for automated workflows that leverage IntelliHR data and actions via MCP.

- **Multi-client compatibility**
  - Can be added to various MCP-enabled chat clients; configuration steps are client-specific and referenced via the general configuration documentation.

## Integration & Configuration
- **Add to chat client**
  - Use the MCP server URL: `https://mcp.pipedream.net/v2`.
  - Follow your specific chat client’s instructions for adding an MCP server.
- **General configuration guide**
  - A full configuration page is available (via Pipedream) for detailed client-specific setup.
