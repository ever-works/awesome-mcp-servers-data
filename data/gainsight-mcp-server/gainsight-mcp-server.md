# Gainsight MCP Server

MCP Server integration for Gainsight NXT, giving MCP-based tools access to customer success and retention data workflows.

- **Website:** https://mcp.pipedream.com/app/gainsight_nxt
- **Category:** Business & Commerce – MCP Servers
- **Brand:** Gainsight
- **Slug:** `gainsight-mcp-server`

## Overview
The Gainsight MCP Server is a static MCP endpoint, hosted via Pipedream Connect, that you can add to compatible chat or MCP-enabled clients to integrate with Gainsight NXT customer success and retention workflows.

## Features
- **Static MCP Endpoint**  
  - Uses a single, static URL that works for all clients:  
    - `https://mcp.pipedream.net/v2`
  - Simplifies configuration across multiple MCP-compatible applications.

- **Client-Agnostic Integration**  
  - Designed to be added to various chat or MCP-enabled clients.  
  - Authentication occurs when you add/configure the server in your client.

- **Gainsight NXT Integration Context**  
  - Intended to expose Gainsight NXT customer success and retention data workflows via MCP-based tools (per item description).  
  - Enables MCP tools (like AI chat clients) to interact with Gainsight-related operations and data through the MCP server.

- **Configuration Documentation**  
  - Additional setup and configuration details are available via a central Configuration page (linked from the product page).

## Authentication
- Authentication is performed when adding the MCP server to your application/client.  
- The same base URL is used for all clients; credentials or tokens are applied within your chosen client’s MCP settings.

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Hosting / Infrastructure:** Powered by Pipedream Connect.
