# HeroBot Chatbot Marketing MCP Server

**Category:** Messaging MCP Servers  
**Brand:** herobot  
**Website:** https://mcp.pipedream.com/app/herobot_chatbot_marketing

## Overview
HeroBot Chatbot Marketing MCP Server provides Model Context Protocol (MCP) access to HeroBotâ€™s multi-channel chatbot marketing platform. It allows applications to interact with and automate messaging flows across multiple communication channels from a single MCP endpoint.

## Features
- **Single MCP Endpoint**  
  - Uses a static MCP server URL: `https://mcp.pipedream.net/v2` for all clients.  
  - Authentication handled when adding the server to your application.

- **Multi-Channel Messaging Integration**  
  - Instagram Direct Messages  
  - Facebook Messenger  
  - Google Business Messages  
  - WhatsApp  
  - Telegram  
  - Email  
  - SMS text messages  
  - Voice calls  
  - Viber  
  - Webchat

- **Chat Client Integration**  
  - Can be added to different chat clients as an MCP server.  
  - Configuration guidance available via a dedicated configuration page.

- **Automation & Orchestration**  
  - Designed to automate and combine messaging across supported channels.  
  - Supports use cases like sales generation, lead capture, and other marketing workflows driven by chatbots.

- **Tools & Actions via MCP**  
  - Exposes available tools/actions through the MCP interface (loaded dynamically in the Pipedream UI) for programmatic access from MCP-compatible clients.

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Access Method:** Add as an MCP server in compatible applications, then authenticate per client.
