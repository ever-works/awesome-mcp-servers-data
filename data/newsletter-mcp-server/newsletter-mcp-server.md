# Newsletter MCP Server

**Brand:** newsletter-wordpress  
**Category:** Business & Commerce – MCP Servers  
**Website:** https://mcp.pipedream.com/app/newsletter

## Overview
Newsletter MCP Server is an MCP (Model Context Protocol) server that connects the WordPress “Newsletter” plugin/solution with AI agents. It allows AI-powered tools and chat clients to send and manage email campaigns using your existing WordPress newsletter setup, via a standardized MCP endpoint hosted on Pipedream.

## Features
- **MCP integration endpoint**  
  - Static MCP base URL for all clients: `https://mcp.pipedream.net/v2`  
  - Works with any compatible MCP client or chat application.

- **WordPress Newsletter integration**  
  - Bridges the WordPress “Newsletter” solution with AI agents.  
  - Enables AI-driven sending and management of email campaigns from a WordPress-backed list and configuration.

- **AI agent campaign control**  
  - Allows AI agents connected via MCP to trigger and manage newsletter email sends.  
  - Intended for AI-powered workflows around email marketing and campaigns.

- **Hosted by Pipedream**  
  - Server is operated via Pipedream’s MCP infrastructure.  
  - Central configuration and documentation available through Pipedream’s configuration page.

- **Client-agnostic access**  
  - Same static MCP URL for all clients; authentication is handled when adding the server to each specific application or chat client.  
  - Compatible with multiple chat clients that support MCP server connections.

## Setup & Configuration
- Use the static MCP URL `https://mcp.pipedream.net/v2` when adding a new MCP server in your client.  
- Authenticate within your chat client or application as prompted when registering the server.  
- Additional configuration instructions are available on the Pipedream Configuration page (linked from the app page).
