# BC Gov News MCP Server

Government of British Columbia news and information streams exposed as an MCP (Model Context Protocol) server for programmatic access via compatible tools and agents.

## Overview
The BC Gov News MCP Server provides a unified MCP endpoint that lets applications and agents access current news and information published by the Government of British Columbia. It is hosted on Pipedream’s MCP infrastructure and can be integrated into MCP-compatible chat clients and tools.

- **Type:** MCP server (web-search / news)
- **Provider / Brand:** Government of British Columbia (hosted via Pipedream)
- **Category:** Web-search MCP servers
- **Use cases:**
  - Pulling current BC government news into AI assistants
  - Programmatic analysis of government news streams
  - Integrating official BC Gov updates into workflows and dashboards

## Features
- **MCP-compatible endpoint**  
  - Single static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Works across supported MCP clients; authentication is handled when adding the server to your app.

- **Government of British Columbia news access**  
  - Surfaces current news and information streams from official BC Gov sources.  
  - Designed for machine consumption (tools, agents, and workflows) rather than just human browsing.

- **Integration with chat clients and tools**  
  - Can be added to MCP-enabled chat or agent clients by configuring the provided server URL.  
  - Client-specific setup guidance is available via the configuration documentation (not specific to this server’s capabilities but to how to add an MCP server in general).

- **Hosted on Pipedream MCP infrastructure**  
  - Uses Pipedream’s managed MCP server platform.  
  - Central configuration page and generic MCP setup instructions available via Pipedream.

## Configuration
- **MCP server URL:** `https://mcp.pipedream.net/v2`  
- **Setup flow:**
  1. Connect a Pipedream account (if required by your environment).
  2. Copy the static MCP server URL.
  3. Add the server URL to your MCP-compatible application or chat client.
  4. Authenticate when prompted in your client.

Detailed, client-specific configuration instructions are available on Pipedream’s configuration pages.
