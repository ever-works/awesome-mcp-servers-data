## Overview

A comprehensive MCP server for Metabase integration providing 70+ ready-to-use tools that cover everything from dashboards to queries, permissions to monitoring. This backend integration layer connects your Metabase instance with AI assistants using the Model Context Protocol, enabling natural language interaction with business intelligence assets.

## Features

- **70+ Tools**: Comprehensive toolset for Metabase operations
- **Dashboard Management**: View, create, and manage dashboards
- **Query Operations**: Execute and manage questions/queries
- **Permission Control**: Manage user permissions and access
- **Monitoring**: Track usage and performance metrics
- **Natural Language Interface**: Interact with BI assets conversationally
- **Multi-Instance Support**: Connect to multiple Metabase instances

## Key Capabilities

- View and analyze dashboards
- Execute saved questions and ad-hoc queries
- Manage user access and permissions
- Monitor query performance
- Create and update visualizations
- Access database schemas and metadata
- Generate insights from data
