## Engage MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Engage  
**Source:** https://mcp.pipedream.com/app/engage

Engage MCP Server is a Message Control Protocol (MCP) server integration that connects the Engage customer messaging platform into MCP-compatible workflows and tools. It enables applications and chat clients to trigger and manage customer communications such as personalized email, push, in-app notifications, and SMS via a unified MCP endpoint.

### Features
- MCP-compatible server integration for Engage.
- Connects the Engage customer messaging platform into MCP workflows and tooling.
- Supports personalized, multi-channel customer messaging:
  - Email
  - Mobile push notifications
  - In-app notifications
  - SMS
- Single static MCP server URL for all clients: `https://mcp.pipedream.net/v2`.
- Authentication performed when adding the server to your client or application.
- Can be added to various chat clients that support MCP.

### Technical Details
- Protocol: Message Control Protocol (MCP) server.
- Endpoint: `https://mcp.pipedream.net/v2` (shared static URL across clients; auth required per client/app).
