# Form.io MCP Server

**Category:** Content Management MCP Servers  
**Tags:** forms, API, no-code

## Overview
Form.io MCP Server is an integration that connects your Form.io account to Model Context Protocol (MCP)-aware tools. It allows AI-powered editors and IDEs to build and manage forms and RESTful APIs backed by Form.io.

## Features
- **MCP integration for Form.io**: Exposes Form.io functionality through an MCP server so MCP-aware tools can interact with your forms and APIs.
- **Form management**: Enables creating and managing forms stored in your Form.io account via AI tools.
- **RESTful API management**: Supports building and managing RESTful APIs that correspond to your Form.io forms and data models.
- **AI tool connectivity**: Designed to work with tools such as Cursor, Claude, and Windsurf through Pipedream’s MCP server.
- **Web and mobile workflows**: Supports workflows for both web and mobile applications that rely on Form.io forms and APIs.
- **No-code / low-code workflows**: Leverages Form.io’s no-code form and API capabilities inside MCP-aware environments.

## Integrations
- **Pipedream MCP Server** – runtime environment for the MCP server.
- **Form.io** – forms and API backend.
- **AI / MCP-aware tools** – including (but not limited to):
  - Cursor
  - Claude
  - Windsurf
