# Modern Treasury MCP Server

**Category:** Finance & Market Data MCP Servers  
**Brand:** Modern Treasury  
**Slug:** `modern-treasury-mcp-server`

Modern Treasury MCP Server connects MCP-compatible agents and chat clients to Modern Treasury’s payment operations platform, enabling automated payment initiation, reconciliation, and cash management workflows.

---

## Features

- **MCP-compatible server endpoint**  
  - Provides a static, shared MCP server URL for all clients:  
    - `https://mcp.pipedream.net/v2`  
  - Works with any MCP-capable chat client or agent framework.

- **Authentication handled at client setup**  
  - Authentication is performed when adding the server to your application or chat client (credentials / method defined on the client side).

- **Integration with Modern Treasury payment operations**  
  - Connects agents to Modern Treasury’s payment operations software to support:  
    - Automated payment initiation workflows  
    - Reconciliation of transactions  
    - Cash management workflows

- **Client setup guidance**  
  - Instructions available for adding the server to different chat clients.  
  - Central configuration documentation via the linked Configuration page.

- **Hosted by Pipedream Connect**  
  - Server infrastructure and access are provided via Pipedream’s MCP hosting environment.

---

## How to Use

1. Configure your MCP-compatible chat client or agent.  
2. Add the MCP server endpoint: `https://mcp.pipedream.net/v2`.  
3. Complete authentication as required by your client.  
4. Use the connected agent to invoke Modern Treasury payment, reconciliation, and cash management workflows.
