# Fillout MCP Server

An MCP Server integration for Fillout that lets MCP clients create and manage forms, surveys, and quizzes within MCP-compatible environments.

- **Website / Source**: https://mcp.pipedream.com/app/fillout
- **Category**: Business & Commerce · MCP Servers
- **Tags**: forms, surveys, quizzes

## Features

- **MCP-compatible server**
  - Exposes Fillout functionality as an MCP Server that can be added to any MCP-compatible client.
  - Uses a single static MCP server URL for all clients: `https://mcp.pipedream.net/v2`.

- **Form, survey, and quiz management**
  - Create and manage forms, surveys, and quizzes through MCP clients.
  - Designed for building structured data-collection experiences your audience can complete.

- **Client-agnostic setup**
  - The same server URL works across different chat or MCP clients.
  - Authentication is handled when you add the server to your chosen application.

- **Configuration guidance**
  - Setup and connection details are available via the platform’s configuration page (linked from the app page).

## Integration & Usage

- **Server URL**: `https://mcp.pipedream.net/v2`
- **How to use**:
  - Add this MCP server URL to your MCP-compatible chat client or application.
  - Authenticate when prompted by your client during setup.
  - Use the exposed tools/endpoints to create and manage Fillout forms, surveys, and quizzes.
