## Features

- Integrates Pulsetic website uptime and availability monitoring into MCP-compatible clients and tools.
- Uses a single static MCP server URL for all clients: `https://mcp.pipedream.net/v2`.
- Authentication occurs when adding the server to your chosen MCP-compatible application.
- Usable across multiple chat or MCP client applications (configuration is client-specific).
- Hosted and powered by Pipedream Connect infrastructure.

---

## Integration & Setup

- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- Add the URL as an MCP server in your preferred chat/MCP client.
- Follow client-specific instructions (via the referenced configuration guide) to authenticate and complete setup.
