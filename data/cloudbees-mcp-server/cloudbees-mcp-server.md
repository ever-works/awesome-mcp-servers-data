# CloudBees MCP Server

## Overview
CloudBees MCP Server is an MCP endpoint that connects CloudBees’ software delivery platform with MCP-compatible clients. It supports MCP-based automation and governance workflows across CI/CD pipelines and DevOps processes.

- **Type:** MCP Server
- **Category:** Cloud / DevOps MCP Servers
- **Vendor / Brand:** CloudBees
- **MCP Base URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible endpoint** for integrating CloudBees with MCP-capable chat or automation clients.
- **Static server URL** (`https://mcp.pipedream.net/v2`) usable across different clients.
- **Supports software delivery workflows**, enabling automation and governance across CI/CD and DevOps processes.
- **Designed for end-to-end software delivery**, aligning with CloudBees’ platform focus on balancing governance and developer autonomy.

## Integration & Usage
- Add the MCP server URL to any supported MCP client or application.
- Authentication is handled when you connect the server within your app.
- Additional configuration guidance is available via the platform’s configuration documentation (not specific to this item in the provided content).
