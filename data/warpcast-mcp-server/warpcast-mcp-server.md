# Warpcast MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Warpcast  
**Source URL:** https://mcp.pipedream.com/app/warpcast

Warpcast MCP Server is an MCP (Model Context Protocol) server that connects to Warpcast, a client for the Farcaster social protocol, enabling MCP-compatible tools and chat clients to interact with Farcaster social data.

## Features

- Connects MCP tools to Warpcast, a Farcaster social client
- Enables interaction with Farcaster social data via MCP-compatible applications
- Uses a single static MCP server URL for all clients:
  - `https://mcp.pipedream.net/v2`
- Authentication occurs when adding the server to an MCP-compatible application
- Can be added to various chat clients that support MCP
- Configuration details available via a dedicated configuration page ("Configuration" documentation)

## Technical Details

- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Integration Type:** MCP server for social media / web3 messaging (Farcaster)
