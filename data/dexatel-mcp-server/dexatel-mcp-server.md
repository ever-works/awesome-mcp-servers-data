# Dexatel MCP Server

## Overview
Dexatel MCP Server is an MCP (Model Context Protocol) server integration that exposes Dexatelâ€™s programmable business text messaging capabilities through MCP endpoints. It enables applications and chat clients to send and manage business SMS via a standardized MCP server URL.

- **Category:** Messaging MCP Servers  
- **Use cases:** Business SMS, programmable text messaging, API-based messaging integration  
- **Base MCP server URL:** `https://mcp.pipedream.net/v2`

## Features
- **Programmable business text messaging** via Dexatel, accessible through MCP endpoints.
- **Static MCP server URL** (`https://mcp.pipedream.net/v2`) that works for all clients; authentication is handled when adding the server to your application.
- **Account-based configuration:** Connect a Dexatel account and select the appropriate client to start using messaging features.
- **Chat client integration:** Designed to be added as a server to various chat clients that support MCP.
- **Central configuration reference:** Additional setup details are available via a dedicated configuration page on the host platform (Pipedream MCP configuration).

## Integration
- Add the MCP server URL to your MCP-compatible application or chat client.
- Authenticate with Dexatel when prompted during server addition.
- Use the exposed MCP tools/actions (business SMS operations) once loading is complete in the client.
