# Statuscake MCP Server

**Category:** Monitoring  
**Brand:** StatusCake  
**Source:** https://mcp.pipedream.com/app/statuscake

## Description
Statuscake MCP Server is a Model Context Protocol (MCP) server that lets AI agents interact with StatusCakeâ€™s monitoring services. It enables programmatic access to website uptime, SSL, page speed, domain, and related monitoring data through MCP-compatible clients.

## Features
- **MCP server endpoint**  
  - Static MCP server URL usable from any compatible client:  
    - `https://mcp.pipedream.net/v2`
  - Authentication handled when adding the server within the client / application.

- **Website uptime monitoring integration**  
  - Access uptime monitoring data for websites via AI agents.

- **SSL monitoring integration**  
  - Interact with SSL certificate monitoring (e.g., for status, expiry) through MCP.

- **Page speed monitoring integration**  
  - Access and work with page speed monitoring metrics programmatically.

- **Domain monitoring integration**  
  - Integrate domain monitoring (e.g., status, expiry tracking) into AI workflows.

- **Website / server monitoring integration**  
  - Support for general website and server health monitoring via StatusCake.

- **Client-agnostic setup**  
  - Single static MCP URL works across different chat / MCP clients.  
  - Configuration instructions available per client type (via the linked configuration page).
