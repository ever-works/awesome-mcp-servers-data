## Slack

**Description**  
An MCP server that integrates with Slack for channel management and messaging, enabling LLM-driven collaboration, notifications, and chat automation.

**Category**  
MCP server directories & lists

**Source**  
[GitHub repository](https://github.com/madhukarkumar/anthropic-mcp-servers/blob/main/src/slack)

### Features
- Slack integration via MCP
- Channel management capabilities
- Messaging support between the LLM and Slack
- LLM-driven collaboration workflows inside Slack
- Automated notifications to Slack channels or users
- Chat automation powered by LLMs through the MCP server
