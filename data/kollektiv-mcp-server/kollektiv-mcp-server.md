# Kollektiv MCP Server

## Overview
Kollektiv MCP Server is an MCP (Model Context Protocol) server that connects AI clients to documentation-related functionality. It exposes an SSE (Server-Sent Events) endpoint and uses OAuth 2.1 for authentication, enabling secure, real-time interactions with documentation from compatible MCP clients and IDEs.

- **Type:** MCP server (AI integration)
- **Category:** AI Integration – MCP Servers
- **Primary use case:** Documentation access and querying via AI clients
- **Brand:** kollektiv
- **Endpoint:** `https://mcp.thekollektiv.ai/sse`
- **Auth:** OAuth 2.1

## Features
- **MCP-compatible server** – Implements the Model Context Protocol so it can be used by MCP-aware clients and IDE integrations.
- **Documentation-focused functionality** – Designed for documentation-related workflows, enabling AI clients to access and work with docs content.
- **SSE endpoint** – Uses a Server-Sent Events endpoint (`/sse`) for streaming responses between the MCP server and clients.
- **OAuth 2.1 authentication** – Secures access to the server with OAuth 2.1, suitable for protected documentation and multi-user environments.
- **AI client integration** – Built to connect with AI tools and editors that support MCP, enabling chat and query experiences over documentation.
