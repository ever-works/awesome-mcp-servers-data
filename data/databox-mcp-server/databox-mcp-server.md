## Overview
Databox MCP Server is an MCP-compatible service (powered by Pipedream Connect) that lets MCP-based agents and workflows connect to Databox. It centralizes performance data from multiple tools so you can track, report on, and monitor metrics across devices without logging into each individual platform.

## Features
- **MCP-compatible server**: Exposes Databox as an MCP Server that can be added to any MCP-capable client or application.
- **Unified performance tracking**: Connects data from multiple tools into a single place for monitoring performance.
- **Cross-device access**: Track and view performance data from various devices.
- **Reporting and dashboards**: Build and share reports based on Databox data sourced through MCP.
- **Trend monitoring**: Monitor trends in KPIs and other performance metrics across connected tools.
- **Insight discovery**: Use centralized data access to help discover insights across tools and workflows.
- **Static MCP server URL**: Uses a single, static endpoint that works across clients:  
  `https://mcp.pipedream.net/v2`
- **Per-client authentication**: Authentication is performed when adding the server to each application or chat client.
- **Client-agnostic configuration**: Can be integrated with multiple chat clients or MCP-based applications via configuration instructions.

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Hosting / provider:** Powered by Pipedream Connect
