# LiveAgent MCP Server

**Category:** Business & Commerce – MCP Servers  
**Tags:** helpdesk, customer-support

## Overview
LiveAgent MCP Server is an MCP (Model Context Protocol) server integration for LiveAgent, a web-based live chat and helpdesk platform. It provides unified MCP access to LiveAgent’s multi-channel support environment, including email, live chat, call center, contact forms, feedback forms, knowledge base, and selected social networks.

## Features
- **Unified MCP integration**
  - Exposes LiveAgent functionality through a single MCP server endpoint.
  - Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`.
- **Multi-channel support access**
  - Email-based ticketing and support.
  - Live chat integration.
  - Call center capabilities.
  - Contact forms and feedback forms.
  - Knowledge base integration.
  - Social network channels: Twitter and Facebook.
- **Centralized environment**
  - Aggregates all supported channels into one environment accessible via MCP.
- **Client-agnostic setup**
  - Same MCP server URL works with different chat / MCP-compatible clients.
  - Authentication handled when adding the server to the client application.

## Setup
- Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server to your MCP-compatible chat client or application.
- Refer to the provider’s Configuration page (linked from the source) for client-specific steps.
