# Yotpo Loyalty & Referrals MCP Server

## Overview
The **Yotpo Loyalty & Referrals MCP Server** is an MCP (Model Context Protocol) server integration for Yotpo’s Loyalty & Referrals product. It provides a static MCP endpoint you can add to compatible chat or AI applications to enable programmatic access to Yotpo loyalty and referral data and related functions.

- **Category:** Business & Commerce MCP Servers  
- **Tags:** loyalty, referrals, e-commerce  
- **Provider / Platform:** Yotpo, delivered via Pipedream Connect  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server endpoint**  
  - Single static URL (`https://mcp.pipedream.net/v2`) used by all clients.  
  - Can be added to any MCP-capable chat or AI application.

- **Access to Yotpo Loyalty & Referrals product**  
  - Designed specifically for Yotpo’s Loyalty & Referrals offering within the Yotpo eCommerce marketing platform.  
  - Enables programmatic interaction with loyalty and referral data and functions (via MCP) once authentication is configured in the client.

- **Client-agnostic setup**  
  - same server URL for every client; authentication is handled when configuring the server in each application.  
  - Documentation and configuration guidance available via the Pipedream “Configuration” page.

- **Integration via Pipedream Connect**  
  - Managed and hosted by Pipedream’s Connect infrastructure.  
  - Can be combined with other Pipedream-powered integrations within MCP-compatible environments.

## Authentication
- Authentication is performed when adding the MCP server to your application.  
- The server URL is static; credentials and configuration are handled at the client side (details provided in the respective client’s setup instructions or the general configuration page).
