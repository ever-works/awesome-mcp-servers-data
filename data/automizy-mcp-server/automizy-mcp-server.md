# Automizy MCP Server

Email marketing automation MCP server powered by Pipedream Connect, enabling MCP-compatible systems to work with Automizy.

---

## Basic Information

- **Name:** Automizy MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Use Case:** Orchestrate email marketing automation workflows via MCP-compatible clients and Pipedream  
- **Source URL:** https://mcp.pipedream.com/app/automizy  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  

---

## Features

- **MCP-Compatible Endpoint**  
  - Provides a single static MCP server URL: `https://mcp.pipedream.net/v2`.  
  - Works across different MCP-compatible chat clients and applications.  

- **Centralized Authentication Flow**  
  - Authentication occurs when adding the server inside the client/application, not by changing the server URL.  

- **Client-Agnostic Integration**  
  - Designed to be added to a variety of chat clients that support MCP servers.  
  - Configuration guidance available via a dedicated configuration page.  

- **Pipedream Connect Backend**  
  - Backed by Pipedream Connect for workflow and integration handling.  
  - Enables orchestration of email marketing automation workflows connected to Automizy.  

- **Config Documentation**  
  - Linked “Configuration” page provides step-by-step instructions for adding the server to supported clients (exact steps depend on the client).

---

## Integration / Usage Overview

1. Use the static MCP server URL `https://mcp.pipedream.net/v2`.  
2. Add this server URL in your MCP-compatible chat client or application.  
3. Authenticate within the client when prompted.  
4. Configure and run workflows that connect to Automizy email marketing automation through Pipedream.
