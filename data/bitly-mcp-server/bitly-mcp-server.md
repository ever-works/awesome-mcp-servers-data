# Bitly MCP Server

An MCP server that connects Bitly’s link management platform to MCP-compatible AI tools.

## Overview
- **Name:** Bitly MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Brand:** Bitly  
- **Website:** https://mcp.pipedream.com/app/bitly  
- **Slug:** `bitly-mcp-server`

## Features
- Connects a Bitly account to AI tools via Pipedream’s MCP server.  
- Compatible with MCP-enabled tools such as Cursor, Claude, and Windsurf.  
- Enables AI tools to interact with Bitly’s link management platform (e.g., shorten and manage URLs, per item description).

## Use Cases
- Letting AI-powered development environments (e.g., Cursor, Windsurf) automatically create and manage short links.  
- Allowing conversational AI tools (e.g., Claude) to generate and organize Bitly links as part of workflows.
