# Welcome MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Welcome  
**Slug:** `welcome-mcp-server`

## Description
The Welcome MCP Server connects MCP-compatible chat clients and workflows to the Welcome webinar platform. It enables applications to create and manage webinars used for marketing and event workflows via a standardized MCP server endpoint.

## Features
- **MCP-compatible server endpoint**
  - Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`.
  - Can be added to any compatible chat client or application that supports MCP.
- **Integration with Welcome webinar software**
  - Designed specifically to work with Welcome, a webinar platform for marketers.
  - Intended for workflows that create and manage webinars for marketing and event use cases.
- **Central configuration**
  - Server is configured once via the static URL; authentication occurs when adding it to an app.
  - Additional configuration guidance available via the referenced configuration documentation.
- **Pipedream Connect infrastructure**
  - Hosted and powered by Pipedream Connect, which provides the MCP server runtime.

## Authentication
- Authentication is performed when adding the MCP server to your application or chat client (exact method depends on the client’s MCP configuration flow).

## Usage
- Add the MCP server URL `https://mcp.pipedream.net/v2` to an MCP-compatible chat client or application.
- Follow your client’s MCP configuration guide or the linked configuration page to complete setup.
- Once configured, use the server within MCP workflows to interact with the Welcome webinar platform (e.g., for marketing webinars and events).
