# LLM Text MCP Server

**Brand:** llm-text  
**Category:** Data Analysis & Exploration MCP Servers  
**Tags:** text-analysis, data-analysis, ai-insights

A Model Context Protocol server for LLM Text that exposes tools to LLM-based agents over SSE. It focuses on discovering and retrieving `llms.txt` information from websites and providing text-oriented utilities.

![LLM Text MCP Server](https://llmtxt.dev/og-image.png)

---

## Features

### General
- Model Context Protocol (MCP) server for LLM Text.
- Communicates with LLM-based agents over Server-Sent Events (SSE).
- Designed to discover and retrieve `llms.txt` files from websites.
- Provides text utilities suitable for analysis and summarization workflows.
- Distributed as a Docker MCP server, integrable via Docker Desktop.

### Tools

#### `git`
- Converts any Git repository into a simple text digest of its codebase.
- Enables LLMs to consume repository content in a more linear text form (e.g., for analysis, search, or summarization).

### Integration & Runtime
- Available in the Docker MCP catalog under **AI Tools**.
- Can be added to Docker Desktop directly from Docker Hub.
- Requires **Docker Desktop version 4.43 or later** for automatic server addition and configuration.
