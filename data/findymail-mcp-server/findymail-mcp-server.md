# Findymail MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Findymail  
**Source:** https://mcp.pipedream.com/app/findymail

## Overview
The Findymail MCP Server is a Model Context Protocol (MCP) server that connects MCP-compatible applications and workflows to Findymail’s B2B email-finding and list-management capabilities. It is designed to support sales, outreach, and lead-generation automations by exposing Findymail actions as tools within chat or agent clients.

## Features

### MCP Server & Integration
- **Static MCP server URL:**
  - Endpoint: `https://mcp.pipedream.net/v2`
  - Works for every client; authentication is handled when adding the server to the application.
- **Client-agnostic setup:**
  - Can be added to various MCP-compatible chat or agent clients.
  - Configuration guidance available via a dedicated configuration page.
- **Account connection:**
  - Connect a Findymail account within Pipedream to access tools.

### Available Tools / Actions (6 total)
1. **Verify Email**  
   - Verifies the deliverability of a specified email address.  
   - Intended for checking if an email is likely valid and reachable before outreach.

2. **List Contact Lists**  
   - Retrieves a list of all contact lists in a connected Findymail account.  
   - Useful for exploring, selecting, or managing existing lists for campaigns.

3. **Find Email**  
   - Finds an email address using a given full name and a website domain.  
   - Suited for B2B lead generation when both the person’s name and company domain are known.

4. **Find Email by Company Domain**  
   - Locates an email address using a company’s website domain.  
   - Helpful when targeting contacts at a specific company and using domain-based discovery.

5. **Delete a List**  
   - Deletes a specified list in Findymail.  
   - Supports cleanup and maintenance of outdated or unused contact lists.

6. **Create List**  
   - Creates a new list of contacts in Findymail.  
   - Enables organizing prospects or leads into dedicated lists for specific campaigns or workflows.
