# BetterVoice MCP Server

MCP Server for BetterVoice, exposing its cloud-based business phone system features to MCP-compatible applications for calling and communication workflows.

- **Website / Source**: https://mcp.pipedream.com/app/bettervoice
- **Category**: Messaging MCP Servers
- **Tags**: phone, VoIP, communication
- **MCP Server URL**: `https://mcp.pipedream.net/v2`

## Features

- Cloud-based, business-class phone system integration via MCP.
- No hardware required; operates entirely over the cloud.
- Exposes BetterVoice calling and communication capabilities to any MCP-compatible client or application.
- Single static MCP server URL (`https://mcp.pipedream.net/v2`) usable across all supported clients.
- Authentication handled when adding the server to your MCP-compatible application or chat client.
- Built and hosted via Pipedream Connect infrastructure.

## Integration

- Add the MCP server URL to supported chat clients or applications to enable BetterVoice functionality.
- Central configuration and setup details available through the Pipedream MCP configuration flow (per the “Configuration page”).
