# AttractWell MCP Server

**Category:** Business & Commerce – MCP Servers  
**Website:** https://mcp.pipedream.com/app/attractwell  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Overview
AttractWell MCP Server exposes AttractWell’s business tools—such as CRM, funnels, courses, payments, and marketing automation—to MCP-based agents and workflows. It is provided via Pipedream Connect as a static MCP endpoint that you authenticate against from compatible chat clients or applications.

## Features
- **MCP integration**
  - Static MCP server URL usable across clients: `https://mcp.pipedream.net/v2`
  - Connects to MCP-based agents and workflows via Pipedream Connect.
- **CRM access**
  - Surfaces AttractWell CRM capabilities for managing contacts and follow-up within MCP workflows.
- **Funnels & website tools**
  - Access to funnels and website / blog-related capabilities for lead capture and nurturing.
- **Courses**
  - Interfaces with AttractWell’s course features (e.g., online programs, trainings) through MCP.
- **Payments**
  - Exposes payment-related functionality (e.g., taking payments for services, products, or courses).
- **Marketing automation**
  - Email and text (SMS) marketing capabilities accessible to MCP agents.
  - Follow-up system and automation workflows for leads and clients.
- **Integrations**
  - Designed for coaching and similar online businesses using tools like Zoom and other connected services within AttractWell.
- **Client setup**
  - Works with multiple chat clients; configuration guidance available via Pipedream’s configuration documentation.

## Usage
- Add `https://mcp.pipedream.net/v2` as an MCP server in your compatible chat client or application.
- Authenticate AttractWell within your client following Pipedream’s configuration instructions.
