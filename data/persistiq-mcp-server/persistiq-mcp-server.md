# PersistIQ MCP Server

PersistIQ MCP Server connects the PersistIQ email automation platform to the Model Context Protocol (MCP), enabling agents and MCP-compatible clients to work with personalized email outreach.

## Overview

- **Type:** MCP Server (integration)
- **Platform:** Pipedream Connect
- **Category:** Business & Commerce – MCP Servers
- **Primary use case:** Automating and managing small batches of personalized emails via MCP-compatible applications using PersistIQ.

## Features

- **Email automation integration**
  - Connects MCP clients to PersistIQ’s email automation platform.
  - Designed for sending and managing small batches of personalized emails.

- **Standard MCP endpoint**
  - Single static MCP server URL for all clients:
    - `https://mcp.pipedream.net/v2`
  - Authentication handled when adding the server to your MCP-compatible application.

- **Client-agnostic setup**
  - Same server URL works across different chat or MCP clients.
  - Configuration instructions available per client via the Pipedream configuration docs.

- **Pipedream-hosted**
  - Operated and provided through Pipedream Connect.
  - Uses Pipedream’s infrastructure, terms of use, and privacy policy.

## Configuration

- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- Add this URL to your MCP-compatible app, then authenticate with your Pipedream / PersistIQ connection as prompted.
- Detailed, client-specific configuration steps are available on the Pipedream **Configuration** page.
