## Description
The Odds API MCP Server exposes The Odds API sports betting odds data within MCP-compatible tools. It provides programmatic access to odds from global bookmakers for major sports leagues such as the NFL, NBA, and English Premier League (EPL), via a simple API endpoint.

## Features
- Access to sports betting odds data from global bookmakers
- Coverage of major leagues including:
  - NFL
  - NBA
  - EPL (English Premier League)
  - Additional leagues as supported by The Odds API
- Simple, static MCP server URL usable across clients: `https://mcp.pipedream.net/v2`
- Works with multiple MCP-based chat or agent clients
- Authentication handled when adding the server to an application

## Integration Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- Addable to compatible chat / agent applications as an MCP server
- Configuration and setup instructions available via the Pipedream configuration page
