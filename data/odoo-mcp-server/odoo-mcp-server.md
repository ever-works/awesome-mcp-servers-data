# Odoo MCP Server

Integrated MCP server that exposes Odoo’s ERP and CRM platform capabilities to MCP agents for enterprise workflow automation.

- **Category:** Business & Commerce MCP Servers  
- **Vendor / Brand:** Odoo  
- **Slug:** `odoo-mcp-server`  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features

- Provides an MCP-compatible interface to Odoo’s integrated ERP and CRM platform.
- Designed for enterprise workflow automation using Odoo data and processes.
- Uses a **single static MCP server URL** (`https://mcp.pipedream.net/v2`) for all clients.
- Authentication is handled when adding the server in the client/application (per-client auth, shared endpoint).
- Compatible with MCP-enabled chat or agent clients (can be added as an MCP server within supported apps).
- Operates as part of the Pipedream Connect MCP infrastructure.

## Usage

1. Use the static MCP server URL: `https://mcp.pipedream.net/v2` when configuring the server in your MCP-compatible client.  
2. Complete authentication within your client when prompted.  
3. Once connected, MCP agents can interact with Odoo’s ERP/CRM features via the MCP interface.
