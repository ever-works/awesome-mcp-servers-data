# Wildberries MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Wildberries  
**Source:** https://mcp.pipedream.com/app/wildberries

Wildberries MCP Server is an MCP (Model Context Protocol) integration that exposes Wildberries’ ecommerce capabilities—such as retail and order-related operations—through a standardized MCP endpoint.

---

## Features

- **MCP-driven access to Wildberries:** Connects applications to Wildberries, Russia’s largest online retailer, via the MCP standard.
- **Static MCP endpoint URL:**
  - Server URL: `https://mcp.pipedream.net/v2`
  - Same URL is used for all clients; authentication is handled when you add the server to your MCP-compatible application.
- **Client-agnostic integration:** Designed to work with multiple MCP-capable chat or AI clients; you select your client and configure it using the same base URL.
- **Ecommerce focus:** Built for ecommerce workflows involving:
  - Retail data and interactions
  - Order-related operations (e.g., retrieving or working with orders in Wildberries)
- **Hosted by Pipedream:** Infrastructure and access managed through Pipedream’s MCP platform.

---

## Configuration

- Add the server to any MCP-compatible application using the static URL: `https://mcp.pipedream.net/v2`.
- Authentication occurs during the “add server” process in your client.
- Additional configuration details are available on the provider’s configuration page (linked from the source URL).
