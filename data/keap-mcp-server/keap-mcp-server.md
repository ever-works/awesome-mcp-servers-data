# Keap MCP Server

CRM, sales & marketing automation MCP server integration for Keap (Infusionsoft).

- **Name:** Keap MCP Server  
- **Category:** Business & Commerce · MCP Servers  
- **Brand:** Keap  
- **Source URL:** https://mcp.pipedream.com/app/infusionsoft  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Overview

Keap MCP Server is an MCP (Model Context Protocol) server integration for Keap (Infusionsoft). It allows MCP-compatible tools and chat clients to interact programmatically with Keap’s CRM, sales, and marketing automation platform.

## Features

- **MCP server endpoint**  
  - Provides a static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same endpoint is used for all clients; authentication occurs when adding the server to the application.

- **CRM, sales & marketing integration**  
  - Designed to connect MCP tools and chat clients with Keap’s CRM, sales, and marketing automation capabilities (exact API actions are not listed in the provided content).

- **Chat client compatibility**  
  - Can be added to different chat clients that support MCP servers.  
  - Setup instructions are available per chat client via the configuration documentation.

- **Central configuration resources**  
  - Full configuration details are accessible through a dedicated configuration page (linked from the source content).
