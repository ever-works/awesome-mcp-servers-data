# Virifi MCP Server

**Category:** Blockchain & Crypto MCP Servers  
**Brand:** Virifi  
**Website / Source:** https://mcp.pipedream.com/app/virifi

## Description
Virifi MCP Server is an MCP-compatible server that allows MCP clients to certify and protect documents on the blockchain using Virifi’s APIs.

## Features
- MCP server endpoint for Virifi: `https://mcp.pipedream.net/v2` (static URL for all clients)
- Supports document certification on the blockchain
- Supports document protection via Virifi’s blockchain APIs
- Works with multiple MCP-compatible chat or agent clients
- Authentication handled when adding the server to an MCP-compatible application
- Central configuration guidance available via a shared Configuration page on Pipedream

## Integration & Usage
- Add the static MCP server URL to any supported MCP client or chat application
- Authenticate within the client when configuring the server connection
- Refer to the Pipedream Configuration page for client-specific setup instructions
