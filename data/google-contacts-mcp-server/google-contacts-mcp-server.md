# Google Contacts MCP Server

**Description**

Google Contacts MCP Server provides an MCP-compatible integration with Google Contacts, backed by the Google People API. It allows applications using the Model Context Protocol (MCP) to access and manage Google contacts data.

**Features**
- MCP server integration for Google Contacts
- Backed by the Google People API for contact data
- Static MCP server URL usable across compatible clients: `https://mcp.pipedream.net/v2`
- Authentication performed when adding/configuring the server in the client application
- Compatible with multiple MCP-enabled chat or agent clients (configured per client)

**Technical Details**
- Server URL: `https://mcp.pipedream.net/v2`
- Provider: Pipedream Connect

**Use Cases**
- Accessing and managing Google Contacts via MCP-enabled applications
- Integrating Google Contacts data into AI/chat clients using MCP
