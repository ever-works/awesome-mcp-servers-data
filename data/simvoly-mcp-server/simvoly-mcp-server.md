# Simvoly MCP Server

**Description**  
Simvoly MCP Server exposes Simvoly’s website and funnel builder platform capabilities (site and funnel management) to MCP-compatible agents and tools via Pipedream Connect.

**Server URL**  
Use this static MCP server URL with any compatible client:

```text
https://mcp.pipedream.net/v2
```

You authenticate when adding the server to your application.

## Features
- MCP-compatible endpoint for Simvoly website and funnel management.
- Single static server URL usable across different MCP clients.
- Integrates with Simvoly’s website and funnel builder platform (for sites, funnels, and related content operations) via Pipedream Connect.
- Can be attached to various chat or agent clients that support MCP.
- Central configuration managed through Pipedream’s MCP configuration system.

## Integration & Setup
- Add the server to any MCP-capable app using the URL: `https://mcp.pipedream.net/v2`.
- Follow client-specific instructions (from your chat or agent client) to configure the MCP server and authenticate.
- Additional configuration details are available on Pipedream’s MCP configuration page.
