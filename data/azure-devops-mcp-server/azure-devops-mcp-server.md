## Azure DevOps MCP Server

**Category:** Development Tools – MCP Servers  
**Brand:** Microsoft Azure DevOps  
**Source:** https://mcp.pipedream.com/app/azure_devops

An MCP server that connects to Azure DevOps, enabling planning, collaboration, and CI/CD operations from MCP‑compatible agents or tools.

---

### Features

- **Azure DevOps integration**: Connects MCP-compatible clients to Azure DevOps services for planning, collaboration, and CI/CD workflows.
- **Static MCP server URL**: Uses a single, static endpoint that works across all supported clients:
  - `https://mcp.pipedream.net/v2`
- **Client-agnostic setup**: Designed to be added to various chat or agent clients; authentication occurs when adding the server to the application.
- **Configuration documentation**: A dedicated configuration page is available for detailed setup and usage instructions.
- **Powered by Pipedream Connect**: Runs on Pipedream’s infrastructure for MCP server hosting and connectivity.
