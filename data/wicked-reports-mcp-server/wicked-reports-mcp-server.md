## Wicked Reports MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Wicked Reports  
**URL:** https://mcp.pipedream.com/app/wicked_reports

### Description
Wicked Reports MCP Server connects a Wicked Reports account to MCP-compatible AI tools, enabling access to sales attribution and marketing analytics data within AI-powered workflows.

### Features
- Connects a Wicked Reports account to Pipedream’s MCP server
- Enables integration with AI tools such as Cursor, Claude, and Windsurf
- Exposes Wicked Reports’ advanced sales attribution capabilities to MCP-based tools
- Provides access to marketing and ecommerce analytics data for use in AI workflows
- Designed for marketing and ecommerce use cases that require attribution insights inside AI assistants or development environments

### Integrations
- Pipedream MCP
- Cursor
- Claude
- Windsurf
