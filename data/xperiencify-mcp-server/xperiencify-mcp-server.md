## Overview
Xperiencify MCP Server is a Model Context Protocol (MCP) server that connects AI agents with the Xperiencify gamified online course platform. It enables agents to help design and manage highly engaging, gamified learning experiences within Xperiencify.

The server is provided via Pipedream Connect and is accessible through a static MCP endpoint that can be used by compatible chat or agent clients.

## Features
- **MCP-compatible server**
  - Exposes Xperiencify functionality through the Model Context Protocol.
  - Can be added as a server in any MCP-capable client or AI agent framework.

- **Static MCP endpoint**
  - Single, reusable URL for all clients:  
    `https://mcp.pipedream.net/v2`
  - Authentication handled when adding the server to your application.

- **Integration with Xperiencify platform**
  - Connects AI agents to the Xperiencify gamified online course platform.
  - Supports use cases around designing and managing engaging, gamified learning experiences for students.

- **Client-agnostic usage**
  - Works with multiple chat / agent clients that support MCP.
  - Configuration details available via a shared configuration page on Pipedream.

## Usage
- Add the MCP server URL `https://mcp.pipedream.net/v2` to your MCP-compatible client.
- Authenticate within your client during setup.
- Use the server to enable AI agents to interact with and orchestrate Xperiencify course experiences.
