# Teach 'n Go MCP Server

## Overview
Teach 'n Go MCP Server is an MCP server that connects the Teach 'n Go school management platform to AI tools, enabling automated interaction with scheduling, student data, and school operations.

## Features
- Integrates Teach 'n Go with AI tools such as Cursor, Claude, and Windsurf via Pipedream's MCP server
- Enables automated access to Teach 'n Go scheduling data
- Supports interaction with student information and school operations data through AI workflows
- Allows AI-assisted workflows for school management tasks using the MCP protocol

## Integrations
- Teach 'n Go school management platform
- Pipedream MCP server
- AI tools:
  - Cursor
  - Claude
  - Windsurf
