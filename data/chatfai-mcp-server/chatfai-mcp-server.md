# ChatFAI MCP Server

An MCP server that connects to ChatFAI, enabling AI-powered character chat experiences with personalities from movies, TV shows, books, history, and more via standardized MCP tools.

## Features
- AI-powered character chat experiences
- Access to characters from:
  - Movies
  - TV shows
  - Books
  - History
  - Other fictional and real-world contexts
- Exposed as a standard MCP (Model Context Protocol) server
- Single static MCP server URL usable across compatible clients
- Works with multiple chat clients that support MCP

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- Authentication occurs when adding the server to your MCP-compatible application
- Configuration and client-specific setup instructions available via the provider’s configuration documentation
