# Vosfactures MCP Server

Online invoicing MCP server integration for businesses and entrepreneurs, enabling chat-based interaction with Vosfacturesâ€™ invoicing and quoting capabilities.

## Overview
- **Type**: MCP server (Model Context Protocol server)
- **Purpose**: Connects chat-based clients to Vosfactures online invoicing features
- **Use Cases**:
  - Create and manage invoices
  - Create and manage quotes
  - Handle basic billing operations for small businesses and entrepreneurs

## Features
- **MCP-based integration**
  - Exposes Vosfactures invoicing and billing functions via a standard MCP server endpoint
  - Works with any compatible MCP client / chat interface
- **Online invoicing**
  - Create custom invoices
  - Manage existing invoices
- **Quotes / estimates**
  - Create and manage quotes for customers
- **Billing for small businesses**
  - Designed for businesses and entrepreneurs
  - Focus on simple, fast, multifunctional invoicing and billing workflows

## Technical Details
- **MCP Server URL**: `https://mcp.pipedream.net/v2`
  - Static URL shared across clients
  - Authentication handled when adding the server to the application
- **Integration context**
  - Powered by Pipedream Connect
  - Can be added to supported chat clients via their MCP configuration
