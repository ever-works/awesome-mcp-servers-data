# Specific MCP Server

Sales call analytics MCP server integrating Specific’s platform to provide call insights and revenue analytics to AI agents.

## Overview
- **Type:** MCP Server (Model Context Protocol)
- **Category:** Business & Commerce MCP Servers
- **Provider / Brand:** Specific (via Pipedream Connect)
- **Primary Use Case:** Enable AI agents and applications to access sales call analytics and revenue-related insights.
- **MCP Endpoint URL:** `https://mcp.pipedream.net/v2`

## Features
- **Sales Call Analytics Integration**  
  - Connects AI agents to Specific’s sales call analytics platform.  
  - Designed to surface insights from sales conversations for downstream tools and agents.

- **Revenue & Conversation Insights**  
  - Provides access to call insights that can be used to analyze revenue performance and sales activity.  
  *(Underlying metrics and specific analytics dimensions are not detailed in the provided content.)*

- **Standard MCP Server Endpoint**  
  - Uses a single static MCP URL (`https://mcp.pipedream.net/v2`) that works for all clients.  
  - Authentication is performed when adding the server to an MCP-compatible application.

- **Client-Agnostic Setup**  
  - Intended to be added to various MCP-compatible chat clients or applications.  
  - Configuration guidance available through a general configuration page on Pipedream (not specific to this server in the provided content).

- **Pipedream Connect Integration**  
  - Operates on top of Pipedream Connect infrastructure.  
  - Benefits from Pipedream’s unified way of connecting external services into MCP.

## Authentication
- Uses authentication at the time you add the MCP server to your app or chat client.  
- The same static URL is used for all clients; credentials / tokens are supplied during configuration (details not specified in the provided content).

## Compatibility & Integration
- **Protocol:** MCP (Model Context Protocol).  
- **Intended Consumers:**
  - MCP-compatible chat clients.  
  - AI applications that can register MCP servers as external tools.
