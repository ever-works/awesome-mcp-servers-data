# Passcreator MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Passcreator  
**Source:** https://mcp.pipedream.com/app/passcreator

## Description
Passcreator MCP Server is an integration that connects a Passcreator account to AI tools via Pipedreamâ€™s Model Context Protocol (MCP). It allows AI agents to manage mobile wallet campaigns that use Passcreator passes, including store cards, event tickets, coupons, and membership cards.

## Features
- **MCP-based integration**: Exposes Passcreator functionality as MCP tools that AI agents can call.  
- **Passcreator account connectivity**: Links an existing Passcreator account to an MCP server running on Pipedream.  
- **AI tool compatibility**: Designed to work with AI developer tools such as Cursor, Claude, and Windsurf.  
- **Mobile wallet campaign management**: Enables AI agents to interact with campaigns built on Passcreator.  
- **Multiple pass types**: Supports passes for:
  - Store cards
  - Event tickets
  - Coupons
  - Membership cards

## Use Cases
- Automating creation and management of mobile wallet campaigns through AI workflows.  
- Letting AI coding assistants (e.g., Cursor, Windsurf) modify or query Passcreator campaigns during development.  
- Integrating Claude or other MCP-compatible AI agents into existing Passcreator-based loyalty, ticketing, or coupon systems.
