# Humanitix MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Humanitix  
**Slug:** `humanitix-mcp-server`

Humanitix MCP Server is an MCP (Model Context Protocol) integration that allows MCP‑compatible tools and chat clients to interact programmatically with Humanitix’s not‑for‑profit event ticketing and booking platform.

---

## Features

- **MCP-compatible server endpoint**  
  - Static MCP server URL for all clients:  
    `https://mcp.pipedream.net/v2`
  - Works across different MCP‑enabled chat applications.

- **Humanitix platform integration**  
  - Designed to connect MCP tools to Humanitix’s event ticketing and booking capabilities.
  - Targets use cases around events, ticketing workflows, and nonprofit/charity‑oriented ticket sales.

- **Client-agnostic configuration**  
  - Same server URL for every MCP client; authentication is handled when adding the server to the application.  
  - Configuration guidance available via a dedicated configuration page (client setup instructions depend on the chosen chat client).

- **Hosted by Pipedream Connect**  
  - MCP server infrastructure and delivery provided through Pipedream Connect.

---

## Technical Details

- **Protocol:** Model Context Protocol (MCP) server.  
- **Base URL:** `https://mcp.pipedream.net/v2`  
- **Authentication:** Performed when the server is added to an MCP‑compatible application (implementation details vary by client).
