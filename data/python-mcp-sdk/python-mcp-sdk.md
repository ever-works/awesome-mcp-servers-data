### Python MCP SDK

**Brand:** model-context-protocol  
**Category:** MCP server directories & lists / SDK  
**Slug:** `python-mcp-sdk`

#### Description
Official Python SDK for implementing Model Context Protocol (MCP) servers and clients. It provides primitives and utilities for defining tools, resources, and capabilities that can be safely exposed to LLMs via the MCP standard.

#### Features
- **Official MCP SDK for Python** – Maintained as the reference Python implementation for the Model Context Protocol.
- **Server implementation support** – Build MCP-compliant servers in Python.
- **Client implementation support** – Interact with MCP servers from Python-based clients.
- **Primitives for tools and resources** – Define tools, resources, and other capabilities in a structured way for MCP.
- **Safety-oriented exposure to LLMs** – Designed for safely exposing tools and resources to language models via the MCP standard.
- **Documentation site** – Hosted documentation at `https://modelcontextprotocol.github.io/python-sdk/`.
- **Examples directory** – Example implementations and usage patterns under the `examples/` folder.
- **Tests suite** – Automated tests in the `tests/` folder to validate SDK behavior.
