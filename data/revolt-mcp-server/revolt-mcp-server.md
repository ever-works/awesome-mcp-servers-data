# Revolt MCP Server

Open-source, user-first chat platform accessible via an MCP (Model Context Protocol) server hosted by Pipedream.

## Features
- Provides an MCP Server integration for the Revolt chat platform.
- Exposes Revolt messaging features to compatible MCP clients.
- Uses a single static MCP server URL that works across all supported clients.
- Authentication is handled when adding/configuring the server in your client or application.
- Can be integrated with different chat clients that support MCP configuration.
- Central configuration and connection management via Pipedream Connect.

## MCP Server URL
- Base MCP server endpoint (static for all clients):
  - `https://mcp.pipedream.net/v2`
