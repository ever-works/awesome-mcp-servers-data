# Formatting MCP Server

## Overview
Formatting MCP Server is a utility-focused MCP server by Pipedream that provides pre-built actions for formatting and manipulating text, numbers, and structured data within MCP-powered workflows.

- **Type:** MCP Server (workflow automation utility)
- **Category:** Workflow Automation MCP Servers
- **Provider / Brand:** Pipedream
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features

### General
- Static MCP server URL usable across compatible MCP clients
- Set of 22 pre-built actions exposed as tools for formatting and data manipulation
- Designed to plug into chat clients and other MCP-enabled applications via configuration

### Text Utilities
- **Encode URL**  
  Encode a string as a URL-safe value.

- **Decode URL**  
  Decode a URL-encoded string back to its original text.

- **Trim Whitespace**  
  Remove leading and trailing whitespace characters from text.

- **Transform Case**  
  Change the case of text input (e.g., upper case, lower case, title case, etc. as supported by the action).

- **Split Text**  
  Split text on a specified character or word and return one or all resulting segments.

- **Set Default Value**  
  Provide and return a default value when the input text is empty.

- **Replace Text**  
  Replace all instances of a specified character, word, or phrase with another in the input text.

- **Extract URL**  
  Find and return the first web URL contained in a text field.

- **Extract Phone Number**  
  Detect and return the first complete phone number in a text field.

- **Extract Number**  
  Find and return the first numeric value contained in a text field.

- **Extract Email Address**  
  Find and return the first email address contained in a text field.

- **Extract by Regular Expression**  
  Find matches for a given regular expression pattern and return all matched groups with their start and end positions.

### Data Utilities
- **Parse JSON**  
  Parse a JSON string into structured data for use in workflows.

### Number & Currency Utilities
- **Format Number**  
  Format a numeric value into a new string style without rounding or padding.

- **Format Currency**  
  Format a number as a currency string (e.g., applying currency symbols and formatting conventions as supported by the action).

> Note: The product page lists a total of 22 actions; only those explicitly described in the available content are enumerated above.
