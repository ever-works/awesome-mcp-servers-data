# Coinbase MCP Server

An MCP server that integrates Coinbase crypto exchange functionality—such as buying, selling, and managing cryptocurrencies—into MCP-based applications.

- **Website / Source**: https://mcp.pipedream.com/app/coinbase
- **Category**: Blockchain & Crypto MCP Servers
- **Brand**: Coinbase
- **Slug**: `coinbase-mcp-server`
- **Tags**: coinbase, cryptocurrency, trading

## Features

- Integrates Coinbase crypto exchange capabilities into MCP-compatible applications.
- Supports exploring a wide range of cryptocurrencies, including Bitcoin, Ethereum, Dogecoin, and many others.
- Enables buying, selling, and managing hundreds of cryptocurrencies through an MCP server interface.
- Provides a static MCP server URL that works across clients: `https://mcp.pipedream.net/v2`.
- Uses client-side authentication when adding the server to your application.
- Compatible with multiple chat or MCP clients (configured per client using the MCP server URL).

## Technical Details

- **MCP Server URL**: `https://mcp.pipedream.net/v2`
- **Authentication**: Performed when adding/configuring the server in your application or chat client.
