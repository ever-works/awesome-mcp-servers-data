# Unbounce MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Unbounce  
**Slug:** `unbounce-mcp-server`

## Description
The Unbounce MCP Server integrates Unbounce’s AI-powered landing page and conversion tools into MCP-based workflows, allowing you to connect your Unbounce account with AI tools via Pipedream’s MCP server.

## Features
- Connects an Unbounce account to MCP-based tools through Pipedream’s MCP server.
- Integrates Unbounce’s AI-powered landing page features into external workflows.
- Enables use of Unbounce conversion tools within AI assistants and IDEs that support MCP (e.g., Cursor, Claude, Windsurf).
- Facilitates automated or semi-automated workflows involving Unbounce data and AI models.

## Integrations
- **Pipedream MCP Server** – acts as the bridge between Unbounce and MCP clients.
- **AI / MCP Clients** (examples given):
  - Cursor
  - Claude
  - Windsurf
