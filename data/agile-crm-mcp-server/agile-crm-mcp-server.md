# Agile CRM MCP Server

**Category:** Business & Commerce MCP Servers  
**Tags:** CRM, sales, marketing-automation

## Overview
The Agile CRM MCP Server integrates Agile CRM with MCP-compatible applications so you can orchestrate sales, marketing, and service automation workflows. It connects to your Agile CRM account via a static MCP server URL and exposes multiple CRM operations as tools.

**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features

### Integration & Configuration
- Connect an Agile CRM account to MCP-compatible chat or automation clients.
- Use a single static MCP server URL for all clients; authentication occurs when adding the server to the application.
- Usable from different chat clients or applications configured for MCP.

### Available Tools (Actions)
The server exposes 13 Agile CRM actions as MCP tools:

#### Deals
- **Create Deal**  
  Create a new deal in Agile CRM.
- **Update Deal**  
  Update an existing deal’s details.
- **Create Deal Note**  
  Add a note to a specific deal.
- **Update Deal Note**  
  Update an existing note attached to a deal.

#### Contacts
- **Create Contact**  
  Create a new contact in Agile CRM.
- **Update Contact**  
  Update an existing contact’s information.
- **Create Contact Note**  
  Add a note to a contact.
- **Add Tags to Contact**  
  Add one or more tags to an existing contact.
- **Add Score to Contact**  
  Change (add or adjust) the score of a contact.

#### Companies
- **Create Company**  
  Create a new company record.
- **Update Company**  
  Update an existing company record.

#### Tasks & Tickets
- **Create Task**  
  Create a new task in Agile CRM.
- **Create Ticket**  
  Create a new ticket in Agile CRM.

Each action is implemented as an MCP tool with further technical details available in the linked source documentation.
