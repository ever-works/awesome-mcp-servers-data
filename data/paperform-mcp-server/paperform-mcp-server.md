# PaperForm MCP Server

**Category:** Business & Commerce – MCP Servers  
**Tags:** forms, surveys, no-code

## Overview
PaperForm MCP Server exposes Paperform’s online form builder capabilities through the Model Context Protocol (MCP), allowing agents and tools to work with forms programmatically.

It connects to the static MCP endpoint operated via Pipedream Connect, so it can be added to any compatible MCP client and then authenticated from within that client.

## Features
- MCP-compatible server for Paperform
- Programmatic access to Paperform’s flexible online form builder capabilities
- Supports working with forms and surveys from agents / tools via MCP
- Single static MCP endpoint usable across different clients
- Authentication handled after adding the server inside the client

## MCP Server Endpoint
- **Server URL:** `https://mcp.pipedream.net/v2`
- **Usage:**
  - Add this URL as an MCP server in your compatible chat or agent client.
  - Authenticate with your Paperform / Pipedream connection when prompted by the client.

## Getting Started
- Add the server URL to your MCP-capable application or chat client.
- Follow the client-specific steps to configure and authenticate the PaperForm MCP Server.
- Optionally refer to the full configuration guide on the provider’s configuration page (Pipedream Connect).
