# Baserow MCP Server

**Category:** Database & Messaging MCP Servers  
**Brand:** Baserow  
**Source:** <https://github.com/baserow/baserow>

## Overview
The Baserow MCP Server is an integration that exposes Baserow databases to AI assistants via the Model Context Protocol (MCP). It enables assistants and tools to discover tables and perform database operations over Baserow’s no‑code, open‑source platform.

## Features
- **MCP server integration**
  - Implements a Model Context Protocol (MCP) server for Baserow.
  - Allows AI assistants to connect to and interact with Baserow workspaces.

- **Database and table access**
  - Connects to Baserow databases and tables.
  - Supports table discovery/listing so assistants can see available data structures.
  - Enables search across Baserow tables.

- **Row‑level CRUD operations**
  - Create new rows in Baserow tables.
  - Read/fetch existing rows.
  - Update existing rows.
  - Delete rows.

- **Automation and AI workflows**
  - Designed to be used by AI agents and assistants to build database‑backed workflows.
  - Integrates with Baserow’s broader no‑code database and automation capabilities.

- **Open‑source platform**
  - Built on the open‑source Baserow project.
  - Can be used with both cloud and self‑hosted Baserow deployments (depending on how the user deploys Baserow itself).
