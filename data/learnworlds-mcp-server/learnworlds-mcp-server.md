# LearnWorlds MCP Server

**Category:** Business & Commerce MCP Servers  
**Tags:** education, course-management, online-store

## Overview
The LearnWorlds MCP Server provides a standard MCP endpoint that lets compatible chat clients integrate with LearnWorlds, an online course platform used to create, sell, and promote courses from your own website.

## MCP Server Details
- **MCP server URL:** `https://mcp.pipedream.net/v2`  
- **URL type:** Static (same for all clients)  
- **Authentication:** Performed when adding the server in your MCP-compatible application or chat client

## Features
- Connects MCP-compatible chat clients to LearnWorlds via a unified MCP endpoint
- Supports workflows related to online course platforms (e.g., creating and selling courses) through the LearnWorlds integration on Pipedream
- Uses a single, static MCP URL for all clients, simplifying configuration
- Can be added to various chat clients that support MCP, following each client’s setup instructions
- Central configuration and management via Pipedream’s configuration page (for advanced setup and integration details)

## Setup & Configuration
- Use `https://mcp.pipedream.net/v2` as the MCP server URL in your client
- Authenticate when prompted by your client during server addition
- Optional: Refer to the Pipedream **Configuration** page for client-specific instructions and advanced options
