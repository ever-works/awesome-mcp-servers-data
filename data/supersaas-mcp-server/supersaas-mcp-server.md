# SuperSaaS MCP Server

**Category:** Business & Commerce • MCP Servers  
**Brand:** Pipedream  
**Slug:** supersaas-mcp-server

## Overview
SuperSaaS MCP Server integrates SuperSaaS’s appointment scheduling capabilities into MCP-compatible environments, enabling scheduling and booking workflows directly from chat clients or other MCP applications.

## Features
- **MCP-compatible appointment scheduling** – Exposes SuperSaaS scheduling and booking capabilities as MCP tools for use in supported chat clients and apps.
- **Static MCP server endpoint** – Uses a single MCP server URL for all clients:
  - `https://mcp.pipedream.net/v2`
- **Per-app authentication** – Authentication occurs when adding the server to an MCP-compatible application, allowing secure access to your SuperSaaS account.
- **Multi-client support** – Designed to be added to various MCP-enabled chat clients; configuration instructions vary by client.
- **Central configuration page** – Additional setup and configuration options are available via the Pipedream configuration page.

## Usage
1. Connect your SuperSaaS account in the Pipedream interface.
2. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
3. Add the server to your chosen MCP-compatible chat client or app.
4. Authenticate when prompted to enable SuperSaaS scheduling tools.
