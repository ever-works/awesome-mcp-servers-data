# Databricks MCP Server (OAuth - Service Principal)

An OAuth-enabled MCP server for Databricks that authenticates using a Databricks Service Principal, enabling secure, authenticated access from MCP-compatible tools to Databricks resources.

- **Category:** data-access-integration-mcp-servers  
- **Brand:** Databricks  
- **Source URL:** https://mcp.pipedream.com/app/databricks_oauth  
- **Slug:** `databricks-mcp-server-oauth-service-principal`

## Features

- **OAuth-based authentication** using a Databricks Service Principal for secure access.  
- **Static MCP server URL** for all clients: `https://mcp.pipedream.net/v2` (authentication is performed when adding the server to the application).  
- **MCP-compatible**: designed to be added as an MCP server within supported chat clients and tools.  
- **Databricks integration**: enables MCP tools to access and interact with Databricks resources via the configured Service Principal.  
- **Central configuration docs**: setup and configuration guidance available via the Pipedream Connect configuration page.
