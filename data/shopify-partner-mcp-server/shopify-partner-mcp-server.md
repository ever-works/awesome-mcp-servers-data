# Shopify Partner MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Shopify  
**Slug:** `shopify-partner-mcp-server`

## Description
The Shopify Partner MCP Server provides an integration with the Shopify Partner API, allowing applications to access partner-related events and data such as app installs, uninstalls, charges, and transactions through the MCP (Model Context Protocol) interface.

## Features
- **Shopify Partner API integration**: Connects to the Shopify Partner API via MCP.
- **Event listening**:
  - App installs
  - App uninstalls
  - Charges
  - Transactions
- **Static MCP server endpoint**: Uses a single static URL that works for any compatible MCP client:
  - `https://mcp.pipedream.net/v2`
- **Client-agnostic setup**: Designed to be added to multiple chat or MCP-enabled clients; authentication occurs when adding the server to the application.
- **Central configuration reference**: Additional configuration details available via a dedicated configuration page (outside the provided content).
