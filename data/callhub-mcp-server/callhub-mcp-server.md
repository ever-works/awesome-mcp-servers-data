# CallHub MCP Server

An MCP server that connects to CallHub, an all-in-one digital organizing platform for political campaigns, nonprofits, and businesses.

- **Category:** Messaging MCP Servers  
- **Brand:** CallHub  
- **Slug:** `callhub-mcp-server`

## Features

- Connects applications to **CallHub**, a digital organizing platform used by:
  - Political campaigns
  - Nonprofits
  - Businesses
- Exposed as a **static MCP server endpoint** usable by any compatible client.
- Authentication is handled when adding the server to your application (per-client auth, not in the URL itself).

## MCP Server URL

Use this base URL for all MCP clients:

```text
https://mcp.pipedream.net/v2
```
