# Team SMS MCP Server

MCP Server for Team SMS, enabling teams to share and manage 2FA SMS messages through MCP-compatible tools.

- **Website / Source**: https://mcp.pipedream.com/app/team_sms
- **Category**: Messaging MCP Servers
- **Tags**: sms, 2fa
- **MCP Server URL**: `https://mcp.pipedream.net/v2`

## Features

- Share 2FA SMS messages across a team.
- Access and manage 2FA SMS via MCP-compatible clients.
- Single static MCP server URL for all clients: `https://mcp.pipedream.net/v2`.
- Authentication handled when adding the server to your application.
- Works with multiple chat / client applications (MCP clients) via configuration.

## Setup / Usage

1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add the server to your MCP-compatible app or chat client.
3. Authenticate when prompted by the client during setup.
