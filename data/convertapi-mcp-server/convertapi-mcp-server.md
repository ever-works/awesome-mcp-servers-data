# ConvertAPI MCP Server

**Category:** Media Processing MCP Servers  
**Brand:** ConvertAPI  
**Slug:** `convertapi-mcp-server`

ConvertAPI MCP Server exposes ConvertAPI’s online file conversion and manipulation capabilities through an MCP-compatible endpoint, so it can be added as a server in MCP-aware applications and chat clients.

---

## Features

- **MCP-compatible server**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Can be added to supported chat clients and applications as an MCP server.  
  - Authentication handled when adding the server to the application.

- **High-performance file conversion service**  
  - Designed for developers and application integration.  
  - Online file conversion and manipulation.  
  - Supports 200+ conversion actions (via the underlying ConvertAPI service).  
  - Can integrate into existing applications or platforms within minutes.  
  - Scales to handle high traffic while using relatively few resources.

- **Available MCP tools (actions)**  
  1. **Convert Web URL to Specified Format**  
     - Converts a website page (web URL) to a chosen output format.  
     - Uses ConvertAPI’s web-to-file rendering / conversion capability.  
  2. **Convert File**  
     - Converts an uploaded file from one format to another specified format.  
  3. **Convert Base64 Encoded File**  
     - Accepts a base64-string-encoded file as input.  
     - Converts it into a user-specified output format.

- **Account-based configuration**  
  - Connect a ConvertAPI account to manage and use the MCP tools.  
  - Account connection handled through the Pipedream configuration flow.

---

## Configuration

- Copy the MCP server URL: `https://mcp.pipedream.net/v2`.  
- Add this server URL to your MCP-compatible app or chat client.  
- Authenticate and connect your ConvertAPI account during setup.  
- Optionally refer to the full configuration documentation on Pipedream for client-specific setup steps.
