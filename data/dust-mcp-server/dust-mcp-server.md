## Description
Dust MCP Server is a Model Context Protocol (MCP) server that connects Dust with compatible AI chat clients. It allows you to create and orchestrate custom AI assistants via Dust, so they can be invoked directly from MCP-enabled applications to help speed up and automate workflows.

## Features
- **MCP-compatible server endpoint**  
  - Provides a static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same URL works across supported clients; authentication occurs when adding the server to your app.

- **Dust integration**  
  - Connects to your Dust account to leverage Dust-based AI assistants.  
  - Enables orchestration of AI assistants through the Model Context Protocol.

- **Works with multiple chat clients**  
  - Can be added to different MCP-compatible chat clients.  
  - Client-specific setup instructions are referenced via the configuration page.

- **Workflow acceleration and automation**  
  - Designed to speed up work by embedding Dust AI assistants directly into your existing tools.  
  - Suitable for use cases involving AI assistants, workflow automation, and orchestration.

- **Central configuration**  
  - Configuration handled through a dedicated configuration page (via Pipedream).  
  - Account connection and client selection are part of the setup flow.
