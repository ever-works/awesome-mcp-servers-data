# Notarize MCP Server

Online notary service MCP server that connects MCP-compatible clients to Notarize’s remote online notarization workflows.

## Overview
- **Type:** MCP server (Model Context Protocol)
- **Category:** Business & Commerce – MCP Servers
- **Provider:** Notarize (via Pipedream Connect)
- **Purpose:** Initiate and manage remote online notarization workflows from MCP-enabled chat or agent clients.

## Server URL
- **MCP Server Endpoint (static for all clients):**
  ```
  https://mcp.pipedream.net/v2
  ```
- Authentication is completed when adding the server to your client/application.

## Features
- Connects MCP clients to Notarize’s online notary service.
- Supports initiating remote online notarization workflows.
- Supports managing existing online notarization workflows (via Notarize) from MCP clients.
- Uses a single, static MCP server URL that works across compatible clients.
- Integrates with multiple MCP-enabled chat / AI clients (configured per client).

## Integration & Configuration
- Add the server to your MCP-compatible app or chat client using the static URL.
- Client-specific setup instructions are available via the “Configuration” page linked from the source.
