# Bilionis MCP Server

**Category:** Business & Commerce MCP Servers  
**Website / Source:** https://mcp.pipedream.com/app/bilionis

## Description
Bilionis MCP Server provides an MCP (Model Context Protocol) integration for Bilionis, a community and catalog platform for Italian professionals and companies that use intelligent catalogues. It allows MCP-compatible tools and chat clients to interface with the Bilionis catalog and community environment via Pipedream.

## Features
- MCP server integration for the Bilionis platform.
- Enables MCP tools and chat clients to connect with Bilionisâ€™ catalog and community for Italian professionals and companies.
- Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) that works across clients.
- Authentication occurs when adding the server to the client/application.
- Hosted and delivered via Pipedream Connect infrastructure.

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Integration host:** Pipedream Connect
- **Usage pattern:** Add the MCP server URL to a compatible chat client or application, then authenticate during setup.
