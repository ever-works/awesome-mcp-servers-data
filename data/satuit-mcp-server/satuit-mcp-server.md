# Satuit MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Satuit Technologies  
**Website / Source:** https://mcp.pipedream.com/app/satuit

## Overview
Satuit MCP Server exposes SatuitCRM—an asset management CRM used by buy-side investment professionals—through the Model Context Protocol (MCP). It allows AI tools and MCP-compatible clients to interact with institutional asset management and investor relations data stored in SatuitCRM.

SatuitCRM is used by:
- Institutional asset managers  
- Private equity firms  
- Wealth managers  
- Hedge funds  
- Alternative investment firms

## MCP Endpoint
- **MCP base URL:** `https://mcp.pipedream.net/v2`  
  - This is a static endpoint shared across clients; authentication is handled when adding the server in your MCP-compatible application.

## Features
- **MCP-based access to SatuitCRM**  
  - Provides a standardized MCP interface to SatuitCRM data.  
  - Designed for integration with AI tools and MCP-compatible chat clients.

- **Access to asset management CRM data**  
  - Connects to CRM data used by buy-side investment professionals.  
  - Targets institutional asset management and investor relations use cases.

- **Client-agnostic configuration**  
  - Single, static MCP URL (`https://mcp.pipedream.net/v2`) for all clients.  
  - Authentication occurs per client when the server is added.

- **Integration via configuration pages and guides**  
  - Can be added to supported chat / MCP clients using documented configuration steps.  
  - Central configuration reference available via the platform’s configuration page.
