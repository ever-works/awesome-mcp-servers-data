## Description
AWS Lambda Tool MCP Server is an MCP server that lets AI agents securely execute AWS Lambda functions as tools. This enables AI workflows to interact with and access private AWS resources via Lambda in a controlled, serverless manner.

## Features
- Exposes AWS Lambda functions as tools to MCP-compatible AI clients.
- Provides secure access to private AWS resources through Lambda execution.
- Built on AWS Lambda and serverless architecture.
- Designed for use by AI agents within the Model Context Protocol ecosystem.
