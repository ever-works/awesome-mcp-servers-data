# Ontraport MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Ontraport  
**Source:** https://mcp.pipedream.com/app/ontraport  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Overview
Ontraport MCP Server connects the Ontraport unified CRM and marketing automation platform to Model Context Protocol (MCP) clients. It enables AI or MCP-compatible applications to interact with Ontraport for workflows such as contact management and campaign automation directly from a chat or other MCP-based interface.

## Features
- Connects Ontraport’s CRM and marketing automation capabilities to MCP-compatible clients.
- Supports workflows related to:
  - Contact and customer data management.
  - Marketing and campaign automation processes.
- Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) for all supported clients.
- Authentication performed when adding the server to an MCP-compatible application.
- Can be integrated into various chat or MCP client applications by adding the server URL in the client’s configuration.

## Integration & Configuration
- Static MCP server URL is added to the client configuration: `https://mcp.pipedream.net/v2`.
- Authentication occurs during setup inside the chosen MCP client.
- Additional configuration details are available via the platform’s configuration documentation (referenced as “Configuration page” in the source).
