# Wachete MCP Server

MCP Server that integrates Wachete’s web page change tracking and monitoring into the MCP environment.

- **Category:** Monitoring  
- **Brand:** Wachete  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features

- Integrates Wachete’s web page change tracking and monitoring into MCP-compatible clients.
- Tracks and monitors changes on web pages (content updates, modifications, etc.).
- Provides a single static MCP server URL that works across supported clients.
- Authentication handled when adding the server to your application.
- Compatible with multiple chat / MCP clients (configured per client in the app).
- Configuration details available via a central configuration page (hosted by Pipedream Connect).
