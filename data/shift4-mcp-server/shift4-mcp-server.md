# Shift4 MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Pipedream  
**Slug:** `shift4-mcp-server`

## Overview
Shift4 MCP Server is a Model Context Protocol (MCP) server provided by Pipedream that exposes Shift4’s commerce and payment processing capabilities to AI agents and automation tools. It enables MCP-compatible clients to interact with Shift4’s payment, e‑commerce, and transaction services through a standardized server endpoint.

## Key Details
- **MCP Server Type:** Static, shared endpoint
- **MCP Base URL:** `https://mcp.pipedream.net/v2`
- **Authentication:** Per-client authentication when adding the server to an application
- **Ecosystem:** Powered by Pipedream Connect

## Features
- **MCP-compatible access to Shift4:**
  - Provides an MCP server interface to Shift4’s commerce and payment processing solutions.
  - Designed for use with MCP-enabled chat clients and agent frameworks.

- **Static MCP server URL:**
  - Single, static endpoint (`https://mcp.pipedream.net/v2`) usable across different clients.
  - Server address does not need to change between applications or environments.

- **Client-side authentication:**
  - Authentication occurs when adding the server to a client or application.
  - Supports secure access control per user or per app (specific methods not detailed in the source content).

- **AI and automation integration:**

## Usage
- **MCP Endpoint:** Add `https://mcp.pipedream.net/v2` as an MCP server in a compatible chat client or application.
- **Setup Flow (high level):**
  1. Configure the MCP server URL in your MCP-compatible client.
  2. Authenticate when prompted by the client, using the method specified in that client’s instructions.
  3. Begin using Shift4 commerce and payment operations through the MCP interface.
