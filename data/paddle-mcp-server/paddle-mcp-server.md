# Paddle MCP Server

**Category:** Finance & Market Data MCP Servers  
**Brand:** Paddle  
**Source:** https://mcp.pipedream.com/app/paddle

## Description
Paddle MCP Server exposes Paddle’s payments and subscription infrastructure for SaaS companies through the Model Context Protocol (MCP). It enables applications and chat-based clients to integrate and manage payment and subscription workflows via a unified MCP server endpoint.

## Features
- Provides access to Paddle’s payments infrastructure for SaaS products via MCP
- Supports subscription-related workflows (e.g., recurring billing scenarios) within MCP-integrated applications
- Uses a single static MCP server URL for all compatible clients
- Authentication occurs when adding/configuring the server in the client application
- Can be added to various chat clients and MCP-compatible tools via configuration

## Integration Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Setup:** Add the above MCP server URL to your MCP-compatible app or chat client and authenticate during configuration
