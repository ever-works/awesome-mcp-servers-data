### SerwerSMS.pl MCP Server

**Category:** Messaging MCP Servers  
**Brand:** serwersmspl  
**Source:** https://mcp.pipedream.com/app/serwersms_pl

#### Description
SerwerSMS.pl MCP Server integrates SerwerSMS SMS capabilities with MCP-based systems, enabling applications to send and receive SMS messages through a standardized MCP server endpoint.

#### Features
- **Send SMS messages**
  - Support for sending individual SMS messages.
  - Support for sending mass / bulk SMS campaigns.
- **Receive SMS messages**
  - Handling of inbound SMS within MCP-based applications.
- **MCP integration**
  - Exposed as an MCP server that can be added to compatible chat or MCP clients.
  - Uses a static MCP server URL: `https://mcp.pipedream.net/v2`.
- **Account-based operation**
  - Requires connecting a SerwerSMS.pl account before use.
  - Authentication handled when adding the server to your application.

#### Configuration
- Connect your SerwerSMS.pl account via the Pipedream MCP configuration flow.
- Use the static MCP server URL (`https://mcp.pipedream.net/v2`) when registering the server in your MCP-compatible app.
