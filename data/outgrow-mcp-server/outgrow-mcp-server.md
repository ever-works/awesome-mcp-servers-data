# Outgrow MCP Server

## Overview
Outgrow MCP Server connects the Outgrow interactive content platform (quizzes, calculators, and other interactive experiences for lead generation) to MCP-compatible chat clients via a static MCP endpoint hosted by Pipedream.

- **Type:** MCP server / integration
- **Category:** Business & Commerce – MCP Servers
- **Provider / Host:** Pipedream Connect
- **Platform Integrated:** Outgrow (interactive content & lead-generation tools)

## Features
- **MCP Endpoint for Outgrow**  
  - Provides a single static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Works with any compatible MCP client.

- **Client-Agnostic Setup**  
  - Same endpoint can be added to multiple chat clients.  
  - Authentication occurs when adding the server to each client (client-specific auth flow, not in the content).

- **Outgrow Integration Context**  
  - Designed to interface with Outgrow’s interactive content capabilities (e.g., quizzes, calculators, and similar lead-generation experiences) through MCP.

- **Configuration Documentation**  
  - Additional configuration details available via a dedicated Configuration page (linked from the source content, though specifics are not included in the provided text).

## Usage
1. Copy the MCP server URL:  
   `https://mcp.pipedream.net/v2`
2. Add the server in your MCP-compatible chat client.  
3. Authenticate when prompted by the client.
