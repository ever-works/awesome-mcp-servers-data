# Polygon.io MCP Server

**Category:** Finance Market Data MCP Servers  
**Brand:** Polygon.io  
**Slug:** `polygonio-mcp-server`

Polygon.io MCP Server provides access to Polygon.io’s financial market data via a developer-focused API, exposed as MCP tools that can be used from compatible chat or AI clients.

---

## Features

- **MCP server endpoint**  
  - Static MCP server URL usable across all compatible clients:  
    - `https://mcp.pipedream.net/v2`
  - Authentication is performed when adding the server to your application.

- **Financial market data access**  
  - Connects to Polygon.io’s market data platform via MCP.  
  - Designed for developer-oriented access to financial markets data (e.g., stocks and general market data via API).

- **Client-agnostic integration**  
  - One static URL works for every supported MCP client.  
  - Can be added to various chat or AI applications that support MCP.

- **Configuration documentation**  
  - Additional setup and usage details available via a configuration page (linked from the app page).

---

## Technical Details

- **MCP Protocol**: Server implements the Model Context Protocol (MCP) to expose Polygon.io tools.  
- **Integration provider**: Delivered via Pipedream Connect.
