### Overview
joinly is an MCP server that enables AI agents to participate in browser-based online meetings. It connects with platforms like Zoom, Microsoft Teams, and Google Meet so agents can join calls, interact via meeting bots, and automate meeting-related workflows.

### Features
- **MCP server for meetings**
  - Implements a Model Context Protocol (MCP) server focused on meeting workflows.
  - Designed to be used by AI agents to access and act within live meetings.

- **Browser-based meeting platform support**
  - Integrates with common browser-based meeting tools such as:
    - Zoom
    - Microsoft Teams
    - Google Meet
  - Uses browser automation to join and interact with meetings.

- **AI agent participation in meetings**
  - Allows AI agents to join meetings as participants via bots.
  - Supports interaction in meetings (e.g., listening, responding, or otherwise engaging via a bot interface).

- **Meeting workflow automation**
  - Automates tasks around meetings, using AI agents connected through MCP.
  - Suitable for building custom workflows such as automated participation, follow-ups, or other meeting-related processes.

- **Developer-oriented project structure**
  - Includes client, common, and server (`joinly`) components.
  - Docker configuration and scripts are provided for running and deploying the server.
  - Example projects and tests included in the repository to demonstrate usage and validate behavior.

### Integrations
- Zoom (browser-based)
- Microsoft Teams (browser-based)
- Google Meet (browser-based)
- MCP-compatible AI agents / tools
