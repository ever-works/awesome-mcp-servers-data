# Domain Group MCP Server

## Overview
The Domain Group MCP Server is an MCP integration for Domain (domain.com.au) that lets AI tools access Australian real estate listings and property data for sale and rent. It is provided via Pipedream and can be connected to AI coding and productivity tools.

## Features
- Connects a Domain Group (domain.com.au) account to AI tools
- Provides access to Australian real estate listings
- Supports both properties for sale and for rent
- Exposes property data for use in MCP-compatible tools and workflows
- Integrates via Pipedreamâ€™s MCP server
- Compatible with AI tools such as Cursor, Claude, and Windsurf
