# Kanbanize MCP Server

## Overview
The Kanbanize MCP Server exposes Lean and Kanban project management workflows from Kanbanize to MCP-compatible clients. It is intended for advanced product teams that want to apply Lean and Kanban principles via MCP agents.

**MCP Server URL:**
```text
https://mcp.pipedream.net/v2
```

## Features
- MCP server interface for Kanbanize
- Provides access to Lean and Kanban project management workflows through MCP agents
- Designed for advanced product teams using Kanbanize
- Static, client-agnostic server URL usable across different MCP-compatible chat clients
- Authentication handled when adding the server to your MCP-compatible application
- Configuration guidance available via a dedicated configuration page (client setup instructions depend on the chosen chat client)
