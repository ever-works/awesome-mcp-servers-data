# Itemize MCP Server

Invoice data extraction, reconciliation, and verification MCP server for use with compatible chat clients and tools.

---

## Overview
- **Name:** Itemize MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Use Case:** Invoice data extraction, reconciliation, and verification workflows  
- **Provider / Platform:** Runs via Pipedream MCP infrastructure  
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

---

## Features
- **Invoice-focused MCP server**  
  - Designed for invoice data extraction, reconciliation, and verification tasks.
- **Shared static MCP URL**  
  - Uses a single static MCP endpoint: `https://mcp.pipedream.net/v2`.
  - Same URL works for all clients.
- **Per-client authentication**  
  - Authentication occurs when adding the server to each client / application (credentials are not embedded in the URL).
- **Multi-client compatibility**  
  - Can be added to different chat clients and tools that support MCP.  
  - Configuration guidance available via the platform’s configuration page.
- **Hosted by Pipedream**  
  - Delivered through Pipedream’s MCP infrastructure (Pipedream Connect).

## Integration & Configuration
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Setup Flow:**  
  1. Copy the static MCP URL.  
  2. Add it as an MCP server in your chat client or MCP-compatible application.  
  3. Authenticate during client-specific setup.  
  4. Optionally consult the platform’s “Configuration” page for detailed client-specific instructions.
