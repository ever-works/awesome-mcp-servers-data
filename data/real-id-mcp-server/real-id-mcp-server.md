# Real ID MCP Server

Real ID MCP Server connects MCP-compatible clients to Real ID’s identity verification platform, enabling secure, automated ID verification flows without custom backend work.

## Overview
- **Type:** MCP Server (Model Context Protocol)
- **Category:** Security & Attestation MCP Servers
- **Provider / Brand:** Real ID (via Pipedream Connect)
- **Use Cases:**
  - Online identity verification
  - KYC / compliance workflows
  - Fraud reduction for selling and shipping
  - Automated ID checks inside chat-based or MCP-enabled applications

## Features
- **MCP-compatible server endpoint**
  - Single static MCP server URL usable across clients: `https://mcp.pipedream.net/v2`
  - Designed to be added as a server in any MCP-capable chat or agent client.

- **Integration with Real ID platform**
  - Connects to Real ID’s identity verification infrastructure.
  - Supports secure, automated ID verification flows.

- **No-code setup via Pipedream Connect**
  - Built on Pipedream Connect for simplified configuration.
  - No custom code required to start using Real ID verification within MCP clients.

- **Secure verification flows**
  - Focus on secure handling of identity verification processes.
  - Suitable for risk-sensitive use cases such as shipping high-value goods or verifying buyers/sellers.

- **Scalable verification**
  - Intended for handling verification at scale (e.g., many transactions or users).

- **Client-agnostic configuration**
  - Same URL and configuration pattern for different chat or MCP clients.
  - Configuration guidance available via the platform’s configuration page.

## Configuration
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Setup flow:**
  - Copy the static URL.
  - Add it as an MCP server in your chosen chat or agent client.
  - Authenticate during server addition/first use as prompted by the client.
  - Optional: refer to the platform’s "Configuration" page for step-by-step instructions.
