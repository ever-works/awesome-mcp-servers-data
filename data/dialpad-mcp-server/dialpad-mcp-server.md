# Dialpad MCP Server

An MCP Server integration for Dialpad that exposes its unified talk, messaging, meetings, and support workspace for automation and orchestration.

- **Category:** Messaging MCP Servers  
- **Brand:** Dialpad  
- **Slug:** `dialpad-mcp-server`

## Features

- Provides MCP (Model Context Protocol) server integration for Dialpad.  
- Exposes Dialpadâ€™s unified communication workspace for:
  - Voice and telephony (talk)
  - Messaging
  - Meetings
  - Support / contact-center style workflows
- Designed for automation and orchestration of Dialpad workflows from compatible MCP clients.
- Uses a **static MCP server URL** for all clients:
  - `https://mcp.pipedream.net/v2`
- Authentication is performed when adding the server inside the client / application.
- Built and hosted via **Pipedream Connect**, enabling integration with other tools and services that support MCP.

## Configuration

- Add the server to any MCP-compatible chat client using the static URL:  
  `https://mcp.pipedream.net/v2`
- Authentication occurs during client setup / server addition.
- Additional configuration details are available on the (external) Configuration page referenced by the source.
