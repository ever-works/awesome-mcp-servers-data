# Typebot MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Typebot  
**Source:** https://mcp.pipedream.com/app/typebot

## Overview
Typebot MCP Server is an MCP (Model Context Protocol) server integration for Typebot that allows AI agents to orchestrate and manage conversational chat flows and collect responses using MCP tools. It provides a static server endpoint you can add to compatible MCP clients to integrate Typebot-powered flows into your applications.

**MCP Server URL:**
```text
https://mcp.pipedream.net/v2
```

## Features
- **MCP Integration for Typebot**: Exposes Typebot capabilities as MCP tools so AI agents can orchestrate and manage conversational flows.
- **Conversational Chat Flows**: Supports building and running interactive, conversational experiences using Typebot’s blocks.
- **Response Collection**: Collects user responses and form-like inputs within chat experiences.
- **Static MCP Endpoint**: Single static MCP server URL (`https://mcp.pipedream.net/v2`) usable across supported clients.
- **Client-Agnostic Server**: Designed to work with multiple MCP-compatible chat clients; you authenticate per client when adding the server.
- **Embeddable Experiences (via Typebot)**: Leverages Typebot’s ability to embed chatbots into web and mobile apps.
- **Configuration Documentation**: Full configuration and setup instructions available via a dedicated configuration page.

## Setup
1. Use the static MCP server URL:
   ```text
   https://mcp.pipedream.net/v2
   ```
2. Add the MCP server to your preferred MCP-compatible chat client.
3. Authenticate within that client as prompted.
4. Configure tools and flows following the linked configuration documentation.
