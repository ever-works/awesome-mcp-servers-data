# Zoho Meeting MCP Server

Zoho Meeting MCP Server is an MCP (Model Context Protocol) server integration for Zoho Meeting, exposing its online meeting and webinar capabilities to compatible chat or AI clients.

---

## Overview
- **Category:** Business & Commerce – MCP Servers
- **Brand:** Zoho
- **Integration Type:** MCP server
- **Purpose:** Provide access to Zoho Meeting’s online meeting and webinar management features (e.g., scheduling, hosting, collaboration) through the Model Context Protocol.

---

## Features
- **MCP Server Endpoint**  
  - Static MCP server URL usable across supported clients:  
    - `https://mcp.pipedream.net/v2`
  - Authentication handled when adding the server to a client/application.

- **Online Meetings & Webinars (via Zoho Meeting)**  
  *As described at a high level in the source content and item metadata:*  
  - Access to secure online meeting functionality.  
  - Access to webinar solution features.  
  - Remote collaboration and communication capabilities.  
  - Support for scheduling and hosting sessions (meetings or webinars).  
  - Collaboration-focused workflows exposed through MCP so tools/agents can interact programmatically with Zoho Meeting.

- **Client Integration**  
  - Designed to be added to compatible chat or AI clients as an MCP server.  
  - Works with multiple clients using the same static URL.  
  - Additional configuration details available via the provider’s configuration documentation (not specific to this item’s functionality).

---

## Configuration
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Connection Flow:**  
  - Add the above URL as an MCP server in your preferred client.  
  - Authenticate during the setup process in that client (exact steps depend on the client).
