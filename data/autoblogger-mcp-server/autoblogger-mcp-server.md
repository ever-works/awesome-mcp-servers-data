# AutoBlogger MCP Server

An MCP server that connects MCP-enabled agents to AutoBlogger so they can automatically create and publish posts.

- **Category:** Content Management MCP Servers  
- **Brand:** autoblogger  
- **Source URL:** https://mcp.pipedream.com/app/autoblogger  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features

- **Automated post creation**
  - Allows MCP-enabled agents to generate blog or content posts via AutoBlogger.
  - Designed for “set it up once” workflows so content creation can run with minimal manual intervention.

- **Automated publishing**
  - Supports publishing of generated posts directly through AutoBlogger.
  - Enables end-to-end automation from content generation to publication.

- **Static MCP endpoint**
  - Uses a single static server URL (`https://mcp.pipedream.net/v2`) that works across compatible MCP clients.
  - Authentication is handled when adding the server to your MCP-enabled application.

- **Client-agnostic integration**
  - Can be added to different chat or MCP clients that support MCP servers.
  - Configuration guidance available via the referenced configuration documentation.

- **Pipedream-powered infrastructure**
  - Hosted and powered by Pipedream Connect’s MCP infrastructure.

## Configuration & Integration

- Add the server to your chat client or MCP-enabled app using the static URL: `https://mcp.pipedream.net/v2`.
- Authenticate as prompted by your client during server setup.
- Additional configuration details are available on the Pipedream configuration page (not included in the provided content).
