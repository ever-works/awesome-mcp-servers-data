## Overview

Official MCP server for Figma's Dev Mode, providing AI access to live design structures.

## Features

- Expose selected Figma layers with full structural detail, not just pixel dimensions
- Surface design tokens, color styles, and typography definitions
- Provide component variants and their properties
- Return auto-layout constraints and spacing rules

## Use Cases

- Implement selected Figma components as React components using Tailwind
- Extract spacing tokens and color variables from designs
- Generate CSS files matching Figma typography styles

## Pricing

Included with Figma Dev Mode access; free tier available.
