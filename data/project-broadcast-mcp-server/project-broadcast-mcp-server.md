# Project Broadcast MCP Server

An MCP server that connects to Project Broadcast to enable automated, relationship-focused business communications within MCP-compatible chat clients.

## Overview
- **Type:** MCP Server (Model Context Protocol)
- **Category:** Business & Commerce MCP Servers
- **Provider / Platform:** Pipedream Connect
- **Integration Target:** Project Broadcast
- **Primary Use Case:** Automating business communications and relationship-building workflows via MCP-aware applications.

## Connection Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
  - Static URL used by all clients
  - Authentication is performed when adding the server to your application

## Features
- Provides an MCP-compliant server endpoint for integrating Project Broadcast with MCP-capable chat clients and tools
- Supports automated business communication workflows powered by Project Broadcast
- Facilitates relationship-building communication flows (e.g., messaging, campaigns) via Project Broadcast through MCP
- Uses a single, static MCP server URL for all clients, simplifying configuration
- Authentication handled at the time of adding/configuring the server in your client
- Documentation and configuration guidance available via the Pipedream Connect configuration page

## Setup & Configuration
- Copy the MCP server URL: `https://mcp.pipedream.net/v2`
- Add the server URL to your MCP-compatible chat client or application
- Authenticate when prompted by your client during setup
- Optional: Refer to the full configuration documentation on the Pipedream Connect Configuration page
