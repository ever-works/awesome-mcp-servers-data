## Overview
Parma MCP Server is an MCP (Model Context Protocol) server that connects a Parma CRM account to AI tools such as Cursor, Claude, and Windsurf. Parma is a minimalistic CRM focused on tracking customers, their preferences, and prior interactions.

## Features
- **Parma CRM integration**: Connects directly to a Parma account to surface CRM data.
- **Customer tracking**: Accesses records about customers, including their history in Parma.
- **Preference management**: Exposes stored customer preferences from Parma to compatible AI tools.
- **Interaction history**: Makes prior customer interactions recorded in Parma available via MCP.
- **AI tool connectivity**: Designed to work with AI tools that support MCP, including:
  - Cursor
  - Claude
  - Windsurf
