# signNow MCP Server

**Category:** Document Management MCP Servers  
**Slug:** `signnow-mcp-server`  
**Source:** https://mcp.pipedream.com/app/signnow

## Overview
signNow MCP Server integrates signNowâ€™s eSignature and workflow automation capabilities into MCP-based applications. It enables document signing, contract workflows, and payment-enabled forms directly within supported chat or MCP clients.

## Features
- **eSignature workflows**
  - Create and scale electronic signature workflows within MCP ecosystems.
  - Sign and send documents for signature.
  - Support for document signing across multiple participants.

- **Document generation & templates**
  - Create new documents from existing signNow templates.
  - Generate agreements and contracts from standardized templates.

- **Contract negotiation & forms**
  - Support for contract negotiation workflows using signNow documents.
  - Payment-enabled forms (collect payments as part of document workflows).

- **Workflow automation**
  - Automate business processes around document preparation, signing, and completion.
  - Integrate with broader airSlate Business Cloud workflows (via signNow).

- **MCP server configuration**
  - Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`.
  - Authentication handled when adding the server to the client application.
  - Can be added to supported chat clients or apps that consume MCP servers.

- **Available tools (actions)**
  - **Upload Document With Tags**
    - Upload files containing signNow text tags for automated field detection.
  - **Send Field Invite**
    - Create and send a field invite so recipients can sign a document.
  - **Prefill Text Fields**
    - Add values to text fields before sending; signers can edit them later during signing.
  - **Create Document From Template**
    - Generate a new document instance from a predefined signNow template.
