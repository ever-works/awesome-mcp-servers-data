# HappyFox Chat MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Pipedream  
**Website:** https://mcp.pipedream.com/app/happyfox_chat

## Overview
HappyFox Chat MCP Server is an integration endpoint on Pipedream that connects HappyFox Chat (live chat software for customer engagement) to MCP-compatible applications. It enables workflows driven by live chat events and customer interactions via a standardized MCP server URL.

## Features
- **Standard MCP Endpoint**  
  - Uses a single static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same URL works for all clients; authentication is performed when adding the server in your application.

- **HappyFox Chat Integration**  
  - Integrates with HappyFox Chat, a live chat tool used to connect, engage, and convert customers.  
  - Designed to be used as a communication-focused MCP server within chat or agent clients.

- **Client-Agnostic Setup**  
  - Can be added to different MCP-compatible chat clients.  
  - Setup instructions are available per client via the Pipedream UI ("Select your chat client to learn how to get started").

- **Configurable via Pipedream**  
  - Connect your HappyFox Chat account through Pipedream.  
  - Configuration details and additional setup guidance are accessible on a dedicated Configuration page.

- **Tooling Integration (MCP Tools)**  
  - Exposes MCP tools / actions that can be loaded by compatible clients (shown as "Available tools" in the UI).  
  - Intended to power automated workflows based on live chat events and customer interaction data from HappyFox Chat.

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Authentication:** Performed when adding the MCP server to your application (details provided in client-specific setup in Pipedream).
