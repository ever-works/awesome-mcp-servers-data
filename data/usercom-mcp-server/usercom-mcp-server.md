# User.com MCP Server

**Category:** Business & Commerce – MCP Servers  
**Tags:** marketing, CRM, automation

## Overview
The User.com MCP Server on Pipedream exposes the User.com platform to Model Context Protocol (MCP) clients, enabling access to CRM, sales, customer success, and marketing automation capabilities through a single automation interface.

It provides a static MCP endpoint that can be added to compatible chat or MCP-enabled applications, allowing them to interact with User.com data and workflows.

## Features
- **Unified automation entry point**  
  - Single MCP server for accessing User.com CRM, sales, customer success, and marketing automation functionality.

- **Static MCP server URL**  
  - Uses a single static MCP endpoint for all clients:  
    - `https://mcp.pipedream.net/v2`  
  - Same URL works across different MCP clients; authentication is handled when adding the server to an application.

- **Account-based configuration**  
  - Connect a User.com account and select a client within the Pipedream interface to begin using the server.
  - Supports checking and validating the connected account before use.

- **Client-agnostic integration**  
  - Can be added to multiple MCP-compatible chat clients or applications.  
  - Per-client setup instructions available via the Pipedream UI.

- **Configurable via Pipedream**  
  - Additional configuration details accessible through a dedicated configuration page on Pipedream.

- **Tools / actions interface**  
  - Exposes User.com functionality as “tools” (actions) loadable by MCP clients (exact tools not listed in the provided content but surfaced dynamically through the server).
