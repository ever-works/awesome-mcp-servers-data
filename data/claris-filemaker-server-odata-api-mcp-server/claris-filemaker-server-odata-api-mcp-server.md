## Claris FileMaker Server OData API MCP Server

An MCP server that connects to the Claris FileMaker Server OData API, enabling standardized OData-based access to FileMaker data from MCP-aware AI tools.

### Features
- Connects to Claris FileMaker Server via its OData API
- Provides standardized OData-based querying of FileMaker data
- Supports updating FileMaker records through OData
- Integrates with MCP-aware AI tools (e.g., Cursor, Claude, Windsurf) via Pipedream’s MCP server
- Enables AI-driven workflows that can read and write FileMaker data

### Integrations
- Claris FileMaker Server (OData API)
- MCP-aware AI tools such as Cursor, Claude, and Windsurf
- Operates through Pipedream’s MCP server framework
