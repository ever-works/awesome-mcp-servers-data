# BugBug MCP Server

**Category:** Testing & Debugging Tools  
**Brand:** bugbug  
**Website / Source:** https://mcp.pipedream.com/app/bugbug

## Description
BugBug MCP Server exposes BugBug’s browser test automation capabilities via the Model Context Protocol (MCP). It allows MCP-compatible clients (such as AI chat interfaces or other MCP apps) to trigger and manage automated browser testing workflows.

## Features
- **MCP-compatible test automation server**  
  - Implements a Model Context Protocol server interface for BugBug test automation.
- **Browser testing workflows**  
  - Designed for running and managing automated browser-based tests.
- **Static MCP server URL**  
  - Single endpoint for all clients:  
    `https://mcp.pipedream.net/v2`
  - Same URL used across different MCP clients; authentication is handled when adding the server to each application.
- **Client-agnostic integration**  
  - Works with any compatible MCP client or chat application that supports configuring external MCP servers.
- **Configuration documentation**  
  - Additional setup and configuration details available via the referenced Configuration page (on the provider’s site).

## How to Use (High-Level)
1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add it as an MCP server in your chosen MCP-compatible chat client or application.
3. Authenticate when prompted by your client to enable BugBug test automation.
