# Voodoo SMS MCP Server

**Category:** Messaging MCP Servers  
**Slug:** voodoo-sms-mcp-server  
**Brand:** Voodoo SMS

## Description
Voodoo SMS MCP Server is an MCP-compatible integration that connects to the Voodoo SMS mobile messaging platform. It enables applications and chat clients to access Voodoo SMS for business SMS campaigns and customer or staff communications via a standardized MCP server endpoint.

## Features
- MCP-based access to the Voodoo SMS platform
- Supports business SMS campaigns
- Enables customer and staff SMS communications
- Static MCP server URL usable across compatible clients
- Authentication handled when adding the server to an MCP-enabled application
- Integration powered by Pipedream Connect

## Technical Details
- **MCP server URL:** `https://mcp.pipedream.net/v2`
- **Integration host:** Pipedream (Pipedream Connect)
