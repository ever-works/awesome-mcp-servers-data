## Nerv MCP Server

**Category:** Business & Commerce · MCP Servers  
**Brand:** nerv  
**Website:** https://mcp.pipedream.com/app/nerv

### Description
Nerv MCP Server is an MCP integration for Nerv digital business cards. It allows AI tools and MCP-compatible clients to access and share Nerv-based digital contact information programmatically.

### Features
- Connects a Nerv digital business card account to MCP-compatible tools.
- Enables AI tools (e.g., Cursor, Claude, Windsurf) to interact with Nerv contact data.
- Provides programmatic access to digital contact information for use in workflows and automations.
- Built on Pipedream’s MCP server infrastructure for integration and orchestration.

### Integrations
- Cursor
- Claude
- Windsurf
- Other MCP-compatible tools via Pipedream MCP server
