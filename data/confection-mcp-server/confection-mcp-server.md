# Confection MCP Server

**Description**  
MCP server integration that connects Confection’s privacy-compliant data collection, storage, and distribution capabilities into model-context workflows. It is designed to operate reliably despite client-side disruptions (such as blocked cookies, cross-domain script issues, and device ID limitations) while remaining compliant with global privacy regulations.

## Features

- **MCP server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Single URL works across compatible MCP clients.  
  - Authentication is performed when adding the server to an application.

- **Confection data integration**  
  - Integrates Confection’s data collection, storage, and distribution into model-context workflows.  
  - Designed to function even when traditional client-side tracking mechanisms fail or are restricted (e.g., cookies, cross-domain scripts, device IDs).

- **Privacy and compliance**  
  - Built to be compliant with global privacy laws.  
  - Focus on privacy-safe data handling suitable for regulated environments.

- **Client integration guidance**  
  - Supports configuration through multiple chat / MCP client applications.  
  - Additional setup and configuration details available via the linked configuration documentation.
