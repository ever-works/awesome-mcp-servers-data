# AirOps MCP Server

MCP server for AirOps Studio that lets LLMs build and orchestrate LLM-powered workflows and chat assistants via the Model Context Protocol.

## Features

- **MCP (Model Context Protocol) server** for integrating AirOps Studio with compatible LLM clients.
- **Supports building LLM-powered workflows**: design, run, and orchestrate complex workflows driven by large language models.
- **Enables chat assistants**: create and manage LLM-based chat assistants connected to AirOps Studio.
- **Static MCP server URL** for all clients:
  - Endpoint: `https://mcp.pipedream.net/v2`
  - Same URL works across different MCP-compatible clients.
- **Client-agnostic authentication flow**: authentication occurs when adding the server to each client/application.
- **Usable with multiple chat clients**: can be added to supported chat/LLM clients that implement MCP.
- **Configuration documentation available** via a central configuration page (for setup steps and options).
- **Hosted by Pipedream Connect**: infrastructure and connectivity handled by Pipedream.

## Integration & Usage

- Add the MCP server URL (`https://mcp.pipedream.net/v2`) to your MCP-compatible chat or LLM client.
- Authenticate within your client when prompted.
- Use AirOps Studio to define and orchestrate workflows and chat assistants that the MCP server exposes to the LLM.
