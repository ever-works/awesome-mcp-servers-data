## Overview

Powered by the open Model Context Protocol (MCP), the Treasure Data MCP Server gives data teams a new capability: the ability to explore and analyze customer data in an easy and effective way using plain language and a conversation window. This enables marketing teams to access customer insights without complex query languages.

## Features

- **Segment Querying**: Query parent segments and segments using natural language
- **Table Exploration**: Explore customer data tables conversationally
- **Data Analysis**: Analyze customer data with AI-assisted insights
- **Plain Language Interface**: Natural language queries for data access
- **Marketing Use Cases**: Optimized for marketing analytics and segmentation
- **Cloud Data Integration**: Connects to enterprise data in cloud stores
- **Governed Access**: Structured and governed data access

## Key Capabilities
