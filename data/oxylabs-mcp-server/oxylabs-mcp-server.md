# Oxylabs MCP Server

An MCP Server integration that lets AI agents access Oxylabs’ proxy and web scraping solutions through any MCP-compatible client.

- **Category:** Content Extraction & Summarization MCP Servers  
- **Provider / Brand:** Oxylabs  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Integration Host:** Pipedream Connect  
- **Slug:** `oxylabs-mcp-server`  
- **Tags:** `web-scraping`, `proxies`, `data-extraction`

## Features

- **MCP-compatible server endpoint**  
  - Single static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Works across all supported MCP clients (e.g., chat-based AI tools that support the Model Context Protocol).

- **Oxylabs proxy and web scraping access**  
  - Designed to connect AI agents to Oxylabs’ market-focused proxy and web scraping services via MCP.  
  - Suitable for data extraction and web scraping workflows that can be orchestrated from an MCP-aware client.

- **Centralized configuration**  
  - Add the MCP server once in your chosen client using the static URL, then authenticate within the client.  
  - Configuration help is available via a dedicated configuration page linked from the app.

- **Pipedream-powered integration**  
  - Integration is provided and hosted via Pipedream Connect, which manages the MCP endpoint for Oxylabs services.

## Authentication

- Authentication is performed when adding the MCP server to your chosen application or chat client (details handled within the client’s configuration flow).
