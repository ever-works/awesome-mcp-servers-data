# Vero MCP Server

An MCP server that connects Vero’s multi-channel messaging and marketing automation capabilities to MCP-compatible clients.

## Overview
- **Type:** MCP server / integration
- **Category:** Messaging MCP servers
- **Provider / Brand:** Vero (via Pipedream Connect)
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`
- **Use Case:** Access Vero’s multi-channel messaging and marketing automation tools from compatible chat or MCP clients.

## Features
- Multi-channel messaging interface via MCP
- Access to Vero’s marketing automation functionality through MCP tools
- Single static MCP server URL for all clients
- Per-client authentication handled when adding the server to an application
- Compatible with multiple chat clients (configured on a per-client basis)

## Integration Details
- **Server URL:** `https://mcp.pipedream.net/v2` (same for all clients)
- **Setup:**
  - Add the MCP server URL to your MCP-compatible chat client or application
  - Authenticate within your client when prompted
  - Optionally refer to the provider’s configuration documentation for client-specific steps
