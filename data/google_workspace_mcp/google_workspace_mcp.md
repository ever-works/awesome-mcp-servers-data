## Overview

google_workspace_mcp is an open‑source Model Context Protocol (MCP) server that integrates Google Workspace (G Suite) apps with AI tools. It enables programmatic and AI-driven control of Gmail, Google Calendar, Docs, Sheets, Slides, Chat, Forms, Tasks, Search, and Drive from a unified MCP interface.

## Features

- **Unified Google Workspace MCP server**
  - Single MCP server for multiple Google services.
  - Designed to be used by AI assistants via the MCP protocol.

- **Gmail integration**
  - Programmatic control of Gmail via MCP (e.g., reading/sending/managing email – implied by scope of a Gmail integration).

- **Google Calendar integration (gcalendar)**
  - Calendar operations exposed through MCP (e.g., events, scheduling – implied by calendar integration module).

- **Google Docs integration (gdocs)**
  - Access and manipulation of Google Docs documents via MCP.

- **Google Drive integration (gdrive)**
  - File and folder operations in Google Drive available through MCP.

- **Google Sheets & Slides support**
  - Repository description notes control over Sheets and Slides as part of the Workspace suite.

- **Google Chat integration (gchat)**
  - Interaction with Google Chat through MCP.

- **Google Forms integration (gforms)**
  - Create or manage Google Forms via MCP.

- **Google Tasks integration**
  - Tasks control is included in the Workspace coverage (noted in repo description).

- **Google Search integration (gsearch)**
  - Search capabilities across Google Workspace via a dedicated gsearch module.

- **Authentication module (auth)**
  - Centralized authentication layer tailored for Google APIs.

- **Core framework (core)**
  - Shared utilities and abstractions used by all service modules.
