## Description
OpenSRS MCP Server exposes the OpenSRS reseller platform capabilities for domains, email, and SSL certificates to MCP-compatible workflows. It provides a single MCP endpoint that applications can use to integrate OpenSRS services.

## Features
- MCP-compatible server for OpenSRS services
- Access to reseller platform functionality for:
  - Domain services
  - Email services
  - SSL certificate services
- Static MCP server URL usable across clients: `https://mcp.pipedream.net/v2`
- Authentication handled when adding the MCP server to an MCP-enabled application
- Designed to plug into chat clients and other MCP consumer applications

## Technical Details
- MCP server URL: `https://mcp.pipedream.net/v2`
- Works as a shared, static endpoint for all clients; authentication is per-client during configuration
