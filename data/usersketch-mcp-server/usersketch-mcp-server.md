# UserSketch MCP Server

CRM platform built for async B2C SaaS teams, exposed as an MCP server compatible with MCP-capable clients.

- **Website / Source**: https://mcp.pipedream.com/app/usersketch
- **MCP Server URL**: `https://mcp.pipedream.net/v2`
- **Category**: Business & Commerce – MCP Servers
- **Tags**: CRM, SaaS, Customer Engagement

## Overview
UserSketch MCP Server provides access to UserSketch CRM capabilities through the Model Context Protocol (MCP), enabling MCP-compatible chat clients and tools to interact with an async-focused B2C SaaS CRM.

## Features
- **MCP-compatible CRM server**  
  Exposes UserSketch CRM capabilities via a standard MCP endpoint so multiple MCP clients can connect.

- **Single static server URL**  
  Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) that works across all supported clients.

- **Client-agnostic integration**  
  Designed to be added to various MCP-enabled chat clients and applications using the same server URL.

- **Async B2C SaaS focus**  
  Tailored for CRM workflows used by asynchronous B2C SaaS teams (e.g., customer engagement and relationship management in non-real-time environments).

- **Configuration documentation**  
  Accompanied by a configuration page (via Pipedream) that explains how to add and configure the server in supported clients.

## Integration & Setup
- Use `https://mcp.pipedream.net/v2` as the MCP server endpoint in your MCP-compatible client.
- Authentication is handled when you add the server in your application’s MCP configuration.
- Additional setup details are available on the linked configuration page from the source URL.
