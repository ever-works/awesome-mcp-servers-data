## Overview

Prisma's MCP server gives AI agents the ability to manage Prisma Postgres databases, including spinning up new database instances and running schema migrations. Prisma Client can be used in any Node.js or TypeScript backend application including serverless applications and microservices.

## Features

- **Database Instance Management**: Spin up new Prisma Postgres database instances
- **Schema Migrations**: Run and manage database schema migrations
- **Database Operations**: Execute database queries and operations
- **Type Safety**: Full TypeScript support with type-safe database access
- **ORM Integration**: Leverage Prisma's powerful ORM capabilities through AI

## Supported Backend Types

Prisma Client works with:
- REST APIs
- GraphQL APIs
- gRPC APIs
- Serverless applications
- Microservices
- Any Node.js or TypeScript backend requiring a database

## Use Cases

- Automated database provisioning
- Schema migration management
