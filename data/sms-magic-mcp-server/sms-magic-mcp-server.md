# SMS Magic MCP Server

**Category:** Messaging MCP Servers  
**Brand:** SMS-Magic

SMS Magic MCP Server is an MCP endpoint that connects AI-driven workflows to SMS-Magic, a global text messaging platform for CRM systems. It enables applications using the Model Context Protocol (MCP) to send and manage SMS communications through a standardized server URL.

---

## Features

- **MCP-compatible SMS endpoint**  
  - Exposes SMS-Magic via a standard MCP server interface.  
  - Can be added as an MCP server to supported chat or AI clients.

- **Static server URL**  
  - Single, reusable MCP endpoint URL for all clients:  
    - `https://mcp.pipedream.net/v2`  
  - Simplifies configuration across different tools and environments.

- **Authentication at connection time**  
  - Uses a shared static URL; authentication happens when adding/connecting the server in your application.  
  - Designed to work with multiple clients while keeping credentials separate.

- **Integration with SMS-Magic messaging platform**  
  - Connects to SMS-Magic, which is used for SMS-based customer engagement.  
  - Built for CRM-centric messaging use cases (e.g., customer communication, engagement flows).

- **Global SMS messaging foundation (via SMS-Magic)**  
  - Backed by SMS-Magic’s infrastructure, including:  
    - Access to 59 direct SMS carriers (as stated in source).  
    - Used by 2,000+ clients of varying sizes.  
  - Suitable for large-scale or international text messaging in CRM workflows.

- **Powered by Pipedream Connect**  
  - Provisioned and hosted by Pipedream’s Connect platform.  
  - Centralized configuration page available for detailed setup instructions.

---

## Setup & Configuration

- **MCP server URL**: `https://mcp.pipedream.net/v2`  
- Add this URL as an MCP server in your compatible chat or AI client.  
- Follow your client’s instructions for authentication when connecting the server.  
- Additional configuration details are available on the provider’s configuration page (linked from the source).
