# kwtsms MCP Server

**Description**  
An MCP Server integration for kwtsms that provides programmatic access to Kuwait’s SMS gateway services within MCP-based systems.

**Category**  
Messaging MCP Servers

**Tags**  
- sms  
- messaging  
- gateway

---

## Features

- MCP server integration for the kwtsms SMS gateway.  
- Provides a static MCP server URL usable across compatible MCP clients:  
  - `https://mcp.pipedream.net/v2`  
- Designed to be added into chat or MCP-enabled applications for sending SMS via Kuwait’s SMS provider infrastructure.  
- Authentication handled when adding the server to the client / application (supports authenticated access to the gateway).  
- Hosted and delivered via Pipedream Connect infrastructure.

---

## Integration & Usage

- Add the static MCP server URL to any supported MCP client.  
- Configure authentication within your client when you register the server.  
- Further configuration details are available on the platform’s configuration page (e.g., client-specific setup steps).
