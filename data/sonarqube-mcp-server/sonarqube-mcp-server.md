## Description
A Model Context Protocol (MCP) server that integrates with SonarQube, giving AI assistants structured access to code quality metrics, issues, and analysis results.

> Note: This repository is archived and no longer maintained. The maintained version is at: https://github.com/SonarSource/sonarqube-mcp-server

## Features
- Implements a **Model Context Protocol (MCP) server** interface.  
- **Integrates with SonarQube**, exposing its analysis data to MCP-compatible AI assistants.  
- Provides **structured access to code quality metrics**, issues, and analysis results from SonarQube.  
- Built with **TypeScript**.  
- Targets both **SonarQube** and **SonarCloud** ecosystems (as indicated by repository topics).  
- Versioned releases available (over 40 releases, latest v1.10.21 as of Oct 21, 2025).

## Technical
- Language: **TypeScript**.
