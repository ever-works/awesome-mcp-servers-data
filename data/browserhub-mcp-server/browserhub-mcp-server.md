# Browserhub MCP Server

Easy-to-use, no-code web scraping and data extraction MCP server for integrating Browserhub’s capabilities into MCP-compatible clients.

## Basic Info
- **Name:** Browserhub MCP Server  
- **Category:** Content Extraction & Summarization MCP Servers  
- **Brand:** browserhub  
- **Source URL:** https://mcp.pipedream.com/app/browserhub  
- **Slug:** `browserhub-mcp-server`  
- **Tags:** `web-scraping`, `data-extraction`, `no-code`

## Features
- **No-code web scraping**  
  - Provides web scraping capabilities without requiring programming.  
  - Intended for use via MCP-compatible chat clients and tools.

- **Data extraction**  
  - Extracts structured data from web pages for downstream use by MCP agents and tools.

- **MCP server endpoint**  
  - Static MCP server URL for all supported clients:  
    - `https://mcp.pipedream.net/v2`  
  - Authentication occurs when you add the server to your chosen client or application.

- **Client-agnostic configuration**  
  - Same base MCP server URL works across different chat clients.  
  - Additional, client-specific configuration is available via Pipedream’s configuration documentation.

- **Integration via Pipedream Connect**  
  - Hosted and powered by Pipedream Connect, enabling simple integration workflows for MCP-compatible apps.

## Setup & Configuration
- Use the MCP server URL: `https://mcp.pipedream.net/v2`.  
- Add this server URL to your MCP-compatible chat client or application.  
- Follow the client-specific configuration steps described on the Pipedream **Configuration** page (linked from the source URL page).
