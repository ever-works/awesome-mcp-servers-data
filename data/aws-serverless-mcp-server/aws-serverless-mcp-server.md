# AWS Serverless MCP Server

**Category:** Cloud & DevOps MCP Servers  
**Brand:** AWS  
**Website:** https://modelcontextprotocol.io/mcp/servers/aws-serverless-mcp-server

## Overview
AWS Serverless MCP Server is a Model Context Protocol (MCP) server that manages the full lifecycle of AWS serverless applications. It integrates with the AWS Serverless Application Model (SAM) CLI to support building, deploying, and operating serverless workloads from MCP-compatible clients.

## Features
- **End-to-end serverless lifecycle management** using the AWS SAM CLI
- **Build automation** for AWS serverless applications (e.g., packaging and preparing functions and resources)
- **Deployment workflows** to provision and update serverless stacks on AWS
- **Operations support** for deployed serverless applications via SAM CLI capabilities (such as updates, redeploys, and related operational commands)
- **MCP integration** so tools that speak the Model Context Protocol can interact with AWS serverless projects through this server
- **AWS-native tooling** built around the standard AWS SAM CLI, ensuring compatibility with existing SAM templates and workflows
