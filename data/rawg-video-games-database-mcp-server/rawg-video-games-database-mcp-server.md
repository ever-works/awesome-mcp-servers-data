# RAWG Video Games Database MCP Server

## Overview
RAWG Video Games Database MCP Server exposes RAWG’s large video game database and discovery APIs through the Model Context Protocol (MCP), enabling programmatic search and retrieval of video game data from compatible MCP clients.

- **Name:** RAWG Video Games Database MCP Server  
- **Brand:** RAWG  
- **Category:** Data Access & Integration – MCP Servers  
- **Slug:** `rawg-video-games-database-mcp-server`  
- **Primary Function:** Access and discover video game data via RAWG’s APIs through an MCP-compatible endpoint.

## MCP Server Endpoint
- **MCP Server URL (static, for all clients):**  
  `https://mcp.pipedream.net/v2`
- Authentication is handled when adding the server to your MCP-enabled application.

## Features
- **MCP Integration**  
  - Exposes RAWG’s video game database and discovery capabilities through the Model Context Protocol.  
  - Usable with any MCP-compatible chat or agent client.

- **Unified Server URL**  
  - Single static base URL (`https://mcp.pipedream.net/v2`) for all supported clients.  
  - No need for per-client or per-environment URLs; configuration is simplified to one endpoint.

- **Programmatic Game Data Access**  
  - Designed for programmatic search and retrieval of video game data (e.g., for chatbots, tools, or agents).  
  - Integrates RAWG’s discovery APIs into workflows that use MCP.

- **Client-Agnostic Setup**  
  - Can be added to multiple MCP-enabled chat clients.  
  - Documentation and configuration guidance available via a central configuration page.

- **Pipedream-Hosted**  
  - Delivered via Pipedream Connect infrastructure.  
  - Operated under Pipedream / Workday terms, privacy, and cookie policies.

## Configuration & Integration
- Add the MCP server by pointing your MCP-compatible client to:  
  `https://mcp.pipedream.net/v2`
- Authentication is performed when the server is added to the application.  
- Additional setup details are provided on the product’s configuration page (not included in the source content).
