# Klipfolio MCP Server

**Category:** Data Visualization  
**Brand:** Klipfolio  
**Slug:** klipfolio-mcp-server

## Description
Klipfolio MCP Server is an MCP (Model Context Protocol) server that connects Klipfolio dashboards and reporting capabilities to AI tools. It enables data-driven dashboards and analytics workflows to be accessed and controlled through MCP-compatible AI environments.

Source: https://mcp.pipedream.com/app/klipfolio

## Features
- MCP-based access to Klipfolio
- Connects Klipfolio to AI tools such as Cursor, Claude, and Windsurf
- Enables data-driven dashboards via MCP
- Supports reporting and analytics workflows through AI-assisted tools
