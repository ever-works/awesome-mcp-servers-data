# Uplisting MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Uplisting  
**Slug:** uplisting-mcp-server

## Description
Uplisting MCP Server is an MCP server integration that connects your Uplisting account (property management and short-term rental operations) to AI tools via Pipedream. It enables MCP-compatible agents to work with your property management and STR business operations.

## Features
- Connects Uplisting accounts to MCP-compatible AI tools (e.g., Cursor, Claude, Windsurf)
- Enables AI-driven interaction with Uplisting property management data
- Supports automation around short-term rental (STR) business operations
- Uses Pipedream's MCP server as the integration layer between Uplisting and AI tools
