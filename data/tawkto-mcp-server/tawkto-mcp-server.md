# tawk.to MCP Server

## Summary
MCP Server integration for tawk.to that lets you use tawk.to’s website live chat, ticketing, contact management, and help center capabilities via MCP-compatible clients (e.g., ChatGPT).

## Features
- Website visitor monitoring and live chat
- Respond to support tickets
- Organize and manage contacts
- Create and manage a help center / knowledge base for self-service support
- Central MCP endpoint for clients:
  - Static MCP server URL: `https://mcp.pipedream.net/v2`
- Usable with MCP-enabled chat clients (e.g., ChatGPT via an MCP configuration guide)

## Integration & Configuration
- Connect a tawk.to account through Pipedream’s MCP interface
- Authenticate when adding the MCP server to your application
- Configuration guidance available via the Pipedream MCP configuration page
