## Overview
The Landbot MCP Server integrates Landbotâ€™s no-code, drag-and-drop chatbot builder with MCP, allowing you to design and orchestrate advanced conversational flows and deploy chatbots across multiple channels.

## Features
- **MCP Integration**
  - Exposes Landbot as an MCP Server that can be added to any compatible MCP client.
  - Uses a single static MCP server URL: `https://mcp.pipedream.net/v2`.

- **No-Code Chatbot Builder**
  - Drag-and-drop interface for building conversational flows.
  - Focus on ease, flexibility, and speed without requiring coding.

- **Advanced Conversational Flows**
  - Design complex, branching conversations.
  - Orchestrate automated interactions via MCP.

- **Multi-Channel Deployment**
  - Deploy chatbots on popular messaging channels.
  - Use on the web as:
    - Pop-ups
    - Widgets
    - Landing pages
    - Website embeds

- **Simple Client Setup**
  - One static URL works for every MCP client; authentication handled when adding the server to the application.
  - Additional setup details available via the configuration page in the source.
