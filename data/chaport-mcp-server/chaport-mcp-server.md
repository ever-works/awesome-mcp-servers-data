# Chaport MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Chaport  
**Source:** https://mcp.pipedream.com/app/chaport

## Description
Chaport MCP Server is an integration that connects a Chaport live chat account to MCP-based systems via Pipedream. It exposes website chat, multichannel messaging, and chatbot-related data and actions so they can be accessed and controlled from AI tools and MCP-compatible clients.

## Features
- Connects a Chaport account to Pipedream's MCP server
- Exposes website live chat data to MCP-based tools
- Supports multichannel messaging integration (e.g., web chat and other connected channels)
- Provides access to chatbot-related data and actions from MCP clients
- Enables use of Chaport inside AI tools such as Cursor, Claude, and Windsurf

## Integrations
- Pipedream MCP Server
- Chaport live chat
- AI tools and editors (e.g., Cursor, Claude, Windsurf)
