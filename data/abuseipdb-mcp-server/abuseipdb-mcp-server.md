# AbuseIPDB MCP Server

**Category:** Security & Attestation MCP Servers  
**Brand:** AbuseIPDB  
**Slug:** `abuseipdb-mcp-server`

AbuseIPDB MCP Server provides access to AbuseIPDB’s centralized blacklist and IP reputation data, enabling applications and chat clients to look up IP addresses associated with malicious activity.

---

## Features

- **Centralized IP blacklist access**  
  - Query IP addresses against AbuseIPDB’s central blacklist.  
  - Retrieve reputation information for IPs associated with malicious or suspicious activity.

- **Threat intelligence integration**  
  - Leverages AbuseIPDB’s crowdsourced reports from webmasters, system administrators, and security teams.  
  - Designed to help identify abusive hosts and improve web and infrastructure security.

- **Static MCP server endpoint**  
  - Single, static MCP server URL for all compatible clients:  
    - `https://mcp.pipedream.net/v2`  
  - Authentication is performed when adding the server within the client or application.

- **MCP-compatible for chat and tools**  
  - Intended for use with MCP-capable chat clients and applications.  
  - Can be added as a server to provide IP reputation checks directly within the client environment.

- **Configuration guidance**  
  - Full configuration details available via the linked configuration page in the original source (for supported clients and setup steps).

- **Hosted by Pipedream Connect**  
  - Delivered as a managed MCP server via Pipedream’s infrastructure.

---

## MCP Server URL

```text
https://mcp.pipedream.net/v2
```

Use this URL when adding the AbuseIPDB MCP Server to your MCP-compatible client or application. Authentication is handled during client configuration.
