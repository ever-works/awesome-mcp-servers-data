# Pointagram MCP Server

## Overview
Pointagram MCP Server exposes Pointagram’s gamification and team motivation tools through the Model Context Protocol (MCP), allowing AI agents and compatible clients to work with points, goals, and achievements.

- **Category:** Business & Commerce – MCP Servers  
- **Use cases:** Gamifying team activities, tracking goals, and managing achievements via MCP-enabled chat or workflow clients.
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server** that can be added to any MCP-capable client via a single static URL.
- **Access to Pointagram gamification tools** for managing:
  - Points 
  - Goals
  - Achievements
- **Team motivation and engagement focus**, enabling workflows that keep teams goal-oriented.
- **Celebration of performance** by surfacing achievements and progress in MCP-integrated applications.
- **Hosted by Pipedream Connect**, so you don’t need to run your own server implementation.
- **Static, reusable server URL** that works across different clients; authentication is handled when adding the server to your app.

## Integration
- Add the MCP server to compatible chat or MCP clients using:
  - **Server URL:** `https://mcp.pipedream.net/v2`
- Configuration details and client-specific setup are provided on the Pipedream **Configuration** page.
