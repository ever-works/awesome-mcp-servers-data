## Overview
The VIES API MCP Server exposes the EU VAT Information Exchange System (VIES) via the MCP protocol, enabling MCP-compatible agents and applications to validate and query EU VAT information.

## Features
- MCP-compatible server endpoint for VIES (VAT Information Exchange System).  
- Supports validation and querying of EU VAT information.  
- Single static MCP server URL (`https://mcp.pipedream.net/v2`) usable across different clients.  
- Authentication handled when adding/configuring the server in client applications.  
- Documentation and setup guidance available via a configuration page in Pipedream Connect.

## Integration
- Can be added to MCP-enabled chat or agent clients using the static server URL.  
- Configuration and connection managed through Pipedream Connect.
