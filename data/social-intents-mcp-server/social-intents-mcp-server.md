## Overview
The Social Intents MCP Server is an integration that exposes the Social Intents live chat and social apps platform to AI tools via the Model Context Protocol (MCP). It allows AI-powered applications to interact with website visitors and provide customer service through collaboration platforms like Microsoft Teams, Slack, Zoom, and Webex.

## Features
- **MCP-compatible endpoint**
  - Static MCP server URL: `https://mcp.pipedream.net/v2`
  - Works as a shared endpoint for all MCP-compatible clients.

- **Live chat integration**
  - Connects AI tools to Social Intents live chat on websites.
  - Enables engagement with website visitors for support and sales use cases.

- **Omnichannel collaboration support**
  - Operates through major collaboration platforms:
    - Microsoft Teams
    - Slack
    - Zoom
    - Webex

- **Customer service and sales workflows**
  - Designed to handle customer service conversations.
  - Supports sales-oriented chat with website visitors.

- **Client-agnostic configuration**
  - Same MCP server URL for all clients.
  - Authentication performed when adding the server in each client.

- **Documentation and configuration guidance**
  - Central configuration instructions available via a dedicated configuration page (outside this summary).

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Protocol:** Model Context Protocol (MCP)
- **Integration platform:** Powered by Pipedream Connect
