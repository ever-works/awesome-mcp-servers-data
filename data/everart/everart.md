# EverArt

## Overview
EverArt is an MCP (Model Context Protocol) server for AI image generation that exposes multiple EverArt image models as tools LLMs can call to create and customize images programmatically.

## Features
- MCP server implementation for AI image generation
- Integration with multiple EverArt image models
- Programmatic image creation from within LLM workflows
- Programmatic image customization (e.g., adjusting outputs via parameters/prompts)
- Designed to be called by LLMs as a tool, enabling automated image generation in apps and agents
