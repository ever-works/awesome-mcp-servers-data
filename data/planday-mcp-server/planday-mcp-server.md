# Planday MCP Server

**Category:** Business & Commerce · MCP Servers  
**Tags:** workforce-management, scheduling, HR

Planday MCP Server is an integration endpoint that exposes Planday’s workforce management and scheduling capabilities through the MCP (Model Context Protocol) ecosystem, allowing compatible chat or MCP-enabled applications to interact with Planday.

---

## Features

- **MCP-based integration**  
  - Provides an MCP Server interface for Planday.  
  - Designed to be added to any MCP-compatible client or chat application.

- **Workforce management access**  
  - Enables access to Planday workforce management functionality via MCP (exact operations not detailed on the page).

- **Scheduling capabilities**  
  - Exposes Planday scheduling features through MCP (e.g., shift / schedule management capabilities, as supported by Planday’s API and MCP tools).

- **Static MCP server endpoint**  
  - Single static URL for all clients:  
    - `https://mcp.pipedream.net/v2`  
  - Same endpoint can be reused across different MCP clients.

- **Authentication at client setup**  
  - Authentication is performed when adding the server to an application (no credentials embedded in the base URL).

- **Configuration resources**  
  - Guided setup for various chat clients.  
  - Central configuration documentation available via the “Configuration” page.

---

## Technical Details

- **Base MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Provider:** Pipedream Connect  
- **Integrates with:** Planday (workforce management and scheduling platform)
