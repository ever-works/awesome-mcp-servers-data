# Webvizio MCP Server

**Category:** Content Management MCP Servers  
**Brand:** Webvizio  
**Source URL:** https://mcp.pipedream.com/app/webvizio

Webvizio MCP Server connects Webvizio’s website feedback and review capabilities to MCP-compatible chat clients, enabling real‑time collaboration on website revisions.

---

## Features

- **MCP integration endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same URL works for every client; authentication occurs when adding the server to your application.

- **Webvizio website feedback & review**  
  - Connects to Webvizio, a tool for collecting website feedback.  
  - Supports website review workflows for teams and managers.  
  - Built for collaboration on website revisions in real time.

- **Chat client compatibility**  
  - Can be added to supported chat clients as an MCP server.  
  - Central configuration guidance available via the platform’s Configuration page.

- **Collaboration-focused**  
  - Designed for managers and teams working together on website changes.  
  - Enables feedback, review, and collaboration actions to be triggered through MCP.

---

## Usage

- Add the MCP server to your MCP-compatible client using:  
  `https://mcp.pipedream.net/v2`
- Authenticate within your application when prompted.
