## Overview

Unity MCP provides Model Context Protocol access to Unity Editor features and runtime game control, allowing AI agents to interact with Unity projects programmatically.

## Features

- Editor automation: menu execution, asset management, scene querying
- Runtime game integration for deployed Unity builds
- WebSocket-based communication for real-time AI control
- Open-source for custom extensions

## Use Cases

- AI-driven game design and testing
- Procedural content generation
- Automated playtesting

Free and open-source.
