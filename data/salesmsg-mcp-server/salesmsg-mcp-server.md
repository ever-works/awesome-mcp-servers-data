# Salesmsg MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Salesmsg  
**Slug:** `salesmsg-mcp-server`

## Description
Salesmsg MCP Server is an MCP (Model Context Protocol) integration that connects Salesmsg’s communication platform with MCP-compatible clients. It enables 2-way SMS texting, SMS marketing, and calling workflows to be used directly within your MCP-enabled applications via a single static server endpoint.

## Features
- **MCP Server Integration**
  - Exposes Salesmsg functionality through a standard MCP server endpoint: `https://mcp.pipedream.net/v2`.
  - Single static URL usable across different MCP clients; authentication is handled when adding the server to the client.

- **2-Way SMS Texting**
  - Send and receive SMS messages through Salesmsg from MCP-connected applications.
  - Supports conversation-style messaging for customer or contact communication.

- **SMS Marketing**
  - Use Salesmsg’s SMS capabilities for marketing and promotional messaging via MCP-integrated workflows.

- **Calling Features**
  - Access Salesmsg calling features (e.g., placing and receiving calls) through MCP-based integrations.

- **Multi-Client Compatibility**
  - Designed to be added to various MCP-compatible chat clients.
  - Central configuration via a single endpoint, with setup details available in the referenced configuration documentation.

## Integration Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Authentication:** Performed when adding the server within your MCP client (details vary by client).
- **Provider:** Implemented and hosted via Pipedream Connect.
