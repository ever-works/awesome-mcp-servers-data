# Float MCP Server

**Category:** Project Management MCP Servers  
**Brand:** Float  
**Source:** https://mcp.pipedream.com/app/float

## Overview
The Float MCP Server is an integration that connects your Float account to AI tools via Pipedream’s MCP server, enabling resource and project-management capabilities to be accessed directly from compatible MCP tools.

## Features
- Connects a Float account through Pipedream’s MCP server.
- Exposes Float’s resource-management data to MCP-compatible tools.
- Exposes Float’s project-management data to MCP-compatible tools.
- Supports workload synchronization across tools using Float data.
- Enables AI tools to access planning and scheduling information from Float.
- Facilitates profitability and workload insights within MCP-based workflows.
- Integrates with AI tools such as:
  - Cursor
  - Claude
  - Windsurf

## Use Cases
- Synchronizing project and resource schedules between Float and AI development tools.
- Pulling team workload and availability data into AI-assisted planning workflows.
- Accessing Float project and resource information from within MCP-enabled IDEs and assistants.
