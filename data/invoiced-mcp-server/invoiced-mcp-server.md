# Invoiced MCP Server

Accounts receivable and billing automation MCP server that connects MCP-based systems to Invoiced via Pipedream Connect.

- **Item Type:** MCP Server / Integration
- **Category:** Business & Commerce MCP Servers
- **Brand:** Invoiced
- **Source / Docs:** https://mcp.pipedream.com/app/invoiced
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Description
The Invoiced MCP Server exposes Invoiced’s accounts receivable and billing automation capabilities to MCP-compatible clients through a static Pipedream MCP endpoint. Authentication is handled when adding the server to your application.

## Features
- **Static MCP Endpoint**  
  - Single MCP server URL (`https://mcp.pipedream.net/v2`) usable across all clients.
- **Invoiced Integration**  
  - Connects MCP-based applications to Invoiced’s accounts receivable and billing automation platform (e.g., invoicing, billing, collections) via Pipedream Connect.
- **Client-Agnostic Setup**  
  - Designed to be added to different MCP-compatible chat clients and applications.
- **Configurable via Pipedream**  
  - Configuration and setup managed through Pipedream’s Connect and configuration pages.
- **Authentication at Connection Time**  
  - Users authenticate to Invoiced when adding the MCP server in their client (no per-client URL changes required).

## Usage
- Add the MCP server URL `https://mcp.pipedream.net/v2` to your MCP-compatible client or application.
- Complete authentication to Invoiced when prompted by your client.
- Use the client’s MCP capabilities to access Invoiced accounts receivable and billing automation features that the server exposes.
