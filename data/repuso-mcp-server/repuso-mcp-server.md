# Repuso MCP Server

Review management MCP Server integration for Repuso, enabling MCP-compatible tools to manage and monitor social reviews and feedback across platforms such as Facebook.

## Overview
- **Type:** MCP Server integration

- **Provider / Brand:** Repuso (via Pipedream Connect)
- **Purpose:** Connect MCP-compatible chat clients and tools to Repuso to monitor and work with social media reviews and feedback.

## Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`  
  - Use the same endpoint across different chat clients or MCP tools.

- **Review monitoring integration**  
  - Integrates with Repuso, a review management solution.  
  - Helps monitor feedback and reviews from social media platforms, including Facebook.

- **Client-agnostic setup**  
  - Designed to be added to multiple MCP-compatible chat clients.  
  - Authentication occurs when adding the server in the client.

- **Centralized configuration docs**  
  - Configuration instructions available via a dedicated configuration page on Pipedream Connect.

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Integration platform:** Pipedream Connect
- **Use case focus:** Social review and feedback management from platforms like Facebook via Repuso.
