# Emailoctopus MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** EmailOctopus  
**Slug:** `emailoctopus-mcp-server`

Emailoctopus MCP Server is an MCP (Model Context Protocol) server running on Pipedream that exposes EmailOctopus email marketing capabilities—such as managing campaigns, subscribers, and sending rich HTML emails—so they can be used as tools inside compatible chat or AI clients.

---

## Features

- **MCP Server Endpoint**  
  - Static MCP server URL for all clients:  
    - `https://mcp.pipedream.net/v2`  
  - Authentication handled when adding the server to your application or client.

- **Email Marketing Integration (via EmailOctopus APIs)**  
  - Connects to EmailOctopus email marketing services through Pipedream.  
  - Intended to support:
    - Managing email marketing campaigns.  
    - Managing subscribers / mailing lists.  
    - Sending HTML email campaigns.  
  - Leverages EmailOctopus’s simple, intuitive UI and rich HTML email capabilities (when used via the core EmailOctopus platform alongside the MCP tools).

- **Client & Configuration Support**  
  - Designed to be added to compatible chat / AI clients as an MCP server.  
  - Central configuration reference available through Pipedream’s MCP Configuration page.

- **Infrastructure & Hosting**  
  - Powered and hosted by Pipedream Connect MCP infrastructure.  
  - Uses a single static URL for all MCP clients, simplifying setup.

---

## Setup

1. Use the MCP server URL: `https://mcp.pipedream.net/v2`.  
2. Add this server URL in your MCP-compatible chat or AI client.  
3. Complete authentication when prompted by your client during server addition.
