## Siteleaf MCP Server

**Category:** Content Management MCP Servers  
**Website:** https://mcp.pipedream.com/app/siteleaf

### Description
The Siteleaf MCP Server is a Pipedream MCP integration that connects Siteleaf’s CMS to AI-driven development and content tools. It is designed for static sites using Jekyll and integrates with GitHub for content and code synchronization.

### Features
- MCP-based access to Siteleaf’s CMS for static websites
- Integration with Pipedream’s MCP server
- Works with AI tools such as Cursor, Claude, and Windsurf
- Supports static site workflows built with Jekyll
- GitHub sync support for content and site source
