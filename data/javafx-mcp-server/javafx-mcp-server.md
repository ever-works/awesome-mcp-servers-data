## JavaFX MCP Server

**Description**  
JavaFX MCP Server is a Model Context Protocol (MCP) server that exposes a JavaFX canvas so AI assistants and LLMs can programmatically create drawings and visualizations using the JavaFX framework.

### Features
- **MCP-compliant server** – Implements the Model Context Protocol (MCP) so it can be used as a tool by compatible AI assistants and development environments.
- **JavaFX canvas exposure** – Provides an accessible JavaFX drawing canvas that LLMs can control programmatically.
- **Programmatic drawing and visualization** – Allows LLMs to generate and manipulate visual content (e.g., custom drawings and data visualizations) via JavaFX APIs rather than manual user interaction.
