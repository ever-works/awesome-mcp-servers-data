# FlippingBook MCP Server

**Category:** Content Management MCP Servers  
**Brand:** FlippingBook  
**Slug:** `flippingbook-mcp-server`

FlippingBook MCP Server connects your FlippingBook account to MCP-compatible AI tools so they can create and work with digital flipbooks generated from PDFs, and access related analytics.

## Features

- **MCP Integration**
  - Exposes FlippingBook functionality through an MCP (Model Context Protocol) server.
  - Allows AI tools that support MCP (such as Cursor, Claude, and Windsurf) to interact with your FlippingBook account.

- **PDF to Digital Flipbook Conversion**
  - Converts PDF files into interactive, digital flipbooks.
  - Makes flipbooks accessible programmatically via MCP-enabled tools.

- **Analytics Access**
  - Provides access to FlippingBookâ€™s flipbook analytics through MCP.
  - Enables AI tools to read and use engagement and performance data for generated flipbooks.

- **Sharing and Management**
  - Allows MCP tools to generate, share, and track digital flipbooks linked to your FlippingBook account.

## Integrations

- **AI / MCP Tools**
  - Cursor
  - Claude
  - Windsurf
  - Any other tools that support the MCP standard and connect via the Pipedream MCP server.
