# Breeze MCP Server

**Category:** Project Management MCP Servers  
**Brand:** Breeze  
**Source:** https://mcp.pipedream.com/app/breeze

## Overview
Breeze MCP Server exposes the Breeze project management platform to MCP-compatible assistants, enabling them to plan, track, and organize work using Breezeâ€™s APIs.

## MCP Server URL
- **Static MCP server URL:** `https://mcp.pipedream.net/v2`
- This URL is used for all MCP-compatible clients.
- Authentication is performed when adding the server to your application.

## Features
- Integration with Breeze project management, which teams use to:
  - Plan work
  - Track work
  - Organize tasks and projects
- MCP-compatible interface, allowing assistants and clients that support MCP to interact with Breeze via its APIs.
- Single static server URL usable across different MCP clients.

## Setup / Configuration
- Add the MCP server to your chat or MCP-compatible application using:
  - **Server URL:** `https://mcp.pipedream.net/v2`
- Client-specific setup instructions are available via the associated configuration documentation (select your chat client or use the full configuration page on the source site).
