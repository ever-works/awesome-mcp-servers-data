# SARE MCP Server

Omnichannel marketing and marketing automation MCP server, provided via Pipedream Connect.

## Overview
The SARE MCP Server exposes SARE’s omnichannel marketing and marketing automation capabilities through a standardized MCP endpoint that can be connected to compatible chat or agent clients.

## MCP Server URL
- Base MCP server URL (static for all clients):
  - `https://mcp.pipedream.net/v2`
- Authentication is performed when adding the server within your client/application.

## Features
- Access to SARE’s omnichannel marketing platform via MCP
- Integration with marketing automation capabilities
- Single, static MCP endpoint usable across different clients
- Compatible with multiple chat / agent clients that support MCP
- Configuration documentation available via Pipedream’s configuration page

## Integration & Setup
- Add the MCP server URL to any supported MCP client
- Authenticate within your client when prompted
- Reference the configuration guide on Pipedream for client-specific setup steps
