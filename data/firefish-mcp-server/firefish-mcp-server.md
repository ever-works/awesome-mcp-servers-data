# Firefish MCP Server

**Category:** Business & Commerce – MCP Servers  
**Tags:** recruiting, CRM, HR

## Overview
Firefish MCP Server is an MCP (Model Context Protocol) server integration for Firefish, a recruitment CRM focused on maximizing recruiter revenue by using data-driven workflows at the core of recruitment agency operations. It is hosted via Pipedream and provides a static MCP endpoint that can be added to compatible chat or AI clients.

## Features
- **Recruitment CRM integration**
  - Connects to Firefish, a recruitment-focused CRM system.
  - Designed for recruitment agencies and HR teams.
  - Centers recruitment operations around data and structured workflows.

- **MCP server endpoint**
  - Single static MCP server URL for all clients:
    - `https://mcp.pipedream.net/v2`
  - Authentication occurs when adding the server within the client/application.
  - Compatible with MCP-capable chat or AI clients.

- **Account configuration**
  - "Configure Firefish" flow to connect a Firefish account.
  - Client selection step to tie a specific Firefish client/tenant to the MCP server.
  - Status feedback such as “Checking your account…” during setup.

- **Tooling and actions (via Pipedream)**
  - Exposes available tools/actions for Firefish through the MCP interface (listed as "Available tools").
  - Tools are dynamically loaded ("Loading actions...", "Loading available tools...") from the Pipedream app configuration.

- **Client integration guidance**
  - Instructions to add the MCP server to supported chat clients.
  - Link to a full configuration page on Pipedream for detailed setup.

## Technical Details
- **Protocol:** Model Context Protocol (MCP) server.
- **Endpoint:** `https://mcp.pipedream.net/v2` (static for all clients).
- **Hosting platform:** Pipedream.
- **Domain focus:** Human Resources / Recruitment.
