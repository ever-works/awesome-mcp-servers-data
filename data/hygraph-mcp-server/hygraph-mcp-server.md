# Hygraph MCP Server

**Category:** Content Management MCP Servers  
**Brand:** Hygraph  
**Source URL:** https://mcp.pipedream.com/app/hygraph  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Overview
Hygraph MCP Server is an MCP (Model Context Protocol) server that connects MCP-compatible clients to Hygraph’s federated content platform. It enables unified access to content, along with tools to structure, enrich, and distribute content across different channels and systems.

## Features
- **MCP-compatible server endpoint**
  - Provides a static MCP server URL (`https://mcp.pipedream.net/v2`) that works across supported clients.
  - Authentication is handled when adding the server in the client/application.

- **Federated content access**
  - Connects MCP clients to Hygraph’s federated content platform.
  - Supports unifying content from multiple sources into a single access layer.

- **Content structuring and enrichment**
  - Allows clients to structure content models and interact with structured content through MCP.
  - Enables enrichment of content pulled from various systems via Hygraph’s platform capabilities.

- **Content distribution**
  - Facilitates distributing content from Hygraph to multiple destinations (“from anywhere to anywhere”) via integrated clients and workflows.

- **Client integration guidance**
  - Works with multiple chat or MCP clients; users can select their client and follow configuration instructions (via the associated configuration page).
