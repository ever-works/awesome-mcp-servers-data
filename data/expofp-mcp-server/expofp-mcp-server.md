# ExpoFP MCP Server

## Overview
ExpoFP MCP Server is a Model Context Protocol (MCP) server that connects to ExpoFP, enabling applications (such as chat clients) to create and interact with interactive expo and trade show floor plans programmatically.

- **Category:** Content Management MCP Servers  
- **Use cases:** Events, trade shows, interactive maps, venue and booth layout management.

## Features
- **ExpoFP integration**  
  - Connects your application to ExpoFP to work with interactive expo and trade show floor plans.  
  - Uses a single static MCP server URL for all ExpoFP clients.

- **MCP server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`.  
  - Authentication is handled when you add the server to your application or chat client.

- **Client-agnostic setup**  
  - Works with multiple chat clients or MCP-compatible applications.  
  - Configuration instructions are available per client type via the configuration flow.

- **Configuration workflow**  
  - Connect your ExpoFP account and select your client to begin using the tools.  
  - Centralized configuration reference available via a dedicated Configuration page.

- **Interactive tooling**  
  - Exposes a set of “tools” (MCP actions) for working with ExpoFP (loaded dynamically in the interface).  
  - Designed for programmatic interaction with expo floor plans (e.g., creating and managing interactive layouts and maps).
