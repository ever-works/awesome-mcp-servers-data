# Sierra Interactive MCP Server

**Category:** Business & Commerce · MCP Servers  
**Tags:** Real estate, Lead management, CRM

## Overview
Sierra Interactive MCP Server connects MCP-compatible clients to Sierra Interactive’s real estate lead generation and management platform. It enables automated handling of real estate leads and follow-ups through a unified MCP server endpoint.

## Features
- **MCP integration endpoint**  
  - Provides a static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Single endpoint usable across different clients and applications.

- **Authentication per client/app**  
  - Authentication occurs when adding the server to an MCP-compatible application.  
  - Same URL can be reused while maintaining client-specific authentication.

- **Client selection and configuration**  
  - Connect a Sierra Interactive account via the Pipedream interface.  
  - Select the target client after account connection to finalize configuration.

- **Tooling / actions via MCP**  
  - Exposes “available tools” (actions) through the MCP interface for use in chat clients and other MCP applications.  
  - Designed to automate aspects of real estate lead generation and lead management workflows.

- **Multi-chat-client support**  
  - Can be added to multiple chat clients that support MCP.  
  - Each chat client can follow specific setup instructions for connecting to the server.

- **Centralized configuration documentation**  
  - Additional setup and configuration guidance available on a dedicated Configuration page (via Pipedream).
