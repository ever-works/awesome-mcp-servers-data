## Description

The Alteryx Analytics Cloud MCP Server provides a standardized MCP (Model Context Protocol) endpoint that connects MCP-compatible clients to Alteryx Analytics Cloud. It enables automated data preparation, analytics, and workflow execution from within MCP-based tools and chat clients.

---

## Features

- **MCP-Compatible Endpoint**  
  - Single static MCP server URL usable across clients:  
    - `https://mcp.pipedream.net/v2`
  - Designed to be added to various MCP-aware chat clients and applications.

- **Authentication at Connection Time**  
  - Authentication handled when adding the server to the client or application (per-client authentication flow rather than per-URL).

- **Integration with Alteryx Analytics Cloud**  
  - Access Alteryx Analytics Cloud capabilities through MCP-based tools.  
  - Supports automated access to data preparation, analytics, and workflow execution (as provided by Alteryx Analytics Cloud).

- **Multi-Client Setup Guidance**  
  - Instructions available per chat client for adding the MCP server.  
  - Centralized configuration documentation via a dedicated configuration page.

- **Pipedream Connect Backing**  
  - MCP server powered by Pipedream Connect infrastructure.
