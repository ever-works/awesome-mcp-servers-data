# Drift MCP Server

**Category:** Business & Commerce MCP Servers  
**Provider:** Pipedream  
**Integration Type:** MCP Server for Drift on Pipedream

Drift MCP Server is an MCP (Model Context Protocol) server integration that connects Pipedream workflows to Drift’s B2B conversational sales and marketing platform.

---

## Features

- **Static MCP Server Endpoint**  
  - Uses a single static MCP server URL for all clients:  
    `https://mcp.pipedream.net/v2`  
  - Same endpoint can be reused across different compatible chat or MCP clients.

- **Per-Client Authentication**  
  - Authentication is performed when adding the server to each client/application.  
  - Supports secure connection of individual chat clients to Drift via Pipedream.

- **Chat Client Integration**  
  - Designed to be added to compatible chat-based applications as an MCP server.  
  - Workflow: select your chat client → add the MCP server URL → authenticate → connect to Drift.

- **Configuration Documentation**  
  - A dedicated configuration page (linked from the app) provides setup instructions for different clients.  
  - Centralized reference for connection details and options.

- **Pipedream Connect Integration**  
  - Built on top of **Pipedream Connect**, enabling integration of Drift with broader Pipedream workflows and automations.  
  - Intended for B2B conversational sales and marketing use cases involving Drift.
