# Workamajig MCP Server

## Overview
The Workamajig MCP Server is an integration endpoint that exposes core Workamajig marketing and creative project management capabilities as MCP-compatible tools, allowing you to connect a Workamajig account to supported chat or MCP client applications.

## Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Works for all clients; authentication occurs when adding the server to your MCP-compatible application.

- **Account connection**  
  - Connect a Workamajig account via Pipedream’s interface.  
  - Select and manage a specific client within Workamajig once connected.

- **Available tools (actions)**  
  The server currently exposes three Workamajig actions as tools:
  - **Update Contact**  
    - Update details for an existing contact in Workamajig.  
    - Useful for keeping client and stakeholder records current from within an MCP client.
  - **Create Company**  
    - Create a new company record in Workamajig.  
    - Supports adding new organizations to your Workamajig CRM and project pipeline.
  - **Create Activity**  
    - Create a new activity in Workamajig.  
    - Enables logging or initiating new work items or interactions directly through MCP-compatible chat interfaces.

- **Client integration guidance**  
  - Instructions and a guide for adding the MCP server to ChatGPT (OpenAI).  
  - Additional configuration details available via the Pipedream MCP configuration page.

## Integrations
- **Workamajig** – marketing and creative project management platform.  
- **Pipedream MCP** – hosting and exposing Workamajig actions as MCP tools.  
- **Chat / MCP clients** – e.g., ChatGPT (OpenAI), via documented setup guides.
