# BuildChatbot MCP Server

No-code MCP server integration for creating and managing chatbots that use private data sources.

## Overview
- **Type:** MCP server / AI integration
- **Category:** AI Integration – MCP Servers
- **Provider / Brand:** BuildChatbot (via Pipedream Connect)
- **Primary Use Case:** Let MCP-compatible agents create and manage no-code chatbots powered by private data.
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **No-code chatbot creation**
  - Build conversational experiences without writing code.
  - Designed for businesses to deploy chatbots quickly.
- **Private data integration**
  - Chatbots can leverage private / internal data sources (as described in product summary).
- **MCP-compatible**
  - Exposes capabilities via the Model Context Protocol so MCP-compatible agents and clients can:
    - Connect to the BuildChatbot server
    - Create and manage chatbots programmatically (as implied by product description)
- **Static server endpoint**
  - Single static MCP server URL: `https://mcp.pipedream.net/v2`.
  - Same endpoint works across supported MCP client applications.
- **Client-agnostic setup**
  - Intended to be added to different MCP-compatible chat clients.
  - Configuration details available via the linked configuration page (per site content).
- **Authentication at connection time**
  - Authentication is handled when adding the MCP server to your application (no per-client URL difference noted).
- **Built on Pipedream Connect**
  - Operates as an app/server within the Pipedream Connect ecosystem.

## Integration & Configuration
- **Add to your app**
  - Use the static MCP URL in your MCP-compatible client.
  - Follow client-specific instructions (referenced “Select your chat client to learn how to get started” and configuration page on the site).
