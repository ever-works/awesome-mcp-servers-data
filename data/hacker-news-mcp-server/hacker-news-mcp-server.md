# Hacker News MCP Server

**Category:** Web Search MCP Servers  
**Brand:** Hacker News  
**Source URL:** https://mcp.pipedream.com/app/hacker_news  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Description
The Hacker News MCP Server is an MCP-compatible service that exposes data from the Hacker News social news website. It enables applications and chat clients to query and automate interactions with Hacker News content through the MCP protocol.

## Features
- Provides access to Hacker News data via the MCP protocol.
- Works as a static MCP server endpoint usable by any compatible client.
- Supports authentication at the time you add the server to your application.
- Designed for use within chat-based or MCP-enabled applications.
- Integrated and powered by Pipedream Connect infrastructure.
- Central configuration guidance available via a dedicated configuration page.

## Usage
- Use the static MCP server URL `https://mcp.pipedream.net/v2` when configuring your MCP-enabled client.
- Authenticate within your client when adding the server.
