# Pocket MCP Server

Service for managing a reading list of articles and videos via an MCP-compatible server.

## Overview
Pocket MCP Server is an MCP server that connects to Pocket, allowing MCP-compatible tools and chat clients to manage and retrieve saved articles and videos from a Pocket reading list.

- **Name:** Pocket MCP Server  
- **Category:** Content management MCP servers  
- **Brand:** Pocket  
- **Source URL:** https://mcp.pipedream.com/app/pocket  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- Connects MCP-compatible tools and chat clients to a Pocket account.  
- Manages a reading list of articles and videos stored in Pocket.  
- Enables retrieval of saved articles and videos from Pocket reading lists.  
- Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) for all clients.  
- Authentication is handled when adding the server to an application or chat client.  
- Can be integrated with multiple chat clients that support MCP by adding the same server URL.  
- Configuration details available on a dedicated configuration page (via the Pipedream Connect site).

## Integration & Usage
- Add the static MCP server URL to any MCP-compatible chat client or application.  
- Authenticate with your Pocket account during setup in the client.
