# SharpSpring MCP Server

MCP Server integration for SharpSpring’s marketing automation platform, allowing MCP-based tools and agents to access marketing, CRM, and automation features programmatically.

- **Website / Source**: https://mcp.pipedream.com/app/sharpspring
- **Category**: Business & Commerce · MCP Servers
- **Brand**: SharpSpring
- **Slug**: `sharpspring-mcp-server`
- **Tags**: marketing, CRM, automation

## Features

- **MCP-based integration**
  - Exposes SharpSpring’s marketing automation and CRM capabilities to MCP-compatible tools and agents.
  - Designed to be added as an MCP server within compatible chat or agent clients.

- **Static MCP server endpoint**
  - Server URL: `https://mcp.pipedream.net/v2`
  - Single static URL used for all clients; authentication is handled when adding the server to your application.

- **Client-agnostic setup**
  - Can be added to multiple chat or agent clients that support MCP servers.
  - Configuration details are documented on the Pipedream MCP Configuration page (external link from source).

- **Pipedream Connect integration**
  - MCP server is powered by Pipedream Connect infrastructure.

## Authentication & Configuration

- Authentication occurs when adding the MCP server to your application; the same base URL is reused for all clients.
- Additional configuration steps and client-specific instructions are available via the referenced configuration page on Pipedream.
