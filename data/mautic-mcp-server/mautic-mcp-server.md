# Mautic MCP Server

## Overview
Mautic MCP Server is an MCP (Model Context Protocol) server that connects the open source Mautic marketing automation platform to MCP-compatible agents and chat clients. It enables automation of campaigns and contact-related workflows via an MCP endpoint.

- **Name:** Mautic MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Brand:** Mautic  
- **Source URL:** https://mcp.pipedream.com/app/mautic  
- **MCP Endpoint URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP Server for Mautic**  
  Exposes Mautic, an open source marketing automation platform, via the Model Context Protocol.

- **Campaign Automation Integration**  
  Allows MCP agents / clients to interact with Mautic for campaign-related automation tasks.

- **Contact Automation Integration**  
  Provides access to contact-related automation capabilities in Mautic (e.g., managing or triggering contact workflows) through MCP.

- **Static MCP Server URL**  
  Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) that works for all compatible clients.

- **Client-Agnostic Setup**  
  Designed to be added to various MCP-enabled chat clients; authentication is performed when the server is added to the client.

- **Hosted by Pipedream Connect**  
  Operates as a Pipedream-hosted MCP server, using Pipedream’s infrastructure for connection and configuration.

## Usage
- Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server to your MCP-compatible app or chat client.
- Authenticate during the setup flow in your client.
