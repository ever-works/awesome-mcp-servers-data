# bot9 MCP Server

AI MCP server integration for using bot9’s AI chatbot builder within MCP-compatible clients for customer support automations.

## Overview
- **Type:** MCP server / integration
- **Category:** AI Integration – MCP Servers
- **Provider / Brand:** bot9 (hosted via Pipedream)
- **Use Case:** Connect MCP tools and chat clients to bot9’s AI chatbot builder to power customer support interactions.

## Features
- **MCP server endpoint**  
  - Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`  
  - Works across different MCP-compatible applications.

- **Authentication-based access**  
  - Single shared URL for all clients, with authentication handled when adding the server to your app.

- **Client-agnostic setup**  
  - Designed to be added to various chat clients that support MCP.  
  - Setup guidance is available per chat client type.

- **Integration with bot9 AI chatbot builder**  
  - Allows MCP tools to call into bot9’s AI chatbot capabilities.  
  - Intended specifically for customer support use cases.

- **Configuration guidance**  
  - Option to connect a bot9 account and select a client for configuration.  
  - Further details via a dedicated configuration page (for full setup steps and options).
