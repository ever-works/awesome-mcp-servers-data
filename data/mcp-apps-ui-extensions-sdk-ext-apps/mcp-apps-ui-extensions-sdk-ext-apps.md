## Overview

MCP Apps / UI Extensions SDK (ext-apps) is the official SDK for building Apps and UI extensions around the Model Context Protocol (MCP) ecosystem. It is intended to help developers create richer user interfaces on top of existing MCP servers.

## Features

- **Official SDK for MCP Apps/UI Extensions**  
  - Designed specifically for building Apps and UI extensions in the Model Context Protocol ecosystem.

- **Integration with MCP Servers**  
  - Supports building richer interfaces on top of MCP servers (as described in the project summary).

- **Documentation**  
  - Dedicated `docs/` directory for project documentation.

- **Examples**  
  - `examples/` directory to demonstrate how to use the SDK in practice.

- **Specification Drafts**  
  - `specification/draft/` directory containing draft specifications related to Apps and UI extensions in the MCP ecosystem.

- **Source Code Structure**  
  - `src/` directory containing the SDK implementation.  
  - `scripts/` directory for project-related scripts (e.g., build, tooling, or automation).  
  - `.github/workflows/` for CI-related workflows.

- **Testing**  
  - `tests/e2e/` directory for end-to-end tests.

- **Developer Tooling**  
  - `.husky/` configuration for Git hooks (e.g., pre-commit, pre-push).
