# Richpanel MCP Server

## Overview
Richpanel MCP Server is an MCP (Model Context Protocol) server integration that connects Richpanel’s help desk and CRM capabilities into MCP-based applications for automation and orchestration.

## Features
- Integrates Richpanel’s help desk and customer support tools into MCP-compatible applications.
- Connects to your Richpanel account and associated client configuration.
- Uses a static MCP server URL (`https://mcp.pipedream.net/v2`) that works across clients.
- Supports authentication at the time you add the server to your MCP-compatible app.
- Can be added to various chat clients or applications that support MCP servers.
- Central configuration and management via the Pipedream configuration interface.

## Integration & Configuration
- **MCP Server URL:** `https://mcp.pipedream.net/v2` (static for all clients).
- **Setup Flow:**
  - Connect a Richpanel account within the Pipedream interface.
  - Select the appropriate Richpanel client.
  - Add the MCP server URL to your chat client or MCP-enabled application.
  - Authenticate as prompted when adding the server.
- **Documentation:** Additional configuration details are available on the Pipedream configuration page.
