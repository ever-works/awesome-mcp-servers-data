# Ticket Source MCP Server

An MCP server that exposes TicketSource’s online ticketing and event management capabilities to MCP-compatible clients, via Pipedream Connect.

- **Category:** Business & Commerce – MCP Servers  
- **Tags:** ticketing, event-management, booking  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features

- **MCP-compatible TicketSource integration**  
  - Provides access to TicketSource’s online ticketing and event management features through the MCP protocol.  
  - Usable with any MCP-compatible chat or agent client.

- **Static server endpoint**  
  - Single static MCP server URL (`https://mcp.pipedream.net/v2`) used across all clients.  
  - No per-client URL differences; configuration is standardized.

- **Client-agnostic setup**  
  - Works with multiple MCP-enabled chat applications.  
  - Configuration guidance available via a dedicated configuration page (general MCP setup instructions).

- **Authentication at configuration time**  
  - Authentication occurs when adding the server to an application/client (exact auth method depends on client and TicketSource/Pipedream setup).

- **Hosted by Pipedream Connect**  
  - Server is operated via Pipedream’s MCP infrastructure.  
  - Centralized terms of use and privacy handling via Pipedream.
