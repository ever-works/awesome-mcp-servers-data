## Features
- **MCP-compatible server endpoint**

  - Works across multiple MCP-capable chat applications.

- **Authentication at client setup**  
  - Authentication performed when adding the MCP server to a client or application (details handled during client configuration).

- **Code-free web automations**  
  - Configure automations without writing code.  
  - Chain multiple actions on the web into integrated workflows.

- **Data extraction on the web**  
  - Extract structured data from websites to support business and marketing use cases.  
  - Suitable for lead lists, audience data, and other growth-related datasets.

- **Lead generation workflows**  
  - Build automated sequences to discover and collect business leads from the web.

- **Audience-building workflows**

- **Growth-focused automations**  
  - Design workflows aimed at business growth, including repetitive web tasks and data collection.

- **Client integration guidance**  
  - Instructions available (via the configuration page) for adding the server to supported chat clients.

## Setup
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- Add this URL as an MCP server in your MCP-compatible chat client, then complete authentication as prompted.
