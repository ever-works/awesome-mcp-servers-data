## Features

- **MCP-compatible server** for integration with tools that support the Model Context Protocol (MCP).
- **Access to product data** from EAN-Search.org.
- **Barcode lookup** capabilities (e.g., using EAN/UPC barcodes).
- **OAuth 2.1-based authentication** for secure access to the API.
- **Remote server endpoint** available at `https://www.ean-search.org/mcp`.
