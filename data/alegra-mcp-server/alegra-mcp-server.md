# Alegra MCP Server

## Overview
Alegra MCP Server is an MCP-compatible server that connects to Alegra’s accounting and billing platform for Latin American businesses. It enables MCP clients (such as chat-based tools or other MCP-compatible applications) to access invoices, billing, and accounting data in context.

- **Category:** Business & Commerce · MCP Servers  
- **Brand:** Alegra  
- **Region focus:** Latin America  
- **Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible server**  
  - Implements the Model Context Protocol (MCP) for integration with MCP clients.  
  - Can be added to compatible chat or agent applications as an external server.

- **Integration with Alegra platform**  
  - Connects to Alegra’s accounting and billing system used by Latin American managers and small businesses.  
  - Provides contextual access to accounting, invoices, and billing data via MCP-compatible clients.

- **Account connection & configuration**  
  - Users connect their Alegra account and select the relevant client within the configuration flow.  
  - Uses a single static server URL (`https://mcp.pipedream.net/v2`) for all clients; authentication occurs when adding the server to the application.  
  - Configuration guidance available through the provider’s configuration page (for supported MCP clients).

- **Tooling exposure**  
  - Exposes available tools / actions from Alegra through the MCP interface (e.g., for working with commerce, invoices, billing, and accounting data).  
  - Tools are loadable and discoverable by compatible MCP clients.
