# Rafflys MCP Server

**Category:** Business & Commerce · MCP Servers  
**Brand:** rafflys  
**Website:** https://mcp.pipedream.com/app/rafflys

## Description
Rafflys MCP Server connects MCP-compatible clients to Rafflys’ Instagram giveaway and sweepstakes tools. It enables automated creation and management of Instagram giveaways directly from supported chat or MCP clients via a standardized server endpoint.

## Features
- **Instagram giveaway integration**: Interfaces with Rafflys’ tools to create and manage Instagram giveaways and sweepstakes.
- **Automated giveaway management**: Supports automated workflows for setting up and handling Instagram giveaways from MCP clients.
- **Static MCP server URL**: Uses a single, static endpoint that works for all compatible MCP clients:
  - `https://mcp.pipedream.net/v2`
- **Client-agnostic access**: Designed to be added to various MCP/chat clients, allowing configuration from different applications.
- **Authentication at client setup**: Authentication is performed when adding the server to an application, rather than changing the server URL.
- **Pipedream Connect integration**: Hosted and powered by Pipedream Connect, benefiting from its MCP server infrastructure.

## Technical Details
- **Server type:** MCP (Model Context Protocol) server
- **Protocol usage:** Exposes Rafflys Instagram giveaway functionality through a standardized MCP endpoint
- **Configuration:** Added as a remote MCP server in compatible chat or automation clients using the static URL
