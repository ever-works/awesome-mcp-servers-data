# Documint MCP Server

**Category:** Document Management MCP Servers  
**Brand:** Documint  
**Website:** https://mcp.pipedream.com/app/documint

## Description
Documint MCP Server integrates Documint’s PDF generation service with the Model Context Protocol (MCP), enabling automated creation of customized, production-ready PDFs from structured data. It runs on Pipedream’s MCP server to connect a Documint account with AI tools such as Cursor, Claude, and Windsurf.

## Features
- **MCP integration**: Connects Documint’s PDF generation capabilities to the Model Context Protocol ecosystem.
- **PDF generation from structured data**: Creates customized, production-ready PDFs using structured inputs.
- **AI tool connectivity**: Works with AI tools like Cursor, Claude, and Windsurf via Pipedream’s MCP server.
- **Documint account integration**: Uses an existing Documint account as the document generation backend.
- **Automation-ready**: Designed for automated document workflows controlled by AI tools.
