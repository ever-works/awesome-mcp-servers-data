# Waitlist MCP Server

**Description**

The Waitlist MCP Server connects your Waitlist account to MCP-compatible AI tools, enabling management of viral waitlists, referrals, and launch metrics directly from tools like Cursor, Claude, and Windsurf.

**Features**

- Connects a Waitlist account to Pipedream's MCP server
- Integrates with AI tools such as Cursor, Claude, and Windsurf
- Enables management of viral waitlists via MCP tools
- Supports handling of referral-based growth workflows
- Provides access to launch and growth metrics from Waitlist through MCP-compatible clients

**Integrations**

- Pipedream MCP
- Cursor
- Claude
- Windsurf
