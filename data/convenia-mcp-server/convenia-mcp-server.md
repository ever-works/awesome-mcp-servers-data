# Convenia MCP Server

## Overview
Convenia MCP Server provides access to Convenia, a cloud-based people management solution designed for small businesses. It focuses on improving efficiency and reducing manual, operational HR work by integrating Convenia into MCP-compatible chat or AI applications via Pipedream Connect.

- **Item type:** MCP server / integration endpoint
- **Use case:** Connect AI/chat clients to Convenia for people management workflows
- **Target users:** Small businesses using Convenia for HR and people operations

## MCP Server Endpoint
- **Base MCP server URL (static for all clients):**
  - `https://mcp.pipedream.net/v2`
- Authentication is performed when you add/configure the server within your client application.

## Features
- **Static MCP Endpoint**
  - Single, fixed MCP server URL usable by all clients.
