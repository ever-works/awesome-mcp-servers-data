# Datarobot MCP Server

**Category:** AI Integration / MCP Servers  
**Brand:** DataRobot  
**Source:** https://mcp.pipedream.com/app/datarobot  
**Slug:** `datarobot-mcp-server`

## Overview
The Datarobot MCP Server is a Model Context Protocol (MCP) server that connects applications to the DataRobot enterprise AI platform via Pipedream. It provides a static MCP endpoint that can be added to compatible chat or AI clients to access DataRobot’s AI and automation capabilities.

## Features
- **MCP-compatible server:** Exposes DataRobot functionality through a standard MCP server endpoint.
- **Static server URL:** Uses a single static URL (`https://mcp.pipedream.net/v2`) that works for all clients.
- **Client-agnostic integration:** Designed to be added to various chat or AI clients; configuration details are provided per client type on the configuration page.
- **Account-based configuration:** Requires connecting a DataRobot account before selecting a client and using the tools.
- **Tool discovery:** Supports loading and listing available tools/actions from the DataRobot integration (shown as "Loading actions..." / "Loading available tools...").
- **Hosted by Pipedream:** Managed and served through Pipedream’s infrastructure, with terms and privacy handled by Pipedream.

## Usage
1. Connect your DataRobot account via the Pipedream configuration interface.
2. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
3. Add the server URL to your MCP-compatible application or chat client.
4. Follow the client-specific instructions on the configuration page to authenticate and start using DataRobot tools.
