# Pitchlane MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Pitchlane  
**Source:** https://mcp.pipedream.com/app/pitchlane

## Description
Pitchlane MCP Server exposes Pitchlane’s video outreach-at-scale capabilities to MCP-based tools. It allows applications that support the Model Context Protocol (MCP) to integrate and automate video outreach workflows programmatically.

## Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same URL can be used for all clients; authentication is handled when adding the server to the application.

- **Pitchlane account integration**  
  - Connect a Pitchlane account within the MCP server configuration.  
  - Select a specific client (where applicable) after connecting the account.

- **Video outreach workflow enablement**  
  - Designed to surface Pitchlane’s video outreach-at-scale capabilities to MCP-based tools.  
  - Enables building automated video outreach workflows via compatible chat clients or applications.

- **Client/app integration**  
  - Can be added to various chat clients or MCP-compatible applications.  
  - Configuration guidance provided via a dedicated configuration page (external to this summary).

- **Tooling exposure**  
  - Exposes “available tools” (actions) from Pitchlane through the MCP server interface (specific tools are not listed in the provided content).  
  - Intended for programmatic access by agents or automation workflows.

## Setup
- Connect your Pitchlane account in the MCP server UI.  
- Choose the appropriate client configuration.  
- Copy and use the static MCP server URL in your MCP-compatible app or chat client.  
- Authenticate within the target app when adding the server.
