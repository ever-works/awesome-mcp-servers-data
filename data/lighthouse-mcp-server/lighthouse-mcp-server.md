# Lighthouse MCP Server

**Category:** Project Management MCP Servers  
**Brand:** Lighthouse  
**Slug:** `lighthouse-mcp-server`

An MCP (Model Context Protocol) server that connects your Lighthouse issue tracking account to AI tools, enabling AI agents and MCP-compatible clients to interact with Lighthouse projects and issues.

---

## Description
Lighthouse MCP Server provides a bridge between your Lighthouse issue tracking system and AI-powered tools via Pipedream’s MCP server. It allows AI agents and MCP clients (such as Cursor, Claude, and Windsurf) to access and work with Lighthouse projects and issues using the Model Context Protocol.

---

## Features
- **Lighthouse integration**: Connects directly to your Lighthouse issue tracking account.  
- **MCP-compatible**: Exposes Lighthouse functionality through the Model Context Protocol for standardized access.  
- **AI tooling connectivity**: Works with AI tools like **Cursor**, **Claude**, and **Windsurf** via Pipedream’s MCP server.  
- **Issue interaction**: Designed to let AI agents and MCP clients interact with Lighthouse issues (e.g., viewing or managing tickets and issue data).  
- **Project-level access**: Supports interaction with Lighthouse projects, enabling project- and ticket-oriented workflows through AI.  
- **Ticketing workflows**: Tailored for issue tracking and ticket management scenarios within Lighthouse.
