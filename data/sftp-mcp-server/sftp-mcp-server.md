# SFTP MCP Server

**Category:** File Management MCP Servers  
**Tags:** sftp, file-management, remote-access

## Description
SFTP MCP Server is an MCP server that lets MCP-compatible clients interact with SFTP (SSH File Transfer Protocol) servers using username and password authentication. It enables remote file management operations over SFTP within MCP-aware applications.

## Features
- Connects to SFTP servers over SSH File Transfer Protocol
- Uses username and passwordâ€“based authentication
- Provides a static MCP server endpoint usable by any compatible client
- Supports interaction from multiple MCP-enabled chat or agent clients
- Central MCP server URL for configuration: `https://mcp.pipedream.net/v2`
- Managed and hosted via Pipedream infrastructure

## Usage
- Add the MCP server URL to your MCP-compatible application or chat client
- Authenticate with your SFTP server credentials (username and password) when configuring the connection

## Endpoint
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
