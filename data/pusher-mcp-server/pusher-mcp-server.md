# Pusher MCP Server

**Description**  
An MCP Server integration for Pusher that provides MCP clients with access to Pusher’s hosted real-time messaging and event APIs via a static, hosted endpoint.

**Category**  
- Messaging MCP Servers

**Tags**  
- messaging  
- real-time  
- api-integration

**Source URL**  
- https://mcp.pipedream.com/app/pusher

---

## Features

- **Hosted MCP server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Single URL works across compatible MCP clients.

- **Pusher integration**  
  - Connects MCP clients to Pusher’s hosted real-time messaging and event APIs.  
  - Enables real-time messaging and event handling from within MCP-compatible tools.

- **Client-agnostic configuration**  
  - Designed to be added to different chat / MCP client applications.  
  - Configuration guidance available via a dedicated configuration page.

- **Authentication at client level**  
  - Authentication occurs when adding the server to an application (per-client authentication, rather than per-URL).

- **Hosted and managed by Pipedream**  
  - Operated through Pipedream’s infrastructure (Pipedream Connect).

---

## Configuration

- **MCP Server URL**: `https://mcp.pipedream.net/v2`  
- Add this URL to your MCP-compatible chat client or application, then follow the client’s instructions to authenticate and complete setup.
