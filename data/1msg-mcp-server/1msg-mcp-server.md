# 1msg MCP Server

Tools and automation for the WhatsApp Business API, exposed as an MCP (Model Context Protocol) server via Pipedream Connect.

## Overview
- **Type:** MCP server / integration endpoint
- **Provider:** 1msg (via Pipedream Connect)
- **Protocol:** MCP (Model Context Protocol)
- **Primary Channel:** WhatsApp Business API
- **Use Case:** Enabling WhatsApp messaging workflows inside MCP-compatible environments and chat clients.

## MCP Server URL
- **Server URL:** `https://mcp.pipedream.net/v2`
  - Static URL shared across clients
  - Authentication is performed when adding the server in the client/application

## Features
- Provides an MCP-compatible server interface for WhatsApp Business API workflows
- Designed to plug into multiple MCP-compatible chat clients
- Supports authentication at the client/application configuration stage
- Hosted and powered by Pipedream Connect (no self-hosted MCP endpoint required)
- Centralized configuration via Pipedream’s configuration page

## Integration & Configuration
- Add the MCP server URL to your MCP-compatible chat client or application
- Authenticate the 1msg MCP server within your client during setup
- Reference Pipedream’s configuration page for detailed setup steps (client-specific instructions available there)
