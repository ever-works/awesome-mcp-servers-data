# MailNinja MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** MailNinja  
**Slug:** `mailninja-mcp-server`

An MCP server that connects your MailNinja email marketing platform to MCP‑compatible AI tools.

## Overview
MailNinja MCP Server exposes MailNinja’s email marketing operations to AI agents via Pipedream’s MCP server, allowing tools like Cursor, Claude, and Windsurf to interact with your MailNinja account.

## Features
- Connect a MailNinja account to Pipedream’s MCP server
- Enable AI tools (e.g., Cursor, Claude, Windsurf) to access MailNinja via MCP
- Surface MailNinja email marketing platform operations to AI agents (e.g., sending, managing, or automating campaigns, where supported by MailNinja’s API)

## Integrations
- **Pipedream MCP** – core server/integration layer
- **AI tools (MCP-compatible)** such as:
  - Cursor
  - Claude
  - Windsurf
