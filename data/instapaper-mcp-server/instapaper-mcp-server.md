# Instapaper MCP Server

**Category:** Document Management MCP Servers  
**Brand:** Instapaper  
**Source:** https://mcp.pipedream.com/app/instapaper

Instapaper MCP Server lets MCP-compatible agents and chat clients interact with Instapaper so users can save and organize web pages and articles to read later, directly from their MCP-enabled applications.

---

## Features

- **Instapaper integration via MCP**  
  Connects MCP-based agents and chat clients to an Instapaper account for later-reading workflows.

- **Static MCP server endpoint**  
  Uses a single static server URL for all clients:  
  `https://mcp.pipedream.net/v2`

- **Per-client authentication**  
  Authentication occurs when adding the server to each MCP-compatible application, allowing secure access to a userâ€™s Instapaper account.

- **Multi-client support**  
  Designed to be added to various chat / MCP clients, enabling similar Instapaper functionality across different tools.

- **Article and web page handling**  
  Supports saving and organizing web pages and articles to Instapaper for later reading (via available MCP tools/actions exposed by the server).

- **Configuration guidance**  
  Provides a configuration page and client-specific instructions for adding the server to supported apps.
