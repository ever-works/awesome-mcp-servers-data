# Storerocket MCP Server

**Category:** Business & Commerce · MCP Servers  
**Tags:** store-locator, maps, local-business

## Overview
Storerocket MCP Server is an MCP integration that exposes Storerocket’s store locator software and widgets through a unified MCP endpoint. It allows compatible chat or MCP-enabled applications to interact with Storerocket’s store locator capabilities.

The integration uses a single static MCP endpoint URL:

```text
https://mcp.pipedream.net/v2
```

Authentication is performed when adding the server to your MCP-compatible application.

## Features
- **MCP integration for Storerocket** – Connects Storerocket’s store locator software and widgets to MCP-enabled apps.
- **Single static MCP endpoint** – Uses a shared MCP server URL (`https://mcp.pipedream.net/v2`) for all clients.
- **Authentication at client setup** – Authentication is handled when you add the MCP server to your own application or chat client.
- **Store locator capabilities** – Provides access (via MCP) to Storerocket’s store locator software designed for websites.
- **Widget and app integration** – Integrates Storerocket’s store locator widget / app so it can be accessed programmatically through MCP.
- **Client-agnostic** – Intended to work with multiple MCP-compatible chat clients; setup instructions depend on the client you choose.

## Usage
- Add the MCP server URL `https://mcp.pipedream.net/v2` to your MCP-compatible client.
- Configure authentication as required by your application during server setup.
- Optionally refer to the platform’s configuration documentation for detailed setup steps.
