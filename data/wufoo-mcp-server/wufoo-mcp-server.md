# Wufoo MCP Server

**Category:** Business & Commerce · MCP Servers  
**Brand:** Wufoo

An MCP Server integration for Wufoo that allows MCP-based agents and clients to create and manage online forms and form submissions using Wufoo’s APIs.

---

## Features

- **MCP server endpoint**  
  - Static MCP server URL for all compatible clients:  
    `https://mcp.pipedream.net/v2`
  - Authentication handled when adding the server to your MCP-compatible application.

- **Wufoo integration**  
  - Connects MCP-based agents to Wufoo’s online form builder and APIs.  
  - Enables creation and management of online forms and form submissions (e.g., registration forms, application forms, surveys, contact forms).

- **Client compatibility**  
  - Designed to be added to various MCP-enabled chat or agent clients.  
  - Central configuration instructions available via a dedicated configuration page.

- **Use cases**  
  - Collect data via custom forms built in Wufoo.  
  - Automate form-related workflows through MCP-based agents (e.g., creating forms, reading submissions, managing entries).
