## Overview
The IBM Cloud - Natural Language Understanding MCP Server is an integration layer that connects IBM Cloud Natural Language Understanding (NLU) to AI tools such as Cursor, Claude, and Windsurf via the Model Context Protocol (MCP). It enables these tools to access IBM NLUâ€™s advanced text analytics capabilities.

## Features
- **MCP-based integration**: Exposes IBM Cloud Natural Language Understanding as an MCP server for use in compatible AI tools.
- **Connection to IBM Cloud NLU**: Bridges an existing IBM Cloud Natural Language Understanding account with external AI environments.
- **Text analytics enablement**: Provides access (via MCP) to IBM NLU capabilities like:
  - Sentiment analysis
  - Entity analysis
  - Other advanced text analysis features supported by IBM Cloud NLU (as configured in your IBM account)
- **Tool compatibility**: Designed to work with AI tools that support MCP, including:
  - Cursor
  - Claude
  - Windsurf

## Integrations
- **IBM Cloud Natural Language Understanding** (data and analysis backend)
