# XeggeX MCP Server

**Category:** Blockchain & Crypto MCP Servers  
**Brand:** XeggeX  
**Description:** An MCP server that allows MCP-compatible clients to interact programmatically with the XeggeX cryptocurrency exchange.

## Features
- MCP server interface for the XeggeX cryptocurrency exchange
- Provides a static MCP server URL usable across compatible clients
- Authentication handled when adding the server to an application/client
- Hosted and powered via Pipedream Connect infrastructure
- Usable with multiple chat or MCP-enabled clients (configuration handled per client)

## Technical Details
- MCP Server URL: `https://mcp.pipedream.net/v2`
