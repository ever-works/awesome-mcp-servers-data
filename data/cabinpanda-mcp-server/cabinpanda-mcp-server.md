# CabinPanda MCP Server

## Overview
CabinPanda MCP Server is an integration that exposes CabinPanda’s online data collection and form capabilities to MCP-compatible tools via a static MCP endpoint hosted on Pipedream.

- **Name:** CabinPanda MCP Server  
- **Category:** Business & Commerce · MCP Servers  
- **Use cases:** Online data collection, form handling, surveys, and related workflows through MCP-enabled clients.

## Features
- **MCP-compatible server endpoint**  
  - Single, static MCP server URL usable by any compatible client:  
    - `https://mcp.pipedream.net/v2`
  - Centralized endpoint for connecting MCP tools to CabinPanda.

- **CabinPanda data collection integration**  
  - Connects MCP tools to CabinPanda’s online data collection infrastructure.  
  - Suitable for collecting various types of online data via forms and surveys (as supported by CabinPanda itself).

- **Client-agnostic setup**  
  - Works with multiple MCP-compatible chat or agent clients.  
  - Authentication handled when adding the server in your chosen client.

- **Configuration documentation**  
  - Guided instructions per chat client (via the referenced configuration resources).  
  - Central configuration page for details on adding and managing the MCP server.

## Integration & Setup
- **Server URL:** `https://mcp.pipedream.net/v2`  
- **Setup flow:**  
  1. Copy the MCP server URL.  
  2. Add it to your MCP-compatible chat or agent client.  
  3. Authenticate within your client when prompted.  
  4. Optionally consult the full configuration page for client-specific steps.
