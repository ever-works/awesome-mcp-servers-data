# Flexmail MCP Server

Flexmail MCP Server connects the Flexmail email marketing platform to the MCP ecosystem, allowing MCP-compatible clients to create and send email campaigns through a unified server interface.

## Overview
- **Name:** Flexmail MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Brand:** Flexmail  
- **Slug:** `flexmail-mcp-server`  
- **Source URL:** https://mcp.pipedream.com/app/flexmail

## Features
- **MCP ecosystem integration**: Exposes Flexmail email marketing capabilities through an MCP-compatible server endpoint.
- **Unified server interface**: MCP-compatible clients can create and send email campaigns using a single, static server URL.
- **Static MCP server URL**: 
  - Base endpoint: `https://mcp.pipedream.net/v2`
  - Same URL works for all supported clients.
- **Client-agnostic configuration**: Designed to be added to different MCP-compatible chat or agent clients; you authenticate when adding the server to your application.
- **Email campaign operations** (implied from description):
  - Create email campaigns via MCP.
  - Send email campaigns via MCP.
- **Pipedream Connect integration**: The server is powered by Pipedream Connect, leveraging Pipedream’s MCP infrastructure.

## Usage
- Add `https://mcp.pipedream.net/v2` as an MCP server in your compatible client.
- Authenticate during server setup within your application.
- Optionally reference the provider’s configuration documentation ("Configuration" page) for client-specific setup instructions.
