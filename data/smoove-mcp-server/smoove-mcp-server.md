# Smoove MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Smoove  
**Source:** https://mcp.pipedream.com/app/smoove  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Overview
Smoove MCP Server is a Model Context Protocol (MCP) server that connects to the Smoove marketing automation platform. It exposes Smoove functionality as tools within compatible chat or AI clients, enabling operations around contacts, lists, and subscriber management directly from your application.

## Features
- **MCP integration with Smoove**  
  - Connects an MCP-compatible client to the Smoove marketing automation platform.  
  - Uses a static MCP server URL; authentication occurs when adding the server to your application.

- **Subscriber and contact management tools (5 actions available)**  
  - **Unsubscribe**  
    - Unsubscribes a contact from communications.  
  - **Find Subscriber**  
    - Retrieves contact information using a unique identifier (contact ID, email, cellphone number, or external ID).  
  - **Find Or Create Subscriber**  
    - Checks whether a contact exists; if not found, creates a new contact.  
  - **Create List**  
    - Creates a new contact list in the Smoove account.  
  - **Add Or Update Subscriber**  
    - Adds a new contact to the account or updates an existing contact’s details.

- **Account connection and configuration**  
  - Connect a Smoove account and select a client to begin managing contacts and lists.  
  - Can be added to various chat clients that support MCP, with configuration guidance available via the platform’s configuration page.

## Use Cases
- Manage Smoove subscribers directly from an MCP-enabled chat or AI interface.  
- Automatically find or create contacts when interacting with users.  
- Maintain and organize marketing lists programmatically.  
- Update or unsubscribe contacts based on interactions within your application.
