# LightSpeed VT MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** LightSpeed VT  
**Tags:** training, online-learning, LMS

LightSpeed VT MCP Server provides a standardized MCP endpoint for integrating the LightSpeed VT web-based training system into compatible chat or MCP-enabled clients.

---

## Features

- **Static MCP Server URL**  
  - Single endpoint for all clients: `https://mcp.pipedream.net/v2`  
  - Works universally across supported MCP/chat clients

- **Authentication at Client Setup**  
  - Authentication is handled when adding the server to your application (per-client configuration)

- **Chat Client Integration**  
  - Can be added to MCP-compatible chat applications  
  - Step-by-step setup is available via the configuration documentation (e.g., selecting your chat client and following integration instructions)

- **Pipedream-Hosted Infrastructure**  
  - Server is provisioned and hosted via Pipedream Connect  
  - Centralized configuration and management through the Pipedream platform

- **Training System Backend**  
  - Connects to LightSpeed VT, a web-based system for delivering and managing online training experiences (LMS-style environment)

---

## Usage

- Use the static MCP endpoint `https://mcp.pipedream.net/v2` when configuring a new MCP server in your chat or development client.
- Complete authentication and any client-specific configuration steps as prompted by your application.
