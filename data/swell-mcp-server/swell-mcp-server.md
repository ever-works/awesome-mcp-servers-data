# Swell MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Swell

Swell MCP Server exposes Swell’s headless ecommerce platform capabilities to MCP-based applications and workflows via a static MCP endpoint provided by Pipedream.

---

## Features

- **Headless ecommerce integration**
  - Connects Swell’s headless ecommerce platform to any MCP-compatible client or application.
  - Enables MCP-based workflows to interact with Swell commerce data and operations.

- **Static MCP server URL**
  - Single, static endpoint for all clients: `https://mcp.pipedream.net/v2`.
  - Same URL works across different MCP-enabled chat or automation clients.

- **Account-based authentication**
  - Authentication occurs when adding the server to your MCP-compatible application.
  - Requires connecting your Swell / Pipedream account before use.

- **Client-agnostic setup**
  - Designed to be added to various chat or MCP clients.
  - Client-specific setup guidance available via the configuration documentation.

- **Configuration documentation**
  - Central configuration page (via Pipedream) for details on connecting accounts and selecting clients.

- **Tooling exposure (MCP tools)**
  - Exposes Swell-related actions as MCP “tools” once the server is connected.
  - Tools are dynamically loaded by compatible MCP clients after configuration.
