# Vendasta MCP Server

An MCP server that connects to Vendasta, enabling automated interactions with Vendasta’s platform for selling marketing solutions to local businesses via MCP-compatible clients.

## Overview
- **Type:** MCP server (Model Context Protocol server)
- **Platform:** Pipedream Connect
- **Use case:** For companies that sell marketing solutions to small and medium-sized local businesses and want to integrate Vendasta into MCP-enabled chat or automation clients.

## Features
- **Vendasta integration via MCP**
  - Connects MCP clients to Vendasta’s platform for marketing and sales workflows targeting local businesses.
- **Static MCP server URL**
  - Single endpoint for all clients: `https://mcp.pipedream.net/v2`.
  - Authentication is handled when adding the server to an MCP-compatible application.
- **Client-agnostic setup**
  - Designed to work with multiple MCP chat clients.
  - Configuration guidance available through a dedicated configuration page.
- **Hosted by Pipedream**
  - Runs on Pipedream’s infrastructure under the Pipedream Connect offering.

## Configuration
- **Server URL:** `https://mcp.pipedream.net/v2`
- **Setup:**
  - Add the above URL as an MCP server in your chat or MCP-capable client.
  - Authenticate within your application during server setup.
  - Optionally consult the full configuration documentation (linked from the Pipedream page) for client-specific steps.
