# Cisco Webex MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Cisco Webex

An MCP Server integration that exposes Cisco Webex video conferencing and collaboration capabilities as programmable endpoints for AI/chat clients.

---

## Features

### MCP Server & Connectivity
- Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`
- Works with multiple chat clients (e.g., ChatGPT / OpenAI) via MCP
- Authentication handled when adding the server to your application
- Central configuration via Pipedream MCP configuration page

### Cisco Webex Capabilities Exposed
- Video conferencing and online meetings (via Cisco Webex APIs)
- Screen sharing support
- Webinar-style sessions and collaboration features (through Webex)

### Available Tools (Actions)
1. **List Messages**
   - Retrieve messages in a specific Webex room
   - Useful for reading conversation history, context gathering, and logging

2. **Create a Room**
   - Programmatically create a new Webex room
   - The authenticated user is automatically added as a member of the room

3. **Create Message**
   - Post a message to a Webex room
   - Supports plain text or rich text messages
   - Optional file attachment support when posting messages

---

## Integration & Setup
- Connect a Cisco Webex account via Pipedream
- Add the MCP server URL to your MCP-compatible client (e.g., ChatGPT)
- Configure and manage connections through the Pipedream interface
