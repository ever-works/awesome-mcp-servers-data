## Description
PulseMCP is a community hub and weekly newsletter focused on the Model Context Protocol (MCP). It provides a searchable index of MCP servers and clients, curated ecosystem updates, and educational content about MCP tools, use cases, and implementations.

## Features
- **MCP Servers Directory**
  - Browse MCP servers that connect AI models to data sources and tools.
  - Searchable/indexed catalog (with API access, per project description).

- **MCP Clients Directory**
  - Listing of apps and tools that connect to MCP servers.

- **Pulse Posts**
