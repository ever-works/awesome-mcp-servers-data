# Invoice Ninja MCP Server

Open-source online invoicing MCP server that connects tools and chat clients to the Invoice Ninja platform for freelancers and businesses.

- **Category:** Business & Commerce – MCP Servers  
- **Slug:** `invoice-ninja-mcp-server`  
- **Tags:** invoicing, billing, small-business  
- **Brand:** Invoice Ninja

---

## Features

- **MCP server endpoint**  
  - Static MCP server URL for all compatible clients:  
    - `https://mcp.pipedream.net/v2`  
  - Central entry point for connecting tools and chat-based clients to Invoice Ninja.

- **Invoice Ninja integration**  
  - Designed specifically for the Invoice Ninja platform, an open‑source online invoicing app for freelancers and businesses.

- **Client-agnostic configuration**  
  - Same URL works across different MCP-compatible chat clients.  
  - Authentication is handled when adding the server in each client.

- **Configuration resources**  
  - Step-by-step connection instructions available via the linked configuration page (per chat client).  
  - Additional configuration details accessible at the platform’s Configuration page.

- **Pipedream Connect infrastructure**  
  - MCP server is powered by Pipedream Connect.

---

## Technical Details

- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Usage model:** Add the MCP server to an MCP-compatible application or chat client, then authenticate within that client.
