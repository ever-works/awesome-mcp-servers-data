## Overview

Dedicated MCP server for interacting with Artifacts MMO, enabling AI-driven gameplay.

## Features

- Query player stats, position, inventory, and world state
- Execute actions: move, attack, craft, trade
- Stream chat, combat, and exploration events
- Multi-session support for fleet of bots

## Use Cases

- AI bots for automated farming and leveling
- Dynamic NPC behaviors powered by LLMs
- Game analytics and simulation testing
