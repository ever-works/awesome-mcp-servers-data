## Front MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Front  
**Integration Type:** MCP Server (via Pipedream)

### Overview
Front MCP Server is an MCP-compatible integration for Front’s customer communication platform. It connects your MCP-enabled chat client or application to Front so you can manage multichannel conversations, messages, teammates, and tickets directly through MCP tools.

- **MCP server URL:** `https://mcp.pipedream.net/v2`
- Works with multiple chat clients; authentication is handled when adding the server to your application.

### Features
- **Front account connection**
  - Connect a Front account to the MCP server through Pipedream.
  - Use a single static MCP URL for all clients, with per-client authentication.

- **Teammate management**
  - **Update Teammate:** Modify details of an existing teammate.
  - **List Teammates:** Retrieve a list of teammates in the company.
  - **Get Teammate:** Retrieve information for a specific teammate by ID.

- **Conversation management**
  - **Update Conversation:** Update properties of an existing conversation.
  - **Tag Conversation:** Add tags to a conversation.
  - **Reply To Conversation:** Send a reply message and append it to an existing conversation.
  - **Send New Message:** Send a new message from a channel and create a new conversation.
  - **Receive Custom Messages:** Receive custom messages into Front.
  - **Import Message:** Append a new message into an inbox.
  - **List Conversations:** List conversations in the company.
  - **List Conversation Comments:** List comments associated with a specific conversation.
  - **List Comment Mentions:** List teammates mentioned in a specific comment.

- **Ticket status management**
  - **List Ticket Statuses:** List ticket statuses available for the company.
  - **Get Ticket Status:** Retrieve a specific ticket status by ID.

- **Message templates**
  - **List Message Templates:** Retrieve available message templates.

- **Tooling coverage**
  - 27 actions exposed as MCP tools (the above are the documented examples; additional actions are available via the MCP server tooling set).

- **Configuration resources**
  - Step-by-step guidance for adding the MCP server to different chat clients.
  - Central configuration reference available on the platform’s Configuration page.
