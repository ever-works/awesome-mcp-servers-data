# Nmbrs MCP Server

## Overview
Nmbrs MCP Server is an MCP (Model Context Protocol) server integration for Visma Nmbrs HR and payroll software. It exposes Nmbrs HR and payroll capabilities to MCP-based automation tools and AI assistants via a static MCP endpoint.

- **Product type:** MCP server integration
- **Category:** Business & Commerce – MCP Servers
- **Primary use cases:** Accessing and automating HR and payroll workflows from MCP-compatible chat clients and AI agents
- **MCP server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible HR & payroll integration**  
  - Connects Visma Nmbrs HR and payroll software to MCP-based applications and assistants.  
  - Designed to expose Nmbrs capabilities (HR and payroll operations) as tools within MCP.

- **Static server URL**  
  - Uses a single static MCP URL: `https://mcp.pipedream.net/v2`.  
  - Same URL for every client; individual authentication is handled when adding the server to a client.

- **Use with multiple chat clients**  
  - Can be added to different MCP-compatible chat clients or AI applications.  
  - Setup instructions are available per client (via the referenced configuration documentation).

- **Configuration documentation**  
  - Full configuration details are available on the provider’s configuration page (for adding and authenticating the server in clients).

## Integration & Access
- **Endpoint:** `https://mcp.pipedream.net/v2` (MCP server endpoint for Nmbrs)
- **Authentication:** Per-client authentication during server setup (no client-specific URL required).
- **Provider platform:** Built and hosted via Pipedream Connect.
