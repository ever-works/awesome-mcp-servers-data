# Transistor.fm MCP Server

**Category:** Business & Commerce MCP Servers  
**Slug:** transistorfm-mcp-server  
**Brand:** Transistor.fm  
**Tags:** podcast, media, analytics

## Overview
Transistor.fm MCP Server exposes Transistor.fm podcast hosting capabilities to MCP-compatible tools. It enables programmatic access to shows, episodes, analytics, and distribution settings through a static MCP endpoint.

## MCP Server Endpoint
- **Server URL:** `https://mcp.pipedream.net/v2`  
- **Usage:** Same static URL for all clients; authentication is performed when adding the server in your MCP-compatible application.

## Features
- Programmatic access to Transistor.fm via MCP tools
- Manage podcast hosting resources, including:
  - Shows
  - Episodes
  - Distribution settings
- Access podcast analytics through MCP
- Works with multiple MCP-enabled chat clients and tools
- Single, static MCP server URL for all clients

## Integration & Configuration
- Add the MCP server URL to any supported MCP client or application
- Authentication occurs during server setup within the client
- Additional configuration details are available from the provider’s configuration documentation (referenced as the “Configuration page”).
