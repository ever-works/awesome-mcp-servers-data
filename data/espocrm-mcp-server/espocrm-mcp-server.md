# EspoCRM MCP Server

**Category:** Business & Commerce – MCP Servers  
**Slug:** espocrm-mcp-server  
**Brand:** EspoCRM  
**Source:** https://mcp.pipedream.com/app/espocrm

## Description
EspoCRM MCP Server is an MCP-based integration endpoint that exposes EspoCRM’s open source, customizable CRM entities and automation capabilities to MCP-compatible clients via a static server URL.

## Features
- Provides MCP-based access to EspoCRM CRM data and entities
- Supports interaction with EspoCRM’s open source, customizable CRM models
- Enables automation workflows built on top of EspoCRM via MCP
- Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) for all compatible clients
- Authentication handled when adding the server within the client/application
- Can be added to multiple chat or MCP-enabled clients
- Hosted and delivered via Pipedream’s infrastructure

## Integration & Usage
- MCP server base URL: `https://mcp.pipedream.net/v2`
- Add the server within your MCP-compatible or chat client configuration
- Authentication occurs during server addition in the client
- Additional configuration guidance is available on the provider’s configuration page (external to this summary)
