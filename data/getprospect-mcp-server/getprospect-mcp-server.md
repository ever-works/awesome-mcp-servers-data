# GetProspect MCP Server

Find and enrich business contact data (including LinkedIn-based emails) directly from MCP-compatible clients using the GetProspect API.

- **Name:** GetProspect MCP Server  
- **Category:** Business & Commerce · MCP Servers  
- **Brand:** GetProspect  
- **Source URL:** https://mcp.pipedream.com/app/getprospect  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Description
The GetProspect MCP Server exposes the GetProspect API to MCP agents and chat-based tools, enabling automated discovery of business email addresses and LinkedIn contact information from within any compatible client. Authentication is handled when adding the server to your application.

## Features
- **Business email discovery**  
  - Find business email addresses for individuals.  
  - Uses GetProspect’s data to return professional contacts.

- **LinkedIn-based enrichment**  
  - Retrieve email addresses starting from LinkedIn profiles.  
  - Designed specifically for prospecting via LinkedIn contact data.

- **MCP-compatible server**  
  - Exposed as an MCP Server that can be added to any MCP-capable chat client or agent.  
  - Uses a single static base URL (`https://mcp.pipedream.net/v2`) for all clients.

- **Centralized configuration**  
  - Authentication and configuration handled when you add the server to your app.  
  - Full configuration instructions available via the linked configuration page (external to this summary).

- **Pipedream Connect integration**  
  - Provided and hosted via Pipedream Connect infrastructure.  
  - Uses Pipedream’s platform for connectivity and API handling.

## Intended Use Cases
- Lead generation and prospecting.  
- Building MCP agents that enrich leads with verified business emails.  
- Sales and marketing workflows that start from LinkedIn profiles.
