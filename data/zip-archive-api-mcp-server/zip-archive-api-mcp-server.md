# Zip Archive API MCP Server

MCP Server for the Zip Archive API that enables compression of files and creation of ZIP archives within MCP workflows.

- **Category:** File Management MCP Servers  
- **Tags:** compression, file-processing, file-management  
- **Source URL:** https://mcp.pipedream.com/app/zip_archive_api  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features

- Provides an MCP-compatible server interface for the Zip Archive API.  
- Enables compression of files and creation of ZIP archives within MCP-based workflows.  
- Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) for all supported clients.  
- Supports authentication at the time you add the server to your MCP-compatible application or chat client.  
- Can be added to various chat clients that support MCP (configuration specifics available via the configuration page).  
- Integrates with Pipedream Connect infrastructure.

## Integration & Configuration

- **Server URL:** `https://mcp.pipedream.net/v2` (static for all clients).
