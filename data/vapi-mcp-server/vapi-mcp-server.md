# Vapi MCP Server

**Category:** Messaging MCP Servers  
**Brand:** Vapi  
**Source:** https://mcp.pipedream.com/app/vapi

## Overview
Vapi MCP Server is an MCP-compatible integration that connects the Vapi voicebot platform to agent environments (such as ChatGPT and other MCP clients). It allows you to build, test, and deploy AI-powered voicebots via Vapi directly from MCP-compatible tools.

## Features
- **MCP-compatible server endpoint**  
  - Static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Single URL works across all supported MCP clients.  
  - Authentication occurs when adding the server to your application.

- **Vapi platform integration**  
  - Leverages Vapiâ€™s capabilities to build, test, and deploy voicebots.  
  - Designed for creating AI-powered voice assistants and conversational voicebots in minutes instead of longer development cycles.

- **Client integration guidance**  
  - Setup guidance for connecting the MCP server to ChatGPT (OpenAI) as a client.  
  - Additional configuration details available via the Pipedream MCP configuration page.

- **Available tools (actions)**  
  1. **Upload File**  
     - Uploads a new file to Vapi.  
     - Useful for providing assets or data (e.g., scripts, documents, or resources) needed by a voice assistant.
  2. **Update Assistant Settings**  
     - Updates configuration settings for a specific Vapi assistant.  
     - Enables programmatic tuning of assistant behavior and parameters from your MCP environment.
  3. **Create Call**  
     - Starts a new conversation with a Vapi assistant.  
     - Allows initiating voicebot calls or sessions directly via MCP.

- **Account connection & management**  
  - Connects to your Vapi account through the Pipedream interface.  
  - Once connected, you can manage multiple assistants and calls via the MCP tools.
