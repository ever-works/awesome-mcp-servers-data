# Poper MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Poper  
**Slug:** `poper-mcp-server`

Poper MCP Server is an MCP (Model Context Protocol) integration that lets AI agents configure and manage AI-driven website popups and engagement campaigns via Poper.

---

## Features

- **MCP-compatible server**
  - Exposes Poper functionality through the Model Context Protocol for use by AI agents.
  - Uses a static MCP server URL usable across compatible clients.

- **AI-driven website popups**
  - Configure AI-powered popups for websites through connected AI agents.
  - Intended to increase on-site user engagement.

- **Engagement campaign management**
  - Manage AI-driven engagement campaigns (e.g., on-site promotions or messaging) via MCP.

- **Central server endpoint**
  - Static MCP server URL: `https://mcp.pipedream.net/v2`
  - Authentication handled when adding the server in the client application.

- **Multi-client support**
  - Designed to be added to different MCP-compatible chat or AI clients.
  - Documentation available via a configuration page (client-agnostic setup details).
