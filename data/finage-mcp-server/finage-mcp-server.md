# Finage MCP Server

**Category:** Finance & Market Data MCP Servers  
**Brand:** Finage  
**Slug:** `finage-mcp-server`

## Description
Finage MCP Server is an MCP-compatible server that exposes Finage’s real-time and historical market data to MCP tools and chat clients. It provides access to data for stocks, currencies, cryptocurrencies, indices, ETFs, and fundamentals through a single MCP endpoint.

## Features
- MCP server endpoint for Finage market data
- Access to **real-time** market data
- Access to **historical** market data
- Coverage for:
  - Stocks
  - Currencies / FX
  - Cryptocurrencies
  - Indices
  - ETFs
  - Fundamental data
- Single static MCP server URL usable across compatible clients
- Authentication handled when adding the server within your application
- Built and hosted via Pipedream Connect infrastructure

## MCP Server URL
Use this static URL for all supported MCP clients (authentication is performed when you add the server to your app):

```text
https://mcp.pipedream.net/v2
```

## Getting Started
- Add the MCP server URL to your MCP-compatible chat client or tool.
- Follow your client’s instructions for configuring authentication and connecting to Finage via MCP.
- Refer to the platform’s configuration documentation for detailed setup steps.
