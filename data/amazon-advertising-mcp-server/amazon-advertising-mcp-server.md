# Amazon Advertising MCP Server

Ad solutions connector that exposes Amazon Advertising capabilities to MCP-based automations.

## Overview
The Amazon Advertising MCP Server is an MCP (Model Context Protocol) server connector provided via Pipedream Connect. It gives MCP-compatible clients and automations programmatic access to Amazon Advertising solutions for reaching, attracting, and engaging Amazon customers across their journey.

- **Type:** MCP server connector
- **Provider / Platform:** Pipedream Connect
- **Category:** Business & Commerce / MCP Servers
- **Use case:** Integrate Amazon Advertising features into AI agents, chat clients, and automated workflows.

## MCP Server URL
- **Base MCP Server URL:** `https://mcp.pipedream.net/v2`
- This is a **static URL** used by all clients.
- Authentication is performed when adding the server to your specific application or client.

## Features
- **MCP-compatible server** for integrating Amazon Advertising into tools and automations that support MCP.
- **Static endpoint** (`https://mcp.pipedream.net/v2`) usable across different clients.
- **Authentication at client setup time**, allowing the same URL to serve multiple authenticated environments.
- **Works with multiple chat clients**: can be added to supported chat or agent clients that understand MCP.
- **Central configuration**: additional setup and options are available via the Pipedream Connect configuration page.
- **Pipedream-hosted** infrastructure, reducing the need to self-host an MCP server.

## Usage
1. Use the static MCP server URL: `https://mcp.pipedream.net/v2` in your MCP-compatible client or application.
2. Configure authentication when adding the server to your app.
3. Refer to the Pipedream **Configuration** page for client-specific setup instructions.
