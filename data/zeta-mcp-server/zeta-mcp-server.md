# Zeta MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Zeta Global (formerly Boomtrain)

## Overview
Zeta MCP Server exposes Zeta’s (formerly Boomtrain) data‑driven marketing capabilities to MCP‑compatible tools and agents. It is accessed through a static MCP endpoint operated via Pipedream Connect.

## MCP Server URL
- Base MCP server URL (static for all clients):  
  `https://mcp.pipedream.net/v2`
- Authentication is performed when adding the server to your application or chat client.

## Features
- **MCP-compatible marketing integration**: Connects Zeta’s data‑driven marketing platform with MCP-compatible tools, agents, and chat clients.
- **Static server endpoint**: Uses a single, stable MCP URL (`https://mcp.pipedream.net/v2`) for all clients.
- **Client-agnostic setup**: Works with multiple chat clients; configuration is handled when you add the server to your specific client.
- **Centralized configuration docs**: Full setup and configuration details are available via the Pipedream Configuration page.
- **Operated via Pipedream Connect**: Hosted and delivered through Pipedream’s integration infrastructure.

## Usage Notes
- Add the MCP server URL to your MCP-compatible client or application.
- Complete authentication during the “add server” or configuration step within your client.
- Refer to the configuration documentation for client-specific setup instructions.
