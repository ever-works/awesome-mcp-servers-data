# CleverReach MCP Server

## Overview
The CleverReach MCP Server is an MCP (Model Context Protocol) server integration that connects MCP-compatible clients to CleverReach’s email marketing platform. It enables creating and sending professional newsletters and managing email marketing campaigns directly from MCP clients.

- **Name:** CleverReach MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Brand:** CleverReach  
- **Source URL:** https://mcp.pipedream.com/app/cleverreach

## Features
- **MCP server integration for CleverReach**  
  - Exposes CleverReach email marketing functionality via a standard MCP server endpoint.  
  - Accessible by any compatible MCP client or chat-based application.

- **Static MCP server URL**  
  - Single, static endpoint for all clients:  
    - `https://mcp.pipedream.net/v2`  
  - Authentication occurs when adding the server within the client or application.

- **Newsletter and campaign creation**  
  - Integrates with CleverReach’s tools to create professional newsletters.  
  - Supports using CleverReach for email marketing and newsletter campaigns from within MCP clients.

- **Sending newsletters**  
  - Trigger sending of newsletters and email campaigns through the MCP integration.

- **Client-agnostic setup**  
  - Works with multiple MCP-compatible chat clients or applications.  
  - Configuration documented via a centralized configuration page.

- **Pipedream-powered infrastructure**  
  - Integration is powered by Pipedream Connect, which hosts and manages the MCP server endpoint.

## Configuration & Setup
- **MCP server endpoint:** `https://mcp.pipedream.net/v2`  
- **Setup flow (high-level):**
  - Add the MCP server URL to your MCP-compatible client or application.  
  - Authenticate the CleverReach MCP server when prompted by your client.  
  - Use the configuration documentation on the Pipedream Configuration page for client-specific setup steps.
