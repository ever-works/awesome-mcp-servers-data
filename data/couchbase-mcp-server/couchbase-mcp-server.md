## Overview

Couchbase is an award-winning distributed NoSQL cloud database that delivers unmatched versatility, performance, scalability, and financial value for cloud, mobile, AI, and edge computing applications. It offers real-time performance, in-memory speed, and GenAI-ready search for scalable NoSQL applications.

## Vector Search Capabilities

Couchbase 8.0 delivers high performance, high recall, and millisecond latency at billion scale – 350+ times faster than MongoDB.

### Vector Indexing Options

- **Hyperscale Vector Index**: For billion-scale data ideal for RAG, agents, and recommendations applications
- **Composite Vector Index**: For search applications where prefiltering massive data is needed
- **Three Indexing Options**: Unique offering to match performance, recall, cost, and query needs

## Features

- **Semantic Search**: Natural language processing for accurate results based on intent and context
- **AI Framework Integration**: Compatible with LangChain and popular AI frameworks
- **Vector MCP Server**: Available through unified Vector MCP Server supporting multiple databases
- **Hybrid Search**: Combine vector and lexical search for optimal retrieval
- **Real-time Performance**: In-memory speed for mission-critical applications
