# Vonage MCP Server

An MCP Server integration for Vonage that brings Vonage’s business phone, VoIP, communication APIs, and contact center capabilities into MCP environments.

- **Website:** https://mcp.pipedream.com/app/vonage
- **Category:** Messaging MCP Servers
- **Brand:** Vonage
- **Slug:** `vonage-mcp-server`
- **Tags:** `voip`, `telecom`, `messaging`

## Features

- Integrates Vonage business phone and VoIP capabilities into MCP-compatible clients.
- Exposes Vonage communication APIs (e.g., messaging/voice) as MCP tools.
- Supports contact center–style communication workflows through MCP.
- Single static MCP server URL for all clients:
  - `https://mcp.pipedream.net/v2`
- Works with multiple chat clients; configuration is done by adding the MCP server URL to the chosen client.
- Authentication handled when adding the server to the application (per-client auth flow).
- Central configuration and documentation available via a dedicated configuration page in Pipedream.

## Setup / Configuration

1. Connect a Vonage account within the Pipedream Vonage MCP app.
2. Copy the static MCP server URL: `https://mcp.pipedream.net/v2`.
3. Add the MCP server to your compatible chat client or application.
4. Complete authentication when prompted in the client.
