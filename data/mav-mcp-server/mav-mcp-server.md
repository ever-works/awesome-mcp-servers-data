# Mav MCP Server

**Category:** Messaging MCP Servers  
**Tags:** sms, lead-generation, customer-engagement

## Description
Mav MCP Server connects Mav’s automated SMS lead conversation platform to MCP-compatible AI tools, enabling conversational lead engagement workflows through those integrations.

It allows you to use your Mav account from AI tools like Cursor, Claude, and Windsurf via Pipedream’s MCP server.

## Features
- Connects a Mav account to MCP via Pipedream’s MCP server
- Integrates Mav with AI tools such as Cursor, Claude, and Windsurf
- Enables automated SMS-based lead conversations through MCP
- Supports building conversational lead engagement workflows using MCP integrations

## Integrations
- Pipedream MCP Server
- Cursor
- Claude
- Windsurf
