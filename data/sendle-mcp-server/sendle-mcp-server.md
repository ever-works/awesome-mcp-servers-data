# Sendle MCP Server

An MCP server that connects applications to Sendle’s 100% carbon neutral shipping services for small businesses via the MCP (Model Context Protocol).

## Source
- **URL:** https://mcp.pipedream.com/app/sendle

## Category
- Business & Commerce → MCP Servers

## Tags
- Shipping
- Logistics
- Small business

## Features
- **MCP integration endpoint**  
  - Static MCP server URL for all compatible clients:  
    `https://mcp.pipedream.net/v2`
  - Works as a shared endpoint; authentication is handled when adding the server in each client/app.

- **Client-agnostic setup**  
  - Designed to be added to different chat or MCP-compatible clients.  
  - Per-client setup instructions available via the configuration flow (referenced "Configuration" page).

- **Sendle shipping service access**  
  - Connects to Sendle’s shipping services optimized for small businesses.  
  - Uses Sendle’s 100% carbon neutral shipping offering for parcel delivery.

- **Hosted by Pipedream**  
  - MCP server is provided and hosted via Pipedream Connect infrastructure.

## Authentication
- Authentication is performed when adding the MCP server to your application or chat client, using the static URL `https://mcp.pipedream.net/v2`.
