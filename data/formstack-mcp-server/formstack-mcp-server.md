# Formstack MCP Server

Workplace productivity and automation MCP server integrating Formstack with MCP-compatible clients.

## Overview
The Formstack MCP Server is a static MCP endpoint provided via Pipedream Connect. It is designed to be added to MCP-compatible chat or agent clients to work with Formstackâ€™s workplace productivity and automation tools (e.g., forms, workflows, data capture) once authenticated.

## MCP Server URL
- Base MCP server URL (static for all clients):
  - `https://mcp.pipedream.net/v2`
- Authentication is handled when adding the server to your MCP-compatible application.

## Features
- **Static MCP endpoint**
  - A single, static MCP server URL used by all clients.
- **Client-agnostic setup**
  - Works with any MCP-compatible chat or agent client.
- **Authentication at client setup**
  - You authenticate as part of adding the server to your application.
- **Pipedream Connect integration**
  - MCP server is powered and hosted via Pipedream Connect.
- **Central configuration resources**
  - Dedicated configuration documentation page is available to guide setup and usage.

## Setup & Configuration
- Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server in your chosen MCP-compatible chat client or app.
- Follow the steps in the client-specific instructions or refer to the full configuration page (linked from the source site) for:
  - Adding the MCP endpoint
  - Handling authentication
  - Managing connection via Pipedream Connect
