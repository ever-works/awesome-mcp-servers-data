# Goodbits MCP Server

**Category:** Content Management MCP Servers  
**Tags:** newsletter, email-marketing, content-curation

## Overview
Goodbits MCP Server is an integration that connects a Goodbits account to AI tooling via Pipedream’s MCP server. It is designed to support the creation of curated email newsletters and the collection of content from the web and popular apps.

## Features
- Connects a Goodbits account to AI tools through Pipedream’s MCP server
- Supports curated email newsletter creation
- Enables content collection from the web
- Integrates with popular apps for sourcing newsletter content
- Usable with AI tools such as:
  - Cursor
  - Claude
  - Windsurf

## Integrations
- **Pipedream MCP Server** – acts as the bridge between Goodbits and AI tools
- **AI Tools** – integration with tools including Cursor, Claude, and Windsurf
