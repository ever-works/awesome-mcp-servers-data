## Description
Node MCP Server is an MCP server that enables Node.js execution inside MCP-based workflows, allowing you to run Node code and leverage most npm packages as part of your automations.

## Features
- Execute arbitrary Node.js code within MCP workflows.
- Use most of npm’s 400,000+ packages in your automations.
- Works as a general-purpose Node.js runtime inside compatible MCP clients.
- Provides a static MCP server URL that can be used across different clients.
- Integrates with MCP-based chat or automation applications by adding the server as an MCP endpoint.

## MCP Server URL
- Static MCP server endpoint (works for every client):
  - `https://mcp.pipedream.net/v2`
- Authentication occurs when adding the server in your application/client.

## Integration & Configuration
- Add the Node MCP Server to your preferred MCP-compatible chat client or application.
- Detailed setup and configuration instructions are available via the platform’s Configuration page (linked from the app page).
