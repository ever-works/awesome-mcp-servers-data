# OpenSea MCP Server

**Category:** Blockchain & Crypto MCP Servers  
**Brand:** OpenSea  
**Slug:** `opensea-mcp-server`

## Overview
The OpenSea MCP Server is an integration that connects MCP-compatible tools and chat clients to the OpenSea NFT marketplace. It enables interaction with OpenSea for discovering, collecting, and selling NFTs through a standardized MCP server endpoint.

## Features
- **MCP-compatible server endpoint**  
  - Provides a static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Same URL works for all supported MCP clients.

- **OpenSea marketplace integration**  
  - Designed to interact with OpenSea, a major NFT marketplace.  
  - Supports workflows related to discovering, collecting, and selling NFTs (via the MCP tools provided by the server).

- **Account-based authentication**  
  - Requires connecting your OpenSea (via Pipedream) account.  
  - Authentication is performed when adding the MCP server to your application.

- **Client-agnostic usage**  
  - Can be added to various MCP-compatible chat clients or applications.  
  - Configuration guidance is provided per client type via the host platform’s configuration pages.

- **Hosted by Pipedream**  
  - MCP server infrastructure is hosted and managed by Pipedream, accessible via the given URL.

## Configuration
- Connect your OpenSea account via the Pipedream interface.  
- Copy and use the static MCP server URL: `https://mcp.pipedream.net/v2`.  
- Add the server to your MCP-compatible app or chat client following that client’s configuration instructions.
