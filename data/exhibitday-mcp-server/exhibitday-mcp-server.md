# ExhibitDay MCP Server

**Category:** Project Management MCP Servers  
**Brand:** ExhibitDay  
**Slug:** exhibitday-mcp-server

## Description
ExhibitDay MCP Server is an MCP integration for ExhibitDay that enables planning and collaboration around trade shows and exhibits directly from MCP-compatible chat or agent clients. It connects your ExhibitDay account so teams can manage event-related workflows through MCP-based tools.

## Features
- **ExhibitDay integration via MCP**: Connects to your ExhibitDay account to work with trade show and exhibit planning data through MCP.
- **Team collaboration for events**: Supports collaboration with event teams to coordinate tasks and activities for trade shows and exhibits.
- **Static MCP server URL**: Uses a single static MCP endpoint for all clients: `https://mcp.pipedream.net/v2`.
- **Per-client authentication**: Authentication occurs when adding the server to each application / chat client, allowing the same URL to be reused across clients.
- **Multi-client support**: Can be added to different MCP-compatible chat or agent clients, enabling interaction with ExhibitDay from various frontends.
- **Configuration via web UI**: Web-based configuration flow to connect your ExhibitDay account and select the relevant client for integration.
- **Tool-based workflow access**: Exposes ExhibitDay-related tools / actions as MCP tools (listed dynamically as “available tools” once loaded) for workflow automation and interaction.

## Setup
- Connect your ExhibitDay account via the configuration interface.
- Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server to your MCP-compatible chat or agent client and complete authentication.
