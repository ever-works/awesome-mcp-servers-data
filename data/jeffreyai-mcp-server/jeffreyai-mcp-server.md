# JeffreyAI MCP Server

**Category:** Business & Commerce – MCP Servers  
**Slug:** `jeffreyai-mcp-server`  
**Source:** https://mcp.pipedream.com/app/jeffreyai

## Overview
JeffreyAI MCP Server is an MCP (Model Context Protocol) server that exposes JeffreyAI’s sales and marketing automation capabilities as tools that can be used within business workflows. It is delivered via Pipedream Connect and can be added as a static MCP endpoint to compatible chat or AI clients.

## MCP Server Endpoint
- **Base MCP URL:** `https://mcp.pipedream.net/v2`  
  - Static URL used across supported clients.  
  - Authentication is performed when the server is added to the client/application.

## Features
- **Sales and marketing automation tools via MCP**  
  - Provides access to JeffreyAI’s sales and marketing automation capabilities through the MCP protocol (exact tool list not detailed in the source content).
- **Standard MCP integration**  
  - Exposed as an MCP server that can be added to MCP-compatible chat or AI applications.  
  - Uses a single static server URL for all supported clients.
- **Authentication on connect**  
  - Authentication flow occurs when the MCP server is added to the client, rather than using a per-user custom URL.
- **Configuration resources**  
  - A dedicated configuration page (linked from the app page) describes how to add and configure the server in various clients.
- **Pipedream Connect integration**  
  - Hosted and delivered through Pipedream Connect infrastructure.

## Integration & Usage
- Add the MCP server to a supported chat client or application using the provided static URL.
- Follow client-specific instructions (via the configuration documentation) to complete setup and authentication.
