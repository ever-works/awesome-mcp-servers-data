# Awesome Crypto MCP Servers

Curated open-source GitHub list of MCP servers enabling crypto APIs, wallets, and blockchain interactions for AI agents.

## Overview

Provides standardized MCP access to cryptocurrency APIs and wallets, supporting transaction querying, NFT retrieval/management, smart contract interactions, and more. Serves as an AI-agent-friendly alternative to Web3.js libraries by exposing blockchain functionality through natural language tool calls.

## Key Features

- **Transaction Querying**: Fetch transaction history, details, confirmations via MCP tools
- **NFT Management**: Retrieve NFT metadata, ownership, transfers
- **Wallet Operations**: Balance checks, signing (secure), address generation
- **DeFi Integration**: Lending/borrowing queries, liquidity pool data, yield farming metrics
- **Real-time Data**: Price feeds, mempool monitoring, block events

## Use Cases

- **DeFi Agents**: Autonomous trading, portfolio rebalancing, yield optimization
- **NFT Agents**: Collection valuation, rarity analysis, marketplace scanning
- **Wallet Assistants**: Transaction simulation, gas estimation, security audits

## Comparison to Web3.js

| Feature | MCP Servers | Web3.js |
|---------|-------------|---------|
| **Integration** | Natural language tool calls | JavaScript SDK calls |
| **AI Compatibility** | Native LLM tool calling | Requires code generation/wrapping |
| **Multi-Chain** | Often unified endpoints | Provider-specific |
| **Standardization** | MCP protocol | EVM-specific |
