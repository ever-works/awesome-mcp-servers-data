# Awesome Crypto MCP Servers

A curated, open-source GitHub “awesome list” of crypto-focused Model Context Protocol (MCP) servers, maintained by Luke Fan (badkk).

## Overview
Awesome Crypto MCP Servers aggregates links and references to MCP server projects and related learning materials in the cryptocurrency and blockchain domain.

## Features
- **Curated list of crypto MCP servers**
  - Focused specifically on Model Context Protocol servers used in crypto and blockchain contexts.
  - Structured as an “awesome list” for easy browsing and discovery.
- **Tutorials section**
  - Dedicated area for learning resources and guides related to crypto MCP servers (as they are added to the list).
- **Server implementations**
  - References to concrete MCP server implementations used in crypto applications.
  - Includes ecosystem projects such as those crafted by Magnet Labs and community collaborators.
- **Resources section**
  - Space for additional documentation, articles, tools, and other helpful links around MCP in crypto.
- **Open-source GitHub repository**
  - Publicly accessible on GitHub for viewing, cloning, and forking.
  - Includes a `LICENSE` file specifying open-source licensing terms.
  - Maintained by contributors from the AI & crypto community.
