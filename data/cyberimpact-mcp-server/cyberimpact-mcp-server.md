# Cyberimpact MCP Server

An MCP server that connects Cyberimpact’s email marketing and campaign management capabilities to MCP-based workflows and automations.

## Overview
- **Type:** MCP server integration
- **Category:** Business & Commerce – MCP Servers
- **Brand:** Cyberimpact
- **Slug:** `cyberimpact-mcp-server`
- **Source URL:** https://mcp.pipedream.com/app/cyberimpact

## Features
- Connects a Cyberimpact account to MCP-compatible AI tools.
- Works with tools such as Cursor, Claude, and Windsurf.
- Exposes Cyberimpact’s email marketing features to automated workflows.
- Exposes campaign management capabilities (e.g., creating, managing, and automating campaigns) to MCP-based tooling.
- Enables building automations that trigger or orchestrate Cyberimpact email and campaign actions from AI-driven environments.

## Integrations
- **Cyberimpact** (email marketing & campaigns)
- **AI / MCP tools:**
  - Cursor
  - Claude
  - Windsurf
