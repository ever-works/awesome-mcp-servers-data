# Whop MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Whop  
**Source:** https://mcp.pipedream.com/app/whop

## Description
Whop MCP Server is an MCP (Model Context Protocol) server that enables MCP-based management and interaction with digital products, memberships, and online communities. It is provided via Pipedream Connect as a static MCP endpoint that can be used from compatible chat or agent clients.

## Features
- MCP-based access to Whop-related functionality for digital products, memberships, and communities
- Single static MCP server URL usable across all supported clients
- Authentication handled when adding the server within each client/application
- Compatible with multiple chat/agent clients that support MCP servers
- Central configuration reference via a dedicated configuration page
- Hosted and managed via Pipedream Connect infrastructure

## Technical Details
- **MCP Server URL:** `https://mcp.pipedream.net/v2`
- **Integration model:** Add as an MCP server in your chat/agent client
- **Authentication:** Performed during server setup inside the client

## Getting Started
1. Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
2. Add it as an MCP server in your preferred MCP-compatible chat client or application.
3. Follow the client-specific instructions or refer to the full configuration page on Pipedream.
