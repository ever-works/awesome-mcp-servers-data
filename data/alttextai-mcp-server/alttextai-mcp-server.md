# AltText.ai MCP Server

## Overview
AltText.ai MCP Server is a Model Context Protocol (MCP) server that enables automatic generation of image alt text in over 130 languages. It is designed to improve website accessibility and SEO, and integrates with platforms like WordPress, Shopify, and various web and CMS systems via Pipedream.

## Features
- **Automatic image alt-text generation**  
  - Generates descriptive alt text for images using AI.  
  - Supports multilingual alt-text creation in over 130 languages.

- **Accessibility improvements**  
  - Helps make websites more accessible for screen-reader users.  
  - Encourages consistent use of alt attributes across images.

- **SEO enhancements**  
  - Alt text generation can support better image search indexing and overall SEO.

- **MCP (Model Context Protocol) integration**  
  - Exposed as an MCP server that can be added to compatible chat or developer tools.  
  - Uses a static MCP server URL: `https://mcp.pipedream.net/v2` for all clients.  
  - Authentication occurs when the server is added to the application.

- **Platform integrations**  
  - Integrations available for:  
    - WordPress  
    - Shopify  
    - Web applications  
    - CMS platforms and other web systems (via Pipedream).

- **Configuration and setup**  
  - Connect an AltText.ai account through Pipedream.  
  - Select a chat client or app and add the MCP server URL.  
  - Additional setup details available via a configuration page on Pipedream.
