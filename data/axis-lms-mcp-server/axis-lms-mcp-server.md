# Axis LMS MCP Server

**Category:** Documentation & Learning Resources  
**Brand:** Axis LMS  
**Slug:** axis-lms-mcp-server

## Description
The Axis LMS MCP Server exposes Axis LMS learning management system capabilities to MCP-aware tools and agents, enabling streamlined access to training and course management features directly from compatible chat clients and applications.

## Features
- Provides a unified MCP endpoint for Axis LMS: `https://mcp.pipedream.net/v2`
- Uses a static MCP server URL that works across clients
- Supports authentication at the time you add the server to your MCP-compatible application
- Integrates with MCP-aware chat clients and tools
- Built and hosted via Pipedream Connect infrastructure
- Central configuration and usage documentation available through the MCP configuration page

## Technical Details
- MCP server base URL: `https://mcp.pipedream.net/v2`
- Authentication: performed when registering/adding the server in the client (exact method depends on the client and Pipedream configuration)
