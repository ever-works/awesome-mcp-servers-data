# Soax MCP Server

Effortless data extraction from any website using managed headless browsers and rotating proxies, exposed as an MCP Server via Pipedream.

- **Website / Source**: https://mcp.pipedream.com/app/scrapein_
- **MCP Server URL**: `https://mcp.pipedream.net/v2`
- **Category**: Content Extraction & Summarization MCP Servers
- **Brand**: Soax
- **Tags**: web-scraping, proxies, data-extraction

## Features

- **Managed headless browsers**
  - Server-side management of headless browsers for web automation and scraping.
  - Eliminates the need to run and maintain your own browser infrastructure.

- **Rotating proxy management**
  - Integrated rotating proxies to reduce blocking and rate-limiting when scraping websites.
  - Proxy handling is abstracted away from the client.

- **MCP-compatible server**
  - Exposed as a Model Context Protocol (MCP) server for use within MCP-enabled environments and chat clients.
  - Single static MCP server URL usable across different clients.

- **Centralized configuration**
  - Uses a static endpoint (`https://mcp.pipedream.net/v2`) for all clients; authentication is handled when adding the server to an application.
  - Configuration details available via the linked configuration page (within Pipedream Connect).
