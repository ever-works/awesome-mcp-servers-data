## Overview
Mem MCP Server is an MCP (Model Context Protocol) server that connects MCP-compatible AI clients to your Mem account. It enables AI knowledge assistant capabilities such as recall and knowledge management directly from tools like Cursor, Claude, and Windsurf.

Source: <https://mcp.pipedream.com/app/mem>

## Features
- **MCP protocol support**: Implements the MCP protocol to act as a bridge between Mem and MCP-compatible AI clients.
- **Mem account integration**: Connects directly to your Mem account to access stored notes and knowledge.
- **AI knowledge assistant capabilities**:
  - Recall information from your Mem knowledge base.
  - Support for knowledge management workflows via AI tools.
- **Multi-client compatibility**:
  - Works with AI development and assistant tools such as:
    - Cursor
    - Claude
    - Windsurf
- **Pipedream-based server**: Runs as a Pipedream-hosted MCP server, simplifying integration and deployment.

## Use Cases
- Query and recall notes and documents stored in Mem from within MCP-compatible editors or AI assistants.
- Enhance AI coding or writing environments (e.g., Cursor, Windsurf) with context drawn from your Mem knowledge base.
- Centralize personal or team knowledge in Mem while exposing it to multiple AI tools through a single MCP integration.
