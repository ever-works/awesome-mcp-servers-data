# Small Improvements MCP Server

## Overview
Small Improvements MCP Server is an MCP (Model Context Protocol) integration for the Small Improvements performance feedback platform. It enables AI tools and chat clients to interact with Small Improvements for performance reviews and feedback-related workflows via a standardized MCP server endpoint.

- **Category:** Business & Commerce – MCP Servers
- **Use Case:** Performance management, HR, and employee engagement workflows

## Features
- **MCP Integration for Small Improvements**  
  Connects AI / MCP-compatible applications to the Small Improvements performance feedback platform.

- **Static MCP Server URL**  
  - Single static endpoint for all clients:  
    `https://mcp.pipedream.net/v2`  
  - Authentication occurs when adding the server to your MCP-compatible application.

- **Client-Agnostic Setup**  
  - Works with multiple chat clients that support MCP.  
  - Instructions available for different chat clients via the configuration UI.

- **Configuration Workflow**  
  - Connect your Small Improvements account through the Pipedream interface.  
  - Select your client and copy the MCP server URL to integrate with your app.  
  - Central configuration page available for more detailed setup steps.

- **Tooling / Actions Support**  
  - Exposes Small Improvements actions as MCP tools (listed as “available tools” once loaded in the interface).  
  - Designed to enable AI-assisted performance review and feedback workflows.
