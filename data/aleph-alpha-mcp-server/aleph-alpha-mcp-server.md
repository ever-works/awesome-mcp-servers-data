# Aleph Alpha MCP Server

## Overview
The Aleph Alpha (Luminous) MCP Server provides access to Aleph Alpha’s Europe-based Luminous large language models via the Model Context Protocol (MCP). It is delivered and hosted through Pipedream Connect as a static MCP endpoint that can be integrated into compatible chat or tooling clients.

- **Name:** Aleph Alpha MCP Server
- **Brand:** Aleph Alpha
- **Category:** AI Integration – MCP Servers
- **Tags:** language-models, generative-ai, ai-integration
- **Source URL:** https://mcp.pipedream.com/app/luminous

## Features
- **MCP-Compatible Server**  
  - Exposes Aleph Alpha’s Luminous models over the Model Context Protocol.  
  - Designed to work with any MCP-compatible client or chat application.

- **Static MCP Endpoint**  
  - Single static server URL usable across clients:  
    - `https://mcp.pipedream.net/v2`  
  - Authentication handled when adding the server to each application.

- **Europe-Based LLM Access**  
  - Connects to Aleph Alpha’s Luminous large language models, hosted in Europe.

- **Client-Agnostic Integration**  
  - Documentation and setup flows for multiple chat clients.  
  - Central configuration guidance via a shared **Configuration** page (on Pipedream) for setup details.

- **Delivered via Pipedream Connect**  
  - MCP server is operated/packaged by Pipedream.  
  - Integrates into broader Pipedream Connect workflows and infrastructure.

## MCP Server URL
- **Base URL:** `https://mcp.pipedream.net/v2`
- **Usage:** Add this URL as an MCP server in your client; complete authentication in the client during setup.
