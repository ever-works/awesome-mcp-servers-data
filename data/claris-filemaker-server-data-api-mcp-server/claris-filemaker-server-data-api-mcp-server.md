# Claris FileMaker Server Data API MCP Server

## Overview
Claris FileMaker Server Data API MCP Server exposes FileMaker Server–hosted databases to MCP-compatible clients, enabling programmatic querying and data manipulation via the FileMaker Data API.

- **Name:** Claris FileMaker Server Data API MCP Server  
- **Category:** Database & Messaging MCP Servers  
- **Brand:** claris-filemaker  
- **Source URL:** https://mcp.pipedream.com/app/claris_filemaker_server_data_api

## Features
- **MCP Server for FileMaker Data API**  
  - Acts as an MCP (Model Context Protocol) server frontend for the Claris FileMaker Server Data API.  
  - Exposes FileMaker databases to compatible MCP clients.

- **Programmatic Data Access**  
  - Allows clients to query data stored in FileMaker-hosted databases.  
  - Supports programmatic manipulation of FileMaker data (e.g., create, read, update, delete operations) through the underlying FileMaker Data API.

- **Static MCP Endpoint**  
  - Uses a single static MCP server URL for all clients:  
    - `https://mcp.pipedream.net/v2`  
  - Authentication is handled when adding / configuring the server in the client.

- **Client Integration**  
  - Designed to be added to compatible chat or MCP-enabled applications as an external server.  
  - Configuration details are available via a dedicated configuration page (referenced, but specifics not provided in the content).

## Usage
- Add the MCP server to your MCP-capable or chat-based application using the static URL: `https://mcp.pipedream.net/v2`.
- Authenticate within your application when prompted to gain access to FileMaker Server–hosted databases.
