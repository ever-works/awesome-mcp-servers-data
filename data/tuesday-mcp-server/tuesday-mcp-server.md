## Overview
The Tuesday MCP Server is hosted via Pipedream and exposes a static MCP endpoint that Tuesday-compatible clients can use to power AI-assisted, personalized outreach and campaign workflows.

## Features
- **Static MCP endpoint**: Single, reusable MCP server URL (`https://mcp.pipedream.net/v2`) that works across supported clients.
- **Client-agnostic URL**: Same endpoint for all compatible MCP-based chat or agent clients; authentication is handled per client during setup.
- **AI-assisted outreach**: Designed for use with Tuesday to help agents scale personalized outreach and campaign activities.
- **Campaign support**: Intended to support marketing and campaign-related workflows such as personalized messaging and campaign automation (as implied by the integration purpose).
- **Configuration documentation**: Dedicated configuration page (via Pipedream) describing how to add and configure the server with different chat clients.
- **Pipedream Connect integration**: Runs on Pipedream Connect infrastructure, benefiting from its standardized connection and configuration model.

## Setup
- Use the static server URL `https://mcp.pipedream.net/v2` in your MCP-compatible client.
- Authenticate within your client when adding the server.
- Refer to the Pipedream configuration page for client-specific setup instructions.
