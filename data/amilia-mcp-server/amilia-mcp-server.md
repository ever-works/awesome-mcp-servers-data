# Amilia MCP Server

**Category:** Business & Commerce · MCP Servers  
**Slug:** `amilia-mcp-server`

## Overview
Amilia MCP Server exposes an MCP-compatible interface to Amilia’s SmartRec eCommerce platform. It enables MCP-based tools and chat clients to interact with SmartRec so organizations can manage and sell community activities and programs through the MCP server.

SmartRec is Amilia’s eCommerce platform for offering, managing, and selling activities to communities.

## Features
- **MCP-compatible interface**
  - Provides an MCP server that can be added to MCP-capable chat clients and tools.
  - Uses a single static MCP server URL for all clients.

- **SmartRec platform integration**
  - Connects to Amilia’s SmartRec eCommerce platform.
  - Supports managing and selling activities and programs offered to communities.

- **Static server endpoint**
  - MCP server URL: `https://mcp.pipedream.net/v2` (same for all clients).
  - Authentication is performed when adding the server to an application or client.

- **Client-agnostic usage**
  - Can be integrated with multiple chat clients that support MCP.
  - Configuration instructions are available per client via the configuration documentation.

- **Hosted by Pipedream Connect**
  - MCP server is powered and hosted through Pipedream’s Connect infrastructure.

## Configuration
- Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
- Authenticate during the “add server” / connection step within your MCP-compatible application or chat client.
- Additional setup and client-specific instructions are available on the provider’s configuration page (linked from the original site).
