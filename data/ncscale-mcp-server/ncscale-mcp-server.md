# ncScale MCP Server

**Category:** Monitoring  
**Brand:** ncScale  
**Website:** https://mcp.pipedream.com/app/ncscale

## Overview
The ncScale MCP Server is an MCP (Model Context Protocol) server integration for ncScale that allows AI agents to monitor and observe no-code applications. It exposes ncScaleâ€™s reliability, security, and visibility tooling to AI clients via a standard MCP endpoint.

## Features
- **MCP Server Integration**
  - Provides a static MCP server URL: `https://mcp.pipedream.net/v2`.
  - Can be added to compatible chat or AI clients that support MCP.

- **No-Code Monitoring and Observability**
  - Designed for monitoring no-code applications.
  - Focuses on improving reliability, security, and visibility of no-code stacks.

- **Centralized Configuration**
  - Connect an ncScale account and select a client to get started.
  - Configuration and setup flow available via a dedicated configuration page on Pipedream.

- **Tooling Exposure to AI Agents**
  - Exposes ncScale tools and actions ("Available tools") through MCP for use by AI agents.
  - Tools are loaded dynamically from the server for the connected client.

- **Authentication at Client Level**
  - The server URL is static; authentication occurs when adding the server to the application/client.

## Installation / Setup
- Copy the MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server URL to a supported MCP-compatible chat or AI client.
- Authenticate and connect your ncScale account within the client.
- Optionally refer to the full configuration page on Pipedream for client-specific instructions.
