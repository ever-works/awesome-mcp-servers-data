# ClassMarker MCP Server

**Description**  
ClassMarker MCP Server is a Model Context Protocol (MCP) server that connects AI tools and chat clients to ClassMarker’s online testing and quiz-making platform, enabling programmatic quiz management and assessment workflows.

**Category**  
Business & Commerce · MCP Servers

**Tags**  
- Assessment  
- Testing  
- Education

**Integration URL**  
- MCP Server base URL (static for all clients):  
  `https://mcp.pipedream.net/v2`

**Usage / Setup**  
- Add the MCP server URL to any compatible MCP client or chat application.  
- Authentication is performed when you add the server to your application.  
- Configuration details are available via the platform’s configuration page/documentation.

## Features
- **Model Context Protocol server** for integrating ClassMarker with AI tools and chat clients.  
- **Programmatic access to ClassMarker** for online testing and quiz creation workflows (via MCP).  
- **Static server URL** (`https://mcp.pipedream.net/v2`) usable across different MCP clients.  
- **Authentication on connection** when adding the server to an application.  
- **Pipedream Connect–hosted integration**, benefiting from Pipedream’s MCP infrastructure.
