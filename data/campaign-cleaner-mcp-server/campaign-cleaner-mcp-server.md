# Campaign Cleaner MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Campaign Cleaner  
**Slug:** `campaign-cleaner-mcp-server`

MCP (Model Context Protocol) server that exposes Campaign Cleaner tools for optimizing and cleaning email campaigns inside MCP-compatible clients and workflows.

---

## Overview

Campaign Cleaner MCP Server provides access to Campaign Cleaner’s email optimization and cleaning capabilities from any MCP-compatible chat client or application. It is delivered via Pipedream Connect as a static MCP endpoint that you add to your client configuration.

---

## MCP Server URL

Use this static URL in any MCP-compatible client:

```text
https://mcp.pipedream.net/v2
```

Authentication is handled when you add the server to your application.

## Usage

1. **Copy the MCP server URL**
   - `https://mcp.pipedream.net/v2`
2. **Add to your MCP client**
   - Add the URL as an MCP server endpoint in your chat client or application.
   - Authenticate when prompted by the client.
3. **Invoke Campaign Cleaner tools**
   - Use the client’s MCP tooling interface to call Campaign Cleaner tools for email campaign cleaning/optimization (exact tools not detailed in the provided content).
