## Typescript MCP SDK

**Category:** MCP server directories & lists  
**Brand:** model-context-protocol

### Overview
Typescript MCP SDK is the official TypeScript/JavaScript SDK for building Model Context Protocol (MCP) servers, clients, and tools. It provides the core interfaces, helpers, and runtime support needed to let LLMs safely access external tools and data through MCP-compliant services.

### Features
- Official SDK for Model Context Protocol (MCP) servers and clients
- Supports building MCP servers and tools in TypeScript and JavaScript
- Core interfaces for implementing MCP-compatible servers
- Helper utilities for wiring up MCP tools and data sources
- Runtime support for safely exposing external tools and data to LLMs via MCP
- Designed to integrate MCP-based capabilities into LLM applications
