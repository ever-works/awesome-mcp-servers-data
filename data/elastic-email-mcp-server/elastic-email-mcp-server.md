# Elastic Email MCP Server

**Category:** Business & Commerce – MCP Servers  
**Slug:** elastic-email-mcp-server  
**Brand:** Elastic Email

## Overview
Elastic Email MCP Server is an MCP (Model Context Protocol) server integration that exposes Elastic Email’s email marketing and delivery capabilities to MCP‑aware agents and tools via the Pipedream MCP platform. It lets applications and chat-based agents perform common email marketing and contact management operations programmatically.

## Features

### MCP Server & Connectivity
- Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`
- Works with multiple chat or MCP‑aware clients (added as a server in the client’s configuration)
- Authentication handled when adding the server to the application
- Configuration documented on a dedicated configuration page (Pipedream MCP configuration)

### Email Sending & Campaign Management
- **Send Email**  
  - Send an email to one or more recipients using Elastic Email.

- **Create Campaign**  
  - Create a new email campaign in an Elastic Email account.

- **Update Campaign**  
  - Modify properties of an existing campaign (e.g., content, settings, or metadata) in an Elastic Email account.

- **List Campaigns**  
  - Retrieve a list of existing campaigns in an Elastic Email account, enabling discovery, selection, or reporting by MCP‑aware tools.

### Contact & List Management
- **Create Contact**  
  - Create a new contact in an Elastic Email account with relevant fields.

- **Update Contact**  
  - Update details of an existing contact in an Elastic Email account.

- **List Contacts**  
  - Fetch a list of contacts from an Elastic Email account, supporting segmentation, analysis, or automation flows.

- **Add Contact to Mailing List**  
  - Add a new or existing contact to a specified mailing list.

- **Unsubscribe Contact**  
  - Unsubscribe a contact from future emails to maintain compliance and list hygiene.

### Segmentation
- **Create Segment**  
  - Create a segment in an Elastic Email account for targeted emailing based on defined criteria.
