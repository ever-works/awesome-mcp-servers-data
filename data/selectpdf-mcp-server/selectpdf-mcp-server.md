# SelectPdf MCP Server

## Overview
SelectPdf MCP Server is a Pipedream-based Model Context Protocol (MCP) integration that exposes SelectPdf’s HTML-to-PDF conversion capabilities via a REST API for use within AI tools and development environments.

## Features
- Connects a SelectPdf account to AI tools and IDEs that support MCP (e.g., Cursor, Claude, Windsurf)
- Provides HTML-to-PDF document conversion through a REST API
- Integrates with Pipedream’s MCP server infrastructure for configuration and connection management
- Designed for document conversion workflows (HTML to PDF) within AI-assisted development environments
