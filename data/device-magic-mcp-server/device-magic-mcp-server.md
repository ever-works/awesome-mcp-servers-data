# Device Magic MCP Server

Industrial-grade mobile data collection MCP server integrating Device Magic with the Model Context Protocol.

## Overview
- **Type:** MCP Server (Model Context Protocol server)
- **Platform:** Device Magic (mobile data collection) via Pipedream Connect
- **Category:** Business & Commerce MCP Servers
- **Use Cases:** Data collection, field service operations, mobile forms

## Features
- **Static MCP Server Endpoint**  
  - Single shared URL for all clients:  
    `https://mcp.pipedream.net/v2`  
  - Authentication handled when adding the server to your MCP-compatible application.

- **MCP Integration**  
  - Designed to work with MCP-compatible chat clients and tools.  
  - Central endpoint to connect Device Magic’s mobile data collection capabilities into MCP workflows.

- **Client-Agnostic Setup**  
  - Same server URL works across different clients.  
  - Setup instructions available per chat client via the “Add the server to your app” flow.  
  - Additional configuration details available on the linked Configuration page (not included in source content).

- **Pipedream Connect Infrastructure**  
  - Hosted and powered by Pipedream Connect.  
  - Covered by Pipedream’s Terms and Privacy Policy.
