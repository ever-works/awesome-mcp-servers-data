# Tidy MCP Server

## Overview
Tidy MCP Server is an MCP server that integrates the TIDY automation platform into MCP-compatible clients, allowing agents to manage and automate cleaning and maintenance tasks (for homes, rentals, and workspaces).

- **Name:** Tidy MCP Server  
- **Category:** Workflow Automation MCP Servers  
- **Brand:** TIDY  
- **Source URL:** https://mcp.pipedream.com/app/tidy  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **TIDY Platform Integration**: Connects MCP-compatible applications to the TIDY automation platform so agents can work with cleaning and maintenance tasks via MCP tools.
- **Automation of Cleaning & Maintenance**: Supports automation workflows for maintaining:
  - Homes
  - Rentals
  - Employee workspaces and offices
- **Static MCP Endpoint**: Uses a single static MCP server URL (`https://mcp.pipedream.net/v2`) that works across compatible clients.
- **Client-Agnostic Setup**: Designed to be added to various chat or agent clients that support MCP, with configuration handled within the client.
- **Authentication at Client Level**: Authentication is performed when adding the server in the client application, not tied to a specific URL per user.

## Usage
- Add the MCP server to a supported MCP client using: `https://mcp.pipedream.net/v2`.
- Configure and authenticate within the client to enable TIDY automation tools for cleaning and maintenance workflows.
