# Teamcamp MCP Server

**Category:** Project Management MCP Servers  
**Slug:** `teamcamp-mcp-server`

## Overview
Teamcamp MCP Server is an MCP-compatible integration for the Teamcamp project management platform. It exposes Teamcamp’s project management, tasks, and collaboration capabilities to MCP-compatible tools and chat clients via a static MCP server URL.

## Features
- **MCP-compatible server**  
  - Provides a static MCP server URL: `https://mcp.pipedream.net/v2`  
  - Can be added to various MCP-capable chat clients and applications.

- **Teamcamp integration**  
  - Connects to a Teamcamp account to surface project management data.  
  - Uses your Teamcamp client/account selection during configuration.

- **Project management & business management data access**  
  - Designed to work with Teamcamp’s all-in-one project and business management capabilities.  
  - Intended to enable access to projects, tasks, and collaboration data from within MCP-compatible tools.

- **Configuration flow**  
  - "Configure Teamcamp" step to connect your Teamcamp account.  
  - Account verification step ("Checking your account…").  
  - Static URL works for every client; authentication occurs when adding the server to your application.  
  - Guidance to add the server to supported chat clients.  
  - Additional setup details available via a configuration page (not detailed in the provided content).

- **Tools exposure (via MCP)**  
  - Exposes Teamcamp functionality as MCP tools/actions (listed as "Available tools" in the interface; concrete tools are not detailed in the provided content).  
  - Designed so MCP-compatible apps can call these tools programmatically.
