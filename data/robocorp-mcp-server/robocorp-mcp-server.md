# Robocorp MCP Server

**Category:** Workflow Automation MCP Servers  
**Brand:** Robocorp  
**Slug:** robocorp-mcp-server

## Description
The Robocorp MCP Server is a Pipedream-hosted MCP (Model Context Protocol) server that connects MCP-compatible clients to Robocorp’s open-source, Python-based automation and GenAI stack. It is designed to simplify automation and enable AI-powered workflows that can be built locally, operated from the cloud, and run in diverse environments.

Source: https://mcp.pipedream.com/app/robocorp

## Features
- **MCP-compatible server endpoint**: Provides a static MCP server URL that can be used with any supported MCP client.
- **Static server URL**: Single endpoint usable across clients:  
  `https://mcp.pipedream.net/v2`
- **Authentication at client setup**: Authentication occurs when adding the server to your chat or MCP-enabled application.
- **Integration with Robocorp stack**: Connects clients to Robocorp’s open-source, Python-based automation tools and GenAI capabilities.
- **Flexible deployment model**: Supports building automation locally, operating from the cloud, and running in varied environments (“run anywhere”).
- **Client-agnostic usage**: Can be added to multiple MCP-capable chat clients; configuration guidance is available per client type.
- **Configuration documentation**: Usage and setup details provided via a dedicated configuration page.

## Setup & Integration
- Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server in your chosen MCP-compatible chat client or application.
- Follow client-specific instructions or refer to the full configuration documentation on the provider’s configuration page.
