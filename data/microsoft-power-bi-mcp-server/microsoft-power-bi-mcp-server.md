# Microsoft Power BI MCP Server

**Category:** Data Visualization  
**Brand:** Microsoft  
**Type:** MCP Server integration / data analytics connector

An MCP server integration for Microsoft Power BI that lets you visualize and analyze data and embed Power BI visuals into MCP-enabled applications and workflows.

---

## Features

### MCP Server & Configuration
- Provides a static MCP server URL for all clients: `https://mcp.pipedream.net/v2`.
- Connects Microsoft Power BI as a managed account within MCP-enabled apps.
- Supports authentication flow when adding the server to an application.
- Works across multiple chat or MCP clients via a common configuration.

### Data Visualization & Analytics
- Enables visualization and analytics of data using Microsoft Power BI.
- Supports embedding Power BI visuals into MCP-enabled applications and workflows.
- Suitable for both self-service analytics and business intelligence scenarios.

### Available Tools (Actions)
The MCP server exposes 6 Power BI actions as tools:

1. **Refresh Dataset**
   - Triggers a refresh operation for a specified Power BI dataset.

2. **Get Reports**
   - Retrieves reports from a specified Power BI workspace.

3. **Get Dataset Refresh**
   - Retrieves information about refresh operations for a specified Power BI dataset.

4. **Create Dataset**
   - Creates a new Push Dataset in Power BI.

5. **Cancel Dataset Refresh**
   - Cancels an ongoing refresh operation for a specified dataset.

6. **Add Rows to Dataset Table**
   - Adds new data rows to a specified table within a specified dataset in **My workspace**.

---

## Integration & Usage
- Add the MCP server URL to your chosen chat or MCP client.
- Sign in with your Microsoft Power BI account to configure and manage the connection.
- Use the exposed tools to automate dataset management, refresh cycles, and data ingestion, and to retrieve reports programmatically within MCP-enabled apps.
