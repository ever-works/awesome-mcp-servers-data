# L2S MCP Server

**Description**

L2S MCP Server is an MCP (Model Context Protocol) server for L2S, providing tools for all‑in‑one link management, including organization, tracking, and sharing. It is delivered via Pipedream Connect and can be used by any compatible MCP client through a static server URL.

**Server URL**

- MCP Server base URL (static for all clients):
  - `https://mcp.pipedream.net/v2`

**Features**

- All‑in‑one link management via MCP
- Link organization
- Link tracking
- Link sharing
- Works with any compatible MCP client using a single static URL
- Authentication handled when adding the server to your application
- Hosted and powered by Pipedream Connect

**Integration & Configuration**

- Add the server to your MCP‑compatible chat/client app using the static URL
- Client‑specific setup instructions available via the configuration documentation ("Configuration" page on Pipedream)
