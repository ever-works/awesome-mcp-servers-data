# Better Proposals MCP Server

**Category:** Business & Commerce – MCP Servers  
**Brand:** Better Proposals  
**Source:** https://mcp.pipedream.com/app/better_proposals

## Overview
The Better Proposals MCP Server connects Better Proposals’ proposal creation and management tools to MCP-compatible clients. It enables automated and streamlined proposal workflows directly from supported chat or MCP applications.

## Features
- **MCP server endpoint**  
  - Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`  
  - Authentication handled when adding the server to your MCP-compatible application.

- **Better Proposals integration**  
  - Connect a Better Proposals account through the configuration flow.  
  - Select a client in your Better Proposals account to start using proposal-related tools (where supported by the integration).

- **Client app compatibility**  
  - Can be added to various MCP chat clients or tools that support MCP servers.  
  - Configuration guidance available per supported client via the “Add the server to your app” flow.

- **Central configuration page**  
  - Dedicated configuration page on Pipedream for managing the Better Proposals MCP Server connection and settings.

*(The Pipedream interface references “available tools” that are loaded dynamically, but specific tool names or capabilities are not listed in the provided content.)*
