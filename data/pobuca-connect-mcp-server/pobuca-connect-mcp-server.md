# Pobuca Connect MCP Server

An MCP server that connects Pobuca Connect’s AI-powered contact management and business communication capabilities into MCP-compatible workflows.

## Overview
- **Type:** MCP server / integration
- **Use case:** Centralize and automate access to business contacts and communication features from Pobuca Connect within any MCP-compatible client.
- **Server URL:** `https://mcp.pipedream.net/v2`

## Features
- **MCP-compatible endpoint**  
  - Single static MCP server URL that works across different MCP clients.
  - Designed to plug Pobuca Connect capabilities into tools that speak the Model Context Protocol (MCP).

- **Pobuca Connect integration**  
  - Connects to Pobuca Connect’s contact management and business communication stack.  
  - Enables AI-supported handling of business contacts (e.g., organizing, searching, and using contacts in workflows).

- **Cross-client support**  
  - Same server URL for different chat / MCP clients.  
  - Authentication performed when adding the server in each client.

- **Workflow-ready**  
  - Built to integrate Pobuca Connect into larger automated workflows powered by Pipedream Connect and MCP.

## Setup & Configuration
- Use the static MCP server URL: `https://mcp.pipedream.net/v2`.
- Add the server within your MCP-compatible app or chat client and complete authentication during setup.
- Additional configuration details are available on the provider’s configuration page (referenced in the original content).
