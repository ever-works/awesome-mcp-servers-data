# Dots MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** dots

An MCP server that connects your Dots! account to AI tools and exposes multi-platform payment sending and receiving capabilities through a unified API.

## Description
Dots MCP Server is an MCP integration that connects your Dots! account to AI-powered tools (such as Cursor, Claude, and Windsurf). It provides a unified API surface for sending and receiving payments across multiple platforms, helping simplify payout workflows for MCP clients.

## Features
- **MCP integration for Dots!**: Connects your Dots! account to MCP-compatible tools.  
- **Multi-platform payments**: Exposes payment sending and receiving capabilities across multiple platforms.  
- **Unified API surface**: Offers a single, consistent API for handling payouts, reducing complexity across different payment channels.  
- **Payout workflow simplification**: Designed to streamline and automate payout workflows for MCP clients.
