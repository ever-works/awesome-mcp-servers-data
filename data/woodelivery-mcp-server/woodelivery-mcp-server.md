# WooDelivery MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** WooDelivery  
**Source:** https://mcp.pipedream.com/app/woodelivery

## Overview
WooDelivery MCP Server exposes WooDelivery’s delivery management and logistics capabilities through the Model Context Protocol (MCP). It allows chat-based or MCP-compatible applications to interact programmatically with WooDelivery for delivery and logistics workflows.

## Features
- MCP-compatible server providing access to WooDelivery delivery management functions.
- Designed for programmatic interaction with WooDelivery’s logistics capabilities from chat clients or other MCP-aware tools.
- Single static MCP endpoint for all clients: `https://mcp.pipedream.net/v2`.
- Authentication handled when adding/configuring the server in the client application.
- Hosted and delivered via Pipedream Connect infrastructure.

## Technical Details
- **MCP endpoint URL:** `https://mcp.pipedream.net/v2`
- **Integration model:** Add as an MCP server in compatible chat clients / applications.
