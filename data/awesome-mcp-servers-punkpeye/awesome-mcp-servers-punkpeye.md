# Awesome MCP Servers (punkpeye)

**Category:** MCP Server Directories & Lists  
**Source:** <https://github.com/punkpeye/awesome-mcp-servers>  
**Author/Maintainer:** Frank Fiegel (punkpeye)

## Overview
Awesome MCP Servers is a GitHub-hosted, curated list of Model Context Protocol (MCP) servers. It serves as a directory of popular MCP server integrations, helping users discover and explore available servers and tools that work with MCP-compatible clients.

## Features
- **Curated MCP server list** – A human-maintained collection of MCP servers gathered in one place.
- **Directory of integrations** – Focuses on MCP server integrations, making it easier to find servers that connect external tools and services via MCP.
- **Central GitHub repository** – Content is organized and versioned on GitHub, allowing users to browse, clone, or watch the repository.
- **Discoverability** – Serves as a starting point for exploring the MCP ecosystem and identifying servers suitable for different use cases.
