# QuickBooks Sandbox MCP Server

**Category:** Finance & Market Data MCP Servers  
**Brand:** Pipedream  
**Website:** https://mcp.pipedream.com/app/quickbooks_sandbox  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Description
QuickBooks Sandbox MCP Server is a Model Context Protocol (MCP) server from Pipedream that provides access to a sandboxed QuickBooks Online environment. It is intended for safe development and testing of MCP-powered financial and accounting workflows without affecting production accounting data.

## Features
- **Sandboxed QuickBooks Online environment** for non-production use, allowing experimentation and testing without impacting real financial records.
- **MCP-compliant server** designed to integrate with MCP-capable applications and chat clients.
- **Static MCP endpoint**: a single URL (`https://mcp.pipedream.net/v2`) used by all clients.
- **Per-client authentication**: authentication occurs when adding the server to each application or chat client (supports secure, user-specific connections).
- **Compatible with multiple chat clients**: can be added to different MCP-enabled chat interfaces (guides available per client type).
- **Configuration documentation**: additional setup and usage details available via a central Configuration page on Pipedream.
- **Built on Pipedream Connect**: leverages Pipedreamâ€™s integration platform for connectivity to QuickBooks Online in a sandbox context.

## Use Cases
- Developing and testing accounting workflows that interact with QuickBooks Online data.
- Prototyping financial automations (e.g., invoices, expenses, ledgers) in a safe environment.
- Validating MCP-based tools and prompts that read or manipulate QuickBooks data before using them in production.
