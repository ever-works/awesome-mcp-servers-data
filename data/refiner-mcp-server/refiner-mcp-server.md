# Refiner MCP Server

MCP Server for Refiner, enabling creation and automation of customer feedback surveys for SaaS and digital products via the Model Context Protocol.

---

## Overview
- **Name:** Refiner MCP Server  
- **Category:** Business & Commerce – MCP Servers  
- **Brand:** Refiner (via Pipedream MCP)  
- **Use Case:** Customer feedback surveys and forms for SaaS and other digital products, accessible via the Model Context Protocol (MCP).

---

## Features
- **MCP-Compatible Server**  
  - Provides a static MCP server endpoint: `https://mcp.pipedream.net/v2`  
  - Works across different MCP-enabled chat clients and applications.

- **Refiner Integration**  
  - Connects to a Refiner account to work with customer feedback surveys.  
  - Designed for SaaS and digital product feedback workflows.

- **Surveys & Forms**  
  - Oriented around creating and managing customer surveys and forms (details of individual actions are loaded dynamically as “available tools”).

- **Multi-Client Support**  
  - Can be added to various chat clients; each client can follow its own setup instructions.  
  - Uses the same static URL for all clients.

- **Authentication at Client Level**  
  - Authentication occurs when adding the server to your MCP-compatible application, rather than per-URL change.

- **Configuration Flow**  
  - Connect your Refiner account.  
  - Select the relevant client within the configuration UI.  
  - Copy and paste the MCP server URL into your chat or MCP client settings.  
  - Optional deeper setup via the full configuration page on Pipedream.

---

## Technical Details
- **MCP Endpoint:** `https://mcp.pipedream.net/v2`  
- **Protocol:** Model Context Protocol (MCP)  
- **Hosting/Provider:** Pipedream (MCP server infrastructure)
