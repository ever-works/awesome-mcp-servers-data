# Heap MCP Server

**Category:** Data Analysis & Exploration – MCP Servers  
**Brand:** Heap  
**Source:** https://mcp.pipedream.com/app/heap

Heap MCP Server is an MCP-compatible server that exposes Heap’s automatically captured web and mobile behavioral analytics to MCP agents, enabling product analysis and insights programmatically.

---

## Features

- **MCP-compatible server endpoint**  
  - Static server URL for all MCP clients: `https://mcp.pipedream.net/v2`  
  - Can be added as an MCP server to compatible chat or agent clients.

- **Access to Heap analytics**  
  - Interfaces with Heap to provide behavioral analytics data from web and mobile applications.  
  - Enables use of automatically captured event data for product and user behavior analysis.

- **Agent / client integration**  
  - Designed for MCP agents and chat clients that support MCP servers.  
  - Authentication handled when adding the server to the application.

- **Retroactive behavioral analysis**  
  - Supports querying historically captured behavioral data without additional client-side instrumentation changes (via Heap’s automatic capture capabilities).

- **No-code / low-code setup for clients**  
  - One static URL for configuration across different MCP-compatible tools.  
  - Detailed setup instructions available via the referenced configuration page (not included here).

---

## Usage

- Add `https://mcp.pipedream.net/v2` as an MCP server in your supported MCP client.  
- Authenticate with Heap when prompted by your client to allow access to your Heap project’s behavioral analytics.
