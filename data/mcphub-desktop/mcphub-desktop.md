# MCPHub Desktop

**Category:** Server Management Tools  
**Slug:** `mcphub-desktop`

## Description
MCPHub Desktop is an open-source macOS and Windows application that provides a graphical interface for discovering, installing, and managing MCP servers on your machine.

## Features
- **Graphical user interface (GUI)** for working with MCP servers
- **Server discovery** to find available MCP servers
- **One-click / guided installation** of MCP servers
- **Server management tools** (e.g., viewing and handling installed MCP servers)
- **Cross-platform desktop app** for macOS and Windows
- **Open-source codebase** hosted on GitHub
