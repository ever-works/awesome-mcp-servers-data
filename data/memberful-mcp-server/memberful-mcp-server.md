# Memberful MCP Server

**Category:** Business & Commerce MCP Servers  
**Brand:** Memberful  
**Slug:** `memberful-mcp-server`

## Description
The Memberful MCP Server connects AI-driven MCP clients (such as Cursor, Claude, and Windsurf) to a Memberful account, enabling access to Memberful’s membership management capabilities for creators, publishers, and educators.

## Features
- Connects a Memberful account to MCP-compatible AI tools via Pipedream’s MCP server
- Bridges AI development environments (e.g., Cursor, Claude, Windsurf) with Memberful membership data and functionality
- Supports workflows related to memberships, subscriptions, and creator-focused membership management
