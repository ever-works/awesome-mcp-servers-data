# Sympla MCP Server

An MCP server that connects your Sympla account (Brazil’s leading self-service event platform) to AI tools.

## Overview
- **Type:** MCP server / integration
- **Category:** Business & Commerce MCP Servers
- **Use case:** Event management and ticketing operations via AI tools
- **Source URL:** https://mcp.pipedream.com/app/sympla

## Features
- Connects a Sympla account to AI tools that support MCP (e.g., Cursor, Claude, Windsurf)
- Provides MCP-based access to event management operations
- Provides MCP-based access to ticketing operations
- Built on Pipedream’s MCP server infrastructure

## Integrations
- Sympla (event and ticketing platform)
- AI tools supporting MCP, including:
  - Cursor
  - Claude
  - Windsurf
