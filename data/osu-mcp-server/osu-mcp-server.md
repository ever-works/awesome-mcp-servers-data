# Osu MCP Server

Osu MCP Server is a Model Context Protocol (MCP) server for the osu! rhythm game ecosystem, enabling MCP-enabled tools and applications to access osu! game data and related services.

- **Website / Source**: https://mcp.pipedream.com/app/osu
- **Category**: Game Server MCP Implementations
- **Tags**: gaming, game-data, community

## Overview

Osu MCP Server integrates the osu! rhythm game ecosystem with MCP-compatible clients, providing a standardized interface for accessing osu!-related data and services.

## MCP Server Endpoint

- **Base MCP Server URL**: `https://mcp.pipedream.net/v2`
- This static URL is used by all supported clients; authentication is handled when adding the server to your application.
