# Rapid7 Insight Platform MCP Server

Security analytics MCP server that enables tools and chat clients to interact with the Rapid7 Insight Platform to help find, manage, and reduce security risk.

**Website:** https://mcp.pipedream.com/app/rapid7_insight_platform  
**MCP Server URL:** `https://mcp.pipedream.net/v2`

## Features
- Connects MCP-compatible clients to the Rapid7 Insight Platform.  
- Provides access to security analytics data for risk-related workflows.  
- Supports interaction through a single static MCP server endpoint (`https://mcp.pipedream.net/v2`).  
- Authentication handled when adding the server within the client or application.  
- Usable across multiple chat clients and MCP-enabled applications via the same URL.  
- Configuration guidance available via a dedicated configuration page (through Pipedream Connect).
