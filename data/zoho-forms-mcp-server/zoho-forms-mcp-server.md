# Zoho Forms MCP Server

An MCP server that connects MCP-based tools and agents to Zoho Forms for online form building and data collection.

## Features

- **MCP Server Endpoint**
  - Static MCP server URL for all clients: `https://mcp.pipedream.net/v2`
  - Authentication occurs when adding the server to a client or application.

- **Form Creation & Sharing**
  - Online form building capabilities.
  - Share forms on the web for collecting responses.

- **Data Collection & Management**
  - Collect responses submitted through Zoho Forms.
  - Manage collected data using integrated applications within the Zoho ecosystem (as supported by Zoho Forms).

- **Notifications**
  - Receive alerts when new form submissions are received ("instant alerts" as supported by Zoho Forms via this integration).

- **Integrations via Pipedream Connect**
  - Powered by Pipedream Connect, enabling usage of Zoho Forms within broader automation and MCP-based workflows.
