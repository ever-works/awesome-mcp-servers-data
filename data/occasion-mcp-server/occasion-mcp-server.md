# Occasion MCP Server

An MCP server that connects to Occasion, enabling booking and scheduling workflows through MCP-compatible clients.

## Overview
- **Type:** MCP Server (Model Context Protocol)
- **Category:** Business & Commerce – MCP Servers
- **Provider / Brand:** Occasion (via Pipedream Connect)
- **Use Cases:** Booking, scheduling, event reservations via chat or MCP-enabled applications
- **MCP Endpoint:** `https://mcp.pipedream.net/v2` (static URL for all clients; authentication handled when adding the server)

## Features
- **Occasion Integration**
  - Connects MCP clients to Occasion’s booking and scheduling capabilities.
  - Designed for fast creation and management of bookings and events.

- **Standard MCP Server Endpoint**
  - Uses a single static URL (`https://mcp.pipedream.net/v2`) that works across compatible clients.
  - Authentication occurs during server configuration inside the client or application.

- **Client-Agnostic Setup**
  - Intended to be added to various MCP-aware chat clients and tools.
  - Configuration guidance available via the linked configuration documentation (Pipedream Connect).

- **Infrastructure by Pipedream Connect**
  - Hosted and delivered through Pipedream’s MCP infrastructure.
  - Benefits from Pipedream’s standardized MCP server deployment model.
