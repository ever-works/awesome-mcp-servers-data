# Multicluster MCP Server

## Overview
Multicluster MCP Server (OCM MCP Server) is an MCP (Model Context Protocol) server that acts as a gateway for Generative AI systems to interact with multiple Kubernetes clusters via Open Cluster Management. It enables multi-cluster Kubernetes resource management, operations, and observability from GenAI-driven workflows.

- **Category:** Cloud & DevOps – MCP Servers  
- **Source:** [GitHub – yanmxa/multicluster-mcp-server](https://github.com/yanmxa/multicluster-mcp-server)  
- **Tags:** kubernetes, cluster-management, cloud-native

## Features

### MCP Tools & Kubernetes Cluster Awareness
- Interact with Kubernetes clusters through the Model Context Protocol (MCP).
- Retrieve Kubernetes resources from the **hub cluster** (current context).
- Retrieve Kubernetes resources from **managed clusters**.
- Connect to a managed cluster using a specified **ClusterRole**.
- Access and operate on resources **across multiple Kubernetes clusters** via Open Cluster Management.
- Retrieve and analyze **metrics, logs, and alerts** from integrated clusters for observability.
- Interact with multi-cluster Open Cluster Management APIs, including:
  - Managed Clusters
  - Policies
  - Add-ons
  - Other multi-cluster resources exposed via OCM
