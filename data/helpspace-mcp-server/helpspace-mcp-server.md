# Helpspace MCP Server

## Overview
Helpspace MCP Server is a Model Context Protocol (MCP) server that connects AI tools to Helpspace’s shared inbox and customer support workspace. It allows applications to access and manage customer messages in Helpspace via a standardized MCP interface.

- **Category:** Business & Commerce – MCP Servers  
- **Brand:** Helpspace  
- **MCP Server URL:** `https://mcp.pipedream.net/v2`  

## Features
- **Shared inbox access**  
  - Connects AI tools to Helpspace’s shared inbox.  
  - Centralizes incoming customer messages in one place for easier handling.

- **Customer support integration**  
  - Provides access to Helpspace’s customer support workspace via MCP.  
  - Designed to let AI tools manage and organize customer conversations.

- **Team collaboration enablement**  
  - Supports organizing messages for efficient teamwork within a shared inbox environment.

- **Static MCP endpoint**  
  - Uses a single, static MCP server URL (`https://mcp.pipedream.net/v2`) for all clients.  
  - Authentication occurs when adding/configuring the server in the client app.

- **Client-agnostic setup**  
  - Can be added to different MCP-compatible chat clients or applications.  
  - Configuration guidance available through a generic "Configuration" page (client-specific instructions not embedded here).

## Integration & Configuration
- Add the MCP server to any compatible client using the static URL.  
- Authentication is performed during server addition in the client.  
- Further configuration details are available through the referenced configuration documentation (not included in this summary).
