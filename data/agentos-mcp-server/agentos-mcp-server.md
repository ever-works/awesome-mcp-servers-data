# agentOS MCP Server

**Category:** Business & Commerce – MCP Servers  
**Website:** https://mcp.pipedream.com/app/agentos

## Description
The agentOS MCP Server connects the agentOS real-estate CRM platform and related PropTech APIs to AI tools via Pipedream’s MCP server, enabling MCP-driven automations for real-estate workflows.

## Features
- Connects an agentOS account to AI tools through Pipedream’s MCP server
- Integrates with AI tools such as Cursor, Claude, and Windsurf
- Exposes agentOS real-estate CRM and PropTech APIs to MCP-based automations
- Enables automation of real-estate CRM and PropTech-related workflows

## Integrations
- **agentOS** (real-estate CRM / PropTech platform)
- **Cursor**
- **Claude**
- **Windsurf**
- **Pipedream MCP Server**
