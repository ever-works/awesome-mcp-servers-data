# Deepgram MCP Server

MCP server integration for Deepgram, providing intelligent, high-accuracy speech recognition for business applications.

- **Category:** Media Processing MCP Servers  
- **Brand:** Deepgram  
- **Source URL:** https://mcp.pipedream.com/app/deepgram  
- **Slug:** `deepgram-mcp-server`

## Features

- **MCP server integration for Deepgram** to enable speech-to-text capabilities inside MCP-compatible clients.
- **High-accuracy speech recognition** designed for business applications.
- **Static MCP server URL** usable across clients:  
  - Endpoint: `https://mcp.pipedream.net/v2`
- **Client-agnostic setup**: the same server URL works for every supported chat client; authentication is handled when adding the server to the application.
- **Configuration guidance available** via a dedicated configuration page (linked from the app page) for setup details.
- **Built on Pipedream Connect** infrastructure for hosting the MCP server.

## Technical Details

- **MCP Server URL:** `https://mcp.pipedream.net/v2`  
- **Authentication:** Performed when you add the MCP server to your chat client or application (no per-client URL change required).
